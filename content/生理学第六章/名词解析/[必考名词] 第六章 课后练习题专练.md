# 第六章 课后练习题名词解释

- **慢波电位**
  - 定义：消化道平滑肌**自发**的、**周期性**产生的缓慢、轻度的**去极化和复极化**电位波动，又称**基本电节律**
  - 正常值/公式：由 **Cajal 间质细胞（ICC）**产生 → ICC 是胃肠运动的**起搏细胞**；频率决定收缩节律 → 胃 **3 次/min**、十二指肠 **11~12 次/min**；幅度 **10~15mV**、时程数秒~十几秒；静息电位 **-50~-60mV**
  - 生理意义/特征：慢波本身不引起收缩，但为动作电位提供**基础**；慢波 → 达**机械阈** → 平滑肌轻微收缩；慢波 → 达**电阈** → 触发 **AP** → 平滑肌明显收缩；慢波频率决定了消化道平滑肌的**收缩节律**
- **脑-肠肽**
  - 定义：既存在于**中枢神经系统**（CNS）又存在于**胃肠道**内，呈**双重分布**的**肽类物质**
  - 正常值/公式：胃肠道内分泌细胞超过 **40 种**（APUD 细胞）；典型代表 → **促胃液素、缩胆囊素、促胰液素、抑胃肽、胃动素**等均为脑-肠肽
  - 生理意义/特征：体现了**神经系统与消化系统**之间在分子水平上的**密切联系**；同一肽类物质在中枢和外周可发挥**不同**的生理功能；是理解**脑-肠轴**概念的物质基础；对研究功能性胃肠病（如肠易激综合征）有重要临床意义
- **容受性舒张**
  - 定义：食物刺激**口腔、咽、食管**等多处感受器，**反射性**地引起**胃底和胃体**肌肉**舒张**、胃容积**扩大**的运动形式，是**胃特有**的运动
  - 正常值/公式：胃容积可由 **50ml 扩大至 1.5L**；机制 → 迷走神经释放**肽类物质（VIP，血管活性肠肽）**引起；区别于紧张性收缩和蠕动
  - 生理意义/特征：**容纳**摄入的大量食物而**胃内压不显著升高** → 防止食物过快进入十二指肠；为胃内消化提供**空间**储备；是实现胃**储存**功能的关键运动形式；切断迷走神经 → 容受性舒张**消失**
- **分节运动**
  - 定义：以肠壁**环形肌**为主的**节律性收缩和舒张**交替进行的运动形式，是**小肠特有**的运动
  - 正常值/公式：环形肌在食糜所在部位交替收缩/舒张 → 将食糜分成许多小节段、随后重新组合；频率由慢波节律决定
  - 生理意义/特征：**搅拌**食糜 → 促进**化学性消化**（食糜与消化液充分混合）；使食糜与肠壁**反复接触** → 促进**吸收**；不主要起推进作用（推进由蠕动完成）；是理解小肠消化吸收**高效性**的运动基础
- **胆盐的肠-肝循环**
  - 定义：胆汁中的**胆盐**在**回肠**被重吸收、经**门静脉**回到肝脏、被肝细胞重新摄取并再次分泌入胆汁的**循环利用**过程
  - 正常值/公式：约 **95%** 的胆盐在**回肠**被重吸收 → 经**门静脉**回肝脏 → 每日循环 **6~12 次**；每次循环约有 5% 的胆盐随粪便排出 → 肝脏需不断合成补充
  - 生理意义/特征：**节约胆盐** → 有限的胆盐池（约 **3~5g**）通过循环满足每日乳化脂肪的**大量需求**（每日分泌胆盐约 **12~24g**）；维持胆汁的**持续分泌**；肠-肝循环**中断**（如回肠切除）→ 胆盐大量流失 → 脂肪**消化吸收障碍** → 脂肪泻；口服**考来烯胺**（胆盐螯合剂）可阻断肠-肝循环 → 降胆固醇
