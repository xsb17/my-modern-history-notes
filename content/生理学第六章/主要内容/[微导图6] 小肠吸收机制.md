# 小肠吸收机制 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 小肠为主要吸收部位的有利条件 ★简答题&nbsp;&nbsp;&nbsp;
  - 消化**充分**（胰液/胆汁/小肠液协同）&nbsp;&nbsp;&nbsp;
  - 食物停留时间长：**3~8h**&nbsp;&nbsp;&nbsp;
  - 吸收面积巨大：**200~250m²**（绒毛+微绒毛）&nbsp;&nbsp;&nbsp;
  - 绒毛内**毛细血管**和**毛细淋巴管**丰富&nbsp;&nbsp;&nbsp;
- 吸收途径与机制&nbsp;&nbsp;&nbsp;
  - 途径：跨细胞途径 + 细胞旁途径&nbsp;&nbsp;&nbsp;
  - 机制：被动转运 + 主动转运 + 出胞/入胞&nbsp;&nbsp;&nbsp;
- 水与无机盐吸收&nbsp;&nbsp;&nbsp;
  - 水：被动吸收 -> 总量 **8 L/d**&nbsp;&nbsp;&nbsp;
  - 钠：腔面膜易化扩散 + 基底侧膜**钠泵**主动转运 -> 促进其他物质吸收&nbsp;&nbsp;&nbsp;
  - 铁：Fe³⁺->Fe²⁺ 后吸收 -> **胃酸/VitC** 促进吸收&nbsp;&nbsp;&nbsp;
  - 钙：受 **VitD₃** 促进 -> 十二指肠主动吸收&nbsp;&nbsp;&nbsp;
- 维生素吸收&nbsp;&nbsp;&nbsp;
  - 水溶性（B/C）：被动扩散/主动转运&nbsp;&nbsp;&nbsp;
  - 脂溶性（A/D/E/K）：与脂类相似 -> 需胆盐参与&nbsp;&nbsp;&nbsp;
- ★三大营养物质吸收对比（必考）&nbsp;&nbsp;&nbsp;
  - 糖：&nbsp;&nbsp;&nbsp;
    - 吸收形式：**单糖**&nbsp;&nbsp;&nbsp;
    - 转运方式：**继发性主动转运**（Na⁺-葡萄糖同向转运）&nbsp;&nbsp;&nbsp;
    - 吸收途径：**血液**&nbsp;&nbsp;&nbsp;
  - 蛋白质：&nbsp;&nbsp;&nbsp;
    - 吸收形式：**氨基酸**&nbsp;&nbsp;&nbsp;
    - 转运方式：**继发性主动转运**&nbsp;&nbsp;&nbsp;
    - 吸收途径：**血液**&nbsp;&nbsp;&nbsp;
  - 脂肪：&nbsp;&nbsp;&nbsp;
    - 吸收形式：**脂肪酸/甘油一酯**&nbsp;&nbsp;&nbsp;
    - 转运方式：与胆盐 -> **混合微胶粒** -> 被动扩散入细胞 -> 重新合成甘油三酯 -> 乳糜微粒&nbsp;&nbsp;&nbsp;
    - 吸收途径：**淋巴途径为主**（长链脂肪酸）-> 经胸导管入血&nbsp;&nbsp;&nbsp;
- 各部位吸收特点&nbsp;&nbsp;&nbsp;
  - 口腔/食管：少数药物&nbsp;&nbsp;&nbsp;
  - 胃：**乙醇、药物**&nbsp;&nbsp;&nbsp;
  - 大肠：水、无机盐、维生素（细菌合成 **VitB/K**）&nbsp;&nbsp;&nbsp;
  - 小肠：主要吸收部位（所有营养物质）&nbsp;&nbsp;&nbsp;
