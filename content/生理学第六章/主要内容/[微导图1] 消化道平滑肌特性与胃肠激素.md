# 消化道平滑肌特性与胃肠激素 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 消化道平滑肌一般特性&nbsp;&nbsp;&nbsp;
  - 自动节律性：节律**缓慢而不规则**（不同于心脏）&nbsp;&nbsp;&nbsp;
  - 兴奋性低、收缩缓慢（不同于骨骼肌）&nbsp;&nbsp;&nbsp;
  - 伸展性大：胃 **50ml → 1500ml**&nbsp;&nbsp;&nbsp;
  - 紧张性：微弱而持续的收缩状态&nbsp;&nbsp;&nbsp;
  - 刺激敏感性：对**牵张/化学/温度**敏感 -> 对**电刺激**不敏感&nbsp;&nbsp;&nbsp;
- 消化道平滑肌电生理特性&nbsp;&nbsp;&nbsp;
  - 静息电位：**-50~-60mV** -> K⁺ 平衡电位为主&nbsp;&nbsp;&nbsp;
  - ★慢波电位（基本电节律）：&nbsp;&nbsp;&nbsp;
    - 由 **Cajal 间质细胞（ICC）**产生 -> 胃肠运动**起搏细胞**&nbsp;&nbsp;&nbsp;
    - 频率：胃 **3次/min** | 十二指肠 **11~12次/min**&nbsp;&nbsp;&nbsp;
    - 幅度 **10~15mV** | 时程数秒~十几秒&nbsp;&nbsp;&nbsp;
    - 慢波 -> 达**机械阈** -> 平滑肌轻微收缩&nbsp;&nbsp;&nbsp;
    - 慢波 -> 达**电阈** -> 触发 **AP** -> 明显收缩&nbsp;&nbsp;&nbsp;
  - 动作电位：&nbsp;&nbsp;&nbsp;
    - 去极缓慢/幅度小 -> 主要由 **Ca²⁺ 内流**引起&nbsp;&nbsp;&nbsp;
    - AP 频率 ∝ 收缩力（**正比**）&nbsp;&nbsp;&nbsp;
- 消化腺分泌概况&nbsp;&nbsp;&nbsp;
  - 分泌量：**6~8 L/d** -> 成分：水+消化酶+电解质&nbsp;&nbsp;&nbsp;
- 神经支配&nbsp;&nbsp;&nbsp;
  - 副交感（迷走/盆）：**ACh** -> 分泌↑/运动↑/括约肌**舒张**&nbsp;&nbsp;&nbsp;
  - 交感：**NA** -> 分泌↓/运动↓/括约肌**收缩**&nbsp;&nbsp;&nbsp;
  - 壁内神经丛：可**独立**调节消化活动&nbsp;&nbsp;&nbsp;
- ★脑-肠肽定义&nbsp;&nbsp;&nbsp;
  - 既存在于**CNS**又存在于**胃肠道** -> 双重分布的肽类物质&nbsp;&nbsp;&nbsp;
- ★五大胃肠激素（必考）&nbsp;&nbsp;&nbsp;
  - 促胃液素：**G 细胞**（胃窦/十二指肠）-> 蛋白质消化产物刺激 -> **胃酸/胃蛋白酶↑** + 胃肠运动↑ + 上皮生长&nbsp;&nbsp;&nbsp;
  - 缩胆囊素：**I 细胞**（小肠上部）-> 蛋白质/脂肪酸刺激 -> **胆囊收缩 + 胰酶↑** + 肠运动↑&nbsp;&nbsp;&nbsp;
  - 促胰液素：**S 细胞**（小肠上部）-> 盐酸/脂肪酸刺激 -> **水/HCO₃⁻↑** + 胃酸/胃运动↓&nbsp;&nbsp;&nbsp;
  - 抑胃肽：**K 细胞**（小肠上部）-> 葡萄糖/脂肪酸刺激 -> **胃酸↓/胃排空↓/胰岛素↑**&nbsp;&nbsp;&nbsp;
  - 胃动素：**Mo 细胞**（小肠）-> 消化期刺激 **胃和小肠运动**&nbsp;&nbsp;&nbsp;
