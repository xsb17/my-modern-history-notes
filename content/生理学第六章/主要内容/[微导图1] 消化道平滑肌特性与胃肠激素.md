# 消化道平滑肌特性与胃肠激素 — 微型速记导图

- 消化道平滑肌一般特性
  - 自动节律性：节律**缓慢而不规则**（不同于心脏）
  - 兴奋性低、收缩缓慢（不同于骨骼肌）
  - 伸展性大：胃 **50ml → 1500ml**
  - 紧张性：微弱而持续的收缩状态
  - 刺激敏感性：对**牵张/化学/温度**敏感 -> 对**电刺激**不敏感
- 消化道平滑肌电生理特性
  - 静息电位：**-50~-60mV** -> K⁺ 平衡电位为主
  - ★慢波电位（基本电节律）：
    - 由 **Cajal 间质细胞（ICC）**产生 -> 胃肠运动**起搏细胞**
    - 频率：胃 **3次/min** | 十二指肠 **11~12次/min**
    - 幅度 **10~15mV** | 时程数秒~十几秒
    - 慢波 -> 达**机械阈** -> 平滑肌轻微收缩
    - 慢波 -> 达**电阈** -> 触发 **AP** -> 明显收缩
  - 动作电位：
    - 去极缓慢/幅度小 -> 主要由 **Ca²⁺ 内流**引起
    - AP 频率 ∝ 收缩力（**正比**）
- 消化腺分泌概况
  - 分泌量：**6~8 L/d** -> 成分：水+消化酶+电解质
- 神经支配
  - 副交感（迷走/盆）：**ACh** -> 分泌↑/运动↑/括约肌**舒张**
  - 交感：**NA** -> 分泌↓/运动↓/括约肌**收缩**
  - 壁内神经丛：可**独立**调节消化活动
- ★脑-肠肽定义
  - 既存在于**CNS**又存在于**胃肠道** -> 双重分布的肽类物质
- ★五大胃肠激素（必考）
  - 促胃液素：**G 细胞**（胃窦/十二指肠）-> 蛋白质消化产物刺激 -> **胃酸/胃蛋白酶↑** + 胃肠运动↑ + 上皮生长
  - 缩胆囊素：**I 细胞**（小肠上部）-> 蛋白质/脂肪酸刺激 -> **胆囊收缩 + 胰酶↑** + 肠运动↑
  - 促胰液素：**S 细胞**（小肠上部）-> 盐酸/脂肪酸刺激 -> **水/HCO₃⁻↑** + 胃酸/胃运动↓
  - 抑胃肽：**K 细胞**（小肠上部）-> 葡萄糖/脂肪酸刺激 -> **胃酸↓/胃排空↓/胰岛素↑**
  - 胃动素：**Mo 细胞**（小肠）-> 消化期刺激 **胃和小肠运动**
