# 胆汁的作用与小肠运动 — 微型速记导图

- 胆汁来源：肝细胞合成 -> 肝胆汁（弱碱性）/ 胆囊胆汁（弱酸性）
- 胆汁成分：水、碳酸氢盐、胆色素、胆固醇、**胆盐**、磷脂
- 关键特征：**无消化酶** -> 胆盐是主要功能成分
- ★胆汁四大作用（必考简答）
  - ① **乳化脂肪**：胆盐+卵磷脂 -> 增加胰脂肪酶**作用面积**
  - ② **促进脂肪分解产物和脂溶性维生素吸收**：
    - 脂肪酸 + 胆盐 -> **混合微胶粒**（水溶性复合物）-> 利于肠粘膜吸收
    - 促进脂溶性维生素 **A/D/E/K** 吸收
  - ③ 中和胃酸
  - ④ 胆盐的**利胆作用**：促进胆汁自身分泌
- ★胆盐的肠-肝循环（必考名词解释）
  - **95%** 的胆盐在**回肠**重吸收 -> 经**门静脉**回肝脏 -> **循环利用**
  - 意义：节约胆盐、维持胆汁持续分泌
- 胆固醇结石机制
  - 胆汁中胆固醇**↑**/胆盐**↓** -> 胆固醇沉积 -> 结石
- 胆汁分泌调节
  - 体液（主要）：缩胆囊素->胆囊收缩排胆汁 + 促胰液素->水/HCO₃⁻ + 胆盐肠肝循环
  - 神经（弱）：迷走神经->ACh->分泌/排出
- 小肠液
  - 性质：弱碱性 -> 含**肠激酶**（激活胰蛋白酶原的关键酶）
- ★小肠运动形式
  - 分节运动（**小肠特有**）：
    - **环形肌**为主的节律性收缩/舒张交替
    - 意义：**搅拌**（促化学性消化）+ **促吸收**
  - 紧张性收缩（共有）
  - 蠕动（共有）：推进食糜；蠕动冲：快速远距离推送
