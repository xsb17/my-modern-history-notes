# 胰液的成分与调节 — 微型速记导图（最重要）

- 胰液定位：**作用最全面、消化力最强、最重要**的消化液
- 性质：碱性 **pH 7.8~8.4** | 分泌量 **1~2 L/d**
- 分泌细胞分工
  - 腺泡细胞 -> **消化酶**
  - 导管细胞 -> **碳酸氢盐 + 水**
- ★胰液五大成分与作用
  - 碳酸氢盐：中和胃酸 -> 为小肠酶提供适宜 pH
  - 糖类消化酶：胰淀粉酶 -> 淀粉 -> 麦芽糖
  - ★蛋白质消化酶（以酶原形式分泌★安全机制）：
    - 胰蛋白酶原 ->（**肠激酶**激活）-> **胰蛋白酶** -> 还可激活糜蛋白酶原
    - 糜蛋白酶原 ->（胰蛋白酶激活）-> 糜蛋白酶
    - 羧基肽酶
  - 脂类消化酶：
    - 胰脂肪酶 + **辅脂酶**（锚定）-> 甘油三酯 -> 甘油/脂肪酸/一酰甘油
    - 胆固醇酯酶
  - 核酸消化酶：DNA酶/RNA酶 -> 核苷酸
- ★急性胰腺炎发生机制（必考）
  - 暴饮暴食 -> 胰液分泌**过多** -> 胰管内压力**↑**
  - -> 胰管和腺泡**破裂** -> 胰蛋白酶原溢入**胰腺间质**
  - -> 被**组织液激活** -> **自身消化**胰腺 -> 急性胰腺炎
  - 正常保护：酶原形式 + 胰蛋白酶抑制物
- ★胰液分泌调节（体液为主）
  - 神经调节：迷走神经 -> **ACh** -> 腺泡细胞 -> **胰酶↑**
  - ★体液调节核心对比（必考）：
    - **促胰液素** -> **导管细胞** -> **水/HCO₃⁻↑**（量大/稀薄）
    - **缩胆囊素**（促胰酶素）-> **腺泡细胞** -> **胰酶↑**（酶多/粘稠）
- 胰液调节口诀
  - 促胰液素 = 导管 = 水多碱高
  - 缩胆囊素 = 腺泡 = 酶多力强
