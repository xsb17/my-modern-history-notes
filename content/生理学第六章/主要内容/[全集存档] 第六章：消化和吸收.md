# 第六章：消化和吸收 — 全集存档

## 第一节 消化生理概述

### 一、消化的方式

- 机械性消化：消化道平滑肌舒缩 → 搅拌、碾磨、推进食物
- 化学性消化：消化液中的消化酶对食物进行化学分解
- 两种方式同时存在，相互配合

### 二、消化道平滑肌的生理特性

#### （一）一般特性

- 自动节律性：节律缓慢而不规则（区别于心脏）
- 兴奋性低、收缩缓慢（区别于骨骼肌）
- 伸展性大：胃容积 **50ml → 1500ml**
- 紧张性：经常保持微弱而持续的收缩状态
- 刺激敏感性：对**牵张、化学、温度**刺激敏感；对电刺激不敏感

#### （二）电生理特性

- 静息电位：**-50~-60mV**，K⁺ 平衡电位为主（Na⁺、Ca²⁺ 参与）
- 慢波电位（基本电节律）：
  - 由 **Cajal 间质细胞（ICC）**产生 → 胃肠运动的**起搏细胞**
  - 频率决定收缩节律：胃 **3 次/min**，十二指肠 **11~12 次/min**
  - 幅度 **10~15mV**，时程数秒~十几秒
  - 慢波 → 达**机械阈** → 平滑肌轻微收缩
  - 慢波 → 达**电阈** → 触发 AP → 平滑肌明显收缩
- 动作电位：去极化缓慢且幅度小；去极化主要由 **Ca²⁺ 内流**引起；AP 频率与收缩力呈**正比**

### 三、消化腺的分泌

- 消化液分泌量：**6~8 L/d**
- 主要成分：水、消化酶、电解质
- 为消化酶提供 pH 环境

### 四、消化道活动的神经调节

#### （一）外来神经系统（自主神经）

- 副交感神经（迷走神经/盆神经）：释放 **ACh** → 消化腺**分泌↑**、消化道**运动↑**、括约肌**舒张**
- 交感神经：释放 **NA** → 消化腺分泌**↓**、消化道运动**↓**、括约肌**收缩**

#### （二）内在神经系统（壁内神经丛）

- 粘膜下神经丛 + 肌间神经丛
- 可**独立**调节消化活动，一般情况下受自主神经支配

### 五、消化道活动的体液调节——胃肠激素

- 胃肠激素：胃肠道粘膜内分泌细胞分泌的激素（**40 多种**，APUD 细胞）
- **脑-肠肽**：既存在于 CNS 又存在于胃肠道内、呈双重分布的肽类物质
- 生理作用：
  - ① 调节消化腺分泌和消化道运动
  - ② 调节其他激素释放（如抑胃肽 → 刺激胰岛素分泌）
  - ③ **营养作用**：促进消化道组织代谢和生长

#### 五种重要胃肠激素

- **促胃液素**（G 细胞，胃窦/十二指肠）：蛋白质消化产物刺激 → 胃酸/胃蛋白酶分泌↑、胃肠运动↑、胃肠上皮生长
- **缩胆囊素**（I 细胞，小肠上部）：蛋白质消化产物/脂肪酸刺激 → 胆囊收缩、胰酶分泌↑、肠运动↑、胰腺外分泌部生长
- **促胰液素**（S 细胞，小肠上部）：盐酸/脂肪酸刺激 → 胰液及胆汁中水/HCO₃⁻ 分泌↑、肠运动↓、胃酸分泌和胃运动↓
- **抑胃肽**（K 细胞，小肠上部）：葡萄糖/脂肪酸/氨基酸刺激 → 胃酸和胃蛋白酶分泌↓、抑制胃排空、胰岛素分泌↑
- **胃动素**（Mo 细胞，小肠）：迷走神经/盐酸/脂肪酸刺激 → 在消化期刺激胃和小肠运动

## 第三节 胃内消化

### 一、胃液的分泌

#### （一）胃液的性质、成分和作用

- 性质：酸性（**pH 0.9~1.5**）
- 分泌量：**1.5~2.5 L/d**
- 成分：盐酸、胃蛋白酶原、内因子、粘液

##### 1. 盐酸（胃酸）

- 分泌细胞：泌酸腺**壁细胞**
- 存在形式：游离酸 + 结合酸
- 基础排酸量：0~5 mmol/h；最大排酸量：20~25 mmol/h（反映壁细胞数目和功能）
- 分泌机制：壁细胞质子泵（**H⁺-K⁺-ATP 酶**）→ 质子泵抑制剂：**奥美拉唑**；餐后碱潮（HCO₃⁻ 入血）
- 生理作用：
  - ① 杀死胃内细菌
  - ② 使蛋白质**变性**，易于消化
  - ③ **激活胃蛋白酶原**，提供酸性环境
  - ④ 刺激促胰液素、缩胆囊素释放
  - ⑤ 促进小肠对 **Ca²⁺、Fe²⁺** 的吸收
- 胃酸不足 → 消化不良；胃酸过多 → 消化性溃疡

##### 2. 胃蛋白酶原

- 分泌细胞：泌酸腺**主细胞**
- 激活：胃蛋白酶原 →（HCl/胃蛋白酶）→ **胃蛋白酶**
- 作用：水解蛋白质 → 胨、胨及少量多肽和氨基酸

##### 3. 粘液

- 分泌细胞：胃粘膜上皮细胞、贲门腺、幽门腺、泌酸腺黏液颈细胞
- 作用：**粘液-碳酸氢盐屏障**（保护胃粘膜、润滑、防 HCl 和胃蛋白酶损伤）

##### 4. 内因子

- 分泌细胞：泌酸腺**壁细胞**
- 作用：保护 VitB₁₂ + 促进**回肠**对 VitB₁₂ 的吸收
- 缺乏 → **巨幼红细胞性贫血**

#### （二）胃的自身保护与损伤因素

- 保护：粘液-碳酸氢盐屏障 + 胃黏膜屏障（上皮细胞紧密连接）+ 细胞保护（PGE₂/PGI₂/EGF）
- 损伤因素：**酒精、阿司匹林、幽门螺杆菌** → 破坏屏障 → 胃炎/胃溃疡

#### （三）消化期胃液分泌的三期

##### 1. 头期（量占 **30%**）

- 食物进入胃之前，刺激头部感受器引起
- 机制：条件反射（视听嗅）+ 非条件反射（食物刺激口腔/食管）→ 迷走神经 → 胃腺 + G 细胞→促胃液素
- 特点：**胃蛋白酶多、胃酸多**

##### 2. 胃期（量占 **60%**）

- 食物入胃，机械+化学刺激引起
- 机制：迷走-迷走反射 + 壁内神经丛反射 + 食物化学成分→G 细胞→促胃液素
- 特点：**胃酸多、胃蛋白酶减少**

##### 3. 肠期（量占 **10%**）

- 食物进入小肠后引起
- 机制：食物 → 十二指肠 G 细胞 → 促胃液素 → 胃腺
- 特点：**胃酸少、胃蛋白酶少**

#### （四）调节胃液分泌的因素

##### 促进因素

- 迷走神经：释放 **ACh + M 受体** → 阻断剂：**阿托品**
- 促胃液素：**G 细胞**分泌 → 阻断剂：**丙谷胺**
- 组胺：肠嗜铬样细胞（**ECL 细胞**）分泌 → 阻断剂：**甲氰咪呱**（H₂ 受体拮抗剂）
- 其他：咖啡、酒精等刺激胃酸分泌

##### 抑制因素

- 盐酸：负反馈 → 抑制促胃液素，刺激促胰液素
- 脂肪：刺激小肠黏膜分泌**肠抑胃素**（促胰液素/缩胆囊素/抑胃肽）
- 高张溶液：肠胃反射

### 二、胃的运动

#### （一）胃的运动形式

##### 1. 容受性舒张（胃特有）

- 概念：食物刺激口腔/咽/食管 → 反射性引起胃底/胃体肌肉舒张 → 胃容积扩大（**50ml→1.5L**）
- 意义：**容纳食物**、维持胃内压不变
- 机制：迷走神经释放**肽类物质（VIP）**引起

##### 2. 紧张性收缩（共有）

- 概念：胃经常保持的一种持续而微弱的收缩状态
- 意义：促进胃液渗入食物、增加胃内压、维持胃的正常形态和位置、是蠕动的基础

##### 3. 蠕动（共有）

- 概念：消化管前端舒张、后端收缩的向前推进波形运动
- 意义：**推进、研磨、搅拌**食物

#### （二）胃排空

- 概念：食物间断地由胃排入十二指肠的过程
- 排空时间：**糖类 < 蛋白质 < 脂肪**，混合食物需 **4~6 小时**
- 胃内促进因素：胃内容物扩张 + 促胃液素 → 胃运动↑
- 十二指肠抑制因素：盐酸/脂肪/高渗 → 肠胃反射 + 肠抑胃素 → 胃排空↓

## 第四节 小肠内消化

### 一、胰液（最重要的消化液）

- 性质：碱性（**pH 7.8~8.4**）
- 分泌量：**1~2 L/d**
- 分泌：腺泡细胞→**消化酶**；导管细胞→**碳酸氢盐+水**

#### （一）胰液成分与作用

##### 1. 碳酸氢盐

- 中和胃酸，为小肠消化酶提供适宜 pH 环境

##### 2. 糖类消化酶：胰淀粉酶 → 淀粉 → 麦芽糖

##### 3. 蛋白质消化酶

- **胰蛋白酶原** →（肠激酶激活）→ **胰蛋白酶**
- 糜蛋白酶原 →（胰蛋白酶激活）→ 糜蛋白酶
- 羧基肽酶

##### 4. 脂类消化酶

- 胰脂肪酶 + 辅脂酶 → 甘油三酯 → 甘油、脂肪酸、一酰甘油
- 胆固醇酯酶 → 胆固醇酯 → 胆固醇、脂肪酸

##### 5. 核酸消化酶：DNA 酶/RNA 酶 → 核苷酸

- 胰液是**作用最全面、消化力最强、最重要的消化液**

#### （二）急性胰腺炎机制

- 暴饮暴食 → 胰液分泌过多 → 胰管内压力↑ → 胰管和腺泡破裂 → 胰蛋白酶原大量溢入胰腺间质 → 被组织液激活 → **自身消化** → 急性胰腺炎
- 正常保护机制：以酶原形式分泌 + 腺泡分泌胰蛋白酶抑制物

#### （三）胰液分泌的调节

##### 1. 神经调节

- 迷走神经 → ACh → 腺泡细胞 → 胰液（**胰酶**）↑ → 同时 G 细胞→促胃液素

##### 2. 体液调节（主要方式）

- **促胰液素** → 胰腺**导管细胞** → 分泌**水、电解质（HCO₃⁻）**↑
- **缩胆囊素**（促胰酶素）→ 胰腺**腺泡细胞** → 分泌**胰酶**↑

### 二、胆汁

#### （一）胆汁的性质与成分

- 肝胆汁：弱碱性；胆囊胆汁：弱酸性
- 成分：水、碳酸氢盐、胆色素（Hb 代谢产物）、胆固醇（脂类代谢产物）、**胆盐**、磷脂
- 胆盐 = 脂类的有机溶剂（无消化酶）
- 胆盐分子：亲水端 + 疏水端 → 形成**混合微胶粒**（水溶性）
- 胆固醇结石：胆汁中胆固醇↑/胆盐↓ → 胆固醇沉积

#### （二）胆汁的作用（无消化酶）

- ① **乳化脂肪**：增加胰脂肪酶作用面积
- ② **促进脂肪分解产物和脂溶性维生素的吸收**：脂肪酸+胆盐 → 混合微胶粒 → 利于肠粘膜吸收
- ③ 中和胃酸
- ④ 胆盐的**利胆作用**：促进胆汁分泌
- **胆盐的肠-肝循环**：95% 的胆盐在回肠重吸收 → 经门静脉回肝脏 → 循环利用

#### （三）胆汁分泌的调节

- 体液调节（主要）：缩胆囊素→胆囊收缩排胆汁；促胰液素→胆管上皮分泌水/HCO₃⁻；促胃液素→肝胆汁分泌；胆盐→肠肝循环利胆
- 神经调节（弱）：迷走神经→ACh→肝细胞分泌、胆囊排出

### 三、小肠液

- 性质：弱碱性
- 成分：水、碳酸氢盐、**肠激酶**
- 作用：中和胃酸；肠激酶 → 激活胰蛋白酶原；肽酶/双糖酶在上皮细胞完成最后消化

### 四、小肠的运动

#### （一）小肠运动形式

##### 1. 分节运动（小肠特有）

- 以**环形肌**为主的节律性收缩和舒张交替进行
- 意义：促进**化学性消化**（搅拌）+ 促进**吸收**

##### 2. 紧张性收缩（共有）

##### 3. 蠕动（共有）

- 推进食糜（距离较短）；**蠕动冲**：将食糜迅速从小肠始端推送至大肠

## 第五节 大肠的功能

- 吸收水、电解质、维生素；形成和排出粪便
- 大肠液：碱性，粘液+碳酸氢盐 → 保护肠粘膜、润滑大便
- 大肠运动：袋状往返运动（吸收水电解质）、分节推进、蠕动、**集团蠕动**（推进远/快，多在进食后发生→十二指肠-结肠反射引起排便）
- 大肠内细菌：发酵糖脂→乳酸/脂肪酸/CO₂；腐败蛋白→氨/硫化氢；合成维生素 **B、K**

## 第六节 吸收

### 一、吸收部位

- 口腔/食管：少数药物
- 胃：乙醇、药物
- 大肠：水、无机盐、维生素
- **小肠**是主要吸收部位，有利条件：
  - 消化充分
  - 食物停留时间长（**3~8h**）
  - 吸收面积大（**200~250m²**）
  - 绒毛内毛细血管和毛细淋巴管丰富

### 二、吸收途径与机制

- 途径：跨细胞途径、细胞旁途径
- 机制：被动转运、主动转运、出胞/入胞

### 三、主要营养物质的吸收

- 水：被动吸收，**8 L/d**
- 钠：腔面膜→易化扩散；基底侧膜→钠泵主动转运
- 铁：Fe³⁺→Fe²⁺ 后吸收；胃酸+VitC 促进吸收
- 钙：受 VitD₃ 促进，十二指肠主动吸收
- 水溶性维生素（B/C）：被动扩散/主动转运
- 脂溶性维生素（A/D/E/K）：与脂类相似
- **糖**：形式→**单糖**；方式→继发性主动转运（Na⁺-葡萄糖协同转运）；途径→**血液**
- **蛋白质**：形式→**氨基酸**；方式→继发性主动转运；途径→**血液**
- **脂肪**：形式→**脂肪酸/甘油一酯**；方式→与胆盐结合形成混合微胶粒后被吸收；途径→**淋巴途径为主**（长链脂肪酸）
