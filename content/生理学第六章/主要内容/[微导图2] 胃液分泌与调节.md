# 胃液分泌与调节 — 微型速记导图

- 胃液性质：酸性 **pH 0.9~1.5** | 分泌量 **1.5~2.5 L/d**
- ★胃液四大成分及细胞来源
  - 盐酸（胃酸）：泌酸腺**壁细胞**
    - 分泌机制：质子泵 **H⁺-K⁺-ATP酶** -> 抑制剂：**奥美拉唑** -> 餐后碱潮（HCO₃⁻入血）
    - 基础排酸 **0~5 mmol/h** | 最大 **20~25 mmol/h**
    - 五大作用：
      - ① 杀死胃内**细菌**
      - ② 使蛋白质**变性**
      - ③ **激活胃蛋白酶原**+提供酸性环境
      - ④ 刺激促胰液素/缩胆囊素释放
      - ⑤ 促进小肠 **Ca²⁺/Fe²⁺** 吸收
    - 过少 -> 消化不良；过多 -> 消化性溃疡
  - 胃蛋白酶原：泌酸腺**主细胞** -> HCl/胃蛋白酶激活 -> **胃蛋白酶** -> 水解蛋白为胨/胨
  - 粘液：上皮细胞/贲门腺/幽门腺/颈细胞 -> **粘液-碳酸氢盐屏障** -> 保护胃粘膜
  - 内因子：泌酸腺**壁细胞** -> 保护 + 促进回肠吸收 **VitB₁₂** -> 缺乏 -> **巨幼红细胞性贫血**
- ★胃的保护与损伤
  - 保护：粘液-HCO₃⁻屏障 + 胃黏膜屏障（紧密连接）+ PGE₂/PGI₂/EGF
  - 损伤：**酒精、阿司匹林、幽门螺杆菌** -> 破坏屏障 -> 胃炎/胃溃疡
- ★消化期胃液分泌三期对比
  - 头期：量**30%** -> 条件+非条件反射 -> 迷走神经 -> **胃蛋白酶多/胃酸多**
  - 胃期：量**60%** -> 迷走-迷走反射 + 壁内丛 + G细胞->促胃液素 -> **胃酸多/胃蛋白酶↓**
  - 肠期：量**10%** -> 十二指肠G细胞->促胃液素 -> **胃酸少/胃蛋白酶少**
- ★促进胃液分泌三因素
  - 迷走神经 -> **ACh + M受体** -> 阻断剂：**阿托品**
  - 促胃液素 -> **G细胞** -> 阻断剂：**丙谷胺**
  - 组胺 -> **ECL细胞**分泌 -> 阻断剂：**甲氰咪呱**（H₂受体拮抗剂）
  - 咖啡/酒精也刺激胃酸
- 抑制胃液分泌三因素
  - 盐酸 -> 负反馈 -> 抑制促胃液素 + 刺激促胰液素
  - 脂肪 -> 刺激**肠抑胃素**（促胰液素/缩胆囊素/抑胃肽）
  - 高张溶液 -> 肠胃反射
