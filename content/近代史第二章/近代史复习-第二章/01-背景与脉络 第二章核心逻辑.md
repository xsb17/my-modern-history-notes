# 第二章核心逻辑：三次早期探索的历史脉络

> 本章讲述的是 1851—1898 年间，中国不同社会力量在面对民族危机时先后作出的三次救国探索：**太平天国运动（农民阶级）→ 洋务运动（地主阶级开明派）→ 维新运动（资产阶级改良派）**。这是一个从"底层革命"到"技术改良"再到"制度改良"的递进过程。

---

## 一、宏观历史背景（鸦片战争后的中国）

### 1. 人口爆炸与土地兼并

- **人口急剧增长**：清朝康乾盛世后人口爆炸式增长。全国人口从 1661 年约 1.05 亿飙升至 1851 年约 4.32 亿。
- **人均土地锐减**：同一时期，人均耕地从 5.32 亩降至 **1.78 亩**，已低于维持生存的基本线（一般认为人均 3 亩是温饱线）。
- **土地兼并严重**：皇室、官僚、地主大量兼并土地，自耕农大批破产沦为佃农或流民。
- **广西尤甚**：广西山多地少，到 1851 年人均土地仅 **1.1 亩**，远低于全国平均水平，矛盾一触即发。

### 2. 白银外流与"银贵钱贱"

- **鸦片走私**：鸦片战争后鸦片输入有增无减，白银大量外流。
- **战争赔款**：《南京条约》等赔款加重了清政府的财政危机，最终转嫁到普通百姓身上。
- **银贵钱贱**：18 世纪 1 两白银兑换 1000 文铜钱，到 1845 年涨至 1 两兑 **2000 文**。农民出售农产品收的是铜钱，纳税却需折合成白银，实际税负翻倍。
- **曾国藩的描述**："昔日卖米三斗，输一亩之课而有余；今日卖米六斗，输一亩之课而不足。"（《备陈民间疾苦疏》）

### 3. 天灾频发与吏治腐败

- 1846—1850 年间，黄河流域、长江流域连年发生严重水旱灾害。
- 1847 年河南旱灾，1849 年鄂、皖、苏、浙水灾，1849 年广西饥荒，1851 年广西旱灾。
- 清政府救灾能力低下，地方官吏横征暴敛，"遇到灾荒反而成了官吏发财的机会"。

### 4. 广西特有的社会矛盾

- **山多地少**：农业生产条件恶劣。
- **土客矛盾**：本地人（土）与客家人之间的长期械斗和冲突（"来土之争"）。
- **改土归流后遗症**：雍正年间改土归流后，原土司地区的社会秩序未能有效重建。
- **民族问题**：广西多民族聚居，民族矛盾与阶级矛盾交织。

## 二、三次探索的演进逻辑（思维主线）

```
鸦片战争（1840）
    ↓
民族危机 + 阶级矛盾激化
    ↓
┌──────────────────────────────────────────────────────────────┐
│  第一次探索：太平天国（1851—1864）                          │
│  探索主体：农民阶级                                         │
│  核心手段：武装革命、建立政权                               │
│  关键词："有田同耕"—— 解决土地问题                         │
│  结果：被中外反动势力联合镇压                               │
│  教训：农民阶级无法领导近代革命                            │
└──────────────────────────────────────────────────────────────┘
    ↓
┌──────────────────────────────────────────────────────────────┐
│  第二次探索：洋务运动（1861—1895）                          │
│  探索主体：地主阶级开明派（洋务派）                        │
│  核心手段：学习西方技术，不触动制度                        │
│  关键词："中体西用"—— 以技术求强求富                      │
│  结果：甲午战败宣告破产                                     │
│  教训：不改革制度，只学技术，行不通                        │
└──────────────────────────────────────────────────────────────┘
    ↓
┌──────────────────────────────────────────────────────────────┐
│  第三次探索：维新运动（1895—1898）                          │
│  探索主体：资产阶级维新派                                   │
│  核心手段：自上而下的制度改良                              │
│  关键词："变法图存"—— 君主立宪                            │
│  结果：慈禧政变，百日维新夭折                               │
│  教训：改良道路在半殖民地半封建中国行不通                  │
└──────────────────────────────────────────────────────────────┘
```

## 三、核心时间轴

| 时间 | 事件 | 意义 |
|------|------|------|
| 1840—1842 | **鸦片战争** | 中国开始沦为半殖民地半封建社会 |
| 1851.1.11 | **金田起义** | 太平天国运动爆发 |
| 1853 | 定都天京；颁布《天朝田亩制度》 | 建立政权；土地纲领出台 |
| 1856 | **天京事变** | 太平天国由盛转衰 |
| 1856—1860 | **第二次鸦片战争** | 进一步加深民族危机 |
| 1859 | 洪仁玕提出《资政新篇》 | 第一个资本主义色彩方案 |
| 1861 | **总理衙门设立** | 洋务运动开始 |
| 1864 | 天京陷落 | 太平天国失败 |
| 1865 | 江南制造总局成立 | 洋务军工企业兴起 |
| 1872 | 第一批留美幼童出发；轮船招商局成立 | 近代教育、民用工业开端 |
| 1888 | 北洋水师正式成军 | 亚洲第一舰队 |
| 1894—1895 | **甲午中日战争** | 北洋水师全军覆没；洋务运动破产 |
| 1895 | **公车上书** | 拉开维新变法序幕 |
| 1895—1898 | 民族资本主义初步发展；维新思潮高涨 | 经济基础与思想准备 |
| 1898.6.11 | 光绪帝颁布"明定国是"诏 | 百日维新开始 |
| 1898.9.21 | **戊戌政变** | 维新运动夭折 |

## 四、补充背景知识（课外拓展）

> 💡 **补充说明——"白银外流"到底有多严重？**
> 据估算，1800—1840 年间，中国因鸦片贸易外流的白银至少在 **6 亿两** 以上，相当于清政府当时两年多的财政收入。整个国家的银根紧缩，导致市场流通的白银减少、物价波动剧烈，底层农民受到的冲击最为直接。

> 💡 **补充说明——中国近代"资本主义初步发展"到底有多"初步"？**
> 1895—1898 年间新创办的 62 家民族企业，资本总额 1240 多万元，平均每家仅 20 万元。相比之下，同期外国在华企业平均资本是民族企业的数倍。民族资产阶级的力量非常弱小，这决定了维新派的政治基础极其薄弱。

> 💡 **补充说明——为什么是广西？**
> 太平天国起义地点的选择并非偶然。广西地处偏远，清政府控制力相对薄弱；山多地少，土客矛盾尖锐；加上连年天灾，大量流民聚集，形成了天然的革命土壤。洪秀全虽然是广东人，但正是在广西桂平紫荆山地区发展了大批信徒，最终从广西起事。
