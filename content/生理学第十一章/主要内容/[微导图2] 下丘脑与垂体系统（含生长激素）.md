# 下丘脑与垂体系统（含生长激素）— 微型速记导图&nbsp;&nbsp;&nbsp;

- 下丘脑-垂体两大系统&nbsp;&nbsp;&nbsp;
  - 下丘脑-腺垂体：下丘脑调节激素 -> **垂体门脉系统** -> 腺垂体 -> 六种激素&nbsp;&nbsp;&nbsp;
  - 下丘脑-神经垂体：激素合成于下丘脑 -> 储存于**神经垂体** -> 需要时释放&nbsp;&nbsp;&nbsp;
- ★下丘脑调节肽（HRP）概念&nbsp;&nbsp;&nbsp;
  - 下丘脑促垂体区小细胞神经元分泌 -> 调节腺垂体活动的激素（多为多肽）&nbsp;&nbsp;&nbsp;
- 下丘脑-腺垂体-靶腺轴三环&nbsp;&nbsp;&nbsp;
  - TRH -> **TSH** -> TH（甲状腺激素）&nbsp;&nbsp;&nbsp;
  - CRH -> **ACTH** -> GC（糖皮质激素）&nbsp;&nbsp;&nbsp;
  - GnRH -> **FSH/LH** -> 性激素&nbsp;&nbsp;&nbsp;
  - GHRH(**+**)/SS(**-**) -> **GH**（生长激素）&nbsp;&nbsp;&nbsp;
- 神经垂体两种激素&nbsp;&nbsp;&nbsp;
  - ADH/VP：生理浓度 -> V₂受体/抗利尿；大剂量 -> V₁受体/升压&nbsp;&nbsp;&nbsp;
  - OT/缩宫素：收缩子宫（妊娠强）+ 射乳反射&nbsp;&nbsp;&nbsp;
- ★生长激素（GH）——腺垂体含量最高&nbsp;&nbsp;&nbsp;
  - 作用机制：直接（GH受体）+ 间接（**IGF/胰岛素样生长因子**：IGF-1出生后/IGF-2胚胎期）&nbsp;&nbsp;&nbsp;
  - ★三大生理作用：&nbsp;&nbsp;&nbsp;
    - ① 促进生长：促骨/肌肉/组织细胞分裂增殖 -> 影响身高（**不影响智力**）&nbsp;&nbsp;&nbsp;
      - 幼年**少** -> **侏儒症**（矮小/**智力正常**）&nbsp;&nbsp;&nbsp;
      - 幼年**多** -> **巨人症**&nbsp;&nbsp;&nbsp;
      - 成年**多** -> **肢端肥大症**&nbsp;&nbsp;&nbsp;
    - ② 调节代谢：**合成蛋白+分解脂肪+升高血糖**（抑外周组织摄糖）&nbsp;&nbsp;&nbsp;
      - 巨人症/肢端肥大症 -> 血糖↑ -> 可伴糖尿&nbsp;&nbsp;&nbsp;
  - ★分泌调节：&nbsp;&nbsp;&nbsp;
    - GHRH(+) / SS(-) -> GH&nbsp;&nbsp;&nbsp;
    - 负反馈：GH↑ -> 抑下丘脑/垂体&nbsp;&nbsp;&nbsp;
    - 促进：**睡眠/饥饿/低血糖/运动** + 甲状腺激素/雌激素/雄激素&nbsp;&nbsp;&nbsp;
- GH vs TH 对生长的区别&nbsp;&nbsp;&nbsp;
  - GH：促身高/**不影响智力** -> 缺乏->侏儒&nbsp;&nbsp;&nbsp;
  - TH：促身高/**促脑发育** -> 缺乏->呆小症&nbsp;&nbsp;&nbsp;
