## 内分泌与内分泌系统

l内分泌（endocrine）**：指内分泌细胞将产生的激素直接分泌到血液或细胞外液等体液中，然后以体液为媒介对靶细胞产生调节作用的分泌形式。

l激素（hormone）**：是由内分泌腺或散在于组织和器官中的内分泌细胞合成与分泌，以体液为媒介，在细胞之间递送调节信息的高效能生物活性物质。

## 激素的分类

1. 胺类激素：氨基酸经化学修饰生成的衍生物，如甲状腺激素、肾上腺素等。

2. 蛋白质、肽类激素：主要有胰岛素、甲状旁腺激素、下丘脑调节肽、神经垂体激素等。

## 3. 脂类激素：

① 类固醇激素：亲脂激素，共同前体是胆固醇，包括肾上腺皮质激素、性激激素，维生素 $\mathsf { D } _ { 3 } .$ 、25-羟胆钙化醇等。

② 二十烷酸类激素：由花生四烯酸转化而来，如前列腺素。

## 四、激素的一般特征

1. 特异性作用 (激素选择性地作用于靶细胞)

2. 信使作用 (传递信息,加强或减弱原有的生理反应 )

3. 高效作用 (激素与受体结合后，产生高效能生物放大作用)

4. 相互作用：

① 协同作用

② 拮抗作用

③ 允许作用**：某激素本身并不能直接对某些组织细胞产生生物效应，但它的存在却是另一种激素发挥效应的必要基础。例如，儿茶酚胺类激素在糖皮质激素存在的条件下，才能发挥其缩血管的作用。

## （2）反馈调节

血液中GC浓度增加可反馈抑制CRH/ACTH的合成释放，且 ACTH 细胞对 CRH 的敏感性下降（长反馈）；
