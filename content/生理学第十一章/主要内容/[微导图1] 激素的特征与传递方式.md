# 激素的特征与传递方式 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 激素定义：内分泌细胞合成分泌 -> 体液为媒介 -> **高效能生物活性物质**&nbsp;&nbsp;&nbsp;
- ★激素传递 5 种方式&nbsp;&nbsp;&nbsp;
  - 远距分泌（内分泌）：经**血液循环** -> 远隔靶器官&nbsp;&nbsp;&nbsp;
  - 神经内分泌：神经细胞释放激素 -> 入血&nbsp;&nbsp;&nbsp;
  - 旁分泌：经**组织液**扩散 -> 邻近细胞&nbsp;&nbsp;&nbsp;
  - 自分泌：作用于**自身**&nbsp;&nbsp;&nbsp;
  - 胞内分泌：细胞内发挥作用&nbsp;&nbsp;&nbsp;
- 激素分类&nbsp;&nbsp;&nbsp;
  - 胺类：**甲状腺激素/肾上腺素**&nbsp;&nbsp;&nbsp;
  - 蛋白质/肽类：**胰岛素/下丘脑调节肽**&nbsp;&nbsp;&nbsp;
  - 脂类：类固醇（肾上腺皮质激素/性激素/VitD₃）+ 二十烷酸（前列腺素）&nbsp;&nbsp;&nbsp;
- ★激素四大一般特征&nbsp;&nbsp;&nbsp;
  - 特异性：选择性作用于**靶细胞**&nbsp;&nbsp;&nbsp;
  - 信使作用：传递信息 -> 加强或减弱原有反应&nbsp;&nbsp;&nbsp;
  - 高效作用：**生物放大**效应（一个激素分子可产生巨大效应）&nbsp;&nbsp;&nbsp;
  - 相互作用：&nbsp;&nbsp;&nbsp;
    - 协同作用 + 拮抗作用 + 竞争作用&nbsp;&nbsp;&nbsp;
    - ★**允许作用**（必考）：某激素本身无直接效应，但为另一激素发挥作用提供**必要基础**&nbsp;&nbsp;&nbsp;
      - 例：**糖皮质激素**为**儿茶酚胺**发挥缩血管效应创造条件&nbsp;&nbsp;&nbsp;
- 激素作用机制&nbsp;&nbsp;&nbsp;
  - 胞内受体：类固醇激素/甲状腺激素 -> 进入细胞核调控基因表达&nbsp;&nbsp;&nbsp;
  - 胞膜受体：蛋白质/肽类/胺类 -> 第二信使系统（cAMP/IP3等）&nbsp;&nbsp;&nbsp;
- 分泌调控：生物钟节律性分泌 + 体液调节 + 神经调节&nbsp;&nbsp;&nbsp;
