# 第十一章：内分泌 — 全集存档&nbsp;&nbsp;&nbsp;

## 第一节 内分泌与激素&nbsp;&nbsp;&nbsp;

### 一、内分泌与内分泌系统&nbsp;&nbsp;&nbsp;

- 内分泌：内分泌细胞将激素分泌到体液中，以体液为媒介对靶细胞产生调节作用&nbsp;&nbsp;&nbsp;
- 激素：由内分泌腺或内分泌细胞合成与分泌，以体液为媒介递送调节信息的**高效能生物活性物质**&nbsp;&nbsp;&nbsp;
- 内分泌系统：由内分泌腺和内分泌细胞组成&nbsp;&nbsp;&nbsp;

#### 激素运输方式&nbsp;&nbsp;&nbsp;

- 远距分泌（内分泌）：经血液循环运输到远隔靶器官&nbsp;&nbsp;&nbsp;
- 神经内分泌：神经内分泌细胞释放激素入血&nbsp;&nbsp;&nbsp;
- 旁分泌：激素经组织液扩散作用于邻近细胞&nbsp;&nbsp;&nbsp;
- 自分泌：激素作用于分泌细胞自身&nbsp;&nbsp;&nbsp;
- 胞内分泌：激素在细胞内发挥作用&nbsp;&nbsp;&nbsp;

### 二、激素的分类&nbsp;&nbsp;&nbsp;

- 胺类激素：氨基酸衍生物（**甲状腺激素、肾上腺素**等）&nbsp;&nbsp;&nbsp;
- 蛋白质/肽类激素：**胰岛素、下丘脑调节肽、神经垂体激素**等&nbsp;&nbsp;&nbsp;
- 脂类激素：&nbsp;&nbsp;&nbsp;
  - **类固醇激素**：亲脂性，前体为胆固醇（肾上腺皮质激素、性激素、VitD₃）&nbsp;&nbsp;&nbsp;
  - **二十烷酸类**：由花生四烯酸转化（前列腺素）&nbsp;&nbsp;&nbsp;

### 三、激素的作用机制&nbsp;&nbsp;&nbsp;

- 胞内受体介导机制（类固醇激素、甲状腺激素）&nbsp;&nbsp;&nbsp;
- 胞膜受体介导机制（蛋白质/肽类激素、胺类激素）&nbsp;&nbsp;&nbsp;

### 四、激素的一般特征&nbsp;&nbsp;&nbsp;

- **特异性作用**：选择性地作用于靶细胞&nbsp;&nbsp;&nbsp;
- **信使作用**：传递信息，加强或减弱原有生理反应&nbsp;&nbsp;&nbsp;
- **高效作用**：与受体结合后产生高效能**生物放大**作用&nbsp;&nbsp;&nbsp;
- **相互作用**：&nbsp;&nbsp;&nbsp;
  - 协同作用&nbsp;&nbsp;&nbsp;
  - 拮抗作用&nbsp;&nbsp;&nbsp;
  - **允许作用**：某激素本身不能直接产生生物效应，但它的存在是另一种激素发挥效应的**必要基础**（例：糖皮质激素为**儿茶酚胺**发挥缩血管效应创造条件）&nbsp;&nbsp;&nbsp;
  - 竞争作用&nbsp;&nbsp;&nbsp;

### 五、激素分泌节律和调控&nbsp;&nbsp;&nbsp;

- 生物节律性分泌：受生物钟调控&nbsp;&nbsp;&nbsp;
- 调控方式：体液调节 + 神经调节&nbsp;&nbsp;&nbsp;

## 第二节 下丘脑-垂体内分泌&nbsp;&nbsp;&nbsp;

### 一、下丘脑-腺垂体系统&nbsp;&nbsp;&nbsp;

- 下丘脑合成下丘脑调节激素 → 经**下丘脑-垂体门脉系统** → 进入腺垂体调节其内分泌活动&nbsp;&nbsp;&nbsp;
- **下丘脑调节肽（HRP）**：下丘脑促垂体区小细胞神经元分泌的能调节腺垂体活动的激素（多为多肽）&nbsp;&nbsp;&nbsp;

#### 下丘脑调节激素与对应的腺垂体激素及靶腺激素&nbsp;&nbsp;&nbsp;

- GHRH（生长激素释放激素）→ **GH**（生长激素）&nbsp;&nbsp;&nbsp;
- SS（生长抑素）→ GH（抑制）&nbsp;&nbsp;&nbsp;
- TRH（促甲状腺激素释放激素）→ **TSH** → **TH**（甲状腺激素）&nbsp;&nbsp;&nbsp;
- CRH（促肾上腺皮质激素释放激素）→ **ACTH** → **GC**（糖皮质激素）&nbsp;&nbsp;&nbsp;
- GnRH（促性腺激素释放激素）→ **FSH/LH** → 性激素&nbsp;&nbsp;&nbsp;
- PRF/PIF → **PRL**（催乳素）&nbsp;&nbsp;&nbsp;

#### 腺垂体六种激素&nbsp;&nbsp;&nbsp;

- **GH**（生长激素）——腺垂体中含量最高的激素，191 个氨基酸&nbsp;&nbsp;&nbsp;
- PRL（催乳素）&nbsp;&nbsp;&nbsp;
- TSH（促甲状腺激素）&nbsp;&nbsp;&nbsp;
- ACTH（促肾上腺皮质激素）&nbsp;&nbsp;&nbsp;
- FSH（卵泡刺激素）&nbsp;&nbsp;&nbsp;
- LH（黄体生成素）&nbsp;&nbsp;&nbsp;

#### 1. 生长激素（GH）★&nbsp;&nbsp;&nbsp;

**作用机制**：① 直接作用（GH 受体）；② 间接作用（诱导靶细胞产生 **IGF/胰岛素样生长因子**，IGF-1 出生后/IGF-2 胚胎期）&nbsp;&nbsp;&nbsp;

**（2）生理作用**：&nbsp;&nbsp;&nbsp;

**① 促进生长**：促进骨、肌肉和其他组织细胞分裂增殖、蛋白质合成增加 → 促进机体生长发育&nbsp;&nbsp;&nbsp;
- GH 分泌异常（**不影响智力**）：&nbsp;&nbsp;&nbsp;
  - 幼年分泌**少** → **侏儒症**（身材矮小/智力正常）&nbsp;&nbsp;&nbsp;
  - 幼年分泌**多** → **巨人症**&nbsp;&nbsp;&nbsp;
  - 成年分泌**多** → **肢端肥大症**&nbsp;&nbsp;&nbsp;

**② 调节代谢**：合成蛋白、分解脂肪、升高血糖&nbsp;&nbsp;&nbsp;
- 促进氨基酸进入细胞，加速蛋白质合成&nbsp;&nbsp;&nbsp;
- 激活对激素敏感的脂肪酶，促进脂肪分解&nbsp;&nbsp;&nbsp;
- 抑制外周组织摄取和利用葡萄糖，升高血糖&nbsp;&nbsp;&nbsp;

**（3）分泌调节**：&nbsp;&nbsp;&nbsp;
- ① GHRH → GH↑；SS → GH↓&nbsp;&nbsp;&nbsp;
- ② GH 负反馈调节下丘脑和垂体&nbsp;&nbsp;&nbsp;
- ③ 睡眠、饥饿、低血糖、运动 → GH↑；甲状腺激素/雌激素/雄激素 → GH↑&nbsp;&nbsp;&nbsp;

#### 2. 催乳素（PRL）&nbsp;&nbsp;&nbsp;

**生理作用**：① 促进乳腺发育、发动和维持泌乳；② 调节性腺功能（小剂量促进/大剂量抑制）；③ 参与应激反应；④ 调节免疫功能&nbsp;&nbsp;&nbsp;

### 二、下丘脑-神经垂体系统&nbsp;&nbsp;&nbsp;

- 下丘脑合成激素，在神经垂体（垂体后叶）**储存**，需要时释放入血&nbsp;&nbsp;&nbsp;

#### （一）抗利尿激素/血管升压素（ADH/VP）&nbsp;&nbsp;&nbsp;

**生理作用**：&nbsp;&nbsp;&nbsp;
- 生理浓度：与肾 V₂ 受体结合 → 抗利尿作用&nbsp;&nbsp;&nbsp;
- 大剂量（失血/脱水）：与血管 V₁ 受体结合 → 收缩血管/升压&nbsp;&nbsp;&nbsp;

#### （二）催产素/缩宫素（OT）&nbsp;&nbsp;&nbsp;

**生理作用**：① 刺激子宫收缩、促进分娩（对妊娠子宫作用强）；② 促进乳腺排乳（射乳反射）&nbsp;&nbsp;&nbsp;

## 第三节 甲状腺内分泌&nbsp;&nbsp;&nbsp;

### 一、甲状腺激素概况&nbsp;&nbsp;&nbsp;

- 甲状腺：人体内最大的内分泌腺，主要结构为**滤泡**&nbsp;&nbsp;&nbsp;
- 滤泡上皮细胞：合成和释放 TH&nbsp;&nbsp;&nbsp;
- 滤泡腔：**储存** TH（储存量大，可供 50~120 天使用）&nbsp;&nbsp;&nbsp;
- 滤泡旁细胞：合成和分泌降钙素&nbsp;&nbsp;&nbsp;
- TH 包括：**T₄**（四碘甲腺原氨酸/占 90%）和 **T₃**（三碘甲腺原氨酸/占 9%）&nbsp;&nbsp;&nbsp;
- 活性：**T₃ > T₄**（T₄ 在外周组织中转化为 T₃）&nbsp;&nbsp;&nbsp;
- 合成原料：甲状腺球蛋白 + **碘**&nbsp;&nbsp;&nbsp;

### 二、甲状腺激素的生物作用★&nbsp;&nbsp;&nbsp;

#### 1. 促进生长发育&nbsp;&nbsp;&nbsp;

- ① 促**神经系统**发育（与 GH 区别：TH 影响智力）&nbsp;&nbsp;&nbsp;
- ② 促**骨骼**生长（与 GH 协同）&nbsp;&nbsp;&nbsp;
- TH 是**胎儿和新生儿脑发育**的关键激素&nbsp;&nbsp;&nbsp;
- 胚胎期或出生后 TH 缺乏 → **呆小症/克汀病**（**身材矮小+智力低下**）&nbsp;&nbsp;&nbsp;
- 先天性甲状腺发育不全患儿应在出生后 **3 个月内**补充 TH&nbsp;&nbsp;&nbsp;
- 与侏儒症区别：侏儒症（GH 缺乏→身材矮小/**智力正常**）；呆小症（TH 缺乏→身材矮小/**智力低下**）&nbsp;&nbsp;&nbsp;

#### 2. 调节新陈代谢&nbsp;&nbsp;&nbsp;

- ① **增强能量代谢（产热效应）**：使绝大多数组织耗氧量和产热量增加&nbsp;&nbsp;&nbsp;
- ② 调节物质代谢（双向调节）：&nbsp;&nbsp;&nbsp;
  - 蛋白质：生理剂量 TH → 促合成；浓度过高 → 促分解&nbsp;&nbsp;&nbsp;
  - 糖代谢：既升高又降低血糖（升糖 > 降糖）&nbsp;&nbsp;&nbsp;
  - 脂肪代谢：促分解 > 促合成&nbsp;&nbsp;&nbsp;
- 甲低 → 黏液性水肿&nbsp;&nbsp;&nbsp;

#### 3. 影响器官系统功能&nbsp;&nbsp;&nbsp;

- 对神经系统：促**交感神经兴奋**&nbsp;&nbsp;&nbsp;
- 对心血管：增加心脏作功&nbsp;&nbsp;&nbsp;
- 对消化道：促消化&nbsp;&nbsp;&nbsp;
- 甲状腺激素主要作用总结：**促生长、促代谢、促兴奋**&nbsp;&nbsp;&nbsp;

### 三、甲状腺功能的调节★&nbsp;&nbsp;&nbsp;

#### （一）下丘脑-腺垂体-甲状腺轴：TRH-TSH-TH 环路&nbsp;&nbsp;&nbsp;

- 下丘脑 **TRH** → 腺垂体 **TSH** → 甲状腺 **TH**（T₃/T₄）&nbsp;&nbsp;&nbsp;
- **1. 下丘脑对腺垂体**：TRH 促进 TSH 合成释放 + 促进 TSH 糖基化保证活性；SS 抑制 TSH&nbsp;&nbsp;&nbsp;
- **2. 腺垂体对甲状腺**：TSH 促进甲状腺滤泡细胞增殖发育 + 促进 TH 合成与分泌&nbsp;&nbsp;&nbsp;
- **3. 甲状腺激素的反馈调控**：&nbsp;&nbsp;&nbsp;
  - TH↑ → 负反馈抑制 TSH 分泌 + 下调 TSH 细胞 TRH 受体数量&nbsp;&nbsp;&nbsp;
  - TH↑ → 直接抑制下丘脑 TRH 合成&nbsp;&nbsp;&nbsp;
- **临床意义**：&nbsp;&nbsp;&nbsp;
  - 甲亢：T₃/T₄**↑**、TSH**↓**&nbsp;&nbsp;&nbsp;
  - 甲减：T₃/T₄**↓**、TSH**↑**&nbsp;&nbsp;&nbsp;
  - 缺碘 → TH↓ → TSH↑ → 甲状腺滤泡细胞增殖 → **甲状腺肿大（大脖子病）**&nbsp;&nbsp;&nbsp;

#### （二）甲状腺自身调节——碘阻滞效应&nbsp;&nbsp;&nbsp;

- 血碘 < 1 mmol/L → 促进 TH 合成&nbsp;&nbsp;&nbsp;
- 血碘 > 10 mmol/L → 抑制碘活化/TH 合成减少（**碘阻滞效应**）&nbsp;&nbsp;&nbsp;

#### （三）神经调节&nbsp;&nbsp;&nbsp;

- 交感神经兴奋 → 促进 TH 分泌&nbsp;&nbsp;&nbsp;

## 第四节 胰岛的内分泌&nbsp;&nbsp;&nbsp;

### 一、胰岛细胞组成&nbsp;&nbsp;&nbsp;

- **α 细胞（25%）**：胰高血糖素&nbsp;&nbsp;&nbsp;
- **β 细胞（60%~70%）**：胰岛素&nbsp;&nbsp;&nbsp;
- **δ 细胞（5%）**：生长抑素（SST）&nbsp;&nbsp;&nbsp;

### 二、胰岛素的生物作用★&nbsp;&nbsp;&nbsp;

#### 1. 对糖代谢：降低血糖&nbsp;&nbsp;&nbsp;

- 胰岛素是体内**唯一**降低血糖的激素&nbsp;&nbsp;&nbsp;
- 机制：减少糖来源（抑制糖原分解/抑制糖异生）+ 增加糖去路（促进糖原合成/促进外周组织摄取和利用葡萄糖）&nbsp;&nbsp;&nbsp;
- 缺乏 → **糖尿病**&nbsp;&nbsp;&nbsp;

#### 2. 对脂肪代谢：促进合成，减少分解&nbsp;&nbsp;&nbsp;

- 促进葡萄糖进入脂肪细胞合成脂肪酸&nbsp;&nbsp;&nbsp;
- 抑制激素敏感性脂肪酶 → 减少脂肪分解&nbsp;&nbsp;&nbsp;

#### 3. 对蛋白质代谢：促进合成，抑制分解&nbsp;&nbsp;&nbsp;

- 促进氨基酸进入细胞合成蛋白质&nbsp;&nbsp;&nbsp;
- 抑制蛋白质分解和糖异生&nbsp;&nbsp;&nbsp;

#### 4. 促生长：与 GH 协同促进生长&nbsp;&nbsp;&nbsp;

### 三、胰岛素分泌的调节★&nbsp;&nbsp;&nbsp;

- ① **血糖浓度**：血糖↑ → 胰岛素分泌↑（**最重要因素**）&nbsp;&nbsp;&nbsp;
- ② 氨基酸和脂肪酸 → 刺激胰岛素分泌&nbsp;&nbsp;&nbsp;
- ③ 促血糖升高的激素可促进胰岛素分泌（胰高血糖素/促胃液素/TH/GH/GC等）&nbsp;&nbsp;&nbsp;
- ④ 副交感神经兴奋 → 促进胰岛素分泌&nbsp;&nbsp;&nbsp;

## 第五节 肾上腺的内分泌&nbsp;&nbsp;&nbsp;

### 一、肾上腺结构&nbsp;&nbsp;&nbsp;

- 皮质（腺垂体靶腺，分泌类固醇激素）：&nbsp;&nbsp;&nbsp;
  - 球状带 → **盐皮质激素**（醛固酮）&nbsp;&nbsp;&nbsp;
  - 束状带 → **糖皮质激素（GC）**&nbsp;&nbsp;&nbsp;
  - 网状带 → 雄激素&nbsp;&nbsp;&nbsp;
- 髓质（交感神经节前纤维支配，分泌儿茶酚胺类激素：**肾上腺素/去甲肾上腺素**）&nbsp;&nbsp;&nbsp;

### 二、肾上腺糖皮质激素的作用★&nbsp;&nbsp;&nbsp;

- 血浆中最多的是**皮质醇**；临床常用：氢化可的松、泼尼松、地塞米松&nbsp;&nbsp;&nbsp;

#### 1. 调节物质代谢&nbsp;&nbsp;&nbsp;

- ① 糖代谢：**抗胰岛素样作用** → 血糖↑（糖尿病人慎用）&nbsp;&nbsp;&nbsp;
- ② 脂肪代谢：促四肢脂肪**分解**、促面部和躯干脂肪**合成** → **向心性肥胖**（满月脸/水牛背）&nbsp;&nbsp;&nbsp;
- ③ 蛋白质：抑制肝外合成、促进分解 → 骨质疏松、皮肤变薄、抑制创伤愈合&nbsp;&nbsp;&nbsp;

#### 2. 影响水盐代谢&nbsp;&nbsp;&nbsp;

- 总效用有利于肾排水 → 缺乏时可致**水中毒**&nbsp;&nbsp;&nbsp;

#### 3. 参与应激反应★&nbsp;&nbsp;&nbsp;

- **应激反应**：机体遭受伤寒性刺激（创伤/手术/感染/中毒/疼痛/缺氧/寒冷/精神紧张）→ 腺垂体释放大量 **ACTH** → GC 快速大量分泌 → 非特异性适应反应&nbsp;&nbsp;&nbsp;
- 应激原：引起应激反应的刺激&nbsp;&nbsp;&nbsp;
- 意义：增加耐受性和抵抗力&nbsp;&nbsp;&nbsp;

#### 4. 对组织器官的影响&nbsp;&nbsp;&nbsp;

- 血液：RBC/中性粒细胞/PLT↑；淋巴细胞和嗜酸性粒细胞↓&nbsp;&nbsp;&nbsp;
- 循环：↑心血管对儿茶酚胺和 AngII 反应性（**允许作用**）；维持心肌正常功能&nbsp;&nbsp;&nbsp;
- 呼吸：↑胎儿肺泡 II 型细胞形成/肺表面活性物质↑&nbsp;&nbsp;&nbsp;
- 消化：↑胃酸和胃蛋白酶原&nbsp;&nbsp;&nbsp;
- 泌尿：↑肾血浆流量/GFR/排水&nbsp;&nbsp;&nbsp;
- 神经系统：维持 CNS 正常功能&nbsp;&nbsp;&nbsp;
- 药理作用：**抗炎、抗毒、抗过敏和抗休克**&nbsp;&nbsp;&nbsp;

### 三、肾上腺糖皮质激素的调节★&nbsp;&nbsp;&nbsp;

#### （一）下丘脑-垂体-肾上腺轴：CRH-ACTH-GC 环路&nbsp;&nbsp;&nbsp;

- 下丘脑 **CRH** → 腺垂体 **ACTH** → 肾上腺皮质 **GC**（皮质醇）&nbsp;&nbsp;&nbsp;
- ACTH 维持肾上腺皮质正常结构，促进 GC 合成和分泌&nbsp;&nbsp;&nbsp;
- 生物钟调控：CRH 分泌清晨觉醒前最高 → 午夜最低&nbsp;&nbsp;&nbsp;

#### （二）反馈调节&nbsp;&nbsp;&nbsp;

- 长反馈：血中 **GC↑** → 负反馈抑制 CRH/ACTH 合成释放 + ACTH 细胞对 CRH 敏感性↓&nbsp;&nbsp;&nbsp;
- 短反馈：ACTH↑ → 抑制 CRH 合成释放&nbsp;&nbsp;&nbsp;
- 超短反馈：CRH 自身调节&nbsp;&nbsp;&nbsp;

#### （三）临床意义&nbsp;&nbsp;&nbsp;

- 长期大剂量使用 GC 药物 → 负反馈抑制 ACTH → **肾上腺皮质萎缩** → 突然停药可致**肾上腺危象**甚至死亡&nbsp;&nbsp;&nbsp;
- 停药原则：**逐渐减量 + 间断补充 ACTH**&nbsp;&nbsp;&nbsp;

#### （四）应激调节&nbsp;&nbsp;&nbsp;

- 应激反应时，CRH-ACTH-GC 大量分泌，不受轴性负反馈影响&nbsp;&nbsp;&nbsp;
