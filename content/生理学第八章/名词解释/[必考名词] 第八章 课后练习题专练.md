# 第八章 课后练习题名词解释&nbsp;&nbsp;&nbsp;

- **排泄**&nbsp;&nbsp;&nbsp;
  - 定义：机体将**代谢产物**、摄入的**过剩物质**、**异物**及**有害物质**等经**血液循环**由相应途径排出体外的过程&nbsp;&nbsp;&nbsp;
  - 正常值/公式：主要途径为**肾脏**（最主要排泄器官），辅助途径包括皮肤（出汗）、呼吸器官（呼出 CO₂）、消化器官（胆汁）&nbsp;&nbsp;&nbsp;
  - 生理意义/特征：维持**内环境稳态**的核心环节；肾脏排泄功能丧失 → 尿毒症；注意与"排遗"区分（排泄=代谢终产物经血液循环排出，排遗=未吸收的食物残渣经消化道排出）&nbsp;&nbsp;&nbsp;
- **肾小球滤过率（GFR）**&nbsp;&nbsp;&nbsp;
  - 定义：**每分钟**双肾生成的**超滤液量**，是衡量肾小球**滤过功能**的定量指标&nbsp;&nbsp;&nbsp;
  - 正常值/公式：正常成人安静时 **125 ml/min** = **180 L/d**&nbsp;&nbsp;&nbsp;
  - 生理意义/特征：反映肾小球滤过功能的**金标准**；GFR↓ → 肾功能减退；是计算**滤过分数**的基础（FF = GFR ÷ 肾血浆流量 ≈ **19%**）；临床常用肌酐清除率估测 GFR&nbsp;&nbsp;&nbsp;
- **肾糖阈**&nbsp;&nbsp;&nbsp;
  - 定义：当血糖浓度升高到一定程度，部分近端小管对葡萄糖的重吸收达到**极限**，**尿中开始出现葡萄糖**时对应的**血浆葡萄糖浓度**&nbsp;&nbsp;&nbsp;
  - 正常值/公式：**180 mg/100ml**；葡萄糖重吸收极限量：男 **375 mg/min**、女 **300 mg/min**；血糖达 **300 mg/100ml** 时尿糖排出与血糖浓度平行增加&nbsp;&nbsp;&nbsp;
  - 生理意义/特征：反映近端小管对葡萄糖**重吸收能力**的界限；是理解和诊断**糖尿**的基础概念（血糖 > 肾糖阈 → 尿糖阳性）；糖尿病诊断和疗效监测的重要参考值&nbsp;&nbsp;&nbsp;
- **渗透性利尿**&nbsp;&nbsp;&nbsp;
  - 定义：由于**小管液中溶质浓度增加**、**渗透压升高**，使水钠重吸收**减少**而引起的**利尿**现象&nbsp;&nbsp;&nbsp;
  - 正常值/公式：机制链：小管液溶质浓度↑ → 小管液渗透压↑ → 对抗水的重吸收 → 水重吸收↓ → **尿量↑**&nbsp;&nbsp;&nbsp;
  - 生理意义/特征：临床典型实例 → **糖尿病多尿**（高血糖超肾糖阈 → 小管液葡萄糖↑ → 渗透压↑ → 多尿）；**甘露醇利尿**（甘露醇不被重吸收 → 滞留小管液 → 渗透压↑ → 达到脱水/降颅压/利尿目的）；渗透性利尿的溶质必须是**不被或不易被重吸收**的物质&nbsp;&nbsp;&nbsp;
- **球-管平衡**&nbsp;&nbsp;&nbsp;
  - 定义：近端小管中钠和水的重吸收率总是占肾小球滤过率的 **65%~70%** 的现象，即近端小管保持**定比重吸收**&nbsp;&nbsp;&nbsp;
  - 正常值/公式：重吸收比例恒定在 **65%~70%**（无论 GFR 如何变化）；若 GFR 为 125 ml/min → 近端小管重吸收约 81~88 ml/min&nbsp;&nbsp;&nbsp;
  - 生理意义/特征：保持**尿量相对恒定**（GFR 在一定范围内波动时尿量不致剧烈变化）；维持**有效循环血量**的稳定；是肾小管与肾小球功能**协调配合**的重要体现；破坏球管平衡 → 尿量异常增多或减少&nbsp;&nbsp;&nbsp;
