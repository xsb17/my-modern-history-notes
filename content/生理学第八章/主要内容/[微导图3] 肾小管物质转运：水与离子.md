# 肾小管物质转运：水与离子 — 微型速记导图

- Na⁺/Cl⁻/水重吸收各段比例与机制
  - 近端小管：**65~70%** -> 重吸收**主要部位**
    - 前半段（2/3）：跨细胞 -> **H⁺-Na⁺ 交换**（腔面）+ Na⁺ 泵（侧膜）-> Cl⁻ 无重吸收 -> 水被动
    - 后半段（1/3）：细胞旁 -> Cl⁻ 顺浓度差 + Na⁺ 顺电位差被动 -> 水被动
    - 特点：**等渗性重吸收**
  - 髓袢：**15~20%**
    - 降支细段：Na⁺/Cl⁻ 不易通透 -> **水被动重吸收**
    - 升支细段：水不易通透 -> Na⁺/Cl⁻ **被动重吸收**
    - 升支粗段★：水**不通透** -> **Na⁺-K⁺-2Cl⁻ 同向转运体（NKCC2）**
      - 2Cl⁻ -> 经通道进入组织间液
      - 1Na⁺ -> 钠泵**主动**泵出
      - 1K⁺ -> 顺浓度梯度**漏回**管腔
      - 1Na⁺ -> 顺电位差从细胞旁路**被动**重吸收
      - 结果：重吸收 **2Cl⁻**（继发主动）+ **2Na⁺**（1 主动 + 1 被动）
  - 远曲小管/集合管：约 **12%**
    - 远曲前段：**Na⁺-Cl⁻ 同向转运体（NCC）**-> 水不通透 -> 噻嗪类利尿药作用点
    - 远曲后段/集合管（主细胞）：Na⁺ 经 **Na⁺ 通道**进入 -> Cl⁻ 经细胞旁被动 -> 水受 **ADH** 调控（AQP-2↑）
- 水重吸收总结
  - 总量：**99%** 被重吸收
  - 部位：近端小管为主（**65~70%**）
  - 机制：被动重吸收，渗透压差驱动
  - 通道：近端小管 **AQP-1** + 集合管 **AQP-2**（受 ADH 调节）
- H⁺ 分泌与 HCO₃⁻ 重吸收（**排酸保碱**）
  - HCO₃⁻ 重吸收：**80%** 在近端小管 -> 以 **CO₂** 形式
  - H⁺ 分泌：近端小管为主要部位 -> **H⁺-Na⁺ 交换**
  - 集合管闰细胞：**H⁺ 泵、H⁺-K⁺ 泵**
  - 碳酸酐酶抑制剂（乙酰唑胺）-> H⁺ 分泌↓ -> 利尿
- NH₃/NH₄⁺ 分泌
  - 近端小管：分泌 **NH₄⁺**
  - 集合管：分泌 **NH₃**
  - H⁺ 分泌 ⇄ NH₃ 分泌 -> **相互促进** -> 促进排酸保碱
- K⁺ 重吸收与分泌
  - 重吸收：原尿 K⁺ 几乎**全部**重吸收 -> **65~70%** 在近端小管
  - 分泌主要部位：**远端小管和集合管**（主细胞）
  - 分泌机制：
    - Na⁺ 泵 -> 细胞内 K⁺↑ -> 顺**浓度差**分泌
    - Na⁺ 重吸收 -> 电位差 -> 顺**电位差**分泌
  - 阿米洛利（保钾利尿剂）：抑制 Na⁺ 通道 -> K⁺ 分泌↓
- ★ H⁺/NH₃/K⁺ 三者分泌关系（必考）
  - H⁺ ⇄ NH₃：**相互促进** -> 排酸保碱
  - H⁺-Na⁺ 交换 ⇄ K⁺-Na⁺ 交换：**竞争性抑制**
  - 酸中毒 -> H⁺ 分泌↑ -> K⁺ 分泌↓ -> **高血钾**
  - 碱中毒 -> H⁺ 分泌↓ -> K⁺ 分泌↑ -> **低血钾**
