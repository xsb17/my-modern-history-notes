# 尿生成的体液调节 — 微型速记导图（大题必考）

- 神经调节概述
  - 肾交感神经兴奋 -> 三效应：
    - ① 入球小动脉**收缩** -> GFR↓
    - ② 球旁细胞分泌**肾素↑** -> NaCl/水重吸收↑
    - ③ 直接促进近端小管/髓袢 NaCl 和水重吸收↑
    - -> 总体：**尿量↓**
- 抗利尿激素（ADH）★核心大题
  - 分泌：下丘脑**视上核、室旁核** -> 储存于神经垂体
  - 作用：远曲小管/集合管 -> 管腔膜 **AQP-2**（水孔蛋白）↑ -> 水通透性↑ -> 水重吸收↑ -> 尿量↓
  - 分泌调节：
    - （1）血浆晶体渗透压（**主要因素**）：
      - 渗透压**↑**（大量出汗/失水）-> 渗透压感受器兴奋 -> ADH**↑** -> 水重吸收↑ -> 尿量↓
      - 渗透压**↓**（大量饮水）-> 渗透压感受器抑制 -> ADH**↓** -> 水重吸收↓ -> 尿量↑
    - （2）循环血量：
      - 血量**↓**（大失血）-> 容量感受器抑制 -> ADH**↑** -> 水重吸收↑ -> 尿量↓
    - （3）其他因素：
      - 寒冷/乙醇 -> ADH**↓** -> 尿量↑
      - 疼痛/紧张 -> ADH**↑** -> 尿量↓
  - ★水利尿（必考大题）
    - 定义：大量饮**清水**后尿量↑的现象
    - 机制链：饮清水 -> 血浆晶体渗透压**↓** -> 下丘脑渗透压感受器**抑制** -> ADH 释放**↓** -> 远曲/集合管水重吸收**↓** -> 尿量**↑**
    - 时间线：30min 开始↑ -> 1h 达峰值 -> 2~3h 恢复正常
    - 对比：饮生理盐水 -> 渗透压不变 -> ADH 不变 -> 尿量无明显变化
- 醛固酮 ★核心大题
  - 分泌：肾上腺皮质**球状带**
  - 作用：**保 Na⁺ 排 K⁺ 保水** -> 远曲小管/集合管主细胞 -> Na⁺/Cl⁻/水重吸收↑ + K⁺ 分泌↑
  - 分泌调节：
    - （1）RAAS 系统（肾素-血管紧张素-醛固酮系统）：
      - 触发 -> 肾素↑ -> AngⅠ -> ACE -> **AngⅡ** -> **AngⅢ** -> 醛固酮↑
      - 肾素分泌触发因素：入球小动脉**牵张感受器** + **致密斑**（NaCl↓）+ **交感神经**
    - （2）血 K⁺/Na⁺ 浓度：
      - 血 **K⁺↑**（最敏感）/血 **Na⁺↓** -> 醛固酮**↑**
- 心房钠尿肽（ANP）
  - 分泌：心房肌细胞
  - 作用：促进**水、钠排出**（与醛固酮效应**对抗**）
- ADH vs 醛固酮 对比速查
  - ADH：来源下丘脑 -> 靶点远曲/集合管 -> 机制 **AQP-2↑** -> 效应**水重吸收↑** -> 调节因素：**晶体渗透压**为主
  - 醛固酮：来源肾上腺皮质 -> 靶点远曲/集合管主细胞 -> 机制**基因转录** -> 效应**保 Na⁺ 排 K⁺ 保水** -> 调节因素：**RAAS + 血 K⁺**
