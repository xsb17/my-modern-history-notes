# 尿生成的体液调节 — 微型速记导图（大题必考）&nbsp;&nbsp;&nbsp;

- 神经调节概述&nbsp;&nbsp;&nbsp;
  - 肾交感神经兴奋 -> 三效应：&nbsp;&nbsp;&nbsp;
    - ① 入球小动脉**收缩** -> GFR↓&nbsp;&nbsp;&nbsp;
    - ② 球旁细胞分泌**肾素↑** -> NaCl/水重吸收↑&nbsp;&nbsp;&nbsp;
    - ③ 直接促进近端小管/髓袢 NaCl 和水重吸收↑&nbsp;&nbsp;&nbsp;
    - -> 总体：**尿量↓**&nbsp;&nbsp;&nbsp;
- 抗利尿激素（ADH）★核心大题&nbsp;&nbsp;&nbsp;
  - 分泌：下丘脑**视上核、室旁核** -> 储存于神经垂体&nbsp;&nbsp;&nbsp;
  - 作用：远曲小管/集合管 -> 管腔膜 **AQP-2**（水孔蛋白）↑ -> 水通透性↑ -> 水重吸收↑ -> 尿量↓&nbsp;&nbsp;&nbsp;
  - 分泌调节：&nbsp;&nbsp;&nbsp;
    - （1）血浆晶体渗透压（**主要因素**）：&nbsp;&nbsp;&nbsp;
      - 渗透压**↑**（大量出汗/失水）-> 渗透压感受器兴奋 -> ADH**↑** -> 水重吸收↑ -> 尿量↓&nbsp;&nbsp;&nbsp;
      - 渗透压**↓**（大量饮水）-> 渗透压感受器抑制 -> ADH**↓** -> 水重吸收↓ -> 尿量↑&nbsp;&nbsp;&nbsp;
    - （2）循环血量：&nbsp;&nbsp;&nbsp;
      - 血量**↓**（大失血）-> 容量感受器抑制 -> ADH**↑** -> 水重吸收↑ -> 尿量↓&nbsp;&nbsp;&nbsp;
    - （3）其他因素：&nbsp;&nbsp;&nbsp;
      - 寒冷/乙醇 -> ADH**↓** -> 尿量↑&nbsp;&nbsp;&nbsp;
      - 疼痛/紧张 -> ADH**↑** -> 尿量↓&nbsp;&nbsp;&nbsp;
  - ★水利尿（必考大题）&nbsp;&nbsp;&nbsp;
    - 定义：大量饮**清水**后尿量↑的现象&nbsp;&nbsp;&nbsp;
    - 机制链：饮清水 -> 血浆晶体渗透压**↓** -> 下丘脑渗透压感受器**抑制** -> ADH 释放**↓** -> 远曲/集合管水重吸收**↓** -> 尿量**↑**&nbsp;&nbsp;&nbsp;
    - 时间线：30min 开始↑ -> 1h 达峰值 -> 2~3h 恢复正常&nbsp;&nbsp;&nbsp;
    - 对比：饮生理盐水 -> 渗透压不变 -> ADH 不变 -> 尿量无明显变化&nbsp;&nbsp;&nbsp;
- 醛固酮 ★核心大题&nbsp;&nbsp;&nbsp;
  - 分泌：肾上腺皮质**球状带**&nbsp;&nbsp;&nbsp;
  - 作用：**保 Na⁺ 排 K⁺ 保水** -> 远曲小管/集合管主细胞 -> Na⁺/Cl⁻/水重吸收↑ + K⁺ 分泌↑&nbsp;&nbsp;&nbsp;
  - 分泌调节：&nbsp;&nbsp;&nbsp;
    - （1）RAAS 系统（肾素-血管紧张素-醛固酮系统）：&nbsp;&nbsp;&nbsp;
      - 触发 -> 肾素↑ -> AngⅠ -> ACE -> **AngⅡ** -> **AngⅢ** -> 醛固酮↑&nbsp;&nbsp;&nbsp;
      - 肾素分泌触发因素：入球小动脉**牵张感受器** + **致密斑**（NaCl↓）+ **交感神经**&nbsp;&nbsp;&nbsp;
    - （2）血 K⁺/Na⁺ 浓度：&nbsp;&nbsp;&nbsp;
      - 血 **K⁺↑**（最敏感）/血 **Na⁺↓** -> 醛固酮**↑**&nbsp;&nbsp;&nbsp;
- 心房钠尿肽（ANP）&nbsp;&nbsp;&nbsp;
  - 分泌：心房肌细胞&nbsp;&nbsp;&nbsp;
  - 作用：促进**水、钠排出**（与醛固酮效应**对抗**）&nbsp;&nbsp;&nbsp;
- ADH vs 醛固酮 对比速查&nbsp;&nbsp;&nbsp;
  - ADH：来源下丘脑 -> 靶点远曲/集合管 -> 机制 **AQP-2↑** -> 效应**水重吸收↑** -> 调节因素：**晶体渗透压**为主&nbsp;&nbsp;&nbsp;
  - 醛固酮：来源肾上腺皮质 -> 靶点远曲/集合管主细胞 -> 机制**基因转录** -> 效应**保 Na⁺ 排 K⁺ 保水** -> 调节因素：**RAAS + 血 K⁺**&nbsp;&nbsp;&nbsp;
