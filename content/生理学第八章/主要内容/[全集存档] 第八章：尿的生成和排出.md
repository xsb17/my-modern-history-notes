# 第八章：尿的生成和排出 — 全集存档

## 第一节 概述

### 一、排泄

- 排泄定义：机体将代谢产物、过剩物质、异物等经血液循环由相应途径排出体外的过程
- 排泄途径：
  - 皮肤：出汗排出水、无机盐、尿素
  - 呼吸器官：呼出 CO₂、水蒸气
  - 消化器官：胆汁排出胆色素等
  - 肾脏：**主要排泄器官**
- 尿量：
  - 正常值：**1000~2000 ml/d**
  - 多尿：> **2500 ml/d**
  - 少尿：**100~400 ml/d**
  - 无尿：< **100 ml/d**
- 尿液理化性质：淡黄色透明，pH **5.0~7.0**（偏酸性）

### 二、肾脏的功能

- （一）排泄功能：排出代谢产物、异物等
- （二）维持水、电解质、酸碱平衡
- （三）内分泌功能：
  - 肾素：**球旁细胞**分泌 → 调节血压
  - 促红细胞生成素（**EPO**）：促进红细胞生成
  - 活性维生素 D₃：肾脏 **1α-羟化酶** → 将 25-羟维生素 D₃ 转化为 1,25-二羟胆骨化醇 → 调节钙磷代谢

### 三、肾脏的功能解剖

#### （一）肾单位

- 肾小体：肾小球 + 肾小囊
- 肾小管：近端小管（近曲小管）→ 髓袢（降支粗段 → 降支细段 → 升支细段 → 升支粗段）→ 远端小管（远曲小管）

#### （二）皮质肾单位 vs 近髓肾单位

- 皮质肾单位：占 **85%~90%**，肾小体位于肾皮质外层，髓袢短（只达外髓），出球小动脉口径（入:出 ≈ **2:1**），有球旁器，功能→**尿液生成**
- 近髓肾单位：占 **10%~15%**，肾小体位于肾皮质近髓层，髓袢长（深入内髓层形成直小血管），入/出球小动脉口径差异小，几乎无球旁器，功能→**尿液浓缩稀释**

#### （三）球旁器（JGA）

- 球旁细胞：**分泌肾素**
- 致密斑：感受小管液中 **NaCl 含量**变化
- 球外系膜细胞：吞噬、收缩功能

### 四、肾血流量的特点及调节

#### （一）肾血液循环特点

- 肾血流量大：双肾 **1200 ml/min**，占心输出量 **20%~25%**
- 分布不均：**94%** 分布于皮质，6% 分布于髓质
- 两套毛细血管网：
  - 肾小球毛细血管网（入球小动脉→毛细血管→出球小动脉）：**血压高** → 有利于肾小球滤过
  - 肾小管周围毛细血管网（出球小动脉→毛细血管→肾静脉）：**血压低** → 有利于肾小管重吸收

#### （二）肾血流量的调节

##### 1. 自身调节

- 范围：肾动脉压 **70~180 mmHg** 时，肾血流量保持**相对恒定**
- 特点：不需神经、体液参与
- 机制：
  - 肌源学说：动脉压↑ → 入球小动脉平滑肌紧张性↑ → 口径↓ → 血流阻力↑ → 肾血流量稳定
  - 管球反馈：小管液流量↑ → 致密斑感受 NaCl↑ → 入球小动脉收缩 → GFR↓

##### 2. 神经体液调节

- 神经调节：交感神经兴奋 → 肾血管**收缩** → 肾血流量↓
- 体液调节：Adr/NA/ATP → 肾血管收缩；PG/NO → 肾血管舒张
- 意义：紧急情况（大失血等）→ 肾血管收缩 → 保证**心脑**等重要器官供血

## 第二节 肾小球的滤过功能

### 一、肾小球滤过

- 定义：血液流经肾小球时，血浆中**除蛋白质外**的成分通过滤过膜滤入肾小囊内形成**原尿**的过程
- 原尿：除大分子蛋白质外，其他成分似血浆 → 是血浆的**超滤液**

#### （一）肾小球滤过率（GFR）与滤过分数（FF）

- GFR：每分钟双肾生成的超滤液量
  - 正常值：**125 ml/min** = **180 L/d**
- FF = GFR ÷ 每分肾血浆流量 × 100%
  - 正常值：125 ÷ 660 × 100% ≈ **19%**
  - 意义：流经肾脏的血浆约有 **1/5** 滤出形成原尿

### 二、滤过结构——滤过膜（三层）

- 内层（毛细血管内皮细胞层）：窗孔 **70~90nm**，阻止血细胞
- 中层（基膜层）：网孔 **2~8nm**，阻止蛋白质，**起主要机械屏障作用**
- 外层（肾小囊脏层上皮细胞层）：裂隙→裂隙膜→小孔，起主要作用

#### 两大屏障

- 机械屏障：三层结构的孔道
  - 分子半径 < **2.0nm**：完全滤过（无机盐、葡萄糖）
  - 分子半径 > **4.2nm**：完全不滤过（蛋白质）
  - 分子半径 2.0~4.2nm：随半径增加滤过减少
- 电学屏障：滤过膜含**带负电荷**的糖蛋白
  - 滤过能力：**负电荷 < 电中性 < 正电荷**
  - 白蛋白（半径 3.6nm，带负电）→ 很难滤出

### 三、滤过动力——有效滤过压

- 公式：有效滤过压 = **肾小球毛细血管压** -（**囊内压** + **血浆胶体渗透压**）
- 入球端：45 -（10 + 25）= **10 mmHg**（有滤过）
- 出球端：45 -（10 + 35）= **0 mmHg**（滤过平衡点）
- 滤过平衡点：有效滤过压最先降为 0 的点 → 此点以前的毛细血管才有滤过功能

#### 影响肾小球滤过的因素推演

- 休克（血压↓）→ 肾小球 Cap 压↓ → 有效滤过压↓ → GFR↓ → **少尿或无尿**
- 大量输入生理盐水 → 血浆蛋白稀释 → 血浆胶渗压↓ → 有效滤过压↑ → GFR↑ → **尿量↑**
- 输尿管结石/肿瘤 → 囊内压↑ → 有效滤过压↓ → GFR↓ → **尿量↓**
- 肾血浆流量↑ → 血浆胶渗压递增速率↓ → 滤过平衡点**后移** → 有效滤过长度↑ → GFR↑

### 四、滤过系数（Kf）

- Kf = **K × S**（K：有效通透系数，S：滤过膜面积 **1.5m²**）
- K↑（肾小球肾炎→通透性↑）→ 出现血尿、蛋白尿
- S↓（肾小球肾炎→Cap 阻塞→有效滤过面积↓）→ 少尿、无尿

## 第三节 肾小管、集合管的物质转运功能

### 一、重吸收

- 定义：小管液中的部分物质通过小管上皮细胞进入周围毛细血管的过程
- 重吸收物质：大部分水、电解质；全部葡萄糖、氨基酸；部分代谢产物（尿素）
- 重吸收主要部位：**近端小管**（重吸收种类多、量大）
- 重吸收方式：
  - 主动重吸收：耗能，逆浓度差/电位差（原发性=生物泵；继发性=间接利用泵能量）
  - 被动重吸收：以渗透压差/浓度差/电位差为动力

### 二、肾小管和集合管的分泌功能

- 分泌：肾小管上皮细胞将其代谢产物或血液中某些物质排入管腔的过程
- 主要分泌物质：**H⁺**、**NH₃**、**K⁺**

### 三、几种重要物质的转运

#### （一）Na⁺、Cl⁻ 和水的重吸收

##### 1. 近端小管（65~70%）

- 前半段（2/3）：跨细胞途径
  - 侧膜/管周膜：Na⁺ 泵主动转运
  - 腔面膜：**H⁺-Na⁺ 交换**（协同转运）
  - Cl⁻ 无重吸收，水被动重吸收
- 后半段（1/3）：细胞旁途径
  - Cl⁻ 顺浓度梯度被动重吸收
  - Na⁺ 顺电位梯度被动重吸收
- 特点：**等渗性重吸收**

##### 2. 髓袢（15~20%）

- 降支细段：Na⁺/Cl⁻ 不易通透，水被动重吸收
- 升支细段：水不易通透，Na⁺/Cl⁻ 被动重吸收
- 升支粗段：水不易通透，存在 **Na⁺-K⁺-2Cl⁻ 同向转运体（NKCC2）**
  - 离子去向：2Cl⁻ 经通道进入组织间液；1Na⁺ 由钠泵主动泵出；1K⁺ 顺浓度梯度经管腔膜返回管腔；1Na⁺ 顺电位差经细胞旁路被动重吸收
  - 结果：重吸收 2Cl⁻（继发性主动）+ 2Na⁺（1 主动 + 1 被动）

##### 3. 远曲小管、集合管（约 12%）

- 远曲小管前段：**Na⁺-Cl⁻ 同向转运体（NCC）**，水不通透（噻嗪类利尿药作用点）
- 远曲小管后段/集合管（主细胞）：
  - Na⁺ 经 Na⁺ 通道进入细胞
  - Cl⁻ 经细胞旁途径被动重吸收
  - 水被动重吸收 → 受 **ADH** 调节（ADH↑ → AQP-2↑ → 水通透性↑）

##### 水重吸收总结

- 重吸收量：**99%**
- 主要部位：**近端小管**（65~70%）
- 机制：被动重吸收，渗透压差驱动
- 近端小管：**AQP-1**
- 集合管：**AQP-2**（受 ADH 调节）

#### （二）HCO₃⁻ 的重吸收与 H⁺ 的分泌

- HCO₃⁻ 重吸收：**80%** 在近端小管，以 **CO₂** 形式重吸收
- H⁺ 分泌：近端小管为主要分泌部位，通过 **H⁺-Na⁺ 交换**分泌
- 集合管（闰细胞）：通过 **H⁺ 泵、H⁺-K⁺ 泵**分泌
- 肾脏的**排酸保碱**功能：分泌 H⁺ + 重吸收 HCO₃⁻
- 碳酸酐酶抑制剂（乙酰唑胺）→ H⁺ 分泌↓ → Na⁺/HCO₃⁻ 重吸收↓ → 利尿

#### （三）NH₃ 与 NH₄⁺ 的分泌

- 近端小管：分泌 **NH₄⁺**
- 集合管：分泌 **NH₃**
- H⁺ 分泌 ↔ NH₃ 分泌 → **相互促进** → 促进排酸保碱

#### （四）K⁺ 的重吸收与分泌

- K⁺ 重吸收：原尿中 K⁺ 几乎全部被重吸收，**65~70%** 在近端小管
- K⁺ 分泌主要部位：**远端小管和集合管**（主细胞）
- 分泌机制：
  - Na⁺ 泵活动 → 细胞内 K⁺↑ → K⁺ 顺浓度差分泌
  - Na⁺ 重吸收造成电位差 → K⁺ 顺电位差分泌
- 阿米洛利（保钾利尿剂）：抑制 Na⁺ 通道 → 抑制 Na⁺/水重吸收 → 抑制 K⁺ 分泌

#### ★ H⁺、NH₃、K⁺ 分泌的关系

- H⁺ 和 NH₃ 分泌 → **相互促进** → 促进排酸保碱
- **H⁺-Na⁺ 交换** 与 **K⁺-Na⁺ 交换** → **竞争性抑制**
- 酸中毒 → H⁺ 分泌↑ → K⁺ 分泌↓ → **高血钾**
- 碱中毒 → H⁺ 分泌↓ → K⁺ 分泌↑ → **低血钾**

#### （五）葡萄糖的重吸收

- 重吸收量：**全部**
- 重吸收部位：**近端小管**（仅限于此）
- 重吸收机制：**Na⁺-葡萄糖协同转运**（继发性主动转运）
- 饱和现象：
  - 肾糖阈：血糖浓度达 **180 mg/100ml** 时，尿中出现葡萄糖 → 此时血糖浓度称为肾糖阈
  - 葡萄糖重吸收极限量：男 **375 mg/min**，女 **300 mg/min**
  - 血糖 **300 mg/100ml** 时，尿糖排出与血糖浓度平行增加

### 四、影响肾小管集合管转运的因素

#### 1. 小管液溶质浓度 → 渗透性利尿

- 小管液溶质浓度↑ → 渗透压↑ → 水重吸收↓ → **尿量↑**
- 糖尿病多尿：血糖↑ → 小管液糖浓度↑ → 渗透压↑ → 尿量↑
- 甘露醇利尿：甘露醇不被重吸收 → 小管液渗透压↑ → 利尿

#### 2. 球管平衡

- 定义：近端小管中钠和水的重吸收率总是占 GFR 的 **65~70%** 的现象（定比重吸收）
- 意义：保持**尿量相对恒定**，维持**有效循环血量**的稳定

## 第五节 尿生成的调节

### 一、神经调节

- 肾交感神经兴奋：
  - ① 入球小动脉收缩 → 肾血流量↓ → GFR↓
  - ② 球旁细胞分泌肾素↑ → 远曲小管 NaCl/水重吸收↑
  - ③ 直接促进近端小管/髓袢 NaCl 和水重吸收↑
  - → 总体效应：**尿量↓**

### 二、体液调节（主要调节）

#### （一）抗利尿激素（ADH）

- 分泌部位：下丘脑**视上核、室旁核**大细胞神经元
- 作用：促进远曲小管/集合管上皮细胞管腔膜 **AQP-2**（水孔蛋白）数量↑ → 水重吸收↑
- 分泌调节：
  - （1）血浆晶体渗透压（主要因素）：渗透压↑ → 渗透压感受器兴奋 → ADH↑ → 水重吸收↑ → 尿量↓
  - （2）循环血量：血量↓ → 容量感受器抑制 → ADH↑ → 水重吸收↑ → 尿量↓
  - （3）其他：寒冷/乙醇 → ADH↓ → 尿量↑；疼痛/紧张 → ADH↑ → 尿量↓
- **水利尿**：大量饮清水后尿量↑的现象
  - 机制：饮水 → 血浆晶体渗透压↓ → 渗透压感受器抑制 → ADH↓ → 远曲/集合管水重吸收↓ → 尿量↑
  - 时间：30min 开始↑，1h 达峰值，2~3h 恢复正常

#### （二）醛固酮

- 分泌部位：肾上腺皮质**球状带**
- 作用：**保 Na⁺ 排 K⁺ 保水**
  - 促进远曲小管/集合管主细胞 → Na⁺/Cl⁻/水重吸收↑，K⁺ 分泌↑
- 分泌调节：
  - （1）RAAS 系统：肾素 → AngⅠ → ACE → AngⅡ → AngⅢ → 醛固酮↑
    - 触发因素：入球小动脉牵张感受器兴奋、致密斑感受器兴奋、交感神经兴奋
  - （2）血 K⁺↑（更敏感）/血 Na⁺↓ → 醛固酮↑

#### （三）心房钠尿肽（ANP）

- 分泌：心房肌细胞合成
- 作用：促进**水、钠排出**（与醛固酮对抗）
