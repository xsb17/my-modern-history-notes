# 葡萄糖转运与小管影响因素 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 葡萄糖重吸收&nbsp;&nbsp;&nbsp;
  - 重吸收量：**全部**（正常血糖时）&nbsp;&nbsp;&nbsp;
  - 重吸收部位：**近端小管**（仅限于此！）&nbsp;&nbsp;&nbsp;
  - 重吸收机制：**Na⁺-葡萄糖协同转运** -> 继发性主动转运&nbsp;&nbsp;&nbsp;
  - 饱和现象：&nbsp;&nbsp;&nbsp;
    - 肾糖阈 = **180 mg/100ml**（尿中出现葡萄糖时的血糖浓度）&nbsp;&nbsp;&nbsp;
    - 葡萄糖重吸收极限量：男 **375 mg/min**，女 **300 mg/min**&nbsp;&nbsp;&nbsp;
    - 血糖 **300 mg/100ml** 时 -> 尿糖排出与血糖浓度**平行增加**&nbsp;&nbsp;&nbsp;
- 渗透性利尿 ★必考&nbsp;&nbsp;&nbsp;
  - 定义：由于**小管液溶质浓度↑**、渗透压↑，使水钠重吸收↓引起的利尿&nbsp;&nbsp;&nbsp;
  - 机制链：小管液溶质浓度↑ -> **小管液渗透压↑** -> 水重吸收↓ -> **尿量↑**&nbsp;&nbsp;&nbsp;
  - 典型例子：&nbsp;&nbsp;&nbsp;
    - 糖尿病多尿：血糖↑ -> 超肾糖阈 -> 小管液糖浓度↑ -> 渗透压↑ -> **多尿**&nbsp;&nbsp;&nbsp;
    - 甘露醇利尿：甘露醇不被重吸收 -> 滞留在小管液 -> 渗透压↑ -> **利尿**&nbsp;&nbsp;&nbsp;
- 球管平衡 ★必考&nbsp;&nbsp;&nbsp;
  - 定义：近端小管中钠和水的重吸收率总是占 GFR 的 **65~70%** -> **定比重吸收**&nbsp;&nbsp;&nbsp;
  - 机制：GFR↑ -> 肾小管周围 Cap 血压↓ + 血浆胶渗压↑ -> 重吸收↑&nbsp;&nbsp;&nbsp;
  - 意义：&nbsp;&nbsp;&nbsp;
    - 保持**尿量相对恒定**&nbsp;&nbsp;&nbsp;
    - 维持**有效循环血量**的稳定&nbsp;&nbsp;&nbsp;
    - 防止因 GFR 波动导致尿量剧烈变化&nbsp;&nbsp;&nbsp;
- 影响肾小管转运的因素速查&nbsp;&nbsp;&nbsp;
  - 小管液溶质浓度↑ -> 渗透性利尿 -> 尿量↑&nbsp;&nbsp;&nbsp;
  - GFR↑ -> 球管平衡 -> 近端小管重吸收按比例↑ -> 尿量不致大幅波动&nbsp;&nbsp;&nbsp;
  - ADH↑ -> AQP-2↑ -> 集合管水通透性↑ -> 水重吸收↑ -> 尿量↓&nbsp;&nbsp;&nbsp;
  - 醛固酮↑ -> Na⁺ 重吸收↑/K⁺ 分泌↑ -> 保 Na⁺ 排 K⁺ 保水&nbsp;&nbsp;&nbsp;
