# 肾小球的滤过功能 — 微型速记导图

- 肾小球滤过定义：血液 -> 肾小球 -> 除**蛋白质**外成分滤入肾小囊 -> 形成**原尿**（血浆超滤液）
- GFR 与滤过分数（FF）
  - GFR = 每分钟双肾生成超滤液量 = **125 ml/min** = **180 L/d**
  - FF = GFR ÷ 肾血浆流量 = 125 ÷ 660 ≈ **19%**
  - 意义：流经肾脏的血浆约 **1/5** 滤出形成原尿
- 滤过膜三层结构（机械屏障）
  - 内层（内皮细胞）：窗孔 **70~90nm** -> 阻止**血细胞**
  - 中层（基膜）：网孔 **2~8nm** -> 阻止**蛋白质** -> **起主要作用**
  - 外层（上皮细胞）：裂隙膜 -> 起主要作用
  - 分子半径 < **2.0nm** -> 完全滤过（无机盐/葡萄糖）
  - 分子半径 > **4.2nm** -> 完全不滤过（蛋白质）
- 电学屏障
  - 滤过膜含**带负电荷**糖蛋白
  - 滤过能力：**负电荷 < 电中性 < 正电荷**
  - 白蛋白（半径 3.6nm + 带负电）-> **很难滤出**
- 有效滤过压公式 ★核心
  - 有效滤过压 = **肾小球 Cap 压** -（**囊内压** + **血浆胶渗压**）
  - 入球端：**45** -（**10** + **25**）= **10 mmHg**（有滤过）
  - 出球端：45 -（10 + **35**）= **0 mmHg**（**滤过平衡点**）
  - 滤过平衡点 = 有效滤过压最先降为 0 的点
  - 平衡点**以前**的 Cap 才有滤过功能
- 影响滤过的四大因素推演 ★必考分析题
  - 休克（血压 < 50mmHg）：
    - 肾小球 Cap 压**↓** -> 有效滤过压↓ -> GFR↓ -> **少尿或无尿**
  - 大量输入生理盐水：
    - 血浆蛋白稀释 -> **血浆胶渗压↓** -> 有效滤过压↑ -> GFR↑ -> **尿量↑**
  - 输尿管结石/肿瘤：
    - 尿液排出受阻 -> **囊内压↑** -> 有效滤过压↓ -> GFR↓ -> **尿量↓**
  - 肾血浆流量变化：
    - 流量↑ -> 胶渗压递增速率↓ -> 滤过平衡点**后移** -> 有效滤过长度↑ -> GFR↑
    - 流量↓（休克）-> 胶渗压递增速率↑ -> 平衡点**前移** -> GFR↓
- 滤过系数 Kf = **K × S**
  - K = 有效通透系数 -> 影响**尿液成分**
    - K↑（肾小球肾炎）-> 血尿、蛋白尿
  - S = 滤过膜面积 **1.5m²** -> 影响**尿量**
    - S↓（肾炎 Cap 阻塞）-> 有效滤过面积↓ -> 少尿、无尿
