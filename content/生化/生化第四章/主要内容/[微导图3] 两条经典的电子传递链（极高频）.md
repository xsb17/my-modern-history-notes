# 两条经典的电子传递链（极高频）&nbsp;&nbsp;&nbsp;

- **呼吸链四大复合体总览**&nbsp;&nbsp;&nbsp;
	- **复合体I** -> **NADH-泛醌还原酶** -> 辅基：FMN, Fe-S -> **有**质子泵功能 -> 泵出 **4H⁺**&nbsp;&nbsp;&nbsp;
	- **复合体II** -> **琥珀酸-泛醌还原酶** -> 辅基：FAD, Fe-S -> **无**质子泵功能 -> 含**琥珀酸脱氢酶**&nbsp;&nbsp;&nbsp;
	- **复合体III** -> **泛醌-Cytc还原酶** -> 辅基：血红素b,c₁, Fe-S -> **有**质子泵功能 -> 泵出 **4H⁺** -> 通过**Q循环**&nbsp;&nbsp;&nbsp;
	- **复合体IV** -> **细胞色素C氧化酶** -> 辅基：血红素a, a₃, Cu -> **有**质子泵功能 -> 泵出 **2H⁺** -> 最终将电子交给**O₂**&nbsp;&nbsp;&nbsp;
	- **2个游离分子** -> **CoQ**（脂溶性，膜内自由移动）/ **Cyt c**（水溶性，膜外侧）&nbsp;&nbsp;&nbsp;
- **NADH 氧化呼吸链（主链）**&nbsp;&nbsp;&nbsp;
	- 电子供体 -> **NADH**（来自TCA循环、糖酵解等）&nbsp;&nbsp;&nbsp;
	- 传递路径 -> **NADH** → **复合体I** → **CoQ** → **复合体III** → **Cyt c** → **复合体IV** → **O₂**&nbsp;&nbsp;&nbsp;
	- 经过复合体 -> **I → III → IV**（跳过了II）&nbsp;&nbsp;&nbsp;
	- 偶联部位数 -> **3个**&nbsp;&nbsp;&nbsp;
	- 能量产出 -> **P/O = 2.5** -> 1分子NADH -> **2.5分子ATP**&nbsp;&nbsp;&nbsp;
	- 质子泵出 -> 复合体I(4H⁺) + 复合体III(4H⁺) + 复合体IV(2H⁺) = **10H⁺**&nbsp;&nbsp;&nbsp;
- **琥珀酸氧化呼吸链（FADH₂呼吸链/次链）**&nbsp;&nbsp;&nbsp;
	- 电子供体 -> **FADH₂**（来自琥珀酸脱氢、脂肪酸β-氧化等）&nbsp;&nbsp;&nbsp;
	- 传递路径 -> **琥珀酸** → **复合体II** → **CoQ** → **复合体III** → **Cyt c** → **复合体IV** → **O₂**&nbsp;&nbsp;&nbsp;
	- 经过复合体 -> **II → III → IV**（跳过了I）&nbsp;&nbsp;&nbsp;
	- 偶联部位数 -> **2个**（少了复合体I的偶联）&nbsp;&nbsp;&nbsp;
	- 能量产出 -> **P/O = 1.5** -> 1分子FADH₂ -> **1.5分子ATP**&nbsp;&nbsp;&nbsp;
	- 质子泵出 -> 复合体III(4H⁺) + 复合体IV(2H⁺) = **6H⁺**&nbsp;&nbsp;&nbsp;
- **两条链的关键对比（必背）**&nbsp;&nbsp;&nbsp;
	- NADH链起点 -> **NADH** -> 经复合体**I**&nbsp;&nbsp;&nbsp;
	- FADH₂链起点 -> **琥珀酸(FADH₂)** -> 经复合体**II**&nbsp;&nbsp;&nbsp;
	- 共同路径 -> CoQ → 复合体III → Cyt c → 复合体IV → O₂&nbsp;&nbsp;&nbsp;
	- **CoQ** -> 两条呼吸链的**汇合点**&nbsp;&nbsp;&nbsp;
	- ATP差 -> NADH比FADH₂多产生 **1分子ATP**（2.5 - 1.5 = 1）&nbsp;&nbsp;&nbsp;
- **胞浆NADH的穿梭机制**&nbsp;&nbsp;&nbsp;
	- 问题 -> 胞浆NADH**不能自由通过**线粒体内膜&nbsp;&nbsp;&nbsp;
	- **苹果酸-天冬氨酸穿梭**&nbsp;&nbsp;&nbsp;
		- 部位 -> **心、肝**&nbsp;&nbsp;&nbsp;
		- 进入 -> **NADH呼吸链** -> 生成 **2.5分子ATP**&nbsp;&nbsp;&nbsp;
	- **α-磷酸甘油穿梭**&nbsp;&nbsp;&nbsp;
		- 部位 -> **脑、神经、肌肉**&nbsp;&nbsp;&nbsp;
		- 进入 -> **FADH₂呼吸链** -> 生成 **1.5分子ATP**&nbsp;&nbsp;&nbsp;
- **线粒体内膜关键转运蛋白**&nbsp;&nbsp;&nbsp;
	- **ATP-ADP转位酶** -> ADP进 / ATP出&nbsp;&nbsp;&nbsp;
	- **α-酮戊二酸转运蛋白** -> 苹果酸进 / α-酮戊二酸出&nbsp;&nbsp;&nbsp;
	- **天冬氨酸-谷氨酸转运蛋白** -> 谷氨酸进 / 天冬氨酸出&nbsp;&nbsp;&nbsp;
	- **三羧酸转运蛋白** -> 苹果酸进 / 柠檬酸出&nbsp;&nbsp;&nbsp;
