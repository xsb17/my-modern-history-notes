# 呼吸链抑制剂与解偶联剂（必考临床联系）&nbsp;&nbsp;&nbsp;

- **抑制剂三大类型**&nbsp;&nbsp;&nbsp;
	- **呼吸链抑制剂** -> 直接阻断**电子传递** -> 电子无法传到O₂&nbsp;&nbsp;&nbsp;
	- **解偶联剂** -> 破坏**质子电化学梯度** -> 氧化和磷酸化解偶联&nbsp;&nbsp;&nbsp;
	- **ATP合酶抑制剂** -> 阻断H⁺通过F₀**回流** -> ATP合成受阻&nbsp;&nbsp;&nbsp;
- **呼吸链抑制剂（按复合体靶点）**&nbsp;&nbsp;&nbsp;
	- **复合体I 抑制剂**（阻断 NADH → CoQ）&nbsp;&nbsp;&nbsp;
		- **鱼藤酮**(rotenone) -> 植物提取物&nbsp;&nbsp;&nbsp;
		- **阿米妥**（异戊巴比妥） -> 巴比妥类药物&nbsp;&nbsp;&nbsp;
		- **粉蝶霉素A**(piericidin A)&nbsp;&nbsp;&nbsp;
		- 结合靶点 -> 复合体I的**铁硫蛋白**&nbsp;&nbsp;&nbsp;
	- **复合体II 抑制剂**&nbsp;&nbsp;&nbsp;
		- **萎锈灵**(carboxin)&nbsp;&nbsp;&nbsp;
	- **复合体III 抑制剂**（阻断 CoQH₂ → Cyt c）&nbsp;&nbsp;&nbsp;
		- **抗霉素A**(antimycin A)&nbsp;&nbsp;&nbsp;
	- **复合体IV 抑制剂**（阻断 Cyt c → O₂）（最重要/最常考）&nbsp;&nbsp;&nbsp;
		- **氰化物(CN⁻)** -> 剧毒 -> 与Cyt a₃的铁离子结合&nbsp;&nbsp;&nbsp;
		- **一氧化碳(CO)** -> 与Cyt a₃结合 -> **人体中毒**常见原因&nbsp;&nbsp;&nbsp;
		- **硫化氢(H₂S)**&nbsp;&nbsp;&nbsp;
		- **叠氮化物(N₃⁻)**&nbsp;&nbsp;&nbsp;
		- 后果 -> 电子传递**完全中断** -> 无ATP合成 -> **组织缺氧** -> 细胞窒息死亡&nbsp;&nbsp;&nbsp;
- **解偶联剂（只产热不产ATP）**&nbsp;&nbsp;&nbsp;
	- 机制 -> 破坏电子传递建立的**跨膜质子电化学梯度**&nbsp;&nbsp;&nbsp;
	- 结果 -> 电子可正常传递 -> 但H⁺不经ATP合酶回流 -> 能量以**热能**释放 -> **ATP合成↓**&nbsp;&nbsp;&nbsp;
	- 外源性解偶联剂&nbsp;&nbsp;&nbsp;
		- **2,4-二硝基苯酚(DNP)** -> 脂溶性 -> 携带H⁺跨膜 -> 曾用为减肥药但毒性大&nbsp;&nbsp;&nbsp;
		- **双香豆素** -> 脂溶性&nbsp;&nbsp;&nbsp;
	- 内源性解偶联剂&nbsp;&nbsp;&nbsp;
		- **解偶联蛋白(UCP)** -> 存在于**褐色脂肪细胞**线粒体&nbsp;&nbsp;&nbsp;
		- 作用 -> 建立**质子漏** -> H⁺不经过ATP合酶而回流 -> 产热&nbsp;&nbsp;&nbsp;
		- 生理意义 -> 新生儿御寒/维持体温&nbsp;&nbsp;&nbsp;
- **ATP合酶抑制剂**&nbsp;&nbsp;&nbsp;
	- **寡霉素**(oligomycin) -> 结合**F₀** -> 阻止H⁺从F₀质子通道回流 -> 抑制ATP生成 -> 抗菌/抗肿瘤&nbsp;&nbsp;&nbsp;
	- **二环己基碳二亚胺(DCCD)** -> 结合**F₀** -> 同样阻止H⁺回流&nbsp;&nbsp;&nbsp;
- **抑制剂速记口诀**&nbsp;&nbsp;&nbsp;
	- **复合体I** -> 鱼藤酮 + 阿米妥 + 粉蝶霉素A -> **"鱼阿粉吃I"**&nbsp;&nbsp;&nbsp;
	- **复合体II** -> 萎锈灵 -> **"二萎"**&nbsp;&nbsp;&nbsp;
	- **复合体III** -> 抗霉素A -> **"三抗"**&nbsp;&nbsp;&nbsp;
	- **复合体IV** -> CN⁻ + CO + H₂S + N₃⁻ -> **"四中毒"** -> 氰化物/煤气/硫化氢/叠氮&nbsp;&nbsp;&nbsp;
	- **解偶联** -> DNP + 双香豆素 + UCP -> **"解联产热"**&nbsp;&nbsp;&nbsp;
	- **ATP合酶** -> 寡霉素 + DCCD -> **"寡D锁F₀"**&nbsp;&nbsp;&nbsp;
- **临床联系**&nbsp;&nbsp;&nbsp;
	- **氰化物中毒** -> CN⁻抑制复合体IV -> 细胞无法利用O₂ -> **细胞内窒息** -> 即使血液O₂充足也会死亡&nbsp;&nbsp;&nbsp;
	- **CO中毒（煤气中毒）** -> CO抑制复合体IV + 与血红蛋白结合 -> 双重打击&nbsp;&nbsp;&nbsp;
	- **鱼藤酮** -> 天然杀虫剂 -> 阻断复合体I -> 对鱼类和昆虫毒性大&nbsp;&nbsp;&nbsp;
	- **甲亢** -> 甲状腺激素诱导UCP + 刺激钠泵 -> 基础代谢率↑ -> 产热↑ -> 怕热多汗&nbsp;&nbsp;&nbsp;
