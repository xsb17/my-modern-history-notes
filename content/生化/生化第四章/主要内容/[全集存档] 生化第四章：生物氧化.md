## 第一节 生物氧化概述&nbsp;&nbsp;&nbsp;

### 一、生物氧化的概念&nbsp;&nbsp;&nbsp;

- **生物氧化(biological oxidation)**：物质在生物体内进行的氧化，主要是指糖、脂肪、蛋白质等营养物质在生物体活细胞内氧化分解，生成 CO₂ 和 H₂O 并逐步释放能量的过程&nbsp;&nbsp;&nbsp;

### 二、生物氧化的特点&nbsp;&nbsp;&nbsp;

#### 生物氧化与体外氧化(燃烧)的比较&nbsp;&nbsp;&nbsp;

- 相同点：&nbsp;&nbsp;&nbsp;
  - 遵循氧化还原反应的一般规律&nbsp;&nbsp;&nbsp;
  - 耗氧量、最终产物、释放能量均相同&nbsp;&nbsp;&nbsp;
- 不同点（生物氧化）：&nbsp;&nbsp;&nbsp;
  - 反应条件：有水、体温、近中性 pH 的细胞内进行（vs 体外：高温、干燥）&nbsp;&nbsp;&nbsp;
  - 催化方式：在酶的催化下逐步进行（vs 体外：无需催化）&nbsp;&nbsp;&nbsp;
  - 能量释放：能量逐步释放，部分以化学能方式贮存、部分以热能释放（vs 体外：能量瞬间大量释放，转换为光和热）&nbsp;&nbsp;&nbsp;
  - 产物生成：加水脱氢使物质间接获得氧，脱下的氢与氧结合生成水；CO₂ 由有机酸脱羧产生（vs 体外：C 和 H 直接与 O₂ 结合生成 CO₂ 和 H₂O）&nbsp;&nbsp;&nbsp;

### 三、参与生物氧化的酶类&nbsp;&nbsp;&nbsp;

- 生物氧化中物质的氧化方式：脱电子、脱氢、加氧等方式，其中**脱氢反应最常见**&nbsp;&nbsp;&nbsp;
- 失去电子或氢原子的物质称为**供电子体/供氢体**；接受电子或氢原子的物质称为**受电子体/受氢体**&nbsp;&nbsp;&nbsp;
- 参与生物氧化的酶分为四类：&nbsp;&nbsp;&nbsp;

#### 1. 氧化酶类&nbsp;&nbsp;&nbsp;

- **氧化酶(oxidase)**：催化代谢底物脱氢，以**氧为受氢体**&nbsp;&nbsp;&nbsp;
- 产物为 H₂O（如**细胞色素氧化酶**）或 H₂O₂（如氨基酸氧化酶、黄嘌呤氧化酶）&nbsp;&nbsp;&nbsp;

#### 2. 脱氢酶类（最主要）&nbsp;&nbsp;&nbsp;

- **脱氢酶(dehydrogenase)**：催化底物脱氢，但**不以氧为直接受氢体**&nbsp;&nbsp;&nbsp;
- 直接受氢体为：**NAD⁺**（或 NADP⁺）、**FMN**（或 FAD）&nbsp;&nbsp;&nbsp;
- 如：琥珀酸脱氢酶&nbsp;&nbsp;&nbsp;

#### 3. 加氧酶类&nbsp;&nbsp;&nbsp;

- **加氧酶(oxygenase)**：主要存在于微粒体，催化向底物加氧原子&nbsp;&nbsp;&nbsp;
- 分为两种：&nbsp;&nbsp;&nbsp;
  - **单加氧酶**：向底物加一个氧原子&nbsp;&nbsp;&nbsp;
  - **双加氧酶**：向底物加两个氧原子&nbsp;&nbsp;&nbsp;

#### 4. 氢过氧化物酶类&nbsp;&nbsp;&nbsp;

- **氢过氧化物酶(hydroperoxidase)**：主要分布于过氧化物酶体，催化有机过氧化物或过氧化氢还原&nbsp;&nbsp;&nbsp;
  - **过氧化物酶**：如谷胱甘肽过氧化物酶&nbsp;&nbsp;&nbsp;
    - 2GSH + H₂O₂ → GSSG + 2H₂O&nbsp;&nbsp;&nbsp;
  - **过氧化氢酶**：&nbsp;&nbsp;&nbsp;
    - 2H₂O₂ → 2H₂O + O₂&nbsp;&nbsp;&nbsp;

### 四、生物氧化中 CO₂ 的生成&nbsp;&nbsp;&nbsp;

- 生物氧化的终产物 CO₂ 来自**脱羧酶**催化的有机酸**脱羧基作用**&nbsp;&nbsp;&nbsp;

#### 分类方式&nbsp;&nbsp;&nbsp;

- 根据脱去 CO₂ 的羧基位置：&nbsp;&nbsp;&nbsp;
  - **α-脱羧**&nbsp;&nbsp;&nbsp;
  - **β-脱羧**&nbsp;&nbsp;&nbsp;
- 根据是否伴有脱氢反应：&nbsp;&nbsp;&nbsp;
  - **氧化脱羧**：伴有脱氢&nbsp;&nbsp;&nbsp;
  - **单纯脱羧**：不伴有脱氢&nbsp;&nbsp;&nbsp;
- 四种组合：&nbsp;&nbsp;&nbsp;
  - **α-单纯脱羧**&nbsp;&nbsp;&nbsp;
  - **α-氧化脱羧**&nbsp;&nbsp;&nbsp;
  - **β-单纯脱羧**&nbsp;&nbsp;&nbsp;
  - **β-氧化脱羧**&nbsp;&nbsp;&nbsp;

## 第二节 线粒体氧化体系&nbsp;&nbsp;&nbsp;

### 线粒体的结构与功能&nbsp;&nbsp;&nbsp;

- **外膜**：小分子和离子能自由通过&nbsp;&nbsp;&nbsp;
- **内膜**：小分子和离子不能自由通过，包含：&nbsp;&nbsp;&nbsp;
  - 呼吸链（复合物 I-IV）&nbsp;&nbsp;&nbsp;
  - ADP-ATP 转位酶&nbsp;&nbsp;&nbsp;
  - ATP 合酶(F₀F₁)&nbsp;&nbsp;&nbsp;
  - 其它膜载体蛋白&nbsp;&nbsp;&nbsp;
- **基质**：包含：&nbsp;&nbsp;&nbsp;
  - 丙酮酸脱氢酶复合物&nbsp;&nbsp;&nbsp;
  - 柠檬酸循环酶&nbsp;&nbsp;&nbsp;
  - 脂肪酸 β-氧化酶&nbsp;&nbsp;&nbsp;
  - 氨基酸氧化酶&nbsp;&nbsp;&nbsp;
  - DNA、核糖体、ATP、ADP&nbsp;&nbsp;&nbsp;

#### 线粒体中的三大过程&nbsp;&nbsp;&nbsp;

- **TCA 循环**（线粒体基质）：丙酮酸和脂肪酸氧化为 CO₂，同时生成还原型辅助因子（NADH 和 FADH₂）&nbsp;&nbsp;&nbsp;
- **呼吸链**（内膜）：NADH 和 FADH₂ 中电子经呼吸链传递给氧，H⁺泵到膜间隙形成跨膜质子梯度&nbsp;&nbsp;&nbsp;
- **氧化磷酸化**（内膜）：贮存于跨膜质子梯度中的能量由 ATP 合酶合成 ATP&nbsp;&nbsp;&nbsp;

### 一、呼吸链&nbsp;&nbsp;&nbsp;

#### 1. 呼吸链的概念(respiratory chain)&nbsp;&nbsp;&nbsp;

- 在线粒体内膜上，由传递氢或传递电子的多种酶与辅助因子组成的复合物，按照一定顺序排列的连锁性电子传递链，将代谢物所脱下的氢传递给氧生成水&nbsp;&nbsp;&nbsp;
- **递氢体**：传递氢的酶和辅助因子（如黄素蛋白含 FMN 辅基）&nbsp;&nbsp;&nbsp;
- **递电子体**：传递电子的酶和辅助因子（如血红素含铁离子：Fe²⁺ ⇌ Fe³⁺ + e⁻）&nbsp;&nbsp;&nbsp;

#### 2. 呼吸链的组成和排列顺序&nbsp;&nbsp;&nbsp;

##### (1) 呼吸链中常见的蛋白质和酶&nbsp;&nbsp;&nbsp;

- **黄素蛋白**：&nbsp;&nbsp;&nbsp;
  - 以 **FAD** 或 **FMN** 为辅助因子&nbsp;&nbsp;&nbsp;
  - 主要功能：传递 H 和电子&nbsp;&nbsp;&nbsp;
  - 含有核黄素衍生物 FMN 或 FAD 的蛋白质和酶类&nbsp;&nbsp;&nbsp;
- **铁硫蛋白**：&nbsp;&nbsp;&nbsp;
  - 辅助因子为**铁硫中心(Fe-S)**&nbsp;&nbsp;&nbsp;
  - 主要功能：传递**单个电子**&nbsp;&nbsp;&nbsp;
  - 铁原子通过 Fe²⁺ → Fe³⁺ + e⁻ 反应传递电子&nbsp;&nbsp;&nbsp;
  - 类型：Fe-S、Fe₂-S₂、Fe₄-S₄&nbsp;&nbsp;&nbsp;
- **泛醌(CoQ)**：&nbsp;&nbsp;&nbsp;
  - 即辅酶 Q（CoQ），**脂溶性**，能在内膜中**自由扩散**&nbsp;&nbsp;&nbsp;
  - 一类脂溶性醌类化合物，含有多个异戊二烯单位组成的侧链（**CoQ₁₀** 最常见）&nbsp;&nbsp;&nbsp;
  - 苯醌母核结构能可逆地加氢还原成对苯二酚化合物&nbsp;&nbsp;&nbsp;
  - 是呼吸链中的**递氢体和递电子体**&nbsp;&nbsp;&nbsp;
  - 存在三种形式：泛醌(氧化型)、泛醌H·(半醌型)、二氢泛醌(还原型)&nbsp;&nbsp;&nbsp;
- **细胞色素(Cytochrome, Cyt)**：&nbsp;&nbsp;&nbsp;
  - 由蛋白质和血红素组成，是以**铁卟啉为辅基**催化电子传递的酶类&nbsp;&nbsp;&nbsp;
  - 根据吸收光谱不同分为：**Cyt a、Cyt b、Cyt c** 三类及不同亚类&nbsp;&nbsp;&nbsp;
  - **Cyt c** 是**水溶性**蛋白质&nbsp;&nbsp;&nbsp;
  - 铁原子通过 Fe²⁺ ⇌ Fe³⁺ + e⁻ 反应传递电子&nbsp;&nbsp;&nbsp;

##### (2) 呼吸链的组成（4个复合体 + 2个游离分子）&nbsp;&nbsp;&nbsp;

- 由上述蛋白组装成 **4 个独立的大分子复合体**：I、II、III、IV&nbsp;&nbsp;&nbsp;
- 另有 **2 个游离分子**：**CoQ**（脂溶性，在线粒体内膜上自由移动）和 **Cyt c**（水溶性，在线粒体内膜外侧）&nbsp;&nbsp;&nbsp;

**复合体 I（NADH-泛醌还原酶）**：

- 又称 NADH-泛醌还原酶&nbsp;&nbsp;&nbsp;
- 电子传递：**NADH → FMN → Fe-S → CoQ**&nbsp;&nbsp;&nbsp;
- 每传递 2 个电子可将 **4 个 H⁺** 从内膜基质侧泵到胞浆侧&nbsp;&nbsp;&nbsp;
- 复合体 I **有质子泵功能**&nbsp;&nbsp;&nbsp;
- 辅酶/辅基：FMN、Fe-S&nbsp;&nbsp;&nbsp;

**复合体 II（琥珀酸-泛醌还原酶）**：

- 又称琥珀酸-泛醌还原酶&nbsp;&nbsp;&nbsp;
- 包含三羧酸循环中的**琥珀酸脱氢酶**&nbsp;&nbsp;&nbsp;
- 电子传递：**琥珀酸 → FAD → Fe-S → CoQ**&nbsp;&nbsp;&nbsp;
- 复合体 II **没有 H⁺泵的功能**&nbsp;&nbsp;&nbsp;
- 辅酶/辅基：FAD、Fe-S&nbsp;&nbsp;&nbsp;

**复合体 III（泛醌-细胞色素 c 还原酶）**：

- 又称泛醌-细胞色素 C 还原酶、细胞色素 b-c1 复合体&nbsp;&nbsp;&nbsp;
- 含有：细胞色素 b(b562, b566)、细胞色素 c1 和可移动的铁硫蛋白(Rieske protein)&nbsp;&nbsp;&nbsp;
- 泛醌从复合体 I、II 募集还原当量和电子并穿梭传递到复合体 III&nbsp;&nbsp;&nbsp;
- 电子传递：**CoQH₂ → (Cyt bL → Cyt bH) → Fe-S → Cyt c1 → Cyt c**&nbsp;&nbsp;&nbsp;
- 电子传递通过 **"Q 循环"** 实现&nbsp;&nbsp;&nbsp;
- 每传递 2 个电子向内膜胞浆侧释放 **4 个 H⁺**&nbsp;&nbsp;&nbsp;
- 复合体 III **有质子泵作用**&nbsp;&nbsp;&nbsp;
- **Cyt c** 是呼吸链唯一**水溶性球状蛋白**，不包含在复合体中&nbsp;&nbsp;&nbsp;

**复合体 IV（细胞色素 C 氧化酶）**：

- 又称细胞色素 C 氧化酶&nbsp;&nbsp;&nbsp;
- 电子传递：**Cyt c → CuA → Cyt a → Cyt a3-CuB → O₂**&nbsp;&nbsp;&nbsp;
- **Cyt a3-CuB** 形成活性双核中心，将电子传递给 O₂&nbsp;&nbsp;&nbsp;
- 每 2 个电子传递过程使 **2 个 H⁺** 跨内膜向胞浆侧转移&nbsp;&nbsp;&nbsp;
- 辅酶/辅基：血红素 a、a3、CuA、CuB&nbsp;&nbsp;&nbsp;

**CoQ**：传递复合体 I 或复合体 II 的电子给复合体 III

**Cyt c**：传递复合体 III 中的电子给复合体 IV

##### (3) 呼吸链组成的排列顺序&nbsp;&nbsp;&nbsp;

确定方法：
- 呼吸链复合物的重组&nbsp;&nbsp;&nbsp;
- 呼吸链各组分氧化态和还原态吸收光谱不同&nbsp;&nbsp;&nbsp;
- 呼吸链抑制剂&nbsp;&nbsp;&nbsp;
- 标准氧化还原电位低（还原性强）到高（氧化性强）的顺序排列&nbsp;&nbsp;&nbsp;

**NADH 氧化呼吸链**：

- NADH → 复合体 I → Q → 复合体 III → Cyt c → 复合体 IV → O₂&nbsp;&nbsp;&nbsp;

**琥珀酸氧化呼吸链（FADH₂ 呼吸链）**：

- 琥珀酸 → 复合体 II → Q → 复合体 III → Cyt c → 复合体 IV → O₂&nbsp;&nbsp;&nbsp;

### 二、氧化磷酸化&nbsp;&nbsp;&nbsp;

#### 1. 氧化磷酸化的概念(Oxidative phosphorylation)&nbsp;&nbsp;&nbsp;

- 营养物质分解代谢过程中脱氢生成 NADH＋H⁺ 和 FADH₂，经线粒体呼吸链传递电子、泵出质子并逐步释放能量，偶联驱动 ADP 磷酸化生成 ATP&nbsp;&nbsp;&nbsp;
- 这种氧化与磷酸化**紧密偶联**的过程即氧化磷酸化&nbsp;&nbsp;&nbsp;
- 氧化磷酸化是体内 **ATP 生成的主要方式**&nbsp;&nbsp;&nbsp;

#### 2. 氧化磷酸化的偶联部位&nbsp;&nbsp;&nbsp;

- 偶联部位通过 **P/O 比值**和**自由能改变**进行推测&nbsp;&nbsp;&nbsp;

##### P/O 比值&nbsp;&nbsp;&nbsp;

- 每消耗 1mol 氧原子所消耗的无机磷的摩尔数，即生成 ATP 的摩尔数&nbsp;&nbsp;&nbsp;
- 实质是电子传递过程中磷酸化的效率（一对电子通过氧化呼吸链传递给氧所生成的 ATP 数）&nbsp;&nbsp;&nbsp;

##### 偶联部位与 ATP 产量&nbsp;&nbsp;&nbsp;

- **NADH 呼吸链**：3 个偶联部位，**P/O = 2.5** → 一分子 NADH 生成 **2.5 个 ATP**&nbsp;&nbsp;&nbsp;
- **FADH₂ 呼吸链**：2 个偶联部位，**P/O = 1.5** → 一分子 FADH₂ 生成 **1.5 个 ATP**&nbsp;&nbsp;&nbsp;

#### 3. 氧化磷酸化偶联机制——化学渗透假说&nbsp;&nbsp;&nbsp;

- 电子经呼吸链传递时，将质子(H⁺)从线粒体内膜的基质侧泵到膜间隙，产生膜内外**氢离子浓度梯度**和**跨膜电位差**，以此储存能量&nbsp;&nbsp;&nbsp;
- 当质子**顺浓度梯度回流**时驱动 ADP 与 Pi 生成 ATP&nbsp;&nbsp;&nbsp;
- **ATP 合酶(F₀F₁)**：&nbsp;&nbsp;&nbsp;
  - **F₀**：疏水部分，镶嵌在线粒体内膜中，形成跨内膜**质子通道**&nbsp;&nbsp;&nbsp;
  - **F₁**：亲水部分，线粒体内膜的基质侧颗粒状突起，**催化 ATP 合成**&nbsp;&nbsp;&nbsp;
  - F₁ β 亚基的三种状态：**L 态**（捕获 ADP 和 Pi）→ **T 态**（合成 ATP）→ **O 态**（释放 ATP）&nbsp;&nbsp;&nbsp;

#### 4. 影响氧化磷酸化的因素&nbsp;&nbsp;&nbsp;

##### (1) ADP 的调节作用&nbsp;&nbsp;&nbsp;

- ADP 是调节人体氧化磷酸化速率的**主要因素**&nbsp;&nbsp;&nbsp;
- ADP/ATP ↑ → 氧化磷酸化 ↑&nbsp;&nbsp;&nbsp;

##### (2) 甲状腺激素&nbsp;&nbsp;&nbsp;

- 诱导**解偶联蛋白**基因表达 → 热能散发，ATP 合成减低&nbsp;&nbsp;&nbsp;
- 刺激**钠钾-ATP 酶（钠泵）** 合成 → 钠泵运转耗能 → ATP/ADP 降低 → 促进氧化磷酸化 → 基础代谢率增高（甲亢患者）&nbsp;&nbsp;&nbsp;

##### (3) 氧化磷酸化的抑制剂&nbsp;&nbsp;&nbsp;

**呼吸链抑制剂（直接阻断电子传递）**：

- 复合体 I：**鱼藤酮**、阿米妥（异戊巴比妥）、**粉蝶霉素 A**（结合复合体 I 铁硫蛋白）&nbsp;&nbsp;&nbsp;
- 复合体 II：**萎锈灵**&nbsp;&nbsp;&nbsp;
- 复合体 III：**抗霉素 A**&nbsp;&nbsp;&nbsp;
- 复合体 IV：**氰化物(CN⁻)**、**CO**（人体中毒）、**H₂S**、叠氮化物(N₃⁻)&nbsp;&nbsp;&nbsp;

**解偶联剂**：

- 破坏电子传递建立的**跨膜质子电化学梯度**，使氧化与磷酸化解偶联&nbsp;&nbsp;&nbsp;
- 电化学梯度储存的能量以**热能**形式释放，合成 ATP 的量**降低**&nbsp;&nbsp;&nbsp;
- 外源性：**二硝基苯酚(DNP)**、双香豆素（脂溶性）&nbsp;&nbsp;&nbsp;
- 内源性：**解偶联蛋白(UCP)**（褐色脂肪细胞线粒体）&nbsp;&nbsp;&nbsp;

**ATP 合酶抑制剂**：

- **二环己基碳二亚胺(DCCD)**（结合 F₀）&nbsp;&nbsp;&nbsp;
- **寡霉素**（抗菌、抗肿瘤）（结合 F₀）&nbsp;&nbsp;&nbsp;
- 寡霉素、DCCD 结合 F₀ → 阻止质子从 F₀ 质子通道回流 → 抑制 ATP 生成&nbsp;&nbsp;&nbsp;

##### (4) 线粒体 DNA 突变&nbsp;&nbsp;&nbsp;

- 线粒体 DNA 编码一些呼吸链蛋白&nbsp;&nbsp;&nbsp;
- 电子传递过程中伴随大量氧自由基产生 → 可造成线粒体 DNA 突变 → 影响氧化磷酸化功能&nbsp;&nbsp;&nbsp;

### 三、ATP 等高能化合物&nbsp;&nbsp;&nbsp;

#### 1. 高能键和高能化合物&nbsp;&nbsp;&nbsp;

- **高能键**：在标准条件下，水解时释放的能量大于 **25kJ/mol** 的化学键，常用 ~ 表示&nbsp;&nbsp;&nbsp;
- **高能化合物**：含有高能键的化合物&nbsp;&nbsp;&nbsp;
- 常见的高能化合物：&nbsp;&nbsp;&nbsp;
  - **高能磷酸化合物**：ATP、磷酸肌酸、1,3-二磷酸甘油酸、磷酸烯醇式丙酮酸(PEP)&nbsp;&nbsp;&nbsp;
  - **高能硫酯化合物**：乙酰 CoA、琥珀酰 CoA、脂酰 CoA、S-腺苷甲硫氨酸、PAPS&nbsp;&nbsp;&nbsp;

#### 磷酸肌酸&nbsp;&nbsp;&nbsp;

- 肌酸 + ATP →（肌酸激酶）→ 磷酸肌酸 + ADP&nbsp;&nbsp;&nbsp;
- 磷酸肌酸作为**肌肉、心脏和脑组织**中能量的一种**贮存形式**&nbsp;&nbsp;&nbsp;
- 当机体 ATP 不足时，磷酸肌酸将 ~P 转移给 ADP 生成 ATP&nbsp;&nbsp;&nbsp;

#### 2. ATP 的生成、储存和利用&nbsp;&nbsp;&nbsp;

##### (1) ATP 的生成方式&nbsp;&nbsp;&nbsp;

- **氧化磷酸化**（主要方式）&nbsp;&nbsp;&nbsp;
- **底物水平磷酸化**（次要方式）&nbsp;&nbsp;&nbsp;
  - 底物水平磷酸化：由于脱氢或脱水引起代谢物分子内部能量聚集形成高能键，然后将高能键裂解将能量转移给 ADP（或 GDP）形成 ATP（或 GTP）的过程&nbsp;&nbsp;&nbsp;

##### 底物水平磷酸化的三个反应（本书仅见）&nbsp;&nbsp;&nbsp;

- 1,3-二磷酸甘油酸 + ADP →（磷酸甘油酸激酶）→ 3-磷酸甘油酸 + ATP&nbsp;&nbsp;&nbsp;
- 磷酸烯醇式丙酮酸 + ADP →（丙酮酸激酶）→ 丙酮酸 + ATP&nbsp;&nbsp;&nbsp;
- 琥珀酰 CoA + H₃PO₄ + GDP →（琥珀酰硫激酶）→ 琥珀酸 + GTP + CoA&nbsp;&nbsp;&nbsp;

##### (2) ATP 的作用&nbsp;&nbsp;&nbsp;

- 生物体内能量的转移、储存和利用均以 ATP 为中心，ATP 被喻为**"能量的通用货币"**，即构成 **ATP 循环**&nbsp;&nbsp;&nbsp;
- 分解代谢或合成代谢时底物的磷酸化或活化&nbsp;&nbsp;&nbsp;
- 参与生物大分子合成等各种生命活动过程&nbsp;&nbsp;&nbsp;
- 转变成磷酸肌酸储存能量&nbsp;&nbsp;&nbsp;

### 四、线粒体内膜的物质转运&nbsp;&nbsp;&nbsp;

#### 线粒体内膜对各种物质进行选择性转运&nbsp;&nbsp;&nbsp;

- 线粒体外膜通透性高，线粒体对物质通过的选择性主要依赖于**内膜中不同转运蛋白**&nbsp;&nbsp;&nbsp;

##### 主要转运蛋白&nbsp;&nbsp;&nbsp;

- **ATP-ADP 转位酶**：进线粒体 → ADP；出线粒体 → ATP&nbsp;&nbsp;&nbsp;
- **α-酮戊二酸转运蛋白**：进线粒体 → 苹果酸；出线粒体 → α-酮戊二酸&nbsp;&nbsp;&nbsp;
- **天冬氨酸-谷氨酸转运蛋白**：进线粒体 → 谷氨酸；出线粒体 → 天冬氨酸&nbsp;&nbsp;&nbsp;
- **三羧酸转运蛋白**：进线粒体 → 苹果酸；出线粒体 → 柠檬酸&nbsp;&nbsp;&nbsp;

#### 胞浆中 NADH 氧化磷酸化的方式（两种穿梭机制）&nbsp;&nbsp;&nbsp;

- 线粒体内的 NADH 可直接参加氧化磷酸化，但胞浆中的 NADH 不能自由通过线粒体内膜&nbsp;&nbsp;&nbsp;
- 必须通过两种穿梭机制进入线粒体：&nbsp;&nbsp;&nbsp;

##### 苹果酸-天冬氨酸穿梭&nbsp;&nbsp;&nbsp;

- 存在部位：**心、肝**等&nbsp;&nbsp;&nbsp;
- 进入的呼吸链：经 NADH 进入 **NADH 呼吸链**&nbsp;&nbsp;&nbsp;
- ATP 生成量：**2.5 分子 ATP**&nbsp;&nbsp;&nbsp;

##### α-磷酸甘油穿梭&nbsp;&nbsp;&nbsp;

- 存在部位：**脑、神经、肌肉**等&nbsp;&nbsp;&nbsp;
- 进入的呼吸链：经 FADH₂ 进入 **FADH₂ 呼吸链**&nbsp;&nbsp;&nbsp;
- ATP 生成量：**1.5 分子 ATP**&nbsp;&nbsp;&nbsp;

## 第三节 非线粒体氧化体系（不生成 ATP）&nbsp;&nbsp;&nbsp;

### 一、概述&nbsp;&nbsp;&nbsp;

- 除线粒体的氧化体系外，在微粒体、过氧化物酶体以及细胞其他部位还存在其他氧化体系&nbsp;&nbsp;&nbsp;
- **特点**：不伴随磷酸化，不能生成 ATP，主要参与体内代谢物、药物和毒物的**生物转化**&nbsp;&nbsp;&nbsp;
- 包括：微粒体氧化体系、过氧化物酶体氧化体系&nbsp;&nbsp;&nbsp;

### 二、微粒体氧化体系&nbsp;&nbsp;&nbsp;

- 主要含有**细胞色素 P450 加单氧酶系**&nbsp;&nbsp;&nbsp;
- 该酶系催化氧分子中的一个氧原子加到底物上（使底物羟化），另一个氧原子被氢（来自 NADPH+H⁺）还原生成水&nbsp;&nbsp;&nbsp;
- 故又称**混合功能氧化酶**&nbsp;&nbsp;&nbsp;
- RH + O₂ + NADPH + H⁺ →（加单氧酶）→ ROH + NADP⁺ + H₂O&nbsp;&nbsp;&nbsp;

### 三、过氧化物酶体氧化体系&nbsp;&nbsp;&nbsp;

- 包含**过氧化氢酶**和**过氧化物酶**&nbsp;&nbsp;&nbsp;
- 过氧化氢酶：2H₂O₂ → 2H₂O + O₂&nbsp;&nbsp;&nbsp;
- 过氧化物酶：R + H₂O₂ → RO + H₂O&nbsp;&nbsp;&nbsp;

### 四、抗氧化酶系&nbsp;&nbsp;&nbsp;

#### 反应活性氧类(Reactive Oxygen Species, ROS)&nbsp;&nbsp;&nbsp;

- ROS 主要指 O₂ 的单电子还原产物&nbsp;&nbsp;&nbsp;
- 种类：&nbsp;&nbsp;&nbsp;
  - **O₂⁻·**（超氧阴离子自由基）&nbsp;&nbsp;&nbsp;
  - **H₂O₂**（过氧化氢）&nbsp;&nbsp;&nbsp;
  - **·OH**（羟自由基）&nbsp;&nbsp;&nbsp;
- 生成过程：O₂ →(e⁻)→ O₂⁻· →(e⁻+2H⁺)→ H₂O₂ →(e⁻+H⁺)→ ·OH →(e⁻+H⁺)→ H₂O&nbsp;&nbsp;&nbsp;
- 来源：&nbsp;&nbsp;&nbsp;
  - **内源性**：细胞内呼吸链电子传递泄露（主要）&nbsp;&nbsp;&nbsp;
  - **外源性**：感染、药物、放射等&nbsp;&nbsp;&nbsp;
- 危害：导致蛋白质、DNA 等大分子损伤，进而破坏正常细胞的结构与功能，引发各种疾病&nbsp;&nbsp;&nbsp;

#### 体内主要的抗氧化体系&nbsp;&nbsp;&nbsp;

- **谷胱甘肽过氧化物酶(GPx)**：可消除 H₂O₂ 和过氧化物(R-OOH)，是体内防止活性氧类损伤的**主要酶**&nbsp;&nbsp;&nbsp;
- **过氧化物酶**：消除 H₂O₂ 和过氧化物&nbsp;&nbsp;&nbsp;
- **超氧化物歧化酶(SOD)**：清除自由基 O₂⁻·，是人体防御内外环境中超氧离子损伤的**重要酶**&nbsp;&nbsp;&nbsp;
  - 2O₂⁻· + 2H⁺ →(SOD)→ H₂O₂ + O₂ →(过氧化氢酶)→ H₂O + O₂&nbsp;&nbsp;&nbsp;
- **其它抗氧化物质**：如维生素 E、A、C、泛醌等&nbsp;&nbsp;&nbsp;
