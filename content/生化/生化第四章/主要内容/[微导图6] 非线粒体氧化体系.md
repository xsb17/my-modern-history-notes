# 非线粒体氧化体系&nbsp;&nbsp;&nbsp;

- **非线粒体氧化体系概述**&nbsp;&nbsp;&nbsp;
	- 位置 -> **微粒体、过氧化物酶体**等细胞其他部位&nbsp;&nbsp;&nbsp;
	- 特点 -> **不伴随磷酸化** -> **不生成ATP** -> 主要参与代谢物、药物和毒物的**生物转化**&nbsp;&nbsp;&nbsp;
	- 包括 -> 微粒体氧化体系 + 过氧化物酶体氧化体系 + 抗氧化酶系&nbsp;&nbsp;&nbsp;
- **微粒体氧化体系**&nbsp;&nbsp;&nbsp;
	- 核心酶 -> **细胞色素P450加单氧酶系**&nbsp;&nbsp;&nbsp;
	- 别名 -> **混合功能氧化酶** / **羟化酶** / **加单氧酶**&nbsp;&nbsp;&nbsp;
	- 反应 -> RH + O₂ + **NADPH + H⁺** → ROH + NADP⁺ + H₂O&nbsp;&nbsp;&nbsp;
	- 机制 -> O₂一个氧原子**加到底物**(羟化) -> 另一个氧原子被NADPH的H还原生成**H₂O**&nbsp;&nbsp;&nbsp;
	- 功能 -> 参与**肝脏生物转化** -> 药物代谢/毒物解毒/类固醇合成&nbsp;&nbsp;&nbsp;
- **过氧化物酶体氧化体系**&nbsp;&nbsp;&nbsp;
	- **过氧化氢酶(CAT)**&nbsp;&nbsp;&nbsp;
		- 反应 -> 2H₂O₂ → 2H₂O + **O₂**&nbsp;&nbsp;&nbsp;
		- 作用 -> 分解**H₂O₂** -> 防止氧化损伤&nbsp;&nbsp;&nbsp;
	- **过氧化物酶**&nbsp;&nbsp;&nbsp;
		- 反应 -> R + H₂O₂ → RO + H₂O&nbsp;&nbsp;&nbsp;
		- 如：**谷胱甘肽过氧化物酶(GPx)** -> 2GSH + H₂O₂ → GSSG + 2H₂O&nbsp;&nbsp;&nbsp;
- **反应活性氧类(ROS)**&nbsp;&nbsp;&nbsp;
	- 定义 -> O₂的**单电子还原产物**&nbsp;&nbsp;&nbsp;
	- 种类&nbsp;&nbsp;&nbsp;
		- **O₂⁻·**（超氧阴离子自由基）&nbsp;&nbsp;&nbsp;
		- **H₂O₂**（过氧化氢）&nbsp;&nbsp;&nbsp;
		- **·OH**（羟自由基）-> 最活跃、毒性最强&nbsp;&nbsp;&nbsp;
	- 生成链 -> **O₂** →(+e⁻)→ **O₂⁻·** →(+e⁻+2H⁺)→ **H₂O₂** →(+e⁻+H⁺)→ **·OH** →(+e⁻+H⁺)→ **H₂O**&nbsp;&nbsp;&nbsp;
	- 来源&nbsp;&nbsp;&nbsp;
		- **内源性**（主要）-> 呼吸链电子传递**泄露** -> 约1-2%的O₂被不完全还原&nbsp;&nbsp;&nbsp;
		- **外源性** -> 感染、药物、放射、吸烟等&nbsp;&nbsp;&nbsp;
	- 危害 -> 攻击**蛋白质/DNA/脂质** -> 氧化损伤 -> 衰老/癌症/动脉粥样硬化等疾病&nbsp;&nbsp;&nbsp;
- **抗氧化防御体系**&nbsp;&nbsp;&nbsp;
	- **超氧化物歧化酶(SOD)**&nbsp;&nbsp;&nbsp;
		- 反应 -> 2O₂⁻· + 2H⁺ → **H₂O₂ + O₂**&nbsp;&nbsp;&nbsp;
		- 地位 -> 人体防御超氧离子损伤的**重要酶**&nbsp;&nbsp;&nbsp;
		- 类型 -> Cu/Zn-SOD(胞浆) / Mn-SOD(线粒体)&nbsp;&nbsp;&nbsp;
	- **谷胱甘肽过氧化物酶(GPx)**&nbsp;&nbsp;&nbsp;
		- 反应 -> 2GSH + H₂O₂(或ROOH) → GSSG + H₂O(或ROH+H₂O)&nbsp;&nbsp;&nbsp;
		- 地位 -> 防止活性氧类损伤的**主要酶**&nbsp;&nbsp;&nbsp;
		- 含**硒(Se)**为辅因子&nbsp;&nbsp;&nbsp;
	- **过氧化氢酶(CAT)**&nbsp;&nbsp;&nbsp;
		- 反应 -> 2H₂O₂ → 2H₂O + O₂&nbsp;&nbsp;&nbsp;
	- 协同路径 -> **SOD**清除O₂⁻· → 生成H₂O₂ → **GPx/CAT**清除H₂O₂&nbsp;&nbsp;&nbsp;
	- **非酶抗氧化剂** -> **维生素E**(VE)/**维生素C**(VC)/维生素A/泛醌(CoQ)&nbsp;&nbsp;&nbsp;
		- VE -> 脂溶性 -> 保护**细胞膜**脂质 -> 阻断脂质过氧化链式反应&nbsp;&nbsp;&nbsp;
		- VC -> 水溶性 -> 再生VE -> 直接清除自由基&nbsp;&nbsp;&nbsp;
- **微导图速记**&nbsp;&nbsp;&nbsp;
	- 非线粒体氧化 -> **不产ATP** -> 核心：**P450**(微粒体) + **ROS**(需氧代谢副产物) + **SOD/GPx/CAT**(三道防线)&nbsp;&nbsp;&nbsp;
	- 抗氧化酶协同 -> SOD(**"扫地"**O₂⁻·→H₂O₂) → GPx/CAT(**"拖地"**H₂O₂→H₂O+O₂)&nbsp;&nbsp;&nbsp;
