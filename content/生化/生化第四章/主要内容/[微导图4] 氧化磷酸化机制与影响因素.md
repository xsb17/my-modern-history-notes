# 氧化磷酸化机制与影响因素&nbsp;&nbsp;&nbsp;

- **氧化磷酸化的概念**&nbsp;&nbsp;&nbsp;
	- 营养物质脱氢 -> 生成 **NADH+FADH₂** -> 经呼吸链传递电子 -> 泵出 **H⁺** -> 逐步**释放能量**&nbsp;&nbsp;&nbsp;
	- 偶联驱动 **ADP + Pi → ATP** -> 氧化与磷酸化**紧密偶联**&nbsp;&nbsp;&nbsp;
	- 是体内 **ATP 生成的主要方式**&nbsp;&nbsp;&nbsp;
- **P/O 比值**&nbsp;&nbsp;&nbsp;
	- 定义 -> 每消耗 **1mol 氧原子**所消耗的无机磷摩尔数 = 生成ATP的摩尔数&nbsp;&nbsp;&nbsp;
	- 实质 -> 电子传递过程中**磷酸化的效率**&nbsp;&nbsp;&nbsp;
	- **NADH呼吸链** -> P/O = **2.5** -> 3个偶联部位&nbsp;&nbsp;&nbsp;
	- **FADH₂呼吸链** -> P/O = **1.5** -> 2个偶联部位&nbsp;&nbsp;&nbsp;
- **化学渗透假说（Peter Mitchell提出）**&nbsp;&nbsp;&nbsp;
	- 核心 -> 电子传递 → 释放能量 → 将 **H⁺(质子)** 从基质侧**泵到膜间隙**&nbsp;&nbsp;&nbsp;
	- 结果 -> 形成跨内膜**质子电化学梯度** -> 质子浓度梯度 + 跨膜电位差 -> 储存能量&nbsp;&nbsp;&nbsp;
	- 驱动 -> H⁺**顺浓度梯度回流** → 通过ATP合酶 → 驱动 **ADP + Pi → ATP**&nbsp;&nbsp;&nbsp;
- **ATP合酶(F₀F₁复合体)**&nbsp;&nbsp;&nbsp;
	- **F₀** -> 疏水部分 -> 镶嵌于**线粒体内膜** -> 形成跨内膜**质子通道**&nbsp;&nbsp;&nbsp;
	- **F₁** -> 亲水部分 -> 基质侧颗粒状突起 -> **催化ATP合成**&nbsp;&nbsp;&nbsp;
	- F₁ β亚基三种状态循环&nbsp;&nbsp;&nbsp;
		- **L态**（松散态） -> 捕获 **ADP + Pi**&nbsp;&nbsp;&nbsp;
		- **T态**（紧密态） -> 使结合的ADP和Pi**合成ATP**&nbsp;&nbsp;&nbsp;
		- **O态**（开放态） -> **释放ATP**&nbsp;&nbsp;&nbsp;
- **影响氧化磷酸化的因素**&nbsp;&nbsp;&nbsp;
	- **ADP的调节作用**（最主要因素）&nbsp;&nbsp;&nbsp;
		- **ADP/ATP ↑** -> 氧化磷酸化**加速**↑&nbsp;&nbsp;&nbsp;
		- **ADP/ATP ↓** -> 氧化磷酸化**减慢**↓&nbsp;&nbsp;&nbsp;
		- ADP是调节人体氧化磷酸化速率的**主要因素**&nbsp;&nbsp;&nbsp;
	- **甲状腺激素**&nbsp;&nbsp;&nbsp;
		- 诱导**解偶联蛋白(UCP)**基因表达 -> 热能散发 -> **ATP合成↓**&nbsp;&nbsp;&nbsp;
		- 刺激**钠钾-ATP酶（钠泵）**合成 -> 钠泵耗能 -> ATP/ADP↓ -> 促进氧化磷酸化&nbsp;&nbsp;&nbsp;
		- 结果 -> **基础代谢率增高** -> 甲亢患者表现&nbsp;&nbsp;&nbsp;
	- **氧化磷酸化抑制剂**（见微导图5）&nbsp;&nbsp;&nbsp;
		- 呼吸链抑制剂 -> 阻断电子传递&nbsp;&nbsp;&nbsp;
		- 解偶联剂 -> 破坏质子梯度&nbsp;&nbsp;&nbsp;
		- ATP合酶抑制剂 -> 阻断质子回流&nbsp;&nbsp;&nbsp;
	- **线粒体DNA突变**&nbsp;&nbsp;&nbsp;
		- 线粒体DNA编码部分呼吸链蛋白&nbsp;&nbsp;&nbsp;
		- 电子传递中产生**氧自由基** -> 损伤mtDNA -> 影响氧化磷酸化&nbsp;&nbsp;&nbsp;
- **ATP — 能量通用货币**&nbsp;&nbsp;&nbsp;
	- 生成方式&nbsp;&nbsp;&nbsp;
		- **氧化磷酸化**（主要）-> 呼吸链偶联 -> 在线粒体内膜&nbsp;&nbsp;&nbsp;
		- **底物水平磷酸化**（次要）-> 代谢物高能键直接转移&nbsp;&nbsp;&nbsp;
	- 功能 -> 能量转移/贮存/利用的**中心** -> 构成 **ATP循环**&nbsp;&nbsp;&nbsp;
	- 可转变为**磷酸肌酸**储存能量（肌肉/心/脑）&nbsp;&nbsp;&nbsp;
