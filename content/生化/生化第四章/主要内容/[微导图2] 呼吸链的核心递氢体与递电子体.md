# 呼吸链的核心递氢体与递电子体&nbsp;&nbsp;&nbsp;

- **呼吸链概述**&nbsp;&nbsp;&nbsp;
	- 位于**线粒体内膜** -> 多种酶与辅助因子按一定顺序排列&nbsp;&nbsp;&nbsp;
	- 功能 -> 将代谢物脱下的**氢**传递给**O₂** -> 生成 **H₂O**&nbsp;&nbsp;&nbsp;
	- **递氢体** -> 传递**氢+H+电子** -> 如：黄素蛋白(FMN/FAD)&nbsp;&nbsp;&nbsp;
	- **递电子体** -> 只传递**电子** -> 如：细胞色素(Fe²⁺⇌Fe³⁺+e⁻)&nbsp;&nbsp;&nbsp;
- **黄素蛋白**&nbsp;&nbsp;&nbsp;
	- 核心成分 -> 含 **FMN** 或 **FAD**&nbsp;&nbsp;&nbsp;
	- 辅基来源 -> **维生素B₂（核黄素）** 衍生物&nbsp;&nbsp;&nbsp;
	- 功能 -> 传递**氢+电子** -> 既可递氢又可递电子&nbsp;&nbsp;&nbsp;
	- FMN/FAD -> 氧化型 ⇌ 还原型（FMNH₂/FADH₂）&nbsp;&nbsp;&nbsp;
	- 典型代表 -> 复合体I含FMN / 复合体II含FAD&nbsp;&nbsp;&nbsp;
- **铁硫蛋白(Fe-S)**&nbsp;&nbsp;&nbsp;
	- 核心成分 -> **铁硫中心(Fe-S)** -> 非血红素铁&nbsp;&nbsp;&nbsp;
	- 功能 -> 传递**单个电子** -> 通过 Fe²⁺ → Fe³⁺ + e⁻&nbsp;&nbsp;&nbsp;
	- 类型 -> Fe-S / Fe₂-S₂ / **Fe₄-S₄**&nbsp;&nbsp;&nbsp;
	- 存在于 -> 复合体I、II、III中&nbsp;&nbsp;&nbsp;
	- 特点 -> 不传递氢，只传递电子&nbsp;&nbsp;&nbsp;
- **泛醌(CoQ/辅酶Q) — 必考核心**&nbsp;&nbsp;&nbsp;
	- 特点 -> 呼吸链中**唯一非蛋白质**组分&nbsp;&nbsp;&nbsp;
	- 性质 -> **脂溶性** -> 可在线粒体内膜中**自由游动/扩散**&nbsp;&nbsp;&nbsp;
	- 侧链 -> 多个异戊二烯单位 -> **CoQ₁₀**最常见（人体)&nbsp;&nbsp;&nbsp;
	- 功能 -> **既可递氢又可递电子** -> 是递氢体和递电子体&nbsp;&nbsp;&nbsp;
	- 三种形式 -> 氧化型(醌型) ⇌ 半醌型 ⇌ 还原型(氢醌型)&nbsp;&nbsp;&nbsp;
	- 关键角色 -> **两条呼吸链的汇合点** -> 从复合体I和II募集电子 -> 传递到复合体III&nbsp;&nbsp;&nbsp;
- **细胞色素(Cyt)**&nbsp;&nbsp;&nbsp;
	- 核心成分 -> 蛋白质 + **血红素（铁卟啉）** 为辅基&nbsp;&nbsp;&nbsp;
	- 分类（按吸收光谱） -> **Cyt a** / **Cyt b** / **Cyt c** 三类及亚类&nbsp;&nbsp;&nbsp;
	- 功能 -> 传递**单个电子** -> 通过 Fe²⁺ ⇌ Fe³⁺ + e⁻&nbsp;&nbsp;&nbsp;
	- **Cyt c** -> 呼吸链中**唯一水溶性**球状蛋白 -> 位于线粒体内膜**外侧** -> 游离分子，不属于四大复合体&nbsp;&nbsp;&nbsp;
	- **Cyt a₃** -> 含于复合体IV -> 与CuB形成**活性双核中心** -> 直接将电子传递给**O₂**&nbsp;&nbsp;&nbsp;
	- 不能传递氢，只能传递电子&nbsp;&nbsp;&nbsp;
- **5大递氢/递电子体速记**&nbsp;&nbsp;&nbsp;
	- **NAD⁺/NADH** -> 水溶性 -> 传递**氢+电子** -> 不直接属于呼吸链组分（底物）&nbsp;&nbsp;&nbsp;
	- **FMN/FAD**（黄素蛋白） -> 传递**氢+电子** -> 复合体I用FMN / 复合体II用FAD&nbsp;&nbsp;&nbsp;
	- **Fe-S**（铁硫蛋白） -> 只传递**电子** -> Fe²⁺⇌Fe³⁺&nbsp;&nbsp;&nbsp;
	- **CoQ**（泛醌） -> 传递**氢+电子** -> 脂溶性/自由游动/两条链的汇合点&nbsp;&nbsp;&nbsp;
	- **Cyt**（细胞色素） -> 只传递**电子** -> 铁卟啉/Fe²⁺⇌Fe³⁺&nbsp;&nbsp;&nbsp;
