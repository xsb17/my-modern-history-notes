# 生物氧化与高能磷酸化合物&nbsp;&nbsp;&nbsp;

- **生物氧化的概念**&nbsp;&nbsp;&nbsp;
	- 物质在**生物体内**进行的氧化 -> 糖、脂肪、蛋白质在活细胞内氧化分解&nbsp;&nbsp;&nbsp;
	- 终产物 -> **CO₂** + **H₂O** -> 逐步释放**能量**&nbsp;&nbsp;&nbsp;
- **生物氧化 vs 体外燃烧**&nbsp;&nbsp;&nbsp;
	- 相同点&nbsp;&nbsp;&nbsp;
		- 遵循氧化还原**一般规律**&nbsp;&nbsp;&nbsp;
		- 耗氧量、**最终产物**、**释放能量**均相同&nbsp;&nbsp;&nbsp;
	- 不同点（生物氧化特征）&nbsp;&nbsp;&nbsp;
		- 条件 -> 有水、**体温**、近中性pH -> 细胞内 -> **条件温和**&nbsp;&nbsp;&nbsp;
		- 催化 -> **酶**催化 -> 逐步进行&nbsp;&nbsp;&nbsp;
		- 能量 -> **逐步释放** -> 部分**化学能**(ATP)贮存 -> 部分**热能**释放&nbsp;&nbsp;&nbsp;
		- 产物机制 -> **加水脱氢**间接获得氧 -> CO₂来自有机酸**脱羧** -> H₂O来自脱下的氢与O₂结合&nbsp;&nbsp;&nbsp;
- **参与生物氧化的酶类**&nbsp;&nbsp;&nbsp;
	- **氧化酶类** -> 以**O₂为受氢体** -> 产物H₂O -> 如：**细胞色素氧化酶**&nbsp;&nbsp;&nbsp;
	- **脱氢酶类**（最主要） -> 不以O₂为直接受氢体 -> 受氢体为**NAD⁺/NADP⁺/FAD/FMN**&nbsp;&nbsp;&nbsp;
	- **加氧酶类** -> 微粒体 -> 分**单加氧酶**和**双加氧酶**&nbsp;&nbsp;&nbsp;
	- **氢过氧化物酶类** -> 过氧化物酶体 -> 包括**过氧化氢酶**和过氧化物酶&nbsp;&nbsp;&nbsp;
- **CO₂的生成方式**&nbsp;&nbsp;&nbsp;
	- 来自**脱羧酶**催化的有机酸脱羧&nbsp;&nbsp;&nbsp;
	- 按羧基位置分 -> **α-脱羧** / **β-脱羧**&nbsp;&nbsp;&nbsp;
	- 按是否伴脱氢分 -> **氧化脱羧** / **单纯脱羧**&nbsp;&nbsp;&nbsp;
	- 四种组合 -> α-单纯脱羧 / α-氧化脱羧 / β-单纯脱羧 / β-氧化脱羧&nbsp;&nbsp;&nbsp;
- **高能化合物**&nbsp;&nbsp;&nbsp;
	- **高能键** -> 水解时释放能量 > **25kJ/mol** -> 用 **~** 表示&nbsp;&nbsp;&nbsp;
	- **高能磷酸化合物** -> ATP、**磷酸肌酸**、1,3-二磷酸甘油酸、**PEP**（磷酸烯醇式丙酮酸）&nbsp;&nbsp;&nbsp;
	- **高能硫酯化合物** -> **乙酰CoA**、琥珀酰CoA、脂酰CoA、S-腺苷甲硫氨酸&nbsp;&nbsp;&nbsp;
- **ATP — 能量的通用货币**&nbsp;&nbsp;&nbsp;
	- 生成方式&nbsp;&nbsp;&nbsp;
		- **氧化磷酸化**（主要方式） -> 呼吸链传递电子偶联ADP磷酸化&nbsp;&nbsp;&nbsp;
		- **底物水平磷酸化**（次要方式） -> 代谢物分子内部能量聚集形成高能键 -> 直接转移给ADP/GDP&nbsp;&nbsp;&nbsp;
	- 三个底物水平磷酸化反应&nbsp;&nbsp;&nbsp;
		- **1,3-二磷酸甘油酸** + ADP -> 3-磷酸甘油酸 + ATP ->（磷酸甘油酸激酶）&nbsp;&nbsp;&nbsp;
		- **PEP** + ADP -> 丙酮酸 + ATP ->（丙酮酸激酶）&nbsp;&nbsp;&nbsp;
		- **琥珀酰CoA** + GDP + Pi -> 琥珀酸 + GTP + CoA ->（琥珀酰硫激酶）&nbsp;&nbsp;&nbsp;
	- ATP 作用 -> 能量转移**中心** -> 构成 **ATP 循环** -> 参与生物大分子合成 -> 转变成磷酸肌酸储存&nbsp;&nbsp;&nbsp;
- **磷酸肌酸(CP)**&nbsp;&nbsp;&nbsp;
	- 肌酸 + ATP ->（**肌酸激酶**）-> 磷酸肌酸 + ADP&nbsp;&nbsp;&nbsp;
	- 分布 -> **肌肉、心脏、脑**组织&nbsp;&nbsp;&nbsp;
	- 功能 -> 能量的**贮存形式** -> ATP不足时 -> CP将~P转给ADP -> 快速生成ATP&nbsp;&nbsp;&nbsp;
	- **肌肉细胞**储能主要形式 -> **磷酸肌酸(CP)**&nbsp;&nbsp;&nbsp;
