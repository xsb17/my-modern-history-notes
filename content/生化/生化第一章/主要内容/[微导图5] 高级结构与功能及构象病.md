# 高级结构与功能及构象病 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★空间结构相似 -> 功能相似&nbsp;&nbsp;&nbsp;
  - 肌红蛋白(Mb)与血红蛋白(Hb)：均含**血红素辅基** -> 均可结合O₂&nbsp;&nbsp;&nbsp;
  - Mb：一条多肽链 -> 结合**1分子**O₂ -> 高亲和力 -> **储氧**&nbsp;&nbsp;&nbsp;
  - Hb：四条多肽链(α₂β₂) -> 结合**4分子**O₂ -> **运氧**&nbsp;&nbsp;&nbsp;
- ★协同效应&nbsp;&nbsp;&nbsp;
  - 定义：一个亚基与配体结合 -> 影响**其他亚基**与配体结合能力&nbsp;&nbsp;&nbsp;
  - **正协同效应**（促进）/ **负协同效应**（抑制）&nbsp;&nbsp;&nbsp;
  - Hb一个亚基结合O₂ -> 其他亚基对O₂亲和力**↑** -> S型氧解离曲线&nbsp;&nbsp;&nbsp;
- ★别构效应（变构效应）&nbsp;&nbsp;&nbsp;
  - 别构效应剂 + 蛋白质特定部位 -> **空间构象改变** -> 功能/活性改变&nbsp;&nbsp;&nbsp;
  - Hb别构：T态(紧张态/紧凑/**低氧亲和**) ⇄ R态(松弛态/**高氧亲和**)&nbsp;&nbsp;&nbsp;
  - 氧合 -> 8对离子键断开 -> T→R构象转变&nbsp;&nbsp;&nbsp;
- ★蛋白质构象病（折叠病）&nbsp;&nbsp;&nbsp;
  - 定义：一级结构不变/**空间构象改变** -> 功能异常 -> 疾病&nbsp;&nbsp;&nbsp;
  - 机制：错误折叠 -> 相互**聚集** -> **淀粉样纤维沉淀** -> 毒性 -> 致病&nbsp;&nbsp;&nbsp;
  - ★疯牛病/人克雅氏病：&nbsp;&nbsp;&nbsp;
    - 病原：**PrPsc**（致病型朊病毒蛋白）&nbsp;&nbsp;&nbsp;
    - 正常：**PrPc**（α-螺旋为主/可溶/无致病性）&nbsp;&nbsp;&nbsp;
    - 病变：PrPc -> PrPsc -> **β-折叠↑** -> 水溶性差/热稳定 -> 聚集致病&nbsp;&nbsp;&nbsp;
  - 其他构象病：**老年痴呆症**、**亨廷顿舞蹈病**、人纹状体脊髓变性病&nbsp;&nbsp;&nbsp;
- ★Hb与Mb对比速查&nbsp;&nbsp;&nbsp;
  - Mb：1条链/储氧/高亲和/双曲线&nbsp;&nbsp;&nbsp;
  - Hb：4亚基/运氧/正协同/S型曲线/别构调节(T⇄R)&nbsp;&nbsp;&nbsp;
