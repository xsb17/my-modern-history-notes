# 氨基酸的分类与特性 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 蛋白质元素组成&nbsp;&nbsp;&nbsp;
  - 主要：**C/H/O/N/S** -> 平均**含氮量16%** -> 凯氏定氮法系数 **6.25**&nbsp;&nbsp;&nbsp;
- ★氨基酸基本结构：**H₂N-CHR-COOH** -> 20种 **L-α-氨基酸**（甘氨酸除外）&nbsp;&nbsp;&nbsp;
- ★五大分类（按R侧链）&nbsp;&nbsp;&nbsp;
  - 非极性疏水侧链：**甘(Gly)/丙(Ala)/缬(Val)/亮(Leu)/异亮(Ile)/脯(Pro)/蛋(Met)** -> 侧链为烃链&nbsp;&nbsp;&nbsp;
  - 极性中性侧链：**丝(Ser)/半胱(Cys)/苏(Thr)/天冬酰胺(Asn)/谷氨酰胺(Gln)** -> 极性不带电荷&nbsp;&nbsp;&nbsp;
  - 芳香族氨基酸：**苯丙(Phe)/色(Trp)/酪(Tyr)** -> 含芳香环 -> **280nm紫外吸收**（Trp/Tyr）&nbsp;&nbsp;&nbsp;
  - 酸性氨基酸(R含-COOH)：**天冬(Asp/D)**、**谷(Glu/E)** -> pI≈3&nbsp;&nbsp;&nbsp;
  - 碱性氨基酸(R含-NH₂)：**精(Arg/R)**、**赖(Lys/K)**、**组(His/H)** -> pI≈7.6~10.8&nbsp;&nbsp;&nbsp;
- ★特殊氨基酸必背&nbsp;&nbsp;&nbsp;
  - 亚氨基酸：**脯氨酸(Pro)** -> 可羟化为羟脯氨酸&nbsp;&nbsp;&nbsp;
  - 含硫：**半胱(Cys)** + **蛋/甲硫(Met)** -> Cys可形成**二硫键**&nbsp;&nbsp;&nbsp;
  - 可磷酸化（含-OH）：**丝(Ser)/苏(Thr)/酪(Tyr)**&nbsp;&nbsp;&nbsp;
  - 含芳香环：**色(Trp)/酪(Tyr)/苯丙(Phe)**&nbsp;&nbsp;&nbsp;
- ★等电点(pI)&nbsp;&nbsp;&nbsp;
  - 定义：净电荷为**零**时的pH值&nbsp;&nbsp;&nbsp;
  - 口诀：**pI - pH > 0 -> 阳离子**（带正电）；**pI - pH < 0 -> 阴离子**（带负电）&nbsp;&nbsp;&nbsp;
- ★肽键与肽链&nbsp;&nbsp;&nbsp;
  - 肽键：α-COOH + α-NH₂ -> 脱水缩合 -> **酰胺键**（-CO-NH-）&nbsp;&nbsp;&nbsp;
  - 寡肽：2~10AA/多肽：10~50AA/蛋白质：>50AA&nbsp;&nbsp;&nbsp;
  - 方向性：**N端→C端**（书写N左C右）&nbsp;&nbsp;&nbsp;
- 谷胱甘肽(GSH)&nbsp;&nbsp;&nbsp;
  - 三肽：谷+半胱+甘 -> 含**-SH** -> **抗氧化**+保护RBC膜+螯合解毒&nbsp;&nbsp;&nbsp;
