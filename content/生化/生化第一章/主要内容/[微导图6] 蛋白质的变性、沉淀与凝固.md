# 蛋白质的变性、沉淀与凝固 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★蛋白质变性（必考核心概念）&nbsp;&nbsp;&nbsp;
  - 定义：理化因素 -> **空间构象被破坏** -> 理化性质改变 + **生物活性丧失**&nbsp;&nbsp;&nbsp;
  - 本质：**空间结构**（非共价键+二硫键）被破坏，**一级结构未改变**&nbsp;&nbsp;&nbsp;
  - 变性因素：**加热/有机溶剂/强酸强碱/重金属离子/生物碱试剂**&nbsp;&nbsp;&nbsp;
  - 变性结果：溶解度↓/粘度↑/结晶能力消失/生物活性丧失/**易被蛋白酶水解**&nbsp;&nbsp;&nbsp;
  - 临床：70~75%酒精消毒(变性细菌蛋白)/高温灭菌/低温保存疫苗(防变性)&nbsp;&nbsp;&nbsp;
  - **复性**：变性程度轻 -> 去除变性因素 -> **恢复原有构象和功能**&nbsp;&nbsp;&nbsp;
- ★蛋白质沉淀&nbsp;&nbsp;&nbsp;
  - 定义：蛋白表面**电荷+水化膜被破坏** -> 蛋白分子聚集析出&nbsp;&nbsp;&nbsp;
  - 沉淀因素：&nbsp;&nbsp;&nbsp;
    - 盐析法：高浓度盐(**硫酸铵/NaCl**) -> 破坏水化层+中和电荷&nbsp;&nbsp;&nbsp;
    - 有机溶剂：乙醇/甲醇/丙酮&nbsp;&nbsp;&nbsp;
    - 酸类：苦味酸/钨酸/**三氯乙酸**&nbsp;&nbsp;&nbsp;
    - 重金属：Pb²⁺/Hg²⁺/Ag⁺（pI<pH时蛋白带负电 -> 与重金属结合）&nbsp;&nbsp;&nbsp;
- ★蛋白质凝固&nbsp;&nbsp;&nbsp;
  - 定义：变性蛋白 -> 调pH至pI -> 絮状物 -> **再加热** -> 坚固凝块（不可逆）&nbsp;&nbsp;&nbsp;
  - 凝固 = 变性的进一步**不可逆**发展&nbsp;&nbsp;&nbsp;
- ★变性/沉淀/凝固三者辩证关系（必考选择）&nbsp;&nbsp;&nbsp;
  - 变性 -> **不一定**沉淀，**不一定**凝固&nbsp;&nbsp;&nbsp;
  - 沉淀 -> **不一定**变性（如盐析沉淀蛋白可不变性）&nbsp;&nbsp;&nbsp;
  - 凝固 -> **一定**发生了变性&nbsp;&nbsp;&nbsp;
- 口诀&nbsp;&nbsp;&nbsp;
  - 变性 = 空构毁/一级在/活性丧&nbsp;&nbsp;&nbsp;
  - 沉淀 = 电荷消/水化破/析出来&nbsp;&nbsp;&nbsp;
  - 凝固 = 变性+加热 = 不可逆&nbsp;&nbsp;&nbsp;
