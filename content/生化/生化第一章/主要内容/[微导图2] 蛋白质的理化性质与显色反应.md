# 蛋白质的理化性质与显色反应 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★元素组成与含氮量&nbsp;&nbsp;&nbsp;
  - 主要元素：**C/H/O/N/S** -> 平均**含氮量16%**&nbsp;&nbsp;&nbsp;
  - 凯氏定氮法：蛋白含量 = 含氮量 × **6.25**（100/16）&nbsp;&nbsp;&nbsp;
  - 三聚氰胺含氮66.6% -> 毒奶粉事件造假原理&nbsp;&nbsp;&nbsp;
- ★紫外吸收特性&nbsp;&nbsp;&nbsp;
  - 吸收峰：**280nm** -> 主要来自**色氨酸(Trp)和酪氨酸(Tyr)**&nbsp;&nbsp;&nbsp;
  - 苯丙氨酸最大吸收：260nm&nbsp;&nbsp;&nbsp;
  - 应用：OD280 ∝ 蛋白质浓度 -> **蛋白质定量分析**&nbsp;&nbsp;&nbsp;
- ★显色反应&nbsp;&nbsp;&nbsp;
  - 茚三酮反应：&nbsp;&nbsp;&nbsp;
    - AA+茚三酮加热 -> **蓝紫色** -> 570nm吸收峰&nbsp;&nbsp;&nbsp;
    - 用途：氨基酸**定性+定量**分析&nbsp;&nbsp;&nbsp;
  - 双缩脲反应：&nbsp;&nbsp;&nbsp;
    - 含**两个及以上相邻肽键** + 碱性Cu²⁺ -> **紫红色**络合物&nbsp;&nbsp;&nbsp;
    - 用途：测定**蛋白质含量**（区别于AA -> 需至少两个肽键）&nbsp;&nbsp;&nbsp;
- ★蛋白质两性电离与等电点&nbsp;&nbsp;&nbsp;
  - 两端α-NH₂/α-COOH + R侧链可电离基团 -> **两性电解质**&nbsp;&nbsp;&nbsp;
  - 等电点(pI)：净电荷为**零**时的pH&nbsp;&nbsp;&nbsp;
  - pI > pH -> 带**正电**（阳离子）；pI < pH -> 带**负电**（阴离子）&nbsp;&nbsp;&nbsp;
  - 电泳：pI > pH向**负极**移动；pI < pH向**正极**移动&nbsp;&nbsp;&nbsp;
- ★胶体性质&nbsp;&nbsp;&nbsp;
  - 蛋白分子直径1~100nm -> 属**胶体颗粒**&nbsp;&nbsp;&nbsp;
  - 稳定两因素：**水化膜**（亲水基团吸水）+ **表面电荷**（同种电荷排斥）&nbsp;&nbsp;&nbsp;
  - 破坏两因素 -> 蛋白质**沉淀**&nbsp;&nbsp;&nbsp;
