# 蛋白质的四级空间结构 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★一级结构&nbsp;&nbsp;&nbsp;
  - 定义：**N端→C端**氨基酸残基的**排列顺序**&nbsp;&nbsp;&nbsp;
  - 维系键：**肽键**（-CO-NH-）&nbsp;&nbsp;&nbsp;
  - 单位：肽链&nbsp;&nbsp;&nbsp;
  - 是空间构象和功能的**基础**&nbsp;&nbsp;&nbsp;
- ★二级结构&nbsp;&nbsp;&nbsp;
  - 定义：肽链**主链骨架原子**的相对空间位置（**不涉及R侧链**）&nbsp;&nbsp;&nbsp;
  - 维系键：**氢键**&nbsp;&nbsp;&nbsp;
  - 肽单元：6原子共面（Cα1-CO-NH-Cα2）-> 肽键有**双键性质**/不能自由旋转&nbsp;&nbsp;&nbsp;
  - 四种形式：&nbsp;&nbsp;&nbsp;
    - **α-螺旋**：右手螺旋/**3.6AA/圈**/螺距**0.54nm**/氢键与轴平行/侧链外伸&nbsp;&nbsp;&nbsp;
    - **β-折叠**：肽链充分伸展/锯齿状/氢键与长轴垂直/侧链交替上下&nbsp;&nbsp;&nbsp;
    - **β-转角**：180°回折处/4个AA/第2个常为**脯氨酸**&nbsp;&nbsp;&nbsp;
    - **无规卷曲**：无确定规律的连接结构&nbsp;&nbsp;&nbsp;
  - 超二级结构（模体）：αα/ββ/βαβ -> 锌指结构/钙结合模体&nbsp;&nbsp;&nbsp;
- ★三级结构&nbsp;&nbsp;&nbsp;
  - 定义：整条肽链**所有原子**在三维空间的整体排布&nbsp;&nbsp;&nbsp;
  - 维系键：**疏水键/氢键/离子键/范德华力**（非共价键为主）&nbsp;&nbsp;&nbsp;
  - 球状蛋白特征：疏水内埋/亲水外露/表面空穴=**活性部位**&nbsp;&nbsp;&nbsp;
  - **结构域**：大分子蛋白可分割为数个独立折叠的功能区域&nbsp;&nbsp;&nbsp;
  - **分子伴侣**：帮助蛋白质正确折叠的特殊蛋白（热激蛋白/伴侣蛋白/核素蛋白）&nbsp;&nbsp;&nbsp;
- ★四级结构&nbsp;&nbsp;&nbsp;
  - 定义：蛋白质分子中**各亚基**的空间排布及相互作用&nbsp;&nbsp;&nbsp;
  - 维系键：**氢键/离子键**&nbsp;&nbsp;&nbsp;
  - 亚基：每条多肽链都有完整三级结构&nbsp;&nbsp;&nbsp;
  - 均一/非均一四级结构&nbsp;&nbsp;&nbsp;
  - 例：**血红蛋白**=4个亚基(α₂β₂)&nbsp;&nbsp;&nbsp;
- ★四级结构对比速查&nbsp;&nbsp;&nbsp;
  - 一级：序列/肽键/肽链&nbsp;&nbsp;&nbsp;
  - 二级：局部主链/氢键/α-螺旋-β-折叠-β-转角-无规卷曲&nbsp;&nbsp;&nbsp;
  - 三级：全链所有原子/疏水键-氢键-离子键-范德华力/结构域&nbsp;&nbsp;&nbsp;
  - 四级：亚基排布/氢键-离子键/亚基&nbsp;&nbsp;&nbsp;
