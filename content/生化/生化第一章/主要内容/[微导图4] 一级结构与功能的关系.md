# 一级结构与功能的关系 — 微型速记导图（必考疾病）&nbsp;&nbsp;&nbsp;

- ★一级结构是空间结构和功能的基础&nbsp;&nbsp;&nbsp;
  - 牛核糖核酸酶(Rnase)变性复性实验：&nbsp;&nbsp;&nbsp;
    - 尿素(变性剂) + β-巯基乙醇(打开**二硫键**) -> 空间构象破坏 -> **活性丧失**&nbsp;&nbsp;&nbsp;
    - 去除变性剂 -> **自动复性** -> 活性恢复&nbsp;&nbsp;&nbsp;
    - 证明：只要**一级结构未破坏** -> 空间构象和功能可恢复&nbsp;&nbsp;&nbsp;
- ★一级结构相似 -> 功能相似&nbsp;&nbsp;&nbsp;
  - 不同哺乳动物胰岛素：51AA中有22个完全相同 -> 空间构象/功能相似&nbsp;&nbsp;&nbsp;
  - 牛胰岛素可治疗**人糖尿病**&nbsp;&nbsp;&nbsp;
- ★关键氨基酸突变 -> 功能异常（必考疾病）&nbsp;&nbsp;&nbsp;
  - 镰刀型红细胞贫血症：&nbsp;&nbsp;&nbsp;
    - 基因突变：β珠蛋白基因点突变 -> 第6位密码子 **GAG→GUG**&nbsp;&nbsp;&nbsp;
    - AA改变：β链N端第6位 **谷氨酸(Glu)** -> **缬氨酸(Val)**（β⁶ Glu→Val）&nbsp;&nbsp;&nbsp;
    - 本质：**酸性亲水AA** 变为 **非极性疏水AA**&nbsp;&nbsp;&nbsp;
    - 结果：**HbS**溶解度↓ -> 缺氧时RBC**镰变** -> 溶血/栓塞&nbsp;&nbsp;&nbsp;
  - 胰岛素突变：&nbsp;&nbsp;&nbsp;
    - A链C端天冬酰胺(N)缺失 -> 活性**完全丧失**&nbsp;&nbsp;&nbsp;
    - B链23-30AA缺失 -> 活性下降**85%**&nbsp;&nbsp;&nbsp;
- ★一级结构决定构象的总结&nbsp;&nbsp;&nbsp;
  - 一级结构 = 空间构象的**基础**&nbsp;&nbsp;&nbsp;
  - 关键部位AA改变 -> 空间构象改变 -> **功能异常** -> 疾病&nbsp;&nbsp;&nbsp;
  - 但一级结构并非决定空间构象的**唯一因素**（需分子伴侣协助折叠）&nbsp;&nbsp;&nbsp;
