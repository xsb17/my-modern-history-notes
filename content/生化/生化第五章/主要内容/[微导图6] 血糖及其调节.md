# 血糖及其调节&nbsp;&nbsp;&nbsp;

- **血糖基本概念**&nbsp;&nbsp;&nbsp;
	- **血糖** -> 血液中的**==葡萄糖==**&nbsp;&nbsp;&nbsp;
	- 正常值 -> ==**3.89～6.11 mmol/L**==&nbsp;&nbsp;&nbsp;
	- **肾糖阈** -> **==8.89-10.00 mmol/L==** -> 超过此值 -> 出现**==尿糖==**&nbsp;&nbsp;&nbsp;
- **血糖恒定的生理意义**&nbsp;&nbsp;&nbsp;
	- 保证**脑**组织能量供应 -> 脑不能利用脂肪酸 -> 主要依赖**==葡萄糖==**&nbsp;&nbsp;&nbsp;
	- 保证**红细胞**供能 -> 无==**线粒体**== -> 完全靠**==糖酵解==**&nbsp;&nbsp;&nbsp;
	- 保证骨髓/神经组织供能 -> 经常利用葡萄糖&nbsp;&nbsp;&nbsp;
- **血糖来源（4条）**&nbsp;&nbsp;&nbsp;
	- **==食物糖==**（主要）-> 消化吸收&nbsp;&nbsp;&nbsp;
	- **==肝糖原分解==** -> 补充血糖&nbsp;&nbsp;&nbsp;
	- **==糖异生==** -> 非糖物质 -> 葡萄糖&nbsp;&nbsp;&nbsp;
	- **==其他单糖==**转化&nbsp;&nbsp;&nbsp;
- **血糖去路（5条）**&nbsp;&nbsp;&nbsp;
	- **==氧化分解==**（主要）-> CO₂ + H₂O -> 供能&nbsp;&nbsp;&nbsp;
	- **==糖原合成==** -> 肝糖原/肌糖原&nbsp;&nbsp;&nbsp;
	- **==磷酸戊糖途径==** -> NADPH + 核糖-5-磷酸&nbsp;&nbsp;&nbsp;
	- 转变成**==其他糖类==**物质&nbsp;&nbsp;&nbsp;
	- 转变成**==脂肪/氨基酸==**等&nbsp;&nbsp;&nbsp;
	- 超肾糖阈 -> **==尿糖==**排出&nbsp;&nbsp;&nbsp;
- **血糖调节三级水平**&nbsp;&nbsp;&nbsp;
	- 神经系统 -> **激素**（主要）-> 组织器官（关键酶）&nbsp;&nbsp;&nbsp;
	- **肝脏** -> 调节血糖浓度**==最重要的器官==**&nbsp;&nbsp;&nbsp;
- **降低血糖的激素 — 胰岛素（唯一）**&nbsp;&nbsp;&nbsp;
	- ==**胰岛素**== -> 机体**唯一降血糖**激素&nbsp;&nbsp;&nbsp;
	- 作用机制&nbsp;&nbsp;&nbsp;
		- 促进葡萄糖**转运入细胞**（GLUT）&nbsp;&nbsp;&nbsp;
		- 促进**糖原合成** -> 激活糖原合酶&nbsp;&nbsp;&nbsp;
		- 促进**糖氧化分解** -> 激活PFK-1等&nbsp;&nbsp;&nbsp;
		- 促进**脂肪合成**&nbsp;&nbsp;&nbsp;
		- **抑制糖异生**&nbsp;&nbsp;&nbsp;
		- 促进氨基酸进入肌组织合成蛋白质 -> 减少糖异生原料&nbsp;&nbsp;&nbsp;
- **升高血糖的激素（4种）**&nbsp;&nbsp;&nbsp;
	- **==胰高血糖素==** -> 促进肝糖原分解 + 促进糖异生&nbsp;&nbsp;&nbsp;
	- **==肾上腺素==** -> 促进肌糖原分解 + 促进糖异生&nbsp;&nbsp;&nbsp;
	- **==糖皮质激素==** -> 促进糖异生 + 抑制糖利用&nbsp;&nbsp;&nbsp;
	- **==生长激素==** -> 抑制糖利用&nbsp;&nbsp;&nbsp;
- **糖耐量与OGTT**&nbsp;&nbsp;&nbsp;
	- **葡萄糖耐量** -> 人体对摄入葡萄糖的耐受能力&nbsp;&nbsp;&nbsp;
	- 正常 -> 一次性大量摄入葡萄糖 -> 血糖不会持续大幅升高&nbsp;&nbsp;&nbsp;
	- **OGTT(口服葡萄糖耐量实验)**&nbsp;&nbsp;&nbsp;
		- 空腹取血 -> 口服75g葡萄糖 -> 30min/1h/2h/3h取血测血糖&nbsp;&nbsp;&nbsp;
		- 用于 -> 血糖在6-7mmol/L之间怀疑糖尿病时 -> 帮助明确诊断&nbsp;&nbsp;&nbsp;
- **血糖水平异常**&nbsp;&nbsp;&nbsp;
	- **高血糖(hyperglycemia)**&nbsp;&nbsp;&nbsp;
		- 诊断标准 -> 空腹血糖 > **==7.1 mmol/L==**&nbsp;&nbsp;&nbsp;
		- 超肾糖阈 -> **==糖尿==**&nbsp;&nbsp;&nbsp;
		- 持续性高血糖+糖尿 -> 主要见于**==糖尿病(diabetes mellitus)==**&nbsp;&nbsp;&nbsp;
	- **低血糖(hypoglycemia)**&nbsp;&nbsp;&nbsp;
		- 诊断标准 -> 血糖 < **==2.8 mmol/L==**&nbsp;&nbsp;&nbsp;
		- 症状 -> 头晕、倦怠无力、心悸 -> 严重时**==昏迷==**(低血糖休克)&nbsp;&nbsp;&nbsp;
		- 原因 -> 脑细胞能量主要来自葡萄糖氧化 -> 低血糖影响脑功能&nbsp;&nbsp;&nbsp;
		- 急救 -> 静脉补充**葡萄糖**&nbsp;&nbsp;&nbsp;
- **饱食/饥饿状态代谢趋势（论述题核心）**&nbsp;&nbsp;&nbsp;
	- **饱食状态**&nbsp;&nbsp;&nbsp;
		- 主要供能物质 -> **==葡萄糖==**&nbsp;&nbsp;&nbsp;
		- 糖代谢 -> 氧化分解供能 -> 多余合成**==糖原/脂肪==**&nbsp;&nbsp;&nbsp;
		- 脂肪代谢 -> 外源脂肪 -> 储存&nbsp;&nbsp;&nbsp;
		- 激素 -> ==**胰岛素**==分泌增加 -> 胰高血糖素抑制&nbsp;&nbsp;&nbsp;
	- **短期饥饿**&nbsp;&nbsp;&nbsp;
		- 肝糖原基本耗尽 -> 血糖降低 -> **==胰高血糖素==**分泌增加&nbsp;&nbsp;&nbsp;
		- 糖代谢 -> **==肝糖异生增强==** -> 维持血糖&nbsp;&nbsp;&nbsp;
		- 脂肪代谢 -> **==脂肪动员增强==** -> 肝酮体合成增强&nbsp;&nbsp;&nbsp;
		- 蛋白质代谢 -> **==蛋白质分解增强==** -> 提供生糖氨基酸&nbsp;&nbsp;&nbsp;
