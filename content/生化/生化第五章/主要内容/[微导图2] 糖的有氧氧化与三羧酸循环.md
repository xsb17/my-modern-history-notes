# 糖的有氧氧化与三羧酸循环&nbsp;&nbsp;&nbsp;

- **有氧氧化概念**&nbsp;&nbsp;&nbsp;
	- 葡萄糖在有氧条件下 -> 彻底氧化 -> **==CO₂ + H₂O==** -> 释放大量能量&nbsp;&nbsp;&nbsp;
	- 是糖氧化的**==主要方式==**&nbsp;&nbsp;&nbsp;
	- 反应部位 -> **==胞质溶胶 + 线粒体==**&nbsp;&nbsp;&nbsp;
- **四个阶段**&nbsp;&nbsp;&nbsp;
	- 第一阶段（胞质）-> 葡萄糖 -> 2×丙酮酸 -> 净生成 **==2 ATP + 2 NADH==**&nbsp;&nbsp;&nbsp;
	- 第二阶段（线粒体）-> 2×丙酮酸 -> 2×**==乙酰CoA==** + 2×CO₂ -> 2 NADH&nbsp;&nbsp;&nbsp;
	- 第三阶段（线粒体）-> 乙酰CoA -> **TCA循环** -> CO₂ + NADH + FADH₂&nbsp;&nbsp;&nbsp;
	- 第四阶段（线粒体）-> NADH+FADH₂ -> **氧化磷酸化** -> H₂O + ATP&nbsp;&nbsp;&nbsp;
- **第二阶段：丙酮酸氧化脱羧**&nbsp;&nbsp;&nbsp;
	- 酶 -> ==**丙酮酸脱氢酶复合体**== -> E1+E2+E3&nbsp;&nbsp;&nbsp;
	- E1(丙酮酸脱氢酶) -> 辅因子 **==TPP==** -> 含**==维生素B₁==**&nbsp;&nbsp;&nbsp;
	- E2(二氢硫辛酰胺转乙酰酶) -> 辅因子**==硫辛酸+CoA==** -> 含硫辛酸+**==泛酸==**&nbsp;&nbsp;&nbsp;
	- E3(二氢硫辛酰胺脱氢酶) -> 辅因子 **==FAD+NAD⁺==** -> 含**==维生素B₂+维生素PP==**&nbsp;&nbsp;&nbsp;
	- 关键维生素 -> **==B₁、B₂、PP、泛酸、硫辛酸==**（无B₆/B₁₂）&nbsp;&nbsp;&nbsp;
- **第三阶段：TCA循环（三羧酸循环/Krebs循环）**&nbsp;&nbsp;&nbsp;
	- 反应部位 -> **==线粒体基质==**&nbsp;&nbsp;&nbsp;
	- 1937 Krebs提出 -> 1953诺贝尔奖&nbsp;&nbsp;&nbsp;
	- 8步反应 -> 3个不可逆反应&nbsp;&nbsp;&nbsp;
	- **经过一次TCA循环**&nbsp;&nbsp;&nbsp;
		- 消耗 -> **==1分子乙酰CoA==**&nbsp;&nbsp;&nbsp;
		- **==4次脱氢==** -> 3 NADH + 1 FADH₂&nbsp;&nbsp;&nbsp;
		- **==2次脱羧==** -> 2 CO₂&nbsp;&nbsp;&nbsp;
		- **==1次底物水平磷酸化==** -> 1 GTP&nbsp;&nbsp;&nbsp;
		- 产量合计 -> **==3 NADH + 1 FADH₂ + 1 GTP + 2 CO₂==**&nbsp;&nbsp;&nbsp;
	- **三大关键酶**&nbsp;&nbsp;&nbsp;
		- ==**柠檬酸合酶**== -> 乙酰CoA+草酰乙酸 -> 柠檬酸 -> 不可逆&nbsp;&nbsp;&nbsp;
		- ==**异柠檬酸脱氢酶**== -> 异柠檬酸 -> α-酮戊二酸 -> 不可逆 -> 辅因子NAD(P)⁺&nbsp;&nbsp;&nbsp;
		- ==**α-酮戊二酸脱氢酶复合体**== -> α-酮戊二酸 -> 琥珀酰CoA -> 不可逆 -> 类似丙酮酸脱氢酶复合体&nbsp;&nbsp;&nbsp;
	- TCA关键酶抑制剂 -> **==ATP、NADH、柠檬酸、琥珀酰CoA==**&nbsp;&nbsp;&nbsp;
	- **==草酰乙酸==** -> 循环速度的关键 -> 需不断补充&nbsp;&nbsp;&nbsp;
	- 整个循环为**==不可逆循环==**&nbsp;&nbsp;&nbsp;
- **TCA循环三大作用**&nbsp;&nbsp;&nbsp;
	- 彻底分解乙酰CoA&nbsp;&nbsp;&nbsp;
	- **==三大营养物质（糖、脂肪、蛋白质）分解供能的共同通路==** -> 三者代谢联系的**==枢纽==**&nbsp;&nbsp;&nbsp;
	- 为生物合成提供**==前体物质==** -> 琥珀酰CoA+甘氨酸 -> 血红素&nbsp;&nbsp;&nbsp;
- **有氧氧化ATP总产量（1分子葡萄糖）**&nbsp;&nbsp;&nbsp;
	- 经历 -> **==6次脱羧，12次脱氢==**&nbsp;&nbsp;&nbsp;
	- 第一阶段（胞浆）&nbsp;&nbsp;&nbsp;
		- 投资：-2 ATP&nbsp;&nbsp;&nbsp;
		- 底物磷酸化：+4 ATP（2次）&nbsp;&nbsp;&nbsp;
		- NADH(胞浆)：**3或5 ATP**（取决于穿梭方式）&nbsp;&nbsp;&nbsp;
	- 第二阶段：2×丙酮酸 -> 2×乙酰CoA -> 2 NADH -> **5 ATP**&nbsp;&nbsp;&nbsp;
	- 第三阶段(TCA)&nbsp;&nbsp;&nbsp;
		- 底物磷酸化：+2 GTP&nbsp;&nbsp;&nbsp;
		- NADH -> 2×3次 -> 6 NADH -> **15 ATP**&nbsp;&nbsp;&nbsp;
		- FADH₂ -> 2×1次 -> 2 FADH₂ -> ==**3 ATP**==&nbsp;&nbsp;&nbsp;
	- **总计** -> **==30 或 32 ATP==**（取决于胞浆NADH穿梭方式）&nbsp;&nbsp;&nbsp;
- **巴斯德效应(Pasteur effect)**&nbsp;&nbsp;&nbsp;
	- 概念 -> 有氧氧化**==抑制==**无氧氧化&nbsp;&nbsp;&nbsp;
	- 有氧时 -> NADH进入线粒体 -> 丙酮酸不生成乳酸&nbsp;&nbsp;&nbsp;
	- 缺氧时 -> NADH堆积 -> 丙酮酸 -> 乳酸&nbsp;&nbsp;&nbsp;
- **有氧氧化关键酶总结（7个）**&nbsp;&nbsp;&nbsp;
	- 糖酵解3个 -> **==HK、PFK-1、PK==**&nbsp;&nbsp;&nbsp;
	- 丙酮酸脱氢酶复合体 -> 1个（含E1/E2/E3）&nbsp;&nbsp;&nbsp;
	- TCA循环3个 -> **==柠檬酸合酶、异柠檬酸脱氢酶、α-酮戊二酸脱氢酶复合体==**&nbsp;&nbsp;&nbsp;
