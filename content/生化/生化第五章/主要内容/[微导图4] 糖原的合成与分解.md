# 糖原的合成与分解&nbsp;&nbsp;&nbsp;

- **糖原概述**&nbsp;&nbsp;&nbsp;
	- **糖原(glycogen)** -> 动物体内糖的**==储存形式==** -> 能迅速动用的能源储备&nbsp;&nbsp;&nbsp;
	- **肝糖原** -> 70-100g -> 维持**==血糖水平==**&nbsp;&nbsp;&nbsp;
	- **肌糖原** -> 180-300g -> 供**==肌肉收缩==**&nbsp;&nbsp;&nbsp;
- **糖原结构特点**&nbsp;&nbsp;&nbsp;
	- 直链 -> **==α-1,4-糖苷键==**连接&nbsp;&nbsp;&nbsp;
	- 分支 -> 约10个葡萄糖单元处 -> **==α-1,6-糖苷键==**连接&nbsp;&nbsp;&nbsp;
	- 分支增加 -> 溶解度增加&nbsp;&nbsp;&nbsp;
	- 非还原端增多 -> 有利于酶分解 -> 每条链终止于**非还原端**&nbsp;&nbsp;&nbsp;
- **糖原合成(glycogenesis)**&nbsp;&nbsp;&nbsp;
	- 部位 -> **肝和肌肉** -> 胞质溶胶&nbsp;&nbsp;&nbsp;
	- 三阶段&nbsp;&nbsp;&nbsp;
		- **第一阶段：形成UDPG（活性葡萄糖）**&nbsp;&nbsp;&nbsp;
			- 葡萄糖 ->(HK)-> G-6-P ->(变位酶)-> G-1-P ->(UDPG焦磷酸化酶)-> **==UDPG==**&nbsp;&nbsp;&nbsp;
		- **第二阶段：糖链延长**&nbsp;&nbsp;&nbsp;
			- UDPG + 糖原n ->(==**糖原合酶**==)-> 糖原n+1 + UDP&nbsp;&nbsp;&nbsp;
			- 以**==α-1,4-糖苷键==**延长&nbsp;&nbsp;&nbsp;
			- 需要**==糖原引物==**（小分子糖原/糖原蛋白）&nbsp;&nbsp;&nbsp;
		- **第三阶段：分支形成**&nbsp;&nbsp;&nbsp;
			- **==分支酶==**催化 -> **==α-1,6-糖苷键==**连接&nbsp;&nbsp;&nbsp;
	- 关键酶 -> ==**糖原合酶**== -> 催化α-1,4-糖苷键&nbsp;&nbsp;&nbsp;
	- 合成方向 -> 在引物的**==非还原端==**添加葡萄糖基&nbsp;&nbsp;&nbsp;
	- 合成耗能 -> 每增加1个葡萄糖基 -> 消耗 ==**2 ATP**==&nbsp;&nbsp;&nbsp;
- **糖原分解(glycogenolysis)**&nbsp;&nbsp;&nbsp;
	- 习惯指 -> 肝糖原分解为**葡萄糖** -> 补充血糖&nbsp;&nbsp;&nbsp;
	- 部位 -> ==**胞质溶胶**==&nbsp;&nbsp;&nbsp;
	- 关键酶&nbsp;&nbsp;&nbsp;
		- ==**糖原磷酸化酶**== -> 裂解α-1,4-糖苷键&nbsp;&nbsp;&nbsp;
		- **糖原脱支酶** -> 处理分支点α-1,6-糖苷键&nbsp;&nbsp;&nbsp;
		- 磷酸葡萄糖变位酶 -> G-1-P -> G-6-P&nbsp;&nbsp;&nbsp;
		- ==**葡糖-6-磷酸酶**== -> G-6-P -> **葡萄糖** -> 仅存在于**==肝、肾==**&nbsp;&nbsp;&nbsp;
- **肝糖原 vs 肌糖原（必考）**&nbsp;&nbsp;&nbsp;
	- **肝糖原** -> 有==**葡糖-6-磷酸酶**== -> 能分解为**葡萄糖** -> **==补充血糖==**&nbsp;&nbsp;&nbsp;
	- **肌糖原** -> **==缺乏葡糖-6-磷酸酶==** -> **==不能分解为葡萄糖==** -> 只能**==糖酵解/有氧氧化==** -> 供肌肉自身收缩&nbsp;&nbsp;&nbsp;
	- 肌糖原 -> 分解为G-6-P -> 进入糖酵解 -> 乳酸 -> 经**乳酸循环(Cori循环)**到肝 -> 糖异生为葡萄糖&nbsp;&nbsp;&nbsp;
- **G-6-P的代谢去路**&nbsp;&nbsp;&nbsp;
	- 肝脏 -> G-6-P -> 葡萄糖（补充血糖）-> 关键酶：==**葡糖-6-磷酸酶**==&nbsp;&nbsp;&nbsp;
	- 所有组织 -> G-6-P -> 糖酵解/有氧氧化&nbsp;&nbsp;&nbsp;
	- 所有组织 -> G-6-P -> **磷酸戊糖途径**&nbsp;&nbsp;&nbsp;
- **糖原合成与分解的调节**&nbsp;&nbsp;&nbsp;
	- 两个关键酶 -> ==**糖原合酶**==(合成) vs ==**糖原磷酸化酶**==(分解)&nbsp;&nbsp;&nbsp;
	- 共价修饰（主要方式）-> **==磷酸化与去磷酸化==**&nbsp;&nbsp;&nbsp;
		- **磷酸化** -> 糖原合酶**==失活==** / 糖原磷酸化酶**==激活==** -> 促进分解&nbsp;&nbsp;&nbsp;
		- **去磷酸化** -> 糖原合酶**==激活==** / 糖原磷酸化酶**==失活==** -> 促进合成&nbsp;&nbsp;&nbsp;
	- 别构调节&nbsp;&nbsp;&nbsp;
		- 糖原合酶 -> 激活：**==ATP/G-6-P==** / 抑制：**==AMP==**&nbsp;&nbsp;&nbsp;
		- 糖原磷酸化酶 -> 激活：**==AMP/ADP==** / 抑制：**==ATP/G-6-P==**&nbsp;&nbsp;&nbsp;
	- 激素&nbsp;&nbsp;&nbsp;
		- **胰高血糖素** -> 促进**==肝糖原==**分解（升血糖）&nbsp;&nbsp;&nbsp;
		- **肾上腺素** -> 促进**==肌糖原==**分解（供能应激）&nbsp;&nbsp;&nbsp;
