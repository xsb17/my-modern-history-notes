# [全集存档] 生化第五章：糖代谢&nbsp;&nbsp;&nbsp;

## 第一节 概述&nbsp;&nbsp;&nbsp;

### 一、糖分子简介&nbsp;&nbsp;&nbsp;

- **糖(carbohydrates)**：即碳水化合物，化学本质为多羟醛或多羟酮类及其衍生物或多聚物&nbsp;&nbsp;&nbsp;
- 糖主要分为四大类（按水解产物）：&nbsp;&nbsp;&nbsp;
  - **单糖**：葡萄糖、果糖、半乳糖、核糖&nbsp;&nbsp;&nbsp;
  - **寡糖**：麦芽糖（两个葡萄糖）、蔗糖（葡萄糖1+果糖1）、乳糖（葡萄糖1+半乳糖1）&nbsp;&nbsp;&nbsp;
  - **多糖**：淀粉、糖原、纤维素&nbsp;&nbsp;&nbsp;
  - **结合糖**：糖脂、糖蛋白、蛋白聚糖&nbsp;&nbsp;&nbsp;

#### 单糖&nbsp;&nbsp;&nbsp;

- 葡萄糖(glucose)：已醛糖&nbsp;&nbsp;&nbsp;
- 果糖(fructose)：已酮糖&nbsp;&nbsp;&nbsp;
- 半乳糖(galactose)：已醛糖&nbsp;&nbsp;&nbsp;
- 核糖(ribose)：戊醛糖&nbsp;&nbsp;&nbsp;

### 二、糖的生理功能&nbsp;&nbsp;&nbsp;

- **体内供能的主要物质**（无氧和有氧代谢）&nbsp;&nbsp;&nbsp;
- **合成其它物质的原料**：&nbsp;&nbsp;&nbsp;
  - 转变成脂类和非必需氨基酸的骨架等&nbsp;&nbsp;&nbsp;
  - 核糖与脱氧核糖是核酸的组成成分&nbsp;&nbsp;&nbsp;
- **其他特定的生理功能**：&nbsp;&nbsp;&nbsp;
  - 糖蛋白、蛋白聚糖、糖脂等参与组成生物膜、结缔组织的组成，执行特定的生理功能&nbsp;&nbsp;&nbsp;

### 三、糖的消化与吸收&nbsp;&nbsp;&nbsp;

#### 消化&nbsp;&nbsp;&nbsp;

- 食物中的糖：植物淀粉（主要）、纤维素、动物糖原、麦芽糖、蔗糖、乳糖、葡萄糖等&nbsp;&nbsp;&nbsp;
- 消化部位：小肠（主要）、口腔&nbsp;&nbsp;&nbsp;
- 消化酶：α-淀粉酶（来源于唾液和胰腺）、α-葡萄糖苷酶（蔗糖酶、乳糖酶）&nbsp;&nbsp;&nbsp;
- α-葡萄糖苷酶抑制剂：米格列醇和阿卡波糖&nbsp;&nbsp;&nbsp;

#### 吸收&nbsp;&nbsp;&nbsp;

- 吸收部位：**==小肠粘膜细胞==**&nbsp;&nbsp;&nbsp;
- 吸收机制：**==主动吸收==**（耗能，Na⁺依赖性葡萄糖转运体，**==SGLT==**）&nbsp;&nbsp;&nbsp;
- 吸收形式：**==单糖（主要是葡萄糖）==**，经过门静脉系统通过肝脏进入血液循环&nbsp;&nbsp;&nbsp;
- 葡萄糖进入细胞依赖于葡萄糖转运体(GLUT)&nbsp;&nbsp;&nbsp;

### 四、糖代谢概况&nbsp;&nbsp;&nbsp;

- 葡萄糖进入哪条代谢途径取决于：机体状态、组织细胞代谢特点&nbsp;&nbsp;&nbsp;

## 第二节 糖的无氧氧化（糖酵解/Glycolysis）&nbsp;&nbsp;&nbsp;

### 一、糖酵解的概念&nbsp;&nbsp;&nbsp;

- **糖的无氧氧化(glycolysis)**：在缺氧或氧利用障碍情况下，葡萄糖或糖原经一系列酶促反应分解生成丙酮酸进而还原生成乳酸，并产生少量能量的过程&nbsp;&nbsp;&nbsp;
- 从葡萄糖到丙酮酸的过程称为**糖酵解**&nbsp;&nbsp;&nbsp;
- **反应部位**：==**胞质溶胶**==&nbsp;&nbsp;&nbsp;

### 二、糖酵解的反应过程（三个阶段）&nbsp;&nbsp;&nbsp;

#### 第一阶段：葡萄糖的裂解（耗能阶段，5步）&nbsp;&nbsp;&nbsp;

- 葡萄糖 → 2分子磷酸丙糖&nbsp;&nbsp;&nbsp;
- 消耗 2 ATP&nbsp;&nbsp;&nbsp;

**第1步：葡萄糖 → 葡糖-6-磷酸**
- 不可逆反应，消耗 ATP&nbsp;&nbsp;&nbsp;
- 酶：**==己糖激酶(hexokinase, HK)==**&nbsp;&nbsp;&nbsp;
- 哺乳动物己糖激酶有 4 种同工酶(I-IV)，肝细胞内为**==己糖激酶 IV（葡萄糖激酶）==**，对葡萄糖亲和力低（Km=10mmol/L），具有维持血糖浓度平衡的重要生理作用&nbsp;&nbsp;&nbsp;
- 己糖激酶是**变构酶**：Mg²⁺是必需激活剂；**==葡糖-6-磷酸、长链脂肪酸==**是变构抑制剂&nbsp;&nbsp;&nbsp;

**第2步：葡糖-6-磷酸 → 果糖-6-磷酸**
- 可逆反应&nbsp;&nbsp;&nbsp;

**第3步：果糖-6-磷酸 → 果糖-1,6-双磷酸**
- 不可逆反应，消耗 ATP&nbsp;&nbsp;&nbsp;
- 酶：**==磷酸果糖激酶-1(PFK-1)==**&nbsp;&nbsp;&nbsp;
- 抑制剂：**==ATP、柠檬酸和长链脂肪酸==**&nbsp;&nbsp;&nbsp;
- 激活剂：**==AMP、ADP、果糖-6-磷酸、果糖-1,6-双磷酸和果糖-2,6-双磷酸==**&nbsp;&nbsp;&nbsp;

**第4步：果糖-1,6-双磷酸 → 磷酸二羟丙酮 + 3-磷酸甘油醛**
- 酶：醛缩酶&nbsp;&nbsp;&nbsp;
- 磷酸己糖裂解为两分子磷酸丙糖&nbsp;&nbsp;&nbsp;

**第5步：磷酸二羟丙酮 → 3-磷酸甘油醛**
- 酶：磷酸丙糖异构酶&nbsp;&nbsp;&nbsp;

**耗能阶段小结**：葡萄糖 + 2 ATP → 2×甘油醛-3-磷酸 + 2 ADP；两个不可逆反应、两个关键酶（HK、PFK-1）

#### 第二阶段：醛氧化为酸（产能阶段，5步）&nbsp;&nbsp;&nbsp;

- 2×甘油醛-3-磷酸 → 2×丙酮酸&nbsp;&nbsp;&nbsp;

**第6步：甘油醛-3-磷酸 → 1,3-二磷酸甘油酸**
- 1,3-二磷酸甘油酸是高能磷酸化合物&nbsp;&nbsp;&nbsp;
- NADH（胞质溶胶）可经穿梭机制进入呼吸链生成 ATP&nbsp;&nbsp;&nbsp;

**第7步：1,3-二磷酸甘油酸 → 3-磷酸甘油酸**
- 糖酵解过程中**第一个产生 ATP**的反应&nbsp;&nbsp;&nbsp;
- **==底物水平磷酸化==**&nbsp;&nbsp;&nbsp;

**第8步：3-磷酸甘油酸 → 2-磷酸甘油酸**
- 酶：磷酸甘油酸变位酶&nbsp;&nbsp;&nbsp;

**第9步：2-磷酸甘油酸 → 磷酸烯醇式丙酮酸(PEP)**
- PEP 是高能磷酸化合物&nbsp;&nbsp;&nbsp;

**第10步：PEP → 丙酮酸**
- 不可逆反应，生成 ATP（底物水平磷酸化）&nbsp;&nbsp;&nbsp;
- 酶：**==丙酮酸激酶(pyruvate kinase, PK)==**&nbsp;&nbsp;&nbsp;

**产能阶段小结**：3-磷酸甘油醛 + Pi + 2 ADP + NAD⁺ → 丙酮酸 + 2 ATP + NADH + H⁺

#### 第三阶段：丙酮酸 → 乳酸&nbsp;&nbsp;&nbsp;

- 反应中的 NADH 和 H⁺ 来自第6步甘油醛-3-磷酸脱氢反应&nbsp;&nbsp;&nbsp;

### 糖无氧氧化小结&nbsp;&nbsp;&nbsp;

- 反应部位：==**胞质溶胶**==&nbsp;&nbsp;&nbsp;
- 不需氧的产能过程&nbsp;&nbsp;&nbsp;
- **三步不可逆反应，三个关键酶**：&nbsp;&nbsp;&nbsp;
  - **==己糖激酶(HK)==**&nbsp;&nbsp;&nbsp;
  - **==磷酸果糖激酶-1(PFK-1)==** → 最重要限速酶&nbsp;&nbsp;&nbsp;
  - **==丙酮酸激酶(PK)==**&nbsp;&nbsp;&nbsp;
- 产能方式和数量：&nbsp;&nbsp;&nbsp;
  - 方式：==**底物水平磷酸化**==&nbsp;&nbsp;&nbsp;
  - 净生成 ATP：从葡萄糖开始→==**2 ATP**==；从糖原开始→==**3 ATP**==&nbsp;&nbsp;&nbsp;
- 脱氢：胞质溶胶中脱氢产生 NADH，用于丙酮酸还原为乳酸（无净生成的 NADH）&nbsp;&nbsp;&nbsp;
- 终产物乳酸的去路：释放入血→分解利用 / 乳酸循环（糖异生）&nbsp;&nbsp;&nbsp;

### 三、糖无氧氧化的调节&nbsp;&nbsp;&nbsp;

- 主要针对三个关键酶进行调节，其中 **==PFK-1 活性最低，是最重要的限速酶==**&nbsp;&nbsp;&nbsp;

#### 关键酶调节一览&nbsp;&nbsp;&nbsp;

- ==**己糖激酶**==：&nbsp;&nbsp;&nbsp;
  - 变构抑制剂：**==葡糖-6-磷酸、长链脂肪酸==**&nbsp;&nbsp;&nbsp;
- ==**磷酸果糖激酶-1**==：&nbsp;&nbsp;&nbsp;
  - 变构激活剂：**==果糖-6-磷酸、果糖-1,6-双磷酸、果糖-2,6-双磷酸、ADP、AMP==**&nbsp;&nbsp;&nbsp;
  - 变构抑制剂：**==柠檬酸、ATP(高浓度)、长链脂肪酸==**&nbsp;&nbsp;&nbsp;
  - 共价修饰：PFK-2 磷酸化失活&nbsp;&nbsp;&nbsp;
- ==**丙酮酸激酶**==：&nbsp;&nbsp;&nbsp;
  - 变构激活剂：**==果糖-1,6-双磷酸、PEP==**&nbsp;&nbsp;&nbsp;
  - 变构抑制剂：**==ATP、柠檬酸、乙酰CoA、丙氨酸==**&nbsp;&nbsp;&nbsp;
  - 共价修饰：磷酸化失活&nbsp;&nbsp;&nbsp;

### 四、无氧氧化的生理意义&nbsp;&nbsp;&nbsp;

- **快速产生 ATP 且不耗氧**&nbsp;&nbsp;&nbsp;
- 机体在缺氧情况下获得能量的一种快捷有效方式&nbsp;&nbsp;&nbsp;
- 某些组织细胞在有氧情况下的重要供能途径：&nbsp;&nbsp;&nbsp;
  - **成熟红细胞**（==**唯一供能途径**==）&nbsp;&nbsp;&nbsp;
  - 白细胞、骨髓细胞、神经细胞等&nbsp;&nbsp;&nbsp;
  - **肿瘤细胞**（==**Warburg 效应**==）→ PET-CT 癌症诊断的辅助手段&nbsp;&nbsp;&nbsp;

### 丙酮酸的去路&nbsp;&nbsp;&nbsp;

- 糖酵解是葡萄糖进行无氧分解和有氧氧化都要经历的首要环节&nbsp;&nbsp;&nbsp;
- 缺氧：丙酮酸 → **==乳酸==**（糖的无氧氧化）&nbsp;&nbsp;&nbsp;
- 有氧：丙酮酸 → **==乙酰CoA==** → TCA 循环 → 呼吸链 → CO₂ + H₂O（糖的有氧氧化）&nbsp;&nbsp;&nbsp;

## 第三节 糖的有氧氧化&nbsp;&nbsp;&nbsp;

### 一、糖有氧氧化的概念&nbsp;&nbsp;&nbsp;

- **糖的有氧氧化(aerobic oxidation)**：葡萄糖在有氧条件下彻底氧化生成 H₂O 和 CO₂，并释放大量能量的反应过程，是**==糖氧化的主要方式==**&nbsp;&nbsp;&nbsp;
- **反应部位**：==**胞质溶胶 + 线粒体**==&nbsp;&nbsp;&nbsp;

### 二、糖有氧氧化的反应过程（四个阶段）&nbsp;&nbsp;&nbsp;

- 第一阶段（糖酵解，胞质溶胶）：葡萄糖 → 2×丙酮酸&nbsp;&nbsp;&nbsp;
- 第二阶段（线粒体）：2×丙酮酸 → 2×乙酰CoA + 2×CO₂&nbsp;&nbsp;&nbsp;
- 第三阶段（TCA循环，线粒体）：乙酰CoA → CO₂ + NADH+H⁺ + FADH₂&nbsp;&nbsp;&nbsp;
- 第四阶段（氧化磷酸化，线粒体）：NADH+H⁺ + FADH₂ + O₂ → H₂O + ATP&nbsp;&nbsp;&nbsp;

#### 第一阶段：糖酵解（同前10步）&nbsp;&nbsp;&nbsp;

- 1×葡萄糖 → 2×丙酮酸，净生成 2×ATP，2×NADH+H⁺（胞质溶胶中）&nbsp;&nbsp;&nbsp;

#### 第二阶段：丙酮酸氧化脱羧生成乙酰CoA&nbsp;&nbsp;&nbsp;

- 反应部位：==**线粒体**==&nbsp;&nbsp;&nbsp;
- 酶：==**丙酮酸脱氢酶复合体**==（由 E1、E2、E3 组成）&nbsp;&nbsp;&nbsp;
- E1（丙酮酸脱氢酶）：辅因子 **==TPP==** → 含**==维生素 B₁==**&nbsp;&nbsp;&nbsp;
- E2（二氢硫辛酰胺转乙酰酶）：辅因子二氢硫辛酸、CoA → 含**==硫辛酸、泛酸==**&nbsp;&nbsp;&nbsp;
- E3（二氢硫辛酰胺脱氢酶）：辅因子 FAD、NAD⁺ → 含**==维生素 B₂、维生素 PP==**&nbsp;&nbsp;&nbsp;

#### 第三阶段：三羧酸循环(TCA cycle/Krebs 循环)&nbsp;&nbsp;&nbsp;

- 指乙酰CoA和草酰乙酸缩合生成含三个羧基的柠檬酸，反复进行脱氢脱羧，又生成草酰乙酸，再重复循环反应的过程&nbsp;&nbsp;&nbsp;
- 反应部位：==**线粒体**==&nbsp;&nbsp;&nbsp;
- 1937年 Hans Krebs 提出，1953年获诺贝尔奖&nbsp;&nbsp;&nbsp;

##### TCA 循环关键步骤&nbsp;&nbsp;&nbsp;

- **第1步**：乙酰CoA + 草酰乙酸 → 柠檬酸（**==柠檬酸合酶==**，不可逆）&nbsp;&nbsp;&nbsp;
  - 抑制剂：**==ATP、NADH、柠檬酸、琥珀酰CoA==**；激活剂：**==ADP==**&nbsp;&nbsp;&nbsp;
- **第3步**：异柠檬酸 → α-酮戊二酸（**==异柠檬酸脱氢酶==**，不可逆）&nbsp;&nbsp;&nbsp;
  - 辅因子：NAD(P)⁺；抑制剂：**==ATP、NADH==**；激活剂：**==ADP、Ca²⁺==**&nbsp;&nbsp;&nbsp;
- **第4步**：α-酮戊二酸 → 琥珀酰CoA（**==α-酮戊二酸脱氢酶复合体==**，不可逆）&nbsp;&nbsp;&nbsp;
  - 组成和催化机制与丙酮酸脱氢酶复合体相似&nbsp;&nbsp;&nbsp;

##### TCA 循环要点（经过一次三羧酸循环）&nbsp;&nbsp;&nbsp;

- 消耗 1 分子乙酰CoA&nbsp;&nbsp;&nbsp;
- **==4 次脱氢，2 次脱羧，1 次底物水平磷酸化==**&nbsp;&nbsp;&nbsp;
- 生成：**==1 分子 FADH₂、3 分子 NADH+H⁺、2 分子 CO₂、1 分子 GTP==**&nbsp;&nbsp;&nbsp;
- **关键酶**：**==柠檬酸合酶、异柠檬酸脱氢酶、α-酮戊二酸脱氢酶复合体==**&nbsp;&nbsp;&nbsp;
- 整个循环反应为不可逆循环&nbsp;&nbsp;&nbsp;
- **==草酰乙酸==**的量直接影响循环速度，需不断补充&nbsp;&nbsp;&nbsp;

##### TCA 循环的作用&nbsp;&nbsp;&nbsp;

- 彻底分解乙酰CoA&nbsp;&nbsp;&nbsp;
- 人体各组织产生的 CO₂ 大部分由 TCA 循环产生&nbsp;&nbsp;&nbsp;
- 驱动呼吸链产生 ATP 的 NADH 和 FADH₂ 大部分由 TCA 循环产生&nbsp;&nbsp;&nbsp;
- **==三大营养物质（糖、脂肪、蛋白质/氨基酸）分解供能的共同代谢通路==**，也是三者代谢联系的**==枢纽==**&nbsp;&nbsp;&nbsp;
- TCA 循环为生物合成提供**==前体物质==**（琥珀酰CoA + 甘氨酸 → 血红素）&nbsp;&nbsp;&nbsp;

#### 第四阶段：氧化磷酸化&nbsp;&nbsp;&nbsp;

- 见第四章&nbsp;&nbsp;&nbsp;

### 三、有氧氧化生成的 ATP&nbsp;&nbsp;&nbsp;

- 1 分子葡萄糖有氧氧化经历：**==6 次脱羧，12 次脱氢==**&nbsp;&nbsp;&nbsp;

#### ATP 生成详表&nbsp;&nbsp;&nbsp;

- 第一阶段（胞浆）：&nbsp;&nbsp;&nbsp;
  - 葡萄糖 → 6-磷酸葡萄糖：-1 ATP&nbsp;&nbsp;&nbsp;
  - 6-磷酸果糖 → 1,6-二磷酸果糖：-1 ATP&nbsp;&nbsp;&nbsp;
  - 2×3-磷酸甘油醛 → 2×1,3-二磷酸甘油酸：2NADH → 3 或 5 ATP（取决于穿梭）&nbsp;&nbsp;&nbsp;
  - 2×1,3-二磷酸甘油酸 → 2×3-磷酸甘油酸：+2 ATP&nbsp;&nbsp;&nbsp;
  - 2×PEP → 2×丙酮酸：+2 ATP&nbsp;&nbsp;&nbsp;
- 第二阶段（线粒体基质）：&nbsp;&nbsp;&nbsp;
  - 2×丙酮酸 → 2×乙酰CoA：2NADH → 5 ATP&nbsp;&nbsp;&nbsp;
- 第三阶段（线粒体基质）：&nbsp;&nbsp;&nbsp;
  - 2×异柠檬酸 → 2×α-酮戊二酸：2NADH → 5 ATP&nbsp;&nbsp;&nbsp;
  - 2×α-酮戊二酸 → 2×琥珀酰CoA：2NADH → 5 ATP&nbsp;&nbsp;&nbsp;
  - 2×琥珀酰CoA → 2×琥珀酸：+2 ATP（底物水平磷酸化）&nbsp;&nbsp;&nbsp;
  - 2×琥珀酸 → 2×延胡索酸：2FADH₂ → 3 ATP&nbsp;&nbsp;&nbsp;
  - 2×苹果酸 → 2×草酰乙酸：2NADH → 5 ATP&nbsp;&nbsp;&nbsp;
- **总计**：1 分子葡萄糖 → **==30 或 32 分子 ATP==**&nbsp;&nbsp;&nbsp;

### 四、糖有氧氧化的调节&nbsp;&nbsp;&nbsp;

- 有氧氧化的调节通过对其关键酶的调节实现&nbsp;&nbsp;&nbsp;
- 关键酶：&nbsp;&nbsp;&nbsp;
  - 糖酵解：己糖激酶、磷酸果糖激酶-1、丙酮酸激酶&nbsp;&nbsp;&nbsp;
  - 丙酮酸氧化脱羧：丙酮酸脱氢酶复合体&nbsp;&nbsp;&nbsp;
  - TCA 循环：柠檬酸合酶、异柠檬酸脱氢酶、α-酮戊二酸脱氢酶复合体&nbsp;&nbsp;&nbsp;
- 调节逻辑：基于机体对能量的需求，ATP/ADP 或 ATP/AMP 比值全程调节，该比值升高时所有关键酶均被抑制&nbsp;&nbsp;&nbsp;
- 氧化磷酸化速率影响 TCA 循环，前者速率降低→后者速率减慢&nbsp;&nbsp;&nbsp;

#### 丙酮酸脱氢酶复合体的调节&nbsp;&nbsp;&nbsp;

- 别构调节：&nbsp;&nbsp;&nbsp;
  - 别构抑制剂：**==乙酰CoA、NADH、ATP、脂肪酸==**&nbsp;&nbsp;&nbsp;
  - 别构激活剂：**==CoA、NAD⁺、AMP(ADP)、Ca²⁺==**&nbsp;&nbsp;&nbsp;
- 共价修饰调节：**==磷酸化失活、去磷酸化激活==**&nbsp;&nbsp;&nbsp;

#### TCA 循环的调节&nbsp;&nbsp;&nbsp;

- **==ATP、柠檬酸、琥珀酰CoA、NADH==** 抑制关键酶&nbsp;&nbsp;&nbsp;

### 五、糖有氧氧化的生理意义&nbsp;&nbsp;&nbsp;

- 正常情况下糖分解供能的**==主要代谢途径==**&nbsp;&nbsp;&nbsp;
- 是机体获得能量（ATP）的**==主要方式==**&nbsp;&nbsp;&nbsp;
- 产能效率高，能量逐步释放，利用率高&nbsp;&nbsp;&nbsp;
- 是体内糖、脂、蛋白质三种主要有机物相互转变的联系体系&nbsp;&nbsp;&nbsp;

### 六、巴斯德效应(Pasteur effect)&nbsp;&nbsp;&nbsp;

- 概念：糖的有氧氧化**==抑制==**无氧氧化的现象&nbsp;&nbsp;&nbsp;
- 机制：&nbsp;&nbsp;&nbsp;
  - 有氧时：NADH+H⁺进入线粒体内氧化，丙酮酸进入线粒体进一步氧化而不生成乳酸&nbsp;&nbsp;&nbsp;
  - 缺氧时：糖无氧氧化加强，NADH+H⁺在胞浆浓度升高，丙酮酸作为氢接受体生成乳酸&nbsp;&nbsp;&nbsp;

### 无氧氧化 vs 有氧氧化对比&nbsp;&nbsp;&nbsp;

- 反应条件：&nbsp;&nbsp;&nbsp;
  - 无氧氧化：无氧&nbsp;&nbsp;&nbsp;
  - 有氧氧化：有氧&nbsp;&nbsp;&nbsp;
- 反应部位：&nbsp;&nbsp;&nbsp;
  - 无氧氧化：**==胞质溶胶==**&nbsp;&nbsp;&nbsp;
  - 有氧氧化：**==胞质溶胶和线粒体==**&nbsp;&nbsp;&nbsp;
- 终产物：&nbsp;&nbsp;&nbsp;
  - 无氧氧化：**==乳酸==**&nbsp;&nbsp;&nbsp;
  - 有氧氧化：**==CO₂ + H₂O==**&nbsp;&nbsp;&nbsp;
- 关键酶：&nbsp;&nbsp;&nbsp;
  - 无氧氧化：**==己糖激酶、PFK-1、丙酮酸激酶==**&nbsp;&nbsp;&nbsp;
  - 有氧氧化：上述三个 + **==丙酮酸脱氢酶复合体、柠檬酸合酶、异柠檬酸脱氢酶、α-酮戊二酸脱氢酶复合体==**&nbsp;&nbsp;&nbsp;
- ATP 生成方式：&nbsp;&nbsp;&nbsp;
  - 无氧氧化：**==底物水平磷酸化==**&nbsp;&nbsp;&nbsp;
  - 有氧氧化：**==氧化磷酸化（为主）+ 底物水平磷酸化==**&nbsp;&nbsp;&nbsp;
- ATP 产量(1 分子葡萄糖)：&nbsp;&nbsp;&nbsp;
  - 无氧氧化：==**2 分子 ATP**==&nbsp;&nbsp;&nbsp;
  - 有氧氧化：==**32 或 30 分子 ATP**==&nbsp;&nbsp;&nbsp;

## 第四节 磷酸戊糖途径&nbsp;&nbsp;&nbsp;

### 一、磷酸戊糖途径的概念&nbsp;&nbsp;&nbsp;

- **磷酸戊糖途径(pentose phosphate pathway)**：亦称磷酸己糖支路(hexose phosphate shunt)&nbsp;&nbsp;&nbsp;
- 指由葡糖-6-磷酸生成**核糖-5-磷酸**及 **NADPH+H⁺**，随后再进一步生成甘油醛-3-磷酸和果糖-6-磷酸的反应过程&nbsp;&nbsp;&nbsp;
- **反应部位**：==**胞质溶胶**==&nbsp;&nbsp;&nbsp;
- 反应的起始物：**==葡糖-6-磷酸==**&nbsp;&nbsp;&nbsp;
- 反应的产物：**==核糖-5-磷酸==**、**==NADPH+H⁺==**、**==CO₂==**&nbsp;&nbsp;&nbsp;

### 二、磷酸戊糖途径的反应过程&nbsp;&nbsp;&nbsp;

#### 第一阶段：氧化阶段（不可逆反应）&nbsp;&nbsp;&nbsp;

- 生成核糖-5-磷酸、NADPH+H⁺、CO₂&nbsp;&nbsp;&nbsp;

**第1步**：葡糖-6-磷酸 → 6-磷酸葡萄糖酸内酯（生成 1 分子 NADPH）
- 不可逆反应&nbsp;&nbsp;&nbsp;
- 酶：==**葡糖-6-磷酸脱氢酶**==（关键酶）&nbsp;&nbsp;&nbsp;

**第2步**：6-磷酸葡萄糖酸内酯 → 6-磷酸葡萄糖酸（水解）

**第3步**：6-磷酸葡萄糖酸 → 核酮糖-5-磷酸 → 核糖-5-磷酸（脱氢脱羧）

#### 第二阶段：非氧化阶段（可逆反应）&nbsp;&nbsp;&nbsp;

- 一系列基团的转移&nbsp;&nbsp;&nbsp;

#### 磷酸戊糖途径的特点&nbsp;&nbsp;&nbsp;

- 脱氢反应以 **NADP⁺** 为受氢体，生成 NADPH+H⁺&nbsp;&nbsp;&nbsp;
- 反应经过了 3、4、5、6、7 碳糖的演变过程&nbsp;&nbsp;&nbsp;
- 反应中生成了重要的中间代谢物——**核糖-5-磷酸**&nbsp;&nbsp;&nbsp;
- 一分子 G-6-P 经过反应，只能发生 **==1 次脱羧和 2 次脱氢==**，生成 1 分子 CO₂ 和 **==2 分子 NADPH+H⁺==**&nbsp;&nbsp;&nbsp;

### 三、磷酸戊糖途径的调节&nbsp;&nbsp;&nbsp;

- 限速酶：==**葡糖-6-磷酸脱氢酶**==，其活性高低决定 G-6-P 进入磷酸戊糖途径的流量&nbsp;&nbsp;&nbsp;
- 该酶活性受 **==NADPH/NADP⁺==** 的反馈抑制调节，NADPH 对该酶有强烈抑制作用&nbsp;&nbsp;&nbsp;
- 磷酸戊糖途径的流量取决于 **==NADPH 的需求==**&nbsp;&nbsp;&nbsp;

### 四、磷酸戊糖途径的生理意义&nbsp;&nbsp;&nbsp;

- **==为核酸的生物合成提供核糖-5-磷酸==**&nbsp;&nbsp;&nbsp;
- **==提供 NADPH==**，作为供氢体参与多种代谢反应：&nbsp;&nbsp;&nbsp;
  - 体内许多合成代谢的供氢体（如脂肪酸、胆固醇、非必需氨基酸的合成）&nbsp;&nbsp;&nbsp;
  - 参与体内羟化反应（如生物转化）&nbsp;&nbsp;&nbsp;
  - **==维持谷胱甘肽的还原状态==**（还原型谷胱甘肽是体内重要的抗氧化剂，可以保护含有游离巯基的酶或蛋白质免受氧化剂损害）&nbsp;&nbsp;&nbsp;

#### 临床联系：蚕豆病&nbsp;&nbsp;&nbsp;

- 6-磷酸葡萄糖脱氢酶先天不足 + 新鲜蚕豆（强氧化剂）→ NADPH 供应不足 → GSH 功能受影响 → 红细胞膜被氧化 → 细胞破裂 → **==溶血性贫血/蚕豆病（胡豆黄）==**&nbsp;&nbsp;&nbsp;

## 第五节 糖原的合成与分解&nbsp;&nbsp;&nbsp;

### 一、糖原的结构&nbsp;&nbsp;&nbsp;

- 摄入过多的糖类，多数转变为脂肪形式储存（长期），少数以糖原形式储存（短期）&nbsp;&nbsp;&nbsp;
- **糖原(glycogen)**：动物体内糖的储存形式，是机体能迅速动用的能源储备&nbsp;&nbsp;&nbsp;
  - **肝糖原**：肝脏，维持血糖水平，**==70-100g==**&nbsp;&nbsp;&nbsp;
  - **肌糖原**：肌肉，肌肉收缩所需，**==180-300g==**&nbsp;&nbsp;&nbsp;
- 糖原的生理意义：&nbsp;&nbsp;&nbsp;
  - 动物体内葡萄糖的一种贮存形式&nbsp;&nbsp;&nbsp;
  - **==肝糖原是血糖的主要来源==**&nbsp;&nbsp;&nbsp;
  - 肌糖原是肌肉收缩时能迅速动用的能源&nbsp;&nbsp;&nbsp;

#### 糖原的结构特点&nbsp;&nbsp;&nbsp;

- 葡萄糖单元以 **==α-1,4-糖苷键==**形成长链&nbsp;&nbsp;&nbsp;
- 约 10 个左右葡萄糖单元处形成分枝，分枝处葡萄糖以 **==α-1,6-糖苷键==**连接，分支增加→溶解度增加&nbsp;&nbsp;&nbsp;
- 每条链都终止于一个**==非还原端==**，非还原端增多→有利于被酶分解&nbsp;&nbsp;&nbsp;

### 二、糖原的合成(glycogenesis)&nbsp;&nbsp;&nbsp;

- **合成部位**：组织定位→**==主要在肝脏和肌肉==**；细胞定位→**==胞质溶胶==**&nbsp;&nbsp;&nbsp;
- **主要的酶**：UDPG 焦磷酸化酶、==**糖原合酶**==、糖原分支酶&nbsp;&nbsp;&nbsp;

#### 第一阶段：葡萄糖 → 尿苷二磷酸葡萄糖(UDPG)&nbsp;&nbsp;&nbsp;

- UDPG 是"活性葡萄糖"，作为葡萄糖供体&nbsp;&nbsp;&nbsp;
- 第1步：葡萄糖 → 葡糖-6-磷酸（己糖激酶）&nbsp;&nbsp;&nbsp;
- 第2步：葡糖-6-磷酸 → 葡糖-1-磷酸（磷酸葡萄糖变位酶）&nbsp;&nbsp;&nbsp;
- 第3步：葡糖-1-磷酸 → UDPG（UDPG 焦磷酸化酶）&nbsp;&nbsp;&nbsp;

#### 第二阶段：糖链的延长&nbsp;&nbsp;&nbsp;

- 糖原n + UDPG →（糖原合酶）→ 糖原n+1 + UDP&nbsp;&nbsp;&nbsp;
- 以 α-1,4-糖苷键的形式延长&nbsp;&nbsp;&nbsp;
- 糖原n 为原有的细胞内较小糖原分子，称为糖原"引物"&nbsp;&nbsp;&nbsp;

#### 第三阶段：糖原分支的形成&nbsp;&nbsp;&nbsp;

- 分支处以 α-1,6-糖苷键连接&nbsp;&nbsp;&nbsp;
- 酶：糖原分支酶&nbsp;&nbsp;&nbsp;

#### 糖原合成特点&nbsp;&nbsp;&nbsp;

- 需要活化的葡萄糖单位（UDPG）&nbsp;&nbsp;&nbsp;
- 需要引物（小分子糖原或糖基化的糖原蛋白）&nbsp;&nbsp;&nbsp;
- 合成关键酶：==**糖原合酶**==（α-1,4 糖苷键）&nbsp;&nbsp;&nbsp;
- 要形成分支：**==分支酶==**催化（α-1,6 糖苷键）&nbsp;&nbsp;&nbsp;
- 合成有方向：在糖原引物糖链的**==非还原端==**加上葡萄糖基&nbsp;&nbsp;&nbsp;
- 合成需要耗能：糖原每增加 1 个葡萄糖基需消耗 **==2 个 ATP==**&nbsp;&nbsp;&nbsp;

### 三、糖原的分解(glycogenolysis)&nbsp;&nbsp;&nbsp;

- 习惯上指**==肝糖原分解成为葡萄糖==**，从而补充血糖的过程&nbsp;&nbsp;&nbsp;
- **细胞定位**：==**胞质溶胶**==&nbsp;&nbsp;&nbsp;
- **主要的酶**：&nbsp;&nbsp;&nbsp;
  - ==**糖原磷酸化酶**==（关键酶）&nbsp;&nbsp;&nbsp;
  - 糖原脱支酶&nbsp;&nbsp;&nbsp;
  - 磷酸葡萄糖变位酶&nbsp;&nbsp;&nbsp;
  - ==**葡糖-6-磷酸酶**==（肌肉中缺乏）&nbsp;&nbsp;&nbsp;

#### 肝糖原与肌糖原的重要区别&nbsp;&nbsp;&nbsp;

- **肝脏**中有葡糖-6-磷酸酶 → 肝糖原可通过糖原分解来**==补充血糖==**&nbsp;&nbsp;&nbsp;
- **肌肉**中**==缺乏葡糖-6-磷酸酶==** → 肌糖原**==不能分解成葡萄糖==**，只能进行糖酵解或有氧氧化供能&nbsp;&nbsp;&nbsp;
- 肌糖原的分解与合成与乳酸循环有关&nbsp;&nbsp;&nbsp;

#### G-6-P 的代谢去路&nbsp;&nbsp;&nbsp;

- 进入磷酸戊糖途径 → 6-磷酸葡萄糖酸内酯&nbsp;&nbsp;&nbsp;
- 进入糖酵解/有氧氧化&nbsp;&nbsp;&nbsp;
- 在肝脏：G-6-P → 葡萄糖（补充血糖）&nbsp;&nbsp;&nbsp;
- 在肌肉：只能进行糖酵解或有氧氧化&nbsp;&nbsp;&nbsp;

### 四、糖原合成与分解的调节&nbsp;&nbsp;&nbsp;

- 关键酶：==**糖原合酶**==（合成）和==**糖原磷酸化酶**==（分解）&nbsp;&nbsp;&nbsp;
- 调节方式：&nbsp;&nbsp;&nbsp;
  - **别构调节**：AMP、ATP、葡糖-6-磷酸&nbsp;&nbsp;&nbsp;
  - **共价修饰调节（主要）**：==**磷酸化与去磷酸化**==&nbsp;&nbsp;&nbsp;
- 激素：&nbsp;&nbsp;&nbsp;
  - **胰高血糖素**：促进**==肝糖原分解==**&nbsp;&nbsp;&nbsp;
  - **肾上腺素**：促进**==肌糖原分解==**&nbsp;&nbsp;&nbsp;

#### 关键酶调节对比&nbsp;&nbsp;&nbsp;

- 糖原合酶（合成）：&nbsp;&nbsp;&nbsp;
  - 变构激活剂：**==ATP、G-6-P==**&nbsp;&nbsp;&nbsp;
  - 变构抑制剂：**==AMP==**&nbsp;&nbsp;&nbsp;
  - 磷酸化：**==无活性==**&nbsp;&nbsp;&nbsp;
  - 去磷酸化：**==有活性==**&nbsp;&nbsp;&nbsp;
- 糖原磷酸化酶（分解）：&nbsp;&nbsp;&nbsp;
  - 变构激活剂：**==AMP、ADP==**&nbsp;&nbsp;&nbsp;
  - 变构抑制剂：**==ATP、G-6-P==**&nbsp;&nbsp;&nbsp;
  - 磷酸化：**==活性高==**&nbsp;&nbsp;&nbsp;
  - 去磷酸化：**==活性低==**&nbsp;&nbsp;&nbsp;
- 共价修饰调节和别构调节对两个关键酶进行**反向调节**&nbsp;&nbsp;&nbsp;

## 第六节 糖异生&nbsp;&nbsp;&nbsp;

### 一、糖异生的概念&nbsp;&nbsp;&nbsp;

- **糖异生(gluconeogenesis)**：由非糖物质转变为**==葡萄糖或糖原==**的过程&nbsp;&nbsp;&nbsp;
- **原料**：**==丙酮酸、乳酸、甘油、生糖氨基酸、TCA 中间物==**等（注意：**==脂肪酸不能作为糖异生原料==**）&nbsp;&nbsp;&nbsp;
- **部位**：**==肝（主要）、肾（次要，饥饿时）==**&nbsp;&nbsp;&nbsp;
- **亚细胞定位**：**==胞质溶胶或胞质溶胶 + 线粒体==**&nbsp;&nbsp;&nbsp;

### 二、糖异生的途径&nbsp;&nbsp;&nbsp;

- 糖异生途径与糖酵解途径大多数反应是共有的、可逆的&nbsp;&nbsp;&nbsp;
- 糖酵解途径中有 **==3 个由关键酶催化的不可逆反应==**，在糖异生时由另外的酶催化&nbsp;&nbsp;&nbsp;
- 糖酵解是产生能量的过程，糖异生需要**==消耗能量==**&nbsp;&nbsp;&nbsp;

#### 糖异生的四个关键酶（绕过三个能障）&nbsp;&nbsp;&nbsp;

**一：丙酮酸 → 磷酸烯醇式丙酮酸(PEP)**
- ① ==**丙酮酸羧化酶**==（线粒体），辅酶为**==生物素==**&nbsp;&nbsp;&nbsp;
- ② **==磷酸烯醇式丙酮酸羧激酶(PEPCK)==**（线粒体、细胞质）&nbsp;&nbsp;&nbsp;
- 此过程相当于消耗 2 分子 ATP&nbsp;&nbsp;&nbsp;

**二：1,6-二磷酸果糖 → 6-磷酸果糖**
- 酶：==**果糖二磷酸酶-1**==&nbsp;&nbsp;&nbsp;

**三：6-磷酸葡萄糖 → 葡萄糖**
- 酶：==**葡糖-6-磷酸酶**==&nbsp;&nbsp;&nbsp;

- 由 2 分子丙酮酸合成 1 分子葡萄糖需要消耗 ==**6 个 ATP**==&nbsp;&nbsp;&nbsp;

### 三、糖异生的调节&nbsp;&nbsp;&nbsp;

- 关键酶：丙酮酸羧化酶、PEPCK、果糖二磷酸酶-1、葡糖-6-磷酸酶&nbsp;&nbsp;&nbsp;
- 激素调节：胰高血糖素、肾上腺素、肾上腺皮质激素、胰岛素等&nbsp;&nbsp;&nbsp;
- 糖异生与糖酵解存在反向别构调节&nbsp;&nbsp;&nbsp;

### 四、糖异生的生理意义&nbsp;&nbsp;&nbsp;

- **==维持血糖浓度恒定==**：长时间空腹或饥饿时，糖异生增强&nbsp;&nbsp;&nbsp;
- **==补充或恢复肝糖原储备的重要途径==**：三碳途径——进食特别是饥饿后进食，大部分葡萄糖先在肝外分解为乳酸或丙酮酸等三碳化合物，再进入肝经糖异生合成糖原&nbsp;&nbsp;&nbsp;
- **==促进肾小管泌氨，调节酸碱平衡==**：长期饥饿，肾糖异生增强&nbsp;&nbsp;&nbsp;
- **==促进乳酸的充分利用——乳酸循环==**&nbsp;&nbsp;&nbsp;

### 五、乳酸循环(Cori 循环)&nbsp;&nbsp;&nbsp;

- 肌收缩（尤其是供氧不足时）葡萄糖或肌糖原通过无氧氧化生成乳酸，释放进入血液后被肝摄入，在肝内糖异生为葡萄糖；葡萄糖进入血液后又可被肌肉摄取，构成一个循环&nbsp;&nbsp;&nbsp;
- 2 分子乳酸异生为 1 分子葡萄糖消耗 **==6 分子 ATP==**&nbsp;&nbsp;&nbsp;
- 生理意义：&nbsp;&nbsp;&nbsp;
  - **==乳酸再利用，避免乳酸损失==**&nbsp;&nbsp;&nbsp;
  - **==防止乳酸的堆积引起酸中毒==**&nbsp;&nbsp;&nbsp;

## 第七节 血糖及其调节&nbsp;&nbsp;&nbsp;

### 一、概述&nbsp;&nbsp;&nbsp;

- **血糖(blood sugar)**：指血液中的葡萄糖&nbsp;&nbsp;&nbsp;
- 血糖水平相对恒定，正常值约 ==**3.89～6.11 mmol/L**==&nbsp;&nbsp;&nbsp;
- 血糖水平恒定的生理意义：&nbsp;&nbsp;&nbsp;
  - 保证脑等重要组织器官的能量供应（脑组织不能利用脂肪酸，主要依赖葡萄糖供能；红细胞没有线粒体，完全通过糖酵解获能；骨髓和神经组织经常利用葡萄糖供能）&nbsp;&nbsp;&nbsp;
- **肾糖阈**为 **==8.89-10.00 mmol/L==**，当血糖浓度高于肾糖阈出现尿糖现象&nbsp;&nbsp;&nbsp;

### 二、血糖的来源和去路&nbsp;&nbsp;&nbsp;

#### 来源（4 条）&nbsp;&nbsp;&nbsp;
- **==食物糖（主要）==**→ 消化吸收&nbsp;&nbsp;&nbsp;
- **==肝糖原==** → 分解&nbsp;&nbsp;&nbsp;
- **==非糖物质==** → 糖异生&nbsp;&nbsp;&nbsp;
- **==其他单糖==** → 转化&nbsp;&nbsp;&nbsp;

#### 去路（5 条）&nbsp;&nbsp;&nbsp;
- **==氧化分解（主要）==**→ CO₂ + H₂O&nbsp;&nbsp;&nbsp;
- **==糖原合成==** → 肝糖原、肌糖原&nbsp;&nbsp;&nbsp;
- **==磷酸戊糖途径==**&nbsp;&nbsp;&nbsp;
- **==转变成其他糖类物质==**&nbsp;&nbsp;&nbsp;
- **==转变成脂肪、氨基酸等==**&nbsp;&nbsp;&nbsp;
- 超肾糖阈 → **==尿糖==**&nbsp;&nbsp;&nbsp;

### 三、血糖水平的调节&nbsp;&nbsp;&nbsp;

- 三级水平：神经系统 → 激素（主要）→ 组织器官（关键酶）&nbsp;&nbsp;&nbsp;
- **降低血糖的激素**：==**胰岛素**==（唯一）&nbsp;&nbsp;&nbsp;
- **升高血糖的激素**：**==胰高血糖素、肾上腺素、糖皮质激素、生长激素==**等&nbsp;&nbsp;&nbsp;
- **肝脏**是调节血糖浓度**==最重要的器官==**&nbsp;&nbsp;&nbsp;

#### 葡萄糖耐量&nbsp;&nbsp;&nbsp;

- 正常人体存在精细的调节糖代谢机制，一次性食入大量葡萄糖后，血糖水平不会出现大的波动和持续升高&nbsp;&nbsp;&nbsp;
- **糖耐量实验(OGTT)**：葡萄糖负荷实验，用于了解机体对葡萄糖的调节能力，帮助诊断糖尿病&nbsp;&nbsp;&nbsp;

### 四、血糖水平异常&nbsp;&nbsp;&nbsp;

- **高血糖(hyperglycemia)**：空腹血糖浓度高于 **==7.1 mmol/L==**&nbsp;&nbsp;&nbsp;
  - 危害：当血糖浓度超过了肾糖阈，可出现糖尿；持续性高血糖和糖尿主要见于**==糖尿病==**&nbsp;&nbsp;&nbsp;
- **低血糖(hypoglycemia)**：血糖浓度低于 **==2.8 mmol/L==**&nbsp;&nbsp;&nbsp;
  - 危害：影响脑的正常功能，出现头晕、倦怠无力、心悸等，严重时昏迷（**==低血糖休克==**），如不及时补充葡萄糖可导致死亡&nbsp;&nbsp;&nbsp;
