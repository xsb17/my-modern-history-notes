# 磷酸戊糖途径（极高频临床联系）&nbsp;&nbsp;&nbsp;

- **磷酸戊糖途径概念**&nbsp;&nbsp;&nbsp;
	- 又称 -> 磷酸己糖支路(hexose phosphate shunt)&nbsp;&nbsp;&nbsp;
	- 由**==葡糖-6-磷酸==**生成 -> **==核糖-5-磷酸 + NADPH + CO₂==**&nbsp;&nbsp;&nbsp;
	- 反应部位 -> ==**胞质溶胶**==&nbsp;&nbsp;&nbsp;
	- 起始物 -> **==葡糖-6-磷酸(G-6-P)==**&nbsp;&nbsp;&nbsp;
- **两大核心产物及功能**&nbsp;&nbsp;&nbsp;
	- **核糖-5-磷酸** -> 为**==核酸生物合成==**提供原料 -> DNA/RNA合成必需&nbsp;&nbsp;&nbsp;
	- **NADPH** -> 供氢体 -> 三大功能&nbsp;&nbsp;&nbsp;
		- 合成代谢供氢 -> **==脂肪酸、胆固醇、非必需氨基酸==**合成&nbsp;&nbsp;&nbsp;
		- 参与**==羟化反应==** -> 生物转化（肝脏解毒）&nbsp;&nbsp;&nbsp;
		- **==维持谷胱甘肽(GSH)还原状态==** -> GSH保护含-SH的酶/蛋白质免遭氧化&nbsp;&nbsp;&nbsp;
- **反应过程两阶段**&nbsp;&nbsp;&nbsp;
	- 第一阶段：**氧化阶段**（不可逆）&nbsp;&nbsp;&nbsp;
		- G-6-P -> 6-磷酸葡萄糖酸内酯 -> 6-磷酸葡萄糖酸 -> 核酮糖-5-磷酸 -> 核糖-5-磷酸&nbsp;&nbsp;&nbsp;
		- 关键酶 -> **==葡糖-6-磷酸脱氢酶(G6PD)==**&nbsp;&nbsp;&nbsp;
		- 一分子G-6-P -> 1次脱羧 + 2次脱氢 -> **==2 NADPH + 1 CO₂ + 1 核糖-5-磷酸==**&nbsp;&nbsp;&nbsp;
	- 第二阶段：**非氧化阶段**（可逆）-> 一系列基团转移&nbsp;&nbsp;&nbsp;
- **磷酸戊糖途径特点**&nbsp;&nbsp;&nbsp;
	- 脱氢以**==NADP⁺为受氢体==**（不是 NAD⁺！）-> 生成 **==NADPH==**&nbsp;&nbsp;&nbsp;
	- 经历了3/4/5/6/7碳糖演变&nbsp;&nbsp;&nbsp;
	- 一分子G-6-P -> **==1次脱羧 + 2次脱氢==** -> 2 NADPH&nbsp;&nbsp;&nbsp;
- **调节**&nbsp;&nbsp;&nbsp;
	- 限速酶 -> **==葡糖-6-磷酸脱氢酶(G6PD)==** -> 决定G-6-P进入PPP流量&nbsp;&nbsp;&nbsp;
	- 受 **==NADPH/NADP⁺==** 反馈抑制 -> NADPH强烈抑制G6PD&nbsp;&nbsp;&nbsp;
	- PPP流量取决于 -> **==NADPH的需求==**&nbsp;&nbsp;&nbsp;
- **蚕豆病（胡豆黄）— 必考临床**&nbsp;&nbsp;&nbsp;
	- 病因 -> **==葡糖-6-磷酸脱氢酶(G6PD)先天缺乏==**&nbsp;&nbsp;&nbsp;
	- 诱因 -> 进食**==新鲜蚕豆==**（强氧化剂）或其他致病因子&nbsp;&nbsp;&nbsp;
	- 机制 -> G6PD缺乏 -> **==NADPH生成减少==** -> GSH(还原型谷胱甘肽)减少 -> 红细胞膜被**==氧化破坏==** -> 细胞破裂 -> 血红蛋白逸出 -> ==**溶血性贫血**==&nbsp;&nbsp;&nbsp;
	- 本质 -> 磷酸戊糖途径受阻导致的红细胞抗氧化能力下降&nbsp;&nbsp;&nbsp;
- **NADPH 与 NADH 区别（必考点）**&nbsp;&nbsp;&nbsp;
	- NADH -> 来自糖酵解/TCA -> 进入**呼吸链** -> 产生**ATP**&nbsp;&nbsp;&nbsp;
	- NADPH -> 来自**磷酸戊糖途径** -> 用于**合成代谢+抗氧化** -> 不进入呼吸链产能&nbsp;&nbsp;&nbsp;
	- 受氢体不同 -> NAD⁺(糖酵解/TCA) vs NADP⁺(PPP)&nbsp;&nbsp;&nbsp;
