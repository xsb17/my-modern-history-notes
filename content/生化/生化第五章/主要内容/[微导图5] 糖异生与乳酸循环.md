# 糖异生与乳酸循环&nbsp;&nbsp;&nbsp;

- **糖异生概念**&nbsp;&nbsp;&nbsp;
	- 由**==非糖物质==**转变为**==葡萄糖或糖原==**的过程&nbsp;&nbsp;&nbsp;
	- 原料 -> **==乳酸、甘油、生糖氨基酸、丙酮酸、TCA中间物==**&nbsp;&nbsp;&nbsp;
	- **==脂肪酸不能作为糖异生原料==**（动物体内乙酰CoA不能净合成糖）&nbsp;&nbsp;&nbsp;
	- 部位 -> **==肝==**(主要) / **==肾==**(次要，饥饿时)&nbsp;&nbsp;&nbsp;
	- 亚细胞定位 -> **==胞质溶胶 / 胞质溶胶+线粒体==**&nbsp;&nbsp;&nbsp;
- **糖异生途径：绕过糖酵解3个能障**&nbsp;&nbsp;&nbsp;
	- 糖异生与糖酵解大多数反应共存 -> 可逆&nbsp;&nbsp;&nbsp;
	- 糖酵解3个不可逆反应需要**4个新酶**绕过&nbsp;&nbsp;&nbsp;
	- **能障1：丙酮酸 -> PEP**（绕过PK催化的不可逆反应）&nbsp;&nbsp;&nbsp;
		- ① ==**丙酮酸羧化酶**==（线粒体）-> 丙酮酸 -> 草酰乙酸 -> 辅酶：**==生物素==**&nbsp;&nbsp;&nbsp;
		- ② **==磷酸烯醇式丙酮酸羧激酶(PEPCK)==** -> 草酰乙酸 -> PEP&nbsp;&nbsp;&nbsp;
		- 消耗 -> ==**2 ATP**==（1 ATP + 1 GTP）&nbsp;&nbsp;&nbsp;
		- 草酰乙酸不能穿膜 -> 转变为**==苹果酸/天冬氨酸==**穿出线粒体&nbsp;&nbsp;&nbsp;
	- **能障2：1,6-二磷酸果糖 -> 6-磷酸果糖**（绕过PFK-1）&nbsp;&nbsp;&nbsp;
		- ③ ==**果糖二磷酸酶-1**== -> 水解掉1位磷酸&nbsp;&nbsp;&nbsp;
	- **能障3：6-磷酸葡萄糖 -> 葡萄糖**（绕过HK）&nbsp;&nbsp;&nbsp;
		- ④ ==**葡糖-6-磷酸酶**== -> 水解掉磷酸 -> 仅存在于**==肝、肾==**&nbsp;&nbsp;&nbsp;
	- 产能消耗 -> 2分子丙酮酸 -> 1分子葡萄糖 -> 消耗 **==6 ATP==**&nbsp;&nbsp;&nbsp;
- **糖异生的四个关键酶（必背）**&nbsp;&nbsp;&nbsp;
	- ==**丙酮酸羧化酶**== -> 线粒体 -> 辅酶生物素&nbsp;&nbsp;&nbsp;
	- **==磷酸烯醇式丙酮酸羧激酶(PEPCK)==**&nbsp;&nbsp;&nbsp;
	- ==**果糖二磷酸酶-1**==&nbsp;&nbsp;&nbsp;
	- ==**葡糖-6-磷酸酶**== -> 仅肝/肾&nbsp;&nbsp;&nbsp;
- **糖异生的调节**&nbsp;&nbsp;&nbsp;
	- 与糖酵解**反向别构调节**&nbsp;&nbsp;&nbsp;
		- 果糖-2,6-双磷酸 -> 激活PFK-1(糖酵解) / 抑制果糖二磷酸酶-1(糖异生)&nbsp;&nbsp;&nbsp;
		- 乙酰CoA -> 激活丙酮酸羧化酶(糖异生) / 抑制PK(糖酵解)&nbsp;&nbsp;&nbsp;
	- 激素调节&nbsp;&nbsp;&nbsp;
		- **胰高血糖素** -> 促进糖异生&nbsp;&nbsp;&nbsp;
		- ==**胰岛素**== -> 抑制糖异生&nbsp;&nbsp;&nbsp;
		- 肾上腺素、糖皮质激素 -> 促进糖异生&nbsp;&nbsp;&nbsp;
- **糖异生三大生理意义**&nbsp;&nbsp;&nbsp;
	- **==维持血糖浓度恒定==** -> 空腹/饥饿时糖异生增强&nbsp;&nbsp;&nbsp;
	- **==补充/恢复肝糖原储备==** -> 三碳途径 -> 乳酸/丙酮酸 -> 肝 -> 糖原&nbsp;&nbsp;&nbsp;
	- **==促进肾小管泌氨，调节酸碱平衡==** -> 长期饥饿肾糖异生增强&nbsp;&nbsp;&nbsp;
- **乳酸循环(Cori循环)**&nbsp;&nbsp;&nbsp;
	- 过程&nbsp;&nbsp;&nbsp;
		- 肌肉 -> 葡萄糖/肌糖原 -> 无氧氧化 -> ==**乳酸**==&nbsp;&nbsp;&nbsp;
		- 乳酸入血 -> **==肝==**摄取 -> 经糖异生 -> **==葡萄糖==**&nbsp;&nbsp;&nbsp;
		- 葡萄糖入血 -> 被**==肌肉==**摄取利用&nbsp;&nbsp;&nbsp;
	- 耗能 -> 2分子乳酸 -> 1分子葡萄糖 -> 消耗 **==6 ATP==**&nbsp;&nbsp;&nbsp;
	- **两大生理意义**&nbsp;&nbsp;&nbsp;
		- **==乳酸再利用==** -> 避免乳酸损失&nbsp;&nbsp;&nbsp;
		- **==防止乳酸堆积==** -> 避免**==酸中毒==**&nbsp;&nbsp;&nbsp;
	- 关键酶联系 -> 肌肉缺乏==**葡糖-6-磷酸酶**== -> 肌糖原不能直接补充血糖 -> 需要乳酸循环"转运"&nbsp;&nbsp;&nbsp;
- **不同原料进入糖异生的路径**&nbsp;&nbsp;&nbsp;
	- **乳酸** -> 乳酸 -> 丙酮酸 -> 草酰乙酸 -> PEP -> ... -> 葡萄糖&nbsp;&nbsp;&nbsp;
	- **甘油** -> 甘油 -> α-磷酸甘油 -> 磷酸二羟丙酮 -> ... -> 葡萄糖&nbsp;&nbsp;&nbsp;
	- **生糖氨基酸** -> 脱氨 -> α-酮酸 -> TCA中间物/丙酮酸 -> ... -> 葡萄糖&nbsp;&nbsp;&nbsp;
