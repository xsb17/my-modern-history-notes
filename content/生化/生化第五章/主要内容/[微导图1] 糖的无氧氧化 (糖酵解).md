# 糖的无氧氧化 (糖酵解)&nbsp;&nbsp;&nbsp;

- **糖酵解概念**&nbsp;&nbsp;&nbsp;
	- 缺氧/氧利用障碍 -> 葡萄糖/糖原 -> 分解为**==丙酮酸==** -> 还原为**==乳酸==** -> 产生少量能量&nbsp;&nbsp;&nbsp;
	- 从葡萄糖到丙酮酸 -> 称为**糖酵解**&nbsp;&nbsp;&nbsp;
	- **反应部位** -> ==**胞质溶胶**==&nbsp;&nbsp;&nbsp;
- **第一阶段：耗能阶段（5步）**&nbsp;&nbsp;&nbsp;
	- 葡萄糖 -> 2×3-磷酸甘油醛 -> 消耗 ==**2 ATP**==&nbsp;&nbsp;&nbsp;
	- **关键酶1**：**==己糖激酶(HK)==**&nbsp;&nbsp;&nbsp;
		- 催化：葡萄糖 -> **==葡糖-6-磷酸==** -> 不可逆&nbsp;&nbsp;&nbsp;
		- 肝内为**==葡萄糖激酶==**(HK-IV) -> Km=10mmol/L -> 维持血糖平衡&nbsp;&nbsp;&nbsp;
		- 变构抑制剂 -> **==葡糖-6-磷酸、长链脂肪酸==**&nbsp;&nbsp;&nbsp;
		- 必需激活剂 -> **==Mg²⁺==**&nbsp;&nbsp;&nbsp;
	- **关键酶2**：**==磷酸果糖激酶-1(PFK-1)==** -> 最重要限速酶&nbsp;&nbsp;&nbsp;
		- 催化：果糖-6-磷酸 -> **==果糖-1,6-双磷酸==** -> 不可逆&nbsp;&nbsp;&nbsp;
		- 变构激活剂 -> **==AMP、ADP、果糖-2,6-双磷酸==**&nbsp;&nbsp;&nbsp;
		- 变构抑制剂 -> **==ATP、柠檬酸、长链脂肪酸==**&nbsp;&nbsp;&nbsp;
	- 醛缩酶 -> 果糖-1,6-双磷酸 -> 磷酸二羟丙酮 + 3-磷酸甘油醛 -> 可逆&nbsp;&nbsp;&nbsp;
- **第二阶段：产能阶段（5步）**&nbsp;&nbsp;&nbsp;
	- 2×3-磷酸甘油醛 -> 2×丙酮酸&nbsp;&nbsp;&nbsp;
	- **第6步**：3-磷酸甘油醛 -> **1,3-二磷酸甘油酸**(高能化合物) -> 生成NADH&nbsp;&nbsp;&nbsp;
	- **第7步**：1,3-二磷酸甘油酸 -> 3-磷酸甘油酸 -> 第一个**==底物水平磷酸化==** -> 产生 **ATP**&nbsp;&nbsp;&nbsp;
	- 第8步：3-磷酸甘油酸 -> 2-磷酸甘油酸 -> 磷酸甘油酸变位酶&nbsp;&nbsp;&nbsp;
	- 第9步：2-磷酸甘油酸 -> **PEP**(高能化合物)&nbsp;&nbsp;&nbsp;
	- **关键酶3**：**==丙酮酸激酶(PK)==**&nbsp;&nbsp;&nbsp;
		- 催化：PEP -> **==丙酮酸==** -> 不可逆 -> 底物水平磷酸化 -> 产生**ATP**&nbsp;&nbsp;&nbsp;
		- 变构抑制剂 -> **==ATP、柠檬酸、乙酰CoA、丙氨酸==**&nbsp;&nbsp;&nbsp;
		- 共价修饰：**==磷酸化失活==**&nbsp;&nbsp;&nbsp;
- **第三阶段：丙酮酸 -> 乳酸**&nbsp;&nbsp;&nbsp;
	- NADH+H⁺来自第6步 -> 供氢给丙酮酸 -> 还原为乳酸&nbsp;&nbsp;&nbsp;
- **糖酵解总产出（1分子葡萄糖）**&nbsp;&nbsp;&nbsp;
	- 方式 -> **==底物水平磷酸化==**&nbsp;&nbsp;&nbsp;
	- 净生成ATP -> 从葡萄糖：==**2 ATP**== / 从糖原：==**3 ATP**==&nbsp;&nbsp;&nbsp;
	- 3步不可逆反应 / 3个关键酶 -> **==HK、PFK-1、PK==**&nbsp;&nbsp;&nbsp;
	- PFK-1活性最低 -> **==最重要限速酶==**&nbsp;&nbsp;&nbsp;
- **生理意义**&nbsp;&nbsp;&nbsp;
	- 缺氧时快速供能 -> **==不耗氧==**&nbsp;&nbsp;&nbsp;
	- **成熟红细胞** -> **==唯一供能途径==**&nbsp;&nbsp;&nbsp;
	- **肿瘤细胞** -> ==**Warburg效应**== -> PET-CT诊断基础&nbsp;&nbsp;&nbsp;
	- 白细胞、骨髓细胞、神经细胞 -> 重要供能途径&nbsp;&nbsp;&nbsp;
- **丙酮酸去路**&nbsp;&nbsp;&nbsp;
	- 缺氧 -> ==**乳酸**==（无氧氧化）&nbsp;&nbsp;&nbsp;
	- 有氧 -> **乙酰CoA** -> TCA循环+呼吸链 -> CO₂+H₂O（有氧氧化）&nbsp;&nbsp;&nbsp;
- **糖酵解关键酶调节速记**&nbsp;&nbsp;&nbsp;
	- ==**HK**== -> 抑制剂：**==G-6-P==** -> 产物反馈抑制&nbsp;&nbsp;&nbsp;
	- ==**PFK-1**== -> 激活：**==AMP/ADP/F-2,6-BP==** / 抑制：**==ATP/柠檬酸==**&nbsp;&nbsp;&nbsp;
	- ==**PK**== -> 抑制：**==ATP/乙酰CoA==** / 修饰：**==磷酸化失活==**&nbsp;&nbsp;&nbsp;
