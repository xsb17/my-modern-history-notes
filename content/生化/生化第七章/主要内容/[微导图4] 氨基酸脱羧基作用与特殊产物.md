# 氨基酸脱羧基作用与特殊产物&nbsp;&nbsp;&nbsp;

- **脱羧基作用概述**&nbsp;&nbsp;&nbsp;
	- 氨基酸 ->(氨基酸脱羧酶/==**磷酸吡哆醛==**)-> **==胺==** + CO₂&nbsp;&nbsp;&nbsp;
	- 胺 ->(单胺氧化酶)-> 醛 + NH₃ + H₂O₂&nbsp;&nbsp;&nbsp;
- **谷氨酸 -> γ-氨基丁酸(GABA)**&nbsp;&nbsp;&nbsp;
	- 酶 -> **谷氨酸脱羧酶**（辅酶：磷酸吡哆醛/B₆）&nbsp;&nbsp;&nbsp;
	- GABA -> **==抑制性神经递质==** -> 对学习记忆、运动**==负向调节==**&nbsp;&nbsp;&nbsp;
	- 临床 -> ==**维生素B₆**==治疗妊娠呕吐和小儿抽搐 -> 增强脱羧酶 -> GABA↑ -> 抑制神经过度兴奋&nbsp;&nbsp;&nbsp;
- **组氨酸 -> 组胺**&nbsp;&nbsp;&nbsp;
	- **==组氨酸脱羧酶==**&nbsp;&nbsp;&nbsp;
	- 组胺作用&nbsp;&nbsp;&nbsp;
		- **==强烈血管舒张剂==** -> 增加毛细血管通透性&nbsp;&nbsp;&nbsp;
		- 使平滑肌收缩 -> 支气管痉挛 -> **==哮喘==**&nbsp;&nbsp;&nbsp;
		- 刺激**==胃蛋白酶原及胃酸分泌==**&nbsp;&nbsp;&nbsp;
- **色氨酸 -> 5-羟色胺(5-HT)**&nbsp;&nbsp;&nbsp;
	- 过程 -> 色氨酸→(羟化酶)→5-羟色氨酸→(**脱羧酶**)-> 5-HT&nbsp;&nbsp;&nbsp;
	- 5-HT -> 脑内**抑制性神经递质**&nbsp;&nbsp;&nbsp;
	- 外周 -> **收缩血管**&nbsp;&nbsp;&nbsp;
- **半胱氨酸 -> 牛磺酸**&nbsp;&nbsp;&nbsp;
	- 半胱氨酸→(氧化)→磺酸丙氨酸→(脱羧酶)→==**牛磺酸**==&nbsp;&nbsp;&nbsp;
	- 功能 -> 促进脑细胞发育 / 提高神经传导和视觉机能 / 可能是抑制性神经递质&nbsp;&nbsp;&nbsp;
- **鸟氨酸 -> 多胺（腐胺、亚精胺、精胺）**&nbsp;&nbsp;&nbsp;
	- 限速酶 -> ==**鸟氨酸脱羧酶**==&nbsp;&nbsp;&nbsp;
	- 多胺功能 -> **==促进细胞增殖==**&nbsp;&nbsp;&nbsp;
	- 临床 -> 测定血/尿多胺水平 -> **肿瘤辅助诊断**和观察病情变化的**生化指标**&nbsp;&nbsp;&nbsp;
- **脱羧产物速记口诀**&nbsp;&nbsp;&nbsp;
	- **谷→GABA**(抑制) / **组→组胺**(扩血管) / **色→5-HT**(抑制+缩血管)&nbsp;&nbsp;&nbsp;
	- **半胱→牛磺酸**(脑发育) / **鸟→多胺**(促增殖/肿瘤标志)&nbsp;&nbsp;&nbsp;
