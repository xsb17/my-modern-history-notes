# 一碳单位代谢&nbsp;&nbsp;&nbsp;

- **一碳单位概念**&nbsp;&nbsp;&nbsp;
	- 定义 -> 某些氨基酸分解代谢产生的只含**一个碳原子**的有机基团&nbsp;&nbsp;&nbsp;
	- 种类（5种）&nbsp;&nbsp;&nbsp;
		- **甲基** (-CH₃)&nbsp;&nbsp;&nbsp;
		- **甲烯基** (-CH₂-)&nbsp;&nbsp;&nbsp;
		- **甲炔基** (-CH=)&nbsp;&nbsp;&nbsp;
		- **甲酰基** (-CHO)&nbsp;&nbsp;&nbsp;
		- **亚胺甲基** (-CH=NH)&nbsp;&nbsp;&nbsp;
	- 特点 -> 是化学基团 -> **不能单独存在** -> 必须由载体携带&nbsp;&nbsp;&nbsp;
- **一碳单位的载体**&nbsp;&nbsp;&nbsp;
	- **==四氢叶酸(FH₄/THF)==** -> **==叶酸的活性形式==**&nbsp;&nbsp;&nbsp;
	- 一碳单位结合在FH₄的 **N⁵、N¹⁰** 位上&nbsp;&nbsp;&nbsp;
	- 不同一碳单位形成不同的FH₄衍生物&nbsp;&nbsp;&nbsp;
		- N¹⁰-CHO-FH₄（N¹⁰-甲酰四氢叶酸）&nbsp;&nbsp;&nbsp;
		- N⁵,N¹⁰=CH-FH₄（N⁵,N¹⁰-甲炔四氢叶酸）&nbsp;&nbsp;&nbsp;
		- N⁵,N¹⁰-CH₂-FH₄（N⁵,N¹⁰-甲烯四氢叶酸）&nbsp;&nbsp;&nbsp;
		- **N⁵-CH₃-FH₄**（N⁵-甲基四氢叶酸）-> **唯一不可逆**的反应&nbsp;&nbsp;&nbsp;
- **一碳单位的来源（4种氨基酸）**&nbsp;&nbsp;&nbsp;
	- ==**丝氨酸**== -> N⁵,N¹⁰-CH₂-FH₄&nbsp;&nbsp;&nbsp;
	- ==**甘氨酸**== -> N⁵,N¹⁰-CH₂-FH₄&nbsp;&nbsp;&nbsp;
	- ==**组氨酸**== -> N⁵,N¹⁰=CH-FH₄&nbsp;&nbsp;&nbsp;
	- ==**色氨酸**== -> N¹⁰-CHO-FH₄&nbsp;&nbsp;&nbsp;
	- 口诀 -> "**施舍竹竿**"（丝-色-组-甘）&nbsp;&nbsp;&nbsp;
- **一碳单位的互变**&nbsp;&nbsp;&nbsp;
	- N¹⁰-甲酰FH₄ ⇌ N⁵,N¹⁰-甲炔FH₄ ⇌ N⁵,N¹⁰-甲烯FH₄ → **N⁵-甲基-FH₄**&nbsp;&nbsp;&nbsp;
	- N⁵-CH₃-FH₄的生成是**不可逆**的 -> 只能通过**甲硫氨酸循环**使FH₄再生&nbsp;&nbsp;&nbsp;
- **一碳单位的生理作用（三大意义）**&nbsp;&nbsp;&nbsp;
	- 合成**嘌呤和嘧啶**的原料 -> 在**==核酸生物合成==**中占重要地位&nbsp;&nbsp;&nbsp;
	- 把**==氨基酸代谢==**与**==核苷酸代谢==**联系起来 -> 桥梁作用&nbsp;&nbsp;&nbsp;
	- 参与体内重要化合物的合成和修饰 -> **肾上腺素、肌酸、胆碱、肉碱**等&nbsp;&nbsp;&nbsp;
- **一碳单位代谢与疾病**&nbsp;&nbsp;&nbsp;
	- 叶酸缺乏 -> FH₄不足 -> 一碳单位代谢障碍 -> dTMP合成↓ -> DNA合成↓ -> ==**巨幼红细胞贫血**==&nbsp;&nbsp;&nbsp;
	- VitB₁₂缺乏 -> N⁵-CH₃-FH₄不能将甲基转给同型半胱氨酸 -> FH₄不能再生 -> **"甲基陷阱"** -> 同样导致巨幼贫&nbsp;&nbsp;&nbsp;
- **一碳单位代谢小结**&nbsp;&nbsp;&nbsp;
	- 来源 -> 丝/色/组/甘氨酸 -> "施舍竹竿"&nbsp;&nbsp;&nbsp;
	- 载体 -> **四氢叶酸FH₄** -> N⁵,N¹⁰位结合&nbsp;&nbsp;&nbsp;
	- 功能 -> 提供核酸合成原料 / 联系氨基酸与核苷酸代谢&nbsp;&nbsp;&nbsp;
