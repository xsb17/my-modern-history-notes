# 芳香族氨基酸代谢与缺陷病&nbsp;&nbsp;&nbsp;

- **芳香族氨基酸家族**&nbsp;&nbsp;&nbsp;
	- 三种 -> ==**苯丙氨酸**==(必需) / ==**酪氨酸**==(非必需，由苯丙氨酸转变) / ==**色氨酸**==(必需)&nbsp;&nbsp;&nbsp;
- **苯丙氨酸的代谢**&nbsp;&nbsp;&nbsp;
	- **主要途径** -> 苯丙氨酸 →(==**苯丙氨酸羟化酶**==)-> ==**酪氨酸**== -> 不可逆 -> 这是苯丙氨酸的主要代谢去路&nbsp;&nbsp;&nbsp;
	- **次要途径** -> 苯丙氨酸 →(转氨酶)→ 苯丙酮酸 → 苯乙酸&nbsp;&nbsp;&nbsp;
	- **苯丙酮酸尿症(PKU)（必考）**&nbsp;&nbsp;&nbsp;
		- 病因 -> 缺乏==**苯丙氨酸羟化酶**== -> 苯丙氨酸不能→酪氨酸&nbsp;&nbsp;&nbsp;
		- 后果 -> 苯丙氨酸堆积 -> 次要途径↑ -> **==苯丙酮酸==**大量生成 -> 尿中排出&nbsp;&nbsp;&nbsp;
		- 临床 -> 常见氨基酸代谢病 -> **智能低下** -> 常染色体隐性遗传&nbsp;&nbsp;&nbsp;
		- 我国发病率 -> 1/16500&nbsp;&nbsp;&nbsp;
		- 防治 -> **新生儿筛查** + 低苯丙氨酸饮食&nbsp;&nbsp;&nbsp;
- **酪氨酸的代谢（三条途径）**&nbsp;&nbsp;&nbsp;
	- **途径1：合成儿茶酚胺**&nbsp;&nbsp;&nbsp;
		- 酪氨酸 →(酪氨酸羟化酶)→ 多巴 → 多巴胺 → 去甲肾上腺素 → **肾上腺素**&nbsp;&nbsp;&nbsp;
		- 部位 -> **肾上腺髓质**和**神经组织**&nbsp;&nbsp;&nbsp;
		- 产物 -> 儿茶酚胺类激素/神经递质&nbsp;&nbsp;&nbsp;
	- **途径2：合成黑色素**&nbsp;&nbsp;&nbsp;
		- 酪氨酸 →(==**酪氨酸酶**==/依赖Cu²⁺)→ 多巴 → **黑色素**&nbsp;&nbsp;&nbsp;
		- 部位 -> 皮肤**黑色素细胞**&nbsp;&nbsp;&nbsp;
		- **白化病(albinism)（必考）**&nbsp;&nbsp;&nbsp;
			- 病因 -> 先天性缺乏==**酪氨酸酶**== -> 不能合成黑色素&nbsp;&nbsp;&nbsp;
			- 表现 -> **==皮肤、毛发发白==** / 畏光&nbsp;&nbsp;&nbsp;
	- **途径3：分解代谢**&nbsp;&nbsp;&nbsp;
		- 酪氨酸 → 尿黑酸 →(==**尿黑酸氧化酶**==)-> 进一步分解&nbsp;&nbsp;&nbsp;
		- **尿黑酸尿症(alkaptonuria)**&nbsp;&nbsp;&nbsp;
			- 病因 -> 先天缺乏==**尿黑酸氧化酶**== -> 尿黑酸堆积&nbsp;&nbsp;&nbsp;
			- 表现 -> 尿液暴露空气中变**==黑色==** / 褐黄病（结缔组织色素沉着）&nbsp;&nbsp;&nbsp;
- **色氨酸的代谢**&nbsp;&nbsp;&nbsp;
	- 分解代谢 -> 色氨酸 → 犬尿氨酸 → 进一步分解&nbsp;&nbsp;&nbsp;
	- 生成**5-羟色胺(5-HT)** -> 抑制性神经递质&nbsp;&nbsp;&nbsp;
	- 生成**一碳单位**（N¹⁰-甲酰FH₄）&nbsp;&nbsp;&nbsp;
	- 可合成少量**烟酸（维生素PP）** -> 约60mg色氨酸→1mg烟酸&nbsp;&nbsp;&nbsp;
- **芳香族氨基酸缺陷病速记**&nbsp;&nbsp;&nbsp;
	- ==**PKU**== -> 缺==**苯丙氨酸羟化酶**== -> 苯丙氨酸↑+苯丙酮酸↑ -> **智力低下**&nbsp;&nbsp;&nbsp;
	- ==**白化病**== -> 缺==**酪氨酸酶**== -> 黑色素↓ -> **皮肤毛发白**&nbsp;&nbsp;&nbsp;
	- ==**尿黑酸尿症**== -> 缺==**尿黑酸氧化酶**== -> 尿黑酸↑ -> **尿变黑**&nbsp;&nbsp;&nbsp;
- **支链氨基酸代谢（了解）**&nbsp;&nbsp;&nbsp;
	- 三种必需氨基酸 -> **亮**(生酮) + **异亮**(生糖兼酮) + **缬**(生糖)&nbsp;&nbsp;&nbsp;
	- 代谢特点 -> 主要在**肌肉**中转氨基 -> 不主要在肝&nbsp;&nbsp;&nbsp;
	- 支链氨基酸酮酸制剂 -> 有**降血氨**作用&nbsp;&nbsp;&nbsp;
