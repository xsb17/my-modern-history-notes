# 氨基酸的一般代谢（脱氨基）&nbsp;&nbsp;&nbsp;

- **氨基酸脱氨基作用概述**&nbsp;&nbsp;&nbsp;
	- 三种方式 -> 转氨作用 / 氧化脱氨 / 联合脱氨&nbsp;&nbsp;&nbsp;
	- **联合脱氨** -> 体内氨基酸脱氨基的**==主要方式==**&nbsp;&nbsp;&nbsp;
- **转氨作用(transamination)**&nbsp;&nbsp;&nbsp;
	- 定义 -> α-氨基酸的氨基 -> 转移到α-酮酸 -> 生成新氨基酸+新α-酮酸&nbsp;&nbsp;&nbsp;
	- 辅酶 -> ==**磷酸吡哆醛**==（==**维生素B₆**==的衍生物）-> 传递氨基&nbsp;&nbsp;&nbsp;
	- 两个临床最重要转氨酶&nbsp;&nbsp;&nbsp;
		- **ALT（丙氨酸转氨酶/GPT）** -> 谷氨酸+丙酮酸 ⇌ α-酮戊二酸+丙氨酸&nbsp;&nbsp;&nbsp;
		- **AST（天冬氨酸转氨酶/GOT）** -> 谷氨酸+草酰乙酸 ⇌ α-酮戊二酸+天冬氨酸&nbsp;&nbsp;&nbsp;
	- 临床应用&nbsp;&nbsp;&nbsp;
		- 血清**ALT↑** -> 诊断==**急性肝炎**==（ALT在肝细胞中含量高）&nbsp;&nbsp;&nbsp;
		- 血清**AST↑** -> 诊断==**心肌梗塞**==（AST在心肌中含量高）&nbsp;&nbsp;&nbsp;
	- 转氨作用特点 -> **可逆** / 不产生游离氨&nbsp;&nbsp;&nbsp;
	- 生理意义 -> 多数氨基酸脱氨 / **==合成非必需氨基酸==** / 联系糖脂代谢&nbsp;&nbsp;&nbsp;
- **氧化脱氨作用**&nbsp;&nbsp;&nbsp;
	- **==L-谷氨酸==** -> 哺乳动物**==唯一==**能高速率氧化脱氨的氨基酸&nbsp;&nbsp;&nbsp;
	- 关键酶 -> ==**L-谷氨酸脱氢酶**== -> 存在于**肝、脑、肾**&nbsp;&nbsp;&nbsp;
	- 辅酶 -> **NAD⁺ 或 NADP⁺**&nbsp;&nbsp;&nbsp;
	- 反应 -> L-谷氨酸 + NAD(P)⁺ → α-酮戊二酸 + **NH₃** + NAD(P)H&nbsp;&nbsp;&nbsp;
	- 调节 -> **==GTP/ATP抑制==** / **==GDP/ADP激活==**&nbsp;&nbsp;&nbsp;
	- 特点 -> 可逆 / 产生游离氨&nbsp;&nbsp;&nbsp;
- **联合脱氨基作用（主要脱氨方式）**&nbsp;&nbsp;&nbsp;
	- 定义 -> 转氨 + 氧化脱氨联合 -> 氨基酸 → α-酮酸 + NH₃&nbsp;&nbsp;&nbsp;
	- 两种类型&nbsp;&nbsp;&nbsp;
		- **转氨酶 + L-谷氨酸脱氢酶联合** -> **==肝、肾、脑==**中&nbsp;&nbsp;&nbsp;
		- **嘌呤核苷酸循环** -> **==骨骼肌、心肌==**中&nbsp;&nbsp;&nbsp;
	- 过程 -> 氨基酸 + α-酮戊二酸 →(转氨酶)→ α-酮酸 + 谷氨酸 →(谷氨酸脱氢酶)→ α-酮戊二酸 + **==NH₃==**&nbsp;&nbsp;&nbsp;
	- 生理意义&nbsp;&nbsp;&nbsp;
		- 氨基酸脱氨基的**==主要方式==**&nbsp;&nbsp;&nbsp;
		- 体内合成**==非必需氨基酸==**的主要方式&nbsp;&nbsp;&nbsp;
		- 可逆反应&nbsp;&nbsp;&nbsp;
- **α-酮酸的代谢去路（三条）**&nbsp;&nbsp;&nbsp;
	- 彻底**==氧化分解==**供能&nbsp;&nbsp;&nbsp;
	- 重新氨基化 -> 合成**==非必需氨基酸==**&nbsp;&nbsp;&nbsp;
	- 转变为**==糖或酮体==**（联系三大代谢）&nbsp;&nbsp;&nbsp;
- **氨基酸按代谢产物分类（必背）**&nbsp;&nbsp;&nbsp;
	- **生糖氨基酸** -> **==13种==** -> 代谢中间物：丙酮酸/α-酮戊二酸/草酰乙酸/琥珀酸&nbsp;&nbsp;&nbsp;
	- **生酮氨基酸** -> **==2种==** -> **==亮氨酸、赖氨酸==** -> 口诀："**同样来**"（酮-亮-赖）&nbsp;&nbsp;&nbsp;
	- **生糖兼生酮氨基酸** -> **==5种==** -> **==异亮、苯丙、酪、色、苏==** -> 口诀："**一本落色书**"（异-苯-酪-色-苏）&nbsp;&nbsp;&nbsp;
