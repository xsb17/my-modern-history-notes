# 氨的去路与尿素循环（绝对重点）&nbsp;&nbsp;&nbsp;

- **尿素循环概述**&nbsp;&nbsp;&nbsp;
	- 氨的**最主要去路** -> 在**==肝==**内合成**==尿素==**&nbsp;&nbsp;&nbsp;
	- 尿素循环 = **鸟氨酸循环** = Krebs-Henseleit循环&nbsp;&nbsp;&nbsp;
	- 合成部位 -> **==肝==**(主要) / 肾及脑(次要)&nbsp;&nbsp;&nbsp;
	- 亚细胞定位 -> **==线粒体 + 胞液==**&nbsp;&nbsp;&nbsp;
- **尿素合成五步反应**&nbsp;&nbsp;&nbsp;
	- **第1步（线粒体）：氨基甲酰磷酸的合成**&nbsp;&nbsp;&nbsp;
		- NH₃ + CO₂ + **==2ATP==** → 氨基甲酰磷酸&nbsp;&nbsp;&nbsp;
		- 关键酶 -> **==氨基甲酰磷酸合成酶Ⅰ(CPS-Ⅰ)==** -> 尿素循环**==启动关键酶==**&nbsp;&nbsp;&nbsp;
		- 激活剂 -> **==N-乙酰谷氨酸(AGA)==**&nbsp;&nbsp;&nbsp;
	- **第2步（线粒体）：瓜氨酸的合成**&nbsp;&nbsp;&nbsp;
		- 氨基甲酰磷酸 + **鸟氨酸** → 瓜氨酸&nbsp;&nbsp;&nbsp;
		- 酶 -> 鸟氨酸氨基甲酰转移酶&nbsp;&nbsp;&nbsp;
		- 瓜氨酸 -> 穿出线粒体进入**胞液**&nbsp;&nbsp;&nbsp;
	- **第3步（胞液）：精氨酸代琥珀酸的合成**&nbsp;&nbsp;&nbsp;
		- 瓜氨酸 + **天冬氨酸**（提供第二个氮原子）+ **ATP**&nbsp;&nbsp;&nbsp;
		- 酶 -> ==**精氨酸代琥珀酸合成酶**== -> 活性最低 -> 尿素合成启动后**限速酶**&nbsp;&nbsp;&nbsp;
	- **第4步（胞液）：精氨酸的生成**&nbsp;&nbsp;&nbsp;
		- 精氨酸代琥珀酸 → **精氨酸** + **延胡索酸**&nbsp;&nbsp;&nbsp;
		- 酶 -> 精氨酸代琥珀酸裂解酶&nbsp;&nbsp;&nbsp;
		- 延胡索酸 -> 进入TCA循环&nbsp;&nbsp;&nbsp;
	- **第5步（胞液）：尿素生成**&nbsp;&nbsp;&nbsp;
		- 精氨酸 + H₂O → **尿素** + **鸟氨酸**&nbsp;&nbsp;&nbsp;
		- 酶 -> ==**精氨酸酶**==&nbsp;&nbsp;&nbsp;
		- 鸟氨酸 -> 重新进入线粒体 -> **循环使用**&nbsp;&nbsp;&nbsp;
- **尿素合成关键总结（必背）**&nbsp;&nbsp;&nbsp;
	- 原料 -> ==**2NH₃==（游离氨+天冬氨酸）** + **CO₂**&nbsp;&nbsp;&nbsp;
	- 耗能 -> ==**3ATP**==（4个高能磷酸键）&nbsp;&nbsp;&nbsp;
	- 总反应 -> 2NH₃ + CO₂ + 3ATP + H₂O → **==尿素==** + 2ADP + AMP + 4Pi&nbsp;&nbsp;&nbsp;
	- 尿素分子中两个氮来源 -> ①**==游离氨==**(CPS-Ⅰ催化) ②**==天冬氨酸的α-氨基==**&nbsp;&nbsp;&nbsp;
	- 关键酶&nbsp;&nbsp;&nbsp;
		- ==**CPS-Ⅰ**==（启动酶/线粒体/需AGA激活）&nbsp;&nbsp;&nbsp;
		- ==**精氨酸代琥珀酸合成酶**==（限速酶/胞液）&nbsp;&nbsp;&nbsp;
		- ==**精氨酸酶**==（产生尿素/胞液）&nbsp;&nbsp;&nbsp;
	- 生理意义 -> 把有毒氨转变为**==中性无毒/易溶于水**==的尿素 -> 肾排出 -> **==解除氨毒==**&nbsp;&nbsp;&nbsp;
- **尿素合成的调节**&nbsp;&nbsp;&nbsp;
	- **AGA**激活CPS-Ⅰ -> 启动尿素合成&nbsp;&nbsp;&nbsp;
	- **精氨酸**促进AGA合成 -> 促进尿素合成 -> 临床**用精氨酸治疗高血氨症**&nbsp;&nbsp;&nbsp;
	- 精氨酸代琥珀酸合成酶 -> 活性最低 -> 调节尿素合成速度&nbsp;&nbsp;&nbsp;
	- 高蛋白质膳食 -> 促进尿素合成&nbsp;&nbsp;&nbsp;
- **高血氨症与肝性脑病（肝昏迷）**&nbsp;&nbsp;&nbsp;
	- 病因 -> 肝功能严重受损 / 尿素合成酶遗传缺陷 -> 尿素合成障碍&nbsp;&nbsp;&nbsp;
	- 血氨↑ -> **高血氨症(hyperammonemia)** -> 氨中毒&nbsp;&nbsp;&nbsp;
	- **肝昏迷的氨中毒学说**&nbsp;&nbsp;&nbsp;
		- 血氨↑ -> 氨+α-酮戊二酸 → 谷氨酸 → 谷氨酰胺&nbsp;&nbsp;&nbsp;
		- **α-酮戊二酸↓** -> **TCA循环↓** -> **脑供能不足** -> 昏迷&nbsp;&nbsp;&nbsp;
		- **谷氨酰胺↑**（渗透剂）-> ==**脑水肿**== -> 加重昏迷&nbsp;&nbsp;&nbsp;
	- 诱因 -> 高蛋白饮食/消化道出血/碱性利尿药等&nbsp;&nbsp;&nbsp;
