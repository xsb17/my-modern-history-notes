# 氨的来源与血液转运&nbsp;&nbsp;&nbsp;

- **氨的毒性**&nbsp;&nbsp;&nbsp;
	- 氨是**有毒物质** -> 血氨增加 -> 脑功能紊乱&nbsp;&nbsp;&nbsp;
	- 正常人血氨 -> ≤ ==**60 μmol/L**==&nbsp;&nbsp;&nbsp;
	- 来源与去路保持**动态平衡**&nbsp;&nbsp;&nbsp;
- **体内氨的三大来源**&nbsp;&nbsp;&nbsp;
	- **==氨基酸脱氨基作用==**（主要来源）&nbsp;&nbsp;&nbsp;
		- 氨基酸脱氨 -> 产生**NH₃**&nbsp;&nbsp;&nbsp;
		- 胺类分解（胺氧化酶）-> 也产生 NH₃&nbsp;&nbsp;&nbsp;
	- **肠道吸收**&nbsp;&nbsp;&nbsp;
		- 肠道细菌**腐败作用**产氨 -> 蛋白质+氨基酸分解&nbsp;&nbsp;&nbsp;
		- 肠道**尿素** ->(细菌**尿素酶**)-> NH₃ -> 吸收入血&nbsp;&nbsp;&nbsp;
		- 对策 -> **酸性灌肠** -> 降低pH -> NH₃→NH₄⁺ -> 减少氨吸收&nbsp;&nbsp;&nbsp;
	- **肾脏产氨**&nbsp;&nbsp;&nbsp;
		- 肾小管上皮细胞 -> ==**谷氨酰胺酶**== -> 谷氨酰胺 → 谷氨酸 + **NH₃**&nbsp;&nbsp;&nbsp;
		- 酸性环境 -> NH₃ + H⁺ → NH₄⁺ 排出&nbsp;&nbsp;&nbsp;
		- 临床注意 -> **肝硬化腹水患者不宜用碱性利尿药**（碱性促进NH₃吸收）&nbsp;&nbsp;&nbsp;
- **氨在血液中的转运（无毒形式）**&nbsp;&nbsp;&nbsp;
	- **丙氨酸-葡萄糖循环**（肌肉→肝运氨）&nbsp;&nbsp;&nbsp;
		- 肌肉中 -> 丙酮酸 + NH₃ →(ALT)→ **丙氨酸** -> 入血 -> 肝&nbsp;&nbsp;&nbsp;
		- 肝中 -> 丙氨酸 →(ALT)→ 丙酮酸 + NH₃ -> NH₃进尿素循环 / 丙酮酸→糖异生→**葡萄糖**&nbsp;&nbsp;&nbsp;
		- 葡萄糖 -> 入血 -> 肌肉利用&nbsp;&nbsp;&nbsp;
		- 意义 -> 肌肉氨以无毒**==丙氨酸==**形式运到肝 / 肝为肌肉提供**==葡萄糖==**&nbsp;&nbsp;&nbsp;
	- **谷氨酰胺的运氨作用**（脑、肌肉→肝/肾运氨）&nbsp;&nbsp;&nbsp;
		- 谷氨酸 + **NH₃** →(谷氨酰胺合成酶/消耗ATP)-> **==谷氨酰胺==** -> 入血运输&nbsp;&nbsp;&nbsp;
		- 谷氨酰胺是**==氨的解毒产物==** -> 也是氨的**==储存及运输形式==**&nbsp;&nbsp;&nbsp;
		- 到达肝/肾 -> 谷氨酰胺酶 -> 谷氨酸 + NH₃&nbsp;&nbsp;&nbsp;
		- 临床应用 -> 治疗氨中毒用**谷氨酸钾/钠**（结合氨生成谷氨酰胺）&nbsp;&nbsp;&nbsp;
- **氨的来源与去路小结**&nbsp;&nbsp;&nbsp;
	- **来源（3条）**&nbsp;&nbsp;&nbsp;
		- 氨基酸脱氨 -> **主要来源** -> 全身组织&nbsp;&nbsp;&nbsp;
		- 肠道吸收 -> 蛋白质腐败 + 尿素分解&nbsp;&nbsp;&nbsp;
		- 肾脏泌氨 -> 谷氨酰胺水解&nbsp;&nbsp;&nbsp;
	- **去路（4条）**&nbsp;&nbsp;&nbsp;
		- 肝合成**==尿素==** -> **==最主要去路==**&nbsp;&nbsp;&nbsp;
		- 合成**谷氨酰胺** -> 解毒+储运&nbsp;&nbsp;&nbsp;
		- 合成**非必需氨基酸**+其他含氮物&nbsp;&nbsp;&nbsp;
		- 肾形成**铵盐**排出&nbsp;&nbsp;&nbsp;
