# 含硫氨基酸代谢&nbsp;&nbsp;&nbsp;

- **含硫氨基酸家族**&nbsp;&nbsp;&nbsp;
	- 三种 -> **甲硫氨酸(蛋氨酸)** / ==**半胱氨酸**== / **胱氨酸**&nbsp;&nbsp;&nbsp;
	- 甲硫氨酸 -> **必需氨基酸**&nbsp;&nbsp;&nbsp;
	- 半胱氨酸 ⇌ 胱氨酸（巯基⇌二硫键互变）&nbsp;&nbsp;&nbsp;
- **甲硫氨酸(蛋氨酸)代谢 — SAM为核心**&nbsp;&nbsp;&nbsp;
	- 活化形式 -> **==S-腺苷甲硫氨酸(SAM)==**&nbsp;&nbsp;&nbsp;
		- 甲硫氨酸 + ATP → SAM&nbsp;&nbsp;&nbsp;
		- SAM的甲基**高度活化** -> 称为**活性甲基**&nbsp;&nbsp;&nbsp;
		- SAM -> 体内**==最主要的甲基供体==**&nbsp;&nbsp;&nbsp;
	- **SAM的转甲基作用**&nbsp;&nbsp;&nbsp;
		- SAM + 受体 →(甲基转移酶)→ S-腺苷同型半胱氨酸 + 甲基化产物&nbsp;&nbsp;&nbsp;
		- 参与 -> 肾上腺素/肌酸/胆碱/肉碱等合成 / DNA甲基化&nbsp;&nbsp;&nbsp;
- **甲硫氨酸循环(methionine cycle)**&nbsp;&nbsp;&nbsp;
	- 过程&nbsp;&nbsp;&nbsp;
		- 甲硫氨酸 → SAM → S-腺苷同型半胱氨酸 → ==**同型半胱氨酸**==&nbsp;&nbsp;&nbsp;
		- 同型半胱氨酸 + **N⁵-CH₃-FH₄** →(VitB₁₂依赖的甲硫氨酸合酶)→ 甲硫氨酸 + FH₄&nbsp;&nbsp;&nbsp;
	- 两大生理意义&nbsp;&nbsp;&nbsp;
		- 为体内甲基化反应**持续提供甲基** -> 减少甲硫氨酸消耗&nbsp;&nbsp;&nbsp;
		- 使**FH₄再生** -> N⁵-CH₃-FH₄的甲基转移后释放FH₄&nbsp;&nbsp;&nbsp;
	- **VitB₁₂缺乏的后果（必考）**&nbsp;&nbsp;&nbsp;
		- 甲硫氨酸合酶受阻 -> 同型半胱氨酸不能甲基化为甲硫氨酸&nbsp;&nbsp;&nbsp;
		- N⁵-CH₃-FH₄**堆积** -> FH₄不能再生 -> 一碳单位代谢障碍&nbsp;&nbsp;&nbsp;
		- -> dTMP↓ -> DNA合成↓ -> **==巨幼红细胞性贫血==**&nbsp;&nbsp;&nbsp;
		- 同型半胱氨酸堆积 -> **==高同型半胱氨酸血症==** -> **==心血管疾病危险因子==**&nbsp;&nbsp;&nbsp;
- **半胱氨酸与胱氨酸代谢**&nbsp;&nbsp;&nbsp;
	- **半胱氨酸 ⇌ 胱氨酸**（-SH ⇌ -S-S-）&nbsp;&nbsp;&nbsp;
		- 二硫键对维持**蛋白质构象**至关重要 -> 如**胰岛素**&nbsp;&nbsp;&nbsp;
		- 半胱氨酸巯基(-SH)是许多**巯基酶**的活性基团&nbsp;&nbsp;&nbsp;
		- **路易士毒气**(有机砷) -> 与巯基结合 -> 酶失活&nbsp;&nbsp;&nbsp;
	- **半胱氨酸 -> 牛磺酸**&nbsp;&nbsp;&nbsp;
		- 半胱氨酸→(氧化)→磺酸丙氨酸→(脱羧)→牛磺酸&nbsp;&nbsp;&nbsp;
		- 牛磺酸 -> 促进脑发育/提高神经传导/抑制性神经递质&nbsp;&nbsp;&nbsp;
	- **半胱氨酸 -> 活性硫酸根(PAPS)**&nbsp;&nbsp;&nbsp;
		- 半胱氨酸 -> ==**PAPS**==(3′-磷酸腺苷-5′-磷酸硫酸)&nbsp;&nbsp;&nbsp;
		- PAPS = **==活性硫酸根==** -> 体内硫酸基供体 -> 参与**==肝生物转化==**&nbsp;&nbsp;&nbsp;
- **含硫氨基酸与临床**&nbsp;&nbsp;&nbsp;
	- **同型半胱氨酸↑** -> 心血管疾病**独立危险因子**&nbsp;&nbsp;&nbsp;
	- VitB₁₂/叶酸缺乏 -> 同型半胱氨酸↑ + 巨幼贫&nbsp;&nbsp;&nbsp;
	- 甲硫氨酸 -> 参与**肌酸**合成（肌酸+ATP⇌磷酸肌酸）&nbsp;&nbsp;&nbsp;
