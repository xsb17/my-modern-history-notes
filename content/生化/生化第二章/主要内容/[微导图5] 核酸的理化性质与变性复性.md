# 核酸的理化性质与变性复性 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★核酸紫外吸收特性&nbsp;&nbsp;&nbsp;
  - 吸收峰：**260nm**（碱基共轭双键）&nbsp;&nbsp;&nbsp;
  - 定量：A₂₆₀=1.0 ≈ **50μg/ml**双链DNA | **40μg/ml** RNA | **20μg/ml**寡核苷酸&nbsp;&nbsp;&nbsp;
  - 纯度判断：纯DNA -> **A₂₆₀/A₂₈₀=1.8** | 纯RNA -> **A₂₆₀/A₂₈₀=2.0**&nbsp;&nbsp;&nbsp;
  - 对比蛋白质：蛋白质吸收峰在**280nm**&nbsp;&nbsp;&nbsp;
- ★DNA变性（Denaturation）&nbsp;&nbsp;&nbsp;
  - 定义：理化因素 -> **氢键+碱基堆积力断裂** -> 双链解为单链&nbsp;&nbsp;&nbsp;
  - 变性因素：**加热**（最常用）/强酸强碱/尿素/甲酰胺&nbsp;&nbsp;&nbsp;
  - 本质：**次级键断裂**/空间结构改变，**一级结构不变**（不涉及共价键）&nbsp;&nbsp;&nbsp;
  - 区别于降解：降解 = **共价键断裂**&nbsp;&nbsp;&nbsp;
  - ★两大标志现象：&nbsp;&nbsp;&nbsp;
    - **增色效应**：260nm紫外吸收**升高**（解链暴露更多碱基）&nbsp;&nbsp;&nbsp;
    - 溶液粘度**降低**&nbsp;&nbsp;&nbsp;
  - ★Tm（解链温度/融解温度）：&nbsp;&nbsp;&nbsp;
    - 定义：紫外吸收增加值达最大值**一半**时的温度&nbsp;&nbsp;&nbsp;
    - Tm与**G-C含量**正相关（G≡C三个氢键更稳定 -> 需更高温）&nbsp;&nbsp;&nbsp;
    - Tm与DNA**均一性**正相关&nbsp;&nbsp;&nbsp;
- ★DNA复性（Renaturation）&nbsp;&nbsp;&nbsp;
  - 定义：变性DNA在适当条件 -> 按碱基配对**重新形成双螺旋**&nbsp;&nbsp;&nbsp;
  - **退火**：热变性DNA**缓慢冷却**后复性&nbsp;&nbsp;&nbsp;
  - 两大标志现象：&nbsp;&nbsp;&nbsp;
    - **减色效应**：260nm吸收**降低**&nbsp;&nbsp;&nbsp;
    - 溶液粘度**升高**&nbsp;&nbsp;&nbsp;
