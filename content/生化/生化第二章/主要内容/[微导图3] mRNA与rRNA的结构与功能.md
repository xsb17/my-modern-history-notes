# mRNA与rRNA的结构与功能 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★mRNA（信使RNA）&nbsp;&nbsp;&nbsp;
  - 含量：总RNA的**1~5%**（最少），但种类最多&nbsp;&nbsp;&nbsp;
  - 功能：**蛋白质合成的模板**（每三个核苷酸=一个密码子）&nbsp;&nbsp;&nbsp;
  - 前体：**hnRNA**（核内不均一RNA）-> 含内含子/外显子 -> 剪切加工 -> 成熟mRNA&nbsp;&nbsp;&nbsp;
  - ★真核成熟mRNA三大结构特征：&nbsp;&nbsp;&nbsp;
    - ① **5'端帽子结构**：**m⁷GpppN**（7-甲基鸟嘌呤-三磷酸核苷）&nbsp;&nbsp;&nbsp;
      - 功能：维持mRNA**稳定** + 协助**核输出** + **翻译起始**识别核糖体&nbsp;&nbsp;&nbsp;
    - ② **3'端Poly-A尾**：20~250个腺苷酸&nbsp;&nbsp;&nbsp;
      - 功能：与poly(A)结合蛋白结合 -> 维持mRNA**稳定性**&nbsp;&nbsp;&nbsp;
    - ③ 编码区：从AUG起始 -> 每三联体密码编码一个AA -> **单顺反子**&nbsp;&nbsp;&nbsp;
- ★rRNA（核糖体RNA）&nbsp;&nbsp;&nbsp;
  - 含量：细胞内**最多**的RNA（>**80%**）&nbsp;&nbsp;&nbsp;
  - 存在形式：**核糖体** = 60% rRNA + 40% 蛋白质&nbsp;&nbsp;&nbsp;
  - 功能：核糖体是**蛋白质合成的场所**&nbsp;&nbsp;&nbsp;
  - 原核核糖体：**70S** = 30S小亚基(16S rRNA) + 50S大亚基(23S/5S rRNA)&nbsp;&nbsp;&nbsp;
  - 真核核糖体：**80S** = 40S小亚基(18S rRNA) + 60S大亚基(28S/5.8S/5S rRNA)&nbsp;&nbsp;&nbsp;
  - 结构：单链回折 -> 局部螺旋区和突环&nbsp;&nbsp;&nbsp;
- mRNA vs rRNA 对比&nbsp;&nbsp;&nbsp;
  - mRNA：**最少**(1~5%)/模板/5'帽+3'polyA&nbsp;&nbsp;&nbsp;
  - rRNA：**最多**(>80%)/核糖体组分/蛋白合成场所&nbsp;&nbsp;&nbsp;
