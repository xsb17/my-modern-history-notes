# 核酸的化学组成 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★DNA vs RNA 根本差异&nbsp;&nbsp;&nbsp;
  - 戊糖：DNA = **脱氧核糖**（2'-H）| RNA = **核糖**（2'-OH）&nbsp;&nbsp;&nbsp;
  - 碱基：DNA = **A/T/G/C** | RNA = **A/U/G/C**（T→U替换）&nbsp;&nbsp;&nbsp;
  - 链形式：DNA 多为**双链** | RNA 多为**单链**&nbsp;&nbsp;&nbsp;
  - 分子量：DNA 大（数千~上亿 bp）| RNA 小（几十~几千 bp）&nbsp;&nbsp;&nbsp;
- 化学组成三层次&nbsp;&nbsp;&nbsp;
  - 碱基 + 戊糖 -> **核苷**（β-N-糖苷键）&nbsp;&nbsp;&nbsp;
  - 核苷 + 磷酸 -> **核苷酸**（酯键）&nbsp;&nbsp;&nbsp;
  - 多个核苷酸 -> **多聚核苷酸链**（3',5'-磷酸二酯键）&nbsp;&nbsp;&nbsp;
- ★碱基分类&nbsp;&nbsp;&nbsp;
  - 嘌呤：**A(腺嘌呤)、G(鸟嘌呤)** -> 双环结构 -> DNA和RNA均有&nbsp;&nbsp;&nbsp;
  - 嘧啶：**C(胞嘧啶)**->DNA/RNA均有；**T(胸腺嘧啶)**->DNA；**U(尿嘧啶)**->RNA&nbsp;&nbsp;&nbsp;
- ★多聚核苷酸链骨架&nbsp;&nbsp;&nbsp;
  - 连接键：**3',5'-磷酸二酯键**&nbsp;&nbsp;&nbsp;
  - 方向：**5'端→3'端**（5'磷酸基团+3'羟基）&nbsp;&nbsp;&nbsp;
  - 骨架：磷酸+戊糖在外侧（亲水），碱基在内侧&nbsp;&nbsp;&nbsp;
- 重要核苷酸衍生物&nbsp;&nbsp;&nbsp;
  - 能量：**ATP**（能量货币）| GTP/CTP/UTP -> RNA合成原料&nbsp;&nbsp;&nbsp;
  - 第二信使：**cAMP、cGMP**&nbsp;&nbsp;&nbsp;
  - 辅酶：**NAD⁺、NADP⁺、FAD**&nbsp;&nbsp;&nbsp;
