# 各组织器官的特定燃料&nbsp;&nbsp;&nbsp;

- **==肝==  -> 人体代谢==中心枢纽==**&nbsp;&nbsp;&nbsp;
	- 糖代谢 -> **==糖异生的主要器官==** / 合成储存糖原 / 维持血糖稳定&nbsp;&nbsp;&nbsp;
	- 脂代谢 -> 脂肪酸 β-氧化 / 酮体生成 / 脂肪合成 + VLDL 输出&nbsp;&nbsp;&nbsp;
	- 蛋白代谢 -> 氨基酸脱氨 / 尿素合成 / 血浆蛋白合成&nbsp;&nbsp;&nbsp;
- **==脑==  -> 主要依赖葡萄糖**&nbsp;&nbsp;&nbsp;
	- 正常状态 -> 主要能源：**==葡萄糖==** -> 耗能大、耗氧多&nbsp;&nbsp;&nbsp;
	- **不能利用脂肪酸**（脂酸不能通过血脑屏障）&nbsp;&nbsp;&nbsp;
	- 长期饥饿/糖供应不足 -> 改用 **==酮体==** 供能&nbsp;&nbsp;&nbsp;
- **==心肌==  -> 优先用脂肪酸**&nbsp;&nbsp;&nbsp;
	- 正常 -> 优先以 **==脂肪酸==** 为燃料产生 ATP&nbsp;&nbsp;&nbsp;
	- 可利用多种能源 -> 脂肪酸 / 葡萄糖 / 酮体 / 乳酸&nbsp;&nbsp;&nbsp;
	- 代谢方式 -> **==有氧氧化==** 为主&nbsp;&nbsp;&nbsp;
- **==成熟红细胞==  -> 只靠糖酵解**&nbsp;&nbsp;&nbsp;
	- 缺乏 **==线粒体==** -> 无法进行有氧氧化&nbsp;&nbsp;&nbsp;
	- 唯一供能途径 -> **==糖无氧氧化 (糖酵解)==**&nbsp;&nbsp;&nbsp;
	- 无细胞核+无细胞器 -> 不能合成蛋白质和核酸&nbsp;&nbsp;&nbsp;
- **==骨骼肌==  -> 安静用脂肪，剧烈用糖**&nbsp;&nbsp;&nbsp;
	- 安静状态 -> 主要依靠 **==脂肪酸氧化==** 供能&nbsp;&nbsp;&nbsp;
	- 剧烈运动 -> 切换到 **==糖酵解==** -> 产生大量乳酸&nbsp;&nbsp;&nbsp;
	- 乳酸 → 入血 → 肝 → **==乳酸循环(Cori循环)==** → 糖异生为葡萄糖 → 回肌肉&nbsp;&nbsp;&nbsp;
	- 储存 **==肌糖原==**（不能直接补充血糖→缺葡糖-6-磷酸酶）&nbsp;&nbsp;&nbsp;
- **==脂肪组织==  -> 储能与动员**&nbsp;&nbsp;&nbsp;
	- 合成及储存 **==脂肪==** 的重要组织&nbsp;&nbsp;&nbsp;
	- 饥饿时 -> 脂肪分解 → **==脂肪酸 + 甘油==** → 供全身组织利用&nbsp;&nbsp;&nbsp;
- **==肾==  -> 糖异生与酮体生成**&nbsp;&nbsp;&nbsp;
	- 肾髓质 -> 无线粒体 -> **==无氧氧化==** 供能&nbsp;&nbsp;&nbsp;
	- 肾皮质 -> **==脂肪酸及酮体==** 供能&nbsp;&nbsp;&nbsp;
	- 长期饥饿 -> **==肾糖异生==** 极度增强 -> 几乎与**肝**相等&nbsp;&nbsp;&nbsp;
	- 生成谷氨酰胺 -> 氨的储存/运输/解毒 / 调节酸碱平衡&nbsp;&nbsp;&nbsp;
- **各器官燃料速记口诀**&nbsp;&nbsp;&nbsp;
	- **脑** -> **==糖主酮备==**（葡萄糖为主，饥饿换酮体）&nbsp;&nbsp;&nbsp;
	- **心** -> **==脂主多能==**（脂肪酸为主，啥都能用）&nbsp;&nbsp;&nbsp;
	- **红细胞** -> **==只酵解==**（唯一糖酵解，无线粒体）&nbsp;&nbsp;&nbsp;
	- **肌肉** -> **==静脂动糖==**（安静用脂，运动酵解）&nbsp;&nbsp;&nbsp;
	- **肝** -> **==全才总管==**（代谢中心，啥都能干）&nbsp;&nbsp;&nbsp;
	- **肾** -> **==饿时帮忙==**（长期饥饿时糖异生≈肝）&nbsp;&nbsp;&nbsp;
