# 应激状态下的代谢改变&nbsp;&nbsp;&nbsp;

- **应激的定义**&nbsp;&nbsp;&nbsp;
	- 机体应对内外环境刺激（中毒/感染/创伤/疼痛/恐惧/大运动量）的非特异性反应&nbsp;&nbsp;&nbsp;
	- 可以是**一过性**或**持续性**&nbsp;&nbsp;&nbsp;
- **应激时的神经-激素改变**&nbsp;&nbsp;&nbsp;
	- **==交感神经兴奋==** -> 肾上腺髓质 + 皮质激素分泌增多&nbsp;&nbsp;&nbsp;
	- **升糖激素全面升高**&nbsp;&nbsp;&nbsp;
		- **==肾上腺素 / 去甲肾上腺素==** ↑&nbsp;&nbsp;&nbsp;
		- **==糖皮质激素（皮质醇）==** ↑&nbsp;&nbsp;&nbsp;
		- **==胰高血糖素==** ↑&nbsp;&nbsp;&nbsp;
		- **==生长激素==** ↑&nbsp;&nbsp;&nbsp;
	- **降糖激素** -> **==胰岛素分泌减少==**&nbsp;&nbsp;&nbsp;
	- HPA 轴激活 -> 下丘脑-垂体-肾上腺&nbsp;&nbsp;&nbsp;
- **代谢改变1：血糖显著升高**&nbsp;&nbsp;&nbsp;
	- 肝糖原分解 **==↑==** + 肝糖异生 **==↑==**&nbsp;&nbsp;&nbsp;
	- 肝糖原合成 **==↓==**&nbsp;&nbsp;&nbsp;
	- 目的 -> 保证**脑和红细胞**的葡萄糖供应&nbsp;&nbsp;&nbsp;
	- 结果 -> 血糖 **==显著升高==**&nbsp;&nbsp;&nbsp;
- **代谢改变2：脂肪动员增强**&nbsp;&nbsp;&nbsp;
	- 脂肪分解 **==↑==** -> 脂肪酸释出 **==↑==**&nbsp;&nbsp;&nbsp;
	- 目的 -> 为**心肌、骨骼肌、肾**等组织供能&nbsp;&nbsp;&nbsp;
	- 结果 -> 血中游离脂肪酸 **==升高==**&nbsp;&nbsp;&nbsp;
- **代谢改变3：蛋白质分解加强**&nbsp;&nbsp;&nbsp;
	- 肌蛋白质分解 **==↑==** -> 释出**丙氨酸**等氨基酸&nbsp;&nbsp;&nbsp;
	- 目的 -> 为**糖异生提供原料**（氨基酸→葡萄糖）&nbsp;&nbsp;&nbsp;
	- 结果 -> ==负氮平衡== / 体重可能减轻&nbsp;&nbsp;&nbsp;
- **应激时三大代谢总趋势**&nbsp;&nbsp;&nbsp;
	- **==分解 > 合成==**（整体呈分解代谢加强）&nbsp;&nbsp;&nbsp;
	- 增加糖、脂、蛋白质分解的能源供应&nbsp;&nbsp;&nbsp;
	- 限制能源存积&nbsp;&nbsp;&nbsp;
- **应激 vs 饥饿 关键区别**&nbsp;&nbsp;&nbsp;
	- **==应激==** -> 激素全面升（肾上腺素+皮质醇+胰高血糖素↑↑ + 胰岛素↓）-> **血糖↑** / 分解全面↑&nbsp;&nbsp;&nbsp;
	- **==饥饿(短期)==** -> 主要是胰高血糖素↑ + 胰岛素↓ -> 糖异生↑ + 脂肪动员↑&nbsp;&nbsp;&nbsp;
	- **==饥饿(长期)==** -> 脑改用酮体 / 蛋白质分解减少（保护）&nbsp;&nbsp;&nbsp;
- **应激的临床意义**&nbsp;&nbsp;&nbsp;
	- 短期应激 -> 适应性的保护反应（为机体提供能源）&nbsp;&nbsp;&nbsp;
	- 长期/持续应激 -> **==代谢紊乱==** -> 高血糖、消瘦、免疫功能下降&nbsp;&nbsp;&nbsp;
