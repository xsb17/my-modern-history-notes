# 细胞水平调节与关键酶特征&nbsp;&nbsp;&nbsp;

- **代谢调节三级水平**&nbsp;&nbsp;&nbsp;
	- **==细胞水平==** -> 基础/原始调节 -> 对**酶**进行调节 -> 一切调节的基础&nbsp;&nbsp;&nbsp;
	- **==激素水平==** -> 激素经信号转导调节靶细胞代谢&nbsp;&nbsp;&nbsp;
	- **==整体水平==** -> 神经系统主导 -> 整合全身代谢&nbsp;&nbsp;&nbsp;
- **酶的区域化分布（必背！）**&nbsp;&nbsp;&nbsp;
	- **==胞液==**（细胞质）&nbsp;&nbsp;&nbsp;
		- 糖酵解 / 磷酸戊糖途径 / 糖异生 / 糖原合成与分解 -> **==胞液==**&nbsp;&nbsp;&nbsp;
		- 脂肪酸合成 -> **==胞液==**&nbsp;&nbsp;&nbsp;
	- **==线粒体==**&nbsp;&nbsp;&nbsp;
		- 三羧酸循环(TCA) / 氧化磷酸化 -> **==线粒体==**&nbsp;&nbsp;&nbsp;
		- 脂肪酸 β-氧化 -> **==线粒体==**&nbsp;&nbsp;&nbsp;
	- **==内质网==**&nbsp;&nbsp;&nbsp;
		- 磷脂合成 -> **==内质网==**&nbsp;&nbsp;&nbsp;
		- 胆固醇合成 -> **==胞液、内质网==**&nbsp;&nbsp;&nbsp;
	- **==线粒体+胞液==**&nbsp;&nbsp;&nbsp;
		- 尿素合成 -> **==胞液、线粒体==**（两步在线粒体、三步在胞液）&nbsp;&nbsp;&nbsp;
	- **==细胞核==**&nbsp;&nbsp;&nbsp;
		- DNA/RNA 合成 -> **==细胞核==**&nbsp;&nbsp;&nbsp;
- **关键酶（限速酶）的三大特征（必背！）**&nbsp;&nbsp;&nbsp;
	- **特征1** -> 催化速度**==最慢==** -> 决定整个代谢途径总速度 -> 又称**==限速酶==**&nbsp;&nbsp;&nbsp;
	- **特征2** -> 催化**==单向不可逆==**反应 -> 决定代谢途径的**方向**&nbsp;&nbsp;&nbsp;
	- **特征3** -> 受多种代谢物/效应剂调节 -> 受**==别构调节==**&nbsp;&nbsp;&nbsp;
- **酶活性的调节方式**&nbsp;&nbsp;&nbsp;
	- **快速调节（秒~分钟）**&nbsp;&nbsp;&nbsp;
		- **==别构调节==**（变构调节）-> 小分子可逆结合 -> 酶构象改变 -> 活性改变&nbsp;&nbsp;&nbsp;
		- **==化学修饰==**（共价修饰）-> 磷酸化/去磷酸化等可逆修饰&nbsp;&nbsp;&nbsp;
		- 酶原激活 -> 暴露酶活性中心&nbsp;&nbsp;&nbsp;
	- **慢速调节（小时~天）**&nbsp;&nbsp;&nbsp;
		- 酶蛋白合成的**诱导与阻遏**&nbsp;&nbsp;&nbsp;
		- 酶蛋白的**降解**&nbsp;&nbsp;&nbsp;
- **各代谢途径关键酶汇总（全系列）**&nbsp;&nbsp;&nbsp;
	- 糖酵解 -> **==己糖激酶 / PFK-1 / 丙酮酸激酶==**&nbsp;&nbsp;&nbsp;
	- TCA 循环 -> **==柠檬酸合酶 / 异柠檬酸脱氢酶 / α-酮戊二酸脱氢酶复合体==**&nbsp;&nbsp;&nbsp;
	- 糖异生 -> **==丙酮酸羧化酶 / PEPCK / 果糖二磷酸酶-1 / 葡糖-6-磷酸酶==**&nbsp;&nbsp;&nbsp;
	- 糖原合成 -> **==糖原合酶==**&nbsp;&nbsp;&nbsp;
	- 糖原分解 -> **==糖原磷酸化酶==**&nbsp;&nbsp;&nbsp;
	- 脂肪酸 β-氧化 -> **==肉碱脂酰转移酶Ⅰ==**&nbsp;&nbsp;&nbsp;
	- 脂肪酸合成 -> **==乙酰CoA 羧化酶==**&nbsp;&nbsp;&nbsp;
	- 胆固醇合成 -> **==HMG-CoA 还原酶==**&nbsp;&nbsp;&nbsp;
	- 酮体生成 -> **==HMG-CoA 合酶==**&nbsp;&nbsp;&nbsp;
