# [全集存档] 生化第九章：物质代谢的联系与调节&nbsp;&nbsp;&nbsp;

## 第一节 物质代谢的特点&nbsp;&nbsp;&nbsp;

- ==整体性==：各种物质代谢同时进行、相互联系、彼此转变&nbsp;&nbsp;&nbsp;
- ==代谢调节==：维持动态平衡&nbsp;&nbsp;&nbsp;
- 各组织、器官物质代谢各具特色&nbsp;&nbsp;&nbsp;
- 各种代谢物均具有共同的==代谢池==&nbsp;&nbsp;&nbsp;
- **==ATP==**是机体能量利用的共同形式&nbsp;&nbsp;&nbsp;
- **==NADPH==**是合成代谢所需的还原当量&nbsp;&nbsp;&nbsp;
- 代谢途径多样性：直线途径、分支途径、循环途径&nbsp;&nbsp;&nbsp;

## 第二节 物质代谢的相互联系&nbsp;&nbsp;&nbsp;

### 一、能量代谢的相互联系和制约&nbsp;&nbsp;&nbsp;

- 三大营养物质（糖、脂、蛋白质）可在体内氧化分解供能，具有共同代谢途径&nbsp;&nbsp;&nbsp;
- 从能量供应角度，三者可互相代替、互相制约&nbsp;&nbsp;&nbsp;
- 机体优先利用燃料的次序：**==糖 → 脂肪 → 蛋白质==**&nbsp;&nbsp;&nbsp;
- 供能以糖及脂为主，尽量节约蛋白质消耗&nbsp;&nbsp;&nbsp;

### 二、代谢过程中间产物的相互联系&nbsp;&nbsp;&nbsp;

#### 1. 糖代谢与脂代谢的相互联系&nbsp;&nbsp;&nbsp;

- **糖可转变为脂肪**：糖摄入过多时，糖 → 乙酰CoA → 脂肪酸 → 甘油三酯&nbsp;&nbsp;&nbsp;
- **脂肪的甘油部分可转变为糖**：甘油 → 磷酸二羟丙酮 → 糖异生 → 葡萄糖（肝、肾、肠含甘油激酶）&nbsp;&nbsp;&nbsp;
- **==脂肪酸不能净生成糖==**：脂肪酸 → 乙酰CoA，动物体内乙酰CoA不能逆行生成丙酮酸&nbsp;&nbsp;&nbsp;
- **脂肪分解受糖代谢影响**：糖供应不足时→草酰乙酸减少→TCA受阻→酮体生成增加&nbsp;&nbsp;&nbsp;

#### 2. 糖代谢与氨基酸代谢的相互联系&nbsp;&nbsp;&nbsp;

- **氨基酸可转变为糖**（除==亮氨酸、赖氨酸==两种生酮氨基酸外）：脱氨基 → α-酮酸 → 糖异生 → 葡萄糖&nbsp;&nbsp;&nbsp;
  - 丙氨酸 → 丙酮酸 → 葡萄糖&nbsp;&nbsp;&nbsp;
  - 天冬氨酸 → 草酰乙酸 → 葡萄糖&nbsp;&nbsp;&nbsp;
- **糖可转变为非必需氨基酸**：糖代谢中间物 → 氨基化 → 非必需氨基酸（如丙酮酸→丙氨酸）&nbsp;&nbsp;&nbsp;

#### 3. 脂代谢与氨基酸代谢的相互联系&nbsp;&nbsp;&nbsp;

- **氨基酸可转变为脂肪**：氨基酸 → 乙酰CoA → 脂肪酸 → 脂肪&nbsp;&nbsp;&nbsp;
- 氨基酸可作为合成==磷脂、胆固醇==的原料（如丝氨酸→胆胺→胆碱）&nbsp;&nbsp;&nbsp;
- 脂肪的**甘油部分**可转变为非必需氨基酸&nbsp;&nbsp;&nbsp;
- **==不能说脂类可转变为氨基酸==**（脂肪酸部分不能氨基化）&nbsp;&nbsp;&nbsp;

#### 4. 核苷酸与其他营养物质代谢的相互联系&nbsp;&nbsp;&nbsp;

- 氨基酸是体内合成核苷酸的重要原料&nbsp;&nbsp;&nbsp;
- ==磷酸戊糖途径==为核苷酸合成提供磷酸核糖&nbsp;&nbsp;&nbsp;
- 核酸参与蛋白质生物合成&nbsp;&nbsp;&nbsp;

## 第三节 组织器官的代谢特点&nbsp;&nbsp;&nbsp;

### 一、肝——人体最重要的物质代谢中心和枢纽&nbsp;&nbsp;&nbsp;

- 在糖、脂、蛋白质、水、盐及维生素代谢中均具有独特而重要的作用&nbsp;&nbsp;&nbsp;
- 糖代谢：合成/储存糖原、分解糖原释出葡萄糖、==糖异生的主要器官==&nbsp;&nbsp;&nbsp;
- 肝在维持==血糖稳定==中起重要作用&nbsp;&nbsp;&nbsp;

### 二、心——可利用多种能源物质，以有氧氧化为主&nbsp;&nbsp;&nbsp;

- 正常优先以==脂酸==为燃料产生 ATP&nbsp;&nbsp;&nbsp;
- 可消耗脂酸、葡萄糖、酮体、乳酸等多种能源&nbsp;&nbsp;&nbsp;

### 三、脑——主要利用葡萄糖供能且耗氧量大&nbsp;&nbsp;&nbsp;

- 耗能大、耗氧多&nbsp;&nbsp;&nbsp;
- 正常以==葡萄糖==为主要能源&nbsp;&nbsp;&nbsp;
- **不能利用脂酸**&nbsp;&nbsp;&nbsp;
- 葡萄糖供应不足时→利用==酮体==供能（长期饥饿）&nbsp;&nbsp;&nbsp;

### 四、肌肉——主要氧化脂肪酸，强烈运动产生大量乳酸&nbsp;&nbsp;&nbsp;

- 合成、储存肌糖原&nbsp;&nbsp;&nbsp;
- 通常以==脂酸氧化==为主要供能方式&nbsp;&nbsp;&nbsp;
- 剧烈运动时→以==糖酵解==为主→产生乳酸→经乳酸循环到肝→糖异生&nbsp;&nbsp;&nbsp;

### 五、成熟红细胞——糖无氧氧化是主要供能途径&nbsp;&nbsp;&nbsp;

- ==缺乏线粒体==，只能进行==糖无氧氧化（糖酵解）==&nbsp;&nbsp;&nbsp;
- 这是成熟红细胞的**==唯一供能途径==**&nbsp;&nbsp;&nbsp;

### 六、脂肪组织——合成、储存脂肪的重要组织&nbsp;&nbsp;&nbsp;

- 合成及储存脂肪的重要组织&nbsp;&nbsp;&nbsp;
- 将脂肪分解成脂酸、甘油，供机体其他组织利用&nbsp;&nbsp;&nbsp;

### 七、肾——可进行糖异生和生成酮体的器官&nbsp;&nbsp;&nbsp;

- 肾髓质：主要由==无氧氧化==供能&nbsp;&nbsp;&nbsp;
- 肾皮质：主要由==脂肪酸及酮体==供能&nbsp;&nbsp;&nbsp;
- 肾生成的谷氨酰胺：氨的储存/运输/解毒，调节体液酸碱平衡&nbsp;&nbsp;&nbsp;
- 长期饥饿时==肾糖异生==明显增强，几乎与肝相等&nbsp;&nbsp;&nbsp;

## 第四节 代谢调节&nbsp;&nbsp;&nbsp;

### 代谢调节的三级水平&nbsp;&nbsp;&nbsp;

- **细胞水平**：通过代谢物浓度变化，对酶活性及含量进行调节（原始调节，一切调节的基础）&nbsp;&nbsp;&nbsp;
- **激素水平**：激素通过与靶细胞受体结合，经信号转导发挥调节作用&nbsp;&nbsp;&nbsp;
- **整体水平**：神经系统主导，调节激素释放，整合不同组织器官的代谢&nbsp;&nbsp;&nbsp;

### 一、细胞水平的代谢调节&nbsp;&nbsp;&nbsp;

#### 1. 细胞内酶的区域化分布&nbsp;&nbsp;&nbsp;

- 糖酵解 → ==胞液==&nbsp;&nbsp;&nbsp;
- 三羧酸循环 → ==线粒体==&nbsp;&nbsp;&nbsp;
- 磷酸戊糖途径 → ==胞液==&nbsp;&nbsp;&nbsp;
- 糖异生 → ==胞液==&nbsp;&nbsp;&nbsp;
- 糖原合成与分解 → ==胞液==&nbsp;&nbsp;&nbsp;
- 脂肪酸 β-氧化 → ==线粒体==&nbsp;&nbsp;&nbsp;
- 脂肪酸合成 → ==胞液==&nbsp;&nbsp;&nbsp;
- 胆固醇合成 → ==胞液、内质网==&nbsp;&nbsp;&nbsp;
- 磷脂合成 → ==内质网==&nbsp;&nbsp;&nbsp;
- 尿素合成 → ==胞液、线粒体==&nbsp;&nbsp;&nbsp;
- 氧化磷酸化 → ==线粒体==&nbsp;&nbsp;&nbsp;
- DNA、RNA 合成 → ==细胞核==&nbsp;&nbsp;&nbsp;

#### 2. 关键酶活性的调节&nbsp;&nbsp;&nbsp;

##### 关键酶（限速酶）的定义&nbsp;&nbsp;&nbsp;

- 代谢途径的速度和方向由其中一个或几个具有调节作用的==关键酶==活性所决定&nbsp;&nbsp;&nbsp;

##### 关键酶的特点&nbsp;&nbsp;&nbsp;

- ① **==速度最慢==**，决定整个代谢途径的总速度，又称**限速酶**&nbsp;&nbsp;&nbsp;
- ② 催化**==单向不可逆==**或非平衡反应，决定代谢途径的**方向**&nbsp;&nbsp;&nbsp;
- ③ 活性受多种代谢物或效应剂调节：受**==别构调节==**&nbsp;&nbsp;&nbsp;

##### 酶活性的调节方式&nbsp;&nbsp;&nbsp;

- **快速调节**（数秒~数分钟）：酶活性的调节&nbsp;&nbsp;&nbsp;
  - ==别构调节==（变构调节）：小分子物质可逆结合 → 酶构象改变&nbsp;&nbsp;&nbsp;
  - ==化学修饰==（共价修饰）：磷酸化/去磷酸化等可逆共价修饰&nbsp;&nbsp;&nbsp;
  - 酶原激活：暴露酶活性中心&nbsp;&nbsp;&nbsp;
- **慢速调节**（数小时~数天）：酶含量的调节&nbsp;&nbsp;&nbsp;
  - 酶蛋白合成的诱导与阻遏&nbsp;&nbsp;&nbsp;
  - 酶蛋白的降解&nbsp;&nbsp;&nbsp;

##### 各代谢途径关键酶汇总&nbsp;&nbsp;&nbsp;

- 糖酵解：==己糖激酶、磷酸果糖激酶-1、丙酮酸激酶==&nbsp;&nbsp;&nbsp;
- 三羧酸循环：==柠檬酸合酶、异柠檬酸脱氢酶、α-酮戊二酸脱氢酶复合体==&nbsp;&nbsp;&nbsp;
- 糖异生：==丙酮酸羧化酶、PEPCK、果糖二磷酸酶-1、葡糖-6-磷酸酶==&nbsp;&nbsp;&nbsp;
- 糖原合成：==糖原合酶==&nbsp;&nbsp;&nbsp;
- 糖原分解：==糖原磷酸化酶==&nbsp;&nbsp;&nbsp;
- 脂肪酸 β-氧化：==肉碱脂酰转移酶Ⅰ==&nbsp;&nbsp;&nbsp;
- 脂肪酸合成：==乙酰CoA 羧化酶==&nbsp;&nbsp;&nbsp;
- 胆固醇合成：==HMG-CoA 还原酶==&nbsp;&nbsp;&nbsp;
- 酮体生成：==HMG-CoA 合酶==&nbsp;&nbsp;&nbsp;

### 二、激素水平的代谢调节&nbsp;&nbsp;&nbsp;

#### 激素受体分类&nbsp;&nbsp;&nbsp;

##### 膜受体激素&nbsp;&nbsp;&nbsp;

- 受体位于**==细胞膜表面==**（跨膜糖蛋白）&nbsp;&nbsp;&nbsp;
- 化学本质：**==亲水性==**&nbsp;&nbsp;&nbsp;
- 包括：&nbsp;&nbsp;&nbsp;
  - 肽类激素：==胰岛素、胰高血糖素、生长激素==&nbsp;&nbsp;&nbsp;
  - 儿茶酚胺类：==肾上腺素、去甲肾上腺素==&nbsp;&nbsp;&nbsp;
- 作用方式：与膜受体结合 → 经**第二信使**（cAMP/cGMP 等）传递信号 → 生物学效应&nbsp;&nbsp;&nbsp;

##### 胞内受体激素&nbsp;&nbsp;&nbsp;

- 受体位于**==细胞内==**&nbsp;&nbsp;&nbsp;
- 化学本质：**==疏水性==**&nbsp;&nbsp;&nbsp;
- 包括：&nbsp;&nbsp;&nbsp;
  - ==类固醇激素==（糖皮质激素、性激素等）&nbsp;&nbsp;&nbsp;
  - ==甲状腺素==&nbsp;&nbsp;&nbsp;
  - 前列腺素等&nbsp;&nbsp;&nbsp;
- 作用方式：直接进入细胞 → 与胞内受体结合 → 调控基因表达&nbsp;&nbsp;&nbsp;

### 三、整体水平的代谢调节&nbsp;&nbsp;&nbsp;

#### （一）饱食状态&nbsp;&nbsp;&nbsp;

- 混合膳食：胰岛素中度升高&nbsp;&nbsp;&nbsp;
  - 机体主要分解==葡萄糖==供能&nbsp;&nbsp;&nbsp;
  - 多余葡萄糖 → 合成**肝糖原/肌糖原**保存；部分→肝合成脂肪（VLDL 输出）&nbsp;&nbsp;&nbsp;
- 高糖膳食：胰岛素明显升高，胰高血糖素降低&nbsp;&nbsp;&nbsp;
- 高脂膳食：胰岛素降低，胰高血糖素升高 → 肝糖原分解+糖异生→补充血糖&nbsp;&nbsp;&nbsp;
- 高蛋白膳食：胰岛素中度升高 + 胰高血糖素升高 → 糖异生增加 + 氨基酸转化&nbsp;&nbsp;&nbsp;

#### （二）空腹状态（餐后12小时以上）&nbsp;&nbsp;&nbsp;

- 胰岛素降低，胰高血糖素升高&nbsp;&nbsp;&nbsp;
- 餐后6~8小时：==肝糖原分解==补充血糖&nbsp;&nbsp;&nbsp;
- 餐后16~18小时：肝糖原将尽→==糖异生==补充血糖；脂肪动员中度增加→肝生酮体→供肌组织&nbsp;&nbsp;&nbsp;

#### （三）饥饿状态&nbsp;&nbsp;&nbsp;

##### 短期饥饿（1~3天未进食）&nbsp;&nbsp;&nbsp;

- 供能转变：==葡萄糖→脂肪==为主（脑和红细胞除外仍依赖葡萄糖）&nbsp;&nbsp;&nbsp;
- ==脂肪动员加强==：脂肪酸约25%在肝氧化生酮体&nbsp;&nbsp;&nbsp;
- ==肝糖异生明显增强==（约150g/天）：原料主要来自氨基酸，部分来自乳酸及甘油&nbsp;&nbsp;&nbsp;
- ==骨骼肌蛋白质分解加强==：氨基酸 → 糖异生&nbsp;&nbsp;&nbsp;

##### 长期饥饿（3天以上未进食）&nbsp;&nbsp;&nbsp;

- 脂肪动员进一步加强→大量酮体生成→==脑利用酮体超过葡萄糖==&nbsp;&nbsp;&nbsp;
- ==蛋白质分解减少==（保护肌肉）：氨基酸释出减少&nbsp;&nbsp;&nbsp;
- ==糖异生明显减少==（与短期饥饿相比）：乳酸和甘油成为主要原料&nbsp;&nbsp;&nbsp;
- ==肾糖异生明显增强==，几乎与肝相等&nbsp;&nbsp;&nbsp;

#### （四）应激状态&nbsp;&nbsp;&nbsp;

- 交感神经兴奋，肾上腺髓质/皮质激素分泌增多&nbsp;&nbsp;&nbsp;
- 血浆胰高血糖素、生长激素↑，**==胰岛素分泌减少==**&nbsp;&nbsp;&nbsp;
- 代谢改变：&nbsp;&nbsp;&nbsp;
  - **==血糖升高==**：糖原分解↑ + 糖异生↑，保证脑和红细胞供能&nbsp;&nbsp;&nbsp;
  - **==脂肪动员增强==**：为心肌、骨骼肌及肾供能&nbsp;&nbsp;&nbsp;
  - **==蛋白质分解加强==**：肌释出丙氨酸等→为糖异生提供原料&nbsp;&nbsp;&nbsp;

#### （五）代谢综合征&nbsp;&nbsp;&nbsp;

- 一组以==肥胖、高血糖、高血压、血脂异常==集结发病的临床征候群&nbsp;&nbsp;&nbsp;
- 核心：**胰岛素抵抗**&nbsp;&nbsp;&nbsp;
- 超重和肥胖在 MS 发生发展中起决定性作用&nbsp;&nbsp;&nbsp;
- 肥胖诊断标准：BMI = 体重(kg) / 身高²(m²)；==BMI > 30== 即为肥胖&nbsp;&nbsp;&nbsp;

#### （六）糖尿病代谢紊乱&nbsp;&nbsp;&nbsp;

- I 型：胰岛素**绝对不足**（儿童/青少年，易酮症酸中毒）&nbsp;&nbsp;&nbsp;
- II 型：胰岛素**生物效应降低**（成年人，占 90%以上）&nbsp;&nbsp;&nbsp;
- 代谢紊乱表现：&nbsp;&nbsp;&nbsp;
  - 肌组织：糖原合成↓、分解↑，蛋白质分解↑ → 体重减轻&nbsp;&nbsp;&nbsp;
  - 肝组织：糖原合成↓，糖原分解↑、糖异生↑ → **血糖↑**；脂肪酸氧化↑、酮体生成↑、胆固醇合成↑&nbsp;&nbsp;&nbsp;
  - 血糖超过肾糖阈 → 尿糖&nbsp;&nbsp;&nbsp;
