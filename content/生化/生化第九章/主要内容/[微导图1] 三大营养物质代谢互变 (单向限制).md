# 三大营养物质代谢互变 (单向限制)&nbsp;&nbsp;&nbsp;

- **糖 → 脂肪**&nbsp;&nbsp;&nbsp;
	- 糖摄入过多 -> 葡萄糖 → 乙酰CoA → **==脂肪酸==** → 甘油三酯&nbsp;&nbsp;&nbsp;
	- 关键路径 -> 柠檬酸-丙酮酸循环 → 乙酰CoA 出线粒体 → **胞液**合成脂肪酸&nbsp;&nbsp;&nbsp;
	- **==可以互变==**&nbsp;&nbsp;&nbsp;
- **糖 → 非必需氨基酸**&nbsp;&nbsp;&nbsp;
	- 糖代谢中间物 -> **氨基化** -> 生成非必需氨基酸&nbsp;&nbsp;&nbsp;
	- 例子 -> 丙酮酸 → **==丙氨酸==** / 草酰乙酸 → **==天冬氨酸==** / α-酮戊二酸 → **==谷氨酸==**&nbsp;&nbsp;&nbsp;
	- **==可以互变==**&nbsp;&nbsp;&nbsp;
- **蛋白质 → 糖（糖异生）**&nbsp;&nbsp;&nbsp;
	- 大部分氨基酸（除 **==亮氨酸==** 和 **==赖氨酸==**）-> 脱氨基 → α-酮酸 → **糖异生** → 葡萄糖&nbsp;&nbsp;&nbsp;
	- **生酮氨基酸**（亮/赖）-> **==不能变糖==**&nbsp;&nbsp;&nbsp;
	- **生糖兼生酮氨基酸**（5种：异-苯-酪-色-苏）-> 可以变糖也可以变酮体&nbsp;&nbsp;&nbsp;
- **蛋白质 → 脂肪**&nbsp;&nbsp;&nbsp;
	- 氨基酸 → 乙酰CoA → 脂肪酸 → 甘油三酯&nbsp;&nbsp;&nbsp;
	- 氨基酸 → 丝氨酸 → 胆胺 → 胆碱 → **==磷脂** / **胆固醇==**合成原料&nbsp;&nbsp;&nbsp;
	- **==可以互变==**&nbsp;&nbsp;&nbsp;
- **脂肪 → 糖（关键限制！）**&nbsp;&nbsp;&nbsp;
	- **==甘油部分==** → 甘油 → 磷酸二羟丙酮 → 糖异生 → **==可以变糖==**&nbsp;&nbsp;&nbsp;
	- **==脂肪酸部分==** → 脂肪酸 → 乙酰CoA → **==不能==** 净生成糖&nbsp;&nbsp;&nbsp;
	- 原因 -> 乙酰CoA **无法逆向**生成丙酮酸（丙酮酸脱氢酶催化不可逆反应）&nbsp;&nbsp;&nbsp;
- **脂肪 → 氨基酸**&nbsp;&nbsp;&nbsp;
	- 甘油部分 → **==可以==** 转变为非必需氨基酸&nbsp;&nbsp;&nbsp;
	- 脂肪酸部分 → **==不能==** 转变为氨基酸（动物体内无乙醛酸循环）&nbsp;&nbsp;&nbsp;
	- **==不能说脂类可转变为氨基酸==**&nbsp;&nbsp;&nbsp;
- **机体燃料优先级**&nbsp;&nbsp;&nbsp;
	- **==糖 → 脂肪 → 蛋白质==**（优先消耗糖和脂，节约蛋白质）&nbsp;&nbsp;&nbsp;
- **互变关系速记**&nbsp;&nbsp;&nbsp;
	- **糖 → 脂** -> **==可以==**&nbsp;&nbsp;&nbsp;
	- **糖 → 非必需氨基酸** -> **==可以==**&nbsp;&nbsp;&nbsp;
	- **蛋白质 → 糖** -> **==可以==**（生糖氨基酸）&nbsp;&nbsp;&nbsp;
	- **蛋白质 → 脂** -> **==可以==**&nbsp;&nbsp;&nbsp;
	- **脂(脂肪酸) → 糖** -> **==不可以！==**&nbsp;&nbsp;&nbsp;
	- **脂(脂肪酸) → 氨基酸** -> **==不可以！==**&nbsp;&nbsp;&nbsp;
