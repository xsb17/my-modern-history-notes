# 饥饿状态下的代谢演变 (绝对重点)&nbsp;&nbsp;&nbsp;

- **短期饥饿 (1~3天未进食)**&nbsp;&nbsp;&nbsp;
	- **供能模式转变** -> 从 **==葡萄糖氧化为主 → 脂肪氧化为主==**&nbsp;&nbsp;&nbsp;
		- 例外 -> 脑和红细胞仍依赖葡萄糖（脑不能利用脂肪酸）&nbsp;&nbsp;&nbsp;
	- **脂肪动员** -> **==加强==**&nbsp;&nbsp;&nbsp;
		- 脂肪酸约 **25%** 在肝氧化生成**酮体**&nbsp;&nbsp;&nbsp;
		- 酮体 → 供肌组织等肝外组织利用&nbsp;&nbsp;&nbsp;
	- **肝糖异生** -> **==明显增强==**（约 ==150g/天==）&nbsp;&nbsp;&nbsp;
		- 原料 -> 主要来自 **==氨基酸==**（蛋白质分解）/ 部分来自乳酸+甘油&nbsp;&nbsp;&nbsp;
		- 饥饿 16~36 小时增加最多&nbsp;&nbsp;&nbsp;
	- **蛋白质分解** -> **==加强==**&nbsp;&nbsp;&nbsp;
		- 骨骼肌蛋白质分解略迟于脂肪动员&nbsp;&nbsp;&nbsp;
		- 氨基酸 → 糖异生 → 维持血糖&nbsp;&nbsp;&nbsp;
- **长期饥饿 (3天以上未进食)**&nbsp;&nbsp;&nbsp;
	- **脂肪动员** -> **==进一步加强==**&nbsp;&nbsp;&nbsp;
		- 大量酮体生成 → **==脑利用酮体超过葡萄糖==**（脑"适应"酮体供能）&nbsp;&nbsp;&nbsp;
		- 肌组织以脂肪酸为主要燃料&nbsp;&nbsp;&nbsp;
	- **蛋白质分解** -> **==减少==**（保护肌肉！）&nbsp;&nbsp;&nbsp;
		- 氨基酸释出减少 → 节约体内蛋白质&nbsp;&nbsp;&nbsp;
	- **糖异生** -> **==明显减少==**（与短期饥饿相比）&nbsp;&nbsp;&nbsp;
		- 原料转变 -> 乳酸和**甘油**成为肝糖异生的**主要原料**&nbsp;&nbsp;&nbsp;
		- **==肾糖异生==** -> 明显增强 -> 几乎**与肝相等**&nbsp;&nbsp;&nbsp;
- **短期 vs 长期饥饿 核心对比**&nbsp;&nbsp;&nbsp;
	- **脑的主要燃料**&nbsp;&nbsp;&nbsp;
		- 短期 -> **==葡萄糖==**&nbsp;&nbsp;&nbsp;
		- 长期 -> **==酮体==**（超过葡萄糖）&nbsp;&nbsp;&nbsp;
	- **糖异生强度**&nbsp;&nbsp;&nbsp;
		- 短期 -> **==肝明显增强==**（约150g/天）&nbsp;&nbsp;&nbsp;
		- 长期 -> **==肝减少 + 肾增强==**（肝肾≈相等）&nbsp;&nbsp;&nbsp;
	- **糖异生原料**&nbsp;&nbsp;&nbsp;
		- 短期 -> 主要是 **==氨基酸==**（蛋白质分解）&nbsp;&nbsp;&nbsp;
		- 长期 -> 主要是 **==乳酸 + 甘油==**&nbsp;&nbsp;&nbsp;
	- **蛋白质分解**&nbsp;&nbsp;&nbsp;
		- 短期 -> **==加强==**&nbsp;&nbsp;&nbsp;
		- 长期 -> **==减少==**（保护肌肉，节俭机制）&nbsp;&nbsp;&nbsp;
	- **脂肪动员**&nbsp;&nbsp;&nbsp;
		- 短期 -> **==加强==**&nbsp;&nbsp;&nbsp;
		- 长期 -> **==进一步加强==**&nbsp;&nbsp;&nbsp;
	- **酮体生成**&nbsp;&nbsp;&nbsp;
		- 短期 -> 增加&nbsp;&nbsp;&nbsp;
		- 长期 -> **==大量增加==**（脑转用酮体）&nbsp;&nbsp;&nbsp;
- **一句话速记**&nbsp;&nbsp;&nbsp;
	- **短期饥饿** -> **==分解自身、拼命糖异生==**（氨基酸为主原料）&nbsp;&nbsp;&nbsp;
	- **长期饥饿** -> **==保护肌肉、脑改吃酮==**（肾也帮忙糖异生）&nbsp;&nbsp;&nbsp;
