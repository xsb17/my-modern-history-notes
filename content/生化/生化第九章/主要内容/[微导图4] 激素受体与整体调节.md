# 激素受体与整体调节&nbsp;&nbsp;&nbsp;

- **激素按受体位置分类**&nbsp;&nbsp;&nbsp;
	- **==膜受体激素==**&nbsp;&nbsp;&nbsp;
		- 受体位置 -> **==细胞膜表面==**（跨膜糖蛋白）&nbsp;&nbsp;&nbsp;
		- 化学本质 -> **==亲水性==**（不能自由通过细胞膜）&nbsp;&nbsp;&nbsp;
		- 包括&nbsp;&nbsp;&nbsp;
			- 肽类激素 -> **==胰岛素、胰高血糖素、生长激素==**&nbsp;&nbsp;&nbsp;
			- 儿茶酚胺类 -> **==肾上腺素、去甲肾上腺素==**&nbsp;&nbsp;&nbsp;
		- 机制 -> 与膜受体结合 → 激活**第二信使**(==cAMP/cGMP==) → 级联放大 → 生物学效应&nbsp;&nbsp;&nbsp;
	- **==胞内受体激素==**&nbsp;&nbsp;&nbsp;
		- 受体位置 -> **==细胞内==**（胞质或核内）&nbsp;&nbsp;&nbsp;
		- 化学本质 -> **==疏水性==**（可自由通过细胞膜）&nbsp;&nbsp;&nbsp;
		- 包括&nbsp;&nbsp;&nbsp;
			- **==类固醇激素==**（糖皮质激素、性激素等）&nbsp;&nbsp;&nbsp;
			- **==甲状腺素==**&nbsp;&nbsp;&nbsp;
			- 前列腺素等&nbsp;&nbsp;&nbsp;
		- 机制 -> 直接进入细胞 → 与胞内受体结合 → 调控**基因表达**→ 慢速持久&nbsp;&nbsp;&nbsp;
- **饱食状态下的代谢趋势**&nbsp;&nbsp;&nbsp;
	- **混合膳食**&nbsp;&nbsp;&nbsp;
		- 激素 -> **==胰岛素中度升高==**&nbsp;&nbsp;&nbsp;
		- 主要供能物质 -> **==葡萄糖==**&nbsp;&nbsp;&nbsp;
		- 多余葡萄糖 -> 合成**肝糖原/肌糖原** + 肝合成脂肪(VLDL 输出)&nbsp;&nbsp;&nbsp;
	- **高糖膳食**&nbsp;&nbsp;&nbsp;
		- 激素 -> **==胰岛素明显升高==** / 胰高血糖素降低&nbsp;&nbsp;&nbsp;
		- 葡萄糖 -> 合成糖原 + 大量转化为**甘油三酯**储存&nbsp;&nbsp;&nbsp;
	- **高脂膳食**&nbsp;&nbsp;&nbsp;
		- 激素 -> **==胰岛素降低==** / **==胰高血糖素升高==**&nbsp;&nbsp;&nbsp;
		- 肝糖原分解补充血糖 / 氨基酸糖异生补充血糖 / 肝生酮体&nbsp;&nbsp;&nbsp;
	- **高蛋白膳食**&nbsp;&nbsp;&nbsp;
		- 激素 -> 胰岛素中度升高 + **==胰高血糖素升高==**&nbsp;&nbsp;&nbsp;
		- 肝糖原分解 + 糖异生补充血糖 / 部分氨基酸→甘油三酯 / 部分→骨骼肌&nbsp;&nbsp;&nbsp;
- **空腹状态（餐后12h以上）**&nbsp;&nbsp;&nbsp;
	- 激素 -> 胰岛素↓ / **==胰高血糖素↑==**&nbsp;&nbsp;&nbsp;
	- 餐后**6~8h** -> **==肝糖原分解==**补充血糖&nbsp;&nbsp;&nbsp;
	- 餐后**16~18h** -> 肝糖原将尽 -> **==糖异生==**补充血糖 / 脂肪动员中度增加 / 肝生酮体供肌&nbsp;&nbsp;&nbsp;
- **调节本质**&nbsp;&nbsp;&nbsp;
	- 血糖高 -> **==胰岛素↑==** -> 促进糖利用+糖原合成+脂肪合成&nbsp;&nbsp;&nbsp;
	- 血糖低 -> **==胰高血糖素↑==** -> 促进糖原分解+糖异生+脂肪动员&nbsp;&nbsp;&nbsp;
