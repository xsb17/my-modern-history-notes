# [全集存档] 生化第八章：核苷酸代谢&nbsp;&nbsp;&nbsp;

## 第一节 核苷酸的代谢概述&nbsp;&nbsp;&nbsp;

### 一、核苷酸的生物学功能&nbsp;&nbsp;&nbsp;

- 作为核酸合成的原料 (dNTP → DNA; NTP → RNA)&nbsp;&nbsp;&nbsp;
- 体内能量的利用形式 (ATP, GTP)&nbsp;&nbsp;&nbsp;
- 参与酶活性的调节 (AMP, ADP, ATP)&nbsp;&nbsp;&nbsp;
- 作为信号分子参与细胞信号转导 (cAMP, cGMP)&nbsp;&nbsp;&nbsp;
- 组成辅酶 (FAD, NAD⁺, NADP⁺, CoA)&nbsp;&nbsp;&nbsp;
- 作为合成代谢的活化中间物 (SAM, UDPG, CDP-胆碱)&nbsp;&nbsp;&nbsp;

### 二、核酸的消化与吸收&nbsp;&nbsp;&nbsp;

- 食物核蛋白 → 胃酸 → 蛋白质 + 核酸&nbsp;&nbsp;&nbsp;
- 核酸 →(胰核酸酶/磷酸二酯酶)→ 核苷酸 →(胰、肠核苷酸酶)→ 核苷&nbsp;&nbsp;&nbsp;
- 核苷 →(核苷水解酶/核苷磷酸化酶)→ 戊糖 + 碱基&nbsp;&nbsp;&nbsp;
- 核苷酸不属于营养必需物质（大部分排出体外，仅小部分被利用）&nbsp;&nbsp;&nbsp;

### 三、核苷酸代谢概述&nbsp;&nbsp;&nbsp;

#### 合成代谢&nbsp;&nbsp;&nbsp;

- **从头合成(de novo synthesis)**&nbsp;&nbsp;&nbsp;
  - 部位：肝脏、小肠粘膜、胸腺&nbsp;&nbsp;&nbsp;
  - 利用简单原料（磷酸核糖、氨基酸、CO₂ 等）经一系列酶促反应合成核苷酸&nbsp;&nbsp;&nbsp;
- **补救合成(salvage pathway)**&nbsp;&nbsp;&nbsp;
  - 部位：脑、骨髓&nbsp;&nbsp;&nbsp;
  - 利用已有的碱基或核苷为原料合成核苷酸&nbsp;&nbsp;&nbsp;

#### 分解代谢&nbsp;&nbsp;&nbsp;

- 核苷酸 → 核苷 → 碱基 + 戊糖 → 进一步分解排出&nbsp;&nbsp;&nbsp;

## 第二节 嘌呤核苷酸代谢&nbsp;&nbsp;&nbsp;

### 一、嘌呤核苷酸的合成代谢&nbsp;&nbsp;&nbsp;

#### 1. 从头合成途径&nbsp;&nbsp;&nbsp;

##### 合成部位&nbsp;&nbsp;&nbsp;

- 肝是体内从头合成嘌呤核苷酸的主要器官，其次是小肠和胸腺&nbsp;&nbsp;&nbsp;
- 脑、骨髓和红细胞等无法进行从头合成途径&nbsp;&nbsp;&nbsp;

##### 合成原料&nbsp;&nbsp;&nbsp;

- 磷酸核糖（PRPP）&nbsp;&nbsp;&nbsp;
- 氨基酸：甘氨酸(Gly)、天冬氨酸(Asp)、谷氨酰胺(Gln)&nbsp;&nbsp;&nbsp;
- 一碳单位（甲酰基，来自 FH₄）&nbsp;&nbsp;&nbsp;
- CO₂&nbsp;&nbsp;&nbsp;

##### 嘌呤环各原子来源口诀&nbsp;&nbsp;&nbsp;

- 甘氨当中站，谷碳坐两边，左上天冬氨，头顶 CO₂（一碳单位补 C₂、C₈）&nbsp;&nbsp;&nbsp;

##### 合成过程&nbsp;&nbsp;&nbsp;

- 第 1 阶段：IMP（次黄嘌呤核苷酸）的合成（8 步反应）&nbsp;&nbsp;&nbsp;
  - PRPP →(PRPP 酰胺转移酶，关键酶)→ 5-磷酸核糖胺 →(甘氨酰胺核苷酸合成酶)→ 甘氨酰胺核苷酸 →…→ IMP&nbsp;&nbsp;&nbsp;
- 第 2 阶段：AMP 和 GMP 的生成&nbsp;&nbsp;&nbsp;
  - IMP →(腺苷酸代琥珀酸合成酶 + 腺苷酸代琥珀酸裂解酶)→ AMP&nbsp;&nbsp;&nbsp;
  - IMP →(IMP 脱氢酶)→ XMP →(GMP 合成酶)→ GMP&nbsp;&nbsp;&nbsp;

##### 从头合成特点&nbsp;&nbsp;&nbsp;

- 嘌呤核苷酸是在磷酸核糖分子上逐步合成 IMP，再由 IMP 转变成 AMP 及 GMP&nbsp;&nbsp;&nbsp;
- 需要大量的 ATP 和氨基酸（Gly、Asp、Gln）等原料&nbsp;&nbsp;&nbsp;
- AMP/GMP →(核苷一磷酸激酶)→ ADP/GDP →(核苷二磷酸激酶)→ ATP/GTP&nbsp;&nbsp;&nbsp;

##### 关键酶&nbsp;&nbsp;&nbsp;

- **PRPP 合成酶**&nbsp;&nbsp;&nbsp;
- **PRPP 酰胺转移酶**（关键限速酶）&nbsp;&nbsp;&nbsp;

#### 2. 补救合成途径&nbsp;&nbsp;&nbsp;

##### 部位与原料&nbsp;&nbsp;&nbsp;

- 部位：主要在脑、骨髓等组织&nbsp;&nbsp;&nbsp;
- 原料：体内现成的嘌呤碱基、嘌呤核苷&nbsp;&nbsp;&nbsp;

##### 关键酶&nbsp;&nbsp;&nbsp;

- **APRT**（腺嘌呤磷酸核糖转移酶）：腺嘌呤 + PRPP → AMP + PPi&nbsp;&nbsp;&nbsp;
- **HGPRT**（次黄嘌呤-鸟嘌呤磷酸核糖转移酶）：次黄嘌呤 + PRPP → IMP + PPi；鸟嘌呤 + PRPP → GMP + PPi&nbsp;&nbsp;&nbsp;
- **腺苷激酶**：腺苷 + ATP → AMP + ADP&nbsp;&nbsp;&nbsp;

##### 生理意义&nbsp;&nbsp;&nbsp;

- 节省能量和减少氨基酸的消耗&nbsp;&nbsp;&nbsp;
- 体内某些组织器官（脑、骨髓等）只能进行补救合成&nbsp;&nbsp;&nbsp;

##### Lesch-Nyhan 综合征（自毁容貌症）&nbsp;&nbsp;&nbsp;

- X 染色体连锁隐性遗传病，常见于男性&nbsp;&nbsp;&nbsp;
- 缺乏 **HGPRT**，导致嘌呤核苷酸补救合成途径障碍&nbsp;&nbsp;&nbsp;
- 表现为尿酸增高及神经异常（智力低下，攻击性强，自毁容貌）&nbsp;&nbsp;&nbsp;

### 二、嘌呤核苷酸的分解代谢&nbsp;&nbsp;&nbsp;

#### 分解过程&nbsp;&nbsp;&nbsp;

- 分解部位：主要在肝、肾、小肠中进行&nbsp;&nbsp;&nbsp;
- AMP → 次黄嘌呤 → 黄嘌呤(GMP → 鸟嘌呤 → 黄嘌呤) → 尿酸&nbsp;&nbsp;&nbsp;

#### 关键酶&nbsp;&nbsp;&nbsp;

- **黄嘌呤氧化酶**：催化次黄嘌呤 → 黄嘌呤 → **尿酸**&nbsp;&nbsp;&nbsp;

#### 尿酸与痛风&nbsp;&nbsp;&nbsp;

- 嘌呤代谢终产物：**尿酸**（水溶性差）&nbsp;&nbsp;&nbsp;
- 高尿酸血症：血清中尿酸浓度超过参考值上限&nbsp;&nbsp;&nbsp;
- 痛风(gout)：尿酸盐结晶沉积在关节、肾和皮下等部位，引起急慢性炎症和组织损伤&nbsp;&nbsp;&nbsp;

#### 别嘌呤醇治疗痛风&nbsp;&nbsp;&nbsp;

- 别嘌呤醇与次黄嘌呤结构类似，是**黄嘌呤氧化酶**的抑制剂，抑制尿酸生成&nbsp;&nbsp;&nbsp;
- 别嘌呤醇与 PRPP 合成别嘌呤醇核苷酸 → PRPP 减少 → 抑制嘌呤从头合成和补救合成&nbsp;&nbsp;&nbsp;
- 别嘌呤醇核苷酸反馈抑制 PRPP 酰胺转移酶 → 抑制嘌呤从头合成&nbsp;&nbsp;&nbsp;

## 第三节 嘧啶核苷酸代谢&nbsp;&nbsp;&nbsp;

### 一、嘧啶核苷酸的合成代谢&nbsp;&nbsp;&nbsp;

#### 1. 从头合成途径&nbsp;&nbsp;&nbsp;

##### 合成部位&nbsp;&nbsp;&nbsp;

- 主要在肝脏中&nbsp;&nbsp;&nbsp;

##### 合成原料&nbsp;&nbsp;&nbsp;

- 谷氨酰胺(Gln)&nbsp;&nbsp;&nbsp;
- CO₂&nbsp;&nbsp;&nbsp;
- 天冬氨酸(Asp)&nbsp;&nbsp;&nbsp;
- PRPP（提供磷酸核糖）&nbsp;&nbsp;&nbsp;

##### 嘧啶环各原子来源口诀&nbsp;&nbsp;&nbsp;

- 天冬氨酸右边站，谷酰直往左上窜，剩余废物二氧化碳&nbsp;&nbsp;&nbsp;

##### 合成过程&nbsp;&nbsp;&nbsp;

- 第 1 阶段：UMP（尿苷酸）的合成&nbsp;&nbsp;&nbsp;
  - CO₂ + Gln + 2ATP →(CPS-Ⅱ)→ 氨基甲酰磷酸 → 与 Asp 结合 →…→ 乳清酸 →(与 PRPP 结合)→ UMP&nbsp;&nbsp;&nbsp;
- 第 2 阶段：CTP 和 dTMP 的合成&nbsp;&nbsp;&nbsp;
  - UMP → UDP → UTP →(CTP 合成酶)→ **CTP**&nbsp;&nbsp;&nbsp;
  - UDP → dUDP → dUMP →(胸苷酸合酶)→ **dTMP**&nbsp;&nbsp;&nbsp;

##### 关键酶&nbsp;&nbsp;&nbsp;

- **氨基甲酰磷酸合成酶Ⅱ(CPS-Ⅱ)**（嘧啶合成关键酶）&nbsp;&nbsp;&nbsp;
- **胸苷酸合酶**（催化 dUMP → dTMP）&nbsp;&nbsp;&nbsp;

##### CPS-Ⅰ 与 CPS-Ⅱ 的主要区别&nbsp;&nbsp;&nbsp;

- 分布：&nbsp;&nbsp;&nbsp;
  - CPS-Ⅰ：线粒体（肝）&nbsp;&nbsp;&nbsp;
  - CPS-Ⅱ：胞液（几乎所有细胞）&nbsp;&nbsp;&nbsp;
- 氮源：&nbsp;&nbsp;&nbsp;
  - CPS-Ⅰ：氨&nbsp;&nbsp;&nbsp;
  - CPS-Ⅱ：谷氨酰胺&nbsp;&nbsp;&nbsp;
- 变构激活剂：&nbsp;&nbsp;&nbsp;
  - CPS-Ⅰ：N-乙酰谷氨酸(AGA)&nbsp;&nbsp;&nbsp;
  - CPS-Ⅱ：PRPP&nbsp;&nbsp;&nbsp;
- 反馈抑制剂：&nbsp;&nbsp;&nbsp;
  - CPS-Ⅰ：无&nbsp;&nbsp;&nbsp;
  - CPS-Ⅱ：UMP&nbsp;&nbsp;&nbsp;
- 功能：&nbsp;&nbsp;&nbsp;
  - CPS-Ⅰ：尿素合成&nbsp;&nbsp;&nbsp;
  - CPS-Ⅱ：嘧啶合成&nbsp;&nbsp;&nbsp;

##### 从头合成特点&nbsp;&nbsp;&nbsp;

- 嘧啶核苷酸先合成嘧啶环，再与 PRPP 的磷酸核糖基结合（与嘌呤相反）&nbsp;&nbsp;&nbsp;
- 先合成 UMP，转变为 UTP，再由 UTP 转变为 CTP，dUMP 转变为 dTMP&nbsp;&nbsp;&nbsp;
- UMP 合成需要 3 个 ATP（4 个高能磷酸键），CTP 合成需要 6 个 ATP（7 个高能磷酸键）&nbsp;&nbsp;&nbsp;

#### 2. 补救合成途径&nbsp;&nbsp;&nbsp;

- 嘧啶碱基（尿嘧啶、胸腺嘧啶、乳清酸，胞嘧啶不能进行该反应）通过补救途径再合成&nbsp;&nbsp;&nbsp;

### 二、嘧啶核苷酸的分解代谢&nbsp;&nbsp;&nbsp;

- 分解部位：主要在肝中进行&nbsp;&nbsp;&nbsp;
- 分解产物：&nbsp;&nbsp;&nbsp;
  - C、U → **β-丙氨酸**&nbsp;&nbsp;&nbsp;
  - T → **β-氨基异丁酸**&nbsp;&nbsp;&nbsp;
- 产物水溶性好，随尿排出，也可进入 TCA 循环 → NH₃ + CO₂&nbsp;&nbsp;&nbsp;

### 嘌呤与嘧啶合成与分解对比&nbsp;&nbsp;&nbsp;

- 合成特点：&nbsp;&nbsp;&nbsp;
  - 嘌呤：碱基在磷酸核糖的基础上逐步合成 → 先合成 IMP&nbsp;&nbsp;&nbsp;
  - 嘧啶：先合成嘧啶环，再与磷酸核糖连接 → 先合成 UMP&nbsp;&nbsp;&nbsp;
- 从头合成部位：&nbsp;&nbsp;&nbsp;
  - 嘌呤：肝（主要）、小肠黏膜、胸腺&nbsp;&nbsp;&nbsp;
  - 嘧啶：肝脏&nbsp;&nbsp;&nbsp;
- 从头合成原料：&nbsp;&nbsp;&nbsp;
  - 嘌呤：天冬氨酸、甘氨酸、谷氨酰胺、CO₂、一碳单位、PRPP&nbsp;&nbsp;&nbsp;
  - 嘧啶：天冬氨酸、谷氨酰胺、CO₂、PRPP&nbsp;&nbsp;&nbsp;
- 从头合成关键酶：&nbsp;&nbsp;&nbsp;
  - 嘌呤：PRPP 合成酶、PRPP 酰胺转移酶&nbsp;&nbsp;&nbsp;
  - 嘧啶：氨基甲酰磷酸合成酶Ⅱ(CPS-Ⅱ)&nbsp;&nbsp;&nbsp;
- 中间物：&nbsp;&nbsp;&nbsp;
  - 嘌呤：IMP&nbsp;&nbsp;&nbsp;
  - 嘧啶：UMP&nbsp;&nbsp;&nbsp;
- 补救合成部位：&nbsp;&nbsp;&nbsp;
  - 嘌呤：脑、骨髓&nbsp;&nbsp;&nbsp;
- 分解产物：&nbsp;&nbsp;&nbsp;
  - 嘌呤：尿酸&nbsp;&nbsp;&nbsp;
  - 嘧啶：C/U → β-丙氨酸；T → β-氨基异丁酸&nbsp;&nbsp;&nbsp;

## 第四节 脱氧核糖核苷酸的合成&nbsp;&nbsp;&nbsp;

### 合成方式&nbsp;&nbsp;&nbsp;

- 在**核苷二磷酸(NDP)**水平上还原生成 dNDP&nbsp;&nbsp;&nbsp;
- 催化酶：**核糖核苷酸还原酶**&nbsp;&nbsp;&nbsp;
- ADP → dADP, GDP → dGDP, CDP → dCDP, UDP → dUDP&nbsp;&nbsp;&nbsp;
- dNDP + ATP →(激酶)→ dNTP + ADP（dNTP 是 DNA 合成原料）&nbsp;&nbsp;&nbsp;

### 脱氧胸苷酸的合成&nbsp;&nbsp;&nbsp;

- UDP → dUDP → dUMP →(胸苷酸合酶)→ dTMP&nbsp;&nbsp;&nbsp;
- 此步骤需 N⁵,N¹⁰-CH₂-FH₄ 提供甲基&nbsp;&nbsp;&nbsp;
- 药物靶点：**氨基蝶呤**、**甲氨蝶呤(MTX)**（抗肿瘤，抑制叶酸还原）&nbsp;&nbsp;&nbsp;
- 药物靶点：**5-氟尿嘧啶(5-FU)**（抗肿瘤，抑制胸苷酸合酶）&nbsp;&nbsp;&nbsp;

## 第五节 核苷酸抗代谢物&nbsp;&nbsp;&nbsp;

### 一、抗代谢物的概念&nbsp;&nbsp;&nbsp;

- **抗代谢物(antimetabolite)**：与体内必需代谢物结构相似的化合物，可与必需代谢物发生特异性拮抗，影响正常代谢&nbsp;&nbsp;&nbsp;

### 二、嘌呤核苷酸的抗代谢物&nbsp;&nbsp;&nbsp;

#### 6-巯基嘌呤(6-MP)&nbsp;&nbsp;&nbsp;

- 嘌呤碱基类似物&nbsp;&nbsp;&nbsp;
- 作用机制：&nbsp;&nbsp;&nbsp;
  - 6-MP 核苷酸结构与 IMP 类似，反馈抑制 PRPP 酰胺转移酶 → 抑制嘌呤从头合成&nbsp;&nbsp;&nbsp;
  - 抑制 IMP 向 AMP 和 GMP 的转变&nbsp;&nbsp;&nbsp;
  - 竞争性抑制 HGPRT → 阻断嘌呤补救合成途径&nbsp;&nbsp;&nbsp;

#### 氮杂丝氨酸&nbsp;&nbsp;&nbsp;

- 谷氨酰胺结构类似物&nbsp;&nbsp;&nbsp;
- 竞争性抑制以谷氨酰胺为底物的酶：PRPP 酰胺转移酶、GMP 合成酶、CPS-Ⅱ、CTP 合成酶等&nbsp;&nbsp;&nbsp;

### 三、嘧啶核苷酸的抗代谢物&nbsp;&nbsp;&nbsp;

#### 5-氟尿嘧啶(5-FU)&nbsp;&nbsp;&nbsp;

- 嘧啶碱基类似物（与胸腺嘧啶 T 结构相似）&nbsp;&nbsp;&nbsp;
- 体内转变为 FdUMP、FdUTP、FUTP 发挥作用&nbsp;&nbsp;&nbsp;
- 作用机制：&nbsp;&nbsp;&nbsp;
  - FdUMP 与 dUMP 结构类似 → **抑制胸苷酸合酶** → 抑制 dTMP 合成&nbsp;&nbsp;&nbsp;
  - FdUTP → 掺入 DNA → DNA 链断裂和细胞死亡&nbsp;&nbsp;&nbsp;
  - FUTP → 掺入 RNA → 破坏 RNA 结构和功能&nbsp;&nbsp;&nbsp;

### 四、叶酸类似物&nbsp;&nbsp;&nbsp;

#### 甲氨蝶呤(MTX)、氨基蝶呤&nbsp;&nbsp;&nbsp;

- 竞争性抑制**二氢叶酸还原酶** → 抑制 FH₄ 生成&nbsp;&nbsp;&nbsp;
- 抑制嘌呤环 C₈ 和 C₂ 的合成 + 抑制 dTMP 合成 → 抑制核酸和蛋白质合成&nbsp;&nbsp;&nbsp;

### 五、核苷类似物&nbsp;&nbsp;&nbsp;

#### 阿糖胞苷、环胞苷&nbsp;&nbsp;&nbsp;

- 转变为相应核苷酸掺入 DNA 链&nbsp;&nbsp;&nbsp;
- 抑制 CDP 还原成 dCDP → 影响 DNA 合成&nbsp;&nbsp;&nbsp;

## 重要概念（来自课后总结）&nbsp;&nbsp;&nbsp;

- **核苷酸从头合成途径**：利用简单原料（磷酸核糖、氨基酸、CO₂ 等）经一系列酶促反应合成核苷酸&nbsp;&nbsp;&nbsp;
- **核苷酸补救合成途径**：利用体内已有碱基或核苷为原料合成核苷酸&nbsp;&nbsp;&nbsp;
- **别嘌呤醇治疗痛风的机制**：竞争性抑制黄嘌呤氧化酶 → 减少尿酸生成；与 PRPP 结合减少从头合成原料&nbsp;&nbsp;&nbsp;
- **抗代谢物**：与体内必需代谢物结构相似，能发生特异性拮抗的化合物&nbsp;&nbsp;&nbsp;
