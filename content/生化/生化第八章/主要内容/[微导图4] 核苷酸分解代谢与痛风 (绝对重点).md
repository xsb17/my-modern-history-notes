# 核苷酸分解代谢与痛风 (绝对重点)&nbsp;&nbsp;&nbsp;

- **嘌呤核苷酸分解代谢**&nbsp;&nbsp;&nbsp;
	- **分解部位** -> **==肝、肾、小肠==**&nbsp;&nbsp;&nbsp;
	- **代谢路径**&nbsp;&nbsp;&nbsp;
		- AMP -> 腺苷 -> 次黄嘌呤核苷 -> **==次黄嘌呤==**&nbsp;&nbsp;&nbsp;
		- GMP -> 鸟苷 -> **==鸟嘌呤==** -> **==黄嘌呤==**&nbsp;&nbsp;&nbsp;
		- 次黄嘌呤 ->(==**黄嘌呤氧化酶**==)-> **==黄嘌呤==**&nbsp;&nbsp;&nbsp;
		- 黄嘌呤 ->(==**黄嘌呤氧化酶**==)-> **==尿酸==**&nbsp;&nbsp;&nbsp;
	- **关键酶** -> **==黄嘌呤氧化酶==**（催化最后两步）&nbsp;&nbsp;&nbsp;
	- **终产物** -> **==尿酸==**（水溶性差！嘌呤代谢的特征终产物）&nbsp;&nbsp;&nbsp;
- **痛风 (Gout)**&nbsp;&nbsp;&nbsp;
	- **发病机制**&nbsp;&nbsp;&nbsp;
		- 嘌呤代谢紊乱 / 尿酸排泄减少&nbsp;&nbsp;&nbsp;
		- 血清尿酸 > 参考值上限 -> **==高尿酸血症==**&nbsp;&nbsp;&nbsp;
		- 尿酸/尿酸盐浓度过饱和 -> **==尿酸盐结晶==**析出&nbsp;&nbsp;&nbsp;
		- 结晶沉积在**关节、肾、皮下** -> **急慢性炎症 + 组织损伤**&nbsp;&nbsp;&nbsp;
	- **诱因** -> **==大量进食肉类/海鲜==**（高嘌呤食物）-> 尿酸生成↑↑&nbsp;&nbsp;&nbsp;
	- **典型表现** -> **==足痛风==**（大拇趾关节红肿剧痛）&nbsp;&nbsp;&nbsp;
	- **诊断指标** -> 血清**尿酸**水平 > 0.36 mmol/L（参考上限）&nbsp;&nbsp;&nbsp;
	- **高尿酸血症** -> 痛风的前期&nbsp;&nbsp;&nbsp;
- **别嘌呤醇治疗痛风（必考论述题）**&nbsp;&nbsp;&nbsp;
	- **药物** -> **==别嘌呤醇 (Allopurinol)==**&nbsp;&nbsp;&nbsp;
	- **机制1：竞争性抑制**&nbsp;&nbsp;&nbsp;
		- 别嘌呤醇结构类似**==次黄嘌呤==**&nbsp;&nbsp;&nbsp;
		- -> 竞争性抑制**==黄嘌呤氧化酶==**&nbsp;&nbsp;&nbsp;
		- -> 次黄嘌呤 → 黄嘌呤 **↓** / 黄嘌呤 → 尿酸 **↓**&nbsp;&nbsp;&nbsp;
		- -> **减少尿酸生成**&nbsp;&nbsp;&nbsp;
	- **机制2：消耗 PRPP**&nbsp;&nbsp;&nbsp;
		- 别嘌呤醇 + PRPP -> **别嘌呤醇核苷酸**&nbsp;&nbsp;&nbsp;
		- -> PRPP 被**消耗减少**&nbsp;&nbsp;&nbsp;
		- -> 嘌呤从头合成 **↓** + 补救合成 **↓**&nbsp;&nbsp;&nbsp;
	- **机制3：反馈抑制**&nbsp;&nbsp;&nbsp;
		- 别嘌呤醇核苷酸 -> 反馈抑制**==PRPP 酰胺转移酶==**&nbsp;&nbsp;&nbsp;
		- -> 嘌呤从头合成 **↓**&nbsp;&nbsp;&nbsp;
	- **一句话总结** -> 别嘌呤醇 = **=="黄嘌呤氧化酶的假底物"==**&nbsp;&nbsp;&nbsp;
- **嘧啶核苷酸分解代谢（对比）**&nbsp;&nbsp;&nbsp;
	- **分解部位** -> **==肝==**（主要）&nbsp;&nbsp;&nbsp;
	- **分解产物**&nbsp;&nbsp;&nbsp;
		- **C、U** -> **==β-丙氨酸==**&nbsp;&nbsp;&nbsp;
		- **T** -> **==β-氨基异丁酸==**&nbsp;&nbsp;&nbsp;
	- **特点** -> 产物**水溶性好** -> 易随尿排出 -> **不产生尿酸！**&nbsp;&nbsp;&nbsp;
	- **不会导致痛风**（与嘌呤的最大区别）&nbsp;&nbsp;&nbsp;
- **嘌呤 vs 嘧啶 分解速记**&nbsp;&nbsp;&nbsp;
	- **嘌呤分解** -> 终产物**==尿酸==** -> 水溶性差 -> **痛风**&nbsp;&nbsp;&nbsp;
	- **嘧啶分解** -> 终产物**==β-丙氨酸 / β-氨基异丁酸==** -> 水溶性好 -> 不致病&nbsp;&nbsp;&nbsp;
	- **嘌呤分解关键酶** -> **==黄嘌呤氧化酶==**&nbsp;&nbsp;&nbsp;
	- **痛风药靶点** -> **==黄嘌呤氧化酶==**（被别嘌呤醇抑制）&nbsp;&nbsp;&nbsp;
