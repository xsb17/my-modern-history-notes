# 核苷酸抗代谢药物 (抗肿瘤机制)&nbsp;&nbsp;&nbsp;

- **抗代谢物概念**&nbsp;&nbsp;&nbsp;
	- **定义** -> 与体内**必需代谢物结构相似**的化合物 -> 发生**特异性拮抗** -> 干扰正常代谢&nbsp;&nbsp;&nbsp;
	- **核苷酸抗代谢物** -> 嘌呤/嘧啶/核苷/氨基酸/叶酸的**类似物** -> 抑制核苷酸和核酸合成&nbsp;&nbsp;&nbsp;
- **嘌呤类抗代谢物**&nbsp;&nbsp;&nbsp;
	- **==6-巯基嘌呤 (6-MP)==**&nbsp;&nbsp;&nbsp;
		- **类别** -> **嘌呤碱基类似物**（类似次黄嘌呤）&nbsp;&nbsp;&nbsp;
		- **机制1** -> 6-MP 核苷酸结构类似 IMP -> **反馈抑制 PRPP 酰胺转移酶** -> 从头合成↓&nbsp;&nbsp;&nbsp;
		- **机制2** -> 抑制 IMP → **AMP** 和 **GMP** 的转变&nbsp;&nbsp;&nbsp;
		- **机制3** -> 竞争性抑制 **==HGPRT==** -> 阻断**补救合成**途径&nbsp;&nbsp;&nbsp;
	- **==氮杂丝氨酸==**&nbsp;&nbsp;&nbsp;
		- **类别** -> **谷氨酰胺 (Gln) 类似物**&nbsp;&nbsp;&nbsp;
		- **机制** -> 竞争性抑制以 Gln 为底物的酶&nbsp;&nbsp;&nbsp;
		- **靶酶** -> PRPP 酰胺转移酶 / GMP 合成酶 / **==CPS-Ⅱ==** / CTP 合成酶&nbsp;&nbsp;&nbsp;
- **嘧啶类抗代谢物**&nbsp;&nbsp;&nbsp;
	- **==5-氟尿嘧啶 (5-FU)==**&nbsp;&nbsp;&nbsp;
		- **类别** -> **嘧啶碱基类似物**（类似**胸腺嘧啶 T**）&nbsp;&nbsp;&nbsp;
		- **体内活化** -> 5-FU → **==FdUMP==** / FdUTP / FUTP&nbsp;&nbsp;&nbsp;
		- **机制1** -> **==FdUMP==** 与 dUMP 结构类似 -> **抑制胸苷酸合酶** -> dTMP↓ -> DNA 合成↓&nbsp;&nbsp;&nbsp;
		- **机制2** -> FdUTP **掺入 DNA** -> 触发错误切除修复 -> DNA 链断裂 -> **细胞死亡**&nbsp;&nbsp;&nbsp;
		- **机制3** -> FUTP **掺入 RNA** -> 取代 UMP -> 破坏 RNA 结构功能&nbsp;&nbsp;&nbsp;
		- **临床用途** -> **抗肿瘤**（阻断快速分裂细胞的 DNA 合成）&nbsp;&nbsp;&nbsp;
- **叶酸类抗代谢物**&nbsp;&nbsp;&nbsp;
	- **==甲氨蝶呤 (MTX)== / ==氨基蝶呤==**&nbsp;&nbsp;&nbsp;
		- **类别** -> **叶酸类似物**&nbsp;&nbsp;&nbsp;
		- **靶酶** -> 竞争性抑制 **==二氢叶酸还原酶==**&nbsp;&nbsp;&nbsp;
		- **后果链** -> FH₄ **不能再生**&nbsp;&nbsp;&nbsp;
			- -> 一碳单位代谢 **↓**&nbsp;&nbsp;&nbsp;
			- -> **嘌呤环 C₈/C₂ 合成 ↓**（需要甲酰基 FH₄）&nbsp;&nbsp;&nbsp;
			- -> **dTMP 合成 ↓**（需要 N⁵,N¹⁰-CH₂-FH₄）&nbsp;&nbsp;&nbsp;
			- -> **DNA 合成 ↓** -> **细胞增殖 ↓**&nbsp;&nbsp;&nbsp;
		- **临床用途** -> 抗肿瘤 / 抗增殖&nbsp;&nbsp;&nbsp;
- **核苷类抗代谢物**&nbsp;&nbsp;&nbsp;
	- **==阿糖胞苷 / 环胞苷==**&nbsp;&nbsp;&nbsp;
		- **类别** -> **核苷类似物**&nbsp;&nbsp;&nbsp;
		- **机制** -> 转变为核苷酸**掺入 DNA 链** / 抑制 CDP → **==dCDP==** → DNA 合成↓&nbsp;&nbsp;&nbsp;
- **五大药物速记对比表**&nbsp;&nbsp;&nbsp;
	- **==6-MP==** -> 嘌呤类似物 -> 抑制**HGPRT** + PRPP 酰胺转移酶 -> 从头+补救合成↓&nbsp;&nbsp;&nbsp;
	- **==5-FU==** -> 嘧啶类似物 -> 抑制**胸苷酸合酶** -> dTMP↓&nbsp;&nbsp;&nbsp;
	- **==MTX==** -> 叶酸类似物 -> 抑制**二氢叶酸还原酶** -> FH₄↓ -> 嘌呤+dTMP↓&nbsp;&nbsp;&nbsp;
	- **==氮杂丝氨酸==** -> Gln 类似物 -> 抑制**CPS-Ⅱ 等多种 Gln 依赖酶**&nbsp;&nbsp;&nbsp;
	- **==阿糖胞苷==** -> 核苷类似物 -> 抑制 CDP → **dCDP**&nbsp;&nbsp;&nbsp;
- **所有药物共同终点**&nbsp;&nbsp;&nbsp;
	- **最终靶点** -> **DNA 合成 ↓** / **RNA 功能 ↓**&nbsp;&nbsp;&nbsp;
	- **最终效应** -> **抑制细胞增殖** -> **==抗肿瘤==**&nbsp;&nbsp;&nbsp;
