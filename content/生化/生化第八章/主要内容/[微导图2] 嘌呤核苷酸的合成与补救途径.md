# 嘌呤核苷酸的合成与补救途径&nbsp;&nbsp;&nbsp;

- **从头合成途径**&nbsp;&nbsp;&nbsp;
	- **合成部位** -> **==肝==**（主要）/ 小肠粘膜 / 胸腺&nbsp;&nbsp;&nbsp;
	- **脑、骨髓、红细胞** -> **==不能从头合成==** -> 只能补救合成&nbsp;&nbsp;&nbsp;
	- **关键底物** -> **==PRPP==**（5-磷酸核糖-1-焦磷酸）-> 活化磷酸核糖供体&nbsp;&nbsp;&nbsp;
	- **两阶段**&nbsp;&nbsp;&nbsp;
		- **第1阶段：IMP 的合成（8步）**&nbsp;&nbsp;&nbsp;
			- PRPP ->(==**PRPP 酰胺转移酶**==/关键限速酶)-> 5-磷酸核糖胺&nbsp;&nbsp;&nbsp;
			- ->(甘氨酰胺核苷酸合成酶 + 甲酰基 + Gln + CO₂ + Asp + 甲酰基)-> **==IMP==**（次黄嘌呤核苷酸）&nbsp;&nbsp;&nbsp;
		- **第2阶段：IMP → AMP / GMP**&nbsp;&nbsp;&nbsp;
			- IMP ->(腺苷酸代琥珀酸合成酶+裂解酶)-> **==AMP==**&nbsp;&nbsp;&nbsp;
			- IMP ->(IMP 脱氢酶)-> XMP ->(GMP 合成酶)-> **==GMP==**&nbsp;&nbsp;&nbsp;
	- **能量消耗** -> 大量 **ATP** 和氨基酸消耗&nbsp;&nbsp;&nbsp;
- **补救合成途径**&nbsp;&nbsp;&nbsp;
	- **合成部位** -> **==脑==** / **==骨髓==**（只能靠补救合成！）&nbsp;&nbsp;&nbsp;
	- **原料** -> 体内现成的**嘌呤碱基** / **嘌呤核苷**&nbsp;&nbsp;&nbsp;
	- **关键酶**&nbsp;&nbsp;&nbsp;
		- **==APRT==**（腺嘌呤磷酸核糖转移酶）-> 腺嘌呤 + PRPP → AMP&nbsp;&nbsp;&nbsp;
		- **==HGPRT==**（次黄嘌呤-鸟嘌呤磷酸核糖转移酶）&nbsp;&nbsp;&nbsp;
			- 次黄嘌呤 + PRPP → IMP&nbsp;&nbsp;&nbsp;
			- 鸟嘌呤 + PRPP → GMP&nbsp;&nbsp;&nbsp;
		- **==腺苷激酶==** -> 腺苷 + ATP → AMP&nbsp;&nbsp;&nbsp;
	- **生理意义**&nbsp;&nbsp;&nbsp;
		- **==节省能量==**和**==减少氨基酸消耗==**&nbsp;&nbsp;&nbsp;
		- 脑、骨髓等**只能补救合成** -> 维持其核苷酸供应&nbsp;&nbsp;&nbsp;
- **自毁容貌症 (Lesch-Nyhan综合征)**&nbsp;&nbsp;&nbsp;
	- **遗传方式** -> **==X染色体连锁隐性遗传==** -> 常见于**男性**&nbsp;&nbsp;&nbsp;
	- **病因** -> 严重缺乏 **==HGPRT==**（次黄嘌呤-鸟嘌呤磷酸核糖转移酶）&nbsp;&nbsp;&nbsp;
	- **后果**&nbsp;&nbsp;&nbsp;
		- 补救合成途径**==障碍==**&nbsp;&nbsp;&nbsp;
		- PRPP 堆积 -> **==嘌呤从头合成↑↑==** -> **尿酸极度增高**&nbsp;&nbsp;&nbsp;
	- **临床表现**&nbsp;&nbsp;&nbsp;
		- **==智力低下==**&nbsp;&nbsp;&nbsp;
		- **==攻击性强==**&nbsp;&nbsp;&nbsp;
		- **==自毁容貌==**（自我咬伤嘴唇/手指）&nbsp;&nbsp;&nbsp;
		- 高尿酸血症 + 痛风&nbsp;&nbsp;&nbsp;
- **嘌呤合成关键总结**&nbsp;&nbsp;&nbsp;
	- **从头合成关键酶** -> **==PRPP 酰胺转移酶==**（限速酶）&nbsp;&nbsp;&nbsp;
	- **补救合成关键酶** -> **==HGPRT==** / **==APRT==**&nbsp;&nbsp;&nbsp;
	- **补救合成关键疾病** -> **==Lesch-Nyhan综合征==** -> 缺 HGPRT&nbsp;&nbsp;&nbsp;
