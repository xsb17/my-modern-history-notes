# 嘧啶核苷酸与脱氧核苷酸合成&nbsp;&nbsp;&nbsp;

- **嘧啶核苷酸从头合成**&nbsp;&nbsp;&nbsp;
	- **合成部位** -> **==肝==**（主要）&nbsp;&nbsp;&nbsp;
	- **关键酶** -> **==CPS-Ⅱ==**（氨基甲酰磷酸合成酶Ⅱ）&nbsp;&nbsp;&nbsp;
	- **两阶段**&nbsp;&nbsp;&nbsp;
		- **第1阶段：UMP 的合成**&nbsp;&nbsp;&nbsp;
			- Gln + CO₂ + 2ATP ->(==**CPS-Ⅱ**==)-> 氨基甲酰磷酸&nbsp;&nbsp;&nbsp;
			- 氨基甲酰磷酸 + Asp -> … -> 乳清酸&nbsp;&nbsp;&nbsp;
			- 乳清酸 + **PRPP** -> **==UMP==**（尿苷酸）&nbsp;&nbsp;&nbsp;
		- **第2阶段：CTP 和 dTMP 的合成**&nbsp;&nbsp;&nbsp;
			- UMP -> UDP -> UTP ->(CTP 合成酶)-> **==CTP==**&nbsp;&nbsp;&nbsp;
			- UDP -> dUDP -> dUMP ->(==**胸苷酸合酶**==)-> **==dTMP==**&nbsp;&nbsp;&nbsp;
	- **合成特点** -> **先合成嘧啶环** -> 再与 PRPP 连接（与嘌呤相反！）&nbsp;&nbsp;&nbsp;
- **CPS-Ⅰ vs CPS-Ⅱ（极其重要对比）**&nbsp;&nbsp;&nbsp;
	- **相同点** -> 都叫氨基甲酰磷酸合成酶&nbsp;&nbsp;&nbsp;
	- **==CPS-Ⅰ==**&nbsp;&nbsp;&nbsp;
		- **分布** -> **==线粒体==**（肝）&nbsp;&nbsp;&nbsp;
		- **氮源** -> **==氨 (NH₃)==**&nbsp;&nbsp;&nbsp;
		- **激活剂** -> **==N-乙酰谷氨酸 (AGA)==**&nbsp;&nbsp;&nbsp;
		- **反馈抑制剂** -> 无&nbsp;&nbsp;&nbsp;
		- **功能** -> **==尿素合成==**&nbsp;&nbsp;&nbsp;
		- **属于** -> 尿素循环（氨基酸代谢/第七章）&nbsp;&nbsp;&nbsp;
	- **==CPS-Ⅱ==**&nbsp;&nbsp;&nbsp;
		- **分布** -> **==胞液==**（几乎所有细胞）&nbsp;&nbsp;&nbsp;
		- **氮源** -> **==谷氨酰胺 (Gln)==**&nbsp;&nbsp;&nbsp;
		- **激活剂** -> **==PRPP==**&nbsp;&nbsp;&nbsp;
		- **反馈抑制剂** -> **==UMP==**&nbsp;&nbsp;&nbsp;
		- **功能** -> **==嘧啶合成==**&nbsp;&nbsp;&nbsp;
		- **属于** -> 核苷酸代谢（第八章）&nbsp;&nbsp;&nbsp;
- **脱氧核苷酸合成 (dNTP)**&nbsp;&nbsp;&nbsp;
	- **反应水平** -> **==核苷二磷酸 (NDP)==** 水平上还原&nbsp;&nbsp;&nbsp;
	- **催化酶** -> **==核糖核苷酸还原酶==**&nbsp;&nbsp;&nbsp;
	- **反应通式** -> ADP → **dADP** / GDP → **dGDP** / CDP → **dCDP** / UDP → **dUDP**&nbsp;&nbsp;&nbsp;
	- **后续** -> dNDP + ATP ->(激酶)-> **dNTP** -> **DNA 合成原料**&nbsp;&nbsp;&nbsp;
- **脱氧胸苷酸 (dTMP) 的特殊合成**&nbsp;&nbsp;&nbsp;
	- **路径** -> dUMP ->(==**胸苷酸合酶**==)-> **==dTMP==**&nbsp;&nbsp;&nbsp;
	- **甲基供体** -> **==N⁵,N¹⁰-CH₂-FH₄==**（一碳单位载体）&nbsp;&nbsp;&nbsp;
	- **药物靶点1** -> **==5-氟尿嘧啶 (5-FU)==** -> 抑制**胸苷酸合酶** -> 阻断 dTMP&nbsp;&nbsp;&nbsp;
	- **药物靶点2** -> **==甲氨蝶呤 (MTX)==** -> 抑制**二氢叶酸还原酶** -> FH₄↓ -> dTMP↓&nbsp;&nbsp;&nbsp;
