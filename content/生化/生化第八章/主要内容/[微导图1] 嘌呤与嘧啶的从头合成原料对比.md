# 嘌呤与嘧啶的从头合成原料对比&nbsp;&nbsp;&nbsp;

- **嘌呤核苷酸从头合成**&nbsp;&nbsp;&nbsp;
	- **合成部位** -> **==肝==**（主要）/ 小肠粘膜 / 胸腺&nbsp;&nbsp;&nbsp;
	- **嘌呤合成原料**&nbsp;&nbsp;&nbsp;
		- **磷酸核糖** -> PRPP（活化供体）&nbsp;&nbsp;&nbsp;
		- **氨基酸** -> **==天冬氨酸 (Asp)==** / **==谷氨酰胺 (Gln)==** / **==甘氨酸 (Gly)==**&nbsp;&nbsp;&nbsp;
		- **一碳单位** -> 甲酰基（来自 FH₄）-> 提供 C₂、C₈&nbsp;&nbsp;&nbsp;
		- **CO₂** -> 提供 C₆&nbsp;&nbsp;&nbsp;
	- **嘌呤环原子来源口诀**&nbsp;&nbsp;&nbsp;
		- **甘氨当中站** -> Gly 提供 C₄/C₅/N₇&nbsp;&nbsp;&nbsp;
		- **谷碳坐两边** -> Gln 提供 N₃/N₉&nbsp;&nbsp;&nbsp;
		- **左上天冬氨** -> Asp 提供 N₁&nbsp;&nbsp;&nbsp;
		- **头顶 CO₂** -> CO₂ 提供 C₆&nbsp;&nbsp;&nbsp;
	- **嘌呤合成特有原料** -> **==甘氨酸 (Gly)==** / **==一碳单位==**（嘧啶不需要！）&nbsp;&nbsp;&nbsp;
	- **关键中间物** -> **==IMP==**（次黄嘌呤核苷酸）-> 再分支生成 AMP / GMP&nbsp;&nbsp;&nbsp;
	- **关键酶** -> **==PRPP 合成酶==** / **==PRPP 酰胺转移酶==**（限速酶）&nbsp;&nbsp;&nbsp;
- **嘧啶核苷酸从头合成**&nbsp;&nbsp;&nbsp;
	- **合成部位** -> **==肝==**（主要）&nbsp;&nbsp;&nbsp;
	- **嘧啶合成原料**&nbsp;&nbsp;&nbsp;
		- **磷酸核糖** -> PRPP&nbsp;&nbsp;&nbsp;
		- **氨基酸** -> **==天冬氨酸 (Asp)==** / **==谷氨酰胺 (Gln)==**&nbsp;&nbsp;&nbsp;
		- **CO₂**&nbsp;&nbsp;&nbsp;
	- **嘧啶环原子来源口诀**&nbsp;&nbsp;&nbsp;
		- **天冬氨酸右边站** / **谷酰直往左上窜** / **剩余废物二氧化碳**&nbsp;&nbsp;&nbsp;
	- **关键中间物** -> **==UMP==**（尿苷酸）-> 再转变为 CTP / dTMP&nbsp;&nbsp;&nbsp;
	- **关键酶** -> **==CPS-Ⅱ==**（氨基甲酰磷酸合成酶Ⅱ）&nbsp;&nbsp;&nbsp;
- **嘌呤 vs 嘧啶 对比速记**&nbsp;&nbsp;&nbsp;
	- **差异1：原料** -> 嘌呤多了 **==甘氨酸==** + **==一碳单位==**&nbsp;&nbsp;&nbsp;
	- **差异2：合成逻辑**&nbsp;&nbsp;&nbsp;
		- 嘌呤 -> **先有核糖骨架再建碱基** -> 生成 IMP&nbsp;&nbsp;&nbsp;
		- 嘧啶 -> **先合成嘧啶环再连磷酸核糖** -> 生成 UMP&nbsp;&nbsp;&nbsp;
	- **差异3：关键酶** -> 嘌呤：**==PRPP 酰胺转移酶==** / 嘧啶：**==CPS-Ⅱ==**&nbsp;&nbsp;&nbsp;
	- **共同原料** -> **==天冬氨酸==** / **==谷氨酰胺==** / **==CO₂==** / **==PRPP==**&nbsp;&nbsp;&nbsp;
	- **共同点** -> 均需要 **PRPP** 作为磷酸核糖供体&nbsp;&nbsp;&nbsp;
