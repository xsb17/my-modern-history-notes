# cAMP-PKA 信号转导通路&nbsp;&nbsp;&nbsp;

- **cAMP-PKA 通路全流程**&nbsp;&nbsp;&nbsp;
	- **信号分子（第一信使）** -> **==肾上腺素==** / **==胰高血糖素==**&nbsp;&nbsp;&nbsp;
	- **受体类型** -> **==G 蛋白偶联型受体==**（与 Gₛ 偶联）&nbsp;&nbsp;&nbsp;
	- **G 蛋白** -> **==Gₛ==**（激活型 G 蛋白）-> αₛ-GTP 活化&nbsp;&nbsp;&nbsp;
	- **效应酶** -> **==腺苷酸环化酶 (AC)==** -> 催化 ATP → **==cAMP==**&nbsp;&nbsp;&nbsp;
	- **第二信使** -> **==cAMP==**（环腺苷酸）&nbsp;&nbsp;&nbsp;
	- **靶激酶** -> **==蛋白激酶 A (PKA)==**&nbsp;&nbsp;&nbsp;
- **PKA 的活化机制**&nbsp;&nbsp;&nbsp;
	- **结构** -> **==2R + 2C==**（2 个调节亚基 R + 2 个催化亚基 C）&nbsp;&nbsp;&nbsp;
	- 无 cAMP 时 -> R₂C₂ 四聚体 -> C 被 R **抑制**（无活性）&nbsp;&nbsp;&nbsp;
	- cAMP 升高 -> **==4 分子 cAMP==** 与 R 亚基结合 -> R 亚基解离 -> **C 亚基释放** -> PKA **活化**&nbsp;&nbsp;&nbsp;
	- PKA 是 **==Ser/Thr 蛋白激酶==**（磷酸化靶蛋白的丝氨酸/苏氨酸残基）&nbsp;&nbsp;&nbsp;
- **PKA 磷酸化的关键底物（知识整合必考！）**&nbsp;&nbsp;&nbsp;
	- **糖原合酶** -> 磷酸化 → **==抑制==** → 糖原合成↓&nbsp;&nbsp;&nbsp;
	- **糖原磷酸化酶 b 激酶** -> 磷酸化 → **==促进==** → 糖原分解↑&nbsp;&nbsp;&nbsp;
	- **激素敏感脂酶(HSL)** -> 磷酸化 → **==促进==** → 甘油三酯分解↑（脂肪动员）&nbsp;&nbsp;&nbsp;
	- **丙酮酸脱氢酶** -> 磷酸化 → **==抑制==** → 丙酮酸→乙酰CoA↓&nbsp;&nbsp;&nbsp;
	- **转录因子 CREB** -> 磷酸化 → 与 **==CRE==** 结合 → 调控基因表达（缓慢持久效应）&nbsp;&nbsp;&nbsp;
- **信号终止机制**&nbsp;&nbsp;&nbsp;
	- **==磷酸二酯酶(PDE)==** 水解 cAMP → 5′-AMP → PKA 失活&nbsp;&nbsp;&nbsp;
	- 蛋白磷酸酶 → 去磷酸化靶蛋白 → 效应终止&nbsp;&nbsp;&nbsp;
- **cAMP-PKA 通路一级流程速记**&nbsp;&nbsp;&nbsp;
	- **肾上腺素** → Gₛ → **==AC==** → **==cAMP==** → **==PKA==** → 磷酸化靶蛋白 → 生理效应&nbsp;&nbsp;&nbsp;
	- 生理效应总方向 -> **==分解↑ 合成↓==**（促糖原分解/脂肪动员/抑制糖原合成）&nbsp;&nbsp;&nbsp;
