# 酶相关受体信号通路&nbsp;&nbsp;&nbsp;

- **酶相关受体的两大类型**&nbsp;&nbsp;&nbsp;
	- **催化型受体** -> 自身具有**==酶活性==** -> 主要是**==酪氨酸蛋白激酶==**&nbsp;&nbsp;&nbsp;
		- 代表 -> **==RTK==**（受体型酪氨酸蛋白激酶）&nbsp;&nbsp;&nbsp;
		- 配体 -> **==生长因子==**（如 EGF/胰岛素）&nbsp;&nbsp;&nbsp;
		- 通路 -> **==Ras-MAPK==** 级联反应&nbsp;&nbsp;&nbsp;
	- **酶偶联型受体** -> 自身**==无酶活性==** -> 需偶联胞内**==酪氨酸蛋白激酶==**&nbsp;&nbsp;&nbsp;
		- 代表 -> **细胞因子受体**&nbsp;&nbsp;&nbsp;
		- 配体 -> **==细胞因子==**（如干扰素/白介素）&nbsp;&nbsp;&nbsp;
		- 通路 -> **==JAK-STAT==**&nbsp;&nbsp;&nbsp;
- **Ras-MAPK 通路详解**&nbsp;&nbsp;&nbsp;
	- **配体** -> 表皮生长因子(EGF) / 胰岛素等&nbsp;&nbsp;&nbsp;
	- **受体** -> **==RTK==**（受体型酪氨酸蛋白激酶）-> 跨膜**1次**&nbsp;&nbsp;&nbsp;
	- **活化过程**&nbsp;&nbsp;&nbsp;
		- 配体结合 → RTK **==二聚化==**&nbsp;&nbsp;&nbsp;
		- RTK 胞内域 **==自身磷酸化==**（Tyr 残基）&nbsp;&nbsp;&nbsp;
		- 接头蛋白 **==Grb2==**（含 ==SH2== 结构域）识别磷酸化 Tyr&nbsp;&nbsp;&nbsp;
		- Grb2 招募 **==Sos==**（鸟苷酸释放因子 GRF）&nbsp;&nbsp;&nbsp;
		- Sos 促使 **==Ras==** 释放 GDP 结合 GTP → **==Ras 活化==**&nbsp;&nbsp;&nbsp;
	- **MAPK 级联反应（三级激酶级联）**&nbsp;&nbsp;&nbsp;
		- Ras-GTP → 激活 **==Raf==**（MAPKKK）&nbsp;&nbsp;&nbsp;
		- Raf → 磷酸化 **==MEK==**（MAPKK）&nbsp;&nbsp;&nbsp;
		- MEK → 磷酸化 **==ERK==**（MAPK）&nbsp;&nbsp;&nbsp;
		- **ERK 入核** → 磷酸化**转录因子** → 调控基因表达 → 促进**==细胞增殖和分化==**&nbsp;&nbsp;&nbsp;
- **JAK-STAT 通路详解**&nbsp;&nbsp;&nbsp;
	- **配体** -> 干扰素 / 白介素等细胞因子&nbsp;&nbsp;&nbsp;
	- **受体** -> **酶偶联型受体**（自身无激酶活性）-> 与 **==JAK==** 偶联&nbsp;&nbsp;&nbsp;
	- **活化过程**&nbsp;&nbsp;&nbsp;
		- 细胞因子 → 受体 **==二聚化==**&nbsp;&nbsp;&nbsp;
		- 激活偶联的 **==JAK==**（Janus 激酶/Tyr 激酶）&nbsp;&nbsp;&nbsp;
		- JAK → **==磷酸化==**受体上的 Tyr 残基&nbsp;&nbsp;&nbsp;
		- **==STAT==**（信号转导子和转录激活子）通过 **==SH2 结构域==** 结合到受体磷酸化 Tyr&nbsp;&nbsp;&nbsp;
		- JAK → 磷酸化 STAT&nbsp;&nbsp;&nbsp;
		- 磷酸化 STAT → **==形成二聚体==** → **==入核==** → 结合 DNA → 调控基因转录&nbsp;&nbsp;&nbsp;
- **四大通路核心对比（考场速查）**&nbsp;&nbsp;&nbsp;
	- **cAMP-PKA** -> GPCR(Gₛ) → AC → **==cAMP==** → **==PKA==**&nbsp;&nbsp;&nbsp;
	- **PLC-IP₃/DAG-PKC** -> GPCR(Gq) → PLC → **==IP₃+DAG==** → **==PKC==**&nbsp;&nbsp;&nbsp;
	- **Ras-MAPK** -> **==RTK==**（催化型）→ Ras → Raf → MEK → **==ERK==**&nbsp;&nbsp;&nbsp;
	- **JAK-STAT** -> 酶偶联型受体 → **==JAK==** → **==STAT==** → 入核&nbsp;&nbsp;&nbsp;
- **Ras vs STAT 入核机制对比**&nbsp;&nbsp;&nbsp;
	- **Ras-MAPK** -> Ras 不直接入核 -> 通过 **==ERK 入核==** 磷酸化转录因子&nbsp;&nbsp;&nbsp;
	- **JAK-STAT** -> **==STAT 二聚体直接入核==** 结合 DNA&nbsp;&nbsp;&nbsp;
