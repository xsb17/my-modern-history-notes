# 常见的第二信使大全&nbsp;&nbsp;&nbsp;

- **第二信使的概念**&nbsp;&nbsp;&nbsp;
	- **定义** -> 细胞外信号（**第一信使**）与受体结合后 -> 在**==细胞内产生==**的传递信号的小分子&nbsp;&nbsp;&nbsp;
	- **特征** -> 浓度可**快速升高和降低** / 具有**==信号放大效应==**&nbsp;&nbsp;&nbsp;
- **四大类第二信使**&nbsp;&nbsp;&nbsp;
	- **==无机离子==**&nbsp;&nbsp;&nbsp;
		- **==Ca²⁺==**（钙离子）-> 最重要的无机离子第二信使&nbsp;&nbsp;&nbsp;
		- 来源 -> 内质网释放（IP₃ 介导）/ 胞外 Ca²⁺ 内流&nbsp;&nbsp;&nbsp;
		- 下游 -> 与**钙调蛋白(CaM)**结合 -> 激活 Ca²⁺/CaM 依赖性激酶&nbsp;&nbsp;&nbsp;
	- **==脂质衍生物==**&nbsp;&nbsp;&nbsp;
		- **==二脂酰甘油 (DAG)==** -> 由 PIP₂ 经 PLC 水解产生&nbsp;&nbsp;&nbsp;
		- 定位 -> 留在**细胞膜**上 -> 与 Ca²⁺ 共同激活 **==PKC==**&nbsp;&nbsp;&nbsp;
	- **==糖类衍生物==**&nbsp;&nbsp;&nbsp;
		- **==三磷酸肌醇 (IP₃)==** -> 由 PIP₂ 经 PLC 水解产生&nbsp;&nbsp;&nbsp;
		- 定位 -> 进入**胞液** -> 与内质网 IP₃ 受体结合 -> 促使内质网**==释放 Ca²⁺==**&nbsp;&nbsp;&nbsp;
	- **==核苷酸类==**&nbsp;&nbsp;&nbsp;
		- **==cAMP==**（环腺苷酸）&nbsp;&nbsp;&nbsp;
			- 合成 -> 腺苷酸环化酶(AC) 催化 ATP → cAMP&nbsp;&nbsp;&nbsp;
			- 降解 -> 磷酸二酯酶(PDE) 水解 cAMP → 5′-AMP&nbsp;&nbsp;&nbsp;
			- 靶激酶 -> **==PKA==**（蛋白激酶 A）&nbsp;&nbsp;&nbsp;
		- **==cGMP==**（环鸟苷酸）&nbsp;&nbsp;&nbsp;
			- 合成 -> 鸟苷酸环化酶(GC) 催化 GTP → cGMP&nbsp;&nbsp;&nbsp;
			- 靶激酶 -> **==PKG==**（蛋白激酶 G）&nbsp;&nbsp;&nbsp;
			- 特殊 -> 视杆细胞中 G_T 激活 cGMP 磷酸二酯酶 -> cGMP↓ → Na⁺通道关闭 → 感光&nbsp;&nbsp;&nbsp;
- **第二信使的来源汇总**&nbsp;&nbsp;&nbsp;
	- **cAMP** -> AC 催化 ATP -> 被 **==磷酸二酯酶(PDE)==** 降解&nbsp;&nbsp;&nbsp;
	- **cGMP** -> GC 催化 GTP -> 被 cGMP 磷酸二酯酶降解&nbsp;&nbsp;&nbsp;
	- **IP₃ + DAG** -> PLC 催化 **==PIP₂==** 水解产生 -> 一个底物两个信使&nbsp;&nbsp;&nbsp;
	- **Ca²⁺** -> 内质网释放（IP₃ 介导）/ 胞外内流&nbsp;&nbsp;&nbsp;
- **考试必背速记**&nbsp;&nbsp;&nbsp;
	- **无机离子** -> **==Ca²⁺==**&nbsp;&nbsp;&nbsp;
	- **脂质** -> **==DAG==**&nbsp;&nbsp;&nbsp;
	- **糖类** -> **==IP₃==**&nbsp;&nbsp;&nbsp;
	- **核苷酸** -> **==cAMP / cGMP==**&nbsp;&nbsp;&nbsp;
	- **PIP₂ 裂解产物** -> **==IP₃ + DAG==**（一对双胞胎第二信使）&nbsp;&nbsp;&nbsp;
