# G蛋白与小G蛋白的分类与功能&nbsp;&nbsp;&nbsp;

- **G 蛋白 (GTP Binding Protein) 概述**&nbsp;&nbsp;&nbsp;
	- **核心特征**（必背！）&nbsp;&nbsp;&nbsp;
		- 与 **==GTP 或 GDP==** 结合&nbsp;&nbsp;&nbsp;
		- **==α 亚基==** 具有 **==GTPase 活性==**（水解 GTP → GDP）&nbsp;&nbsp;&nbsp;
		- 组成 -> **α + β + γ** 三个亚基（异源三聚体）&nbsp;&nbsp;&nbsp;
	- **活化机制** -> 配体-受体 → G 蛋白释放 GDP 结合 GTP → α 亚基与 βγ 解离 → α-GTP **活化**&nbsp;&nbsp;&nbsp;
	- **失活机制** -> α 亚基 GTPase 水解 GTP → GDP → α-GDP 与 βγ 重新结合 → **失活**&nbsp;&nbsp;&nbsp;
- **六大经典 G 蛋白类型及其功能（必背！）**&nbsp;&nbsp;&nbsp;
	- **==Gₛ==**（stimulatory）&nbsp;&nbsp;&nbsp;
		- α 亚基 -> **αₛ**&nbsp;&nbsp;&nbsp;
		- 功能 -> **==激活腺苷酸环化酶(AC)==** / 激活 Ca²⁺ 通道&nbsp;&nbsp;&nbsp;
		- 所属通路 -> **cAMP-PKA**&nbsp;&nbsp;&nbsp;
		- 代表配体 -> 肾上腺素(β受体)/胰高血糖素&nbsp;&nbsp;&nbsp;
	- **==Gᵢ==**（inhibitory）&nbsp;&nbsp;&nbsp;
		- α 亚基 -> **αᵢ**&nbsp;&nbsp;&nbsp;
		- 功能 -> **==抑制腺苷酸环化酶(AC)==**&nbsp;&nbsp;&nbsp;
		- 与 Gₛ 相反 -> 降低 cAMP 水平&nbsp;&nbsp;&nbsp;
	- **==Gq==**（激活 PLC）&nbsp;&nbsp;&nbsp;
		- α 亚基 -> **αq**&nbsp;&nbsp;&nbsp;
		- 功能 -> **==激活磷脂酶 C(PLC)==**&nbsp;&nbsp;&nbsp;
		- 所属通路 -> **PLC-IP₃/DAG-PKC**&nbsp;&nbsp;&nbsp;
		- 代表配体 -> 促甲状腺素释放激素/去甲肾上腺素(α₁受体)&nbsp;&nbsp;&nbsp;
	- **==G_T==**（transducin/转导素）&nbsp;&nbsp;&nbsp;
		- α 亚基 -> **α_T**&nbsp;&nbsp;&nbsp;
		- 功能 -> 激活 **==cGMP 磷酸二酯酶==**（视杆细胞）&nbsp;&nbsp;&nbsp;
		- 作用 -> cGMP↓ → Na⁺通道关闭 → **感光**&nbsp;&nbsp;&nbsp;
	- **==G_olf==**（olfactory）&nbsp;&nbsp;&nbsp;
		- 功能 -> 激活腺苷酸环化酶（**嗅觉神经细胞**）&nbsp;&nbsp;&nbsp;
	- **==Gₒ==**（other）&nbsp;&nbsp;&nbsp;
		- 功能 -> **调节离子通道**&nbsp;&nbsp;&nbsp;
- **小 G 蛋白 (Small GTPase) 家族（必背！）**&nbsp;&nbsp;&nbsp;
	- **特征** -> 分子量约 ==21kDa== / 单体 / 也具有 GTPase 活性&nbsp;&nbsp;&nbsp;
	- **六大类型及其功能**&nbsp;&nbsp;&nbsp;
		- **==Ras==** -> 生长因子信号转导 -> **==Ras-MAPK 通路==**核心&nbsp;&nbsp;&nbsp;
		- **==Rab==** -> 囊泡**靶向输送和融合**&nbsp;&nbsp;&nbsp;
		- **==ARF==** -> 囊泡外层蛋白复合体&nbsp;&nbsp;&nbsp;
		- **==Ran==** -> 蛋白**出核和入核**&nbsp;&nbsp;&nbsp;
		- **==Rho==** -> 调节**肌动蛋白骨架**&nbsp;&nbsp;&nbsp;
		- **翻译起始/延长因子** -> 蛋白质生物合成&nbsp;&nbsp;&nbsp;
	- **Ras 活化机制**（RTK 通路核心）&nbsp;&nbsp;&nbsp;
		- Ras-GDP（**失活**）+ **鸟苷酸释放因子(GRF/Sos)** -> Ras-GTP（**活化**）&nbsp;&nbsp;&nbsp;
- **G 蛋白 vs 小 G 蛋白 核心区别**&nbsp;&nbsp;&nbsp;
	- **G 蛋白** -> **异源三聚体**(αβγ) / 分子量大 / 与 GPCR 偶联&nbsp;&nbsp;&nbsp;
	- **小 G 蛋白** -> **单体** / ~21kDa / 多种细胞功能（Ras/Rab/Ran/Rho等）&nbsp;&nbsp;&nbsp;
	- **共同点** -> 都具有 **==GTPase 活性==** / 活化型为 GTP 结合态&nbsp;&nbsp;&nbsp;
