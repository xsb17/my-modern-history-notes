# [全集存档] 生化第16章：细胞信号转导&nbsp;&nbsp;&nbsp;

## 第一节 细胞信号转导的分子基础&nbsp;&nbsp;&nbsp;

### 一、细胞外信号&nbsp;&nbsp;&nbsp;

#### 细胞外信号的作用形式&nbsp;&nbsp;&nbsp;

- **接触通讯**：膜结合型信号分子 → 直接接触靶细胞&nbsp;&nbsp;&nbsp;
- **旁分泌**：信号分子经扩散作用于邻近靶细胞&nbsp;&nbsp;&nbsp;
- **内分泌**：激素经血液循环运送到远距离靶细胞&nbsp;&nbsp;&nbsp;
- **突触传递**：神经递质经突触间隙传递到突触后膜&nbsp;&nbsp;&nbsp;
- **自分泌**：信号分子作用于分泌细胞自身&nbsp;&nbsp;&nbsp;

#### 细胞外信号分子的种类&nbsp;&nbsp;&nbsp;

- **水溶性信号分子**：蛋白质类、肽类、儿茶酚胺类激素等 → 必须结合**膜受体**&nbsp;&nbsp;&nbsp;
- **脂溶性信号分子**：类固醇类激素、前列腺素、甲状腺素、维生素 D₃ 等 → 结合**胞内受体**&nbsp;&nbsp;&nbsp;

### 二、受体&nbsp;&nbsp;&nbsp;

#### 受体的定义&nbsp;&nbsp;&nbsp;

- **受体(receptor)**：细胞膜或细胞内的天然分子，能识别和结合有生物活性的外源化学信号分子（配体），启动信号转导，产生生物学效应&nbsp;&nbsp;&nbsp;
- 受体的作用：==识别外源信号分子== + ==转换配体信号==&nbsp;&nbsp;&nbsp;

#### 受体与配体结合的特征（重点）&nbsp;&nbsp;&nbsp;

- **==特异性==**：受体选择性地与特定配体结合&nbsp;&nbsp;&nbsp;
- **==亲和性==**：解离常数 Kd 值**越小**表示亲和力**越高**&nbsp;&nbsp;&nbsp;
- **==饱和性==**：受体与配体的结合曲线呈矩形双曲线&nbsp;&nbsp;&nbsp;
- **==可逆性==**：受体与配体以**非共价键**结合，两者的结合可逆&nbsp;&nbsp;&nbsp;
- **产生特定的生理效应**&nbsp;&nbsp;&nbsp;

#### 三类受体的特征对比&nbsp;&nbsp;&nbsp;

##### 离子通道型受体&nbsp;&nbsp;&nbsp;

- 配体：神经递质&nbsp;&nbsp;&nbsp;
- 结构：寡聚体形成的孔道，跨膜 ==4 次==&nbsp;&nbsp;&nbsp;
- 功能：离子通道&nbsp;&nbsp;&nbsp;
- 细胞应答：去极化与超极化&nbsp;&nbsp;&nbsp;

##### G蛋白偶联型受体&nbsp;&nbsp;&nbsp;

- 配体：神经递质、激素、外源刺激（光、味）&nbsp;&nbsp;&nbsp;
- 结构：单体，跨膜 ==7 次==&nbsp;&nbsp;&nbsp;
- 功能：激活 G 蛋白&nbsp;&nbsp;&nbsp;
- 细胞应答：去极化与超极化，调节蛋白功能和表达水平&nbsp;&nbsp;&nbsp;

##### 催化型和酶偶联型受体&nbsp;&nbsp;&nbsp;

- 配体：生长因子、细胞因子&nbsp;&nbsp;&nbsp;
- 结构：单体，跨膜 ==1 次==&nbsp;&nbsp;&nbsp;
- 功能：激活酪氨酸蛋白激酶等&nbsp;&nbsp;&nbsp;
- 细胞应答：调节蛋白功能和表达水平，调节细胞增殖和分化&nbsp;&nbsp;&nbsp;

### 三、常见的细胞内信号转导分子&nbsp;&nbsp;&nbsp;

#### 第二信使的概念&nbsp;&nbsp;&nbsp;

- **第二信使(second messengers)**：细胞外信号分子（第一信使）与受体结合后，在细胞内产生的传递信号的小分子物质&nbsp;&nbsp;&nbsp;
- 特点：浓度可快速升高和降低；具有信号放大效应&nbsp;&nbsp;&nbsp;
- 主要包括：==cAMP、cGMP、DAG、IP₃、Ca²⁺==等&nbsp;&nbsp;&nbsp;

#### 第二信使的分类&nbsp;&nbsp;&nbsp;

- **无机离子**：==Ca²⁺==&nbsp;&nbsp;&nbsp;
- **脂质衍生物**：==二脂酰甘油(DAG)==&nbsp;&nbsp;&nbsp;
- **糖类衍生物**：==三磷酸肌醇(IP₃)==&nbsp;&nbsp;&nbsp;
- **核苷酸类**：==cAMP、cGMP==&nbsp;&nbsp;&nbsp;

#### 其他细胞内信号转导分子&nbsp;&nbsp;&nbsp;

- **G 蛋白**（GTP binding protein）：与 GTP 或 GDP 结合；α 亚基具有 GTPase 活性&nbsp;&nbsp;&nbsp;
- **小 G 蛋白**（small GTP binding protein）：约 21kDa，如 Ras 参与生长因子信号转导&nbsp;&nbsp;&nbsp;
- **蛋白激酶/蛋白磷酸酶系统**&nbsp;&nbsp;&nbsp;
- **接头蛋白和支架蛋白**&nbsp;&nbsp;&nbsp;

## 第二节 常见细胞信号转导通路&nbsp;&nbsp;&nbsp;

### 一、cAMP-PKA 信号通路&nbsp;&nbsp;&nbsp;

- **信号分子**：肾上腺素、胰高血糖素等 → 与 ==G 蛋白偶联型受体==结合&nbsp;&nbsp;&nbsp;
- **G 蛋白类型**：==Gₛ==（激活型 G 蛋白）&nbsp;&nbsp;&nbsp;
- **效应酶**：==腺苷酸环化酶(AC)== → 催化 ATP → **==cAMP==**&nbsp;&nbsp;&nbsp;
- **靶激酶**：==蛋白激酶 A(PKA)==&nbsp;&nbsp;&nbsp;
  - PKA 由 2 个调节亚基(R) + 2 个催化亚基(C)组成&nbsp;&nbsp;&nbsp;
  - cAMP 与 R 亚基结合 → R 亚基解离 → C 亚基释放 → 活化&nbsp;&nbsp;&nbsp;
  - PKA 是 Ser/Thr 蛋白激酶&nbsp;&nbsp;&nbsp;
- **PKA 磷酸化的底物**：&nbsp;&nbsp;&nbsp;
  - 糖原合酶 → 抑制糖原合成&nbsp;&nbsp;&nbsp;
  - 糖原磷酸化酶 b 激酶 → 促进糖原分解&nbsp;&nbsp;&nbsp;
  - 激素敏感脂酶 → 促进甘油三酯分解和脂肪酸氧化&nbsp;&nbsp;&nbsp;
  - 丙酮酸脱氢酶 → 抑制丙酮酸 → 乙酰CoA&nbsp;&nbsp;&nbsp;
  - 转录因子 CREB → 调控基因表达&nbsp;&nbsp;&nbsp;

### 二、PLC-IP₃/DAG-PKC 信号通路&nbsp;&nbsp;&nbsp;

- **信号分子**：促甲状腺素释放激素、去甲肾上腺素等&nbsp;&nbsp;&nbsp;
- **G 蛋白类型**：==Gq== → 激活 ==磷脂酶 C(PLC)==&nbsp;&nbsp;&nbsp;
- **PLC 催化**：==PIP₂==（磷脂酰肌醇-4,5-二磷酸）→ **==IP₃==** + **==DAG==**&nbsp;&nbsp;&nbsp;
- **IP₃ 的作用**：进入胞液 → 与内质网 IP₃ 受体结合 → 内质网释放 ==Ca²⁺== → 胞液 [Ca²⁺] 升高&nbsp;&nbsp;&nbsp;
- **DAG 的作用**：留在细胞膜上 → 与 Ca²⁺ 共同激活 ==蛋白激酶 C(PKC)==&nbsp;&nbsp;&nbsp;
- **Ca²⁺ 的其他作用**：与 ==钙调蛋白(CaM)== 结合 → Ca²⁺/CaM 复合物 → 激活 Ca²⁺/CaM 依赖性激酶(CaMK)等多种蛋白&nbsp;&nbsp;&nbsp;

### 三、Ras-MAPK 信号通路&nbsp;&nbsp;&nbsp;

- **受体类型**：==催化型受体（RTK）==→ 具有酪氨酸蛋白激酶活性&nbsp;&nbsp;&nbsp;
- **信号分子**：表皮生长因子(EGF)、胰岛素等&nbsp;&nbsp;&nbsp;
- **通路过程**：&nbsp;&nbsp;&nbsp;
  - 配体 → RTK 二聚化 → 自身磷酸化（Tyr 残基）&nbsp;&nbsp;&nbsp;
  - 接头蛋白 Grb2（含 SH2 结构域）识别磷酸化 Tyr&nbsp;&nbsp;&nbsp;
  - Sos（鸟苷酸释放因子 GRF）→ 促使 Ras 释放 GDP 结合 GTP → ==Ras 活化==&nbsp;&nbsp;&nbsp;
  - Ras-GTP → 激活 Raf → MEK → ERK（MAPK 级联反应）&nbsp;&nbsp;&nbsp;
  - ERK 入核 → 磷酸化转录因子 → 调控基因表达 → 细胞增殖、分化&nbsp;&nbsp;&nbsp;

### 四、JAK-STAT 信号通路&nbsp;&nbsp;&nbsp;

- **受体类型**：==酶偶联型受体==（细胞因子受体，自身无激酶活性）&nbsp;&nbsp;&nbsp;
- **信号分子**：干扰素、白介素等细胞因子&nbsp;&nbsp;&nbsp;
- **通路过程**：&nbsp;&nbsp;&nbsp;
  - 细胞因子 → 受体二聚化 → 激活偶联的 ==JAK==(Janus 激酶)&nbsp;&nbsp;&nbsp;
  - JAK → 磷酸化受体上的 Tyr 残基&nbsp;&nbsp;&nbsp;
  - ==STAT==(signal transducers and activators of transcription) → 通过 SH2 结构域结合到受体 → 被 JAK 磷酸化&nbsp;&nbsp;&nbsp;
  - 磷酸化 STAT 形成二聚体 → 入核 → 调控基因转录&nbsp;&nbsp;&nbsp;

### 五、各通路信号分子-受体-效应酶-靶激酶速查&nbsp;&nbsp;&nbsp;

| 通路 | 受体类型 | G蛋白 | 效应器 | 第二信使 | 靶激酶 |
|---|---|---|---|---|---|
| cAMP-PKA | G蛋白偶联型(Gₛ) | Gₛ | 腺苷酸环化酶(AC) | cAMP | PKA |
| PLC-IP₃/DAG-PKC | G蛋白偶联型(Gq) | Gq | 磷脂酶C(PLC) | IP₃/DAG/Ca²⁺ | PKC |
| Ras-MAPK | 催化型(RTK) | Ras(小G) | — | — | MAPK级联 |
| JAK-STAT | 酶偶联型 | — | JAK | — | STAT |
| cGMP-PKG | 催化型(鸟苷酸环化酶) | — | 鸟苷酸环化酶 | cGMP | PKG |

## 第三节 细胞信号转导的特点和规律&nbsp;&nbsp;&nbsp;

- **信号转导网络(signal transduction network)**：多种信号通路相互交叉、相互调节&nbsp;&nbsp;&nbsp;
- **cross-talking**：GPCR 和 RTK 通路之间的交互对话&nbsp;&nbsp;&nbsp;
- 介导信号分子相互作用和区域化分布&nbsp;&nbsp;&nbsp;
- 共同调节模块中的信号蛋白&nbsp;&nbsp;&nbsp;

## 第四节 细胞信号转导异常与疾病&nbsp;&nbsp;&nbsp;

- 信号转导异常可发生于多个环节：胞外信号分子 → 受体 → 胞内信号转导分子 → 效应分子&nbsp;&nbsp;&nbsp;
- 信号转导异常与多种疾病相关（如肿瘤中 Ras 基因突变导致持续活化）&nbsp;&nbsp;&nbsp;

## 重要概念&nbsp;&nbsp;&nbsp;

- **细胞信号转导**：细胞对环境信号的应答，启动胞内信号转导通路，调节生理代谢反应和基因表达&nbsp;&nbsp;&nbsp;
- **受体**：能识别和结合配体，启动信号转导产生生物学效应的天然分子&nbsp;&nbsp;&nbsp;
- **第二信使**：细胞外信号与受体结合后在细胞内产生的传递信号的小分子（cAMP/cGMP/DAG/IP₃/Ca²⁺）&nbsp;&nbsp;&nbsp;
- **G 蛋白**：与 GTP 或 GDP 结合的蛋白，α 亚基具有 GTPase 活性，介导受体与效应器之间的信号转导&nbsp;&nbsp;&nbsp;
- **蛋白激酶**：催化蛋白质磷酸化的酶（PKA/PKC/MAPK/JAK 等）&nbsp;&nbsp;&nbsp;
- **信号转导级联反应**：信号经多级蛋白激酶逐级放大传递的过程&nbsp;&nbsp;&nbsp;
