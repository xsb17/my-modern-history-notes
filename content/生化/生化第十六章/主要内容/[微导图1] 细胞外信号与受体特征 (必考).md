# 细胞外信号与受体特征 (必考)&nbsp;&nbsp;&nbsp;

- **细胞通讯方式**&nbsp;&nbsp;&nbsp;
	- **==接触通讯==** -> 膜结合型信号分子 -> 直接接触靶细胞&nbsp;&nbsp;&nbsp;
	- **==旁分泌==** -> 信号分子经**扩散** -> 作用于邻近靶细胞&nbsp;&nbsp;&nbsp;
	- **==内分泌==** -> 激素经**血液循环** -> 远距离靶细胞&nbsp;&nbsp;&nbsp;
	- **==突触传递==** -> 神经递质经**突触间隙** -> 突触后膜&nbsp;&nbsp;&nbsp;
	- **==自分泌==** -> 信号作用于**分泌细胞自身**&nbsp;&nbsp;&nbsp;
- **信号分子按溶解性分类**&nbsp;&nbsp;&nbsp;
	- **水溶性信号分子** -> **==必须结合膜受体==**（不能自由通过细胞膜）&nbsp;&nbsp;&nbsp;
		- 包括 -> 蛋白质类 / 肽类 / 儿茶酚胺类激素（如**肾上腺素**）&nbsp;&nbsp;&nbsp;
	- **脂溶性信号分子** -> **==结合胞内受体==**（可自由通过细胞膜）&nbsp;&nbsp;&nbsp;
		- 包括 -> **==类固醇激素 / 甲状腺素 / 前列腺素 / Vit D₃==**&nbsp;&nbsp;&nbsp;
- **受体的定义与作用**&nbsp;&nbsp;&nbsp;
	- **受体(receptor)** -> 识别和结合**配体**的天然分子 -> 启动信号转导 -> 产生生物学效应&nbsp;&nbsp;&nbsp;
	- 两大作用 -> **==识别外源信号分子==** + **==转换配体信号==**&nbsp;&nbsp;&nbsp;
- **受体与配体结合的五大特征（必背！）**&nbsp;&nbsp;&nbsp;
	- **==特异性==** -> 受体**选择性**地与特定配体结合&nbsp;&nbsp;&nbsp;
	- **==亲和性==** -> **==Kd 值越小==** -> 亲和力**越高**&nbsp;&nbsp;&nbsp;
	- **==饱和性==** -> 结合曲线呈**==矩形双曲线==**（受体数量有限）&nbsp;&nbsp;&nbsp;
	- **==可逆性==** -> 受体与配体以**==非共价键==**结合 -> 可解离&nbsp;&nbsp;&nbsp;
	- **==产生特定的生理效应==** -> 结合后启动信号转导 → 细胞应答&nbsp;&nbsp;&nbsp;
- **三类受体的跨膜次数速记**&nbsp;&nbsp;&nbsp;
	- **离子通道型** -> 跨膜 **==4 次==** -> 配体：神经递质&nbsp;&nbsp;&nbsp;
	- **G 蛋白偶联型** -> 跨膜 **==7 次==** -> 配体：神经递质/激素/光/味&nbsp;&nbsp;&nbsp;
	- **催化型/酶偶联型** -> 跨膜 **==1 次==** -> 配体：生长因子/细胞因子&nbsp;&nbsp;&nbsp;
