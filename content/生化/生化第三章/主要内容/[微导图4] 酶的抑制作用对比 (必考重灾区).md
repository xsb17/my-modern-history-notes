# 酶的抑制作用对比 (必考重灾区)&nbsp;&nbsp;&nbsp;

- **抑制剂**&nbsp;&nbsp;&nbsp;
	- 使酶催化活性**下降**但不引起构象**非常显著**变化的物质&nbsp;&nbsp;&nbsp;
	- 与变性剂的区别 -> 抑制剂有**选择性**，变性剂无选择性&nbsp;&nbsp;&nbsp;
- **抑制作用分类**&nbsp;&nbsp;&nbsp;
	- 按结合紧密程度分&nbsp;&nbsp;&nbsp;
		- **不可逆抑制** -> **共价键**结合 -> 透析/超滤**不能**除去&nbsp;&nbsp;&nbsp;
		- **可逆抑制** -> **非共价键**结合 -> 透析/超滤**可以**除去&nbsp;&nbsp;&nbsp;
- **不可逆抑制作用**&nbsp;&nbsp;&nbsp;
	- **有机磷农药**中毒&nbsp;&nbsp;&nbsp;
		- 靶酶：**乙酰胆碱酯酶**&nbsp;&nbsp;&nbsp;
		- 机制 -> 与活性中心**丝氨酸残基的羟基**共价结合 -> 酶失活&nbsp;&nbsp;&nbsp;
		- 后果 -> **乙酰胆碱堆积** -> 迷走神经兴奋 -> 恶心呕吐、多汗、肌肉震颤、瞳孔缩小&nbsp;&nbsp;&nbsp;
		- 解毒剂1 -> **阿托品**（乙酰胆碱拮抗剂）&nbsp;&nbsp;&nbsp;
		- 解毒剂2 -> **解磷定(PAM)**（胆碱酯酶复活剂）&nbsp;&nbsp;&nbsp;
	- **重金属离子**中毒&nbsp;&nbsp;&nbsp;
		- 代表离子：**Hg²⁺**、**Ag⁺**、**As³⁺**&nbsp;&nbsp;&nbsp;
		- 机制 -> 与酶分子**巯基(-SH)** 结合 -> 酶失活&nbsp;&nbsp;&nbsp;
		- 解毒剂 -> **二巯基丁二酸钠**&nbsp;&nbsp;&nbsp;
	- **路易士气**（含砷化合物）中毒&nbsp;&nbsp;&nbsp;
		- 机制 -> 与酶**巯基**结合 -> 失活&nbsp;&nbsp;&nbsp;
		- 解毒剂 -> **二巯基丙醇(BAL)**&nbsp;&nbsp;&nbsp;
- **可逆抑制作用对比（必背）**&nbsp;&nbsp;&nbsp;
	- **竞争性抑制**&nbsp;&nbsp;&nbsp;
		- 结合部位 -> 与 **E** 结合（活性中心）&nbsp;&nbsp;&nbsp;
		- 抑制剂与底物关系 -> 结构**相似** -> 竞争**活性中心**&nbsp;&nbsp;&nbsp;
		- 能由增加[S]解除 -> **是**&nbsp;&nbsp;&nbsp;
		- **Vmax** -> **不变**&nbsp;&nbsp;&nbsp;
		- **Km** -> **增大**&nbsp;&nbsp;&nbsp;
		- 林-贝氏图斜率 -> 增大&nbsp;&nbsp;&nbsp;
		- 纵轴截距(1/Vmax) -> 不变&nbsp;&nbsp;&nbsp;
		- 横轴截距(-1/Km) -> 增大&nbsp;&nbsp;&nbsp;
		- 实例 -> **磺胺类药物** vs **PABA** 竞争**二氢蝶酸合酶**&nbsp;&nbsp;&nbsp;
		- 实例 -> **丙二酸** vs 琥珀酸竞争**琥珀酸脱氢酶**&nbsp;&nbsp;&nbsp;
	- **非竞争性抑制**&nbsp;&nbsp;&nbsp;
		- 结合部位 -> 与 **E 和 ES** 均结合（活性中心**外**）&nbsp;&nbsp;&nbsp;
		- 抑制剂与底物关系 -> **无竞争关系**&nbsp;&nbsp;&nbsp;
		- 能由增加[S]解除 -> **否**（取决于[I]）&nbsp;&nbsp;&nbsp;
		- **Vmax** -> **降低**&nbsp;&nbsp;&nbsp;
		- **Km** -> **不变**&nbsp;&nbsp;&nbsp;
		- 林-贝氏图斜率 -> 增大&nbsp;&nbsp;&nbsp;
		- 纵轴截距(1/Vmax) -> 增大&nbsp;&nbsp;&nbsp;
		- 横轴截距(-1/Km) -> 不变&nbsp;&nbsp;&nbsp;
	- **反竞争性抑制**&nbsp;&nbsp;&nbsp;
		- 结合部位 -> 只与 **ES** 结合&nbsp;&nbsp;&nbsp;
		- 抑制剂与底物关系 -> 需先形成ES复合物&nbsp;&nbsp;&nbsp;
		- 能由增加[S]解除 -> **否**&nbsp;&nbsp;&nbsp;
		- **Vmax** -> **降低**&nbsp;&nbsp;&nbsp;
		- **Km** -> **降低**&nbsp;&nbsp;&nbsp;
		- 林-贝氏图斜率 -> 不变&nbsp;&nbsp;&nbsp;
		- 纵轴截距(1/Vmax) -> 增大&nbsp;&nbsp;&nbsp;
		- 横轴截距(-1/Km) -> 增大&nbsp;&nbsp;&nbsp;
- **三种可逆抑制口诀记忆**&nbsp;&nbsp;&nbsp;
	- **竞大不不变（竞：Km↑ Vmax不变）**&nbsp;&nbsp;&nbsp;
	- **非不小不变（非：Vmax↓ Km不变）**&nbsp;&nbsp;&nbsp;
	- **反都小（反：Vmax↓ Km↓）**&nbsp;&nbsp;&nbsp;
- **激活剂**&nbsp;&nbsp;&nbsp;
	- 使酶从无活性→有活性或活性增加的物质&nbsp;&nbsp;&nbsp;
	- **必需激活剂** -> 没有就不工作&nbsp;&nbsp;&nbsp;
	- **非必需激活剂** -> 没有也有一定活性&nbsp;&nbsp;&nbsp;
	- 实例 -> Mg²⁺、K⁺、Mn²⁺、Cl⁻、胆汁酸盐&nbsp;&nbsp;&nbsp;
