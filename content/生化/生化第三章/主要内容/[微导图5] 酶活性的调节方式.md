# 酶活性的调节方式&nbsp;&nbsp;&nbsp;

- **酶的调节总览**&nbsp;&nbsp;&nbsp;
	- **酶活性调节（快速调节）**&nbsp;&nbsp;&nbsp;
		- 别构调节&nbsp;&nbsp;&nbsp;
		- 化学修饰调节&nbsp;&nbsp;&nbsp;
		- 酶原激活&nbsp;&nbsp;&nbsp;
	- **酶含量调节（慢速调节）**&nbsp;&nbsp;&nbsp;
		- 酶蛋白合成的诱导与阻遏&nbsp;&nbsp;&nbsp;
		- 酶蛋白降解&nbsp;&nbsp;&nbsp;
- **别构调节**&nbsp;&nbsp;&nbsp;
	- 小分子物质与酶活性中心或活性中心**外**的特定部位**可逆**结合&nbsp;&nbsp;&nbsp;
	- 结合后 -> 酶构象变化 -> 活性改变（激活或抑制）&nbsp;&nbsp;&nbsp;
	- **别构酶** -> 受别构调节的酶&nbsp;&nbsp;&nbsp;
	- **别构效应剂** -> 引起别构效应的小分子&nbsp;&nbsp;&nbsp;
	- **别构部位** -> 效应剂结合的部位&nbsp;&nbsp;&nbsp;
	- 别构调节特点&nbsp;&nbsp;&nbsp;
		- 大多为**寡聚酶** -> 含 ≥2 个亚基&nbsp;&nbsp;&nbsp;
		- 催化部位和调节部位可在**同一亚基**或**不同亚基**&nbsp;&nbsp;&nbsp;
		- 具有**协同效应**&nbsp;&nbsp;&nbsp;
			- **正协同效应** -> 一个催化部位结合后 -> 其它部位对底物亲和力**增强**&nbsp;&nbsp;&nbsp;
			- **负协同效应** -> 一个催化部位结合后 -> 其它部位对底物亲和力**降低**&nbsp;&nbsp;&nbsp;
		- 动力学曲线 -> **S形**曲线 -> **不符合**米氏方程&nbsp;&nbsp;&nbsp;
		- 催化**非平衡反应** -> 多为代谢途径**关键酶**&nbsp;&nbsp;&nbsp;
- **化学修饰调节（共价修饰）**&nbsp;&nbsp;&nbsp;
	- 酶蛋白氨基酸残基与基团在另一酶催化下**共价结合**或**脱去** -> 活性改变&nbsp;&nbsp;&nbsp;
	- 最常见方式 -> **磷酸化与去磷酸化**&nbsp;&nbsp;&nbsp;
	- 其它方式&nbsp;&nbsp;&nbsp;
		- 乙酰化与去乙酰化&nbsp;&nbsp;&nbsp;
		- 甲基化与去甲基化&nbsp;&nbsp;&nbsp;
		- 腺苷化与去腺苷化&nbsp;&nbsp;&nbsp;
		- **-SH 与 -S-S- 互变**&nbsp;&nbsp;&nbsp;
	- 属于**共价键**改变&nbsp;&nbsp;&nbsp;
- **酶原激活**&nbsp;&nbsp;&nbsp;
	- **酶原** -> 酶在细胞内合成时的**无活性前体**&nbsp;&nbsp;&nbsp;
	- **酶原激活** -> 无活性酶原→有活性酶的过程&nbsp;&nbsp;&nbsp;
	- 激活本质 -> **活性中心形成或暴露**&nbsp;&nbsp;&nbsp;
	- 激活机理&nbsp;&nbsp;&nbsp;
		- 酶原 -> 特定条件下**肽键断裂** -> 水解掉短肽 -> 构象改变 -> 活性中心形成/暴露&nbsp;&nbsp;&nbsp;
	- 经典级联实例&nbsp;&nbsp;&nbsp;
		- **胰蛋白酶原** ->（**肠激酶**）-> **胰蛋白酶**&nbsp;&nbsp;&nbsp;
		- 胰蛋白酶 -> 继而激活：糜蛋白酶原 → 糜蛋白酶 / 弹性蛋白酶原 → 弹性蛋白酶 / 羧基肽酶原 → 羧基肽酶&nbsp;&nbsp;&nbsp;
	- 生理意义&nbsp;&nbsp;&nbsp;
		- 保证酶在**特定部位和环境下**发挥催化作用&nbsp;&nbsp;&nbsp;
		- 避免组织**自身消化**（如消化酶原、凝血酶原）&nbsp;&nbsp;&nbsp;
		- 酶原可视为酶的**贮存形式**&nbsp;&nbsp;&nbsp;
- **酶含量的调节（慢速调节）**&nbsp;&nbsp;&nbsp;
	- **酶蛋白合成诱导** -> 转录水平促进酶合成&nbsp;&nbsp;&nbsp;
	- **酶蛋白合成阻遏** -> 转录水平减少酶合成&nbsp;&nbsp;&nbsp;
	- **酶蛋白降解**&nbsp;&nbsp;&nbsp;
		- 溶酶体途径（不依赖 ATP）&nbsp;&nbsp;&nbsp;
		- 非溶酶体途径（依赖 **ATP** + **泛素**）&nbsp;&nbsp;&nbsp;
