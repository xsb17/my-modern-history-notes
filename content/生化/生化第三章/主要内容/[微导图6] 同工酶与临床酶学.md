# 同工酶与临床酶学&nbsp;&nbsp;&nbsp;

- **同工酶概念**&nbsp;&nbsp;&nbsp;
	- 催化**相同**化学反应&nbsp;&nbsp;&nbsp;
	- 酶蛋白的**分子结构**不同&nbsp;&nbsp;&nbsp;
	- **理化性质**不同&nbsp;&nbsp;&nbsp;
	- **免疫学性质**不同&nbsp;&nbsp;&nbsp;
	- **组织学分布**不同&nbsp;&nbsp;&nbsp;
	- 存在于同一个体**不同组织**或同一细胞**不同亚细胞结构**&nbsp;&nbsp;&nbsp;
	- 临床意义 -> 用于**临床诊断**&nbsp;&nbsp;&nbsp;
- **乳酸脱氢酶(LDH) — 最经典同工酶**&nbsp;&nbsp;&nbsp;
	- 结构 -> **四聚体酶** -> 最先发现的同工酶&nbsp;&nbsp;&nbsp;
	- 亚基类型&nbsp;&nbsp;&nbsp;
		- **M 型**（骨骼肌型）&nbsp;&nbsp;&nbsp;
		- **H 型**（心肌型）&nbsp;&nbsp;&nbsp;
	- 五种同工酶 LDH1~LDH5&nbsp;&nbsp;&nbsp;
		- **LDH1** -> **H₄** -> **心肌**含量最高&nbsp;&nbsp;&nbsp;
		- **LDH2** -> H₃M&nbsp;&nbsp;&nbsp;
		- **LDH3** -> H₂M₂&nbsp;&nbsp;&nbsp;
		- **LDH4** -> HM₃&nbsp;&nbsp;&nbsp;
		- **LDH5** -> **M₄** -> **骨骼肌**和**肝**含量最高&nbsp;&nbsp;&nbsp;
	- 组织分布&nbsp;&nbsp;&nbsp;
		- **心** -> LDH1(35-70%) > LDH2(28-45%)&nbsp;&nbsp;&nbsp;
		- **肝** -> LDH5(8-30%) 相对较高&nbsp;&nbsp;&nbsp;
		- **骨骼肌** -> LDH5(40-97%) 占绝对优势&nbsp;&nbsp;&nbsp;
		- **正常血清** -> LDH2 > LDH1 > LDH3 > LDH4 > LDH5&nbsp;&nbsp;&nbsp;
	- 临床意义&nbsp;&nbsp;&nbsp;
		- **心肌梗死** -> **LDH1** 升高，LDH1 > LDH2（翻转现象）&nbsp;&nbsp;&nbsp;
		- **肝病/骨骼肌损伤** -> **LDH5** 升高&nbsp;&nbsp;&nbsp;
- **酶的分类（7大类）**&nbsp;&nbsp;&nbsp;
	- **氧化还原酶类** -> 催化氧化还原 -> 如：乳酸脱氢酶&nbsp;&nbsp;&nbsp;
	- **转移酶类** -> 催化基团转移或交换 -> 如：氨基转移酶&nbsp;&nbsp;&nbsp;
	- **水解酶类** -> 催化水解反应 -> 如：淀粉酶、脂肪酶&nbsp;&nbsp;&nbsp;
	- **裂解酶类** -> 移去基团留下双键或其逆反应 -> 如：醛缩酶&nbsp;&nbsp;&nbsp;
	- **异构酶类** -> 催化同分异构体互变 -> 如：磷酸丙糖异构酶&nbsp;&nbsp;&nbsp;
	- **连接酶/合成酶类** -> 偶联**ATP**断裂的两分子合成 -> 合成酶(需ATP) ≠ 合酶(不需ATP)&nbsp;&nbsp;&nbsp;
	- **易位酶** -> 催化跨膜转运或膜内易位 -> 如：ATP合酶&nbsp;&nbsp;&nbsp;
- **酶活性单位**&nbsp;&nbsp;&nbsp;
	- **国际单位(U)** -> 每分钟催化 **1μmol** 底物转化为产物&nbsp;&nbsp;&nbsp;
	- **催量单位(kat)** -> 每秒钟催化 **1mol** 底物转化为产物&nbsp;&nbsp;&nbsp;
	- 换算：**1 kat = 6.0 × 10⁷ U**&nbsp;&nbsp;&nbsp;
