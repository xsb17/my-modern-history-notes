# 维生素与对应的辅酶&nbsp;&nbsp;&nbsp;

- **维生素总览**&nbsp;&nbsp;&nbsp;
	- 维持正常生理功能必需的微量有机物 -> 体内**不能合成或合成量很少** -> 必须食物供给&nbsp;&nbsp;&nbsp;
	- **脂溶性** -> VitA、D、E、K -> 不溶于水，与脂类共吸收，排泄慢&nbsp;&nbsp;&nbsp;
	- **水溶性** -> B族维生素 + VitC -> 易随尿排出，需经常摄取，主要构成辅酶/辅基&nbsp;&nbsp;&nbsp;
- **维生素 B₁（硫胺素）**&nbsp;&nbsp;&nbsp;
	- 活性形式 -> **TPP**（焦磷酸硫胺素）&nbsp;&nbsp;&nbsp;
	- 辅酶功能 -> **α-酮酸氧化脱羧酶**辅酶、**转酮醇酶**辅酶&nbsp;&nbsp;&nbsp;
	- 缺乏症 -> **脚气病**、末梢神经炎&nbsp;&nbsp;&nbsp;
- **维生素 B₂（核黄素）**&nbsp;&nbsp;&nbsp;
	- 活性形式 -> **FMN**（黄素单核苷酸）、**FAD**（黄素腺嘌呤二核苷酸）&nbsp;&nbsp;&nbsp;
	- 辅酶功能 -> 氧化还原酶的**辅基** -> 传递**氢**（氢传递体）&nbsp;&nbsp;&nbsp;
	- 缺乏症 -> **口角炎、唇炎、阴囊炎、眼睑炎**&nbsp;&nbsp;&nbsp;
- **维生素 PP（尼克酸/尼克酰胺）**&nbsp;&nbsp;&nbsp;
	- 活性形式 -> **NAD⁺**、**NADP⁺**&nbsp;&nbsp;&nbsp;
	- 辅酶功能 -> 不需氧脱氢酶的**辅酶** -> 传递**氢**&nbsp;&nbsp;&nbsp;
	- 缺乏症 -> **糙皮病（赖皮病）** -> 皮炎、腹泻、痴呆&nbsp;&nbsp;&nbsp;
- **维生素 B₆（吡哆醛/吡哆胺）**&nbsp;&nbsp;&nbsp;
	- 活性形式 -> **磷酸吡哆醛**、**磷酸吡哆胺**&nbsp;&nbsp;&nbsp;
	- 辅酶功能 -> 氨基酸**转氨酶**和**脱羧酶**的辅酶 -> **ALA合酶**的辅酶 -> 同型半胱氨酸代谢酶辅酶&nbsp;&nbsp;&nbsp;
	- 缺乏症 -> 少见（低血色素小细胞性贫血、高同型半胱氨酸血症）&nbsp;&nbsp;&nbsp;
- **泛酸（遍多酸）**&nbsp;&nbsp;&nbsp;
	- 活性形式 -> **CoA**（辅酶A）、**ACP**（酰基载体蛋白）&nbsp;&nbsp;&nbsp;
	- 辅酶功能 -> 酰基转移酶辅酶 -> 传递**酰基**&nbsp;&nbsp;&nbsp;
	- 缺乏症 -> 少见&nbsp;&nbsp;&nbsp;
- **生物素（维生素H/B₇）**&nbsp;&nbsp;&nbsp;
	- 活性形式 -> **生物素**（本身就是活性形式）&nbsp;&nbsp;&nbsp;
	- 辅酶功能 -> **羧化酶**辅基 -> 参与 **CO₂ 固定**&nbsp;&nbsp;&nbsp;
	- 缺乏症 -> 少见&nbsp;&nbsp;&nbsp;
- **叶酸**&nbsp;&nbsp;&nbsp;
	- 活性形式 -> **FH₄**（四氢叶酸）&nbsp;&nbsp;&nbsp;
	- 辅酶功能 -> **一碳单位**载体 -> 一碳单位转移酶辅酶 -> 参与嘌呤和胸腺嘧啶核苷酸合成&nbsp;&nbsp;&nbsp;
	- N⁵、N¹⁰ 是一碳单位结合位点&nbsp;&nbsp;&nbsp;
	- 缺乏症 -> **巨幼红细胞贫血** -> 高同型半胱氨酸血症 -> 孕妇：胎儿脊柱裂和**神经管缺陷**&nbsp;&nbsp;&nbsp;
- **维生素 B₁₂（钴胺素）**&nbsp;&nbsp;&nbsp;
	- 活性形式 -> **甲基钴胺素**、**5'-脱氧腺苷钴胺素**&nbsp;&nbsp;&nbsp;
	- 辅酶功能 -> **甲硫氨酸合成酶**辅酶（同型半胱氨酸→甲硫氨酸）-> **L-甲基丙二酰CoA变位酶**辅酶&nbsp;&nbsp;&nbsp;
	- 特点 -> 唯一含**金属元素（钴）** 的维生素 -> 不存在于植物&nbsp;&nbsp;&nbsp;
	- 缺乏症 -> **巨幼红细胞贫血** -> 高同型半胱氨酸血症&nbsp;&nbsp;&nbsp;
- **维生素 C（抗坏血酸）**&nbsp;&nbsp;&nbsp;
	- 活性形式 -> **L-抗坏血酸**（本身就是活性形式）&nbsp;&nbsp;&nbsp;
	- 辅酶功能 -> 羟化酶辅酶（如**脯氨酸羟化酶**、**赖氨酸羟化酶**）-> **抗氧化剂** -> 增强**免疫力**&nbsp;&nbsp;&nbsp;
	- 对**热不稳定**&nbsp;&nbsp;&nbsp;
	- C₂ 和 C₃ 羟基可氧化脱氢 ⇌ 脱氢抗坏血酸&nbsp;&nbsp;&nbsp;
	- 缺乏症 -> **坏血病(scurvy)**&nbsp;&nbsp;&nbsp;
- **脂溶性维生素**&nbsp;&nbsp;&nbsp;
	- **VitA** -> 视黄醛（视觉）&nbsp;&nbsp;&nbsp;
	- **VitD** -> 钙磷代谢调节&nbsp;&nbsp;&nbsp;
	- **VitE** -> 抗氧化、生育酚&nbsp;&nbsp;&nbsp;
	- **VitK** -> 凝血因子合成（γ-羧化酶辅酶）&nbsp;&nbsp;&nbsp;
