# 酶促反应的特点与机制&nbsp;&nbsp;&nbsp;

- **酶与一般催化剂的共性**&nbsp;&nbsp;&nbsp;
	- 反应前后质和量**不变**&nbsp;&nbsp;&nbsp;
	- 只能催化热力学**允许**的反应&nbsp;&nbsp;&nbsp;
	- 只缩短时间，**不改变**平衡点和平衡常数&nbsp;&nbsp;&nbsp;
	- 加速机制 -> 降低反应的**活化能** -> 酶能**更有效**地降低&nbsp;&nbsp;&nbsp;
- **酶的特性（与一般催化剂的区别）**&nbsp;&nbsp;&nbsp;
	- **高效性** -> 极高的催化效率&nbsp;&nbsp;&nbsp;
		- 比非催化反应高 **10⁸~10²⁰** 倍&nbsp;&nbsp;&nbsp;
		- 比一般催化剂高 **10⁷~10¹³** 倍&nbsp;&nbsp;&nbsp;
		- 催化的反应条件**温和**（常温常压）&nbsp;&nbsp;&nbsp;
		- 高效性原因 -> 更有效降低**活化能** -> 促进形成**过渡态**&nbsp;&nbsp;&nbsp;
	- **高度专一性（特异性）**&nbsp;&nbsp;&nbsp;
		- 按特异性高低分&nbsp;&nbsp;&nbsp;
			- **绝对特异性** -> 一种酶只作用于**一种**底物 -> 如：**脲酶**只催化尿素&nbsp;&nbsp;&nbsp;
			- **相对特异性** -> 作用于**一类**化合物或**一种**化学键 -> 如：蛋白水解酶&nbsp;&nbsp;&nbsp;
		- 按立体异构分&nbsp;&nbsp;&nbsp;
			- **立体异构体专一性** -> 如：延胡索酸酶仅催化**反丁烯二酸**，不能作用顺丁烯二酸&nbsp;&nbsp;&nbsp;
			- **光学异构体专一性** -> 如：乳酸脱氢酶只能氧化**L-乳酸**，不能作用 D-乳酸&nbsp;&nbsp;&nbsp;
	- **可调节性** -> 受多种因素调控 -> 酶含量调节 + 酶活性调节&nbsp;&nbsp;&nbsp;
	- **不稳定性** -> 对**环境因素敏感** -> 强酸/强碱/高温 → 变性失活&nbsp;&nbsp;&nbsp;
- **酶促反应机制**&nbsp;&nbsp;&nbsp;
	- **酶-底物复合物(E-S复合物)形成**&nbsp;&nbsp;&nbsp;
		- E + S ⇌ ES → E + P&nbsp;&nbsp;&nbsp;
		- 有利于底物形成**过渡态** -> 转变为产物&nbsp;&nbsp;&nbsp;
		- 酶释放后可**循环利用**&nbsp;&nbsp;&nbsp;
	- **诱导契合学说(induced-fit theory)**&nbsp;&nbsp;&nbsp;
		- 酶与底物相互**诱导、变形、适应**&nbsp;&nbsp;&nbsp;
		- 酶构象改变 -> 利于结合底物&nbsp;&nbsp;&nbsp;
		- 底物变形 -> 处于**不稳定过渡态** -> 易受催化攻击&nbsp;&nbsp;&nbsp;
		- 与锁-钥模式不同：**动态**而非机械&nbsp;&nbsp;&nbsp;
		- 实例：**己糖激酶** -> 结合葡萄糖后构象发生显著改变，两叶闭合&nbsp;&nbsp;&nbsp;
	- **催化机制的多元性（综合作用）**&nbsp;&nbsp;&nbsp;
		- **邻近与定向效应** -> 增加有效碰撞&nbsp;&nbsp;&nbsp;
		- **酸碱催化** -> His、Asp 传递质子&nbsp;&nbsp;&nbsp;
		- **疏水/表面效应** -> 排除水分子，利于接触&nbsp;&nbsp;&nbsp;
		- **张力/变形效应** -> 诱导底物变形，促进断键&nbsp;&nbsp;&nbsp;
		- **共价催化** -> 形成过渡性共价中间体&nbsp;&nbsp;&nbsp;
