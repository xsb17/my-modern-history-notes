# 酶促反应动力学基础 (米氏方程)&nbsp;&nbsp;&nbsp;

- **酶促反应动力学前提**&nbsp;&nbsp;&nbsp;
	- 研究某一因素时 -> **固定其它因素不变**&nbsp;&nbsp;&nbsp;
	- 单底物、单产物反应&nbsp;&nbsp;&nbsp;
	- 反应速度取**初速度** -> 底物消耗量在 **5%** 以内&nbsp;&nbsp;&nbsp;
- **反应级数**&nbsp;&nbsp;&nbsp;
	- **一级反应** -> v = kc -> V 与 [S] 成正比&nbsp;&nbsp;&nbsp;
	- **二级反应** -> v = kc₁c₂&nbsp;&nbsp;&nbsp;
	- **零级反应** -> v = k -> V 与底物浓度无关&nbsp;&nbsp;&nbsp;
- **底物浓度对反应速度的影响**&nbsp;&nbsp;&nbsp;
	- [S] 较低 -> V与[S]成正比 -> **一级反应**&nbsp;&nbsp;&nbsp;
	- [S] 增高 -> V不成正比加速 -> **混合级反应**&nbsp;&nbsp;&nbsp;
	- [S] 极高 -> V达最大值(Vmax) -> **零级反应** -> **底物饱和现象**&nbsp;&nbsp;&nbsp;
	- 曲线类型：**矩形双曲线**&nbsp;&nbsp;&nbsp;
- **米-曼氏方程 (Michaelis-Menten equation)**&nbsp;&nbsp;&nbsp;
	- 公式：**V = (Vmax × [S]) / (Km + [S])**&nbsp;&nbsp;&nbsp;
	- 提出者：**Michaelis** 和 **Menten**（1913年）&nbsp;&nbsp;&nbsp;
	- 前提：中间复合物学说 E + S ⇌ ES → E + P&nbsp;&nbsp;&nbsp;
- **米氏方程三种特殊情况**&nbsp;&nbsp;&nbsp;
	- **[S] << Km** -> V ≈ (Vmax/Km) × [S] -> **一级反应** -> V与[S]成正比&nbsp;&nbsp;&nbsp;
	- **[S] >> Km** -> V ≈ Vmax -> **零级反应** -> 酶被底物饱和&nbsp;&nbsp;&nbsp;
	- **[S] = Km** -> V = **Vmax/2**&nbsp;&nbsp;&nbsp;
- **Km 值（米氏常数）**&nbsp;&nbsp;&nbsp;
	- 定义 -> 反应速度达到 **Vmax 一半**时的底物浓度&nbsp;&nbsp;&nbsp;
	- 单位：**浓度单位**（mol/L）&nbsp;&nbsp;&nbsp;
	- 意义 -> Km 反映酶与底物的**亲和力**&nbsp;&nbsp;&nbsp;
		- **Km 越小 -> 亲和力越大**&nbsp;&nbsp;&nbsp;
		- **Km 越大 -> 亲和力越小**&nbsp;&nbsp;&nbsp;
	- 推导：Kₘ = (k₋₁ + k₂) / k₁&nbsp;&nbsp;&nbsp;
	- 当 k₋₁ >> k₂ 时 -> Km ≈ Ks（ES解离常数）&nbsp;&nbsp;&nbsp;
	- Km 是酶的**特征性常数** -> 取决于酶与底物结构&nbsp;&nbsp;&nbsp;
	- 与酶和底物的**浓度无关**&nbsp;&nbsp;&nbsp;
	- 受环境影响 -> pH、温度、离子强度&nbsp;&nbsp;&nbsp;
	- **天然底物** -> 同一酶对多种底物中 **Km 最小**者&nbsp;&nbsp;&nbsp;
- **Vmax 值（最大反应速度）**&nbsp;&nbsp;&nbsp;
	- 定义 -> 酶**完全被底物饱和**时的反应速度&nbsp;&nbsp;&nbsp;
	- 与**酶浓度成正比**，与底物浓度无关&nbsp;&nbsp;&nbsp;
	- Vmax = **k₂[E]**&nbsp;&nbsp;&nbsp;
	- **k₂ (转换数/kcat)** -> 每秒钟每个酶分子转换底物的**分子数**&nbsp;&nbsp;&nbsp;
- **Km 和 Vmax 的测定 — 双倒数作图法（林-贝氏作图法）**&nbsp;&nbsp;&nbsp;
	- 公式：**1/V = (Km/Vmax) × 1/[S] + 1/Vmax**&nbsp;&nbsp;&nbsp;
	- 以 1/V 对 1/[S] 作图得**直线**&nbsp;&nbsp;&nbsp;
	- **纵轴截距** = **1/Vmax**&nbsp;&nbsp;&nbsp;
	- **横轴截距** = **-1/Km**&nbsp;&nbsp;&nbsp;
	- **斜率** = **Km/Vmax**&nbsp;&nbsp;&nbsp;
- **酶浓度的影响**&nbsp;&nbsp;&nbsp;
	- 当 **[S] >> [E]** 时 -> V 与 [E] **成正比**&nbsp;&nbsp;&nbsp;
- **温度的影响**&nbsp;&nbsp;&nbsp;
	- **双重效应** -> 升高温度 → 加快反应 + 增加变性&nbsp;&nbsp;&nbsp;
	- **最适温度** -> 反应速度最快时的温度&nbsp;&nbsp;&nbsp;
	- 最适温度**不是**特征性常数&nbsp;&nbsp;&nbsp;
- **pH 的影响**&nbsp;&nbsp;&nbsp;
	- pH 改变酶和底物的**解离状态**&nbsp;&nbsp;&nbsp;
	- **最适 pH** -> 酶活性最大时的 pH&nbsp;&nbsp;&nbsp;
	- 最适 pH **不是**特征性常数&nbsp;&nbsp;&nbsp;
