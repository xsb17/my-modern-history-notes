# 酶的分子组成与活性中心&nbsp;&nbsp;&nbsp;

- **酶的分类（按分子组成）**&nbsp;&nbsp;&nbsp;
	- **单纯酶** -> 仅由氨基酸残基组成 -> 只有蛋白质部分&nbsp;&nbsp;&nbsp;
	- **结合酶（缀合酶）** -> **全酶** = 酶蛋白 + 辅助因子&nbsp;&nbsp;&nbsp;
		- **酶蛋白(apoenzyme)** -> 决定**反应特异性** -> 催化**什么底物**&nbsp;&nbsp;&nbsp;
		- **辅助因子(cofactor)** -> 决定**反应类型** -> 即**酶催化类型**&nbsp;&nbsp;&nbsp;
- **辅助因子按化学本质分类**&nbsp;&nbsp;&nbsp;
	- **无机金属离子** -> 最常见的辅助因子 -> 占 **2/3**&nbsp;&nbsp;&nbsp;
		- **金属酶** -> 金属离子与酶结合**紧密** -> 如：**超氧化物歧化酶**&nbsp;&nbsp;&nbsp;
		- **金属激活酶** -> 金属离子与酶结合**不紧密** -> 如：**己糖激酶**&nbsp;&nbsp;&nbsp;
		- 金属离子作用 -> 传递电子 / 桥梁作用稳定构象 / 中和阴离子降低斥力&nbsp;&nbsp;&nbsp;
	- **有机小分子化合物** -> 多为**维生素**或其衍生物 -> 如醌类、卟啉衍生物&nbsp;&nbsp;&nbsp;
- **辅助因子按结合紧密程度分类**&nbsp;&nbsp;&nbsp;
	- **辅酶(coenzyme)**&nbsp;&nbsp;&nbsp;
		- 与酶蛋白结合**疏松**&nbsp;&nbsp;&nbsp;
		- 可用**透析或超滤**方法除去&nbsp;&nbsp;&nbsp;
		- 作为辅助底物接受质子或基团后离开&nbsp;&nbsp;&nbsp;
		- 实例：**NAD⁺**、**NADP⁺**&nbsp;&nbsp;&nbsp;
	- **辅基(prosthetic group)**&nbsp;&nbsp;&nbsp;
		- 与酶蛋白结合**紧密**&nbsp;&nbsp;&nbsp;
		- **不能**用透析或超滤除去&nbsp;&nbsp;&nbsp;
		- 反应中与酶蛋白不发生解离&nbsp;&nbsp;&nbsp;
		- 实例：**FAD**、**FMN**&nbsp;&nbsp;&nbsp;
- **酶按分子结构分类**&nbsp;&nbsp;&nbsp;
	- **单体酶** -> 仅有三级结构&nbsp;&nbsp;&nbsp;
	- **寡聚酶** -> 多个亚基以**非共价键**连接&nbsp;&nbsp;&nbsp;
	- **多酶体系** -> 参与某一代谢途径的一组相关酶&nbsp;&nbsp;&nbsp;
	- **多酶复合物** -> 催化相关反应的酶彼此聚集结合&nbsp;&nbsp;&nbsp;
	- **多功能酶/串联酶** -> 多种催化功能存在于**一条多肽链**&nbsp;&nbsp;&nbsp;
- **酶的活性中心**&nbsp;&nbsp;&nbsp;
	- **必需基团** -> 酶发挥催化活性**必需的**化学基团&nbsp;&nbsp;&nbsp;
	- **活性中心** -> 必需基团在空间上彼此靠近 -> 形成**口袋或裂穴** -> 与底物特异结合并转化为产物&nbsp;&nbsp;&nbsp;
	- 活性中心特点&nbsp;&nbsp;&nbsp;
		- 只占酶总体积的**一小部分**&nbsp;&nbsp;&nbsp;
		- 是**三维**空间结构&nbsp;&nbsp;&nbsp;
		- 辅助因子参与活性中心的形成&nbsp;&nbsp;&nbsp;
	- 活性中心内的必需基团&nbsp;&nbsp;&nbsp;
		- **结合基团** -> 与底物相结合&nbsp;&nbsp;&nbsp;
		- **催化基团** -> 催化底物转变成产物&nbsp;&nbsp;&nbsp;
	- 活性中心以外的必需基团&nbsp;&nbsp;&nbsp;
		- **结构必需基团** -> 维持活性中心精确构象&nbsp;&nbsp;&nbsp;
	- 实例：**溶菌酶**&nbsp;&nbsp;&nbsp;
		- 催化基团 -> **35位Glu**、**52位Asp**&nbsp;&nbsp;&nbsp;
		- 结合基团 -> **101位Asp**、**108位Trp**&nbsp;&nbsp;&nbsp;
