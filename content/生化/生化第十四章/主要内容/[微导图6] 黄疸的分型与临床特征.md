# 黄疸的分型与临床特征&nbsp;&nbsp;&nbsp;

- **黄疸概述**&nbsp;&nbsp;&nbsp;
	- **定义** -> 胆红素代谢异常 → 血中胆红素↑ → **皮肤、黏膜及巩膜黄染**&nbsp;&nbsp;&nbsp;
	- **正常血清胆红素** -> 总胆红素 = **==游离胆红素(4/5) + 结合胆红素(1/5)==**&nbsp;&nbsp;&nbsp;
	- **隐性黄疸** -> TB：17.1~34.2 μmol/L&nbsp;&nbsp;&nbsp;
	- **显性黄疸** -> TB > **==34.2 μmol/L==**&nbsp;&nbsp;&nbsp;
	- **发生机制** -> ①生成过多 ②摄取↓ ③转化↓ ④排泄↓ → 高胆红素血症 → 黄疸&nbsp;&nbsp;&nbsp;
- **胆红素在肠道的正常代谢**&nbsp;&nbsp;&nbsp;
	- 结合胆红素 → 肠道 →(细菌作用)→ 脱去葡萄糖醛酸 → 还原为 **==胆素原==**&nbsp;&nbsp;&nbsp;
	- 大部分胆素原 → 粪便排出 → 氧化为**胆素** → 粪便的**==主要色素==**&nbsp;&nbsp;&nbsp;
	- 小部分胆素原 → 重吸收入血 → 部分经肾排出(**0.5-4mg/天**)&nbsp;&nbsp;&nbsp;
	- **==尿三胆==** -> 尿胆红素 + 尿胆素原 + 尿胆素 → 肝功能检查指标&nbsp;&nbsp;&nbsp;
- **==溶血性黄疸（肝前性）==**&nbsp;&nbsp;&nbsp;
	- **病因** -> 红细胞大量破坏 → 单核吞噬系统产生胆红素**过多** → 超过肝处理能力&nbsp;&nbsp;&nbsp;
	- **血清特征**&nbsp;&nbsp;&nbsp;
		- **==未结合胆红素==** -> **==↑↑==**（显著增高）&nbsp;&nbsp;&nbsp;
		- 结合胆红素 -> 正常或轻度↑&nbsp;&nbsp;&nbsp;
	- **尿液特征**&nbsp;&nbsp;&nbsp;
		- 尿胆红素 -> **==阴性（-）==**（未结合胆红素不能经肾排出）&nbsp;&nbsp;&nbsp;
		- 尿胆素原 -> **==↑↑==**（肝处理结合胆红素增多 → 肠道胆素原↑→ 重吸收↑）&nbsp;&nbsp;&nbsp;
	- **粪便特征** -> **==颜色加深==**（肠道胆素原↑→胆素↑）&nbsp;&nbsp;&nbsp;
	- **一句话** -> "红细胞破太多，肝来不及处理，全是未结合型"&nbsp;&nbsp;&nbsp;
- **==肝细胞性黄疸（肝原性）==**&nbsp;&nbsp;&nbsp;
	- **病因** -> 肝细胞功能**受损** → 摄取/转化/排泄胆红素能力**均降低**&nbsp;&nbsp;&nbsp;
	- **血清特征**&nbsp;&nbsp;&nbsp;
		- **未结合胆红素** -> **==↑==**（摄取+转化↓）&nbsp;&nbsp;&nbsp;
		- **结合胆红素** -> **==↑==**（排泄↓，结合胆红素逆流入血）&nbsp;&nbsp;&nbsp;
	- **尿液特征**&nbsp;&nbsp;&nbsp;
		- 尿胆红素 -> **==阳性（+）==**（结合胆红素可经肾排出）&nbsp;&nbsp;&nbsp;
		- 尿胆素原 -> **==↑==**（肝处理能力↓→肠道重吸收→肾排出↑）&nbsp;&nbsp;&nbsp;
	- **粪便特征** -> **==颜色变浅==**（排入肠道胆红素减少）&nbsp;&nbsp;&nbsp;
	- **一句话** -> "肝细胞坏，啥都降，两种胆红素都升高"&nbsp;&nbsp;&nbsp;
- **==阻塞性黄疸（肝后性）==**&nbsp;&nbsp;&nbsp;
	- **病因** -> 胆管系统**阻塞** → 胆汁排泄**障碍** → **==结合胆红素==**逆流入血&nbsp;&nbsp;&nbsp;
	- **血清特征**&nbsp;&nbsp;&nbsp;
		- 未结合胆红素 -> 正常或轻度↑&nbsp;&nbsp;&nbsp;
		- **==结合胆红素==** -> **==↑↑==**（显著增高）&nbsp;&nbsp;&nbsp;
	- **尿液特征**&nbsp;&nbsp;&nbsp;
		- 尿胆红素 -> **==强阳性（++）==**（大量结合胆红素经肾排出 → **深黄色尿**）&nbsp;&nbsp;&nbsp;
		- 尿胆素原 -> **==↓ 或消失（-）==**（胆红素无法入肠→无胆素原生成）&nbsp;&nbsp;&nbsp;
	- **粪便特征** -> **==白陶土色==**（无胆素→粪便失去正常色素）&nbsp;&nbsp;&nbsp;
	- **一句话** -> "胆管堵死，结合型逆流，尿黑便白"&nbsp;&nbsp;&nbsp;
- **三种黄疸血清-尿-粪速记对比**&nbsp;&nbsp;&nbsp;
	- **==溶血性==** -> 血清未结合↑↑ / 尿胆红素(-) / 尿胆素原↑↑ / 粪便**深**&nbsp;&nbsp;&nbsp;
	- **==肝细胞性==** -> 血清两者均↑ / 尿胆红素(+) / 尿胆素原↑ / 粪便**浅**&nbsp;&nbsp;&nbsp;
	- **==阻塞性==** -> 血清结合↑↑ / 尿胆红素(++) / 尿胆素原(-) / 粪便**==白陶土色==**&nbsp;&nbsp;&nbsp;
