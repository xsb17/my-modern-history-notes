# [全集存档] 生化第14章：肝的生物化学&nbsp;&nbsp;&nbsp;

## 第一节 肝在物质代谢中的作用&nbsp;&nbsp;&nbsp;

### 肝的结构与功能概述&nbsp;&nbsp;&nbsp;

- 两条输入通道：肝门静脉、肝动脉&nbsp;&nbsp;&nbsp;
- 两条输出通道：肝静脉、胆道系统&nbsp;&nbsp;&nbsp;
- 丰富的亚细胞结构：线粒体、内质网、微粒体、溶酶体&nbsp;&nbsp;&nbsp;
- 肝是**==物质代谢中枢==** + 生物转化作用 + 分泌作用 + 排泄作用&nbsp;&nbsp;&nbsp;

### 一、肝在糖代谢中的作用&nbsp;&nbsp;&nbsp;

- 肝是维持==血糖水平相对稳定==的重要器官&nbsp;&nbsp;&nbsp;
- 肝可进行的糖代谢途径：&nbsp;&nbsp;&nbsp;
  - ==糖异生==&nbsp;&nbsp;&nbsp;
  - 肝糖原的合成与分解&nbsp;&nbsp;&nbsp;
  - 糖酵解&nbsp;&nbsp;&nbsp;
  - 糖的有氧氧化&nbsp;&nbsp;&nbsp;
  - 磷酸戊糖途径&nbsp;&nbsp;&nbsp;

### 二、肝在脂代谢中的作用&nbsp;&nbsp;&nbsp;

- 肝在脂类的消化、吸收、合成、分解与运输等方面均具有重要作用&nbsp;&nbsp;&nbsp;
- 肝可进行的脂代谢途径：&nbsp;&nbsp;&nbsp;
  - 脂肪酸的氧化&nbsp;&nbsp;&nbsp;
  - 脂肪酸的合成及脂肪的合成&nbsp;&nbsp;&nbsp;
  - ==酮体的生成==（肝是酮体生成的**==唯一场所==**）&nbsp;&nbsp;&nbsp;
  - 胆固醇的合成与转变（→胆汁酸）&nbsp;&nbsp;&nbsp;
  - 脂蛋白与载脂蛋白的合成（==VLDL、HDL==等）&nbsp;&nbsp;&nbsp;

### 三、肝在蛋白质代谢中的作用&nbsp;&nbsp;&nbsp;

- 合成、分泌、清除==血浆蛋白质==的重要器官&nbsp;&nbsp;&nbsp;
- 氨基酸分解和转化的重要场所&nbsp;&nbsp;&nbsp;
- ==氨的解毒==（合成尿素）和胺类物质的生物转化&nbsp;&nbsp;&nbsp;
- ==尿素合成的主要场所==&nbsp;&nbsp;&nbsp;

### 四、肝在维生素代谢中的作用&nbsp;&nbsp;&nbsp;

- 吸收：胆汁酸帮助==脂溶性维生素==吸收&nbsp;&nbsp;&nbsp;
- 储存：Vit A、Vit K、Vit E 和 Vit B₁₂&nbsp;&nbsp;&nbsp;
- 运输：合成视黄醇结合蛋白和 Vit D 结合蛋白&nbsp;&nbsp;&nbsp;
- 转化：B 族维生素 → 辅酶或辅基；维生素 A 原 → 维生素 A；维生素 D₃ → 25-羟维生素 D₃&nbsp;&nbsp;&nbsp;

### 五、肝在激素代谢中的作用&nbsp;&nbsp;&nbsp;

- ==激素灭活==：激素发挥作用后降解或失去活性的过程&nbsp;&nbsp;&nbsp;
- 肝参与激素的灭活和排泄&nbsp;&nbsp;&nbsp;

## 第二节 肝的生物转化作用&nbsp;&nbsp;&nbsp;

### 一、生物转化的概念&nbsp;&nbsp;&nbsp;

- **==生物转化(biotransformation)==**：机体将极性/水溶性较低、不容易排出体外的==非营养物质==进行化学转变，增加其极性/水溶性，使其容易排出体外的过程&nbsp;&nbsp;&nbsp;
- **非营养物质**：既不作为构建组织细胞的原料，又不作为能源物质供能；某些还对人体有一定的生物学效应或潜在毒性&nbsp;&nbsp;&nbsp;
- **生物转化的意义**：&nbsp;&nbsp;&nbsp;
  - 增加非营养物质的==水溶性和极性==，使其易于随粪便或尿液排出&nbsp;&nbsp;&nbsp;
  - 使大部分非营养物质的生物学活性降低或丧失（灭活），或使有毒物质的毒性减低（解毒）&nbsp;&nbsp;&nbsp;
  - 部分非营养物质经生物转化后毒性或生物学活性会**增加**，即"==解毒致毒双重性=="&nbsp;&nbsp;&nbsp;
- **生物转化部位**：&nbsp;&nbsp;&nbsp;
  - 肝：==80~90%==，是最重要的生物转化器官&nbsp;&nbsp;&nbsp;
  - 亚细胞定位：肝细胞的==微粒体、胞液、线粒体==，以微粒体为主&nbsp;&nbsp;&nbsp;
  - 其他组织：肺、肾、皮肤、胃肠道等&nbsp;&nbsp;&nbsp;

### 二、生物转化的主要类型&nbsp;&nbsp;&nbsp;

#### 第一相反应&nbsp;&nbsp;&nbsp;

- 包括**氧化、还原、水解**反应&nbsp;&nbsp;&nbsp;
- 使分子中非极性基团转变为极性基团，增强水溶性&nbsp;&nbsp;&nbsp;
- **氧化反应**：&nbsp;&nbsp;&nbsp;
  - ==加单氧酶系==（细胞色素 P450）：异源物 → 羟化产物&nbsp;&nbsp;&nbsp;
  - 单胺氧化酶系：胺类物质&nbsp;&nbsp;&nbsp;
  - 脱氢酶系：醇类或醛类（如乙醇代谢：醇脱氢酶 ADH → 醛脱氢酶 ALDH）&nbsp;&nbsp;&nbsp;
- **还原反应**：硝基还原酶（硝基化合物）、偶氮还原酶（偶氮化合物）&nbsp;&nbsp;&nbsp;
- **水解反应**：酯酶（脂类）、酰胺酶（酰胺类）、糖苷酶（糖苷类）&nbsp;&nbsp;&nbsp;

#### 第二相反应（结合反应）&nbsp;&nbsp;&nbsp;

- 最普遍、最重要的生物转化方式&nbsp;&nbsp;&nbsp;
- 结合底物：含有羟基、巯基、羧基或氨基的物质&nbsp;&nbsp;&nbsp;
- 主要结合类型：&nbsp;&nbsp;&nbsp;
  - **==葡萄糖醛酸结合==**（最普遍）：供体 ==UDPGA==，酶 ==葡糖醛酸基转移酶(UGT)==，底物为醇/酚/胺/羧酸类&nbsp;&nbsp;&nbsp;
  - **硫酸结合**：供体 ==PAPS==，酶 = 硫酸转移酶，底物为类固醇/醇/酚/芳香胺 → 激素灭活&nbsp;&nbsp;&nbsp;
  - **谷胱甘肽结合**：供体 = 谷胱甘肽，酶 = 谷胱甘肽 S-转移酶(GST)，底物为卤代物/环氧化物（致癌物解毒）&nbsp;&nbsp;&nbsp;
  - **乙酰基结合**：供体 = 乙酰CoA，底物为芳香胺&nbsp;&nbsp;&nbsp;
  - **甲基结合**：供体 ==SAM==，底物为含羟基/巯基/氨基的物质&nbsp;&nbsp;&nbsp;
  - **氨基酸结合**：供体 = 甘氨酸/牛磺酸，底物为羧酸类&nbsp;&nbsp;&nbsp;

### 三、生物转化反应的特点&nbsp;&nbsp;&nbsp;

- **反应的连续性**：如黄曲霉素先氧化再结合&nbsp;&nbsp;&nbsp;
- **类型的多样性**：一种物质可经多种反应进行转化&nbsp;&nbsp;&nbsp;
- **==解毒与致毒的双重性==**：黄曲霉素 B₁ 经加单氧酶氧化后生成的环氧黄曲霉素具有致癌性，可与 DNA 鸟嘌呤结合引起突变（肝癌危险因素）&nbsp;&nbsp;&nbsp;

### 四、影响生物转化的因素&nbsp;&nbsp;&nbsp;

- **个体差异**：遗传因素、生理因素（年龄、性别）&nbsp;&nbsp;&nbsp;
- **生理状态**：病理因素（肝脏疾病）、营养状况&nbsp;&nbsp;&nbsp;
- **外界因素**：药物等可诱导或抑制特定生物转化酶（如苯巴比妥可诱导加单氧酶和 UGT）&nbsp;&nbsp;&nbsp;

## 第三节 胆汁与胆汁酸代谢&nbsp;&nbsp;&nbsp;

### 一、胆汁及其成分&nbsp;&nbsp;&nbsp;

- **胆汁(bile)**：由肝细胞分泌 → 肝胆管 → 胆囊（储存浓缩）→ 胆管系统 → 肠道&nbsp;&nbsp;&nbsp;
- 肝胆汁：金黄色、透明澄清、偏碱性、水分多&nbsp;&nbsp;&nbsp;
- 胆囊胆汁：黄褐色、水分少、比重大&nbsp;&nbsp;&nbsp;
- **胆汁的生理功能**：&nbsp;&nbsp;&nbsp;
  - 作为消化液：胆汁酸盐 → 促进==脂类消化和吸收==&nbsp;&nbsp;&nbsp;
  - 作为排泄液：将胆红素等代谢产物排入肠腔 → 随粪便排出&nbsp;&nbsp;&nbsp;

### 二、胆汁酸的种类&nbsp;&nbsp;&nbsp;

- **胆汁酸(bile acids)**：肝细胞以==胆固醇==为原料合成的 24 碳类固醇化合物，是胆固醇在体内**主要的代谢产物**&nbsp;&nbsp;&nbsp;
- **初级胆汁酸**：在肝细胞以胆固醇为原料直接合成&nbsp;&nbsp;&nbsp;
  - 游离型：==胆酸==、==鹅脱氧胆酸==&nbsp;&nbsp;&nbsp;
  - 结合型：甘氨胆酸、牛磺胆酸、甘氨鹅脱氧胆酸、牛磺鹅脱氧胆酸&nbsp;&nbsp;&nbsp;
- **次级胆汁酸**：在肠道受细菌作用，初级胆汁酸第 7 位 α-羟基脱氧生成&nbsp;&nbsp;&nbsp;
  - 游离型：==脱氧胆酸==、==石胆酸==&nbsp;&nbsp;&nbsp;
  - 结合型：甘氨脱氧胆酸、牛磺脱氧胆酸、甘氨石胆酸、牛磺石胆酸&nbsp;&nbsp;&nbsp;

### 三、胆汁酸的代谢&nbsp;&nbsp;&nbsp;

#### 1. 初级胆汁酸的生成&nbsp;&nbsp;&nbsp;

- 部位：==肝细胞的胞液和微粒体==&nbsp;&nbsp;&nbsp;
- 原料：==胆固醇==&nbsp;&nbsp;&nbsp;
- 产物：胆酸、鹅脱氧胆酸及其结合型胆汁酸&nbsp;&nbsp;&nbsp;

#### 2. 次级胆汁酸的生成&nbsp;&nbsp;&nbsp;

- 部位：==小肠下段和大肠==&nbsp;&nbsp;&nbsp;
- 原料：初级胆汁酸&nbsp;&nbsp;&nbsp;
- 产物：脱氧胆酸、石胆酸（经肠道细菌 7α-脱羟基作用）&nbsp;&nbsp;&nbsp;

#### 3. 胆汁酸的肠肝循环&nbsp;&nbsp;&nbsp;

- 胆汁酸排入肠腔后，约==95%==被肠壁重吸收 → 经==门静脉==回肝 → 肝内转变为结合胆汁酸 → 与新合成的胆汁酸一同再次排入肠腔&nbsp;&nbsp;&nbsp;
- **生理意义**：使有限的胆汁酸（代谢池==3-5g==）循环利用，满足机体每日对胆汁酸的需求（==16-32g==/天）&nbsp;&nbsp;&nbsp;

### 四、胆汁酸的生理作用&nbsp;&nbsp;&nbsp;

- **促进脂类的消化和吸收**（==乳化剂==）&nbsp;&nbsp;&nbsp;
- **抑制胆汁中胆固醇的析出**（防止==胆结石==）&nbsp;&nbsp;&nbsp;
- 临床：鹅脱氧胆酸/熊脱氧胆酸治疗胆固醇结石&nbsp;&nbsp;&nbsp;

## 第四节 胆色素代谢&nbsp;&nbsp;&nbsp;

### 胆色素概述&nbsp;&nbsp;&nbsp;

- 胆色素是体内铁卟啉化合物的分解代谢产物，包括胆绿素、胆红素、胆素原和胆素等&nbsp;&nbsp;&nbsp;
- 胆红素处于胆色素代谢的中心，是人体胆汁中的主要色素&nbsp;&nbsp;&nbsp;
- 胆红素极性基团集中在分子内部 → ==亲脂疏水== → 易自由透过细胞膜&nbsp;&nbsp;&nbsp;

### 一、胆红素的生成与转运&nbsp;&nbsp;&nbsp;

#### 1. 胆红素的生成&nbsp;&nbsp;&nbsp;

- 部位：==肝、脾、骨髓==等单核吞噬细胞微粒体与胞液中&nbsp;&nbsp;&nbsp;
- 来源：血红素的降解产物，主要源于==衰老红细胞==破坏所释放的血红蛋白（约占==80%==）&nbsp;&nbsp;&nbsp;
- 过程：血红素 →(血红素加氧酶)→ 胆绿素 →(胆绿素还原酶)→ 胆红素&nbsp;&nbsp;&nbsp;

#### 2. 胆红素的转运&nbsp;&nbsp;&nbsp;

- 运输形式：==胆红素-清蛋白复合体==（游离胆红素 = 间接胆红素 = 未结合胆红素）&nbsp;&nbsp;&nbsp;
- 意义：增加水溶性、提高运输能力；限制自由通透细胞膜 → ==解毒作用==&nbsp;&nbsp;&nbsp;
- **竞争结合剂**：磺胺药、水杨酸、胆汁酸等可竞争结合清蛋白 → 游离胆红素↑ → 可致==胆红素脑病（核黄疸）==&nbsp;&nbsp;&nbsp;
- **新生儿黄疸**：新生儿血脑屏障不健全，游离胆红素可通过血脑屏障与脑基底核脂类结合 → ==核黄疸==&nbsp;&nbsp;&nbsp;

### 二、胆红素在肝中的代谢&nbsp;&nbsp;&nbsp;

#### 1. 肝细胞对胆红素的摄取&nbsp;&nbsp;&nbsp;

- 胆红素自由通透肝细胞膜进入肝细胞&nbsp;&nbsp;&nbsp;
- 在胞浆与配体蛋白（==Y蛋白或Z蛋白==）结合 → 携带至滑面内质网&nbsp;&nbsp;&nbsp;

#### 2. 肝细胞对胆红素的转化作用&nbsp;&nbsp;&nbsp;

- 部位：==滑面内质网==&nbsp;&nbsp;&nbsp;
- 反应：结合反应（主要结合物为==UDPGA==）&nbsp;&nbsp;&nbsp;
- 酶：==葡萄糖醛酸基转移酶(UGT)==&nbsp;&nbsp;&nbsp;
- 产物：==结合胆红素==（直接胆红素/肝胆红素）——主要为双葡糖醛酸胆红素&nbsp;&nbsp;&nbsp;
- **意义**：将脂溶性有毒游离胆红素 → 水溶性无毒结合胆红素 → 易随胆汁/尿液排出（==胆红素在体内解毒的重要方式==）&nbsp;&nbsp;&nbsp;
- 临床应用：苯巴比妥可==诱导 UGT== → 治疗新生儿黄疸&nbsp;&nbsp;&nbsp;

#### 3. 肝细胞对胆红素的排泄作用&nbsp;&nbsp;&nbsp;

- 肝经毛细胆管排泄结合胆红素至肠道是==肝脏代谢胆红素的限速步骤==&nbsp;&nbsp;&nbsp;
- 糖皮质激素可诱导 UGT → 促进结合胆红素排泄 → 治疗高胆红素血症&nbsp;&nbsp;&nbsp;

### 三、胆红素在肠道的变化&nbsp;&nbsp;&nbsp;

- 结合胆红素 → 肠道 →(细菌作用)→ 脱去葡萄糖醛酸 → 还原为==胆素原==（尿胆素原 + 粪胆素原）&nbsp;&nbsp;&nbsp;
- 大部分胆素原随粪便排出 → 氧化为==胆素==（粪便的主要色素）&nbsp;&nbsp;&nbsp;
- 小部分胆素原重吸收入血 → 部分→肾→尿液排出（0.5-4mg/天）&nbsp;&nbsp;&nbsp;
- **尿三胆**：尿液中胆红素、胆素原、胆素 → 肝功能检查指标&nbsp;&nbsp;&nbsp;

### 四、两种胆红素理化性质比较（重点）&nbsp;&nbsp;&nbsp;

#### 未结合胆红素（间接胆红素/游离胆红素）&nbsp;&nbsp;&nbsp;

- 与葡萄糖醛酸结合：==未结合==&nbsp;&nbsp;&nbsp;
- 水溶性：==小== / 脂溶性：==大==&nbsp;&nbsp;&nbsp;
- 透过细胞膜能力及毒性：==大==&nbsp;&nbsp;&nbsp;
- 能否透过肾小球随尿排出：==不能==&nbsp;&nbsp;&nbsp;
- 对脑的毒性：==大==&nbsp;&nbsp;&nbsp;
- 与重氮试剂反应：间接阳性&nbsp;&nbsp;&nbsp;

#### 结合胆红素（直接胆红素/肝胆红素）&nbsp;&nbsp;&nbsp;

- 与葡萄糖醛酸结合：==结合==&nbsp;&nbsp;&nbsp;
- 水溶性：==大== / 脂溶性：==小==&nbsp;&nbsp;&nbsp;
- 透过细胞膜能力及毒性：==小==&nbsp;&nbsp;&nbsp;
- 能否透过肾小球随尿排出：==能==&nbsp;&nbsp;&nbsp;
- 对脑的毒性：==小==&nbsp;&nbsp;&nbsp;
- 与重氮试剂反应：直接阳性&nbsp;&nbsp;&nbsp;

### 五、血清胆红素与黄疸&nbsp;&nbsp;&nbsp;

- 正常人血清：总胆红素 = 游离胆红素(==4/5==) + 结合胆红素(==1/5==)&nbsp;&nbsp;&nbsp;
- **黄疸(jaundice)**：胆红素代谢异常 → 血中胆红素升高 → 皮肤、黏膜及巩膜黄染&nbsp;&nbsp;&nbsp;
- 隐性黄疸：TB 17.1-34.2 μmol/L；显性黄疸：TB > ==34.2 μmol/L==&nbsp;&nbsp;&nbsp;

#### 三种黄疸的鉴别&nbsp;&nbsp;&nbsp;

- **==溶血性黄疸（肝前性）==**&nbsp;&nbsp;&nbsp;
  - 机制：红细胞大量破坏 → 胆红素生成过多超过肝处理能力 → ==未结合胆红素==显著增高&nbsp;&nbsp;&nbsp;
  - 血清：未结合胆红素 ↑↑&nbsp;&nbsp;&nbsp;
  - 尿液：尿胆素原 ↑↑，尿胆红素（-）&nbsp;&nbsp;&nbsp;
  - 粪便：颜色加深&nbsp;&nbsp;&nbsp;
- **==肝细胞性黄疸（肝原性）==**&nbsp;&nbsp;&nbsp;
  - 机制：肝细胞功能受损 → 摄取、转化和排泄胆红素能力均降低&nbsp;&nbsp;&nbsp;
  - 血清：未结合 + 结合胆红素均 ↑&nbsp;&nbsp;&nbsp;
  - 尿液：尿胆素原 ↑，尿胆红素（+）&nbsp;&nbsp;&nbsp;
  - 粪便：颜色变浅&nbsp;&nbsp;&nbsp;
- **==阻塞性黄疸（肝后性）==**&nbsp;&nbsp;&nbsp;
  - 机制：胆管系统阻塞 → 胆汁排泄障碍 → ==结合胆红素==逆流入血&nbsp;&nbsp;&nbsp;
  - 血清：结合胆红素 ↑↑&nbsp;&nbsp;&nbsp;
  - 尿液：尿胆素原 ↓/（-），尿胆红素（++）&nbsp;&nbsp;&nbsp;
  - 粪便：==白陶土色==（无胆素）&nbsp;&nbsp;&nbsp;

## 重要概念（课后总结）&nbsp;&nbsp;&nbsp;

- **生物转化**：机体将非营养物质进行化学转变，增加其水溶性，使其容易排出体外的过程&nbsp;&nbsp;&nbsp;
- **非营养物质**：既不作为组织构建原料也不供能的物质，某些有生物学效应或毒性&nbsp;&nbsp;&nbsp;
- **胆汁酸肠肝循环**：约 95% 胆汁酸经肠道重吸收 → 门静脉→肝 → 再次排入肠腔的循环过程&nbsp;&nbsp;&nbsp;
- **胆汁酸生理作用**：促进脂类消化吸收（乳化剂）；抑制胆汁中胆固醇析出（防胆结石）&nbsp;&nbsp;&nbsp;
- **胆色素**：体内铁卟啉化合物的分解代谢产物（胆绿素、胆红素、胆素原和胆素）&nbsp;&nbsp;&nbsp;
- **结合胆红素（直接胆红素）**：与葡萄糖醛酸结合的胆红素，水溶性大，能经肾排出&nbsp;&nbsp;&nbsp;
