# 生物转化的第二相反应与影响因素&nbsp;&nbsp;&nbsp;

- **第二相反应（结合反应）**&nbsp;&nbsp;&nbsp;
	- **地位** -> 最普遍、最重要的生物转化方式&nbsp;&nbsp;&nbsp;
	- **底物特征** -> 含**羟基/巯基/羧基/氨基**的药物、毒物或激素&nbsp;&nbsp;&nbsp;
	- **目标** -> 与**水溶性更强的物质结合** -> 进一步增加水溶性 -> 利于排出&nbsp;&nbsp;&nbsp;
- **六大结合反应类型**&nbsp;&nbsp;&nbsp;
	- **==葡萄糖醛酸结合==**（最普遍！）&nbsp;&nbsp;&nbsp;
		- 活性供体 -> **==UDPGA==**（尿苷二磷酸葡萄糖醛酸）&nbsp;&nbsp;&nbsp;
		- 催化酶 -> **==葡糖醛酸基转移酶 (UGT)==**&nbsp;&nbsp;&nbsp;
		- 底物 -> 醇/酚/胺/羧酸类 -> 产物：葡萄糖醛酸苷&nbsp;&nbsp;&nbsp;
		- 实例 -> 胆红素/类固醇激素/氯霉素/吗啡/苯巴比妥 → 经此途径转化&nbsp;&nbsp;&nbsp;
		- 临床 -> 葡醛内酯治疗肝病 → 增强生物转化功能&nbsp;&nbsp;&nbsp;
	- **==硫酸结合==**&nbsp;&nbsp;&nbsp;
		- 活性供体 -> **==PAPS==**（3′-磷酸腺苷-5′-磷酸硫酸/活性硫酸根）&nbsp;&nbsp;&nbsp;
		- 催化酶 -> 硫酸转移酶&nbsp;&nbsp;&nbsp;
		- 底物 -> 类固醇/醇/酚/芳香胺 -> 产物：硫酸酯&nbsp;&nbsp;&nbsp;
		- **特殊功能** -> **==激素灭活==**（激素在肝中经此途径降解失活）&nbsp;&nbsp;&nbsp;
		- 肝功受损 -> 激素灭活↓ -> **==肝掌/蜘蛛痣/男性乳房发育==**&nbsp;&nbsp;&nbsp;
	- **==谷胱甘肽结合==**&nbsp;&nbsp;&nbsp;
		- 催化酶 -> **谷胱甘肽 S-转移酶 (GST)**&nbsp;&nbsp;&nbsp;
		- 底物 -> 卤代物/环氧化物（致癌物/抗癌药/环境污染物）&nbsp;&nbsp;&nbsp;
		- 实例 -> 黄曲霉素 B₁ 环氧化物 → GST → 谷胱甘肽结合物 → 胆汁排出&nbsp;&nbsp;&nbsp;
	- **==乙酰基结合==** -> 供体：**乙酰CoA** / 底物：芳香胺 / 产物：乙酰衍生物&nbsp;&nbsp;&nbsp;
	- **==甲基结合==** -> 供体：**==SAM==**（S-腺苷甲硫氨酸）/ 底物：含羟基/巯基/氨基&nbsp;&nbsp;&nbsp;
	- **==氨基酸结合==** -> 供体：**甘氨酸/牛磺酸** / 底物：羧酸类 / 产物：氨基酸结合物&nbsp;&nbsp;&nbsp;
- **第一相 vs 第二相对应关系**&nbsp;&nbsp;&nbsp;
	- **第一相** -> 氧化/还原/水解 -> 暴露或引入**极性基团**（-OH/-NH₂/-COOH）&nbsp;&nbsp;&nbsp;
	- **第二相** -> 结合反应 -> 极性基团 + 水溶性物质 → **高度水溶性** → **==易于排泄==**&nbsp;&nbsp;&nbsp;
	- 有些物质 -> 直接进入第二相（本身已有极性基团）&nbsp;&nbsp;&nbsp;
	- 有些物质 -> 必须经第一相 → 暴露极性基团 → 才能进入第二相&nbsp;&nbsp;&nbsp;
- **影响生物转化的因素**&nbsp;&nbsp;&nbsp;
	- **个体差异** -> 遗传因素 + 年龄/性别&nbsp;&nbsp;&nbsp;
	- **生理状态** -> 肝疾病 / 营养状况&nbsp;&nbsp;&nbsp;
	- **药物影响（重点）** -> 某些药物可**==诱导或抑制==**生物转化酶&nbsp;&nbsp;&nbsp;
		- **==苯巴比妥==** -> 诱导**加单氧酶**（P450）+ **UGT**（葡萄糖醛酸基转移酶）↑ -> 生物转化↑&nbsp;&nbsp;&nbsp;
		- 苯巴比妥临床应用 -> 治疗新生儿黄疸（诱导 UGT → 加快胆红素结合转化）&nbsp;&nbsp;&nbsp;
- **三大结合供体速记**&nbsp;&nbsp;&nbsp;
	- **==UDPGA==** -> 葡萄糖醛酸结合（最普遍、最重要）&nbsp;&nbsp;&nbsp;
	- **==PAPS==** -> 硫酸结合（激素灭活核心）&nbsp;&nbsp;&nbsp;
	- **==SAM==** -> 甲基结合（甲基供体）&nbsp;&nbsp;&nbsp;
