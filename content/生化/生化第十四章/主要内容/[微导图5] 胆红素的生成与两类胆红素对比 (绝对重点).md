# 胆红素的生成与两类胆红素对比 (绝对重点)&nbsp;&nbsp;&nbsp;

- **胆色素概述**&nbsp;&nbsp;&nbsp;
	- **定义** -> 体内**==铁卟啉化合物==**的分解代谢产物 -> 包括胆绿素/胆红素/胆素原/胆素&nbsp;&nbsp;&nbsp;
	- **胆红素** -> 处于胆色素代谢的**中心** -> 是人体胆汁中的主要色素&nbsp;&nbsp;&nbsp;
	- **结构特征** -> 极性基团集中在分子内部 -> **==亲脂疏水==** -> 易自由透过细胞膜&nbsp;&nbsp;&nbsp;
- **胆红素的生成**&nbsp;&nbsp;&nbsp;
	- **生成部位** -> **==肝、脾、骨髓==** 等单核吞噬细胞（微粒体与胞液）&nbsp;&nbsp;&nbsp;
	- **主要来源** -> **==衰老红细胞==** 破坏释放的血红蛋白（约占 ==80%==）&nbsp;&nbsp;&nbsp;
	- **反应过程**&nbsp;&nbsp;&nbsp;
		- 血红素 ->(==**血红素加氧酶**==)-> **胆绿素**&nbsp;&nbsp;&nbsp;
		- 胆绿素 ->(==**胆绿素还原酶**==)-> **==胆红素==**（游离/未结合/间接）&nbsp;&nbsp;&nbsp;
- **胆红素的血液转运**&nbsp;&nbsp;&nbsp;
	- **运输形式** -> **==胆红素-清蛋白复合体==**（游离胆红素 = 间接胆红素 = 未结合胆红素）&nbsp;&nbsp;&nbsp;
	- **意义1** -> 增加胆红素水溶性 -> 提高血浆运输能力&nbsp;&nbsp;&nbsp;
	- **意义2** -> 限制胆红素自由通透细胞膜 -> **==暂时性解毒==**&nbsp;&nbsp;&nbsp;
	- **竞争结合危险** -> **磺胺药/水杨酸/胆汁酸**竞争结合清蛋白 -> 游离胆红素 ↑ -> 可致**==胆红素脑病（核黄疸）==**&nbsp;&nbsp;&nbsp;
	- **新生儿黄疸** -> 血脑屏障不健全 -> 游离胆红素入脑 -> **==核黄疸==**&nbsp;&nbsp;&nbsp;
- **胆红素在肝中的代谢（三步骤）**&nbsp;&nbsp;&nbsp;
	- **步骤1：摄取** -> 胆红素自由通透肝细胞膜 → 胞浆与 **==Y蛋白/Z蛋白==** 结合 → 携带至**滑面内质网**&nbsp;&nbsp;&nbsp;
	- **步骤2：转化（核心！）**&nbsp;&nbsp;&nbsp;
		- 部位 -> **==滑面内质网==**&nbsp;&nbsp;&nbsp;
		- 反应类型 -> **==结合反应==**（第二相反应）-> 主要结合物：**==UDPGA==**&nbsp;&nbsp;&nbsp;
		- 催化酶 -> **==葡萄糖醛酸基转移酶 (UGT)==**&nbsp;&nbsp;&nbsp;
		- 产物 -> **==结合胆红素==**（直接胆红素/肝胆红素）-> 主要是**双葡糖醛酸胆红素**&nbsp;&nbsp;&nbsp;
		- **意义** -> 脂溶性有毒游离胆红素 → **水溶性无毒**结合胆红素 → 胆红素在体内**==解毒的重要方式==**&nbsp;&nbsp;&nbsp;
	- **步骤3：排泄** -> 肝经毛细胆管排泄结合胆红素 → **==限速步骤==**（主动耗能分泌）&nbsp;&nbsp;&nbsp;
- **==未结合胆红素 vs 结合胆红素 极限对比（必背！）==**&nbsp;&nbsp;&nbsp;
	- **==未结合胆红素==**（间接胆红素 / 游离胆红素）&nbsp;&nbsp;&nbsp;
		- 葡萄糖醛酸 -> **==未结合==**&nbsp;&nbsp;&nbsp;
		- 水溶性 -> **==小==** / 脂溶性 -> **==大==**&nbsp;&nbsp;&nbsp;
		- 透过细胞膜 + 毒性 -> **==大==**&nbsp;&nbsp;&nbsp;
		- 经肾小球随尿排出 -> **==不能==**&nbsp;&nbsp;&nbsp;
		- 对脑毒性 -> **==大==**（可通过血脑屏障）&nbsp;&nbsp;&nbsp;
		- 重氮试剂反应 -> **间接阳性**（需加乙醇才反应）&nbsp;&nbsp;&nbsp;
	- **==结合胆红素==**（直接胆红素 / 肝胆红素）&nbsp;&nbsp;&nbsp;
		- 葡萄糖醛酸 -> **==结合==**&nbsp;&nbsp;&nbsp;
		- 水溶性 -> **==大==** / 脂溶性 -> **==小==**&nbsp;&nbsp;&nbsp;
		- 透过细胞膜 + 毒性 -> **==小==**&nbsp;&nbsp;&nbsp;
		- 经肾小球随尿排出 -> **==能==**&nbsp;&nbsp;&nbsp;
		- 对脑毒性 -> **==小==**&nbsp;&nbsp;&nbsp;
		- 重氮试剂反应 -> **直接阳性**&nbsp;&nbsp;&nbsp;
- **胆红素代谢的临床药物关联**&nbsp;&nbsp;&nbsp;
	- **苯巴比妥** -> 诱导 **==UGT==** ↑ -> 促进游离→结合胆红素 → 治疗**新生儿黄疸**&nbsp;&nbsp;&nbsp;
	- **糖皮质激素** -> 诱导 UGT + 促进结合胆红素排泄 → 治疗**高胆红素血症**&nbsp;&nbsp;&nbsp;
