# 肝的生物转化作用 (第一相反应)&nbsp;&nbsp;&nbsp;

- **生物转化概念**&nbsp;&nbsp;&nbsp;
	- **定义** -> 机体将**==非营养物质==**进行化学转变 -> 增加**==水溶性与极性==** -> 使其容易排出体外&nbsp;&nbsp;&nbsp;
	- **非营养物质** -> 不构建组织 / 不供能 / 某些有生物学效应或潜在毒性&nbsp;&nbsp;&nbsp;
	- **主要部位** -> **==肝==**（承担 ==80-90%==）-> 亚细胞定位：**==微粒体==**为主 + 胞液 + 线粒体&nbsp;&nbsp;&nbsp;
- **生物转化的三大意义**&nbsp;&nbsp;&nbsp;
	- **意义1** -> 增加==水溶性和极性== -> 易随**粪便或尿液**排出&nbsp;&nbsp;&nbsp;
	- **意义2** -> 使非营养物质**==灭活/解毒==**（生物学活性降低或毒性减低）&nbsp;&nbsp;&nbsp;
	- **意义3** -> 具有**==解毒致毒双重性==** -> 部分物质转化后毒性反而增加（如黄曲霉素 B₁）&nbsp;&nbsp;&nbsp;
- **第一相反应**&nbsp;&nbsp;&nbsp;
	- **总目标** -> 使非极性基团 → 极性基团 -> 增强**==水溶性==**&nbsp;&nbsp;&nbsp;
	- **==氧化反应==**（最主要的第一相反应）&nbsp;&nbsp;&nbsp;
		- **==加单氧酶系==**（细胞色素 P450）-> 异源物 → 羟化产物 -> 参与药物和毒物转化&nbsp;&nbsp;&nbsp;
		- **单胺氧化酶系** -> 胺类物质&nbsp;&nbsp;&nbsp;
		- **脱氢酶系** -> 醇类或醛类 -> 如**乙醇代谢**&nbsp;&nbsp;&nbsp;
			- 乙醇 ->(==**醇脱氢酶 ADH**==)-> 乙醛 ->(==**醛脱氢酶 ALDH**==)-> 乙酸&nbsp;&nbsp;&nbsp;
			- ADH 活性稳定 / ALDH 因人而异&nbsp;&nbsp;&nbsp;
			- 乙醛蓄积 -> **血管扩张/面红/心动过速/头痛**&nbsp;&nbsp;&nbsp;
	- **==还原反应==**&nbsp;&nbsp;&nbsp;
		- **硝基还原酶** -> 硝基化合物&nbsp;&nbsp;&nbsp;
		- **偶氮还原酶** -> 偶氮化合物&nbsp;&nbsp;&nbsp;
	- **==水解反应==**&nbsp;&nbsp;&nbsp;
		- **酯酶** -> 脂类 / **酰胺酶** -> 酰胺类 / **糖苷酶** -> 糖苷类&nbsp;&nbsp;&nbsp;
- **加单氧酶系的核心地位**&nbsp;&nbsp;&nbsp;
	- **又称** -> 细胞色素 ==P450== / 混合功能氧化酶&nbsp;&nbsp;&nbsp;
	- **反应式** -> RH + O₂ + NADPH + H⁺ → **==ROH + NADP⁺ + H₂O==**&nbsp;&nbsp;&nbsp;
	- **能直接激活氧分子** -> 一个氧原子加入产物（羟化）、另一个生成水&nbsp;&nbsp;&nbsp;
	- **生理意义** -> 参与药物/毒物羟化 → 增水溶性 / 参与体内代谢过程中的羟化反应&nbsp;&nbsp;&nbsp;
- **生物转化的特点**&nbsp;&nbsp;&nbsp;
	- **连续性** -> 先第一相再第二相（如黄曲霉素先氧化再结合）&nbsp;&nbsp;&nbsp;
	- **多样性** -> 一种物质可经多种反应进行转化&nbsp;&nbsp;&nbsp;
	- **==解毒与致毒双重性==** -> 黄曲霉素 B₁ 经 P450 氧化 → **==环氧黄曲霉素==** → 与 DNA 鸟嘌呤结合 → 突变 → **==肝癌==**&nbsp;&nbsp;&nbsp;
