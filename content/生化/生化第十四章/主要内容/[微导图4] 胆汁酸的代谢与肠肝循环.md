# 胆汁酸的代谢与肠肝循环&nbsp;&nbsp;&nbsp;

- **胆汁概述**&nbsp;&nbsp;&nbsp;
	- **定义** -> 由**肝细胞**分泌 → 肝胆管 → 胆囊（储存浓缩）→ 肠道&nbsp;&nbsp;&nbsp;
	- **双重功能** -> 消化液（促进脂类消化吸收）+ 排泄液（排出胆红素等代谢物）&nbsp;&nbsp;&nbsp;
- **胆汁酸概述**&nbsp;&nbsp;&nbsp;
	- **定义** -> 肝细胞以**==胆固醇==**为原料合成的 24 碳类固醇化合物&nbsp;&nbsp;&nbsp;
	- **地位** -> 胆固醇在体内**==主要的代谢产物==** -> 以钠盐/钾盐形式存在 = 胆汁酸盐（胆盐）&nbsp;&nbsp;&nbsp;
- **胆汁酸的分类**&nbsp;&nbsp;&nbsp;
	- **初级胆汁酸（肝内合成）**&nbsp;&nbsp;&nbsp;
		- 合成部位 -> **==肝细胞的胞液和微粒体==**&nbsp;&nbsp;&nbsp;
		- 原料 -> **==胆固醇==**&nbsp;&nbsp;&nbsp;
		- 游离型 -> **==胆酸==** / **==鹅脱氧胆酸==**&nbsp;&nbsp;&nbsp;
		- 结合型 -> 甘氨胆酸 / 牛磺胆酸 / 甘氨鹅脱氧胆酸 / 牛磺鹅脱氧胆酸&nbsp;&nbsp;&nbsp;
	- **次级胆汁酸（肠道内生成）**&nbsp;&nbsp;&nbsp;
		- 生成部位 -> **==小肠下段和大肠==**&nbsp;&nbsp;&nbsp;
		- 原料 -> 初级胆汁酸&nbsp;&nbsp;&nbsp;
		- 关键反应 -> 肠道细菌催化 → **第 7 位 α-羟基脱氧**&nbsp;&nbsp;&nbsp;
		- 游离型 -> **==脱氧胆酸==**（来自胆酸）/ **==石胆酸==**（来自鹅脱氧胆酸）&nbsp;&nbsp;&nbsp;
		- 结合型 -> 甘氨脱氧胆酸 / 牛磺脱氧胆酸 / 甘氨石胆酸 / 牛磺石胆酸&nbsp;&nbsp;&nbsp;
- **胆汁酸的肠肝循环**&nbsp;&nbsp;&nbsp;
	- **定义** -> 胆汁酸排入肠腔 → 约 **==95%==** 被肠壁重吸收 → 经**==门静脉==**回肝 → 肝内转变成结合胆汁酸 → 与新合成胆汁酸一起 → 再次排入肠腔&nbsp;&nbsp;&nbsp;
	- **循环参数**&nbsp;&nbsp;&nbsp;
		- 代谢池总量 -> **==3-5g==**&nbsp;&nbsp;&nbsp;
		- 每日需要量 -> **==16-32g==**&nbsp;&nbsp;&nbsp;
		- 循环次数 -> 每日 6-10 次&nbsp;&nbsp;&nbsp;
	- **生理意义** -> 使有限的胆汁酸经**==循环利用==** -> 满足机体对胆汁酸的巨大需求&nbsp;&nbsp;&nbsp;
- **胆汁酸的生理作用**&nbsp;&nbsp;&nbsp;
	- **作用1** -> 促进脂类的消化和吸收 -> 作为**==乳化剂==** -> 降低油水界面张力&nbsp;&nbsp;&nbsp;
	- **作用2** -> 抑制胆汁中胆固醇的析出 -> 防止**==胆结石==**形成&nbsp;&nbsp;&nbsp;
	- **临床应用** -> **鹅脱氧胆酸 / 熊脱氧胆酸**治疗胆固醇结石&nbsp;&nbsp;&nbsp;
- **胆汁酸的代谢调控链**&nbsp;&nbsp;&nbsp;
	- **胆固醇 → 胆汁酸** = 胆固醇的**==主要代谢去路==**&nbsp;&nbsp;&nbsp;
	- 肠肝循环被阻断（如服用胆汁酸结合树脂）-> 肝胆固醇转化为胆汁酸 ↑ -> 血胆固醇 ↓&nbsp;&nbsp;&nbsp;
	- 这是降胆固醇药物的原理之一&nbsp;&nbsp;&nbsp;
