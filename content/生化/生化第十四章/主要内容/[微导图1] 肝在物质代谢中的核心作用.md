# 肝在物质代谢中的核心作用&nbsp;&nbsp;&nbsp;

- **肝的定位**&nbsp;&nbsp;&nbsp;
	- 肝 = **==物质代谢中枢==** + 生物转化 + 分泌 + 排泄&nbsp;&nbsp;&nbsp;
	- 结构基础 -> **==肝门静脉 + 肝动脉==**（双输入）/ **==肝静脉 + 胆道==**（双输出）&nbsp;&nbsp;&nbsp;
- **肝在糖代谢的作用**&nbsp;&nbsp;&nbsp;
	- 核心任务 -> 维持**==血糖水平相对稳定==** -> 保障大脑和红细胞供能&nbsp;&nbsp;&nbsp;
	- 能吃能干的途径 -> **==糖异生==** / 糖原合成分解 / 糖酵解 / 有氧氧化 / 磷酸戊糖途径&nbsp;&nbsp;&nbsp;
	- 一句话 -> 肝是**==糖异生的主要器官==**&nbsp;&nbsp;&nbsp;
- **肝在脂代谢的作用**&nbsp;&nbsp;&nbsp;
	- 酮体生成 -> 肝是酮体生成的**==唯一场所==**（肝内生成、肝外利用）&nbsp;&nbsp;&nbsp;
	- 脂蛋白输出 -> 合成**==VLDL / HDL==** -> 将脂肪输送到肝外&nbsp;&nbsp;&nbsp;
	- 胆固醇代谢 -> 合成胆固醇 + 转变为**==胆汁酸==**（胆固醇主要去路）&nbsp;&nbsp;&nbsp;
	- 其余 -> 脂肪酸 β-氧化 / 脂肪酸合成 / 脂肪合成&nbsp;&nbsp;&nbsp;
- **肝在蛋白质代谢的作用**&nbsp;&nbsp;&nbsp;
	- 合成**==血浆蛋白质==**的重要器官（白蛋白、凝血因子等）&nbsp;&nbsp;&nbsp;
	- **==尿素合成==**的主要场所 -> 氨的解毒 -> **==尿素循环==**&nbsp;&nbsp;&nbsp;
	- 氨基酸分解和转化的重要场所&nbsp;&nbsp;&nbsp;
- **肝在维生素代谢的作用**&nbsp;&nbsp;&nbsp;
	- 吸收 -> 胆汁酸帮助**==脂溶性维生素==**吸收&nbsp;&nbsp;&nbsp;
	- 储存 -> **==Vit A / Vit K / Vit E / Vit B₁₂==**&nbsp;&nbsp;&nbsp;
	- 转化 -> Vit D₃ → **==25-羟维生素 D₃==**（肝内羟化）&nbsp;&nbsp;&nbsp;
- **肝在激素代谢的作用**&nbsp;&nbsp;&nbsp;
	- **==激素灭活==** -> 激素降解或失活 -> 肝参与灭活和排泄&nbsp;&nbsp;&nbsp;
	- 肝功能受损 -> 激素灭活↓ -> 肝掌/蜘蛛痣/男性乳房发育&nbsp;&nbsp;&nbsp;
- **肝的独有功能速记**&nbsp;&nbsp;&nbsp;
	- **酮体生成** -> **==唯一场所==**&nbsp;&nbsp;&nbsp;
	- **尿素合成** -> **==主要场所==**&nbsp;&nbsp;&nbsp;
	- **糖异生** -> **==主要器官==**&nbsp;&nbsp;&nbsp;
	- **VLDL 合成** -> **==主要器官==**&nbsp;&nbsp;&nbsp;
	- **胆汁酸合成** -> **==唯一场所==**&nbsp;&nbsp;&nbsp;
	- **血浆蛋白合成** -> **==主要场所==**&nbsp;&nbsp;&nbsp;
