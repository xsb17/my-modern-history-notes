# 脂肪酸的合成代谢&nbsp;&nbsp;&nbsp;

- **脂肪酸合成概述**&nbsp;&nbsp;&nbsp;
	- 主要产物 -> **==软脂酸（16C棕榈酸）==**&nbsp;&nbsp;&nbsp;
	- 组织定位 -> **==肝==**(主要场所)/大多数组织&nbsp;&nbsp;&nbsp;
	- 亚细胞定位 -> ==**胞液**==（关键！与β-氧化的线粒体相反）&nbsp;&nbsp;&nbsp;
	- 原料 -> **==乙酰CoA、NADPH==**、ATP、HCO₃⁻、Mn²⁺&nbsp;&nbsp;&nbsp;
- **乙酰CoA的来源与转运**&nbsp;&nbsp;&nbsp;
	- 乙酰CoA全部在**==线粒体内==**产生 -> 不能自由通过线粒体内膜&nbsp;&nbsp;&nbsp;
	- 出线粒体机制 -> ==**柠檬酸-丙酮酸循环**==&nbsp;&nbsp;&nbsp;
		- 线粒体内 -> 乙酰CoA + 草酰乙酸 -> 柠檬酸 -> 穿出线粒体&nbsp;&nbsp;&nbsp;
		- 胞液中 -> 柠檬酸 ->(柠檬酸裂解酶/ATP-柠檬酸裂解酶)-> **乙酰CoA + 草酰乙酸**&nbsp;&nbsp;&nbsp;
	- 关键 -> 柠檬酸裂解酶 -> 将柠檬酸裂解为乙酰CoA（胞液）&nbsp;&nbsp;&nbsp;
- **NADPH的来源**&nbsp;&nbsp;&nbsp;
	- **磷酸戊糖途径** -> **==主要来源==**&nbsp;&nbsp;&nbsp;
	- 胞液中异柠檬酸脱氢酶&nbsp;&nbsp;&nbsp;
	- 胞液中**苹果酸酶**（丙酮酸-苹果酸循环中）&nbsp;&nbsp;&nbsp;
- **乙酰CoA的活化**&nbsp;&nbsp;&nbsp;
	- 乙酰CoA ->(==**乙酰CoA羧化酶**==)-> **==丙二酰CoA==**&nbsp;&nbsp;&nbsp;
	- ==**乙酰CoA羧化酶**== -> 脂肪酸合成的**==限速酶==**&nbsp;&nbsp;&nbsp;
	- 辅基 -> ==**生物素**==（维生素H/B₇）&nbsp;&nbsp;&nbsp;
	- 激活剂 -> **Mn²⁺**&nbsp;&nbsp;&nbsp;
	- 调节&nbsp;&nbsp;&nbsp;
		- 别构激活剂 -> **==柠檬酸、异柠檬酸==**&nbsp;&nbsp;&nbsp;
		- 别构抑制剂 -> **==软脂酰CoA==**（产物反馈抑制）、长链脂酰CoA&nbsp;&nbsp;&nbsp;
		- 共价修饰 -> **胰高血糖素**磷酸化**失活** / **胰岛素**去磷酸化**复活**&nbsp;&nbsp;&nbsp;
	- **丙二酰CoA** -> 是==**肉碱脂酰转移酶Ⅰ**==的变构抑制剂 -> **合成时抑制β-氧化**（避免"无效循环"）&nbsp;&nbsp;&nbsp;
- **脂肪酸合成酶系**&nbsp;&nbsp;&nbsp;
	- 大肠杆菌 -> 7种酶 + **==ACP==**(酰基载体蛋白) -> 多酶复合物&nbsp;&nbsp;&nbsp;
	- **哺乳动物** -> 7种酶活性在**一条多肽链**上 -> **==多功能酶==** -> 有活性的酶为两个相同亚基首尾相连的**==二聚体==**&nbsp;&nbsp;&nbsp;
- **合成过程**&nbsp;&nbsp;&nbsp;
	- 从乙酰CoA + 丙二酰CoA -> 每次延长**==2个碳原子==**&nbsp;&nbsp;&nbsp;
	- 经过**==7轮循环==** -> 释出**==软脂酸(16C)==**&nbsp;&nbsp;&nbsp;
	- 总反应式 -> 乙酰CoA + **==7丙二酰CoA==** + 14NADPH + 14H⁺ -> 软脂酸 + 7CO₂ + 6H₂O + 8CoA + 14NADP⁺&nbsp;&nbsp;&nbsp;
- **脂肪酸碳链延长与去饱和**&nbsp;&nbsp;&nbsp;
	- 碳链延长&nbsp;&nbsp;&nbsp;
		- 内质网 -> 酰基载体：CoA-SH -> 可延长至24C -> 以18C硬脂酸最多&nbsp;&nbsp;&nbsp;
		- 线粒体 -> 过程与β-氧化逆反应相似&nbsp;&nbsp;&nbsp;
	- 去饱和 -> 微粒体脂酰CoA去饱和酶（混合功能氧化酶）&nbsp;&nbsp;&nbsp;
- **必需脂肪酸（不能自身合成）**&nbsp;&nbsp;&nbsp;
	- **==亚油酸、α-亚麻酸、花生四烯酸==**（多不饱和脂肪酸）&nbsp;&nbsp;&nbsp;
	- 口诀 -> "炸麻（亚麻酸）花（花生四烯酸）要油（亚油酸）"&nbsp;&nbsp;&nbsp;
	- 花生四烯酸 -> 合成**前列腺素/血栓烷/白三烯** -> 炎症/血栓/血管舒缩&nbsp;&nbsp;&nbsp;
- **脂肪合成的调节**&nbsp;&nbsp;&nbsp;
	- 进食糖类 -> 糖代谢↑ -> NADPH+乙酰CoA↑ + 柠檬酸↑ -> **脂肪酸合成↑**&nbsp;&nbsp;&nbsp;
	- 激素 -> **胰岛素**促进脂酸合成 / **胰高血糖素+肾上腺素**抑制&nbsp;&nbsp;&nbsp;
