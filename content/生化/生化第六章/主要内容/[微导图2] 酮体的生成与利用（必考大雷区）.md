# 酮体的生成与利用（必考大雷区）&nbsp;&nbsp;&nbsp;

- **酮体概念**&nbsp;&nbsp;&nbsp;
	- 脂肪酸在**==肝脏==**分解代谢时的**==特有中间代谢物==**&nbsp;&nbsp;&nbsp;
	- 三种成分&nbsp;&nbsp;&nbsp;
		- ==**乙酰乙酸**== -> 约占 **30%**&nbsp;&nbsp;&nbsp;
		- ==**β-羟丁酸**== -> 约占 **70%**&nbsp;&nbsp;&nbsp;
		- **丙酮** -> 微量（有气味，经呼吸排出）&nbsp;&nbsp;&nbsp;
- **酮体代谢定位（必考核心）**&nbsp;&nbsp;&nbsp;
	- **生成** -> **==肝细胞线粒体==**（肝有生成酮体的酶）&nbsp;&nbsp;&nbsp;
	- **利用** -> **==肝外组织线粒体==**（心、肾、脑、骨骼肌等）&nbsp;&nbsp;&nbsp;
	- **肝脏不能利用酮体！** -> 缺乏**==琥珀酰CoA转硫酶==**等利用酮体的酶&nbsp;&nbsp;&nbsp;
	- 口诀 -> **"肝内生成，肝外利用"**&nbsp;&nbsp;&nbsp;
- **酮体生成过程**&nbsp;&nbsp;&nbsp;
	- 原料 -> **==乙酰CoA==**（来自脂肪酸β-氧化）&nbsp;&nbsp;&nbsp;
	- 关键酶 -> ==**HMG-CoA合酶**==（羟甲基戊二酸单酰CoA合酶）&nbsp;&nbsp;&nbsp;
	- 路径 -> 2乙酰CoA -> 乙酰乙酰CoA -> ==**HMG-CoA**== ->(==**HMG-CoA裂解酶**==)-> 乙酰乙酸 -> 还原 -> β-羟丁酸 / 脱羧 -> 丙酮&nbsp;&nbsp;&nbsp;
	- 注意与胆固醇区别 -> ==**HMG-CoA裂解酶**==(酮体) vs ==**HMG-CoA还原酶**==(胆固醇) -> 线粒体中裂解/胞液中还原&nbsp;&nbsp;&nbsp;
- **酮体利用**&nbsp;&nbsp;&nbsp;
	- 肝外组织具有的酶&nbsp;&nbsp;&nbsp;
		- **==琥珀酰CoA转硫酶==** -> 心、肾、脑、骨骼肌线粒体&nbsp;&nbsp;&nbsp;
		- **==乙酰乙酰CoA硫激酶==** -> 心、肾、脑线粒体&nbsp;&nbsp;&nbsp;
		- **==乙酰乙酰CoA硫解酶==** -> 心、肾、脑、骨骼肌线粒体&nbsp;&nbsp;&nbsp;
	- β-羟丁酸 -> 乙酰乙酸 -> 乙酰乙酰CoA -> 2乙酰CoA -> **TCA循环**&nbsp;&nbsp;&nbsp;
- **酮体生理意义（三点必背）**&nbsp;&nbsp;&nbsp;
	- 酮体是**==肝输出能源的一种形式==** -> 将脂肪酸能量运输到肝外&nbsp;&nbsp;&nbsp;
	- **长期饥饿/糖供应不足**时 -> **==脑细胞可利用酮体供能==** -> 酮体分子量小/易溶于水/**==易透过血脑屏障==**&nbsp;&nbsp;&nbsp;
	- 正常情况下血中仅有少量酮体&nbsp;&nbsp;&nbsp;
- **酮体代谢失衡 — 酮症酸中毒**&nbsp;&nbsp;&nbsp;
	- 病因 -> 长期饥饿/高脂低糖饮食/**==严重糖尿病==**&nbsp;&nbsp;&nbsp;
	- 机制 -> 酮体生成 > 肝外利用能力 -> **酮血症** -> **酮尿症** -> ==**酮症酸中毒**==&nbsp;&nbsp;&nbsp;
	- 糖尿病酮症酸中毒 -> **烂苹果味**呼吸（丙酮气味）&nbsp;&nbsp;&nbsp;
- **酮体生成的调节**&nbsp;&nbsp;&nbsp;
	- 饱食 -> 胰岛素↑ -> 脂肪动员↓ -> β-氧化↓ -> 酮体↓&nbsp;&nbsp;&nbsp;
	- 饥饿 -> 胰高血糖素↑ -> 脂肪动员↑ -> β-氧化↑ -> 酮体↑&nbsp;&nbsp;&nbsp;
	- **丙二酰CoA** -> 竞争性抑制==**肉碱脂酰转移酶Ⅰ**== -> 抑制脂酰CoA进入线粒体 -> **β-氧化↓ -> 酮体↓**&nbsp;&nbsp;&nbsp;
	- 糖代谢减弱 -> 丙二酰CoA↓ -> β-氧化↑ -> 酮体↑（糖供应不足时酮体生成增加的根本原因）&nbsp;&nbsp;&nbsp;
