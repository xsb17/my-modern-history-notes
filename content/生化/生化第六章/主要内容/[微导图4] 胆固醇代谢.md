# 胆固醇代谢&nbsp;&nbsp;&nbsp;

- **胆固醇概述**&nbsp;&nbsp;&nbsp;
	- 基本结构 -> ==**环戊烷多氢菲**== -> **27碳**&nbsp;&nbsp;&nbsp;
	- 弱两性分子&nbsp;&nbsp;&nbsp;
	- 人体含量 -> 约**140克** -> 广泛分布 -> 脑/神经组织约占25%&nbsp;&nbsp;&nbsp;
	- 存在形式 -> **游离胆固醇** + **胆固醇酯**（与脂肪酸结合）&nbsp;&nbsp;&nbsp;
- **胆固醇生理功能**&nbsp;&nbsp;&nbsp;
	- 构成**生物膜**成分 -> 控制膜流动性&nbsp;&nbsp;&nbsp;
	- 合成==**胆汁酸**==（主要去路）&nbsp;&nbsp;&nbsp;
	- 合成**==类固醇激素==**（肾上腺皮质激素、雄激素、雌激素等）&nbsp;&nbsp;&nbsp;
	- 合成**==维生素D₃==**（皮肤中7-脱氢胆固醇 -> 紫外线 -> VitD₃）&nbsp;&nbsp;&nbsp;
- **胆固醇的合成**&nbsp;&nbsp;&nbsp;
	- 组织定位 -> **==肝==**(主要)/小肠 -> 所有组织(不包括脑和成熟红细胞)&nbsp;&nbsp;&nbsp;
	- 细胞定位 -> **==胞质溶胶 + 滑面内质网==**&nbsp;&nbsp;&nbsp;
	- 原料 -> **==乙酰CoA + NADPH==** -> 1分子胆固醇 -> **==18乙酰CoA + 36ATP + 16NADPH==**（"三高"）&nbsp;&nbsp;&nbsp;
	- 乙酰CoA出线粒体 -> ==**柠檬酸-丙酮酸循环**==&nbsp;&nbsp;&nbsp;
- **胆固醇合成三阶段**&nbsp;&nbsp;&nbsp;
	- **第一阶段：甲羟戊酸合成（关键！）**&nbsp;&nbsp;&nbsp;
		- 3乙酰CoA -> HMG-CoA ->(==**HMG-CoA还原酶**==)-> **甲羟戊酸(C6)**&nbsp;&nbsp;&nbsp;
		- ==**HMG-CoA还原酶**== -> 胆固醇合成的**==限速酶==**&nbsp;&nbsp;&nbsp;
		- 区别 -> 线粒体中HMG-CoA裂解 -> **酮体** / 胞液中HMG-CoA还原 -> **胆固醇**&nbsp;&nbsp;&nbsp;
		- 他汀类药物 -> 竞争性抑制==**HMG-CoA还原酶**== -> 降胆固醇&nbsp;&nbsp;&nbsp;
	- 第二阶段：鲨烯合成 -> 甲羟戊酸 -> 活化的异戊烯(C5) -> **鲨烯(C30)**&nbsp;&nbsp;&nbsp;
	- 第三阶段：胆固醇合成 -> 鲨烯 -> **胆固醇(C27)**&nbsp;&nbsp;&nbsp;
- **HMG-CoA还原酶的调节（多考点）**&nbsp;&nbsp;&nbsp;
	- **胆固醇**（主要）-> **==反馈抑制==**&nbsp;&nbsp;&nbsp;
	- 饱食 -> 酶活性↑ / 饥饿 -> 酶活性↓&nbsp;&nbsp;&nbsp;
	- 激素&nbsp;&nbsp;&nbsp;
		- **胰岛素** -> 促进 -> 去磷酸化激活&nbsp;&nbsp;&nbsp;
		- **胰高血糖素** -> 抑制 -> 磷酸化失活&nbsp;&nbsp;&nbsp;
	- **甲状腺素** -> 诱导酶合成↑，但更强化胆固醇→胆汁酸转化 -> **==甲亢病人低胆固醇==**&nbsp;&nbsp;&nbsp;
	- **昼夜节律** -> 午夜最高，中午最低&nbsp;&nbsp;&nbsp;
- **胆固醇的酯化**&nbsp;&nbsp;&nbsp;
	- **组织细胞内** -> ==**ACAT**==（脂酰CoA胆固醇脂酰转移酶）-> 脂酰CoA提供脂肪酸&nbsp;&nbsp;&nbsp;
	- **血浆中** -> ==**LCAT**==（卵磷脂胆固醇脂酰转移酶）-> 卵磷脂提供脂肪酸 -> 激活剂：==**apo AⅠ**==&nbsp;&nbsp;&nbsp;
- **胆固醇的转化与排泄（三大去路）**&nbsp;&nbsp;&nbsp;
	- **转变为胆汁酸**（肝细胞中）-> **主要去路** -> 随胆汁进入肠道&nbsp;&nbsp;&nbsp;
	- **转变为类固醇激素** -> 肾上腺/性腺 -> 雄激素/雌激素/肾上腺皮质激素&nbsp;&nbsp;&nbsp;
	- **转变为维生素D₃** -> 皮肤中7-脱氢胆固醇 ->（紫外线）-> VitD₃&nbsp;&nbsp;&nbsp;
	- 肠道残留胆固醇 -> 粪固醇 -> 直接排出&nbsp;&nbsp;&nbsp;
- **乙酰CoA来源与去路（重要串联考点）**&nbsp;&nbsp;&nbsp;
	- 来源 -> 糖有氧氧化 / 脂肪酸β-氧化 / 氨基酸氧化 / 酮体氧化 / 柠檬酸裂解&nbsp;&nbsp;&nbsp;
	- 去路 -> TCA彻底氧化 / 合成脂肪酸 / **==合成胆固醇==** / **==合成酮体==** / 乙酰化反应&nbsp;&nbsp;&nbsp;
