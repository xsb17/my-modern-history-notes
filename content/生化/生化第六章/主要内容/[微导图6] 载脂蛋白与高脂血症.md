# 载脂蛋白与高脂血症&nbsp;&nbsp;&nbsp;

- **载脂蛋白(apo)概述**&nbsp;&nbsp;&nbsp;
	- 定义 -> 血浆脂蛋白中的**蛋白质部分**&nbsp;&nbsp;&nbsp;
	- 功能 -> ==**结合转运脂质 / 稳定脂蛋白结构 / 识别受体 / 调节酶活性**==&nbsp;&nbsp;&nbsp;
	- 20多种 -> 主要：apo A/B/C/D/E&nbsp;&nbsp;&nbsp;
- **核心载脂蛋白的归属与功能**&nbsp;&nbsp;&nbsp;
	- **Apo AⅠ**&nbsp;&nbsp;&nbsp;
		- 归属 -> **HDL**主要蛋白&nbsp;&nbsp;&nbsp;
		- 功能 -> 激活==**LCAT**==（卵磷脂胆固醇酯酰转移酶）/ 识别HDL受体&nbsp;&nbsp;&nbsp;
		- 临床 -> HDL标志物 / **抗动脉粥样硬化**&nbsp;&nbsp;&nbsp;
	- **Apo AⅡ**&nbsp;&nbsp;&nbsp;
		- 归属 -> HDL&nbsp;&nbsp;&nbsp;
		- 功能 -> 激活**HL**（肝脂肪酶）&nbsp;&nbsp;&nbsp;
	- **Apo B48**&nbsp;&nbsp;&nbsp;
		- 归属 -> **CM**（唯一）&nbsp;&nbsp;&nbsp;
		- 功能 -> 结构蛋白 / CM合成和分泌必需&nbsp;&nbsp;&nbsp;
		- 来源 -> 小肠（B100 mRNA编辑产物）&nbsp;&nbsp;&nbsp;
	- **Apo B100**&nbsp;&nbsp;&nbsp;
		- 归属 -> **VLDL + LDL**（主要）&nbsp;&nbsp;&nbsp;
		- 功能 -> 结构蛋白 / **识别LDL受体**（apo B/E受体）&nbsp;&nbsp;&nbsp;
		- 临床 -> LDL的标志物 / 致AS&nbsp;&nbsp;&nbsp;
	- **Apo CⅡ**&nbsp;&nbsp;&nbsp;
		- 归属 -> CM、VLDL、HDL均有&nbsp;&nbsp;&nbsp;
		- 功能 -> **激活LPL**（脂蛋白脂肪酶）-> 促进TG水解&nbsp;&nbsp;&nbsp;
		- 临床 -> CⅡ缺乏 -> LPL不能激活 -> **高TG血症**&nbsp;&nbsp;&nbsp;
	- **Apo CⅢ**&nbsp;&nbsp;&nbsp;
		- 归属 -> CM、VLDL、HDL&nbsp;&nbsp;&nbsp;
		- 功能 -> **抑制LPL** -> 与CⅡ拮抗&nbsp;&nbsp;&nbsp;
	- **Apo E**&nbsp;&nbsp;&nbsp;
		- 归属 -> CM、VLDL、HDL&nbsp;&nbsp;&nbsp;
		- 功能 -> **==识别LDL受体==**（apo B/E受体）-> 介导CM残粒/IDL摄取&nbsp;&nbsp;&nbsp;
		- 临床 -> E变异 -> **Ⅲ型高脂血症**&nbsp;&nbsp;&nbsp;
- **载脂蛋白激活酶速记**&nbsp;&nbsp;&nbsp;
	- **AⅠ激活LCAT** / **AⅡ激活HL** / **CⅡ激活LPL** / **CⅢ抑制LPL**&nbsp;&nbsp;&nbsp;
	- 口诀 -> "AI开L锁(AⅠ→LCAT) / AII开H锁(AⅡ→HL) / CII开L锁(CⅡ→LPL)"&nbsp;&nbsp;&nbsp;
- **LDL受体途径**&nbsp;&nbsp;&nbsp;
	- LDL受体 -> **apo B/E受体** -> 识别含**apo B100或apo E**的脂蛋白&nbsp;&nbsp;&nbsp;
	- 分布 -> 肝(主要)、肾上腺皮质、卵巢、睾丸、动脉壁等&nbsp;&nbsp;&nbsp;
	- 约**2/3** LDL经此途径降解&nbsp;&nbsp;&nbsp;
- **非受体途径（清道夫途径）**&nbsp;&nbsp;&nbsp;
	- 修饰LDL(ox-LDL) -> 被**清道夫受体(SR)**识别 -> 巨噬细胞/血管内皮细胞清除&nbsp;&nbsp;&nbsp;
	- 约**1/3** LDL经此途径&nbsp;&nbsp;&nbsp;
	- ==**ox-LDL**== -> 巨噬细胞吞噬 -> ==**泡沫细胞**==形成 -> **动脉粥样硬化**&nbsp;&nbsp;&nbsp;
- **高脂血症（高脂蛋白血症）**&nbsp;&nbsp;&nbsp;
	- 定义 -> 空腹血脂高于正常参考值上限&nbsp;&nbsp;&nbsp;
	- 诊断 -> 成人空腹12-14h -> TG > **==200mg/dL==** / 胆固醇 > **==240mg/dL==**&nbsp;&nbsp;&nbsp;
	- 常见类型 -> **==高甘油三脂血症 + 高胆固醇血症==**&nbsp;&nbsp;&nbsp;
	- 六型分型（按脂蛋白变化）&nbsp;&nbsp;&nbsp;
		- **Ⅰ型** -> CM↑ -> TG↑↑↑&nbsp;&nbsp;&nbsp;
		- **Ⅱa型** -> LDL↑ -> 胆固醇↑↑&nbsp;&nbsp;&nbsp;
		- **Ⅱb型** -> LDL↑ + VLDL↑&nbsp;&nbsp;&nbsp;
		- **Ⅲ型** -> IDL↑（apo E异常）&nbsp;&nbsp;&nbsp;
		- **Ⅳ型** -> VLDL↑ -> TG↑↑（最常见）&nbsp;&nbsp;&nbsp;
		- **Ⅴ型** -> VLDL↑ + CM↑&nbsp;&nbsp;&nbsp;
- **动脉粥样硬化(AS)的脂蛋白机制**&nbsp;&nbsp;&nbsp;
	- **致AS因素**&nbsp;&nbsp;&nbsp;
		- **LDL↑** -> 与AS发病率正相关&nbsp;&nbsp;&nbsp;
		- **VLDL↑** -> 致AS&nbsp;&nbsp;&nbsp;
		- ox-LDL -> 巨噬细胞清道夫受体 -> 泡沫细胞&nbsp;&nbsp;&nbsp;
	- **抗AS因素**&nbsp;&nbsp;&nbsp;
		- **HDL↑** -> 与AS发病率**负相关**&nbsp;&nbsp;&nbsp;
		- HDL逆向转运胆固醇 -> 肝代谢排出&nbsp;&nbsp;&nbsp;
- **临床速记串讲**&nbsp;&nbsp;&nbsp;
	- 降胆固醇药 -> ==**他汀类**== -> 抑制==**HMG-CoA还原酶**== -> LDL受体↑ -> 血浆LDL↓&nbsp;&nbsp;&nbsp;
	- 降TG药 -> 贝特类 -> 激活PPARα -> LPL活性↑&nbsp;&nbsp;&nbsp;
	- 减肥药 -> **奥利司他** -> 抑制肠道**胰脂酶** -> 阻断脂肪吸收&nbsp;&nbsp;&nbsp;
	- 减肥理论 -> **左旋肉碱** -> 促进脂肪酸进入线粒体&nbsp;&nbsp;&nbsp;
