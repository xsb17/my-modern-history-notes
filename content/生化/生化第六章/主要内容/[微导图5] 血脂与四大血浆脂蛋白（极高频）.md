# 血脂与四大血浆脂蛋白（极高频）&nbsp;&nbsp;&nbsp;

- **血脂概念**&nbsp;&nbsp;&nbsp;
	- 血浆中脂质的总称 -> **==TG、磷脂、胆固醇、胆固醇酯、游离脂肪酸==**&nbsp;&nbsp;&nbsp;
	- 来源 -> 外源性(食物) / 内源性(肝/脂肪合成+脂肪动员)&nbsp;&nbsp;&nbsp;
	- 去路 -> 氧化供能/脂肪储存/生物膜/转化&nbsp;&nbsp;&nbsp;
	- 空腹正常值参考 -> TG 0.11-1.69 / TC 2.59-6.47 mmol/L&nbsp;&nbsp;&nbsp;
- **血浆脂蛋白结构**&nbsp;&nbsp;&nbsp;
	- = 血脂 + **载脂蛋白(apo)** -> 亲水复合体&nbsp;&nbsp;&nbsp;
	- 内核 -> 疏水的**TG + 胆固醇酯**&nbsp;&nbsp;&nbsp;
	- 外层 -> 两性分子**磷脂 + 游离胆固醇 + 载脂蛋白**&nbsp;&nbsp;&nbsp;
- **四大脂蛋白分类（超速离心法，按密度从小到大）**&nbsp;&nbsp;&nbsp;
	- **CM（乳糜微粒）**&nbsp;&nbsp;&nbsp;
		- 密度 -> **<0.95**（最低）&nbsp;&nbsp;&nbsp;
		- 脂类 -> 含**TG最多**(80-95%)&nbsp;&nbsp;&nbsp;
		- 蛋白质 -> **最少**(1%)&nbsp;&nbsp;&nbsp;
		- 载脂蛋白 -> **ApoB48**、AⅠ、AⅡ、AⅣ、CⅠ/CⅡ/CⅢ、E&nbsp;&nbsp;&nbsp;
		- 合成部位 -> **小肠粘膜细胞**&nbsp;&nbsp;&nbsp;
		- **核心功能** -> 转运**==外源性TG和胆固醇==**&nbsp;&nbsp;&nbsp;
		- 电泳位置 -> 原点（CM）&nbsp;&nbsp;&nbsp;
	- **VLDL（极低密度脂蛋白）**&nbsp;&nbsp;&nbsp;
		- 密度 -> 0.95-1.006&nbsp;&nbsp;&nbsp;
		- 脂类 -> 含TG 50-70%&nbsp;&nbsp;&nbsp;
		- 蛋白质 -> 5-10%&nbsp;&nbsp;&nbsp;
		- 载脂蛋白 -> **ApoB100**、CⅠ/CⅡ/CⅢ、E&nbsp;&nbsp;&nbsp;
		- 合成部位 -> **肝细胞**&nbsp;&nbsp;&nbsp;
		- **核心功能** -> 转运**==内源性TG和胆固醇==**&nbsp;&nbsp;&nbsp;
		- 电泳位置 -> 前β-脂蛋白&nbsp;&nbsp;&nbsp;
	- **LDL（低密度脂蛋白）**&nbsp;&nbsp;&nbsp;
		- 密度 -> 1.006-1.063&nbsp;&nbsp;&nbsp;
		- 脂类 -> 含**胆固醇及其酯最多**(40-50%)&nbsp;&nbsp;&nbsp;
		- 蛋白质 -> 20-25%&nbsp;&nbsp;&nbsp;
		- 载脂蛋白 -> **ApoB100**（几乎唯一）&nbsp;&nbsp;&nbsp;
		- 合成部位 -> **血浆**（由VLDL转变而来）&nbsp;&nbsp;&nbsp;
		- **核心功能** -> 转运内源性**胆固醇**（**==肝→肝外==**）&nbsp;&nbsp;&nbsp;
		- 电泳位置 -> β-脂蛋白&nbsp;&nbsp;&nbsp;
		- **致动脉粥样硬化** -> LDL↑与AS正相关&nbsp;&nbsp;&nbsp;
	- **HDL（高密度脂蛋白）**&nbsp;&nbsp;&nbsp;
		- 密度 -> **1.063-1.21**（最高）&nbsp;&nbsp;&nbsp;
		- 脂类 -> 含脂类50%&nbsp;&nbsp;&nbsp;
		- 蛋白质 -> **最多**(约50%)&nbsp;&nbsp;&nbsp;
		- 载脂蛋白 -> **Apo AⅠ、AⅡ**&nbsp;&nbsp;&nbsp;
		- 合成部位 -> **肝、肠、血浆**等&nbsp;&nbsp;&nbsp;
		- **核心功能** -> **逆向转运胆固醇**（**==肝外→肝==**）&nbsp;&nbsp;&nbsp;
		- 电泳位置 -> **α-脂蛋白**&nbsp;&nbsp;&nbsp;
		- **抗动脉粥样硬化** -> HDL↑与AS负相关&nbsp;&nbsp;&nbsp;
- **四大脂蛋白功能口诀（必背）**&nbsp;&nbsp;&nbsp;
	- **CM** -> **"外"** -> 外源性TG&nbsp;&nbsp;&nbsp;
	- **VLDL** -> **"内"** -> 内源性TG&nbsp;&nbsp;&nbsp;
	- **LDL** -> **"胆到外"** -> 胆固醇肝→肝外&nbsp;&nbsp;&nbsp;
	- **HDL** -> **"胆回肝"** -> 胆固醇肝外→肝（逆向转运）&nbsp;&nbsp;&nbsp;
- **脂蛋白代谢关键酶**&nbsp;&nbsp;&nbsp;
	- **LPL（脂蛋白脂肪酶）** -> 水解CM/VLDL中TG -> 激活剂：==**apo CⅡ**==&nbsp;&nbsp;&nbsp;
	- **HL（肝脂肪酶）** -> 水解脂蛋白中TG/磷脂 -> 激活剂：apo AⅡ&nbsp;&nbsp;&nbsp;
	- **LCAT（卵磷脂胆固醇酯酰转移酶）** -> 催化胆固醇酯化(血浆) -> 激活剂：==**apo AⅠ**==&nbsp;&nbsp;&nbsp;
- **动脉粥样硬化(AS)关系**&nbsp;&nbsp;&nbsp;
	- **LDL + VLDL** -> **致AS** -> LDL↑与AS正相关 -> ox-LDL被巨噬细胞清道夫受体吞噬 -> ==**泡沫细胞**==&nbsp;&nbsp;&nbsp;
	- **HDL** -> **抗AS** -> HDL↑与AS负相关 -> 逆向转运胆固醇到肝代谢&nbsp;&nbsp;&nbsp;
