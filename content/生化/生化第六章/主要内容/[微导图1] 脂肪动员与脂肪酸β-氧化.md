# 脂肪动员与脂肪酸β-氧化&nbsp;&nbsp;&nbsp;

- **脂肪动员**&nbsp;&nbsp;&nbsp;
	- 定义 -> 脂肪细胞中的脂肪 -> 经脂肪酶逐步水解 -> **游离脂肪酸 + 甘油** -> 释放入血&nbsp;&nbsp;&nbsp;
	- **限速酶** -> **==激素敏感性脂肪酶(HSL)==**&nbsp;&nbsp;&nbsp;
	- **脂解激素**（促进）-> **==肾上腺素、去甲肾上腺素、胰高血糖素==**、促肾上腺皮质激素&nbsp;&nbsp;&nbsp;
	- **抗脂解激素**（抑制）-> **==胰岛素==**、前列腺素E₂&nbsp;&nbsp;&nbsp;
	- 产物去向&nbsp;&nbsp;&nbsp;
		- **甘油** -> 血 -> 肝/肾/肠 -> 甘油激酶 -> 3-磷酸甘油 -> 磷酸二羟丙酮 -> **==糖酵解/糖异生==**&nbsp;&nbsp;&nbsp;
		- **脂肪酸** -> 血(与**==清蛋白==**结合) -> 骨骼肌/心肌 -> **==氧化供能==** / 肝 -> **==合成酮体==**&nbsp;&nbsp;&nbsp;
- **脂肪酸的活化（胞质溶胶）**&nbsp;&nbsp;&nbsp;
	- 脂肪酸 + CoA ->（脂酰CoA合成酶）-> **==脂酰CoA==**（活性形式，含高能硫酯键）&nbsp;&nbsp;&nbsp;
	- 消耗 -> ==**2个高能磷酸键**==（ATP -> AMP + PPi -> 2Pi）&nbsp;&nbsp;&nbsp;
- **脂酰CoA进入线粒体 — 关键限速步骤**&nbsp;&nbsp;&nbsp;
	- 短链脂酰CoA -> 直接进入线粒体&nbsp;&nbsp;&nbsp;
	- **长链脂酰CoA** -> **不能自由通过**线粒体内膜 -> 需**==肉碱(carnitine)==**转运&nbsp;&nbsp;&nbsp;
	- 限速酶 -> ==**肉碱脂酰转移酶Ⅰ**== -> 脂肪酸氧化分解的**限速酶**&nbsp;&nbsp;&nbsp;
	- 临床联系 -> **左旋肉碱**减肥产品（促进脂肪酸转运进入线粒体）&nbsp;&nbsp;&nbsp;
- **脂肪酸β-氧化（线粒体中）**&nbsp;&nbsp;&nbsp;
	- β-氧化 -> 脂肪酸体内氧化的**主要方式**&nbsp;&nbsp;&nbsp;
	- 组织定位 -> **==肝和肌肉==**(主要) -> 脑组织除外&nbsp;&nbsp;&nbsp;
	- 概念 -> 从脂酰基羧基端**β-碳原子**开始 -> 每次循环断裂产生 **1乙酰CoA + 1少2C的脂酰CoA**&nbsp;&nbsp;&nbsp;
	- **四步循环（必背）**&nbsp;&nbsp;&nbsp;
		- ① **脱氢** -> 脂酰CoA脱氢酶 -> 生成 **==FADH₂==**&nbsp;&nbsp;&nbsp;
		- ② **加水** -> Δ²-烯脂酰CoA水化酶&nbsp;&nbsp;&nbsp;
		- ③ **再脱氢** -> L(+)-β-羟脂酰CoA脱氢酶 -> 生成 **==NADH==**&nbsp;&nbsp;&nbsp;
		- ④ **硫解** -> β-酮脂酰CoA硫解酶 -> 产生**==乙酰CoA==** + 少2C脂酰CoA&nbsp;&nbsp;&nbsp;
- **能量计算（以16C软脂酸为例）**&nbsp;&nbsp;&nbsp;
	- 活化 -> 消耗 **==2 ATP==**&nbsp;&nbsp;&nbsp;
	- β-氧化 -> **==7轮循环==** -> 8乙酰CoA + **==7 FADH₂ + 7 NADH==**&nbsp;&nbsp;&nbsp;
	- TCA -> 8乙酰CoA × 10 = **==80 ATP==**&nbsp;&nbsp;&nbsp;
	- 呼吸链 -> 7FADH₂×1.5 + 7NADH×2.5 = 10.5 + 17.5 = **==28 ATP==**&nbsp;&nbsp;&nbsp;
	- 总生成 = 80 + 28 = **==108 ATP==** / 净生成 = 108 - 2 = ==**106 ATP**==&nbsp;&nbsp;&nbsp;
- **奇数碳脂肪酸氧化**&nbsp;&nbsp;&nbsp;
	- 偶数碳 -> 全部**==乙酰CoA==**&nbsp;&nbsp;&nbsp;
	- 奇数碳 -> 乙酰CoA + **==1丙酰CoA==** -> 丙酰CoA -> 甲基丙二酰CoA -> 琥珀酰CoA -> TCA&nbsp;&nbsp;&nbsp;
- **脂肪酸β-氧化 vs 合成 关键对比**&nbsp;&nbsp;&nbsp;
	- 场所 -> 氧化：==**线粒体**== / 合成：==**胞液**==&nbsp;&nbsp;&nbsp;
	- 酰基载体 -> 氧化：**==CoA-SH==** / 合成：**==ACP==**&nbsp;&nbsp;&nbsp;
	- 限速酶 -> 氧化：==**肉碱脂酰转移酶Ⅰ**== / 合成：==**乙酰CoA羧化酶**==&nbsp;&nbsp;&nbsp;
	- 供/受氢体 -> 氧化：**==FAD+NAD⁺(受)==** / 合成：**==NADPH(供)==**&nbsp;&nbsp;&nbsp;
	- 能量 -> 氧化：**==产能==** / 合成：**==耗能==**&nbsp;&nbsp;&nbsp;
