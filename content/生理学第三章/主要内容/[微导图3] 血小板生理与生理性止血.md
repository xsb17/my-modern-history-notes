# 血小板生理与生理性止血 — 微型速记导图

- 血小板基本特征
  - 来源：骨髓**巨核细胞**胞质小块（**无核**）
  - 寿命：**7~14天**（仅前2天有功能）
  - 正常值：**(100~300)×10⁹/L**
  - < **50×10⁹/L** -> 出血倾向 | > **1000×10⁹/L** -> 易血栓
- ★血小板五大生理特性
  - ① 黏附 ② 释放 ③ 聚集 ④ 收缩 ⑤ 吸附
- 血小板三大功能
  - 维持毛细血管内皮完整性
  - 参与生理性止血
  - 促进血液凝固
- ★生理性止血三个基本过程（必考）
  - 出血时间：< **9min**（模板法）
  - 第一期（一期止血）：
    - 血管损伤 -> 暴露**胶原** -> 血小板**黏附/聚集/释放**
    - 释放 **5-HT/TXA₂** -> **血管收缩**
    - -> 形成**松软的血小板血栓**（白色血栓/初期止血）
  - 第二期（二期止血）：
    - 凝血系统激活 -> **纤维蛋白网**形成 -> 加固止血栓（红色血栓）
  - 完整链：损伤->胶原暴露->血小板激活->5-HT/TXA₂->血管收缩+血小板血栓->凝血系统->纤维蛋白网
