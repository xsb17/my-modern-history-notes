# 红细胞生理与贫血 — 微型速记导图

- 红细胞正常值
  - 男：RBC **(4.0~5.5)×10¹²/L** | Hb **120~160g/L**
  - 女：RBC **(3.0~5.0)×10¹²/L** | Hb **110~150g/L**
  - 形态：**双凹圆盘状**/无核/胞浆充满 Hb
- ★三大生理特性（必考）
  - 可塑变形性：外力下**变形**通过窄毛细血管
  - 悬浮稳定性：
    - 血沉（ESR）：男 **<15mm/h** | 女 **<20mm/h**
    - ESR↑ = 悬浮稳定性**↓** = RBC **叠连**
    - ESR↑ 原因：**纤维蛋白原↑/球蛋白↑/胆固醇↑**（白蛋白↑则ESR↓）
    - 临床：**活动性结核/风湿热/肿瘤/炎症** -> ESR↑
  - 渗透脆性：
    - 低渗溶液中 RBC 膨胀**破裂**的特性
    - 渗透脆性大 -> 抵抗力**小** -> 易溶血
    - 衰老RBC、**遗传性球形红细胞增多症** -> 脆性↑
- 红细胞功能：运输 **O₂/CO₂** + 缓冲 + 参与免疫清除
- 生成原料与贫血类型（知识库补全）
  - **铁/蛋白质** 缺乏 -> RBC 数量或 Hb↓ -> **缺铁性贫血**（小细胞低色素）
  - **叶酸/VitB12** 缺乏 -> DNA 合成障碍 -> RBC 巨幼变 -> **巨幼红细胞性贫血**
- 贫血判定：单位容积中 **RBC/Hb/Hct** 低于正常
