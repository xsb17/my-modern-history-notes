# 血液凝固的核心途径 — 微型速记导图（大题重灾区）&nbsp;&nbsp;&nbsp;

- ★凝血定义与实质&nbsp;&nbsp;&nbsp;
  - 血液：流动液体 -> **凝胶状态**&nbsp;&nbsp;&nbsp;
  - 实质：可溶性**纤维蛋白原** -> 不溶性**纤维蛋白**&nbsp;&nbsp;&nbsp;
  - 血清 = 血浆 - 纤维蛋白原（凝血后析出液体）&nbsp;&nbsp;&nbsp;
- ★凝血三大基本步骤（核心）&nbsp;&nbsp;&nbsp;
  - ① 凝血酶原酶复合物形成（内源/外源两条途径汇聚于此）&nbsp;&nbsp;&nbsp;
  - ② 凝血酶原（**II**）-> **凝血酶**（IIa）&nbsp;&nbsp;&nbsp;
  - ③ 纤维蛋白原（**I**）-> **纤维蛋白**（Ia，可溶->不溶）&nbsp;&nbsp;&nbsp;
- ★凝血因子要点&nbsp;&nbsp;&nbsp;
  - FIV = **Ca²⁺**（其余均为蛋白质）&nbsp;&nbsp;&nbsp;
  - 以**酶原**形式存在（FII/FVII/FIX/FX/FXI/FXII）&nbsp;&nbsp;&nbsp;
  - 多数在**肝脏**合成&nbsp;&nbsp;&nbsp;
  - **FII/FVII/FIX/FX** 需 **维生素 K** 参与&nbsp;&nbsp;&nbsp;
  - FIII = **组织因子**（唯一不在血浆中的因子）&nbsp;&nbsp;&nbsp;
- ★内源性 vs 外源性凝血途径（必考）&nbsp;&nbsp;&nbsp;
  - 内源性：&nbsp;&nbsp;&nbsp;
    - 启动：**FXII** 接触异物/胶原激活&nbsp;&nbsp;&nbsp;
    - 因子分布：全部在**血液中**&nbsp;&nbsp;&nbsp;
    - 参与因子：XII/XI/IX/VIII/V/IV/II/XIII/I&nbsp;&nbsp;&nbsp;
    - 特点：步骤**多**/速度**慢**（约数分钟）&nbsp;&nbsp;&nbsp;
  - 外源性：&nbsp;&nbsp;&nbsp;
    - 启动：**FIII（组织因子）** 释放&nbsp;&nbsp;&nbsp;
    - 因子分布：**组织和血液中**&nbsp;&nbsp;&nbsp;
    - 参与因子：III/VII/V/IV/II/XIII/I&nbsp;&nbsp;&nbsp;
    - 特点：步骤**少**/速度**快**（约十几秒）&nbsp;&nbsp;&nbsp;
    - 在体内生理性凝血中起**关键作用**&nbsp;&nbsp;&nbsp;
- ★血友病（FVIII/FIX/FXI 缺乏）&nbsp;&nbsp;&nbsp;
  - 甲型：**FVIII** 缺乏 | 乙型：**FIX** 缺乏 | 丙型：**FXI** 缺乏&nbsp;&nbsp;&nbsp;
- 临床联系&nbsp;&nbsp;&nbsp;
  - 抽血后凝血 = 内源性途径（接触玻璃异物）为主&nbsp;&nbsp;&nbsp;
  - 组织创伤 = 外源性途径快速启动 + 内源性途径放大&nbsp;&nbsp;&nbsp;
