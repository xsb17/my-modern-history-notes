# 第三章：血液 — 全集存档

## 第一节 血液的组成及理化特性

### 一、血量和血液的组成

- 血量：全身血液总量，占体重 **7~8%**（**70~80ml/kg**）
- 循环血量：心血管系统中快速循环流动的部分
- 储存血量：滞留于肝、肺、腹腔静脉和皮下静脉丛内缓慢流动的部分
- 失血分级：
  - 轻度：< 500ml（< 总量 10%）→ 储存血量可补充
  - 中度：约 1000ml（总量 20%）→ 代偿功能不足
  - 重度：约 1500ml（总量 30%）→ 危及生命

#### 血液组成

- 血液 = 血浆（约 55%）+ 血细胞（约 45%）
- **血细胞比容**：血细胞在全血中所占容积百分比
  - 男性：**40~50%**，女性：**37~48%**，新生儿：**55%**

#### 血常规主要参考值

- RBC：男 **(4.0~5.5)×10¹²/L**，女 **(3.0~5.0)×10¹²/L**
- Hb：男 **120~160 g/L**，女 **110~150 g/L**
- WBC：**(4.0~10.0)×10⁹/L**
- PLT：**(100~300)×10⁹/L**

### 二、血液的理化特征

#### （四）血浆渗透压（掌握）

- 渗透压：溶液中溶质分子通过半透膜对水的吸引力（与溶质颗粒多少呈正相关）

##### 晶体渗透压

- 来源：由晶体物质（主要是 **NaCl**）形成 → 占血浆渗透压的 **99%**
- 意义：保持**细胞内外**水平衡 → 维持细胞的正常体积及形态
- 等渗溶液 = 细胞内液渗透压 = 血浆渗透压
- 等张溶液 = 红细胞保持正常形态的等渗溶液（溶质不能自由通过细胞膜）

##### 胶体渗透压

- 来源：由血浆蛋白质（主要是 **白蛋白**）形成 → 占血浆渗透压的 **1%**
- 意义：调节**血管内外**水平衡 → 维持正常血浆容量 → 降低导致**水肿**

#### （五）血浆酸碱度

- 正常 pH：**7.35~7.45**
- pH < 7.35 → **酸中毒**；pH > 7.45 → **碱中毒**
- 血浆缓冲对：NaHCO₃/H₂CO₃（最重要）、蛋白质钠盐/蛋白质、Na₂HPO₄/NaH₂PO₄

## 第二节 血细胞生理

### 一、红细胞生理

#### 红细胞数量和血红蛋白

- 男：RBC **(4.0~5.5)×10¹²/L**，Hb **120~160 g/L**
- 女：RBC **(3.0~5.0)×10¹²/L**，Hb **110~150 g/L**
- 新生儿：RBC > 6.0×10¹²/L，Hb > 200 g/L
- 贫血：RBC/Hb/Hct 低于正常；红细胞增多：高于正常
- 形态：双凹圆盘状，无核无细胞器，胞浆充满 Hb

#### （一）红细胞的生理特性

##### 1. 可塑变形性

- 正常红细胞在外力作用下**变形**的能力 → 通过直径更小的毛细血管

##### 2. 悬浮稳定性

- 红细胞能较稳定地悬浮于血浆中的特性
- 原因：双凹形态（摩擦大）+ 表面带负电荷（相互排斥）+ 血液流动
- 衡量指标：**红细胞沉降率（ESR/血沉）**
  - 男性 **< 15mm/h**，女性 **< 20mm/h**
- 沉降率越大 → 悬浮稳定性越差
- 影响因素：**红细胞叠连**
  - 不在于红细胞本身，而在于**血浆成分**
  - ESR↑（叠连加速）：**纤维蛋白原↑、球蛋白↑、胆固醇↑**
  - ESR↓：**白蛋白↑、卵磷脂↑**
- 临床意义：急性炎症、风湿热、结核、肿瘤 → ESR↑

##### 3. 渗透脆性

- 红细胞在低渗溶液中发生膨胀破裂的特性
- 渗透脆性大 → 抵抗力小，易溶血
- 渗透脆性小 → 抵抗力大，不易溶血
- 衰老红细胞、遗传性球形红细胞增多症 → 脆性增大

#### （二）红细胞的生理功能

- 运输 **O₂ 和 CO₂**（最主要）
- 缓冲作用
- 参与免疫复合物清除

---

### 二、血小板生理

#### （一）血小板形态和数量

- 来自骨髓巨核细胞的胞质小块（无核）
- 直径 2~3μm，双凹圆盘状
- 寿命 **7~14 天**，仅前 2 天有生理功能
- 正常值：**(100~300)×10⁹/L**
- < 50×10⁹/L → 出血倾向
- > 1000×10⁹/L → 易发生血栓

#### （二）血小板的生理特性

- ① 黏附 ② 释放 ③ 聚集 ④ 收缩 ⑤ 吸附

#### （三）血小板的生理功能

- 维持毛细血管内皮完整性
- 参与生理性止血
- 促进血液凝固

## 第三节 生理性止血

### 一、生理性止血的基本过程

- 生理性止血：正常人小血管破损后出血在几分钟内**自行停止**的现象
- 出血时间：< **9min**（模板法）
- 基本过程三步：
  - **一期止血**：血管收缩 + 血小板黏附/聚集/释放 → **血小板血栓**形成（松软）
  - **二期止血**：凝血系统激活 → **纤维蛋白网**形成 → 加固止血栓
  - 血管损伤 → 暴露胶原 → 血小板激活（黏附/聚集/释放）→ 释放 5-HT、TXA₂ → 血管收缩 + 血小板止血栓 → 凝血系统激活 → 纤维蛋白网形成

### 二、血液凝固（血凝）

- 定义：血液由流动液体状态变成不能流动的**凝胶状态**
- 实质：可溶性**纤维蛋白原** → 不溶性**纤维蛋白**
- 血清 = 血浆 - 纤维蛋白原（凝血后析出的液体）

#### （一）凝血因子

- 定义：血浆和组织中直接参与凝血过程的物质（罗马数字编号 FI~FXIII）
- FIV = **Ca²⁺**（其余为蛋白质）
- 丝氨酸蛋白酶（FII/FVII/FIX/FX/FXI/FXII/前激肽释放酶）：以**酶原**形式存在
- 除 FIII（组织因子）外，其余存在于新鲜血浆
- 多数在**肝脏**合成 → 其中 **FII/FVII/FIX/FX** 需**维生素 K** 参与

#### （二）凝血过程——三大步骤

- ① **凝血酶原酶复合物形成**
- ② 凝血酶原（II）→ **凝血酶**（IIa）
- ③ 纤维蛋白原（I）→ **纤维蛋白**（Ia）（可溶 → 不溶）

##### 两条途径对比

- **内源性凝血途径**：
  - 启动因子：**FXII**（接触异物/胶原表面激活）
  - 凝血因子：全部在**血液中**
  - 参与因子：XII/XI/IX/VIII/V/IV/II/XIII/I
  - 特点：步骤**多**、速度**慢**（约数分钟）
- **外源性凝血途径**：
  - 启动因子：**FIII（组织因子）**（受损组织释放）
  - 凝血因子：在**组织和血液中**
  - 参与因子：III/VII/V/IV/II/XIII/I
  - 特点：步骤**少**、速度**快**（约十几秒）
  - 在体内生理性凝血启动中起**关键作用**
- 血友病：FVIII 缺乏 → 甲型；FIX 缺乏 → 乙型；FXI 缺乏 → 丙型

#### （三）血液凝固的负性调控（抗凝系统）

- 细胞抗凝：血管内皮 + 单核-吞噬细胞系统
- 体液抗凝：
  - **抗凝血酶 III**（丝氨酸蛋白酶抑制物）→ 与肝素结合后活性↑
  - **蛋白质 C 系统**：肝脏合成，需维生素 K
  - **组织因子途径抑制物（TFPI）**：主要生理性抗凝物质
  - **肝素**：间接抗凝 → 增强抗凝血酶 III 作用

### 三、纤维蛋白溶解（纤溶）

- 定义：纤维蛋白被降解液化的过程
- 基本步骤：纤溶酶原 → **纤溶酶** → 降解纤维蛋白
- 意义：维持血液流体状态 + 溶解血栓使血流通畅

## 第四节 血型和输血原则

### 一、ABO 血型系统

- 分型依据：红细胞膜上的**凝集原（抗原）**
- A 型：A 凝集原，血清含**抗 B 凝集素**
- B 型：B 凝集原，血清含**抗 A 凝集素**
- AB 型：A+B 凝集原，血清**无**凝集素（万能受血者）
- O 型：无凝集原，血清含**抗 A + 抗 B** 凝集素（万能供血者）

### 二、交叉配血试验

- **主侧**：**供血者红细胞** + **受血者血清** → 绝对不能凝集
- **次侧**：受血者红细胞 + 供血者血清 → 允许轻微凝集

### 三、Rh 血型系统

- Rh 阳性：红细胞膜上有 **D 抗原**
- Rh 阴性：红细胞膜上无 D 抗原
- Rh 阴性母亲孕育 Rh 阳性胎儿 → 第二胎可能发生**新生儿溶血**
