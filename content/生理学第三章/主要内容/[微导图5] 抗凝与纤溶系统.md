# 抗凝与纤溶系统 — 微型速记导图

- ★血液凝固的负性调控——抗凝系统
  - 细胞抗凝：血管内皮 + **单核-吞噬细胞**系统
  - 体液抗凝：
    - **抗凝血酶 III**：丝氨酸蛋白酶抑制物 -> 与肝素结合活性↑ -> 灭活凝血酶/FXa 等
    - **蛋白质 C 系统**：肝脏合成，需**维生素 K** -> 灭活 FVa/FVIIIa
    - **组织因子途径抑制物（TFPI）**：**主要生理性抗凝物质** -> 抑制 FVIIa/TF 复合物
    - **肝素**：间接抗凝 -> 结合并**增强抗凝血酶 III** 活性 -> 临床常用抗凝药
- 抗凝系统作用机制
  - 抗凝血酶 III + 肝素 -> 灭活**凝血酶（FIIa）** 和 **FXa**
  - TFPI -> 抑制外源性途径启动
  - 蛋白质 C -> 灭活 FVa/FVIIIa（需维生素 K）
- ★纤溶系统（纤维蛋白溶解）
  - 定义：纤维蛋白被**降解液化**的过程
  - 基本步骤：
    - 纤溶酶原 ->（纤溶酶原激活物）-> **纤溶酶**
    - -> 降解纤维蛋白/纤维蛋白原
  - 意义：
    - 维持血液**流体状态**
    - 溶解**血栓**使血流通畅
- 凝血 vs 抗凝 vs 纤溶平衡
  - 凝血↑/抗凝↓ -> **血栓**
  - 凝血↓/纤溶↑ -> **出血**
  - 正常：三者动态平衡 -> 血液流体/破损时及时止血
- 临床药物
  - 肝素/华法林（VitK拮抗剂）-> 抗凝
  - 链激酶/尿激酶 -> 激活纤溶 -> 溶栓
