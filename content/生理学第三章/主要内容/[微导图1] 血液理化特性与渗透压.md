# 血液理化特性与渗透压 — 微型速记导图（必考）

- 血量
  - 占体重 **7~8%** -> 约 **70~80ml/kg**
  - 循环血量 + 储存血量
  - 失血 < **500ml**(10%) -> 代偿 | > **1000ml**(20%) -> 症状 | > **1500ml**(30%) -> 危及生命
- 血液组成
  - 血细胞比容：男 **40~50%** | 女 **37~48%**
  - 血浆 55% + 血细胞 45%
- ★血浆渗透压（必考对比）
  - 渗透压影响因素：溶质**颗粒数**（与种类/大小无关）
  - 晶体渗透压：
    - 来源：**NaCl** 等晶体物质 -> 占 **99%**
    - 意义：维持**细胞内外**水平衡 -> 保持 RBC 形态
    - 等渗 ≠ 等张（等张=等渗且溶质不能自由通过膜）
  - 胶体渗透压：
    - 来源：**白蛋白** 等血浆蛋白 -> 占 **1%**
    - 意义：维持**血管内外**水平衡 -> **胶渗压↓ = 水肿**
- 血浆 pH
  - 正常 **7.35~7.45**（弱碱性）
  - <7.35 -> **酸中毒** | >7.45 -> **碱中毒**
  - 最主要缓冲对：**NaHCO₃/H₂CO₃**
- 渗透压 vs 水肿
  - 胶渗压↓（肝病/肾病/营养不良->白蛋白↓）-> 血管内水外流 -> **水肿**
