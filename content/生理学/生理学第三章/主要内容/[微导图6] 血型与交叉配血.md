# 血型与交叉配血 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★ABO 血型系统&nbsp;&nbsp;&nbsp;
  - 分型依据：RBC 膜上的**凝集原**（抗原）&nbsp;&nbsp;&nbsp;
  - A 型：**A 凝集原** + 血清抗 B 凝集素&nbsp;&nbsp;&nbsp;
  - B 型：**B 凝集原** + 血清抗 A 凝集素&nbsp;&nbsp;&nbsp;
  - AB 型：A+B 凝集原 + 血清**无**凝集素 -> **万能受血者**&nbsp;&nbsp;&nbsp;
  - O 型：**无**凝集原 + 血清抗 A + 抗 B -> **万能供血者**&nbsp;&nbsp;&nbsp;
- ★交叉配血试验（必考）&nbsp;&nbsp;&nbsp;
  - 主侧：**供血者 RBC** + **受血者血清** -> **绝对不能凝集**&nbsp;&nbsp;&nbsp;
  - 次侧：受血者 RBC + 供血者血清 -> 允许轻微凝集&nbsp;&nbsp;&nbsp;
- 输血原则&nbsp;&nbsp;&nbsp;
  - 首选**同型**输血&nbsp;&nbsp;&nbsp;
  - 交叉配血主侧不凝集方可输血&nbsp;&nbsp;&nbsp;
  - O 型 -> 紧急时可输给任何血型（仅限 RBC，不含血浆）&nbsp;&nbsp;&nbsp;
- ★Rh 血型系统&nbsp;&nbsp;&nbsp;
  - Rh 阳性：RBC 膜有 **D 抗原**（汉族约 99%）&nbsp;&nbsp;&nbsp;
  - Rh 阴性：RBC 膜无 D 抗原（约 1%）&nbsp;&nbsp;&nbsp;
  - Rh 阴性母亲 + Rh 阳性胎儿：&nbsp;&nbsp;&nbsp;
    - 第一胎：分娩时 Rh 阳性 RBC 进入母体 -> 母体产生**抗 D 抗体**&nbsp;&nbsp;&nbsp;
    - 第二胎（又是 Rh 阳性）：抗 D 抗体 IgG 通过胎盘 -> 攻击胎儿 RBC -> **新生儿溶血病**&nbsp;&nbsp;&nbsp;
  - 预防：第一胎分娩后 72h 内给母亲注射 **抗 D 免疫球蛋白**&nbsp;&nbsp;&nbsp;
- Rh 血型临床特征&nbsp;&nbsp;&nbsp;
  - Rh 阴性者**无天然**抗 D 抗体（需免疫后产生，不同于 ABO 天然抗体）&nbsp;&nbsp;&nbsp;
  - Rh 阴性者第一次接受 Rh 阳性血 -> 产生抗 D 抗体 -> 再次输 Rh 阳性血 -> 溶血反应&nbsp;&nbsp;&nbsp;
