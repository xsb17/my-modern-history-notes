# 血液理化特性与渗透压 — 微型速记导图（必考）&nbsp;&nbsp;&nbsp;

- 血量&nbsp;&nbsp;&nbsp;
  - 占体重 **7~8%** -> 约 **70~80ml/kg**&nbsp;&nbsp;&nbsp;
  - 循环血量 + 储存血量&nbsp;&nbsp;&nbsp;
  - 失血 < **500ml**(10%) -> 代偿 | > **1000ml**(20%) -> 症状 | > **1500ml**(30%) -> 危及生命&nbsp;&nbsp;&nbsp;
- 血液组成&nbsp;&nbsp;&nbsp;
  - 血细胞比容：男 **40~50%** | 女 **37~48%**&nbsp;&nbsp;&nbsp;
  - 血浆 55% + 血细胞 45%&nbsp;&nbsp;&nbsp;
- ★血浆渗透压（必考对比）&nbsp;&nbsp;&nbsp;
  - 渗透压影响因素：溶质**颗粒数**（与种类/大小无关）&nbsp;&nbsp;&nbsp;
  - 晶体渗透压：&nbsp;&nbsp;&nbsp;
    - 来源：**NaCl** 等晶体物质 -> 占 **99%**&nbsp;&nbsp;&nbsp;
    - 意义：维持**细胞内外**水平衡 -> 保持 RBC 形态&nbsp;&nbsp;&nbsp;
    - 等渗 ≠ 等张（等张=等渗且溶质不能自由通过膜）&nbsp;&nbsp;&nbsp;
  - 胶体渗透压：&nbsp;&nbsp;&nbsp;
    - 来源：**白蛋白** 等血浆蛋白 -> 占 **1%**&nbsp;&nbsp;&nbsp;
    - 意义：维持**血管内外**水平衡 -> **胶渗压↓ = 水肿**&nbsp;&nbsp;&nbsp;
- 血浆 pH&nbsp;&nbsp;&nbsp;
  - 正常 **7.35~7.45**（弱碱性）&nbsp;&nbsp;&nbsp;
  - <7.35 -> **酸中毒** | >7.45 -> **碱中毒**&nbsp;&nbsp;&nbsp;
  - 最主要缓冲对：**NaHCO₃/H₂CO₃**&nbsp;&nbsp;&nbsp;
- 渗透压 vs 水肿&nbsp;&nbsp;&nbsp;
  - 胶渗压↓（肝病/肾病/营养不良->白蛋白↓）-> 血管内水外流 -> **水肿**&nbsp;&nbsp;&nbsp;
