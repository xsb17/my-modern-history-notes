# 脑干、基底核与小脑的运动调节 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 运动调控三级水平&nbsp;&nbsp;&nbsp;
  - 设计：皮层联络区 + **基底神经节** + **皮层小脑**&nbsp;&nbsp;&nbsp;
  - 实施/协调：皮层运动区 + **脊髓小脑**&nbsp;&nbsp;&nbsp;
  - 执行：**脑干 + 脊髓**&nbsp;&nbsp;&nbsp;
  - 最高级中枢：大脑皮层；最基本中枢：脊髓&nbsp;&nbsp;&nbsp;
- 脑干网状结构调节肌紧张&nbsp;&nbsp;&nbsp;
  - 易化区：范围**广、强** -> 加强伸肌肌紧张&nbsp;&nbsp;&nbsp;
  - 抑制区：范围**小、弱** -> 肌张力降低&nbsp;&nbsp;&nbsp;
  - 上级抑制：大脑皮层运动区、纹状体、小脑前叶蚓部&nbsp;&nbsp;&nbsp;
  - 上级易化：前庭核、小脑前叶两侧部、后叶中间部&nbsp;&nbsp;&nbsp;
- ★去大脑僵直（必考）&nbsp;&nbsp;&nbsp;
  - 定位：**中脑上、下丘之间**切断脑干&nbsp;&nbsp;&nbsp;
  - 现象：全身**伸肌肌紧张亢进** -> 四肢伸直/头尾昂起/脊柱挺硬 -> **角弓反张**&nbsp;&nbsp;&nbsp;
  - 机制：切除较多**抑制系统** -> **易化区 >> 抑制区** -> 牵张反射增强&nbsp;&nbsp;&nbsp;
  - 类型：α 僵直 + **γ 僵直**（经典僵直 -> 网状脊髓束→γ↑→肌梭敏感↑）&nbsp;&nbsp;&nbsp;
- ★基底神经节损伤两大疾病&nbsp;&nbsp;&nbsp;
  - 帕金森病（震颤麻痹）：&nbsp;&nbsp;&nbsp;
    - 病变：**黑质**受损 -> **多巴胺能神经元↓**&nbsp;&nbsp;&nbsp;
    - 症状：**肌紧张↑** + 随意运动**减少** + 静止性震颤 + 面具脸&nbsp;&nbsp;&nbsp;
  - 亨庭顿病（舞蹈病）：&nbsp;&nbsp;&nbsp;
    - 病变：**纹状体**受损 -> 胆碱能/GABA 能↓ -> 多巴胺相对**亢进**&nbsp;&nbsp;&nbsp;
    - 症状：**肌紧张↓** + 不自主**舞蹈样**动作&nbsp;&nbsp;&nbsp;
- 小脑三大分区与功能&nbsp;&nbsp;&nbsp;
  - 前庭小脑（绒球小结叶）：&nbsp;&nbsp;&nbsp;
    - 功能：**维持身体平衡**&nbsp;&nbsp;&nbsp;
    - 损伤：平衡障碍、步态蹒跚、**眼球震颤**&nbsp;&nbsp;&nbsp;
  - 脊髓小脑（前叶/后叶中间带）：&nbsp;&nbsp;&nbsp;
    - 功能：**调节肌紧张** + **协调随意运动**&nbsp;&nbsp;&nbsp;
    - 损伤：意向性震颤、指鼻不稳、醉酒步态 -> **小脑性共济失调**&nbsp;&nbsp;&nbsp;
  - 皮层小脑（后叶外侧部）：&nbsp;&nbsp;&nbsp;
    - 功能：运动**设计**和**编制程序**&nbsp;&nbsp;&nbsp;
- 大脑皮层运动区（中央前回 4 区）&nbsp;&nbsp;&nbsp;
  - 规律：**左右交叉**（上面部双侧）+ **上下倒置** + 面积 ∝ 运动**精细程度**&nbsp;&nbsp;&nbsp;
  - 传导束：皮层脊髓侧束（**80%** 交叉/精细）+ 前束（**20%** 不交叉/粗略）&nbsp;&nbsp;&nbsp;
