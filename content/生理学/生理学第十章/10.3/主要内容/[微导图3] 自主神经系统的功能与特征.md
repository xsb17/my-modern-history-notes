# 自主神经系统的功能与特征 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 自主神经结构特征对比 ★必考&nbsp;&nbsp;&nbsp;
  - 交感神经：&nbsp;&nbsp;&nbsp;
    - 起源：脊髓 **T₁~L₃** 灰质侧角&nbsp;&nbsp;&nbsp;
    - 节前**短** / 节后**长**&nbsp;&nbsp;&nbsp;
    - 节前:节后 = **1:多个**（弥散效应）&nbsp;&nbsp;&nbsp;
    - 分布：**广泛** -> 几乎支配所有器官&nbsp;&nbsp;&nbsp;
  - 副交感神经：&nbsp;&nbsp;&nbsp;
    - 起源：脑干副交感核 + 脊髓 **S₂₋₄** 灰质侧角&nbsp;&nbsp;&nbsp;
    - 节前**长** / 节后**短**&nbsp;&nbsp;&nbsp;
    - 节前:节后 = **1:1~2**（局限效应）&nbsp;&nbsp;&nbsp;
    - 分布：较**局限**&nbsp;&nbsp;&nbsp;
- 自主神经主要功能对比&nbsp;&nbsp;&nbsp;
  - 循环：交感→心脏**兴奋**/血管**收缩** | 副交感→心脏**抑制**/部分血管舒张&nbsp;&nbsp;&nbsp;
  - 呼吸：交感→支气管**舒张** | 副交感→支气管**收缩**&nbsp;&nbsp;&nbsp;
  - 消化：交感→胃肠运动**减弱** | 副交感→胃肠运动**增强**&nbsp;&nbsp;&nbsp;
  - 泌尿：交感→**抑制**排尿 | 副交感→**促进**排尿&nbsp;&nbsp;&nbsp;
  - 眼：交感→瞳孔**扩大** | 副交感→瞳孔**缩小**&nbsp;&nbsp;&nbsp;
  - 皮肤：交感→竖毛肌收缩/**汗腺分泌**（仅交感支配）&nbsp;&nbsp;&nbsp;
  - 内分泌：交感→促进**肾上腺髓质**分泌 | 副交感→促进**胰岛素**分泌&nbsp;&nbsp;&nbsp;
- ★自主神经 4 大功能特征&nbsp;&nbsp;&nbsp;
  - ① **紧张性作用**：持续发放低频冲动 -> 维持效应器基础活动&nbsp;&nbsp;&nbsp;
  - ② **双重支配、相互拮抗**：&nbsp;&nbsp;&nbsp;
    - 大多数器官受双重支配&nbsp;&nbsp;&nbsp;
    - 汗腺/竖毛肌等仅受**交感**支配&nbsp;&nbsp;&nbsp;
    - 唾液腺：两者**协同**（交感→粘稠少量；副交感→稀薄大量）&nbsp;&nbsp;&nbsp;
  - ③ **受效应器功能状态影响**：&nbsp;&nbsp;&nbsp;
    - 有孕子宫：α₁ 受体 -> **收缩**&nbsp;&nbsp;&nbsp;
    - 未孕子宫：β₂ 受体 -> **舒张**&nbsp;&nbsp;&nbsp;
  - ④ **整体调节意义**：&nbsp;&nbsp;&nbsp;
    - 交感-肾上腺髓质系统 -> **应急/应激** -> 动员储备力 -> 适应环境剧变&nbsp;&nbsp;&nbsp;
    - 副交感-胰岛素系统 -> **休整/恢复** -> 促进消化吸收/积蓄能量&nbsp;&nbsp;&nbsp;
- 各级中枢对内脏的调节&nbsp;&nbsp;&nbsp;
  - 脊髓：初级排尿/排便中枢&nbsp;&nbsp;&nbsp;
  - 延髓（**生命中枢**）：心血管基本中枢 + 呼吸基本中枢&nbsp;&nbsp;&nbsp;
