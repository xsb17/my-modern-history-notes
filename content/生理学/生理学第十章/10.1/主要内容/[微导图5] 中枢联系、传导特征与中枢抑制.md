# 中枢联系、传导特征与中枢抑制 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 中枢神经元 5 种联系方式&nbsp;&nbsp;&nbsp;
  - 单线式：**感觉传入** -> 点对点精确传导&nbsp;&nbsp;&nbsp;
  - 辐散式：一个神经元 -> 多个神经元 -> 扩大影响范围&nbsp;&nbsp;&nbsp;
  - 聚合式：多个神经元 -> 一个神经元 -> **整合**信息&nbsp;&nbsp;&nbsp;
  - 链锁式：逐级传递 -> 空间/时间上扩大作用&nbsp;&nbsp;&nbsp;
  - 环路式：形成回路 -> 后发放的基础（**运动传出**）&nbsp;&nbsp;&nbsp;
- ★中枢兴奋传递 7 大特征（必考简答）&nbsp;&nbsp;&nbsp;
  - 单向传递：突触前膜 -> 突触后膜（不可逆）&nbsp;&nbsp;&nbsp;
  - 突触延搁：一次传递需 **0.3~0.5ms**（递质释放+扩散+结合受体）&nbsp;&nbsp;&nbsp;
  - 总和效应：EPSP/IPSP 是**局部电位** -> 时间总和 + 空间总和&nbsp;&nbsp;&nbsp;
  - 兴奋节律可变性：传出 ≠ 传入冲动频率&nbsp;&nbsp;&nbsp;
  - 后发放：刺激停止后传出仍发放 -> 环式联系&nbsp;&nbsp;&nbsp;
  - 敏感性高：缺 **O₂** / **CO₂** 潴留 / 酸碱 / 药物 -> 最先受影响&nbsp;&nbsp;&nbsp;
  - 易疲劳性：递质**耗竭** -> 突触传递疲劳（区别于神经纤维的不疲劳）&nbsp;&nbsp;&nbsp;
- 中枢抑制分类&nbsp;&nbsp;&nbsp;
  - 突触后抑制：抑制性中间神经元 -> 释放抑制性递质 -> IPSP&nbsp;&nbsp;&nbsp;
  - 突触前抑制：突触前膜兴奋性递质释放**减少** -> EPSP 降低&nbsp;&nbsp;&nbsp;
- ★传入侧支性抑制（交互抑制）&nbsp;&nbsp;&nbsp;
  - 机制：传入纤维 ->&nbsp;&nbsp;&nbsp;
    - ① 兴奋一个中枢神经元&nbsp;&nbsp;&nbsp;
    - ② 经**侧支**兴奋**抑制性中间神经元** -> 抑制另一个中枢神经元&nbsp;&nbsp;&nbsp;
  - 意义：使**拮抗中枢**活动协调 -> 屈肌收缩时伸肌舒张&nbsp;&nbsp;&nbsp;
  - 实例：屈肌反射、伸肌反射&nbsp;&nbsp;&nbsp;
- ★回返性抑制&nbsp;&nbsp;&nbsp;
  - 机制：神经元兴奋沿轴突外传 ->&nbsp;&nbsp;&nbsp;
    - 经**轴突侧支**兴奋**闰绍细胞**（抑制性中间神经元）&nbsp;&nbsp;&nbsp;
    - -> 闰绍细胞释放**甘氨酸**&nbsp;&nbsp;&nbsp;
    - -> 反过来**抑制**原先兴奋的神经元&nbsp;&nbsp;&nbsp;
  - 意义：&nbsp;&nbsp;&nbsp;
    - **及时终止**运动神经元活动（防止过度兴奋）&nbsp;&nbsp;&nbsp;
    - 促使神经元间活动**同步化**&nbsp;&nbsp;&nbsp;
- 突触后抑制 vs 回返性抑制对比&nbsp;&nbsp;&nbsp;
  - 传入侧支性：传入兴奋 -> 侧支 -> 抑中间N -> 抑另一中枢 -> **协调拮抗**&nbsp;&nbsp;&nbsp;
  - 回返性：传出兴奋 -> 侧支 -> 闰绍细胞(**甘氨酸**) -> 抑原N -> **终止/同步**&nbsp;&nbsp;&nbsp;
- 中枢 vs 神经纤维传导对比&nbsp;&nbsp;&nbsp;
  - 中枢：单向/延搁/总和/可变/后发放/敏感/易疲劳&nbsp;&nbsp;&nbsp;
  - 纤维：双向/不延搁/无总和/保真/无后发放/不敏感/不疲劳&nbsp;&nbsp;&nbsp;
