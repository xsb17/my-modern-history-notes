# ACh 神经递质与受体系统 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 神经递质定义&nbsp;&nbsp;&nbsp;
  - 突触前神经元**合成** + 末梢**释放** -> 特异性作用于**受体** -> 产生效应的信息传递物质&nbsp;&nbsp;&nbsp;
  - 调质：调节递质效应的化学物质&nbsp;&nbsp;&nbsp;
  - 递质共存：一个神经元可含**两种以上**递质 -> 协调生理活动&nbsp;&nbsp;&nbsp;
- 外周胆碱能纤维分布（释放 **ACh**）★必考&nbsp;&nbsp;&nbsp;
  - 躯体运动神经纤维&nbsp;&nbsp;&nbsp;
  - **交感和副交感**神经的节前纤维&nbsp;&nbsp;&nbsp;
  - 多数**副交感**神经节后纤维&nbsp;&nbsp;&nbsp;
  - 少数交感神经节后纤维：**小汗腺** + 骨骼肌交感舒血管神经&nbsp;&nbsp;&nbsp;
- 胆碱能受体分类&nbsp;&nbsp;&nbsp;
  - **M 受体**（毒蕈碱受体，M1~M5）：G 蛋白耦联受体&nbsp;&nbsp;&nbsp;
  - **N 受体**（烟碱受体）：离子通道型受体&nbsp;&nbsp;&nbsp;
    - **N₁**：自主神经节后神经元&nbsp;&nbsp;&nbsp;
    - **N₂**：骨骼肌运动终板膜&nbsp;&nbsp;&nbsp;
- ★M 受体分布 + M 样作用 + 阻断剂&nbsp;&nbsp;&nbsp;
  - 心脏 -> 心率**↓**、传导**↓**、收缩力**↓**&nbsp;&nbsp;&nbsp;
  - 血管（间接）-> **舒张**&nbsp;&nbsp;&nbsp;
  - 支气管/胃肠平滑肌 -> **收缩**、腺体分泌**↑**&nbsp;&nbsp;&nbsp;
  - 膀胱逼尿肌 -> **收缩**、尿道内括约肌**舒张**&nbsp;&nbsp;&nbsp;
  - 汗腺 -> 分泌**↑**&nbsp;&nbsp;&nbsp;
  - 瞳孔括约肌 -> **收缩**（瞳孔**缩小**）&nbsp;&nbsp;&nbsp;
  - 睫状肌 -> **收缩**（视近物）&nbsp;&nbsp;&nbsp;
  - 阻断剂：**阿托品**&nbsp;&nbsp;&nbsp;
- ★N 受体分布 + N 样作用 + 阻断剂&nbsp;&nbsp;&nbsp;
  - N₁（自主神经节）-> 节后神经元**兴奋**&nbsp;&nbsp;&nbsp;
  - N₂（骨骼肌终板）-> 终板电位 -> 肌肉**收缩**&nbsp;&nbsp;&nbsp;
  - 阻断剂：**筒箭毒碱**（N₂）、六烃季胺（N₁）、十烃季胺（N₂）&nbsp;&nbsp;&nbsp;
- 中枢胆碱能系统&nbsp;&nbsp;&nbsp;
  - 分布：脊髓前角运动神经元、**脑干网状结构上行激活系统**等&nbsp;&nbsp;&nbsp;
  - 主要起**兴奋性递质**作用&nbsp;&nbsp;&nbsp;
  - 功能：学习和记忆、觉醒与睡眠、感觉与运动、内脏活动调节&nbsp;&nbsp;&nbsp;
- 递质共存实例&nbsp;&nbsp;&nbsp;
  - 副交感：ACh（分泌↑）+ **VIP**（血管舒张）-> 大量**稀薄**唾液&nbsp;&nbsp;&nbsp;
  - 交感：**NE**（分泌↑）+ **NPY**（血管收缩）-> 少量**粘稠**唾液&nbsp;&nbsp;&nbsp;
- ACh 系统核心考点口诀&nbsp;&nbsp;&nbsp;
  - 胆碱能纤维 = 躯体运动 + 节前全 + 副节后多 + 交感汗/舒血管&nbsp;&nbsp;&nbsp;
  - M 样 = 心脏抑/平滑肌缩/腺体泌/瞳孔缩 -> 阿托品阻断&nbsp;&nbsp;&nbsp;
  - N 样 = 节兴奋/肌收缩 -> 筒箭毒碱阻断&nbsp;&nbsp;&nbsp;
