# 两大感觉投射系统对比 — 微型速记导图（必考）&nbsp;&nbsp;&nbsp;

- ★特异性投射系统&nbsp;&nbsp;&nbsp;
  - 通路：感受器 -> **三级神经元**传送 -> 丘脑**特异感觉接替核** -> 皮层**特定感觉区域**&nbsp;&nbsp;&nbsp;
  - 特点：&nbsp;&nbsp;&nbsp;
    - 传入冲动有**特异性**&nbsp;&nbsp;&nbsp;
    - **点对点**精确投射 -> 定位精确&nbsp;&nbsp;&nbsp;
    - 三级神经元接替（少突触）&nbsp;&nbsp;&nbsp;
  - 功能：产生**特定感觉** + 激发大脑皮层发出神经冲动&nbsp;&nbsp;&nbsp;
  - 实例：视/听/触-压/本体感觉&nbsp;&nbsp;&nbsp;
- ★非特异性投射系统&nbsp;&nbsp;&nbsp;
  - 通路：各种感觉传入 -> **脑干网状结构**（多突触换元）-> 丘脑**髓板内核群** -> 皮层**广泛区域弥散性投射**&nbsp;&nbsp;&nbsp;
  - 特点：&nbsp;&nbsp;&nbsp;
    - 传入冲动**无特异性**&nbsp;&nbsp;&nbsp;
    - **弥散性**投射 -> 无点对点关系&nbsp;&nbsp;&nbsp;
    - **多级神经元、多突触**传送&nbsp;&nbsp;&nbsp;
  - 功能：**维持大脑皮层的兴奋状态**（维持**觉醒**）&nbsp;&nbsp;&nbsp;
  - 脑干网状结构**上行激活系统** -> 维持觉醒的核心通路&nbsp;&nbsp;&nbsp;
    - 损伤 -> **昏睡不醒**&nbsp;&nbsp;&nbsp;
    - 易受**巴比妥类安眠药**抑制 -> 催眠&nbsp;&nbsp;&nbsp;
  - 生理地位：是产生特定感觉的**基础**（无觉醒则无感觉）&nbsp;&nbsp;&nbsp;
- ★两大系统全面对比（必考简答/论述）&nbsp;&nbsp;&nbsp;
  - 传入冲动：特异 = **有特异性** | 非特异 = **无特异性**&nbsp;&nbsp;&nbsp;
  - 换元次数：特异 = **三级**（少） | 非特异 = **多级/多突触**（多）&nbsp;&nbsp;&nbsp;
  - 投射部位：特异 = 皮层**特定**区 | 非特异 = 皮层**广泛**区&nbsp;&nbsp;&nbsp;
  - 投射关系：特异 = **点对点**精确 | 非特异 = **弥散性**粗糙&nbsp;&nbsp;&nbsp;
  - 生理作用：特异 = 产生**特定感觉** | 非特异 = **维持觉醒**&nbsp;&nbsp;&nbsp;
  - 相互关系：非特异是特异的**基础**（无觉醒则无法产生特定感觉）&nbsp;&nbsp;&nbsp;
  - 药物敏感：特异 = 不易受抑制 | 非特异 = **易受巴比妥类**抑制&nbsp;&nbsp;&nbsp;
- 临床实例联系&nbsp;&nbsp;&nbsp;
  - 巴比妥类安眠药 -> 抑制上行激活系统 -> 非特异系统↓ -> 催眠&nbsp;&nbsp;&nbsp;
  - 脑干网状结构损伤 -> 上行激活系统阻断 -> 昏迷/昏睡&nbsp;&nbsp;&nbsp;
  - 全麻 -> 抑制非特异性投射系统 -> 意识消失&nbsp;&nbsp;&nbsp;
