# 躯体痛与内脏痛觉 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 痛觉定义&nbsp;&nbsp;&nbsp;
  - 伤害性刺激 -> 不愉快感觉 + **情绪变化** + **防御反应** + 自主神经反应&nbsp;&nbsp;&nbsp;
  - = 机体受伤害的**警报**&nbsp;&nbsp;&nbsp;
- 躯体痛分类&nbsp;&nbsp;&nbsp;
  - 体表痛（皮肤）&nbsp;&nbsp;&nbsp;
    - **快痛**：立即产生 -> 尖锐刺痛 -> 迅速短暂 + **定位准确** -> 伴逃避反射&nbsp;&nbsp;&nbsp;
    - **慢痛**：较慢出现 -> 烧灼痛 -> 缓慢持久 + **定位不准** -> 伴自主神经 + 情绪反应&nbsp;&nbsp;&nbsp;
  - 深部痛（关节/肌肉）：一般为**慢痛**&nbsp;&nbsp;&nbsp;
- 快痛 vs 慢痛对比&nbsp;&nbsp;&nbsp;
  - 快痛：立即/尖锐/短暂/定位**准**/逃避反射&nbsp;&nbsp;&nbsp;
  - 慢痛：延迟/烧灼/持久/定位**不准**/自主神经+情绪&nbsp;&nbsp;&nbsp;
- ★内脏痛四大特征（必考）&nbsp;&nbsp;&nbsp;
  - ① **慢痛**为主&nbsp;&nbsp;&nbsp;
  - ② **定位不准确**&nbsp;&nbsp;&nbsp;
  - ③ 发生**缓慢** + 持续时间**较长**&nbsp;&nbsp;&nbsp;
  - ④ 对**牵拉/炎症/缺血**敏感 -> 对**切割/烧灼**不敏感&nbsp;&nbsp;&nbsp;
  - ⑤ 伴有自主神经反应 + 情绪变化&nbsp;&nbsp;&nbsp;
  - ⑥ 常伴**牵涉痛**&nbsp;&nbsp;&nbsp;
- ★牵涉痛（referred pain）&nbsp;&nbsp;&nbsp;
  - 定义：内脏疾患 -> 远隔**特定体表部位**发生疼痛或痛觉过敏&nbsp;&nbsp;&nbsp;
  - 机制：会聚学说（内脏+体表传入 -> 同一脊髓后角神经元）+ 易化学说&nbsp;&nbsp;&nbsp;
- ★牵涉痛常见对应表（必考选择）&nbsp;&nbsp;&nbsp;
  - **心**（心绞痛）-> **心前区 + 左上臂尺侧**&nbsp;&nbsp;&nbsp;
  - **胃、胰** -> **左上腹 + 肩胛间**&nbsp;&nbsp;&nbsp;
  - **肝、胆囊** -> **右肩胛**&nbsp;&nbsp;&nbsp;
  - **肾结石** -> **腹股沟区**&nbsp;&nbsp;&nbsp;
  - **阑尾炎** -> **上腹部或脐周**&nbsp;&nbsp;&nbsp;
- 牵涉痛诊断口诀&nbsp;&nbsp;&nbsp;
  - 心左臂、胆右肩、肾结石腹股沟、阑尾上腹脐周圈&nbsp;&nbsp;&nbsp;
