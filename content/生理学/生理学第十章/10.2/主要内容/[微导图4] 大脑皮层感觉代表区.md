# 大脑皮层感觉代表区 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 第一体表感觉区（**中央后回**）★主要区域&nbsp;&nbsp;&nbsp;
  - 三大投射规律：&nbsp;&nbsp;&nbsp;
    - ① **左右交叉**：单侧感觉 -> 对侧皮层（**头面部为双侧性**例外）&nbsp;&nbsp;&nbsp;
    - ② **上下倒置**：下肢在上方、头面部在下方&nbsp;&nbsp;&nbsp;
    - ③ 投射面积 **∝ 感觉灵敏度**：灵敏度越高 -> 代表区越大（手指/口唇面积大）&nbsp;&nbsp;&nbsp;
  - 功能：产生**精细触-压觉**和**本体感觉**的精确感知&nbsp;&nbsp;&nbsp;
  - 定位：**精确** -> 点对点投射&nbsp;&nbsp;&nbsp;
- 第二体表感觉区（中央前回与岛叶之间）&nbsp;&nbsp;&nbsp;
  - 特点：&nbsp;&nbsp;&nbsp;
    - **双侧性**投射（不同于第一区的交叉）&nbsp;&nbsp;&nbsp;
    - **正立**分布（不同于第一区的倒置）&nbsp;&nbsp;&nbsp;
    - **定位较差**&nbsp;&nbsp;&nbsp;
  - 功能：对感觉作**粗糙分析**&nbsp;&nbsp;&nbsp;
- 第一感觉区 vs 第二感觉区对比&nbsp;&nbsp;&nbsp;
  - 位置：第一区 = **中央后回** | 第二区 = 中央前回与**岛叶**之间&nbsp;&nbsp;&nbsp;
  - 交叉性：第一区 = **左右交叉**（头面双侧）| 第二区 = **双侧性**&nbsp;&nbsp;&nbsp;
  - 分布：第一区 = **倒置** | 第二区 = **正立**&nbsp;&nbsp;&nbsp;
  - 定位：第一区 = **精确** | 第二区 = **较差**&nbsp;&nbsp;&nbsp;
  - 功能：第一区 = **精细感觉** | 第二区 = **粗糙分析**&nbsp;&nbsp;&nbsp;
- 本体感觉投射区&nbsp;&nbsp;&nbsp;
  - 投射至：**中央前回**（运动区）&nbsp;&nbsp;&nbsp;
  - 意义：感觉与运动在中枢紧密联系&nbsp;&nbsp;&nbsp;
- 感觉皮层的可塑性&nbsp;&nbsp;&nbsp;
  - 截肢/神经损伤 -> 皮层代表区**重组** -> 邻近代表区扩张&nbsp;&nbsp;&nbsp;
  - 训练（如盲人读盲文）-> 手指代表区**扩大**&nbsp;&nbsp;&nbsp;
- 口诀记忆&nbsp;&nbsp;&nbsp;
  - 第一区中央后回：**交叉、倒置、灵敏度正比**&nbsp;&nbsp;&nbsp;
  - 第二区岛叶旁：**双侧、正立、粗糙分析**&nbsp;&nbsp;&nbsp;
