## （四）影响静息电位的因素

## 1.改变细胞外液中K<sup>+</sup>的浓度

细胞外液中K<sup>+</sup>浓度↑→细胞内、外液中K<sup>+</sup>浓度差↓→ K<sup>+</sup>平衡电位↓

## 2.细胞膜对K<sup>+</sup>和Na<sup>+</sup>的通透性改变

Na<sup>+</sup>通透性↑→ Na<sup>+</sup>内流↑→静息电位↓

K<sup>+</sup>通透性↑→ K<sup>+</sup>外流↑→静息电位↑

## 3.钠泵活动强度的改变

钠泵活动↓→ 细胞外液K<sup>+</sup>浓度↑→细胞内、外液中K<sup>+</sup>浓度差→静息电位↓

## AP的产生机制

## 动作电位去极相

条件：

①静息时膜内外 $\mathbb { N } \mathtt { a } ^ { + }$ 不均匀分布(外高内低)。

②有效刺激使膜对 $\mathsf { N a } ^ { + }$ 通透性突然增大$( N a ^ { + }$ 通道大量开放）。

机制：由 $\mathsf { N a } ^ { + }$ 内流形成 $\mathsf { N a } ^ { + }$ 的平衡电位有效刺激→部分 $\mathsf { N a } ^ { + }$ 通道开放→ $\mathsf { N a } ^ { + }$ 内流↑→膜去极化→刺激更多 $\mathsf { N a } ^ { \sf + }$ 通道开放→大量 $\mathsf { N a } ^ { + }$ 内流（正反馈）→膜内负电位消失，出现正电位

## AP的产生机制

## 动作电位复极相

条件：

① $\mathbb { N } \mathtt { a } ^ { + }$ 通道关闭。

② K<sup>+</sup>通透性升高。

机制： $\mathsf { N a } ^ { + }$ 通道失活、K<sup>+</sup>通透性升高→ $\mathsf { N a } ^ { + }$ 内流停止，K<sup>+</sup>外流↑→膜内电位向负值变化→恢复至静息电位
