## 兴奋-收缩耦联过程

① 动作电位沿肌膜和横管膜传播到肌细胞深处；

② 激活的L型钙通道再激活终池膜上的钙释放通道，使肌质网内的 $\mathsf { C a } ^ { 2 + }$ 释放入胞质；

③ 胞质内 $\mathsf { C a } ^ { 2 + }$ 浓度的升高促使肌钙蛋白与 $\mathsf { C a } ^ { 2 + }$ 结合并触发肌丝滑行；

④ 肌质网膜上的钙泵将胞质中的 $\mathsf { C a } ^ { 2 + }$ 回收入肌质网，导致肌肉舒张。
