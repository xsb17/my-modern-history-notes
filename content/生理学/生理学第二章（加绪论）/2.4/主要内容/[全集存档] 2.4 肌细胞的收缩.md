# 第二章第四节：肌细胞的收缩 — 全集存档&nbsp;&nbsp;&nbsp;

## 一、骨骼肌神经-肌肉接头处兴奋的传递&nbsp;&nbsp;&nbsp;

### （一）神经-肌肉接头结构&nbsp;&nbsp;&nbsp;

- 定义：运动神经末梢与其所支配的骨骼肌细胞之间进行信息传递的**特化结构**&nbsp;&nbsp;&nbsp;
- 接头前膜：含**线粒体**、含 **ACh** 的突触小泡、**钙通道**&nbsp;&nbsp;&nbsp;
- 接头间隙&nbsp;&nbsp;&nbsp;
- 接头后膜（终板膜）：**皱襞**、化学门控通道（**N 型 ACh 受体**）、**胆碱酯酶**&nbsp;&nbsp;&nbsp;

### （二）兴奋传递过程（四个阶段）&nbsp;&nbsp;&nbsp;

#### 第一阶段：ACh 的释放&nbsp;&nbsp;&nbsp;

- 运动神经末梢 AP → 接头前膜去极化 → 电压门控 **Ca²⁺ 通道开放** → **Ca²⁺ 内流** → 触发突触小泡以**胞吐**方式释放 ACh（量子式释放）&nbsp;&nbsp;&nbsp;

#### 第二阶段：终板电位（EPP）的形成&nbsp;&nbsp;&nbsp;

- ACh 经接头间隙扩散 → 与终板膜上 **N 型 ACh 受体**特异性结合 → 阳离子通道开放（以 **Na⁺ 内流为主**）→ 形成**终板电位**&nbsp;&nbsp;&nbsp;
- 本质：大量突触囊泡同步释放引起的微终板电位总和 → 属于**局部电位**&nbsp;&nbsp;&nbsp;

#### 第三阶段：肌细胞膜上 AP 的形成&nbsp;&nbsp;&nbsp;

- 终板电位 → 电紧张扩布至邻近普通肌膜 → Na⁺ 内流和去极化 → 达**阈电位**水平 → 触发**动作电位**&nbsp;&nbsp;&nbsp;

#### 第四阶段：ACh 的降解&nbsp;&nbsp;&nbsp;

- ACh 被终板膜上的**胆碱酯酶**水解为**胆碱和乙酸** → 终止传递&nbsp;&nbsp;&nbsp;

### （三）传递特征&nbsp;&nbsp;&nbsp;

- **单向传递**（电→化学→电）&nbsp;&nbsp;&nbsp;
- **时间延搁**（约 0.5~1.0ms）&nbsp;&nbsp;&nbsp;
- **易疲劳、易受药物和环境因素影响**&nbsp;&nbsp;&nbsp;

### （四）影响神经-肌接头传递的因素&nbsp;&nbsp;&nbsp;

#### 1. 影响 ACh 释放&nbsp;&nbsp;&nbsp;

- 细胞外 **低 Ca²⁺** / **高 Mg²⁺** → Ca²⁺ 内流↓ → ACh 释放↓&nbsp;&nbsp;&nbsp;
- **肉毒素**：阻止 ACh 释放 → 肌肉麻痹&nbsp;&nbsp;&nbsp;

#### 2. 影响 ACh 受体&nbsp;&nbsp;&nbsp;

- **筒箭毒、α-银环蛇毒**：阻断 N 型 ACh 受体 → 传递受阻 → 肌肉松弛/麻痹&nbsp;&nbsp;&nbsp;
- **重症肌无力**：自身抗体破坏 ACh 受体 → 传递障碍 → 肌无力&nbsp;&nbsp;&nbsp;

#### 3. 影响 ACh 清除&nbsp;&nbsp;&nbsp;

- **新斯的明**：胆碱酯酶抑制剂 → ACh 蓄积 → 增强传递&nbsp;&nbsp;&nbsp;
- **有机磷农药**：使胆碱酯酶失活 → ACh 蓄积 → 肌肉痉挛&nbsp;&nbsp;&nbsp;

## 二、横纹肌细胞的结构特征&nbsp;&nbsp;&nbsp;

### （一）肌原纤维和肌节&nbsp;&nbsp;&nbsp;

- 肌节：相邻两 **Z 线**之间的区域 → 肌肉收缩的基本单位&nbsp;&nbsp;&nbsp;
- 粗肌丝：**肌球蛋白**（含横桥，具有 ATP 酶活性）&nbsp;&nbsp;&nbsp;
- 细肌丝：**肌动蛋白**（含横桥结合位点）+ **原肌球蛋白**（覆盖结合位点）+ **肌钙蛋白**（与 Ca²⁺ 结合）&nbsp;&nbsp;&nbsp;

### （二）肌管系统&nbsp;&nbsp;&nbsp;

- **横管（T 管）**：走行与肌原纤维垂直 → 将 AP 传入细胞深处&nbsp;&nbsp;&nbsp;
- **纵管（肌质网/SR）**：纵管末端膨大形成**终池** → 终池内含高浓度 **Ca²⁺**&nbsp;&nbsp;&nbsp;
- **三联管**：骨骼肌 → 横管 + 两侧终池 → 兴奋-收缩耦联的**关键部位**&nbsp;&nbsp;&nbsp;
- **二联管**：心肌 → 横管 + 单侧终池&nbsp;&nbsp;&nbsp;

---&nbsp;&nbsp;&nbsp;

## 三、横纹肌的收缩机制——肌丝滑行理论&nbsp;&nbsp;&nbsp;

### 核心论点&nbsp;&nbsp;&nbsp;

- 肌肉收缩时肌丝本身**并未缩短**，而是**细肌丝向粗肌丝滑行** → Z 线互相靠近 → 肌小节变短 → 肌原纤维/肌细胞/整块肌肉缩短&nbsp;&nbsp;&nbsp;

### 肌丝滑行的过程&nbsp;&nbsp;&nbsp;

- 肌质中 **Ca²⁺ 升高** → Ca²⁺ 与**肌钙蛋白**结合&nbsp;&nbsp;&nbsp;
- → 原肌球蛋白双螺旋结构发生**扭转** → 肌动蛋白上横桥结合位点**暴露**&nbsp;&nbsp;&nbsp;
- → 横桥与肌动蛋白结合 → 横桥**扭动**（拉动细肌丝向 M 线方向滑动）→ 横桥**复位**&nbsp;&nbsp;&nbsp;
- → 横桥周期：结合 → 扭动（产生张力/缩短）→ 复位 → 再结合 → 再扭动&nbsp;&nbsp;&nbsp;
- 肌肉舒张：Ca²⁺ 被钙泵回收入肌质网 → 肌钙蛋白与 Ca²⁺ 解离 → 原肌球蛋白**重新覆盖**结合位点&nbsp;&nbsp;&nbsp;

### 力学表现&nbsp;&nbsp;&nbsp;

- 张力：众多活化横桥产生张力的**总和**&nbsp;&nbsp;&nbsp;
- 缩短速度：横桥周期越短，收缩速度**越快**&nbsp;&nbsp;&nbsp;

## 四、横纹肌的兴奋-收缩耦联（ECC）&nbsp;&nbsp;&nbsp;

### 定义&nbsp;&nbsp;&nbsp;

- 肌细胞以**动作电位**为特征的兴奋过程与以**肌丝滑行**为基础的机械收缩过程联系起来的**中介过程**&nbsp;&nbsp;&nbsp;

### 结构基础&nbsp;&nbsp;&nbsp;

- 骨骼肌：**三联管**&nbsp;&nbsp;&nbsp;
- 心肌：**二联管**&nbsp;&nbsp;&nbsp;
- 耦联关键因子：**Ca²⁺**&nbsp;&nbsp;&nbsp;

### 耦联过程（四个关键步骤）&nbsp;&nbsp;&nbsp;

- ① AP 沿肌膜和**横管膜**传播到肌细胞深处&nbsp;&nbsp;&nbsp;
- ② 激活的 **L 型钙通道**（横管膜上）→ 激活终池膜上的**钙释放通道**（RYR 受体）→ 肌质网内 **Ca²⁺** 释放入胞质&nbsp;&nbsp;&nbsp;
- ③ 胞质 Ca²⁺ 浓度↑ → Ca²⁺ 与**肌钙蛋白**结合 → 触发**肌丝滑行** → **收缩**&nbsp;&nbsp;&nbsp;
- ④ 肌质网膜上的**钙泵**将胞质 Ca²⁺ 回收入肌质网 → 胞质 Ca²⁺↓ → **舒张**&nbsp;&nbsp;&nbsp;

---&nbsp;&nbsp;&nbsp;

## 五、影响横纹肌收缩效能的因素&nbsp;&nbsp;&nbsp;

### （一）收缩形式&nbsp;&nbsp;&nbsp;

- **等长收缩**：肌肉**长度不变**，张力增加（如维持姿势）&nbsp;&nbsp;&nbsp;
- **等张收缩**：**张力不变**，肌肉缩短（如举起重物）&nbsp;&nbsp;&nbsp;

### （二）影响因素&nbsp;&nbsp;&nbsp;

#### 1. 前负荷（初长度）&nbsp;&nbsp;&nbsp;

- 肌肉收缩**前**已存在的负荷 → 决定**初长度**&nbsp;&nbsp;&nbsp;
- 最适初长度：粗、细肌丝重叠最佳 → 产生**最大张力**&nbsp;&nbsp;&nbsp;

#### 2. 后负荷&nbsp;&nbsp;&nbsp;

- 肌肉收缩**中**承受的负荷（阻力）&nbsp;&nbsp;&nbsp;
- 后负荷↑ → 缩短速度↓ → 缩短程度↓&nbsp;&nbsp;&nbsp;

#### 3. 肌肉收缩能力&nbsp;&nbsp;&nbsp;

- 与肌肉本身**功能状态**有关（与前/后负荷无关）&nbsp;&nbsp;&nbsp;
- 取决于：兴奋-收缩耦联中胞浆 **Ca²⁺ 水平**和肌球蛋白 **ATP 酶活性**&nbsp;&nbsp;&nbsp;

#### 4. 收缩总和&nbsp;&nbsp;&nbsp;

- 多纤维总和：参与收缩的运动单位数量↑ → 收缩力↑&nbsp;&nbsp;&nbsp;
- 频率总和（强直收缩）：刺激频率↑ → 收缩波融合 → 张力↑（完全强直收缩 > 不完全强直收缩 > 单收缩）&nbsp;&nbsp;&nbsp;
