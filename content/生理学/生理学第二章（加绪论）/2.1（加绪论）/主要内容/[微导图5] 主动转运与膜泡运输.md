# 主动转运与膜泡运输 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 主动转运共性&nbsp;&nbsp;&nbsp;
  - 方向：**低浓度 -> 高浓度**（逆浓度/电位梯度）&nbsp;&nbsp;&nbsp;
  - 耗能：**耗能**&nbsp;&nbsp;&nbsp;
  - 膜蛋白：**必需**&nbsp;&nbsp;&nbsp;
  - 饱和性：有&nbsp;&nbsp;&nbsp;
- ★原发性主动转运&nbsp;&nbsp;&nbsp;
  - 定义：**直接利用 ATP** 分解供能&nbsp;&nbsp;&nbsp;
  - ★钠-钾泵（**Na⁺-K⁺-ATP 酶**）— 核心考点&nbsp;&nbsp;&nbsp;
    - 本质：钠-钾依赖式 **ATP 酶**（离子泵）&nbsp;&nbsp;&nbsp;
    - 激活条件：膜内 **Na⁺↑** + 膜外 **K⁺↑**&nbsp;&nbsp;&nbsp;
    - 每次循环：分解 **1 个 ATP** -> 泵出 **3 个 Na⁺** + 泵入 **2 个 K⁺**&nbsp;&nbsp;&nbsp;
    - 净效应：**生电性**（净泵出 1 个正电荷 -> 膜外正电位↑）&nbsp;&nbsp;&nbsp;
    - 被**哇巴因**（ouabain）特异性抑制&nbsp;&nbsp;&nbsp;
  - 钠泵四大生理意义：&nbsp;&nbsp;&nbsp;
    - ① 建立 **Na⁺/K⁺ 不均衡分布** -> 神经肌肉**兴奋性**基础&nbsp;&nbsp;&nbsp;
    - ② 维持胞内**渗透压**和细胞**容积**正常（防止细胞水肿）&nbsp;&nbsp;&nbsp;
    - ③ 建立 **Na⁺ 浓度势能贮备** -> 供继发性主动转运利用&nbsp;&nbsp;&nbsp;
    - ④ **生电性** -> 影响静息膜电位&nbsp;&nbsp;&nbsp;
  - 其他泵：**钙泵**（Ca²⁺-ATP 酶）、**质子泵**（H⁺-K⁺-ATP 酶，胃壁细胞）&nbsp;&nbsp;&nbsp;
- ★继发性主动转运&nbsp;&nbsp;&nbsp;
  - 定义：能量来自其他离子（**Na⁺**）的浓度/电位梯度 -> **间接利用** ATP&nbsp;&nbsp;&nbsp;
  - 同向转运：**Na⁺-葡萄糖同向转运**（小肠上皮/肾近端小管吸收葡萄糖）&nbsp;&nbsp;&nbsp;
  - 逆向转运：**Na⁺-Ca²⁺ 交换**（心肌细胞）、**Na⁺-H⁺ 交换**（肾小管上皮）&nbsp;&nbsp;&nbsp;
- ★膜泡运输（大分子）&nbsp;&nbsp;&nbsp;
  - 特点：**耗能**、通过囊泡形成和融合&nbsp;&nbsp;&nbsp;
  - 入胞（内吞）：&nbsp;&nbsp;&nbsp;
    - **吞噬**：固态物质（如巨噬细胞吞噬细菌）&nbsp;&nbsp;&nbsp;
    - **吞饮**：液态物质&nbsp;&nbsp;&nbsp;
  - 出胞（胞吐）：&nbsp;&nbsp;&nbsp;
    - **激素分泌**、**神经递质释放**、消化液分泌&nbsp;&nbsp;&nbsp;
  - 例：突触前膜释放 ACh = 出胞&nbsp;&nbsp;&nbsp;
- 四种转运方式速查对比&nbsp;&nbsp;&nbsp;
  - 单纯扩散：脂溶性小分子/顺浓度/不耗能/不需蛋白&nbsp;&nbsp;&nbsp;
  - 易化扩散（通道）：离子/顺浓度/不耗能/需通道蛋白/离子选择性+门控&nbsp;&nbsp;&nbsp;
  - 易化扩散（载体）：葡萄糖AA/顺浓度/不耗能/需载体蛋白/**有饱和+竞争抑制**&nbsp;&nbsp;&nbsp;
  - 主动转运（原发）：Na⁺/K⁺/Ca²⁺/逆浓度/**耗能(ATP)**/需泵蛋白/**3Na⁺出2K⁺入**&nbsp;&nbsp;&nbsp;
  - 主动转运（继发）：葡萄糖AA/逆浓度/耗能(间接)/需转运体/耦合 Na⁺&nbsp;&nbsp;&nbsp;
  - 膜泡运输：大分子/需囊泡/耗能/入胞+出胞&nbsp;&nbsp;&nbsp;
