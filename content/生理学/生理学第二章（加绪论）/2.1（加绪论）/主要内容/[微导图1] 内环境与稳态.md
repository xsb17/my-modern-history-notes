# 内环境与稳态 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 体液分布（占体重 **60%**）&nbsp;&nbsp;&nbsp;
  - 细胞内液：**2/3**（40%体重）&nbsp;&nbsp;&nbsp;
  - 细胞外液：**1/3**（20%体重）-> 分为：&nbsp;&nbsp;&nbsp;
    - 组织液：**3/4**（15%体重）&nbsp;&nbsp;&nbsp;
    - 血浆：**1/4**（5%体重）&nbsp;&nbsp;&nbsp;
    - 淋巴液、脑脊液（少量）&nbsp;&nbsp;&nbsp;
- ★内环境 = **细胞外液**（组织液+血浆+淋巴液等）-> 细胞直接生存的环境&nbsp;&nbsp;&nbsp;
- ★稳态（Homeostasis）&nbsp;&nbsp;&nbsp;
  - 定义：内环境理化因素（体温/血压/pH/渗透压等）在调节下达到**动态平衡**的相对稳定状态&nbsp;&nbsp;&nbsp;
  - 关键：**不是恒定不变** -> 在一定范围内波动 -> 动态平衡&nbsp;&nbsp;&nbsp;
  - 例：血液生化指标有参考范围，表明内环境在波动中维持相对稳定&nbsp;&nbsp;&nbsp;
- 生物节律&nbsp;&nbsp;&nbsp;
  - 机体功能按时间顺序发生**周期性变化**（日/月/年）&nbsp;&nbsp;&nbsp;
  - 2017 年诺贝尔奖：**昼夜节律**分子机制&nbsp;&nbsp;&nbsp;
- 生命基本特征&nbsp;&nbsp;&nbsp;
  - 新陈代谢：物质+能量交换 -> **最基本**&nbsp;&nbsp;&nbsp;
  - 兴奋性：对刺激发生产生**兴奋或抑制**反应的能力&nbsp;&nbsp;&nbsp;
  - 适应性：**生理性适应**（高原红细胞↑）+ **行为性适应**（添衣）&nbsp;&nbsp;&nbsp;
  - 生殖：无性生殖 + 有性生殖&nbsp;&nbsp;&nbsp;
- 刺激与反应&nbsp;&nbsp;&nbsp;
  - 刺激：引起反应的**内外环境变化**&nbsp;&nbsp;&nbsp;
  - 反应：**兴奋**（静止→活跃）+ **抑制**（活跃→静止）&nbsp;&nbsp;&nbsp;
  - 可兴奋组织：**神经、肌肉、腺体**&nbsp;&nbsp;&nbsp;
