# 生理功能的调节方式 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★神经调节（先锋营）— 最主要&nbsp;&nbsp;&nbsp;
  - 基本方式：**反射** -> 反射弧（感受器→传入→中枢→传出→效应器）&nbsp;&nbsp;&nbsp;
  - 特点：**迅速、准确、局限、短暂**&nbsp;&nbsp;&nbsp;
  - 例：缩手反射、膝跳反射&nbsp;&nbsp;&nbsp;
- ★体液调节（主力军）&nbsp;&nbsp;&nbsp;
  - 方式：细胞分泌**化学物质（激素等）**-> 体液运输 -> 靶细胞受体&nbsp;&nbsp;&nbsp;
  - 分泌方式：远距分泌、旁分泌、自分泌、神经内分泌&nbsp;&nbsp;&nbsp;
  - 神经-体液调节：内分泌腺受神经支配 -> 体液调节成为反射弧传出部分（如胃液头期分泌）&nbsp;&nbsp;&nbsp;
  - 特点：**缓慢、弥散、持久**&nbsp;&nbsp;&nbsp;
  - 例：胰岛素降血糖&nbsp;&nbsp;&nbsp;
- ★自身调节（地方军）&nbsp;&nbsp;&nbsp;
  - 方式：细胞/组织**不依赖神经体液** -> 凭借本身内在特性对环境变化产生适应性反应&nbsp;&nbsp;&nbsp;
  - 特点：**强度弱、范围小、灵敏度低**&nbsp;&nbsp;&nbsp;
  - 例：肾血流量自身调节（70~180mmHg 范围内恒定）&nbsp;&nbsp;&nbsp;
- ★三种调节方式全面对比&nbsp;&nbsp;&nbsp;
  - 神经调节：反射弧 -> **快/准/局限/短**&nbsp;&nbsp;&nbsp;
  - 体液调节：化学物质 -> **慢/散/持久**&nbsp;&nbsp;&nbsp;
  - 自身调节：组织自身 -> **弱/小/低**&nbsp;&nbsp;&nbsp;
- 口诀&nbsp;&nbsp;&nbsp;
  - 神经 = 快准短&nbsp;&nbsp;&nbsp;
  - 体液 = 慢散久&nbsp;&nbsp;&nbsp;
  - 自身 = 弱小低&nbsp;&nbsp;&nbsp;
