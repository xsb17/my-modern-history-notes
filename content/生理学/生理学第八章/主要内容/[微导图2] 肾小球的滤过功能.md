# 肾小球的滤过功能 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 肾小球滤过定义：血液 -> 肾小球 -> 除**蛋白质**外成分滤入肾小囊 -> 形成**原尿**（血浆超滤液）&nbsp;&nbsp;&nbsp;
- GFR 与滤过分数（FF）&nbsp;&nbsp;&nbsp;
  - GFR = 每分钟双肾生成超滤液量 = **125 ml/min** = **180 L/d**&nbsp;&nbsp;&nbsp;
  - FF = GFR ÷ 肾血浆流量 = 125 ÷ 660 ≈ **19%**&nbsp;&nbsp;&nbsp;
  - 意义：流经肾脏的血浆约 **1/5** 滤出形成原尿&nbsp;&nbsp;&nbsp;
- 滤过膜三层结构（机械屏障）&nbsp;&nbsp;&nbsp;
  - 内层（内皮细胞）：窗孔 **70~90nm** -> 阻止**血细胞**&nbsp;&nbsp;&nbsp;
  - 中层（基膜）：网孔 **2~8nm** -> 阻止**蛋白质** -> **起主要作用**&nbsp;&nbsp;&nbsp;
  - 外层（上皮细胞）：裂隙膜 -> 起主要作用&nbsp;&nbsp;&nbsp;
  - 分子半径 < **2.0nm** -> 完全滤过（无机盐/葡萄糖）&nbsp;&nbsp;&nbsp;
  - 分子半径 > **4.2nm** -> 完全不滤过（蛋白质）&nbsp;&nbsp;&nbsp;
- 电学屏障&nbsp;&nbsp;&nbsp;
  - 滤过膜含**带负电荷**糖蛋白&nbsp;&nbsp;&nbsp;
  - 滤过能力：**负电荷 < 电中性 < 正电荷**&nbsp;&nbsp;&nbsp;
  - 白蛋白（半径 3.6nm + 带负电）-> **很难滤出**&nbsp;&nbsp;&nbsp;
- 有效滤过压公式 ★核心&nbsp;&nbsp;&nbsp;
  - 有效滤过压 = **肾小球 Cap 压** -（**囊内压** + **血浆胶渗压**）&nbsp;&nbsp;&nbsp;
  - 入球端：**45** -（**10** + **25**）= **10 mmHg**（有滤过）&nbsp;&nbsp;&nbsp;
  - 出球端：45 -（10 + **35**）= **0 mmHg**（**滤过平衡点**）&nbsp;&nbsp;&nbsp;
  - 滤过平衡点 = 有效滤过压最先降为 0 的点&nbsp;&nbsp;&nbsp;
  - 平衡点**以前**的 Cap 才有滤过功能&nbsp;&nbsp;&nbsp;
- 影响滤过的四大因素推演 ★必考分析题&nbsp;&nbsp;&nbsp;
  - 休克（血压 < 50mmHg）：&nbsp;&nbsp;&nbsp;
    - 肾小球 Cap 压**↓** -> 有效滤过压↓ -> GFR↓ -> **少尿或无尿**&nbsp;&nbsp;&nbsp;
  - 大量输入生理盐水：&nbsp;&nbsp;&nbsp;
    - 血浆蛋白稀释 -> **血浆胶渗压↓** -> 有效滤过压↑ -> GFR↑ -> **尿量↑**&nbsp;&nbsp;&nbsp;
  - 输尿管结石/肿瘤：&nbsp;&nbsp;&nbsp;
    - 尿液排出受阻 -> **囊内压↑** -> 有效滤过压↓ -> GFR↓ -> **尿量↓**&nbsp;&nbsp;&nbsp;
  - 肾血浆流量变化：&nbsp;&nbsp;&nbsp;
    - 流量↑ -> 胶渗压递增速率↓ -> 滤过平衡点**后移** -> 有效滤过长度↑ -> GFR↑&nbsp;&nbsp;&nbsp;
    - 流量↓（休克）-> 胶渗压递增速率↑ -> 平衡点**前移** -> GFR↓&nbsp;&nbsp;&nbsp;
- 滤过系数 Kf = **K × S**&nbsp;&nbsp;&nbsp;
  - K = 有效通透系数 -> 影响**尿液成分**&nbsp;&nbsp;&nbsp;
    - K↑（肾小球肾炎）-> 血尿、蛋白尿&nbsp;&nbsp;&nbsp;
  - S = 滤过膜面积 **1.5m²** -> 影响**尿量**&nbsp;&nbsp;&nbsp;
    - S↓（肾炎 Cap 阻塞）-> 有效滤过面积↓ -> 少尿、无尿&nbsp;&nbsp;&nbsp;
