# 肾脏基础与血流调节 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 尿量正常值与异常界定&nbsp;&nbsp;&nbsp;
  - 正常：**1000~2000 ml/d**&nbsp;&nbsp;&nbsp;
  - 多尿：> **2500 ml/d**&nbsp;&nbsp;&nbsp;
  - 少尿：**100~400 ml/d**&nbsp;&nbsp;&nbsp;
  - 无尿：< **100 ml/d**&nbsp;&nbsp;&nbsp;
- 尿液理化性质：淡黄透明 -> pH **5.0~7.0**（偏酸）&nbsp;&nbsp;&nbsp;
- 肾脏三大功能&nbsp;&nbsp;&nbsp;
  - 排泄：主要排泄器官 -> 排出代谢产物/异物&nbsp;&nbsp;&nbsp;
  - 调节：维持**水、电解质、酸碱**平衡&nbsp;&nbsp;&nbsp;
  - 内分泌：&nbsp;&nbsp;&nbsp;
    - 球旁细胞 -> 分泌**肾素** -> 调节血压&nbsp;&nbsp;&nbsp;
    - 分泌**EPO**（促红细胞生成素）-> 促进红细胞生成&nbsp;&nbsp;&nbsp;
    - 肾脏 **1α-羟化酶** -> 活性 **Vit D₃** -> 调节钙磷代谢&nbsp;&nbsp;&nbsp;
- 肾单位组成&nbsp;&nbsp;&nbsp;
  - 肾小体：肾小球 + 肾小囊&nbsp;&nbsp;&nbsp;
  - 肾小管：近端小管 -> 髓袢（降粗→降细→升细→升粗）-> 远端小管&nbsp;&nbsp;&nbsp;
- 皮质 vs 近髓肾单位对比&nbsp;&nbsp;&nbsp;
  - 皮质肾单位：占 **85~90%** -> 髓袢短 -> 入:出 ≈ **2:1** -> 有球旁器 -> **尿液生成**&nbsp;&nbsp;&nbsp;
  - 近髓肾单位：占 **10~15%** -> 髓袢长（深入内髓）-> 口径差异小 -> 几乎无球旁器 -> **尿液浓缩稀释**&nbsp;&nbsp;&nbsp;
- 球旁器（JGA）★★必考&nbsp;&nbsp;&nbsp;
  - 球旁细胞：分泌**肾素**&nbsp;&nbsp;&nbsp;
  - 致密斑：感受小管液中 **NaCl 含量**变化&nbsp;&nbsp;&nbsp;
  - 球外系膜细胞：吞噬、收缩功能&nbsp;&nbsp;&nbsp;
- 肾血流特点&nbsp;&nbsp;&nbsp;
  - 总流量：**1200 ml/min** -> 占 CO 的 **20~25%**&nbsp;&nbsp;&nbsp;
  - 分布：**94%** 皮质 + 6% 髓质&nbsp;&nbsp;&nbsp;
  - 肾小球 Cap 网（入球→Cap→出球）：**血压高** -> 利于**滤过**&nbsp;&nbsp;&nbsp;
  - 肾小管周围 Cap 网（出球→Cap→肾静脉）：**血压低** -> 利于**重吸收**&nbsp;&nbsp;&nbsp;
- 肾血流自身调节（**70~180 mmHg** 范围内恒定）&nbsp;&nbsp;&nbsp;
  - 肌源学说：&nbsp;&nbsp;&nbsp;
    - 动脉压↑ -> 入球小动脉平滑肌**紧张性↑** -> 口径↓ -> 血流阻力↑ -> 肾血流量**稳定**&nbsp;&nbsp;&nbsp;
    - 动脉压↓ -> 平滑肌紧张性↓ -> 口径↑ -> 血流阻力↓ -> 肾血流量**稳定**&nbsp;&nbsp;&nbsp;
  - 管球反馈：&nbsp;&nbsp;&nbsp;
    - GFR↑ -> 小管液流量↑ -> 致密斑感受 **NaCl↑** -> 入球小动脉**收缩** -> GFR↓（负反馈）&nbsp;&nbsp;&nbsp;
- 神经体液调节（紧急情况）&nbsp;&nbsp;&nbsp;
  - 交感神经兴奋 -> 肾血管**收缩** -> 肾血流量↓&nbsp;&nbsp;&nbsp;
  - Adr/NA/ATP -> 缩血管；PG/NO -> 舒血管&nbsp;&nbsp;&nbsp;
  - 意义：大失血时 -> 肾血管收缩 -> 保证**心脑供血**&nbsp;&nbsp;&nbsp;
