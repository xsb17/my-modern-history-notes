# 胰岛内分泌（胰岛素为主）— 微型速记导图&nbsp;&nbsp;&nbsp;

- 胰岛细胞组成&nbsp;&nbsp;&nbsp;
  - α 细胞（**25%**）：**胰高血糖素** -> 升血糖&nbsp;&nbsp;&nbsp;
  - β 细胞（**60~70%**）：**胰岛素** -> 降血糖&nbsp;&nbsp;&nbsp;
  - δ 细胞（5%）：生长抑素&nbsp;&nbsp;&nbsp;
- ★胰岛素——体内**唯一**降血糖激素&nbsp;&nbsp;&nbsp;
  - 糖代谢：**降低血糖**&nbsp;&nbsp;&nbsp;
    - 减少来源：抑制糖原分解 + 抑制糖异生&nbsp;&nbsp;&nbsp;
    - 增加去路：促进糖原合成 + 促进外周组织**摄取和利用**葡萄糖&nbsp;&nbsp;&nbsp;
    - 缺乏 -> **糖尿病**&nbsp;&nbsp;&nbsp;
  - 脂肪代谢：**促进合成/减少分解**&nbsp;&nbsp;&nbsp;
    - 促葡萄糖入脂肪细胞合成脂肪酸&nbsp;&nbsp;&nbsp;
    - 抑制激素敏感性脂肪酶 -> ↓脂肪分解&nbsp;&nbsp;&nbsp;
  - 蛋白质代谢：**促进合成/抑制分解**&nbsp;&nbsp;&nbsp;
    - 促氨基酸入细胞合成蛋白&nbsp;&nbsp;&nbsp;
    - 抑制蛋白分解和糖异生&nbsp;&nbsp;&nbsp;
  - 促生长：与 GH 协同&nbsp;&nbsp;&nbsp;
- ★胰岛素分泌调节&nbsp;&nbsp;&nbsp;
  - ① **血糖浓度**：血糖↑ -> 胰岛素分泌↑（**最重要**）&nbsp;&nbsp;&nbsp;
  - ② 氨基酸/脂肪酸 -> 刺激分泌&nbsp;&nbsp;&nbsp;
  - ③ 促血糖升高的激素可促胰岛素分泌（胰高血糖素/TH/GH/GC）&nbsp;&nbsp;&nbsp;
  - ④ 副交感神经(+) -> 促分泌&nbsp;&nbsp;&nbsp;
  - 临床联系：长期高血糖/高血脂/高氨基酸 -> 胰岛素持续分泌 -> β细胞**衰竭** -> 糖尿病&nbsp;&nbsp;&nbsp;
- 胰高血糖素——升血糖&nbsp;&nbsp;&nbsp;
  - 促进糖原分解 + 糖异生 -> **升高血糖**&nbsp;&nbsp;&nbsp;
  - 与胰岛素**拮抗**，共同维持血糖稳态&nbsp;&nbsp;&nbsp;
- 糖尿病概要&nbsp;&nbsp;&nbsp;
  - 1 型：β细胞破坏 -> 胰岛素绝对缺乏&nbsp;&nbsp;&nbsp;
  - 2 型：胰岛素抵抗 + β细胞功能减退&nbsp;&nbsp;&nbsp;
  - 并发症：心血管/肾/视网膜/神经病变&nbsp;&nbsp;&nbsp;
