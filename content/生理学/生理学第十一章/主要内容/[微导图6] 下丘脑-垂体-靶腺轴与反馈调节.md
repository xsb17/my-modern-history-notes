# 下丘脑-垂体-靶腺轴与反馈调节 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★两大负反馈轴（必考）&nbsp;&nbsp;&nbsp;
  - 甲状腺轴：TRH -> TSH -> TH（T₃/T₄）&nbsp;&nbsp;&nbsp;
  - 肾上腺轴：CRH -> ACTH -> GC（皮质醇）&nbsp;&nbsp;&nbsp;
- 甲状腺轴（TRH-TSH-TH）&nbsp;&nbsp;&nbsp;
  - 下丘脑：**TRH** -> 促TSH合成释放 + 促TSH糖基化（保证活性）&nbsp;&nbsp;&nbsp;
  - 腺垂体：**TSH** -> 促甲状腺增殖 + 促TH合成分泌&nbsp;&nbsp;&nbsp;
  - 甲状腺：分泌 **T₃/T₄**&nbsp;&nbsp;&nbsp;
  - 负反馈：TH↑ -> 抑制TSH（下调TRH受体）+ 抑制TRH合成&nbsp;&nbsp;&nbsp;
  - 甲亢：T₃/T₄**↑** + TSH**↓**&nbsp;&nbsp;&nbsp;
  - 甲减：T₃/T₄**↓** + TSH**↑**&nbsp;&nbsp;&nbsp;
  - ★缺碘→TH↓→TSH↑→**甲状腺肿大**&nbsp;&nbsp;&nbsp;
- 肾上腺轴（CRH-ACTH-GC）&nbsp;&nbsp;&nbsp;
  - 下丘脑：**CRH** -> 促 ACTH 分泌&nbsp;&nbsp;&nbsp;
  - 腺垂体：**ACTH** -> 维持肾上腺皮质结构 + 促 GC 合成分泌&nbsp;&nbsp;&nbsp;
  - 肾上腺皮质：分泌 **GC（皮质醇）**&nbsp;&nbsp;&nbsp;
  - 生物钟：CRH清晨**最高** / 午夜**最低** -> GC 呈昼夜节律&nbsp;&nbsp;&nbsp;
  - 负反馈：GC↑ -> 抑制 CRH/ACTH&nbsp;&nbsp;&nbsp;
  - ★长期用GC药物 -> ACTH↓ -> 肾上腺皮质**萎缩** -> 突然停药 -> **肾上腺危象/死亡**&nbsp;&nbsp;&nbsp;
  - 停药原则：**逐渐减量 + 间断补ACTH**&nbsp;&nbsp;&nbsp;
- ★反馈层级对比&nbsp;&nbsp;&nbsp;
  - 长反馈：**靶腺激素** -> 抑下丘脑/腺垂体（最重要）&nbsp;&nbsp;&nbsp;
  - 短反馈：**腺垂体激素** -> 抑下丘脑&nbsp;&nbsp;&nbsp;
  - 超短反馈：**下丘脑激素** -> 抑自身&nbsp;&nbsp;&nbsp;
- ★应激反应的特殊性&nbsp;&nbsp;&nbsp;
  - 应激时：下丘脑-垂体-肾上腺轴大量激活 -> GC 大量分泌&nbsp;&nbsp;&nbsp;
  - **不受轴性负反馈影响**（区别于日常节律调控）&nbsp;&nbsp;&nbsp;
- 临床联系&nbsp;&nbsp;&nbsp;
  - 长期用泼尼松/地塞米松 -> 必须**缓慢减量**（防肾上腺皮质功能不全）&nbsp;&nbsp;&nbsp;
  - 库欣综合征：GC过多 -> 满月脸/水牛背/高血压/高血糖&nbsp;&nbsp;&nbsp;
  - 艾迪生病：GC不足 -> 低血糖/低血压/色素沉着&nbsp;&nbsp;&nbsp;
