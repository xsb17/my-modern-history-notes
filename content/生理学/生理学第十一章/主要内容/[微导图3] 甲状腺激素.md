# 甲状腺激素 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 甲状腺概况&nbsp;&nbsp;&nbsp;
  - 人体**最大**内分泌腺 -> 结构：滤泡（上皮细胞合成TH + 腔内**储存**TH）&nbsp;&nbsp;&nbsp;
  - 储存量：可供 **50~120天**使用&nbsp;&nbsp;&nbsp;
  - TH 组成：**T₄**（占 **90%**）+ **T₃**（占 9%）&nbsp;&nbsp;&nbsp;
  - 活性：**T₃ > T₄**（T₄ 在外周转化为 T₃）&nbsp;&nbsp;&nbsp;
  - 合成原料：甲状腺球蛋白 + **碘**&nbsp;&nbsp;&nbsp;
- ★TH 生物作用——促生长、促代谢、促兴奋&nbsp;&nbsp;&nbsp;
  - ① 促进生长发育：&nbsp;&nbsp;&nbsp;
    - 促**神经系统**发育（胎儿/新生儿脑发育的**关键激素**）&nbsp;&nbsp;&nbsp;
    - 促**骨骼**生长（与 GH 协同）&nbsp;&nbsp;&nbsp;
    - 缺乏 -> **呆小症/克汀病**（**身材矮小+智力低下**）&nbsp;&nbsp;&nbsp;
    - 与侏儒症区别：侏儒（GH缺）->矮小/**智力正常**；呆小症（TH缺）->矮小/**智力低下**&nbsp;&nbsp;&nbsp;
  - ② 调节新陈代谢：&nbsp;&nbsp;&nbsp;
    - 产热效应：↑绝大多数组织耗氧量和产热量&nbsp;&nbsp;&nbsp;
    - 蛋白质：生理浓度促合成/高浓度促分解&nbsp;&nbsp;&nbsp;
    - 糖：升糖 > 降糖&nbsp;&nbsp;&nbsp;
    - 脂肪：促分解 > 促合成&nbsp;&nbsp;&nbsp;
  - ③ 影响器官系统：&nbsp;&nbsp;&nbsp;
    - 促**交感神经兴奋** -> 心率↑/出汗↑/脾气不好&nbsp;&nbsp;&nbsp;
    - 促消化 -> 吃得多还不胖&nbsp;&nbsp;&nbsp;
- ★TH 分泌调节——TRH-TSH-TH 轴&nbsp;&nbsp;&nbsp;
  - 下丘脑 **TRH**(+) -> 腺垂体 **TSH**(+) -> 甲状腺 **TH**(T₃/T₄)&nbsp;&nbsp;&nbsp;
  - TH **↑** -> 负反馈抑制 TSH + 抑制 TRH&nbsp;&nbsp;&nbsp;
  - 甲亢：T₃/T₄**↑** + TSH**↓**&nbsp;&nbsp;&nbsp;
  - 甲减：T₃/T₄**↓** + TSH**↑**&nbsp;&nbsp;&nbsp;
- ★缺碘导致甲状腺肿大机制（必考）&nbsp;&nbsp;&nbsp;
  - 缺碘 -> TH 合成↓ -> 负反馈抑制减弱 -> **TSH↑**&nbsp;&nbsp;&nbsp;
  - -> 甲状腺滤泡细胞**持续增殖** -> **甲状腺肿大（大脖子病）**&nbsp;&nbsp;&nbsp;
- 碘阻滞效应：血碘 > 10mmol/L -> 抑制 TH 合成&nbsp;&nbsp;&nbsp;
- 临床要点&nbsp;&nbsp;&nbsp;
  - 先天性甲低 -> 出生后 **3个月**内必须补 TH（否则智力不可逆损伤）&nbsp;&nbsp;&nbsp;
  - 甲亢症状：心率快/出汗多/消瘦/易激动/甲状腺肿大/突眼&nbsp;&nbsp;&nbsp;
  - 甲减症状：畏寒/乏力/黏液性水肿/反应迟钝&nbsp;&nbsp;&nbsp;
