# 糖皮质激素 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 肾上腺结构&nbsp;&nbsp;&nbsp;
  - 皮质：球状带(盐皮质激素) + **束状带(GC)** + 网状带(雄激素)&nbsp;&nbsp;&nbsp;
  - 髓质：**肾上腺素/去甲肾上腺素**（交感节前纤维支配）&nbsp;&nbsp;&nbsp;
- ★糖皮质激素（GC）——血浆最多为**皮质醇**&nbsp;&nbsp;&nbsp;
  - 临床常用：**氢化可的松/泼尼松/地塞米松**&nbsp;&nbsp;&nbsp;
- ★GC 生理作用&nbsp;&nbsp;&nbsp;
  - ① 糖代谢：**抗胰岛素样作用** -> **升高血糖**（糖尿病人慎用）&nbsp;&nbsp;&nbsp;
  - ② 脂肪代谢：促四肢脂肪**分解** + 促面/躯干脂肪**合成** -> **向心性肥胖**（满月脸/水牛背）&nbsp;&nbsp;&nbsp;
  - ③ 蛋白质：抑制肝外合成/促进分解 -> 骨质疏松/皮肤变薄/抑制创伤愈合&nbsp;&nbsp;&nbsp;
  - ④ 水盐代谢：利排水 -> 缺乏->**水中毒**&nbsp;&nbsp;&nbsp;
  - ⑤ ★应激反应（必考）：&nbsp;&nbsp;&nbsp;
    - 伤害性刺激（创伤/手术/感染/缺氧等）-> 腺垂体释放**大量ACTH** -> GC大量分泌&nbsp;&nbsp;&nbsp;
    - 意义：增加**耐受性和抵抗力**&nbsp;&nbsp;&nbsp;
    - GC 是应激反应的**核心激素**&nbsp;&nbsp;&nbsp;
  - ⑥ 对组织器官影响：&nbsp;&nbsp;&nbsp;
    - 血液：RBC/中性粒/PLT**↑** + 淋巴/嗜酸性粒**↓**&nbsp;&nbsp;&nbsp;
    - 循环：**允许作用**（↑心血管对儿茶酚胺/AngII反应性）&nbsp;&nbsp;&nbsp;
    - 药理：**抗炎/抗毒/抗过敏/抗休克**（四抗）&nbsp;&nbsp;&nbsp;
- ★GC 分泌调节——CRH-ACTH-GC 轴&nbsp;&nbsp;&nbsp;
  - 下丘脑 **CRH**(+) -> 腺垂体 **ACTH**(+) -> 肾上腺皮质 **GC**&nbsp;&nbsp;&nbsp;
  - 生物钟：CRH清晨觉醒前**最高** -> 午夜**最低**&nbsp;&nbsp;&nbsp;
  - 长反馈：GC↑ -> 抑 CRH/ACTH&nbsp;&nbsp;&nbsp;
  - 短反馈：ACTH↑ -> 抑 CRH&nbsp;&nbsp;&nbsp;
  - ★临床重点：**长期大剂量用GC -> 负反馈抑ACTH -> 肾上腺皮质萎缩 -> 突然停药->肾上腺危象可致死**&nbsp;&nbsp;&nbsp;
  - 停药原则：**逐渐减量 + 间断补充ACTH**&nbsp;&nbsp;&nbsp;
  - 应激时：CRH-ACTH-GC大量分泌，**不受轴性负反馈影响**&nbsp;&nbsp;&nbsp;
