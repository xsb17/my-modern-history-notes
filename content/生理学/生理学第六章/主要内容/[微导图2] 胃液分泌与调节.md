# 胃液分泌与调节 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 胃液性质：酸性 **pH 0.9~1.5** | 分泌量 **1.5~2.5 L/d**&nbsp;&nbsp;&nbsp;
- ★胃液四大成分及细胞来源&nbsp;&nbsp;&nbsp;
  - 盐酸（胃酸）：泌酸腺**壁细胞**&nbsp;&nbsp;&nbsp;
    - 分泌机制：质子泵 **H⁺-K⁺-ATP酶** -> 抑制剂：**奥美拉唑** -> 餐后碱潮（HCO₃⁻入血）&nbsp;&nbsp;&nbsp;
    - 基础排酸 **0~5 mmol/h** | 最大 **20~25 mmol/h**&nbsp;&nbsp;&nbsp;
    - 五大作用：&nbsp;&nbsp;&nbsp;
      - ① 杀死胃内**细菌**&nbsp;&nbsp;&nbsp;
      - ② 使蛋白质**变性**&nbsp;&nbsp;&nbsp;
      - ③ **激活胃蛋白酶原**+提供酸性环境&nbsp;&nbsp;&nbsp;
      - ④ 刺激促胰液素/缩胆囊素释放&nbsp;&nbsp;&nbsp;
      - ⑤ 促进小肠 **Ca²⁺/Fe²⁺** 吸收&nbsp;&nbsp;&nbsp;
    - 过少 -> 消化不良；过多 -> 消化性溃疡&nbsp;&nbsp;&nbsp;
  - 胃蛋白酶原：泌酸腺**主细胞** -> HCl/胃蛋白酶激活 -> **胃蛋白酶** -> 水解蛋白为胨/胨&nbsp;&nbsp;&nbsp;
  - 粘液：上皮细胞/贲门腺/幽门腺/颈细胞 -> **粘液-碳酸氢盐屏障** -> 保护胃粘膜&nbsp;&nbsp;&nbsp;
  - 内因子：泌酸腺**壁细胞** -> 保护 + 促进回肠吸收 **VitB₁₂** -> 缺乏 -> **巨幼红细胞性贫血**&nbsp;&nbsp;&nbsp;
- ★胃的保护与损伤&nbsp;&nbsp;&nbsp;
  - 保护：粘液-HCO₃⁻屏障 + 胃黏膜屏障（紧密连接）+ PGE₂/PGI₂/EGF&nbsp;&nbsp;&nbsp;
  - 损伤：**酒精、阿司匹林、幽门螺杆菌** -> 破坏屏障 -> 胃炎/胃溃疡&nbsp;&nbsp;&nbsp;
- ★消化期胃液分泌三期对比&nbsp;&nbsp;&nbsp;
  - 头期：量**30%** -> 条件+非条件反射 -> 迷走神经 -> **胃蛋白酶多/胃酸多**&nbsp;&nbsp;&nbsp;
  - 胃期：量**60%** -> 迷走-迷走反射 + 壁内丛 + G细胞->促胃液素 -> **胃酸多/胃蛋白酶↓**&nbsp;&nbsp;&nbsp;
  - 肠期：量**10%** -> 十二指肠G细胞->促胃液素 -> **胃酸少/胃蛋白酶少**&nbsp;&nbsp;&nbsp;
- ★促进胃液分泌三因素&nbsp;&nbsp;&nbsp;
  - 迷走神经 -> **ACh + M受体** -> 阻断剂：**阿托品**&nbsp;&nbsp;&nbsp;
  - 促胃液素 -> **G细胞** -> 阻断剂：**丙谷胺**&nbsp;&nbsp;&nbsp;
  - 组胺 -> **ECL细胞**分泌 -> 阻断剂：**甲氰咪呱**（H₂受体拮抗剂）&nbsp;&nbsp;&nbsp;
  - 咖啡/酒精也刺激胃酸&nbsp;&nbsp;&nbsp;
- 抑制胃液分泌三因素&nbsp;&nbsp;&nbsp;
  - 盐酸 -> 负反馈 -> 抑制促胃液素 + 刺激促胰液素&nbsp;&nbsp;&nbsp;
  - 脂肪 -> 刺激**肠抑胃素**（促胰液素/缩胆囊素/抑胃肽）&nbsp;&nbsp;&nbsp;
  - 高张溶液 -> 肠胃反射&nbsp;&nbsp;&nbsp;
