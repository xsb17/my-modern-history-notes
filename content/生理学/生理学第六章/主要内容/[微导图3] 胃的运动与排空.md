# 胃的运动与排空 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★胃的三种运动形式&nbsp;&nbsp;&nbsp;
  - 容受性舒张（**胃特有**）★必考：&nbsp;&nbsp;&nbsp;
    - 食物刺激口腔/咽/食管 -> 反射性胃底/胃体**舒张** -> 容积 **50ml→1.5L**&nbsp;&nbsp;&nbsp;
    - 意义：**容纳食物** + 维持胃内压**不变**&nbsp;&nbsp;&nbsp;
    - 机制：迷走神经释放**肽类物质（VIP）**&nbsp;&nbsp;&nbsp;
  - 紧张性收缩（共有）：&nbsp;&nbsp;&nbsp;
    - 胃经常持续微弱收缩&nbsp;&nbsp;&nbsp;
    - 意义：促进胃液渗入食物 + 增加胃内压 + 维持形态位置 + 是**蠕动的基础**&nbsp;&nbsp;&nbsp;
  - 蠕动（共有）：&nbsp;&nbsp;&nbsp;
    - 前端舒张/后端收缩的波形运动&nbsp;&nbsp;&nbsp;
    - 意义：**推进/研磨/搅拌**食物&nbsp;&nbsp;&nbsp;
- 胃运动的调节&nbsp;&nbsp;&nbsp;
  - 神经：**迷走+** | 交感-&nbsp;&nbsp;&nbsp;
  - 体液：**胃动素+/促胃液素+** | 缩胆囊素-/促胰液素-&nbsp;&nbsp;&nbsp;
- ★胃排空（必考）&nbsp;&nbsp;&nbsp;
  - 概念：食物间断地由胃**排入十二指肠**的过程&nbsp;&nbsp;&nbsp;
  - 排空速度：**糖类 < 蛋白质 < 脂肪** -> 混合食物需 **4~6h**&nbsp;&nbsp;&nbsp;
  - 胃内促进因素：&nbsp;&nbsp;&nbsp;
    - 胃内容物扩张 -> 迷走-迷走反射 + 壁内丛 -> 胃运动↑&nbsp;&nbsp;&nbsp;
    - 促胃液素 -> 胃运动↑&nbsp;&nbsp;&nbsp;
  - 十二指肠抑制因素（反馈调控）：&nbsp;&nbsp;&nbsp;
    - 盐酸/脂肪/高渗 -> 肠胃反射 + **肠抑胃素** -> 胃排空**↓**&nbsp;&nbsp;&nbsp;
- 胃排空调控口诀&nbsp;&nbsp;&nbsp;
  - 胃内推、肠内刹 -> 糖最快、脂最慢 -> 混合食4-6h&nbsp;&nbsp;&nbsp;
