# 眼的折光系统与视近物调节 — 微型速记导图（极高频）&nbsp;&nbsp;&nbsp;

- 折光系统组成：**角膜 -> 房水 -> 晶状体 -> 玻璃体**&nbsp;&nbsp;&nbsp;
- 简化眼&nbsp;&nbsp;&nbsp;
  - 前后径 **20mm** | 折射率 **1.333** | 曲率半径 **5mm**&nbsp;&nbsp;&nbsp;
  - 节点(n)：角膜后方 **5mm** 处 -> 后主焦点 = 视网膜&nbsp;&nbsp;&nbsp;
- ★视近物调节——近反射三大变化（必考）&nbsp;&nbsp;&nbsp;
  - ① 晶状体调节（**最主要**）：&nbsp;&nbsp;&nbsp;
    - 视远物（**6m以外**）-> 不需调节&nbsp;&nbsp;&nbsp;
      - 睫状肌**舒张** -> 悬韧带**紧张** -> 晶状体**扁平**&nbsp;&nbsp;&nbsp;
    - 视近物 -> 成像在视网膜后(模糊) ->&nbsp;&nbsp;&nbsp;
      - 动眼神经(+) -> **睫状肌收缩** -> **悬韧带松弛** -> 晶状体**变凸**(前面为主)&nbsp;&nbsp;&nbsp;
      - -> 折光力↑ -> 焦距缩短 -> 物像前移回到视网膜&nbsp;&nbsp;&nbsp;
  - ② 瞳孔调节：&nbsp;&nbsp;&nbsp;
    - 瞳孔对光反射：强光->**缩小**/弱光->散大 -> 调节入眼光量 -> **双侧性**&nbsp;&nbsp;&nbsp;
    - 瞳孔近反射：看近物 -> 反射性**缩小** -> 减少球面像差和色相差&nbsp;&nbsp;&nbsp;
  - ③ 视轴会聚（辐辏反射）：&nbsp;&nbsp;&nbsp;
    - 双眼看近物 -> 两视轴向**鼻侧聚拢**&nbsp;&nbsp;&nbsp;
    - 意义：物像落在**对称点** -> 避免**复视**&nbsp;&nbsp;&nbsp;
- ★近点 = 眼最大调节时能看清的**最近距离** -> 反映晶状体弹性&nbsp;&nbsp;&nbsp;
  - 10岁 **8.8cm** -> 20岁 **10.4cm** -> 40岁 **18.5cm** -> 60岁 **83.3cm**&nbsp;&nbsp;&nbsp;
  - 年龄↑ -> 晶状体弹性↓ -> 近点**远移**&nbsp;&nbsp;&nbsp;
- ★四种屈光异常及矫正镜片（必考选择）&nbsp;&nbsp;&nbsp;
  - 近视：眼轴**过长**/折光力过强 -> 远物成像于视网膜**前** -> 纠正：**凹透镜**&nbsp;&nbsp;&nbsp;
  - 远视：眼轴**过短**/折光力过弱 -> 近物成像于视网膜**后** -> 纠正：**凸透镜**&nbsp;&nbsp;&nbsp;
  - 散光：角膜/晶状体曲率**不均** -> 不同方向折光不一 -> 纠正：**圆柱镜**&nbsp;&nbsp;&nbsp;
  - 老视：晶状体弹性**↓** -> 近点远移 -> 纠正：**凸透镜**&nbsp;&nbsp;&nbsp;
