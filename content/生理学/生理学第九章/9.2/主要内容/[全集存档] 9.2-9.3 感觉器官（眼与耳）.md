# 第九章第二至三节：感觉器官（眼与耳）— 全集存档&nbsp;&nbsp;&nbsp;

## 一、屈光不正（折光异常）&nbsp;&nbsp;&nbsp;

- 定义：因折光系统折光力异常或眼球形态异常，使平行光线不能聚焦在视网膜上的现象&nbsp;&nbsp;&nbsp;

### （一）近视（myopia）&nbsp;&nbsp;&nbsp;

- 原因：眼球前后径**过长**、折光力**过强**&nbsp;&nbsp;&nbsp;
- 视近物：辐散光可聚焦在视网膜上&nbsp;&nbsp;&nbsp;
- 视远物：平行光聚焦在视网膜**前方**&nbsp;&nbsp;&nbsp;
- 矫正：配带**凹透镜**&nbsp;&nbsp;&nbsp;

### （二）远视（hyperopia）&nbsp;&nbsp;&nbsp;

- 原因：眼球前后径**过短**、折光力**过弱**&nbsp;&nbsp;&nbsp;
- 视远物：平行光聚焦在视网膜**后方**（可调节）&nbsp;&nbsp;&nbsp;
- 视近物：物像更靠后&nbsp;&nbsp;&nbsp;
- 矫正：配带**凸透镜**&nbsp;&nbsp;&nbsp;

### （三）散光（astigmatism）&nbsp;&nbsp;&nbsp;

- 原因：眼球在不同方位的折光力**不一致**（不同方位折光面曲率半径不相等）&nbsp;&nbsp;&nbsp;
- 矫正：配带**圆柱镜**&nbsp;&nbsp;&nbsp;

## 二、眼的感光换能系统&nbsp;&nbsp;&nbsp;

### （一）视网膜的结构特点&nbsp;&nbsp;&nbsp;

- 分层：色素细胞 → 感光细胞（视锥/视杆）→ 双极细胞 → 神经节细胞&nbsp;&nbsp;&nbsp;
- **生理盲点**：视神经穿过视网膜处形成视神经乳头，此处**没有感光细胞**，没有感光功能（双眼视物对侧补偿）&nbsp;&nbsp;&nbsp;
- **黄斑中央凹**：视神经乳头向颞侧约 3mm，感光**最敏锐**的地方&nbsp;&nbsp;&nbsp;

### （二）视网膜两种感光系统对比&nbsp;&nbsp;&nbsp;

#### 视锥系统（明视觉）&nbsp;&nbsp;&nbsp;

- 光敏感性：**差**（感受强光）&nbsp;&nbsp;&nbsp;
- 分辨力：**精细**，有**色觉**&nbsp;&nbsp;&nbsp;
- 外形：短圆锥状&nbsp;&nbsp;&nbsp;
- 分布：密集于**中央凹**处，周边少&nbsp;&nbsp;&nbsp;
- 感光色素：红视锥、绿视锥、蓝视锥（**三原色学说**）&nbsp;&nbsp;&nbsp;
- 突触联系：**单线式**联系 → 利于提高分辨力&nbsp;&nbsp;&nbsp;
- 代表动物：鸡（几乎只有视锥细胞）&nbsp;&nbsp;&nbsp;

#### 视杆系统（暗视觉）&nbsp;&nbsp;&nbsp;

- 光敏感性：**高**（感受弱光）&nbsp;&nbsp;&nbsp;
- 分辨力：**粗糙**，无**色觉**&nbsp;&nbsp;&nbsp;
- 外形：长杆状&nbsp;&nbsp;&nbsp;
- 分布：视网膜**周边**多，中央凹无&nbsp;&nbsp;&nbsp;
- 感光色素：**视紫红质**&nbsp;&nbsp;&nbsp;
- 突触联系：**聚合式**联系 → 可将弱刺激总和&nbsp;&nbsp;&nbsp;
- 代表动物：猫头鹰（只有视杆细胞）&nbsp;&nbsp;&nbsp;

### （三）视杆细胞的感光换能机制&nbsp;&nbsp;&nbsp;

#### 1. 视紫红质的光化学反应&nbsp;&nbsp;&nbsp;

- 视紫红质 = **视蛋白 + 11-顺型视黄醛**&nbsp;&nbsp;&nbsp;
- 光照：11-顺型视黄醛 → **全反型视黄醛**（与视蛋白分离）→ 视紫红质**分解**（漂白）&nbsp;&nbsp;&nbsp;
- 暗处：全反型视黄醛 → 11-顺型视黄醛（需异构酶）→ 与视蛋白结合 → 视紫红质**合成**&nbsp;&nbsp;&nbsp;

#### 2. 视杆细胞的感受器电位&nbsp;&nbsp;&nbsp;

- 暗处静息电位：**-30~-40mV**&nbsp;&nbsp;&nbsp;
- 暗电流：暗处胞质 cGMP 含量高 → cGMP 门控 Na⁺ 通道**开放** → Na⁺ 持续内流（内向暗电流）&nbsp;&nbsp;&nbsp;
- 光照时：视紫红质光化学反应 → 激活 G 蛋白 → 激活磷酸二酯酶 → cGMP 分解失活 → Na⁺ 通道**关闭** → 暗电流消失 → 只有 K⁺ 外流 → **超极化**&nbsp;&nbsp;&nbsp;

### （四）视锥细胞的感光原理和色觉&nbsp;&nbsp;&nbsp;

- 感光换能机制与视杆细胞相似&nbsp;&nbsp;&nbsp;
- 色觉：不同波长的可见光刺激人眼后在脑内引起的主观感觉&nbsp;&nbsp;&nbsp;
- 人眼能分辨波长 380~760nm 之间约 **150 种颜色**&nbsp;&nbsp;&nbsp;
- 三原色学说：&nbsp;&nbsp;&nbsp;
  - 红视锥（**564nm**）、绿视锥（**534nm**）、蓝视锥（**420nm**）&nbsp;&nbsp;&nbsp;
  - 不同波长 → 三种视锥以不同比例兴奋 → 产生不同色觉&nbsp;&nbsp;&nbsp;
- 色盲：缺乏编码相应感光色素的基因（多为遗传性）→ 红色盲/绿色盲/蓝色盲/全色盲&nbsp;&nbsp;&nbsp;
- 色弱：某种视锥细胞反应能力较弱（多为后天因素）&nbsp;&nbsp;&nbsp;

## 三、视觉相关生理现象&nbsp;&nbsp;&nbsp;

### （一）暗适应&nbsp;&nbsp;&nbsp;

- 从**亮处进入暗室**，最初看不清物体，经一定时间后逐渐恢复暗处视力&nbsp;&nbsp;&nbsp;
- 时间**较长**，与**视紫红质合成**有关&nbsp;&nbsp;&nbsp;

### （二）明适应&nbsp;&nbsp;&nbsp;

- 从**暗处来到强光**下，最初强光耀眼不能视物，稍待片刻恢复视觉&nbsp;&nbsp;&nbsp;
- 时间**较短**，与**视紫红质分解**有关&nbsp;&nbsp;&nbsp;

### （三）视野&nbsp;&nbsp;&nbsp;

- 单眼固定注视前方一点时所能看到的范围&nbsp;&nbsp;&nbsp;
- 鼻侧和上方较小，颞侧和下方较大&nbsp;&nbsp;&nbsp;
- 大小排名：**白色 > 黄色 > 蓝色 > 红色 > 绿色**&nbsp;&nbsp;&nbsp;

## 四、听觉器官&nbsp;&nbsp;&nbsp;

### 第三节 听觉器官&nbsp;&nbsp;&nbsp;

- 感受器：**螺旋器**（Corti 器）&nbsp;&nbsp;&nbsp;
- 适宜刺激：频率 **20~20000Hz** 声波&nbsp;&nbsp;&nbsp;
- 听阈：每个频率声波刚能引起听觉的最小声波强度&nbsp;&nbsp;&nbsp;
- 最大可听阈：人耳能忍受的最强声压&nbsp;&nbsp;&nbsp;
- 听域：人耳能感受到的整个声波频率和强度范围&nbsp;&nbsp;&nbsp;
- 最敏感频率：**1000~3000Hz**；日常语言频率：**300~3000Hz**&nbsp;&nbsp;&nbsp;

### （一）外耳的功能&nbsp;&nbsp;&nbsp;

- 耳廓：**收集声波**&nbsp;&nbsp;&nbsp;
- 外耳道：**传导声波**&nbsp;&nbsp;&nbsp;

### （二）中耳的功能&nbsp;&nbsp;&nbsp;

#### 1. 鼓膜&nbsp;&nbsp;&nbsp;

- 椭圆形半透明薄膜，振动与声波振动同始同终，失真度小&nbsp;&nbsp;&nbsp;

#### 2. 听骨链&nbsp;&nbsp;&nbsp;

- 鼓膜 → 锤骨 → 砧骨 → 镫骨 → 前庭窗&nbsp;&nbsp;&nbsp;
- **鼓膜和听骨链的增压效应**：能将声波压强**增大**、振幅减小，提高声波传递效率&nbsp;&nbsp;&nbsp;
  - 鼓膜面积:前庭窗 ≈ **17.2:1**&nbsp;&nbsp;&nbsp;
  - 听骨链杠杆长臂:短臂 ≈ **1.3:1**&nbsp;&nbsp;&nbsp;
  - 总增压：17.2 × 1.3 = **22.4 倍**，振幅减小约 **1/4**&nbsp;&nbsp;&nbsp;
  - 意义：补偿气体声阻抗与液体声阻抗差异导致的能量衰减&nbsp;&nbsp;&nbsp;

#### 3. 咽鼓管的功能&nbsp;&nbsp;&nbsp;

- 连通鼻咽部和鼓室，鼻咽部开口在吞咽/张口时开放&nbsp;&nbsp;&nbsp;
- 调节鼓室内空气压力，与外界大气压保持平衡&nbsp;&nbsp;&nbsp;
- 意义：维持鼓膜正常位置、形状和振动&nbsp;&nbsp;&nbsp;
- 儿童咽鼓管短、粗且平直 → 鼻炎/咽炎时易引起中耳炎&nbsp;&nbsp;&nbsp;

### （三）声波传入内耳的途径&nbsp;&nbsp;&nbsp;

#### 1. 气传导（主要途径）&nbsp;&nbsp;&nbsp;

- 声波 → 外耳道 → 鼓膜 → 听骨链 → 前庭窗膜 → 耳蜗&nbsp;&nbsp;&nbsp;
- 鼓室空气 → 蜗窗膜&nbsp;&nbsp;&nbsp;

#### 2. 骨传导&nbsp;&nbsp;&nbsp;

- 声波 → 颅骨振动 → 耳蜗内淋巴振动&nbsp;&nbsp;&nbsp;
- 较强声波、自己说话时较明显&nbsp;&nbsp;&nbsp;

## 五、内耳耳蜗的感音功能&nbsp;&nbsp;&nbsp;

### （一）基底膜的振动与感音换能功能&nbsp;&nbsp;&nbsp;

- 耳蜗：骨性管道长约 **35mm**，盘旋 **2.5~2.75 圈**&nbsp;&nbsp;&nbsp;
- 感音换能链：声波振动 → 鼓膜振动 → 听骨链 → 前庭窗膜振动 → 耳蜗淋巴液 → **基底膜振动** → 基底膜与盖膜间相对位置变化 → 听毛弯曲 → 毛细胞产生感受器电位 → 蜗神经产生 AP&nbsp;&nbsp;&nbsp;

#### 行波学说&nbsp;&nbsp;&nbsp;

- 基底膜振动从前庭窗处以**行波**方式向耳蜗顶部传导&nbsp;&nbsp;&nbsp;
- **高频**声波 → 行波传播**近** → 感受部位在耳蜗**底部**&nbsp;&nbsp;&nbsp;
- **低频**声波 → 行波传播**远** → 感受部位在耳蜗**顶部**&nbsp;&nbsp;&nbsp;

### （二）耳蜗的生物电现象&nbsp;&nbsp;&nbsp;

#### 1. 耳蜗内电位&nbsp;&nbsp;&nbsp;

- 耳蜗未受声波刺激时不同部位的电位差&nbsp;&nbsp;&nbsp;
- 毛细胞静息电位 **-70~-80mV**，耳蜗内电位（内淋巴）**+80mV**&nbsp;&nbsp;&nbsp;

#### 2. 耳蜗微音器电位&nbsp;&nbsp;&nbsp;

- 耳蜗受到声波刺激时，产生类似声波作用于微音器产生的电变化&nbsp;&nbsp;&nbsp;
- 多个毛细胞产生的感受器电位的**复合型电位变化**&nbsp;&nbsp;&nbsp;
- 可以总和，触发蜗神经产生动作电位&nbsp;&nbsp;&nbsp;

### （三）听神经动作电位&nbsp;&nbsp;&nbsp;

- 通过神经冲动的节律、间隔时间等传递不同形式的声音信息&nbsp;&nbsp;&nbsp;
