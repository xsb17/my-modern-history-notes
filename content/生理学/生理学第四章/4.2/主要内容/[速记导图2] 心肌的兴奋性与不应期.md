# 心肌的兴奋性与不应期 — 极简速记导图&nbsp;&nbsp;&nbsp;

- 兴奋性定义：心肌受刺激产生**动作电位**的能力 -> 衡量指标：**阈值**&nbsp;&nbsp;&nbsp;
- 影响兴奋性的因素&nbsp;&nbsp;&nbsp;
  - 静息电位与阈电位**差距**：差距↑ -> 所需刺激↑ -> 兴奋性**↓**&nbsp;&nbsp;&nbsp;
  - 0 期离子通道**性状**&nbsp;&nbsp;&nbsp;
    - 备用（关闭可激活）-> 激活（开放）-> 失活（关闭不可激活）-> 复活（失活→备用）&nbsp;&nbsp;&nbsp;
    - Na⁺ 通道只有在**备用**状态才能被激活&nbsp;&nbsp;&nbsp;
- 兴奋性的周期性变化（★必考）&nbsp;&nbsp;&nbsp;
  - 有效不应期（ERP）&nbsp;&nbsp;&nbsp;
    - 绝对不应期：0 期 -> 复极至 **-55mV** -> Na⁺ 通道**完全失活** -> 对任何刺激**均无反应**&nbsp;&nbsp;&nbsp;
    - 局部反应期：**-55mV → -60mV** -> Na⁺ 通道刚开始复活 -> 强刺激仅产生**局部反应** -> **不能产生 AP**&nbsp;&nbsp;&nbsp;
    - ERP = 绝对不应期 + 局部反应期 = 0 期到 **-60mV**&nbsp;&nbsp;&nbsp;
  - 相对不应期（RRP）&nbsp;&nbsp;&nbsp;
    - **-60mV → -80mV** -> Na⁺ 通道**大部分复活** -> 需**阈上刺激** -> 可产生 AP（幅度小、0 期慢）&nbsp;&nbsp;&nbsp;
  - 超常期（SNP）&nbsp;&nbsp;&nbsp;
    - **-80mV → -90mV** -> Na⁺ 通道完全**复活至备用** -> 距阈电位**近** -> **阈下刺激**即可产生 AP&nbsp;&nbsp;&nbsp;
    - 兴奋性**高于**正常 -> 但 AP 幅度和速度仍**低于**正常&nbsp;&nbsp;&nbsp;
- ERP 特别长的生理意义（★核心）&nbsp;&nbsp;&nbsp;
  - 心肌 **ERP 特别长** = 收缩期 + 舒张早期&nbsp;&nbsp;&nbsp;
  - 保证心肌**不发生完全强直收缩** -> 保持**收缩与舒张交替** -> 实现泵血功能&nbsp;&nbsp;&nbsp;
  - 骨骼肌 ERP 短 -> 可发生强直收缩 -> 维持持续张力&nbsp;&nbsp;&nbsp;
- 期前收缩与代偿间歇（★必考）&nbsp;&nbsp;&nbsp;
  - 期前收缩&nbsp;&nbsp;&nbsp;
    - 在 ERP 之后、下一次窦房结兴奋到达**之前**&nbsp;&nbsp;&nbsp;
    - 受到**额外刺激**（异位起搏点兴奋）-> 产生**提前**的一次收缩&nbsp;&nbsp;&nbsp;
  - 代偿间歇&nbsp;&nbsp;&nbsp;
    - 期前收缩后出现一段**较长**的舒张期&nbsp;&nbsp;&nbsp;
    - 原因：期前兴奋也有自己的 **ERP** -> 下一次正常窦房结冲动到达时 -> 恰好落在期前兴奋的 **ERP 内** -> **不能引起**心室兴奋 -> 需等再下一次窦房结冲动 -> 形成代偿间歇&nbsp;&nbsp;&nbsp;
- 记忆口诀&nbsp;&nbsp;&nbsp;
  - 兴奋性变化：**-55** 绝对 → **-60** 局部 → **-80** 相对 → **-90** 超常&nbsp;&nbsp;&nbsp;
  - ERP 最长 = 不强制 = 收缩舒张交替&nbsp;&nbsp;&nbsp;
  - 期前收缩 + 代偿间歇：额外刺激 → 提前跳 → 漏一跳 → 补回来&nbsp;&nbsp;&nbsp;
