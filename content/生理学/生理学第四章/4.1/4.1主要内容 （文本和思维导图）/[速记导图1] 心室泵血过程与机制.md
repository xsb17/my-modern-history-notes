# 心室泵血过程与机制 — 极简速记导图&nbsp;&nbsp;&nbsp;

- 心动周期 = 一次收缩 + 一次舒张&nbsp;&nbsp;&nbsp;
  - 与心率成**反比**：周期(s) = **60 ÷ 心率**&nbsp;&nbsp;&nbsp;
  - 心率 **75 次/min** -> 周期 = **0.8s**&nbsp;&nbsp;&nbsp;
  - 收缩期**短于**舒张期&nbsp;&nbsp;&nbsp;
  - 全心舒张期：心室舒张前 **0.4s** 心房也在舒张&nbsp;&nbsp;&nbsp;
  - 临床：心动过速 -> 舒张期缩短比例**更大** -> 冠脉供血**减少**&nbsp;&nbsp;&nbsp;
- 泵血核心动力：心脏舒缩 -> **压力差**&nbsp;&nbsp;&nbsp;
- 方向决定因素：**瓣膜启闭**（房室瓣 / 动脉瓣）&nbsp;&nbsp;&nbsp;
- 心房收缩期（周期起点，约 **0.1s**）&nbsp;&nbsp;&nbsp;
  - 心房收缩 -> 房内压**升高** > 室内压 -> 房室瓣**开**&nbsp;&nbsp;&nbsp;
  - 血液（**20%~30%**）主动挤入心室 -> 贡献充盈量约 **1/4**&nbsp;&nbsp;&nbsp;
  - 心房舒张期 + 全心舒张期 -> 血液（**70%~80%**）被动流入心室&nbsp;&nbsp;&nbsp;
- 等容收缩期&nbsp;&nbsp;&nbsp;
  - 心室肌收缩 -> 室内压**急剧↑** > 房内压 -> 房室瓣**关**&nbsp;&nbsp;&nbsp;
  - 室内压仍 < 动脉压 -> 动脉瓣仍**关**&nbsp;&nbsp;&nbsp;
  - 两瓣均**关** -> 容积**不变** -> 室内压**急剧↑**（最快上升期）&nbsp;&nbsp;&nbsp;
  - 压力：房压 < **室压** < 动脉压&nbsp;&nbsp;&nbsp;
- 快速射血期&nbsp;&nbsp;&nbsp;
  - 室内压**↑** > 动脉压 -> 动脉瓣**开**&nbsp;&nbsp;&nbsp;
  - 射血量占总射血量 **70%**（2/3）-> 室内压达**峰值**&nbsp;&nbsp;&nbsp;
  - 房室瓣**关** + 动脉瓣**开** -> 容积**迅速↓**&nbsp;&nbsp;&nbsp;
  - 压力：房压 < **室压** > 动脉压&nbsp;&nbsp;&nbsp;
- 减慢射血期&nbsp;&nbsp;&nbsp;
  - 室内压由峰值**↓** -> 射血速度**减慢** -> 靠**惯性**射血&nbsp;&nbsp;&nbsp;
  - 射血量占总射血量 **30%**（1/3）-> 容积减至**最低**（ESV）&nbsp;&nbsp;&nbsp;
  - 房室瓣**关** + 动脉瓣**开**&nbsp;&nbsp;&nbsp;
- 等容舒张期&nbsp;&nbsp;&nbsp;
  - 心室肌舒张 -> 室内压**↓** < 动脉压 -> 动脉瓣**关**&nbsp;&nbsp;&nbsp;
  - 室内压仍 > 房内压 -> 房室瓣仍**关**&nbsp;&nbsp;&nbsp;
  - 两瓣均**关** -> 容积**不变** -> 血液**不进不出**&nbsp;&nbsp;&nbsp;
  - 压力：房压 < **室压** < 动脉压&nbsp;&nbsp;&nbsp;
- 快速充盈期（**负压抽吸**）&nbsp;&nbsp;&nbsp;
  - 室内压**↓** < 房内压 -> 房室瓣**开**&nbsp;&nbsp;&nbsp;
  - 血液快速流入心室 -> 容积**迅速↑** -> 充盈量占**大部分**&nbsp;&nbsp;&nbsp;
  - 房室瓣**开** + 动脉瓣**关**&nbsp;&nbsp;&nbsp;
- 减慢充盈期&nbsp;&nbsp;&nbsp;
  - 压力梯度↓ -> 血流速度**减慢** -> 容积**继续↑**&nbsp;&nbsp;&nbsp;
  - 房室瓣**开** + 动脉瓣**关**&nbsp;&nbsp;&nbsp;
- 心房收缩期（舒张末期最后 **0.1s**）&nbsp;&nbsp;&nbsp;
  - 心房收缩 -> 充盈量再增加 **1/4** -> 容积达**最大**（EDV）&nbsp;&nbsp;&nbsp;
  - 心房 = **初级泵**&nbsp;&nbsp;&nbsp;
- 心音&nbsp;&nbsp;&nbsp;
  - S1：**房室瓣关** + 心室肌收缩振动 -> 标志**收缩期开始**&nbsp;&nbsp;&nbsp;
  - S2：**半月瓣关** -> 标志**舒张期开始**&nbsp;&nbsp;&nbsp;
- 核心口诀：缩期射血 -> 舒期充盈 -> 两等容期瓣全关容积不变&nbsp;&nbsp;&nbsp;
