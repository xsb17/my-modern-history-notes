# 躯体痛与内脏痛觉 — 微型速记导图

- 痛觉定义
  - 伤害性刺激 -> 不愉快感觉 + **情绪变化** + **防御反应** + 自主神经反应
  - = 机体受伤害的**警报**
- 躯体痛分类
  - 体表痛（皮肤）
    - **快痛**：立即产生 -> 尖锐刺痛 -> 迅速短暂 + **定位准确** -> 伴逃避反射
    - **慢痛**：较慢出现 -> 烧灼痛 -> 缓慢持久 + **定位不准** -> 伴自主神经 + 情绪反应
  - 深部痛（关节/肌肉）：一般为**慢痛**
- 快痛 vs 慢痛对比
  - 快痛：立即/尖锐/短暂/定位**准**/逃避反射
  - 慢痛：延迟/烧灼/持久/定位**不准**/自主神经+情绪
- ★内脏痛四大特征（必考）
  - ① **慢痛**为主
  - ② **定位不准确**
  - ③ 发生**缓慢** + 持续时间**较长**
  - ④ 对**牵拉/炎症/缺血**敏感 -> 对**切割/烧灼**不敏感
  - ⑤ 伴有自主神经反应 + 情绪变化
  - ⑥ 常伴**牵涉痛**
- ★牵涉痛（referred pain）
  - 定义：内脏疾患 -> 远隔**特定体表部位**发生疼痛或痛觉过敏
  - 机制：会聚学说（内脏+体表传入 -> 同一脊髓后角神经元）+ 易化学说
- ★牵涉痛常见对应表（必考选择）
  - **心**（心绞痛）-> **心前区 + 左上臂尺侧**
  - **胃、胰** -> **左上腹 + 肩胛间**
  - **肝、胆囊** -> **右肩胛**
  - **肾结石** -> **腹股沟区**
  - **阑尾炎** -> **上腹部或脐周**
- 牵涉痛诊断口诀
  - 心左臂、胆右肩、肾结石腹股沟、阑尾上腹脐周圈
