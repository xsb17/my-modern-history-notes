# 丘脑的三大核团 — 微型速记导图

- 丘脑定位：各种感觉的**中继站**（除**嗅觉**外）
- ★特异感觉接替核
  - 传入通路：
    - 三叉丘系 -> **后内侧腹核**
    - 脊髓丘脑束 -> **后外侧腹核**
    - 视觉 -> **外侧膝状体**
    - 听觉 -> **内侧膝状体**
  - 功能：产生**特定的感觉** -> 投射至大脑皮层**特定感觉区域**
  - 特点：**点对点**精确投射
- 联络核
  - 通路：感觉接替核 -> **联络核** -> 大脑皮层特定区域
    - 下丘脑 -> **丘脑前核**
    - 后腹核 -> **腹外侧核**
    - 内外侧膝状体 -> **丘脑枕**
  - 功能：协调各种感觉在**丘脑和大脑皮层之间**的联系
  - 特点：起**联络协调**作用
- ★非特异性投射核群（髓板内核群）
  - 通路：**脑干网状结构** -> 髓板内核群 -> 大脑皮层**广泛区域**
  - 包括：**中央中核** + **束旁核**
  - 功能：**维持大脑皮层的兴奋状态**
  - 特点：弥散投射，无特异性
  - 脑干网状结构**上行激活系统** -> 维持**觉醒状态**
    - 损伤 -> **昏睡不醒**
    - 易受**巴比妥类安眠药**抑制 -> 催眠
- 三大核团功能口诀
  - 特异接替 = **产生特定感觉**（点对点）
  - 联络核 = **协调联系**（中转站）
  - 非特异核群（髓板内核）= **维持皮层兴奋/觉醒**（弥散）
