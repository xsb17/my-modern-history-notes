# 10.1 课后练习题名词解释

- **突触**
  - 定义：**神经元之间**或**神经元与效应细胞**之间相互紧密联系并**传递信息**的特殊**结构部位**
  - 正常值/公式：分类 → **电突触**（缝隙连接，双向快传）+ **化学性突触**（定向/非定向）；经典突触间隙 **20~40nm**；每个神经元可形成数百~数十万个突触
  - 生理意义/特征：是实现**神经元间信息传递**的结构基础；化学性突触通过**递质释放**实现单向传递；突触可塑性是**学习和记忆**的细胞基础
- **兴奋性突触后电位（EPSP）**
  - 定义：**兴奋性递质**与突触后膜受体结合 → 后膜对 **Na⁺ 通透性↑** → Na⁺ 内流为主 → 突触后膜发生**去极化**的局部电位
  - 正常值/公式：突触前神经元为**兴奋性神经元**；递质为**兴奋性递质**；后膜状态为**去极化**
  - 生理意义/特征：使突触后神经元**兴奋性升高** → 去极化达到阈电位时可触发 **AP**；与 IPSP 的代数和决定突触后神经元最终是否兴奋；是中枢兴奋传递的基本形式
- **抑制性突触后电位（IPSP）**
  - 定义：**抑制性递质**与突触后膜受体结合 → 后膜对 **Cl⁻ 通透性↑** → Cl⁻ 内流为主 → 突触后膜发生**超极化**的局部电位
  - 正常值/公式：突触前神经元为**抑制性神经元**；递质为**抑制性递质**；后膜状态为**超极化**
  - 生理意义/特征：使突触后神经元**兴奋性降低** → 不易产生 AP；是实现**中枢抑制**（突触后抑制）的离子基础；与 EPSP 共同构成中枢神经系统**兴奋-抑制平衡**
- **神经递质**
  - 定义：由**突触前神经元合成**并在**末梢释放**，能**特异性**作用于突触后神经元或效应器细胞的**受体**，而产生一定效应的**信息传递物质**
  - 正常值/公式：分类 → 胆碱类（**ACh**）、胺类（**NA/Adr/DA/5-HT**）、氨基酸类（**谷氨酸/GABA/甘氨酸**）、肽类（阿片肽等）、嘌呤类（ATP/腺苷）、气体分子（**NO/CO**）
  - 生理意义/特征：是化学性突触传递的**核心媒介**；递质与**调质**配合（调质调节递质效应）；**递质共存**（一个神经元可释放两种以上递质）→ 协调生理活动；递质代谢异常 → 多种神经精神疾病
- **脊休克**
  - 定义：脊髓与**高位中枢离断**后，断面以下暂时丧失**反射活动**、进入**无反应状态**的现象
  - 正常值/公式：主要表现为断面以下**躯体运动反射**和**内脏反射**消失（肌张力↓、血压↓、发汗反射消失、大小便潴留）；随后反射可逐渐**恢复**（恢复快慢因动物种类而异）
  - 生理意义/特征：说明脊髓本身可完成一些**基本反射**，但正常情况下受**高位中枢调控**；脊休克的产生和恢复说明高位中枢对脊髓具有**易化作用**；临床上脊髓横断损伤后可观察到类似现象
- **骨骼肌的牵张反射**
  - 定义：有神经支配的骨骼肌受到**外力牵拉伸长**时，引起受牵拉的**同一肌肉收缩**的反射
  - 正常值/公式：分为两型 → **腱反射**（快速牵拉肌腱→单突触反射，如膝跳反射）+ **肌紧张**（缓慢持续牵拉→多突触反射，维持姿势）；感受器为**肌梭**，传入/传出均为同一肌肉
  - 生理意义/特征：是维持**姿势**的基础（肌紧张）；临床上通过检查腱反射判断**神经系统功能**状态（反射亢进→上运动神经元损伤；反射减弱→下运动神经元损伤）
- **去大脑僵直**
  - 定义：在**中脑上、下丘之间**切断脑干后，动物出现**伸肌紧张性亢进**、四肢**伸直**、头尾**昂起**、脊柱**挺硬**的现象
  - 正常值/公式：本质是**γ 僵直**→ 网状结构**抑制区**活动减弱 + **易化区**活动相对增强 → γ 运动神经元兴奋↑ → 肌梭敏感性↑ → 伸肌紧张↑
  - 生理意义/特征：说明脑干网状结构存在调节肌紧张的**抑制区**和**易化区**；正常情况下两者保持**平衡**；去大脑僵直是两者失衡（抑制区活动↓）的结果；去大脑僵直 = **γ 僵直**（经典僵直）
- **牵涉痛**
  - 定义：某些**内脏疾病**引起远隔的**体表特定部位**发生**疼痛**或**痛觉过敏**的现象
  - 正常值/公式：机制 → **会聚学说**（内脏传入与体表传入会聚于同一脊髓后角神经元）+ **易化学说**（内脏传入易化体表传入通路）
  - 生理意义/特征：有助于**定位诊断**内脏疾病（如心绞痛→**左臂尺侧**；胆囊炎→**右肩**；阑尾炎→**脐周/右下腹**；肾结石→**腹股沟**）；是临床体格检查的重要线索
- **屈肌反射**
  - 定义：肢体皮肤受到**伤害性刺激**时，受刺激一侧肢体**屈肌收缩、伸肌舒张**，使肢体**迅速回缩**的反射
  - 正常值/公式：属于**多突触反射**、**防御性反射**；与**交叉伸肌反射**协调：同侧回缩 + 对侧伸直支撑
  - 生理意义/特征：具有**保护**意义（远离伤害源）；屈肌反射的强弱可反映 CNS 兴奋性；病理情况下可出现**巴宾斯基征**（Babinski sign）→ 提示**锥体束**损伤
- **轴浆运输**
  - 定义：通过**轴浆流动**来实现**胞体**与**轴突末梢**之间**物质运输**的现象
  - 正常值/公式：分两向 → **顺向运输**（胞体→末梢：快速=递质囊泡/分泌颗粒；慢速=胞质蛋白/基质）+ **逆向运输**（末梢→胞体：内吞体/神经营养因子）
  - 生理意义/特征：维持**轴突结构和功能**的物质基础；顺向运输→递质和蛋白的**供应线**；逆向运输→反馈**信号通路**（神经营养因子反哺神经元）；病毒（**狂犬病毒**）和毒素（**破伤风毒素**）可借逆向运输**入侵 CNS**
