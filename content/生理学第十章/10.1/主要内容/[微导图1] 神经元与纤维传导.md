# 神经元与纤维传导 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 神经元基本结构（约 **1000 亿**个）&nbsp;&nbsp;&nbsp;
  - 树突：**接受信息** -> 多少不一&nbsp;&nbsp;&nbsp;
  - 胞体：**整合信息**&nbsp;&nbsp;&nbsp;
  - 始段（轴突起始部）：产生 **AP** 的起始部位&nbsp;&nbsp;&nbsp;
  - 轴突：通常只有**一个** -> **传导兴奋**&nbsp;&nbsp;&nbsp;
  - 突触小体（末梢膨大）：**释放递质**&nbsp;&nbsp;&nbsp;
- 神经纤维&nbsp;&nbsp;&nbsp;
  - 神经纤维 = 轴索 + 髓鞘或神经膜&nbsp;&nbsp;&nbsp;
  - 有髓 vs 无髓：有髓传导快（**跳跃式传导**）&nbsp;&nbsp;&nbsp;
- 神经冲动传导速度影响因素&nbsp;&nbsp;&nbsp;
  - 直径**粗** -> 快；直径细 -> 慢&nbsp;&nbsp;&nbsp;
  - 有**髓鞘** -> 快；无髓鞘 -> 慢&nbsp;&nbsp;&nbsp;
  - 温度**↓** -> 慢（**低温麻醉**原理）&nbsp;&nbsp;&nbsp;
- 神经纤维传导兴奋 4 大特征 ★必考&nbsp;&nbsp;&nbsp;
  - **生理完整性**：结构和功能必须完整&nbsp;&nbsp;&nbsp;
  - **绝缘性**：一条神经干内纤维间互不干扰&nbsp;&nbsp;&nbsp;
  - **双向性**：离体时刺激任一点 -> AP 向两端传导&nbsp;&nbsp;&nbsp;
  - **相对不疲劳性**：耗能极少 -> 不易疲劳&nbsp;&nbsp;&nbsp;
- 轴浆运输&nbsp;&nbsp;&nbsp;
  - 定义：通过轴浆流动实现**胞体 ↔ 轴突末梢**物质运输&nbsp;&nbsp;&nbsp;
  - 顺向运输（胞体 → 末梢）：&nbsp;&nbsp;&nbsp;
    - 快速：**递质囊泡**、分泌颗粒&nbsp;&nbsp;&nbsp;
    - 慢速：胞质蛋白、基质成分&nbsp;&nbsp;&nbsp;
  - 逆向运输（末梢 → 胞体）：&nbsp;&nbsp;&nbsp;
    - **神经营养因子** -> 内吞体进入胞体&nbsp;&nbsp;&nbsp;
    - 病毒（**狂犬病毒**）、毒素（**破伤风**）可借此入侵 CNS&nbsp;&nbsp;&nbsp;
- 神经的营养性作用&nbsp;&nbsp;&nbsp;
  - 神经末梢释放**营养因子** -> 调节靶组织代谢&nbsp;&nbsp;&nbsp;
  - 神经**损伤**后 -> 被支配组织**萎缩**（如坐骨神经离断）&nbsp;&nbsp;&nbsp;
  - 效应组织 + 胶质细胞 -> 产生神经营养因子 -> 反哺神经元&nbsp;&nbsp;&nbsp;
- 神经胶质细胞 6 大功能&nbsp;&nbsp;&nbsp;
  - 支持（网状支架）+ 修复（增生填充）+ 绝缘（形成髓鞘）+ 屏障（**血脑屏障**）+ 代谢营养 + 免疫&nbsp;&nbsp;&nbsp;
