# 中枢联系、传导特征与中枢抑制 — 微型速记导图

- 中枢神经元 5 种联系方式
  - 单线式：**感觉传入** -> 点对点精确传导
  - 辐散式：一个神经元 -> 多个神经元 -> 扩大影响范围
  - 聚合式：多个神经元 -> 一个神经元 -> **整合**信息
  - 链锁式：逐级传递 -> 空间/时间上扩大作用
  - 环路式：形成回路 -> 后发放的基础（**运动传出**）
- ★中枢兴奋传递 7 大特征（必考简答）
  - 单向传递：突触前膜 -> 突触后膜（不可逆）
  - 突触延搁：一次传递需 **0.3~0.5ms**（递质释放+扩散+结合受体）
  - 总和效应：EPSP/IPSP 是**局部电位** -> 时间总和 + 空间总和
  - 兴奋节律可变性：传出 ≠ 传入冲动频率
  - 后发放：刺激停止后传出仍发放 -> 环式联系
  - 敏感性高：缺 **O₂** / **CO₂** 潴留 / 酸碱 / 药物 -> 最先受影响
  - 易疲劳性：递质**耗竭** -> 突触传递疲劳（区别于神经纤维的不疲劳）
- 中枢抑制分类
  - 突触后抑制：抑制性中间神经元 -> 释放抑制性递质 -> IPSP
  - 突触前抑制：突触前膜兴奋性递质释放**减少** -> EPSP 降低
- ★传入侧支性抑制（交互抑制）
  - 机制：传入纤维 ->
    - ① 兴奋一个中枢神经元
    - ② 经**侧支**兴奋**抑制性中间神经元** -> 抑制另一个中枢神经元
  - 意义：使**拮抗中枢**活动协调 -> 屈肌收缩时伸肌舒张
  - 实例：屈肌反射、伸肌反射
- ★回返性抑制
  - 机制：神经元兴奋沿轴突外传 ->
    - 经**轴突侧支**兴奋**闰绍细胞**（抑制性中间神经元）
    - -> 闰绍细胞释放**甘氨酸**
    - -> 反过来**抑制**原先兴奋的神经元
  - 意义：
    - **及时终止**运动神经元活动（防止过度兴奋）
    - 促使神经元间活动**同步化**
- 突触后抑制 vs 回返性抑制对比
  - 传入侧支性：传入兴奋 -> 侧支 -> 抑中间N -> 抑另一中枢 -> **协调拮抗**
  - 回返性：传出兴奋 -> 侧支 -> 闰绍细胞(**甘氨酸**) -> 抑原N -> **终止/同步**
- 中枢 vs 神经纤维传导对比
  - 中枢：单向/延搁/总和/可变/后发放/敏感/易疲劳
  - 纤维：双向/不延搁/无总和/保真/无后发放/不敏感/不疲劳
