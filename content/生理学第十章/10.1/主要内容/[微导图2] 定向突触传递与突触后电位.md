# 定向突触传递与突触后电位 — 微型速记导图

- 突触分类
  - 电突触：**缝隙连接** -> 双向、快
  - 化学性突触：定向（经典）+ 非定向
- 经典突触基本结构
  - 突触前膜：含囊泡 -> **释放递质**
  - 突触间隙：**20~40nm**
  - 突触后膜：含**受体**
  - 囊泡三种：
    - 小而清亮透明 -> **ACh**、氨基酸
    - 小有致密中心 -> 儿茶酚胺类
    - 大有致密中心 -> 肽类
- 经典突触分类（按接触部位）
  - **轴-树**（最多见）、轴-体、轴-轴、树-树、树-体、树-轴
- ★经典突触传递过程（核心大题）
  - 突触前神经元兴奋 -> 冲动传至末梢
  - -> 突触前膜**去极化** -> 电压门控 **Ca²⁺ 通道开放**
  - -> **Ca²⁺ 内流** -> 囊泡动员/摆渡/锚定/融合
  - -> 递质**量子式释放**（出胞）
  - -> 递质 + 突触后膜**受体**
  - -> 后膜离子通透性改变 -> **EPSP** 或 **IPSP**
- ★兴奋性突触后电位（EPSP）
  - 兴奋性递质 + 受体 -> 后膜 **Na⁺ 内流为主**
  - -> 后膜**去极化** -> **EPSP**
  - -> 突触后神经元兴奋性**↑**
- ★抑制性突触后电位（IPSP）
  - 抑制性递质 + 受体 -> 后膜 **Cl⁻ 内流为主**
  - -> 后膜**超极化** -> **IPSP**
  - -> 突触后神经元兴奋性**↓**
- EPSP vs IPSP 对比口诀
  - EPSP：兴奋递质 -> **Na⁺进** -> **去极化** -> 兴奋↑
  - IPSP：抑制递质 -> **Cl⁻进** -> **超极化** -> 兴奋↓
- 影响突触传递的因素 ★必考
  - 递质释放：
    - **Ca²⁺ 内流↑** -> 递质释放↑ -> 传递增强
    - 肉毒杆菌毒素 -> 阻止 **ACh 释放** -> 肌肉**麻痹**（美容除皱）
  - 递质消除：
    - 有机磷农药 -> 抑制**胆碱酯酶** -> ACh 蓄积 -> 肌肉**痉挛/震颤**
  - 受体阻断：
    - 筒箭毒碱/α-银环蛇毒 -> 阻断 **N₂ 受体** -> 神经-肌接头受阻 -> **肌肉松弛/麻痹**
- 突触后整合
  - 突触后膜电位 = **EPSP + IPSP 的代数和**
  - 去极化达阈电位 -> **触发 AP** -> 兴奋
  - 超极化 -> **抑制**
