# 定向突触传递与突触后电位 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 突触分类&nbsp;&nbsp;&nbsp;
  - 电突触：**缝隙连接** -> 双向、快&nbsp;&nbsp;&nbsp;
  - 化学性突触：定向（经典）+ 非定向&nbsp;&nbsp;&nbsp;
- 经典突触基本结构&nbsp;&nbsp;&nbsp;
  - 突触前膜：含囊泡 -> **释放递质**&nbsp;&nbsp;&nbsp;
  - 突触间隙：**20~40nm**&nbsp;&nbsp;&nbsp;
  - 突触后膜：含**受体**&nbsp;&nbsp;&nbsp;
  - 囊泡三种：&nbsp;&nbsp;&nbsp;
    - 小而清亮透明 -> **ACh**、氨基酸&nbsp;&nbsp;&nbsp;
    - 小有致密中心 -> 儿茶酚胺类&nbsp;&nbsp;&nbsp;
    - 大有致密中心 -> 肽类&nbsp;&nbsp;&nbsp;
- 经典突触分类（按接触部位）&nbsp;&nbsp;&nbsp;
  - **轴-树**（最多见）、轴-体、轴-轴、树-树、树-体、树-轴&nbsp;&nbsp;&nbsp;
- ★经典突触传递过程（核心大题）&nbsp;&nbsp;&nbsp;
  - 突触前神经元兴奋 -> 冲动传至末梢&nbsp;&nbsp;&nbsp;
  - -> 突触前膜**去极化** -> 电压门控 **Ca²⁺ 通道开放**&nbsp;&nbsp;&nbsp;
  - -> **Ca²⁺ 内流** -> 囊泡动员/摆渡/锚定/融合&nbsp;&nbsp;&nbsp;
  - -> 递质**量子式释放**（出胞）&nbsp;&nbsp;&nbsp;
  - -> 递质 + 突触后膜**受体**&nbsp;&nbsp;&nbsp;
  - -> 后膜离子通透性改变 -> **EPSP** 或 **IPSP**&nbsp;&nbsp;&nbsp;
- ★兴奋性突触后电位（EPSP）&nbsp;&nbsp;&nbsp;
  - 兴奋性递质 + 受体 -> 后膜 **Na⁺ 内流为主**&nbsp;&nbsp;&nbsp;
  - -> 后膜**去极化** -> **EPSP**&nbsp;&nbsp;&nbsp;
  - -> 突触后神经元兴奋性**↑**&nbsp;&nbsp;&nbsp;
- ★抑制性突触后电位（IPSP）&nbsp;&nbsp;&nbsp;
  - 抑制性递质 + 受体 -> 后膜 **Cl⁻ 内流为主**&nbsp;&nbsp;&nbsp;
  - -> 后膜**超极化** -> **IPSP**&nbsp;&nbsp;&nbsp;
  - -> 突触后神经元兴奋性**↓**&nbsp;&nbsp;&nbsp;
- EPSP vs IPSP 对比口诀&nbsp;&nbsp;&nbsp;
  - EPSP：兴奋递质 -> **Na⁺进** -> **去极化** -> 兴奋↑&nbsp;&nbsp;&nbsp;
  - IPSP：抑制递质 -> **Cl⁻进** -> **超极化** -> 兴奋↓&nbsp;&nbsp;&nbsp;
- 影响突触传递的因素 ★必考&nbsp;&nbsp;&nbsp;
  - 递质释放：&nbsp;&nbsp;&nbsp;
    - **Ca²⁺ 内流↑** -> 递质释放↑ -> 传递增强&nbsp;&nbsp;&nbsp;
    - 肉毒杆菌毒素 -> 阻止 **ACh 释放** -> 肌肉**麻痹**（美容除皱）&nbsp;&nbsp;&nbsp;
  - 递质消除：&nbsp;&nbsp;&nbsp;
    - 有机磷农药 -> 抑制**胆碱酯酶** -> ACh 蓄积 -> 肌肉**痉挛/震颤**&nbsp;&nbsp;&nbsp;
  - 受体阻断：&nbsp;&nbsp;&nbsp;
    - 筒箭毒碱/α-银环蛇毒 -> 阻断 **N₂ 受体** -> 神经-肌接头受阻 -> **肌肉松弛/麻痹**&nbsp;&nbsp;&nbsp;
- 突触后整合&nbsp;&nbsp;&nbsp;
  - 突触后膜电位 = **EPSP + IPSP 的代数和**&nbsp;&nbsp;&nbsp;
  - 去极化达阈电位 -> **触发 AP** -> 兴奋&nbsp;&nbsp;&nbsp;
  - 超极化 -> **抑制**&nbsp;&nbsp;&nbsp;
