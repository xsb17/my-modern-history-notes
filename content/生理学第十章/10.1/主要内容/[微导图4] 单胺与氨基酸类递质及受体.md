# 单胺与氨基酸类递质及受体 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 外周肾上腺素能纤维（释放 **NA**）&nbsp;&nbsp;&nbsp;
  - 绝大多数**交感神经节后纤维**&nbsp;&nbsp;&nbsp;
- 肾上腺素能受体分类&nbsp;&nbsp;&nbsp;
  - **α 受体**：α₁、α₂&nbsp;&nbsp;&nbsp;
  - **β 受体**：β₁、β₂、β₃&nbsp;&nbsp;&nbsp;
- ★α₁ 受体分布 + 效应 + 阻断剂&nbsp;&nbsp;&nbsp;
  - 血管平滑肌（腹腔内脏）-> **收缩** -> 外周阻力↑&nbsp;&nbsp;&nbsp;
  - 瞳孔开大肌 -> **收缩** -> 瞳孔**散大**&nbsp;&nbsp;&nbsp;
  - 子宫（已孕）-> **收缩**&nbsp;&nbsp;&nbsp;
  - 阻断剂：**酚妥拉明**&nbsp;&nbsp;&nbsp;
- ★α₂ 受体分布 + 效应&nbsp;&nbsp;&nbsp;
  - 突触前膜 -> 调节**递质释放**（负反馈抑制）&nbsp;&nbsp;&nbsp;
  - 小肠平滑肌 -> **舒张**&nbsp;&nbsp;&nbsp;
- ★β₁ 受体分布 + 效应&nbsp;&nbsp;&nbsp;
  - **心脏组织** -> **兴奋** -> 心率↑ + 收缩力↑ + 传导↑（**强心**）&nbsp;&nbsp;&nbsp;
  - 阻断剂：**普萘洛尔（心得安）**&nbsp;&nbsp;&nbsp;
- ★β₂ 受体分布 + 效应&nbsp;&nbsp;&nbsp;
  - 支气管平滑肌 -> **舒张**&nbsp;&nbsp;&nbsp;
  - 胃肠平滑肌、膀胱逼尿肌 -> **舒张**&nbsp;&nbsp;&nbsp;
  - 血管平滑肌（骨骼肌）-> **舒张** -> 血流量↑&nbsp;&nbsp;&nbsp;
  - 阻断剂：普萘洛尔（心得安）&nbsp;&nbsp;&nbsp;
- 多巴胺及受体&nbsp;&nbsp;&nbsp;
  - 分布：**黑质-纹状体**、中脑-边缘前脑、结节-漏斗&nbsp;&nbsp;&nbsp;
  - 功能：躯体运动、精神情绪、垂体内分泌调节&nbsp;&nbsp;&nbsp;
  - 黑质-纹状体多巴胺↓ -> **帕金森病**&nbsp;&nbsp;&nbsp;
- 5-羟色胺（5-HT）及受体&nbsp;&nbsp;&nbsp;
  - 功能：**痛觉**、**情绪**、**睡眠**调节&nbsp;&nbsp;&nbsp;
- 组胺及受体&nbsp;&nbsp;&nbsp;
  - H₁/H₂/H₃，组胺 + **H₂ 受体** -> 壁细胞 -> **胃酸分泌↑**&nbsp;&nbsp;&nbsp;
- ★氨基酸类递质&nbsp;&nbsp;&nbsp;
  - 兴奋性：**谷氨酸**、门冬氨酸&nbsp;&nbsp;&nbsp;
  - 抑制性：**GABA**（γ-氨基丁酸）、**甘氨酸**&nbsp;&nbsp;&nbsp;
- 肽类递质&nbsp;&nbsp;&nbsp;
  - 阿片肽：**镇痛**&nbsp;&nbsp;&nbsp;
  - 下丘脑调节性多肽、脑-肠肽&nbsp;&nbsp;&nbsp;
- 嘌呤类递质：**腺苷、ATP**&nbsp;&nbsp;&nbsp;
- 气体分子类递质：**NO、CO**&nbsp;&nbsp;&nbsp;
- 肾上腺素能受体核心口诀&nbsp;&nbsp;&nbsp;
  - α₁ = **血管缩**/瞳孔大（酚妥拉明）&nbsp;&nbsp;&nbsp;
  - α₂ = **突触前**负反馈&nbsp;&nbsp;&nbsp;
  - β₁ = **心脏兴**/强心（普萘洛尔）&nbsp;&nbsp;&nbsp;
  - β₂ = **支气扩**/血管舒（普萘洛尔）&nbsp;&nbsp;&nbsp;
