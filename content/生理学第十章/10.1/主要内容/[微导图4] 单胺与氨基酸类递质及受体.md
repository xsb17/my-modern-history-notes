# 单胺与氨基酸类递质及受体 — 微型速记导图

- 外周肾上腺素能纤维（释放 **NA**）
  - 绝大多数**交感神经节后纤维**
- 肾上腺素能受体分类
  - **α 受体**：α₁、α₂
  - **β 受体**：β₁、β₂、β₃
- ★α₁ 受体分布 + 效应 + 阻断剂
  - 血管平滑肌（腹腔内脏）-> **收缩** -> 外周阻力↑
  - 瞳孔开大肌 -> **收缩** -> 瞳孔**散大**
  - 子宫（已孕）-> **收缩**
  - 阻断剂：**酚妥拉明**
- ★α₂ 受体分布 + 效应
  - 突触前膜 -> 调节**递质释放**（负反馈抑制）
  - 小肠平滑肌 -> **舒张**
- ★β₁ 受体分布 + 效应
  - **心脏组织** -> **兴奋** -> 心率↑ + 收缩力↑ + 传导↑（**强心**）
  - 阻断剂：**普萘洛尔（心得安）**
- ★β₂ 受体分布 + 效应
  - 支气管平滑肌 -> **舒张**
  - 胃肠平滑肌、膀胱逼尿肌 -> **舒张**
  - 血管平滑肌（骨骼肌）-> **舒张** -> 血流量↑
  - 阻断剂：普萘洛尔（心得安）
- 多巴胺及受体
  - 分布：**黑质-纹状体**、中脑-边缘前脑、结节-漏斗
  - 功能：躯体运动、精神情绪、垂体内分泌调节
  - 黑质-纹状体多巴胺↓ -> **帕金森病**
- 5-羟色胺（5-HT）及受体
  - 功能：**痛觉**、**情绪**、**睡眠**调节
- 组胺及受体
  - H₁/H₂/H₃，组胺 + **H₂ 受体** -> 壁细胞 -> **胃酸分泌↑**
- ★氨基酸类递质
  - 兴奋性：**谷氨酸**、门冬氨酸
  - 抑制性：**GABA**（γ-氨基丁酸）、**甘氨酸**
- 肽类递质
  - 阿片肽：**镇痛**
  - 下丘脑调节性多肽、脑-肠肽
- 嘌呤类递质：**腺苷、ATP**
- 气体分子类递质：**NO、CO**
- 肾上腺素能受体核心口诀
  - α₁ = **血管缩**/瞳孔大（酚妥拉明）
  - α₂ = **突触前**负反馈
  - β₁ = **心脏兴**/强心（普萘洛尔）
  - β₂ = **支气扩**/血管舒（普萘洛尔）
