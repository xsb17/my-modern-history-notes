# 第十章第三至六节：躯体/内脏运动与高级功能 — 全集存档

## 第三节 神经系统对躯体运动的调控

### 一、运动的分类

- 反射运动：最简单、最基本，不受意识控制（如膝跳反射）
- 随意运动：由主观意愿发动，较复杂
- 节律运动：可自动重复，可受意识控制（如呼吸运动）

### 二、运动调控的三级水平

- 大脑皮层联络区、基底神经节、皮层小脑：运动的设计
- 大脑皮层运动区、脊髓小脑：运动的实施与协调
- 脑干、脊髓：运动的执行
- 躯体运动最高级中枢：大脑皮层；最基本中枢：脊髓

---

### 三、脊髓对躯体运动的调节

#### （一）脊髓运动神经元

- α 运动神经元：支配**梭外肌纤维**（骨骼肌）
- γ 运动神经元：支配**梭内肌纤维**（肌梭）
- β 运动神经元：功能不明
- 运动单位：一个 α 运动神经元 + 其所支配的全部肌纤维
  - 小运动单位：精细运动（如眼外肌）
  - 大运动单位：产生巨大肌张力（如四肢肌）

#### （二）牵张反射（stretch reflex）

- 定义：有神经支配的骨骼肌受外力牵拉伸长时，引起受牵拉的**同一肌肉收缩**的反射

##### 1. 牵张反射类型

**（1）腱反射**
- 快速牵拉肌腱 → 被牵拉肌肉快速而明显的收缩（如膝跳反射）
- 特点：**单突触反射**、收缩力强、迅速短暂、快肌纤维收缩
- 意义：临床**神经检查**

**（2）肌紧张**
- 缓慢持续牵拉肌腱 → 肌肉处于持续的轻度收缩状态
- 表现：受牵拉肌肉持续轻度收缩
- 特点：收缩缓慢持久不易疲劳、同一肌肉不同运动单位**交替收缩**、慢肌纤维收缩、**伸肌**表现明显
- 生理意义：**维持躯体姿势**，是最基本的反射

##### 2. 牵张反射的反射弧

- 感受器：**肌梭**（长度感受器）
  - 梭内肌：中间感受装置 + 两端收缩部分
- 传入纤维：**Ⅰa、Ⅱ类纤维**
- 中枢：脊髓前角 **α、γ 运动神经元**
- 传出纤维：α 传出纤维→梭外肌；γ 传出纤维→梭内肌
- 过程：骨骼肌受外力牵拉变长 → 肌梭被拉长 → Ⅰa/Ⅱ类纤维传入冲动↑ → 脊髓前角 α 运动神经元活动↑ → 梭外肌收缩

##### 3. γ 环路

- 高位中枢 → γ 传出冲动↑ → 梭内肌纤维收缩 → 对感受装置的牵拉↑ → Ⅰa 纤维传入冲动↑ → α 运动神经元活动↑ → 梭外肌收缩加强 → **提高肌梭敏感性**

#### （三）反牵张反射

- 感受器：**腱器官**（张力感受器）
- 机制：梭外肌收缩 → 张力↑ → 腱器官兴奋 → Ⅰb 传入↑ → 抑制性中间神经元 → α 运动神经元抑制 → **抑制牵张反射** → 避免肌肉损伤

#### （四）屈肌反射与对侧伸肌反射

- 屈肌反射：一侧肢体受伤害性刺激 → 同侧屈肌收缩/伸肌舒张 → 肢体屈曲 → **躲避伤害**（不属于姿势反射）
- 对侧伸肌反射：伤害性刺激较强 → 同侧屈曲 + 对侧伸直 → **保持躯体平衡**（属于姿势反射）

#### （五）脊休克

- 定义：脊髓与高位中枢离断后，断面以下脊髓反射活动**暂时丧失**、进入无反应状态
- 表现：横断面以下躯体和内脏反射均减退消失 → 肌紧张↓、血压↓、粪尿潴留等
- 恢复：越高等恢复越慢（蛙几分钟、犬几天、人**数周**以上）→ 知觉和随意运动**永久丧失**，内脏反射部分恢复（大小便失禁）
- 原因：断面以下脊髓突然**失去高位中枢的调控**

### 四、脑干对肌紧张的调节

#### （一）脑干网状结构

- 易化区：范围**广、强** → 加强伸肌肌紧张
- 抑制区：范围**小、弱** → 肌张力降低

#### （二）上级中枢对脑干的作用

- 抑制作用：大脑皮层运动区、纹状体、小脑前叶蚓部
- 易化作用：前庭核、小脑前叶两侧部、后叶中间部

#### （三）去大脑僵直

- 现象：中脑上、下丘之间切断脑干 → 全身**伸肌肌紧张亢进** → 四肢伸直、头尾昂起、脊柱挺硬（**角弓反张**）
- 机制：切断较多抑制系统（来自大脑皮层、纹状体等）→ **易化区活动 >> 抑制区活动** → 牵张反射增强
- 类型：α 僵直（前庭脊髓束→α 运动神经元↑）、γ 僵直（网状脊髓束→γ 运动神经元↑→肌梭敏感性↑）

---

### 五、基底核对躯体运动的调节

- 组成：纹状体（尾状核+壳核）、苍白球、丘脑底核、中脑黑质/红核
- 主要功能：调节**随意运动**和**肌紧张**

#### 基底神经节损伤相关疾病

- 帕金森病（震颤麻痹）：
  - 病变：**黑质**受损 → 多巴胺能神经元↓
  - 症状：**肌紧张增强**、随意运动减少、静止性震颤、面具脸
- 亨庭顿病（舞蹈病）：
  - 病变：**纹状体**受损 → 胆碱能/GABA 能神经元↓ → 多巴胺相对亢进
  - 症状：**肌紧张减弱**、不自主的舞蹈样动作

---

### 六、小脑对躯体运动的调节

#### 小脑三大分区与功能

- 前庭小脑（绒球小结叶）：**维持身体平衡**
  - 损伤 → 平衡障碍、步态蹒跚、眼球震颤
- 脊髓小脑（前叶/后叶中间带）：**调节肌紧张** + **协调随意运动**
  - 损伤 → 意向性震颤、指鼻不稳、醉酒样步态（**小脑性共济失调**）
- 皮层小脑（后叶外侧部）：参与运动**设计**和**编制程序**

---

### 七、大脑皮层对躯体运动的调节

- 主要位置：**中央前回**（4区）、运动前区（6区）
- 投射规律：**左右交叉**（上面部双侧支配）、**上下倒置**、定位精细、运动区大小 ∝ 运动精细程度
- 运动传导系统：
  - 皮层脊髓侧束（80%）：交叉，控制**精细运动**
  - 皮层脊髓前束（20%）：不交叉，控制**粗略运动**

## 第四节 神经系统对内脏活动的调节

### 一、自主神经系统

#### （一）结构特征对比

- 交感神经：
  - 起源：脊髓 **T₁~L₃** 灰质侧角
  - 节前纤维**短**、节后纤维**长**
  - 节前:节后 = 1:多个（**弥散**效应）
  - 分布**广泛**，几乎支配所有器官
- 副交感神经：
  - 起源：脑干副交感神经核 + 脊髓 **S₂₋₄** 灰质侧角
  - 节前纤维**长**、节后纤维**短**
  - 节前:节后 = 1:1~2（**局限**效应）
  - 分布较**局限**

#### （二）自主神经主要功能

- 交感神经：心脏**兴奋**、大部分血管**收缩**、支气管**舒张**、胃肠运动**减弱**、抑制排尿、瞳孔**扩大**、竖毛肌收缩/汗腺分泌、促进肾上腺髓质分泌
- 副交感神经：心脏**抑制**、部分血管舒张、支气管**收缩**、胃肠运动**增强**、促进排尿、瞳孔**缩小**、睫状肌收缩、促进胰岛素分泌

#### （三）自主神经功能特征

- 紧张性作用：持续发放低频冲动，维持效应器基础活动状态
- 双重支配、相互拮抗（汗腺/竖毛肌等只受交感支配；唾液腺为协同）
- 受效应器功能状态影响（有孕子宫 α₁→收缩；未孕子宫 β₂→舒张）
- 整体调节意义：
  - 交感-肾上腺髓质系统：**应急/应激**，动员器官储备力
  - 副交感-胰岛素系统：**休整/恢复**，促进消化吸收、积蓄能量

---

### 二、各级中枢对内脏活动的调节

#### （一）脊髓

- 初级排尿中枢、初级排便中枢

#### （二）脑干

- 延髓**生命中枢**：心血管基本中枢 + 呼吸基本中枢

#### （三）下丘脑（6 大功能）

- 体温调节：PO/AH（视前区-下丘脑前部）→ 体温调节中枢
- 摄食调节：外侧区→**摄食中枢**；腹内侧核→**饱中枢**
- 水平衡调节：外侧区→口渴中枢；视上核/室旁核→**ADH** 分泌
- 内分泌调节：调节腺垂体激素分泌
- 情绪调节：刺激下丘脑 → **假怒**
- 生物节律调控：**视交叉上核** → 生物钟

#### （四）大脑皮层

- 边缘系统 → 内脏脑 → 情绪、记忆

## 第五节 脑电活动

### 脑电图（EEG）

- 定义：用仪器在头皮表面记录到的自发脑电活动

#### 四种基本脑电波形

- **δ 波**：频率 **0.5~3 Hz**，振幅 **20~200 μV** → 成人深睡、婴儿、深度麻醉/缺氧/器质性病变 → 颞叶/枕叶
- **θ 波**：频率 **4~7 Hz**，振幅 **100~150 μV** → 少年主要成分、成人困倦/精神压抑 → 颞叶/顶叶（CNS 抑制状态）
- **α 波**：频率 **8~13 Hz**，振幅 **20~100 μV** → 闭目静息时明显，呈梭形波群；睁眼/受刺激时消失→出现 β 波（**α-阻断**）→ 枕叶
- **β 波**：频率 **14~30 Hz**，振幅 **5~20 μV** → 清醒时主要成分，积极精神活动时 → 额叶/顶叶

## 第六节 脑的高级功能

### 一、学习和记忆

- 条件反射的建立：非条件刺激（食物）+ 无关刺激（铃声）→ 强化 → 条件刺激 → 条件反射
- 意义：提高机体对**环境变化的适应能力**

### 二、人类语言功能

#### 大脑皮层语言中枢与失语症

- 写语言中枢：额中回后部受损 → **失写症**（能听会说能读，不会写）
- 读语言中枢：角回受损 → **失读症**（能听说写，看不懂文字）
- 说语言中枢（运动语言中枢）：额下回后部（Broca 区）受损 → **运动失语症**（能看懂听懂，不能说）
- 听语言中枢（感觉语言中枢）：颞上回后部（Wernicke 区）受损 → **感觉失语症**（能说能写，听不懂）
