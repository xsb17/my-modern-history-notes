# 脑干、基底核与小脑的运动调节 — 微型速记导图

- 运动调控三级水平
  - 设计：皮层联络区 + **基底神经节** + **皮层小脑**
  - 实施/协调：皮层运动区 + **脊髓小脑**
  - 执行：**脑干 + 脊髓**
  - 最高级中枢：大脑皮层；最基本中枢：脊髓
- 脑干网状结构调节肌紧张
  - 易化区：范围**广、强** -> 加强伸肌肌紧张
  - 抑制区：范围**小、弱** -> 肌张力降低
  - 上级抑制：大脑皮层运动区、纹状体、小脑前叶蚓部
  - 上级易化：前庭核、小脑前叶两侧部、后叶中间部
- ★去大脑僵直（必考）
  - 定位：**中脑上、下丘之间**切断脑干
  - 现象：全身**伸肌肌紧张亢进** -> 四肢伸直/头尾昂起/脊柱挺硬 -> **角弓反张**
  - 机制：切除较多**抑制系统** -> **易化区 >> 抑制区** -> 牵张反射增强
  - 类型：α 僵直 + **γ 僵直**（经典僵直 -> 网状脊髓束→γ↑→肌梭敏感↑）
- ★基底神经节损伤两大疾病
  - 帕金森病（震颤麻痹）：
    - 病变：**黑质**受损 -> **多巴胺能神经元↓**
    - 症状：**肌紧张↑** + 随意运动**减少** + 静止性震颤 + 面具脸
  - 亨庭顿病（舞蹈病）：
    - 病变：**纹状体**受损 -> 胆碱能/GABA 能↓ -> 多巴胺相对**亢进**
    - 症状：**肌紧张↓** + 不自主**舞蹈样**动作
- 小脑三大分区与功能
  - 前庭小脑（绒球小结叶）：
    - 功能：**维持身体平衡**
    - 损伤：平衡障碍、步态蹒跚、**眼球震颤**
  - 脊髓小脑（前叶/后叶中间带）：
    - 功能：**调节肌紧张** + **协调随意运动**
    - 损伤：意向性震颤、指鼻不稳、醉酒步态 -> **小脑性共济失调**
  - 皮层小脑（后叶外侧部）：
    - 功能：运动**设计**和**编制程序**
- 大脑皮层运动区（中央前回 4 区）
  - 规律：**左右交叉**（上面部双侧）+ **上下倒置** + 面积 ∝ 运动**精细程度**
  - 传导束：皮层脊髓侧束（**80%** 交叉/精细）+ 前束（**20%** 不交叉/粗略）
