# 下丘脑对内脏的调节 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 下丘脑定位：较高级的**内脏调节中枢**&nbsp;&nbsp;&nbsp;
- ★下丘脑 6 大核心功能（必考）&nbsp;&nbsp;&nbsp;
  - ① 体温调节：&nbsp;&nbsp;&nbsp;
    - 中枢：**PO/AH**（视前区-下丘脑前部）&nbsp;&nbsp;&nbsp;
    - 感受温度变化 -> 调节产热/散热&nbsp;&nbsp;&nbsp;
  - ② 摄食调节：&nbsp;&nbsp;&nbsp;
    - **外侧区** = **摄食中枢**（破坏→拒食/消瘦）&nbsp;&nbsp;&nbsp;
    - **腹内侧核** = **饱中枢**（破坏→贪食/肥胖）&nbsp;&nbsp;&nbsp;
  - ③ 水平衡调节：&nbsp;&nbsp;&nbsp;
    - 外侧区：**口渴中枢**（刺激→饮水↑）&nbsp;&nbsp;&nbsp;
    - 视上核/室旁核：分泌 **ADH**（抗利尿激素）-> 促进肾水重吸收&nbsp;&nbsp;&nbsp;
  - ④ 内分泌调节：&nbsp;&nbsp;&nbsp;
    - 调节**腺垂体**激素分泌（释放/抑制激素）-> 控制甲状腺/肾上腺/性腺轴&nbsp;&nbsp;&nbsp;
  - ⑤ 情绪调节：&nbsp;&nbsp;&nbsp;
    - 刺激下丘脑情绪反应区 -> **假怒**（sham rage）&nbsp;&nbsp;&nbsp;
    - 与边缘系统协同调控&nbsp;&nbsp;&nbsp;
  - ⑥ 生物节律调控：&nbsp;&nbsp;&nbsp;
    - **视交叉上核** = **生物钟**&nbsp;&nbsp;&nbsp;
    - 调控睡眠-觉醒周期/体温/激素分泌昼夜节律&nbsp;&nbsp;&nbsp;
- 下丘脑核团与功能对应口诀&nbsp;&nbsp;&nbsp;
  - PO/AH = 体温调节&nbsp;&nbsp;&nbsp;
  - 外侧区 = 摄食中枢 + 口渴中枢（外侧 = 饿=渴=向外索取）&nbsp;&nbsp;&nbsp;
  - 腹内侧核 = 饱中枢（内侧 = 饱=向内知足）&nbsp;&nbsp;&nbsp;
  - 视上核/室旁核 = ADH（保水）&nbsp;&nbsp;&nbsp;
  - 视交叉上核 = 生物钟（视交叉=看昼夜）&nbsp;&nbsp;&nbsp;
