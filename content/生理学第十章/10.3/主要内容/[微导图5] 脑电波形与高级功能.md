# 脑电波形与高级功能 — 微型速记导图

- 脑电图（EEG）定义
  - 用仪器在**头皮表面**记录到的**自发脑电活动**
- ★四种正常脑电波形（必考选择）
  - δ 波：
    - 频率：**0.5~3 Hz** | 振幅：**20~200 μV**
    - 状态：成人**深睡**、婴儿、深度麻醉/缺氧/器质性病变
    - 部位：颞叶/枕叶
  - θ 波：
    - 频率：**4~7 Hz** | 振幅：**100~150 μV**
    - 状态：少年主要成分、成人**困倦/精神压抑** -> CNS 抑制状态
    - 部位：颞叶/顶叶
  - α 波：
    - 频率：**8~13 Hz** | 振幅：**20~100 μV**
    - 状态：**闭目静息**时明显（梭形波群）-> 睁眼/受刺激 -> α 消失→β 出现（**α-阻断**）
    - 部位：枕叶
  - β 波：
    - 频率：**14~30 Hz** | 振幅：**5~20 μV**
    - 状态：**清醒时主要成分**、积极精神活动时
    - 部位：额叶/顶叶
- 脑电波形快速记忆
  - δ=深睡（0.5-3/高振幅）、θ=困倦（4-7）、α=闭目静息（8-13）、β=清醒活跃（14-30/低振幅）
  - 频率越高 -> 振幅越低 -> 大脑越"清醒"
  - α-阻断：睁眼 -> α 消失换 β = 皮层激活
- 条件反射建立
  - 非条件刺激（食物）+ 无关刺激（铃声）-> **强化** -> 无关变**条件刺激** -> 条件反射
  - 意义：提高机体对**环境变化的适应能力**
- ★大脑皮层四大语言中枢与失语症
  - 写语言中枢（额中回后部受损）：
    - 症状：**失写症** -> 能听会说能读，**不会写**
  - 读语言中枢（角回受损）：
    - 症状：**失读症** -> 能听说写，**看不懂文字**
  - 说语言中枢/运动语言中枢（额下回后部/**Broca 区**受损）：
    - 症状：**运动失语症** -> 能看懂听懂，**不能说**
  - 听语言中枢/感觉语言中枢（颞上回后部/**Wernicke 区**受损）：
    - 症状：**感觉失语症** -> 能说能写，**听不懂**
- 语言中枢口诀
  - 额中写、角回读、Broca 说、Wernicke 听
  - 写=手、读=眼、说=嘴、听=耳 -> 部位与功能对应
