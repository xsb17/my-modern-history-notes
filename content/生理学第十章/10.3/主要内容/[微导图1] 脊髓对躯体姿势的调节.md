# 脊髓对躯体姿势的调节 — 微型速记导图（大题常考）

- 脊髓运动神经元
  - α 运动神经元：支配**梭外肌纤维**（骨骼肌）
  - γ 运动神经元：支配**梭内肌纤维**（肌梭）
  - 运动单位 = 一个 **α 神经元** + 全部支配肌纤维
    - 小单位：**精细运动**（眼外肌）
    - 大单位：巨大**肌张力**（四肢肌）
- ★牵张反射定义
  - 骨骼肌受**外力牵拉伸长** -> 引起**同一肌肉收缩**的反射
- ★牵张反射两大类型对比（必考大题）
  - 腱反射：
    - 快速牵拉肌腱 -> 快速明显收缩（膝跳反射）
    - 特点：**单突触反射**、收缩力强、迅速短暂、**快肌纤维**收缩
    - 意义：临床**神经系统检查**
  - 肌紧张：
    - 缓慢持续牵拉肌腱 -> 持续轻度收缩
    - 特点：缓慢持久**不易疲劳**、不同运动单位**交替收缩**、**慢肌纤维**收缩、**伸肌**表现明显
    - 意义：**维持躯体姿势**（最基本反射）
- ★牵张反射反射弧（五段式）
  - 感受器：**肌梭**（长度感受器）-> 梭内肌中间感受装置
  - 传入纤维：**Ⅰa、Ⅱ类纤维**
  - 中枢：脊髓前角 **α、γ 运动神经元**
  - 传出纤维：α 传出 -> 梭外肌；γ 传出 -> 梭内肌
  - 效应器：同一肌肉的**梭外肌**
- ★γ 环路（提高肌梭敏感性）
  - 高位中枢 -> **γ 传出↑** -> 梭内肌收缩 -> 感受装置牵拉↑ -> Ⅰa 传入↑ -> **α 活动↑** -> 梭外肌收缩↑
- 反牵张反射（保护机制）
  - 感受器：**腱器官**（张力感受器）
  - 梭外肌收缩 -> 张力↑ -> 腱器官兴奋 -> Ⅰb 传入↑ -> 抑制 α 神经元 -> 抑制牵张反射 -> 避免**肌肉损伤**
- 屈肌反射与对侧伸肌反射
  - 屈肌反射：伤害性刺激 -> 同侧**屈肌收缩/伸肌舒张** -> 肢体回缩 -> **躲避伤害**（非姿势反射）
  - 对侧伸肌反射：强刺激 -> 同侧屈 + **对侧伸** -> **保持躯体平衡**（姿势反射）
- ★脊休克（必考）
  - 定义：脊髓与**高位中枢离断**后 -> 断面以下反射**暂时丧失** -> 无反应状态
  - 表现：肌紧张↓、血压↓、粪尿**潴留**
  - 恢复：蛙几分钟、犬几天、人**数周**以上 -> 知觉/随意运动**永久丧失** -> 内脏反射部分恢复（大小便**失禁**）
  - 原因：断面以下脊髓突然**失去高位中枢调控**
