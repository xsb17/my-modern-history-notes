# 脊髓对躯体姿势的调节 — 微型速记导图（大题常考）&nbsp;&nbsp;&nbsp;

- 脊髓运动神经元&nbsp;&nbsp;&nbsp;
  - α 运动神经元：支配**梭外肌纤维**（骨骼肌）&nbsp;&nbsp;&nbsp;
  - γ 运动神经元：支配**梭内肌纤维**（肌梭）&nbsp;&nbsp;&nbsp;
  - 运动单位 = 一个 **α 神经元** + 全部支配肌纤维&nbsp;&nbsp;&nbsp;
    - 小单位：**精细运动**（眼外肌）&nbsp;&nbsp;&nbsp;
    - 大单位：巨大**肌张力**（四肢肌）&nbsp;&nbsp;&nbsp;
- ★牵张反射定义&nbsp;&nbsp;&nbsp;
  - 骨骼肌受**外力牵拉伸长** -> 引起**同一肌肉收缩**的反射&nbsp;&nbsp;&nbsp;
- ★牵张反射两大类型对比（必考大题）&nbsp;&nbsp;&nbsp;
  - 腱反射：&nbsp;&nbsp;&nbsp;
    - 快速牵拉肌腱 -> 快速明显收缩（膝跳反射）&nbsp;&nbsp;&nbsp;
    - 特点：**单突触反射**、收缩力强、迅速短暂、**快肌纤维**收缩&nbsp;&nbsp;&nbsp;
    - 意义：临床**神经系统检查**&nbsp;&nbsp;&nbsp;
  - 肌紧张：&nbsp;&nbsp;&nbsp;
    - 缓慢持续牵拉肌腱 -> 持续轻度收缩&nbsp;&nbsp;&nbsp;
    - 特点：缓慢持久**不易疲劳**、不同运动单位**交替收缩**、**慢肌纤维**收缩、**伸肌**表现明显&nbsp;&nbsp;&nbsp;
    - 意义：**维持躯体姿势**（最基本反射）&nbsp;&nbsp;&nbsp;
- ★牵张反射反射弧（五段式）&nbsp;&nbsp;&nbsp;
  - 感受器：**肌梭**（长度感受器）-> 梭内肌中间感受装置&nbsp;&nbsp;&nbsp;
  - 传入纤维：**Ⅰa、Ⅱ类纤维**&nbsp;&nbsp;&nbsp;
  - 中枢：脊髓前角 **α、γ 运动神经元**&nbsp;&nbsp;&nbsp;
  - 传出纤维：α 传出 -> 梭外肌；γ 传出 -> 梭内肌&nbsp;&nbsp;&nbsp;
  - 效应器：同一肌肉的**梭外肌**&nbsp;&nbsp;&nbsp;
- ★γ 环路（提高肌梭敏感性）&nbsp;&nbsp;&nbsp;
  - 高位中枢 -> **γ 传出↑** -> 梭内肌收缩 -> 感受装置牵拉↑ -> Ⅰa 传入↑ -> **α 活动↑** -> 梭外肌收缩↑&nbsp;&nbsp;&nbsp;
- 反牵张反射（保护机制）&nbsp;&nbsp;&nbsp;
  - 感受器：**腱器官**（张力感受器）&nbsp;&nbsp;&nbsp;
  - 梭外肌收缩 -> 张力↑ -> 腱器官兴奋 -> Ⅰb 传入↑ -> 抑制 α 神经元 -> 抑制牵张反射 -> 避免**肌肉损伤**&nbsp;&nbsp;&nbsp;
- 屈肌反射与对侧伸肌反射&nbsp;&nbsp;&nbsp;
  - 屈肌反射：伤害性刺激 -> 同侧**屈肌收缩/伸肌舒张** -> 肢体回缩 -> **躲避伤害**（非姿势反射）&nbsp;&nbsp;&nbsp;
  - 对侧伸肌反射：强刺激 -> 同侧屈 + **对侧伸** -> **保持躯体平衡**（姿势反射）&nbsp;&nbsp;&nbsp;
- ★脊休克（必考）&nbsp;&nbsp;&nbsp;
  - 定义：脊髓与**高位中枢离断**后 -> 断面以下反射**暂时丧失** -> 无反应状态&nbsp;&nbsp;&nbsp;
  - 表现：肌紧张↓、血压↓、粪尿**潴留**&nbsp;&nbsp;&nbsp;
  - 恢复：蛙几分钟、犬几天、人**数周**以上 -> 知觉/随意运动**永久丧失** -> 内脏反射部分恢复（大小便**失禁**）&nbsp;&nbsp;&nbsp;
  - 原因：断面以下脊髓突然**失去高位中枢调控**&nbsp;&nbsp;&nbsp;
