# 02-重点展开 正面战场与敌后战场的协同

> [!quote] 本节聚焦
> 详细展开抗日战争的**两个战场**——国民党正面战场与共产党敌后战场的各自表现、战略配合及相互关系。这是考试中**材料分析题和比较题**的高频出题区域。

---

## 一、抗日战争的三个阶段

| 阶段 | 时间 | 特征 |
|------|------|------|
| **战略防御阶段** | 1937.7—1938.10 | 以国民党军队为主体的正面战场抗击日军战略进攻 |
| **战略相持阶段** | 1938.10—1943.12 | 敌后游击战成为主要抗日作战方式 |
| **战略反攻阶段** | 1944.1—1945.8 | 逐步转入反攻，最终取得胜利 |

---

## 二、战略防御阶段的正面战场（1937.7—1938.10）

在战略防御阶段，以国民党军队为主体的正面战场担负了抗击日军战略进攻的主要任务，组织了**淞沪、太原、徐州、武汉会战**等一系列大战役。

### （一）四大战役对比

| 战役 | 时间 | 中国投入兵力 | 歼敌 | 效果/意义 |
|------|------|-------------|------|----------|
| **淞沪会战** | 1937.8—11 | **70万** | 4万多 | **粉碎日本三个月灭亡中国的计划** |
| **太原会战（忻口战役）** | 1937.10—11 | 8万多 | 伤亡2万余 | **国共精诚合作** |
| **徐州会战（台儿庄战役）** | 1938.1—5 | 40万 | 1万多 | **抗战以来的最大胜仗** |
| **武汉会战** | 1938.6—10 | **100万** | 伤亡10余万 | **规模最大、时间最长、歼敌最多**，标志着战争进入相持阶段 |

### （二）重点战役详述

#### 1. 淞沪会战——八百壮士

> 在淞沪会战中，第八十八师五二四团团附**谢晋元**率孤军据守**四行仓库**，被上海市民喻为"**八百壮士**"。
> **「一寸山河一寸血」**——四行仓库保卫战打出了中国军队的血性本色。

> 💡 **补充说明：** "八百壮士"实际人数约400余人（一个加强营），为了壮大声势对外称800人。他们坚守四行仓库四昼夜，在日军重重包围下击退多次进攻。这场战斗在上海租界民众的注视下进行，极大鼓舞了全国军民的抗战士气。

#### 2. 台儿庄大战

- **时间：** 1938年3—4月
- **指挥：** 李宗仁
- **战果：** 歼敌1万余人
- **地位：** **抗战以来正面战场取得的最大胜利**

#### 3. 武汉会战

| 项目 | 内容 |
|------|------|
| **规模** | 抗日战争战略防御阶段**规模最大、时间最长、歼敌最多**的一次战役 |
| **战场范围** | 安徽、河南、江西、湖北4省广大地区 |
| **战略思想** | "以空间换时间"，利用地形和工事逐次抵抗消耗日军 |
| **三大目的** | ①以空间换取时间，消耗挫败敌人；②内迁工厂、内运物资、整备军队；③争取国际同情和支援 |
| **总指挥** | 蒋介石自任总指挥 |

> 1938年6月中旬，国民政府军委会制定了保卫武汉的作战计划，其中心思想是立足外线，保持部队高度的机动性，利用地形和工事，逐次抵抗消耗日军，以空间换时间，最后转变敌攻我守的战争态势。

### （三）抗战初期国民政府的积极政策

| 政策 | 时间 |
|------|------|
| 改组军事委员会 | 1938年初 |
| 成立**国民参政会** | 1938年6月 |
| **民族工业西迁** | 抗战初期 |

> **毛泽东评价：** 「从一九三七年七月七日卢沟桥事变到一九三八年十月武汉失守这一个时期内，国民党政府的对日作战是比较努力的。」
> ——《毛泽东选集》第3卷

### （四）一次伟大的西迁——民族工业西迁

> **被誉为"中国迁厂之父"的林继庸**，与同事们奋力将**152家上海民用企业**，连同江苏浙江的超过**200家**企业的机器设备拆卸运送至大后方进行组装重建。

- **意义：** 对保存国家实力，支援前方的抗战起了极大作用
- 与西迁至后方的高校、军工企业一道为我国保留了**文明复兴的种子与命脉**
- 为抗战的最终胜利奠定了**工业基础**

### （五）战略防御阶段正面战场的评价

| 积极方面 | 消极方面 |
|----------|----------|
| ①粉碎了日军"速战速决"灭亡中国的计划 | 虽然取得了局部的胜利，但**总体败退** |
| ②客观上掩护了八路军、新四军游击战略的实施 | 华中、华北、华南大片地区沦陷 |
| ③为世界反法西斯战争做出了贡献 | 广州、武汉被日军占领 |

## 三、战略相持阶段的正面战场（1938.10—1943.12）

### （一）为什么抗战会进入相持阶段？

| 方面 | 原因 |
|------|------|
| **日本方面** | 战线太长，兵力、财力、物力不足；加上**敌后抗日根据地的发展和壮大**严重威胁敌人后方，所以不得不停止对正面战场的大规模进攻 |
| **中国方面** | 正面战场连续败退，人民抗日力量还需要经过长期艰苦斗争才能打败侵略者 |

### （二）日本对华政策调整

| 方向 | 策略 |
|------|------|
| 总方针 | **"以华制华"**和**"以战养战"** |
| 对抗日根据地 | 集中大量兵力进行"**扫荡**" |
| 对国民政府 | **政治诱降为主，军事进攻为辅** |
| 对日占区 | 加紧扶植傀儡政权，建立和发展汉奸组织 |

### （三）国民党五届五中全会（1939年1月）

武汉失守标志着中国抗日战争进入战略相持阶段。1939年1月，国民党五届五中全会决定成立"**防共委员会**"，确定了"**防共、限共、溶共、反共**"的方针。

> **这标志着国民党政府逐步转变为消极抗战。**

### （四）汪精卫叛国投敌

- **1938年底**，汪精卫等人叛国投敌
- **1940年3月**，在南京成立了伪国民政府
- 协同日本侵略军进攻抗日根据地

### （五）相持阶段的重要战役

| 战役 | 时间 | 指挥 | 意义 |
|------|------|------|------|
| **长沙会战** | 1939.9—1942.2（三次） | **薛岳** vs **岗村宁次** | ①相持阶段后国民党正面战场的重大胜利；②中国军队第一次以武力迫使日军回到原战略态势；③粉碎了日本"以战迫降"的战略目标；④提高了中国在世界反法西斯阵营的地位 |
| **枣宜会战** | 1940 | **张自忠**殉国 | 第三十三集团军总司令张自忠将军在枣宜会战中壮烈牺牲 |
| **中条山战役** | 1941 | — | 华北正面战场最后一战 |
| **昆仑关战役** | 1939—1940 | — | 桂南会战重要战役 |
| **中国远征军** | 1942—1945 | — | 入缅作战，支援盟军 |

> 💡 **补充说明：** **张自忠**是抗日战争中**牺牲的最高级别中国将领**（集团军总司令）。他在枣宜会战中身负重伤仍坚持指挥，最终壮烈殉国。周恩来评价他"**忠义之志，壮烈之气**"。张自忠的牺牲是相持阶段国民党军坚持抗战的象征。

### （六）豫湘桂战役大溃败（1944.4—1945.1）

| 项目 | 数据 |
|------|------|
| 日军目的 | 发动打通中国大陆交通线的作战 |
| 结果 | 国民党军队惨遭**大溃败** |
| 损失 | 损失**50多万**兵力，丢失**146座**大小城市、**6000万**人口的**20多万平方公里**的国土 |

> 💡 **补充说明：** 豫湘桂战役的大溃败暴露了国民党在相持阶段后期**腐败无能、指挥混乱**的严重问题。当时日军已经是强弩之末，在太平洋战场节节败退，但在中国战场上仍能发动如此大规模的进攻并取得胜利，充分说明了国民党统治的腐朽。

## 四、全面抗战路线与持久战战略总方针

### （一）洛川会议（1937年8月）

中国共产党在陕北洛川召开中央政治局扩大会议，制定了**抗日救国十大纲领**：

| 序号 | 纲领内容 |
|------|----------|
| 一 | 打倒日本帝国主义 |
| 二 | 全国军事的总动员 |
| 三 | 全国人民的总动员 |
| 四 | 改革政治机构 |
| 五 | 抗日的外交政策 |
| 六 | 战时的财政经济政策 |
| 七 | 改良人民生活 |
| 八 | 抗日的教育政策 |
| 九 | 肃清汉奸卖国贼亲日派，巩固后方 |
| 十 | 抗日的民族团结 |

> **核心思想：** 强调要打倒日本帝国主义，关键在于使已经发动的抗战成为**全面的全民族的抗战**。

### （二）《论持久战》（1938年5—6月）

毛泽东总结全民族抗战10个月来的经验，发表《论持久战》。

| 方面 | 内容 |
|------|------|
| **批驳错误观点** | 批驳了"**亡国论**"和"**速胜论**"等错误观点 |
| **核心判断** | 系统阐明了**持久战**方针 |
| **为什么是持久战？** | 日本是强国，中国是弱国（决定了不能速胜）；日本是小国，侵略战争失道寡助，中国是大国，反抗侵略得道多助（决定了不会亡国）；有中国共产党及其领导的抗日根据地和人民军队（决定了胜利最终属于中国） |

#### 毛泽东科学预见了抗战三阶段

```
战略防御阶段 ──→ 战略相持阶段 ──→ 战略反攻阶段
```

#### 《论持久战》的地位

| 方面 | 意义 |
|------|------|
| **纲领性文献** | 是中国共产党领导抗日战争的**纲领性文献** |
| **指明前景** | 指明了必须持久抗战才能取得最后胜利的前景 |
| **动员群众** | 提出了一整套动员人民群众、在持久战争中不断削弱敌方优势、生长自己力量的办法 |
| **增强信心** | **大大增强了人们坚持抗战的决心和信心** |

> 💡 **补充说明：** **《论持久战》最精彩之处在于它的预判完全准确**——毛泽东在1938年就预见到抗战要经过"防御→相持→反攻"三个阶段。事实上战争的发展完全遵循了这一预判。他还特别强调**战略相持阶段是持久战转到最后胜利的"枢纽"阶段**——谁能熬过相持阶段，谁就能赢得战争。这一理论为全民族抗战提供了科学的战略指导。

## 五、敌后战场的开辟与游击战争的发展

### （一）平型关大捷（1937年）

| 项目 | 内容 |
|------|------|
| 性质 | **八路军出师华北前线打的第一个大胜仗** |
| 地位 | **全国抗战以来的第一个大胜仗** |
| 意义 | **打破了日军不可战胜的神话**，振奋了人心，鼓舞了士气 |

> 1937年11月太原失守后，在华北，以国民党为主体的正规战争结束，**以共产党为主体的游击战争上升到主要地位**。

### （二）敌后抗日根据地的创建

到1938年10月，八路军和新四军同日、伪军作战**1600多次**，到1940年底，在华北、华中、华南创建了**16块根据地**，包括：

| 华北 | 华中 |
|------|------|
| 晋察冀根据地 | 苏北抗日根据地 |
| 晋绥根据地 | 苏中抗日根据地 |
| 冀热辽根据地 | 淮北抗日根据地 |
| 晋冀豫根据地 | 淮南抗日根据地 |
| 冀鲁豫根据地 | 皖中抗日根据地 |
| 山东根据地 | — |

### （三）人民战争的战略战术

面对日军的疯狂进攻，敌后军民进行了艰苦卓绝的斗争，发展了人民战争的战略战术：

| 战术 | 说明 |
|------|------|
| **地道战** | 挖地道连接村庄，神出鬼没打击敌人 |
| **地雷战** | 在道路上埋设地雷，炸毁日军车辆人员 |
| **麻雀战** | 小股部队分散出击，骚扰消耗敌人 |
| **破袭战** | 破坏日军交通线、通讯设施 |
| **敌后武工队** | 深入敌后的小分队武装工作 |
| **水上游击队** | 在河湖港汊开展游击战 |

### （四）游击战争的战略地位

| 阶段 | 游击战地位 | 作用 |
|------|-----------|------|
| **战略防御阶段** | **辅助的** | 阻止日军进攻、减轻正面战场压力，使战争转入相持阶段的关键性作用 |
| **战略相持阶段** | **主要的抗日作战方式** | 削弱敌人、壮大自己，逐步改变敌强我弱态势，为实行战略反攻准备条件 |

### （五）百团大战（1940年8月—10月）

| 项目 | 内容 |
|------|------|
| **指挥** | **彭德怀** |
| **范围** | 华北**2000多公里**的战线上 |
| **规模** | 八路军**100多个团共20万人**，战斗**1800多次** |
| **目的** | 争取抗战形势好转，振奋人民的抗战信心 |
| **战绩** | 毙伤日伪军**4万多**，破坏交通线**2000多公里**，拔掉据点**近3000个** |
| **意义** | **沉重地打击了敌人，鼓舞了全国人民，提高了共产党及军队的威望** |

## 六、坚持抗战、团结、进步的方针

### （一）统一战线中的独立自主原则

| 原则 | 内容 |
|------|------|
| 1 | 必须保持在思想上、政治上和组织上的**独立性**，放手发动群众，壮大人民力量 |
| 2 | 必须坚持对人民军队的**绝对领导**，冲破国民党的限制和束缚，努力发展人民武装和抗日根据地 |
| 3 | 必须对国民党采取**又团结又斗争、以斗争求团结**的方针 |

### （二）三大口号（1939年7月）

| 口号 | 对应 |
|------|------|
| **坚持抗战到底** | **反对中途妥协** |
| **巩固国内团结** | **反对内部分裂** |
| **力求全国进步** | **反对向后倒退** |

### （三）国民党三次反共高潮

| 次数 | 时间 | 事件 |
|------|------|------|
| **第一次** | 1939年冬至1940年春 | 国民党发动第一次反共高潮 |
| **第二次** | **1941年1月——皖南事变** | 反共高潮的高峰 |
| **第三次** | 1943年春 | 第三次反共高潮 |

### （四）皖南事变（1941年1月6日）

| 项目 | 内容 |
|------|------|
| 经过 | 新四军军部及所属部队在皖南遭国民党军队包围袭击 |
| 损失 | 军长**叶挺被俘**，副军长**项英**、参谋长**周子昆**突围后遇难，政治部主任**袁国平**牺牲。除**傅秋涛**率2000余人分散突围外，少数被俘，**大部分壮烈牺牲** |
| 性质 | 国民党**第二次反共高潮的高峰** |

#### 中共的针锋相对斗争

| 手段 | 内容 |
|------|------|
| **舆论反击** | 《新华日报》登载**周恩来**的题诗——「**千古奇冤，江南一叶；同室操戈，相煎何急？！**」 |
| **政治揭露** | 1943年7月12日，毛泽东亲自为《解放日报》撰写《**质问国民党**》的社论，揭露国民党顽固派破坏团结抗战，号召全国人民起来制止内战危机 |

> 💡 **补充说明：** 皖南事变后，中共坚持"**有理、有利、有节**"的斗争原则。一方面在政治上彻底揭露国民党的反共罪行，另一方面在军事上保持克制，不使国共合作完全破裂。这种"**又团结又斗争**"的策略，既维护了抗日民族统一战线的大局，又使国民党在政治上陷入被动。

### （五）巩固抗日民族统一战线的策略总方针

> **「发展进步势力，争取中间势力，反对顽固势力」**

| 势力 | 包含群体 | 策略 |
|------|----------|------|
| **进步势力** | 工人、农民和城市小资产阶级 | **发展** |
| **中间势力** | 民族资产阶级、开明绅士和地方实力派 | **争取** |
| **顽固势力** | 大地主大资产阶级的抗日派（蒋介石集团） | **反对**（又团结又斗争） |
