- 第四章 血液循环 — 第一节 心脏的泵血功能
  - 心脏的泵血功能：心脏通过节律性跳动泵出血液，并提供动力使血液在血管内循环流动
  - 【心肌细胞的收缩特点】
    - ① 同步收缩（"全或无"式收缩）
      - 两个合胞体（心房合胞体 + 心室合胞体）各自同步收缩
      - 心房肌与心室肌之间由结缔组织分隔，互不传导
    - ② 不发生完全强直收缩
      - 心肌有效不应期特别长（相当于整个收缩期+舒张早期）
      - 保证收缩与舒张交替进行，实现泵血功能
    - ③ 对细胞外 Ca²⁺ 高度依赖
      - 心肌细胞肌质网不如骨骼肌发达，贮Ca²⁺较少
      - 需细胞外Ca²⁺内流触发钙释放（钙诱导钙释放机制，CICR）
  - 一、心脏的泵血过程和机制
    - （一）心动周期
      - 定义：心脏一次收缩和舒张构成的机械活动周期，包括收缩期和舒张期
      - 心动周期长度与心率成反比：心动周期（s）= 60 ÷ 心率
      - 心率：单位时间内心脏搏动的次数，安静状态下正常成人 60~100 次/min
      - 心率为 75 次/min 时：心动周期 = 60s ÷ 75 = 0.8s
      - 心动周期的特点：
        - ① 心房和心室按照一定顺序和时程先后收缩（心房先收缩 -> 心室后收缩）
        - ② 心室舒张前 0.4s 内，心房也在舒张 → 全心舒张期
        - ③ 无论心房还是心室，收缩期均**短于**舒张期
        - ④ 临床意义：长时间心动过速 -> 心动周期缩短 -> 舒张期缩短比例**更大** -> 冠脉供血减少 -> 不利于心脏长期工作
    - （二）心脏的泵血过程
      - 核心动力：心脏舒缩产生的**压力差**
      - 方向决定因素：瓣膜启闭（房室瓣 / 动脉瓣）
      - 以心房收缩期作为心动周期起始点，左右两侧泵血原理基本相似
      - 1. 心房收缩期与心房舒张期
        - 心房收缩 -> 房内压**升高** > 室内压 -> 房室瓣**开** -> 血液（20%~30%）由心房**主动挤入**心室（持续约 0.1s）
        - 之后转入心房舒张期
        - 进入全心舒张期 -> 心房、心室均舒张 -> 房室瓣**开** -> 血液（70%~80%）经心房**快速流入**心室（被动充盈）
        - 特点：心房收缩为主动充盈，贡献心室总充盈量约 **1/4**
      - 2. 心室收缩期
        - （1）等容收缩期
          - 心室肌收缩 -> 室内压**急剧升高** > 房内压 -> 房室瓣**关**
          - 室内压仍 < 主动脉压 -> 主动脉瓣仍**关**
          - 压力关系：房压 < 室压 < 动脉压
          - 特点：
            - 两瓣均**关**（房室瓣**关** + 主动脉瓣**关**）
            - 心室容积**不变**（血液不进不出）
            - 室内压**急剧升高**（室内压上升最快的时期）
        - （2）快速射血期
          - 心室肌继续收缩 -> 室内压**升高** > 动脉压 -> 主动脉瓣**开**
          - 血液由心室快速射入动脉
          - 压力关系：房压 < 室压 > 动脉压（室内压达**峰值**）
          - 特点：
            - 房室瓣**关**，主动脉瓣**开**
            - 室内压达**峰值**
            - 射血量占总射血量约 **70%**（2/3）
            - 心室容积**迅速减小**
        - （3）减慢射血期
          - 室内压由峰值逐渐**下降** -> 射血速度**减慢**
          - 依靠**惯性**射血（此时室内压已略低于动脉压，血液靠惯性继续射出）
          - 特点：
            - 房室瓣**关**，主动脉瓣**开**
            - 心室容积继续减小至**最低**（收缩末期容积 ESV）
            - 射血量占总射血量约 **30%**（1/3）
      - 3. 心室舒张期
        - （1）等容舒张期
          - 心室肌舒张 -> 室内压**下降** < 动脉压 -> 主动脉瓣**关**
          - 室内压仍 > 房内压 -> 房室瓣仍**关**
          - 压力关系：房压 < 室压 < 动脉压
          - 特点：
            - 两瓣均**关**（房室瓣**关** + 主动脉瓣**关**）
            - 心室容积**不变**（血液**不进不出**）
            - 室内压**急剧下降**
        - （2）快速充盈期（负压抽吸）
          - 室内压继续下降 < 房内压 -> 房室瓣**开**
          - 血液快速由心房流入心室 -> 心室容积**迅速增大**
          - 特点：
            - 房室瓣**开**，主动脉瓣**关**
            - "负压抽吸"效应：心室舒张使室内压快速降低，产生抽吸力
            - 充盈量占总充盈量的大部分
        - （3）减慢充盈期
          - 房室间压力梯度减小 -> 入室血流速度**减慢**
          - 心室容积**继续增大**
          - 特点：房室瓣**开**，主动脉瓣**关**
        - （4）心房收缩期（心室舒张末期最后 0.1s）
          - 心房收缩 -> 心室充盈量再增加约 **1/4**
          - 心室容积达**最大**（舒张末期容积 EDV）
          - 特点：心房发挥**初级泵**作用
      - 【泵血过程总览——各期瓣膜、压力、容积与血流方向】
        - 心室收缩期：
          - 等容收缩期：房室瓣**关** + 半月瓣**关**，室内压**急剧↑**，容积**不变**，血流在心室
          - 快速射血期：房室瓣**关** + 半月瓣**开**，室内压达**峰值**，容积**迅速↓**，射入动脉占 **2/3**
          - 减慢射血期：房室瓣**关** + 半月瓣**开**，室内压**逐渐↓**，容积**↓至最低**，射入动脉占 **1/3**
        - 心室舒张期：
          - 等容舒张期：房室瓣**关** + 半月瓣**关**，室内压**急剧↓**，容积**不变**，血流在心室
          - 快速充盈期：房室瓣**开** + 半月瓣**关**，室内压**快速↓**，容积**迅速↑**，血液快速流入心室
          - 减慢充盈期：房室瓣**开** + 半月瓣**关**，室内压**＜房压**，容积**继续↑**，血液慢速流入心室
          - 心房收缩期：房室瓣**开** + 半月瓣**关**，房压**＞室压**，容积**↑至最大**，心房血液挤入心室
    - （三）心脏泵血的特点
      - ① 心室发挥**主导**作用
      - ② 心房发挥**初级泵**作用（贡献心室总充盈量约 **1/4**）
      - ③ 心脏瓣膜防止血液**倒流**，保证单向流动
      - ④ 产生心音
        - 第一心音：房室瓣关闭 + 心室肌收缩振动（标志收缩期开始）
        - 第二心音：半月瓣关闭（标志舒张期开始）
        - 由心肌收缩、瓣膜启闭、血液流速改变形成的涡流和血液撞击心室壁/大动脉壁引起振动产生
  - 二、心脏泵血功能的评价
    - （一）基于泵血量的心功能评价指标
      - 1. 搏出量（每搏输出量，SV）和射血分数（EF）
        - 搏出量（SV）
          - 定义：一侧心室一次心搏所射出的血液量
          - 正常值：安静状态下约 **70mL**（60~80mL）
          - 公式：SV = EDV - ESV（舒张末期容积 - 收缩末期容积）
          - 生理意义：心功能的基础参数
        - 射血分数（EF）
          - 定义：搏出量占心室舒张末期容积的百分比
          - 公式：EF = SV ÷ EDV × 100%
          - 正常值：健康成年人 **55%~65%**
          - 生理意义：与搏出量相比，更能反映心脏**射血效率**
          - 临床意义：心室功能减退、心室异常扩大的患者 -> EF **明显降低**
      - 2. 心输出量（每分输出量，CO）和心指数（CI）
        - 心输出量（CO）
          - 定义：一侧心室 1 分钟射出的血液量
          - 公式：CO = SV × HR（搏出量 × 心率）
          - 正常值：健康成年男性安静状态下 **4.5~6.0 L/min**
          - 生理意义：评价不同状态下的心功能
        - 心指数（CI）
          - 定义：单位体表面积（每平方米）的心输出量
          - 公式：CI = CO ÷ 体表面积
          - 正常值：中等身材成年人静息心指数 **3.0~3.5 L/(min·m²)**
          - 生理意义：评价不同身材个体的心功能，消除体型差异影响
    - （二）基于心脏做功的心功能评价指标
      - 1. 每搏功（搏功）：心室一次收缩射血所做的功
        - 搏功 = 搏出量 ×（平均动脉压 - 平均心房压）+ 动能
      - 2. 每分功（分功）：心室 1 分钟内收缩射血所做的功
        - 分功 = 搏功 × 心率
      - 生理意义：用心脏做功比单纯泵血量更全面评价泵血功能
        - 尤其在动脉血压水平不同的个体之间
        - 或同一个体动脉血压发生改变前后
        - 因为血压升高时，心脏需做更多功才能射出等量血液
      - 3. 心脏耗氧率：衡量心脏工作时消耗能量的指标，可间接反映心脏做功量
  - 三、心脏泵血功能的储备（心力储备）
    - 定义：心输出量随机体代谢需要而**增加**的能力，反映心脏泵血功能对代谢需要的适应能力
    - 安静状态：心输出量约 **5 L/min**
    - 剧烈运动时：可达 **25~30 L/min**（安静时的 5~6 倍），运动员可达 **40 L/min**（近 7 倍）
    - 储备来源：心输出量 = 搏出量 × 心率
    - （一）搏出量储备
      - 收缩期储备：提高心输出量的**重要途径**，正常成年人 **35~40mL**
        - 机制：增强心肌收缩力 -> 减少收缩末期容积（ESV↓）-> 增加搏出量
      - 舒张期储备：轻度储备作用，仅约 **15mL**
        - 机制：增加舒张末期容积（EDV↑）-> 增加搏出量
        - 受心包限制，储备空间有限
    - （二）心率储备
      - 提高心输出量的**重要途径**
      - 心率在一定范围内加快（达 **160~180 次/分**）-> 心输出量可增至静息时的 2~2.5 倍
      - 是最常被调动的**心力储备**
      - 注意（心率过快反而降低心输出量）：
        - 心率 > 180 次/分 -> 舒张期**显著缩短** -> 心室充盈**不足** -> 搏出量↓ -> 心输出量反而**降低**
        - 原因：冠脉供血主要在舒张期，心率过快 -> 冠脉供血减少 + 心肌耗氧增加 -> 心肌收缩力减弱
  - 四、影响心输出量的因素
    - 核心公式：心输出量 = 搏出量 × 心率
    - （一）影响搏出量的因素
      - 1. 前负荷（异长自身调节）
        - 定义：肌肉收缩前所承受的负荷 -> 使肌肉被拉长到初长度
        - 心脏中的对应：
          - 心室舒张末期压力 ≈ 前负荷
          - 心室舒张末期容积（EDV）≈ 初长度
        - （1）心室功能曲线（Frank-Starling 曲线）
          - 概念：表示心室舒张末容积（前负荷）与搏出量（或搏功）关系的曲线
          - 机制：在一定范围内，增加心室舒张末容积 -> 心肌肌小节初长度增加（最适初长度 2.0~2.2μm）-> 粗细肌丝有效重叠增加 -> 心肌收缩力**增强** -> 搏出量**增多**
          - 特点：
            - 心室肌有较大初长度储备（正常初长度约 1.8~2.0μm，远小于最适初长度 2.2μm）
            - 超过最适初长度后，搏出量**不变或轻微减少**（骨骼肌则会急剧下降）
            - 正常心室工作于功能曲线**升支段**，有较大储备空间
        - （2）异长自身调节（Starling 机制）
          - 定义：通过改变心肌细胞初长度（前负荷）而引起心肌收缩强度改变，进而调节搏出量的机制
          - 核心：前负荷改变 -> 初长度改变 -> 收缩力改变（不依赖神经体液因素）
          - 生理意义：对搏出量的微小变化进行精细调节，使心室射血量与静脉回心血量保持**平衡**
            - 回心血量↑ -> EDV↑ -> 心肌拉长 -> 收缩力↑ -> 搏出量↑ -> 防止血液在心室内淤积
            - 保证左右心室输出量平衡，防止肺循环淤血/体循环淤血
        - （3）影响前负荷（心室舒张末期容积）的因素
          - 心室充盈时间：心率**增快** -> 舒张期缩短 -> 充盈时间**缩短** -> EDV**减小** -> 前负荷**减小**
          - 静脉回流速度：静脉输液/体位改变 -> 回流速度**加快** -> EDV**增大** -> 前负荷**增大**
          - 心包腔内压：心包炎/心包积液 -> 心包腔内压**升高** -> 限制心室充盈 -> EDV**减小** -> 前负荷**减小**
          - 心室顺应性：心肌肥厚/纤维化 -> 顺应性**降低** -> 充盈受限 -> EDV**减小** -> 前负荷**减小**
          - 心室舒张功能：心肌缺血缺氧 -> 舒张功能**障碍** -> 充盈不足 -> EDV**减小** -> 前负荷**减小**
          - 射血后心室内剩余血量：剩余血量**增多** -> EDV**增大** -> 前负荷**增大**
      - 2. 后负荷
        - 定义：心室肌收缩射血时必须对抗的负荷
        - 在体情况下：大动脉血压 = 心室收缩时的后负荷
        - 动脉血压对心输出量的影响：
          - 动脉血压短暂升高（不超过 160mmHg 时，正常心脏可代偿）：
            - 动脉血压↑ -> 等容收缩期室内压峰值**增加** -> 等容收缩期**延长** -> 射血期**缩短** -> 搏出量**暂时减少**
            - 心室内剩余血量**增加** -> 若回心血量不变 -> 心舒末期容积（EDV）**增大** -> 初长度**增加**
            - 通过异长自身调节（Starling机制）-> 心肌收缩力**增强** -> 搏出量**恢复**正常
          - 动脉血压长期持续升高（高血压病的病理过程）：
            - 左心室后负荷持续增大 -> 心室肌代偿性**肥大**（心肌细胞增粗）-> 心肌收缩力**增强** -> 短期内维持正常搏出量
            - 但长期代偿 -> 心肌重构（纤维化、凋亡）-> 心室扩张 -> 心肌收缩力**减弱** -> 可致**心力衰竭**
        - 临床意义：高血压是导致心力衰竭的重要病因之一
      - 3. 心肌收缩能力（等长调节）
        - 定义：心肌不依赖于前后负荷而改变其力学活动（收缩强度与速度）的内在特性，又称心肌的**变力状态**（inotropic state）
        - 等长调节：通过心肌细胞本身收缩能力的改变而使搏出量和搏功发生相应改变的调节方式
          - "等长"意为前负荷（初长度）不变而搏出量改变
        - 影响因素：
          - 活化横桥数目（决定收缩力的直接因素）：
            - 胞质内 Ca²⁺ 浓度↑ -> 活化横桥↑ -> 收缩力**增强**（受交感神经、儿茶酚胺等影响）
            - 肌钙蛋白对 Ca²⁺ 的亲和力↑ -> 活化横桥↑ -> 收缩力**增强**（受茶碱、咖啡因等影响）
          - 肌球蛋白头部 ATP 酶活性：甲状腺激素↑ -> ATP酶活性↑ -> 收缩力**增强**
          - 心率（阶梯现象）：心率增快 -> 细胞内 Na⁺ 和 Ca²⁺ 积累 -> 收缩能力**增强**
        - 正性变力作用：交感神经兴奋、儿茶酚胺（肾上腺素、去甲肾上腺素）-> 心肌收缩能力**增强** -> 心室功能曲线**左上移**（同等前负荷下搏出量增加）
        - 负性变力作用：心肌缺血、酸中毒、心力衰竭 -> 心肌收缩能力**减弱** -> 心室功能曲线**右下移**（同等前负荷下搏出量减少）
        - 三种调节方式的对比理解：
          - 异长自身调节（前负荷调节）：改变初长度 -> 改变搏出量，适应回心血量变化
          - 等长调节（收缩能力调节）：不改变初长度 -> 改变搏出量，适应代谢需求变化
          - 后负荷调节：动脉血压改变 -> 暂时影响搏出量 -> 继而引发异长自身调节代偿
    - （二）影响心率的因素
      - 心率与心输出量的关系（CO = SV × HR）：
        - 心率正常（60~100 次/分）：心输出量正常
        - 心率适度加快（不超过 160~180 次/分）：心率↑ -> 舒张期虽有缩短但充盈尚可 -> 心输出量**增加**
        - 心率过快（超过 160~180 次/分）：心率↑↑ -> 舒张期**显著缩短** -> 心室充盈**严重不足** -> 搏出量**大幅下降** -> 心输出量反而**降低**
        - 心率过慢（低于 40 次/分）：心率↓↓ -> 虽搏出量有所增加（充盈充分）但不足以代偿 -> 心输出量**降低**
      - 影响心率的具体因素：
        - 生理因素：年龄、性别、睡眠状态
        - 神经因素：交感神经兴奋 -> 心率**加快**（正性变时作用）；迷走神经（副交感）兴奋 -> 心率**减慢**（负性变时作用）
        - 体液因素：肾上腺素、去甲肾上腺素 -> 心率**加快**；甲状腺激素 -> 心率**加快**
        - 离子浓度：血 K⁺↑ -> 心率**减慢**（高钾抑制自律性）；血 Ca²⁺↑ -> 心率**加快**
        - 体温：体温↑ -> 心率**加快**（体温每升高 1℃，心率增加约 10 次/分）；体温↓ -> 心率**减慢**
  - 【核心概念对比速查】
    - 搏出量（SV）vs 射血分数（EF）：SV 反映一次射血绝对量（70mL），EF 反映射血效率（55%~65%），EF 更敏感地反映心功能减退
    - 心输出量（CO）vs 心指数（CI）：CO 反映整体泵血量（4.5~6.0 L/min），CI 消除体型差异（3.0~3.5 L/(min·m²)）
    - 异长自身调节 vs 等长调节：前者通过改变初长度（前负荷）调节，后者通过改变收缩能力（不依赖初长度）调节
    - 收缩期储备 vs 舒张期储备：收缩期储备 35~40mL（主要储备），舒张期储备仅约 15mL（心包限制）
    - 前负荷 vs 后负荷：前负荷 = 心室舒张末期容积/压力（牵拉心肌），后负荷 = 大动脉血压（对抗射血）
    - 等容收缩期 vs 等容舒张期：共同点为两瓣均**关**、容积**不变**；区别为前者室内压**急剧↑**（收缩），后者室内压**急剧↓**（舒张）
    - 快速射血期 vs 快速充盈期：前者射血入动脉（房室瓣**关** + 半月瓣**开**），后者心房血入心室（房室瓣**开** + 半月瓣**关**）
