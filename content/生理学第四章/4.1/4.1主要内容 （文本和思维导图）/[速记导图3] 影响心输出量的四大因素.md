# 影响心输出量的四大因素 — 极简速记导图

- 核心公式：**CO** = SV × HR
- 前负荷（异长自身调节 / Starling 机制）
  - 本质：**心室舒张末期容积**（EDV）-> 代表**初长度**
  - 前负荷**↑** -> 心肌初长度↑ -> 粗细肌丝重叠↑ -> 收缩力**↑** -> 搏出量**↑**
  - 心室功能曲线（Frank-Starling曲线）：升支段工作 -> 有**较大初长度储备** -> **不出现降支**（区别于骨骼肌）
  - 异长自身调节：改**初长度** -> 改搏出量 -> 精细调节 -> 使射血量 = 回心血量 -> 防肺淤血
  - 前负荷↑的因素：静脉回流**↑**、输液 -> 充盈↑
  - 前负荷↓的因素：心率**↑**（充盈时间↓）、心包积液、心肌肥厚（顺应性↓）
  - 关键：不依赖神经体液 -> **内在适应性**调节
- 后负荷（大动脉血压）
  - 本质：心室射血必须对抗的**大动脉血压**
  - 后负荷**↑** -> 等容收缩期**延长** -> 射血期**缩短** -> 搏出量**暂时↓**
  - 代偿：剩余血量↑ -> EDV↑ -> 异长自身调节 -> 搏出量**恢复**
  - 短期 ≤ **160mmHg**：可完全代偿
  - 长期↑（高血压）：心室肌**代偿性肥大** -> 心肌重构 -> **心力衰竭**
  - 临床：高血压是心衰**重要病因**
- 心肌收缩能力（等长调节 / 变力状态）
  - 本质：**不依赖**前后负荷 -> 改变收缩**强度**与**速度**的内在特性
  - 等长调节：初长度**不变** -> 收缩力改变 -> 搏出量改变
  - 收缩力**↑**的因素：
    - 交感兴奋/儿茶酚胺 -> Ca²⁺内流↑ -> 活化横桥↑ -> **正性变力** -> 曲线**左上移**
    - 甲状腺激素 -> ATP酶活性↑
    - 心率↑ -> 阶梯现象 -> Ca²⁺积累
  - 收缩力**↓**的因素：心肌缺血、酸中毒 -> **负性变力** -> 曲线**右下移**
  - 意义：适应**代谢需求**变化 -> 神经体液调节的主要途径
- 心率（HR）
  - 正常：**60~100 次/分** -> CO 正常
  - 心率**↑**（≤ **160~180** 次/分）：CO**↑**（可达 2~2.5 倍）
  - 心率**↑↑**（> **180** 次/分）：舒张期**显著缩短** -> 充盈**不足** -> SV**↓** -> CO**↓**
  - 心率**↓↓**（< **40** 次/分）：SV 虽有↑但不足以代偿 -> CO**↓**
  - 影响因素：交感↑ -> 心率↑；迷走↑ -> 心率↓；体温**每↑1℃** -> 心率↑约 **10 次/分**
  - 血 K⁺↑ -> 心率↓；血 Ca²⁺↑ -> 心率↑
- 三种调节的横向对比
  - 异长自身调节：改**初长度** -> 适应**回心血量**变化 -> 内在机制
  - 等长调节：改**收缩力** -> 适应**代谢需求**变化 -> 依赖神经体液
  - 后负荷调节：血压变 -> 暂时影响 -> 引发异长调节**代偿**
