# 4.1 课后练习题名词解释

- **心动周期**
  - 定义：心脏**一次收缩和舒张**构成的**机械活动周期**，包括**收缩期**和**舒张期**
  - 正常值/公式：与心率成**反比**，心率 75 次/min 时为 **0.8s**（60s ÷ 75）
  - 生理意义/特征：心脏泵血活动的**基本时间单位**；收缩期**短于**舒张期；心动过速时**舒张期缩短比例更大**，不利于心脏长期工作
- **心指数**
  - 定义：**单位体表面积**（每平方米）的**心输出量**
  - 正常值/公式：CI = **CO ÷ 体表面积**；静息心指数 **3.0~3.5 L/(min·m²)**（中等身材成人）
  - 生理意义/特征：评价**不同身材个体**的心功能，**消除体型差异**影响；CI < **2.5** L/(min·m²) 提示心功能不全
- **异长自身调节**
  - 定义：通过改变**心肌细胞初长度**（前负荷）而引起心肌收缩强度改变，进而调节**搏出量**的机制，又称 **Starling 机制**
  - 正常值/公式：前负荷改变 -> 初长度改变 -> 收缩力改变；**不依赖神经体液**因素
  - 生理意义/特征：对搏出量进行**精细调节**，使**心室射血量**与**静脉回心血量**保持**平衡**；保证**左右心室**输出量一致；是心脏的**内在适应性**调节
- **房室延搁**
  - 定义：兴奋在**房室交界区**传导速度**最慢**（仅约 0.02~0.05 m/s），耗时约 **0.1s**，使心房兴奋传递到心室出现**延迟**的现象
  - 正常值/公式：延搁时间约 **0.1s**（房室交界区传导速度最慢）
  - 生理意义/特征：保证**心房收缩完毕后**心室才开始收缩 -> 实现房室**顺序收缩** -> 心室有充分时间**充盈** -> 防止房室**同时收缩**
- **平均动脉压**
  - 定义：一个**心动周期**中动脉血压的**平均值**（非收缩压与舒张压的简单平均）
  - 正常值/公式：MAP = **舒张压 + 1/3 脉压**（脉压 = 收缩压 - 舒张压）；正常值约 **100 mmHg**（≈ 70 + 1/3×40）
  - 生理意义/特征：反映**器官灌注压**的核心指标；是计算**搏功**和评价组织**血液供应**的基础参数
- **微循环**
  - 定义：**微动脉**与**微静脉**之间的**血液循环**，是血液与组织液进行**物质交换**的场所
  - 正常值/公式：由七部分组成：微动脉 + 后微动脉 + 毛细血管前括约肌 + 真毛细血管 + 通血毛细血管 + 动-静脉吻合支 + 微静脉
  - 生理意义/特征：实现**物质交换**（营养、O₂、CO₂、代谢产物）；三条通路：迂回通路（交换）、直捷通路（快速回流）、动-静脉短路（体温调节）
- **心血管中枢**
  - 定义：**中枢神经系统**内与**心血管反射**有关的**神经元集中**部位
  - 正常值/公式：基本中枢位于**延髓**，包括心交感中枢、心迷走中枢（心抑制中枢）、交感缩血管中枢
  - 生理意义/特征：持续发放**紧张性冲动**，维持心血管系统基础活动；延髓以上高位中枢（下丘脑、大脑皮层）起**整合调节**作用
- **中心静脉压**
  - 定义：**右心房**和**胸腔内大静脉**的血压，反映**右心功能**与**静脉回心血量**之间的平衡关系
  - 正常值/公式：正常值 **4~12 cmH₂O**
  - 生理意义/特征：CVP**↑** -> 回心血量↑ 或 右心功能↓；CVP**↓** -> 回心血量不足或血容量减少；临床用于指导**补液**（CVP < 5 cmH₂O 提示血容量不足）
- **正常起搏点**
  - 定义：主导整个心脏**节律性兴奋**的自律细胞所在部位，即**窦房结**
  - 正常值/公式：窦房结自律性**最高**（约 **100 次/min**）> 房室交界（40~60 次/min）> 浦肯野纤维（15~40 次/min）
  - 生理意义/特征：窦房结通过**抢先占领**和**超速驱动压抑**控制其他自律组织；窦房结以外的自律组织为**潜在起搏点**（异位起搏点）；窦房结功能障碍时潜在起搏点可取而代之
- **弹性储器血管**
  - 定义：**主动脉**和**肺动脉**主干及其**大分支**，管壁富含**弹性纤维**，具有**扩张**和**弹性回缩**能力的大血管
  - 正常值/公式：包括主动脉、肺动脉主干及大分支；管壁弹性纤维丰富、平滑肌少
  - 生理意义/特征：射血期**扩张储存**血液 -> 将心室间断射血变为血管内**连续血流**；舒张期**弹性回缩** -> 推动血液继续流动 -> 维持**舒张压**；缓冲**收缩压**，减小脉压
