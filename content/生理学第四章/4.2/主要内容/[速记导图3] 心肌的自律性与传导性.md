# 心肌的自律性与传导性 — 极简速记导图

- 自律性定义：**无外来刺激** -> 自动发生**节律性兴奋** -> 基础 = **4 期自动去极化**
- 衡量指标：**心率**（高低）、**心律**（规整度）
- 心脏的起搏点（★必考）
  - 正常起搏点：**窦房结 P 细胞**
    - 自律性**最高**：约 **100 次/分**
    - 主导整个心脏 -> **窦性节律**
  - 潜在起搏点（自律性较低，正常不表现）
    - 房室交界：约 **50 次/分**
    - 浦肯野细胞：约 **25 次/分**
  - 异位起搏点：潜在起搏点在病理情况下取而代之 -> **异位节律**
  - 窦房结控制潜在起搏点的机制
    - **抢先占领**：窦房结自律性最高 -> 抢先发放冲动 -> 潜在起搏点未达阈电位即被兴奋
    - **超速驱动压抑**：窦房结频率快 -> 持续快速驱动 -> 压抑潜在起搏点自律性
- 影响自律性的因素
  - 最大复极电位与阈电位**差距**：差距↑ -> 自律性**↓**
  - **4 期自动去极化速率**（最重要）：速率↑ -> 自律性**↑**
    - 交感神经 -> 4 期去极速度↑ -> 自律性**↑** -> 心率**↑**
    - 迷走神经 -> 4 期去极速度↓ -> 自律性**↓** -> 心率**↓**
- 心肌传导性定义：传导兴奋的能力 -> 衡量指标：**传播速度**
- 心脏内兴奋传导途径
  - 窦房结 -> 心房肌（优势传导通路）-> **房室交界** -> 房室束 -> 左右束支 -> **浦肯野纤维** -> 心室肌
  - 传导速度：浦肯野纤维**最快** -> 房室交界**最慢**
- 房室延搁（★必考大题）
  - 位置：**房室交界** -> 传导速度**最慢**（仅 0.02~0.05 m/s）
  - 耗时：约 **0.1s**
  - 机制：房室交界细胞为**慢反应细胞** -> 0 期 Ca²⁺ 内流 -> 去极速度慢 -> 传导慢
  - 生理意义：保证**心房收缩完毕后**心室才开始收缩
    - -> 心室有充分时间**充盈**
    - -> 防止房室**同时收缩**
    - -> 利于心室**射血**
  - 临床联系：房室传导阻滞 -> 房室延搁延长或中断 -> 影响心室充盈和射血
- 影响传导性的因素
  - 结构因素：细胞**直径**大 -> 传导快；**缝隙连接**多 -> 传导快
  - 生理因素
    - **0 期去极速度与幅度**：速度↑幅度↑ -> 传导**↑**
    - **邻近未兴奋膜的兴奋性**：兴奋性↑ -> 传导**↑**
- 三大电生理特性对比速查
  - 兴奋性：能产生 AP -> 衡量：阈值 -> ERP 最长 = 不强制
  - 自律性：能自动产生 AP -> 衡量：心率/心律 -> 基础 = 4 期自动去极
  - 传导性：能传导 AP -> 衡量：传播速度 -> 房室延搁 = 最慢 = 保护性延迟
