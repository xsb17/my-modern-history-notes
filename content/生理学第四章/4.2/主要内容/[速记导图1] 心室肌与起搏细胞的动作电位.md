# 心室肌与起搏细胞的动作电位 — 极简速记导图

- 心肌细胞分类
  - 工作细胞（心房肌/心室肌）：有收缩性 -> 无自律性 -> **静息电位稳定**
  - 自律细胞（窦房结/浦肯野等）：无收缩性 -> 有自律性 -> **4 期自动去极化**
- 心室肌细胞动作电位（快反应细胞，5 个时相）
  - 静息电位：**-80~-90mV** -> 由 **K⁺ 外流**（Ik₁）维持
  - 0 期（去极化期）
    - **-90mV → +30mV** -> 幅度约 **120mV** -> 仅 **1~2ms**
    - 机制：**Na⁺ 快速内流**（快钠通道 INa）-> 阈电位 **-70mV**
    - 特点：速度快、幅度大 -> **快反应动作电位**
  - 1 期（快速复极初期）
    - **+30mV → 0mV** -> 约 **10ms**
    - 机制：Na⁺ 通道**失活** -> Na⁺ 停止内流 + **K⁺ 快速外流**（Ito 通道）
    - 与 0 期构成**锋电位**
  - 2 期（平台期）★核心考点
    - 膜电位维持 **0mV 左右** -> 约 **100~150ms**
    - 机制：**Ca²⁺ 缓慢内流**（L 型钙通道）与 **K⁺ 外流** 达到**平衡**
    - 内向 = 外向 -> 形成平台
    - 意义：导致 AP 时程**最长** -> 是心肌**区别于骨骼肌和神经**的**主要特征**
  - 3 期（快速复极末期）
    - **0mV → -90mV** -> 约 **100~150ms**
    - 机制：**Ca²⁺ 通道失活** -> Ca²⁺ 停止内流 + **K⁺ 迅速外流**↑
  - 4 期（静息期）
    - 膜电位稳定于**静息电位**
    - 机制：**钠泵** + **钠钙交换体** + **钙泵** -> 恢复离子分布
    - 工作细胞 4 期**无自动去极化**
- 窦房结 P 细胞动作电位（慢反应细胞，自律性最高）
  - 最大复极电位：**-70mV** -> 阈电位：**-40mV**
  - 仅有 **0 期、3 期、4 期** -> 无 1 期和 2 期（无平台期）
  - 0 期：**Ca²⁺ 内流**（慢钙通道）-> 速度**慢**、幅度**小** -> **慢反应电位**
  - 3 期：**K⁺ 快速外流**（Ik 通道）
  - 4 期自动去极化（★核心机制）
    - **K⁺ 外流进行性衰减**（Ik，最重要离子基础）
    - **Na⁺ 内流进行性增强**（If）
    - **Ca²⁺ 内流**（T 型钙通道，ICa-T）
    - 自动去极速度**最快** -> 自律性**最高**（约 100 次/min）
- 浦肯野纤维动作电位（快反应自律细胞）
  - 最大复极电位：**-90mV** -> 阈电位：**-70mV**
  - 0~3 期与心室肌**基本相似**（有平台期）
  - 4 期有自动去极化 -> **Na⁺ 内流**（If）-> 速度**慢于**窦房结 -> 自律性**较低**
- 快反应 vs 慢反应对比
  - 快反应（心室肌/浦肯野）：0 期 **Na⁺ 内流** -> 速度快、幅度大 -> 有**平台期** -> 5 个时相
  - 慢反应（窦房结/房室交界）：0 期 **Ca²⁺ 内流** -> 速度慢、幅度小 -> **无平台期** -> 3 个时相
