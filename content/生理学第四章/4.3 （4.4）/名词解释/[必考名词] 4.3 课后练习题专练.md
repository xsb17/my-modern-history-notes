# 4.3 & 4.4 课后练习题名词解释

- **平均动脉压**
  - 定义：一个**心动周期**中每一瞬间动脉血压的**平均值**，反映**器官灌注压**的核心指标
  - 正常值/公式：MAP ≈ **舒张压 + 1/3 脉压**（脉压 = 收缩压 - 舒张压）；正常值约 **100 mmHg**
  - 生理意义/特征：反映循环系统对组织器官的**血液灌注**能力；是计算**搏功**和评价组织**血液供应**的基础参数；压力感受性反射在 MAP ≈ **100 mmHg** 处最敏感
- **微循环**
  - 定义：**微动脉**与**微静脉**之间的**血液循环**，是血液与组织细胞进行**物质交换**的场所
  - 正常值/公式：典型微循环由七部分组成 → 微动脉 + 后微动脉 + 毛细血管前括约肌 + 真毛细血管 + 通血毛细血管 + 动静脉吻合支 + 微静脉
  - 生理意义/特征：三条血流通路各具功能 → **迂回通路**完成**物质和气体交换**（营养通路）、**直捷通路**保证**回心血量**、**动静脉短路**利于**散热**调节**体温**
- **心血管中枢**
  - 定义：**中枢神经系统**内与**心血管反射**有关的**神经元集中**部位，最基本的中枢位于**延髓**
  - 正常值/公式：延髓分为四个功能区 → **缩血管区**（延髓头端腹外侧，交感紧张起源地）+ **舒血管区**（延髓尾端腹外侧）+ **心抑制区**（迷走背核和疑核）+ **传入接替站**（孤束核）
  - 生理意义/特征：持续发放**紧张性冲动**，维持心血管系统基础活动；延髓是**最基本**的心血管中枢（生命中枢）；延髓以上高位中枢（下丘脑、大脑皮层）起**整合调节**作用
- **中心静脉压**
  - 定义：**右心房**和**胸腔内大静脉**（上、下腔静脉胸腔段）的血压，反映**右心功能**与**静脉回心血量**之间的平衡关系
  - 正常值/公式：正常值 **4~12 cmH₂O**；取决于心室**射血能力**与静脉**回心血量**的相互关系
  - 生理意义/特征：CVP**↑** → 回心血量↑ 或 右心功能↓（心衰/大量快速输液）；CVP**↓** → 回心血量不足或血容量减少（失血）；临床上常用来监测**补液速度**和**补液量**，是反映**心血管功能**的重要指标
- **弹性储器血管**
  - 定义：**主动脉**和**肺动脉**主干及其**最大分支**，管壁富含**弹性纤维**，具有**扩张储存**和**弹性回缩**能力的大血管
  - 正常值/公式：管壁厚，弹性纤维丰富，弹性大，可扩张性强
  - 生理意义/特征：射血期**扩张储存**血液 → 将心室**间断射血**转变为血管内**连续血流**；舒张期**弹性回缩** → 推动血液继续流向外周，维持**舒张压**；**缓冲**动脉血压的大幅波动，减小**脉压**；老年人弹性↓ → 收缩压↑ + 舒张压↓ → 脉压**增大**
- **心动周期**
  - 定义：心脏**一次收缩和舒张**构成的**机械活动周期**，包括收缩期和舒张期
  - 正常值/公式：与心率成**反比**，心率 75 次/min 时为 **0.8s**
  - 生理意义/特征：心脏泵血活动的**基本时间单位**；收缩期短于舒张期；心动过速时**舒张期缩短比例更大**，不利于心脏长期工作
- **心指数**
  - 定义：**单位体表面积**的**心输出量**，消除体型差异
  - 正常值/公式：CI = CO ÷ 体表面积；静息心指数 **3.0~3.5 L/(min·m²)**
  - 生理意义/特征：评价**不同身材个体**的心功能；CI < **2.5** 提示心功能不全
- **异长自身调节**
  - 定义：通过改变**心肌细胞初长度**（前负荷）而引起收缩强度改变，调节**搏出量**的机制（Starling 机制）
  - 正常值/公式：不依赖**神经体液**因素
  - 生理意义/特征：使**心室射血量**与**静脉回心血量**保持平衡；左右心室输出量一致；心脏的**内在适应性**调节
- **房室延搁**
  - 定义：兴奋在**房室交界区**传导速度**最慢**（慢反应细胞，0 期 Ca²⁺ 内流），耗时约 **0.1s**，使房室兴奋出现**延迟**
  - 正常值/公式：延搁约 **0.1s**；房室交界是兴奋传入心室的**唯一通路**
  - 生理意义/特征：保证**心房收缩完毕后**心室才开始收缩 → 心室充分**充盈** → 利于**射血**；防止房室**同时收缩**
- **正常起搏点**
  - 定义：主导心脏**节律性兴奋**的自律细胞部位 → **窦房结 P 细胞** → 形成**窦性节律**
  - 正常值/公式：自律性约 **100 次/分**（最高）；潜在起搏点：房室交界约 50、浦肯野约 25 次/分
  - 生理意义/特征：通过**抢先占领**和**超速驱动压抑**控制其他自律组织；病理时潜在起搏点可成为**异位起搏点** → 异位节律
