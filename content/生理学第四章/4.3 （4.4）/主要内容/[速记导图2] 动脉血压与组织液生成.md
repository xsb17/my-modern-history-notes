# 动脉血压与组织液生成 — 极简速记导图

- 血压定义：血管内血液对管壁的**侧压力**
- 动脉血压形成：**一个前提 + 两个因素 + 一个辅助**
  - 前提：循环系统**足够血液充盈** -> 平均充盈压 ≈ **7mmHg**
  - 因素1：**心脏射血** -> 能量来源 -> 必要条件
  - 因素2：**外周阻力**（小/微动脉）-> 阻止血液过快流走 -> 维持舒张压
  - 辅助：大动脉**弹性储器**作用 -> 缓冲 + 连续血流
- 动脉血压正常值（安静健康青年）
  - 收缩压：**100~120 mmHg**
  - 舒张压：**60~80 mmHg**
  - 脉压：**30~40 mmHg**（收缩压 - 舒张压）
  - 平均动脉压：≈ **舒张压 + 1/3脉压** ≈ **100 mmHg**
- 影响动脉血压的五大因素（假定单一因素变化）
  - 搏出量**↑** -> 收缩压**↑↑** -> 脉压**↑**（收缩压主要反映**搏出量**）
  - 心率**↑** -> 舒张期缩短 -> 舒张压**↑↑** -> 脉压**↓**（心率主要影响**舒张压**）
  - 外周阻力**↑** -> 舒张压**↑↑** -> 脉压**↓**（舒张压反映**外周阻力**大小）
  - 大动脉弹性**↓**（老年）-> 收缩压**↑** + 舒张压**↓** -> 脉压**↑↑**
  - 循环血量**↓** / 血管容量**↑** -> 平均充盈压↓ -> 血压**↓**
- 中心静脉压（CVP）
  - 定义：**右心房**和胸腔内大静脉血压
  - 正常值：**4~12 cmH₂O**
  - 取决于：心室**射血能力** 与 静脉**回心血量** 的平衡
  - CVP **↓** -> 血容量不足；CVP **↑** -> 心衰/输液过量
- 影响静脉回心血量的因素
  - 体循环平均充盈压**↑** -> 回心血量**↑**
  - 心肌收缩力**↑** -> 舒张期**抽吸力**↑ -> 回心血量**↑**
  - 卧→立位 -> 下肢血管扩张 -> 回心血量**↓**
  - 骨骼肌运动**↑** -> 挤压+抽吸 -> 回心血量**↑**
  - 吸气 -> 胸内负压**↑** -> 腔静脉扩张 -> 回心血量**↑**
- 微循环
  - 定义：微动脉 ↔ 微静脉之间的血液循环
  - 三条通路：
    - 迂回通路（营养通路）：真毛细血管 -> **物质交换**
    - 直捷通路：通血毛细血管 -> **保证回心血量**
    - 动静脉短路：吻合支 -> **体温调节**
- 组织液生成
  - 动力：**有效滤过压**
  - 有效滤过压 =（**毛细血管血压** + 组织液胶渗压）-（**血浆胶渗压** + 组织液静水压）
  - 正值 -> **滤过**（生成）；负值 -> **重吸收**（回流）
- 影响组织液生成的四大因素（→ 水肿机制★必考）
  - 毛细血管血压**↑**（右心衰/炎症）-> 有效滤过压↑ -> 生成↑ -> **水肿**
  - 血浆胶体渗透压**↓**（肝病/肾病/营养不良 -> 血浆蛋白↓）-> 有效滤过压↑ -> 生成↑ -> **水肿**
  - 毛细血管壁通透性**↑**（过敏/烧伤 -> 蛋白漏出）-> 组织液胶渗压↑ -> 生成↑ -> **水肿**
  - 淋巴回流**受阻**（丝虫病）-> 组织液积聚 -> **水肿（象皮肿）**
