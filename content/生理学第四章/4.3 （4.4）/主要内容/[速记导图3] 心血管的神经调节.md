# 心血管的神经调节 — 极简速记导图

- 心交感神经（**三正作用**）
  - 起源：脊髓 **T1~T5** 中间外侧柱
  - 右交感 -> **窦房结**（变时）；左交感 -> **房室交界**（变传导）
  - 递质：节后纤维释放 **NE**（去甲肾上腺素）
  - 受体：心肌细胞膜 **β₁ 受体**
  - 效应：
    - 正性**变时**：心率**↑**
    - 正性**变力**：心肌收缩力**↑**
    - 正性**变传导**：房室传导速度**↑**
  - 机制：NE + β₁ -> cAMP↑ -> Ca²⁺内流↑ -> 收缩力↑ + 心率↑
- 心迷走神经（**三负作用**）
  - 起源：延髓**迷走神经背核**和**疑核**
  - 右迷走 -> **窦房结**；左迷走 -> **房室交界**
  - 递质：节后纤维释放 **ACh**（乙酰胆碱）
  - 受体：心肌细胞膜 **M 受体**
  - 效应：
    - 负性**变时**：心率**↓**
    - 负性**变力**：心肌收缩力**↓**（心房为主，心室弱）
    - 负性**变传导**：房室传导速度**↓**
  - 机制：ACh + M -> K⁺外流↑ + Ca²⁺内流↓ -> 自律性↓ + 收缩力↓
- 交感神经 vs 迷走神经对比
  - 对心脏：交感 -> **NE + β₁** -> 三正；迷走 -> **ACh + M** -> 三负
  - 安静时：**迷走紧张**占优势 -> 维持心率在 60~100 次/分
  - 运动/应激：**交感紧张**增强 -> 心率↑、收缩力↑
- 交感缩血管神经（★最主要血管神经）
  - 递质：节后纤维释放 **NE**（去甲肾上腺素）
  - 受体：血管平滑肌 **α₁ 受体**（主要）+ β₂ 受体
  - α₁ 受体亲和力 **>** β₂ -> 主要产生**缩血管效应**
  - 支配特点：
    - 大多数血管**仅接受**交感缩血管神经支配
    - 密度：**皮肤 > 骨骼肌/内脏 > 冠脉/脑血管**
    - 持续发放**紧张性冲动** -> 维持血管基础张力
  - 紧张性↑ -> 血管收缩 -> 外周阻力↑ -> 血压**↑**
  - 紧张性↓ -> 血管舒张 -> 外周阻力↓ -> 血压**↓**
- 舒血管神经纤维（次要，末梢均释放 **ACh**）
  - 交感舒血管神经：支配**骨骼肌血管**
  - 副交感舒血管神经：支配脑膜/唾液腺/胃肠外分泌腺/外生殖器血管
  - 脊髓背根舒血管神经：支配皮肤微动脉（轴突反射）
- 心血管中枢
  - 脊髓：最后传出通路
  - 延髓（**最基本心血管中枢**）：
    - 缩血管区（**延髓头端腹外侧**）：交感紧张起源地
    - 舒血管区（延髓尾端腹外侧）：抑制缩血管紧张
    - 心抑制区（**迷走背核+疑核**）：迷走紧张起源地
    - 传入接替站（**孤束核**）：接受分配传入信息
  - 延髓以上（下丘脑等）：**整合**作用
- 记忆口诀
  - 交感三正 = NE + β₁ = **快、强、通**
  - 迷走三负 = ACh + M = **慢、弱、堵**
  - 血管收缩 = NE + α₁ = **皮肤最密、冠脉最稀**
