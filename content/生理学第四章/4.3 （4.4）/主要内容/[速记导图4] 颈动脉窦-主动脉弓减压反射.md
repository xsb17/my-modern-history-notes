# 颈动脉窦-主动脉弓减压反射 — 极简速记导图

- 压力感受性反射（减压反射）★必考论述题
- 反射弧组成（五段式）
  - 感受器：**颈动脉窦** + **主动脉弓** 压力感受器
    - 实质：血管外膜下**感觉神经末梢**
    - 感受的不是血压本身 -> 而是血管壁的**机械牵张程度**
    - 牵张↑ -> 传入冲动频率↑（在一定范围内与扩张程度成正比）
  - 传入神经：窦神经（→**舌咽神经**）+ **迷走神经**
  - 中枢：**延髓心血管中枢**
  - 传出神经：心交感神经 + **心迷走神经** + 交感缩血管神经
  - 效应器：**心脏** + **血管**
- 反射过程（血压↑时）★箭头必考链
  - 血压**↑** -> 动脉扩张 -> 管壁牵张**增强**
  - -> 压力感受器**兴奋** -> 传入冲动**增多**
  - -> 延髓心血管中枢整合：
    - 心迷走紧张**↑**（心抑制区兴奋）
    - 心交感紧张**↓**（缩血管区抑制）
    - 交感缩血管紧张**↓**
  - -> 传出效应：
    - 心率**↓** + 心肌收缩力**↓** -> 心输出量**↓**
    - 血管**舒张** -> 外周阻力**↓**
  - -> **血压回降**（负反馈）
- 血压↓时的反射（反向）
  - 血压**↓** -> 牵张**减弱** -> 传入冲动**减少**
  - -> 心迷走紧张**↓** + 心交感紧张**↑** + 缩血管紧张**↑**
  - -> 心率**↑** + 收缩力**↑** + 血管**收缩**
  - -> **血压回升**
- 特点
  - 对血压**突然变化**最敏感（缓慢变化不敏感）
  - 最敏感范围：MAP ≈ **100 mmHg** 附近
  - 可进行**重调定**（长期高血压 -> 调定点上移）
  - **快速、双向**调节
- 生理意义（★必考）
  - **缓冲**动脉血压的**突然波动** -> 维持血压**相对稳定**
  - 是机体最重要的**负反馈**血压调节机制
  - 经常性监视血压波动 -> 使血压不致过高或过低
- 化学感受性反射（对比补充）
  - 感受器：**颈动脉体** + **主动脉体**
  - 刺激：**PO₂↓** / **PCO₂↑** / **[H⁺]↑**
  - 平时主要调节**呼吸**（呼吸加深加快）
  - 严重缺氧/大失血时 -> 升压效应 -> **血液重新分配** -> 保证**心脑供血**
- 肾素-血管紧张素系统（RAS）★体液调节核心
  - 链：血管紧张素原 -> **肾素** -> AngⅠ -> **ACE** -> AngⅡ -> AngⅢ
  - AngⅡ（最重要）：
    - 收缩**微动脉** -> 外周阻力↑ -> 血压↑
    - 促进**NE**释放 -> 缩血管↑
    - 刺激**醛固酮** -> 水钠潴留 -> 血容量↑
    - 刺激**渴觉** -> 饮水↑
- 肾上腺素 vs 去甲肾上腺素
  - Adr（肾上腺素）：**强心**为主 -> **β₁**（心肌↑）+ β₂（舒张）+ α₁（收缩皮肤内脏）
  - NE（去甲肾上腺素）：**缩血管/升压**为主 -> **α₁**（全身缩）+ β₁（弱）+ β₂（几乎无作用）
  - 肾上腺髓质分泌：80% Adr + 20% NE
- 减压反射核心口诀
  - 血压↑ -> 牵张↑ -> 迷走↑/交感↓ -> 心慢+血管扩 -> 血压↓
  - 血压↓ -> 牵张↓ -> 迷走↓/交感↑ -> 心快+血管缩 -> 血压↑
  - 快、准、稳 -> 负反馈维持稳态
