# 第四章第三/四节：血管生理与心血管活动的调节 — 全集存档&nbsp;&nbsp;&nbsp;

## 第三节 血管生理&nbsp;&nbsp;&nbsp;

### 一、各类血管的功能特点&nbsp;&nbsp;&nbsp;

#### （一）弹性储器血管（大动脉）&nbsp;&nbsp;&nbsp;

- 结构：主动脉、肺动脉主干及最大分支&nbsp;&nbsp;&nbsp;
- 特点：管壁厚，富含**弹性纤维**，弹性大，可扩张性强&nbsp;&nbsp;&nbsp;
- 作用：容纳血液、**缓冲血压**、将心脏**间断射血**转变为血管内**连续血流**&nbsp;&nbsp;&nbsp;
- 射血期：扩张储存血液；舒张期：弹性回缩推动血液继续流动&nbsp;&nbsp;&nbsp;

#### （二）分配血管（中动脉）&nbsp;&nbsp;&nbsp;

- 结构：弹性储器血管之后、分支到小动脉前的动脉管道&nbsp;&nbsp;&nbsp;
- 特点：富含**平滑肌**纤维，收缩性强&nbsp;&nbsp;&nbsp;
- 作用：将血液**输送**至各组织器官&nbsp;&nbsp;&nbsp;

#### （三）毛细血管前阻力血管（小动脉、微动脉）&nbsp;&nbsp;&nbsp;

- 特点：口径小，管壁含**平滑肌较多**，对血流阻力大&nbsp;&nbsp;&nbsp;
- 作用：改变**血流阻力**、调控**组织血流量**，在**动脉血压的形成和维持**中起重要作用&nbsp;&nbsp;&nbsp;

#### （四）毛细血管前括约肌&nbsp;&nbsp;&nbsp;

- 位置：分布于真毛细血管起始部的平滑肌&nbsp;&nbsp;&nbsp;
- 作用：舒缩活动调控毛细血管**开闭**，决定某一时间内毛细血管**开放数量**&nbsp;&nbsp;&nbsp;

#### （五）交换血管（真毛细血管）&nbsp;&nbsp;&nbsp;

- 特点：管壁很薄，仅由单层**内皮细胞**和薄层基膜组成，**通透性很高**&nbsp;&nbsp;&nbsp;
- 作用：血液与组织细胞进行**物质交换**的主要场所&nbsp;&nbsp;&nbsp;

#### （六）毛细血管后阻力血管（微静脉）&nbsp;&nbsp;&nbsp;

- 特点：管径较小，有一定血流阻力&nbsp;&nbsp;&nbsp;
- 作用：通过舒缩调节毛细血管前/后阻力比值，改变**毛细血管压**，影响**组织液滤过**和血容量&nbsp;&nbsp;&nbsp;

#### （七）容量血管（静脉系统）&nbsp;&nbsp;&nbsp;

- 特点：数量多、管径大、壁薄、扩张性大&nbsp;&nbsp;&nbsp;
- 作用：容纳较多血液，**60%~70%** 循环血量储存在静脉中，调节**静脉回心血量**&nbsp;&nbsp;&nbsp;

#### （八）动静脉短路&nbsp;&nbsp;&nbsp;

- 结构：微动脉与微静脉之间的直接**吻合支**&nbsp;&nbsp;&nbsp;
- 分布：手指、足趾、耳郭等皮肤中较多&nbsp;&nbsp;&nbsp;
- 作用：开闭与**体温调节**有关&nbsp;&nbsp;&nbsp;

### 三、动脉血压与动脉脉搏&nbsp;&nbsp;&nbsp;

#### （一）动脉血压&nbsp;&nbsp;&nbsp;

- 血压定义：血管内流动的血液对单位面积管壁产生的**侧压力**&nbsp;&nbsp;&nbsp;
- 动脉血压定义：动脉内的血液对单位面积血管壁产生的侧压力&nbsp;&nbsp;&nbsp;

##### 1. 动脉血压的形成&nbsp;&nbsp;&nbsp;

- 一个前提：循环系统内有**足够的血液充盈**&nbsp;&nbsp;&nbsp;
  - 平均充盈压：表示血液充盈程度，取决于循环血量与血管容量的相对关系，约 **7mmHg**&nbsp;&nbsp;&nbsp;
- 两个因素：&nbsp;&nbsp;&nbsp;
  - 心脏射血：动脉血压形成的**必要条件**，压强能的**能量来源**&nbsp;&nbsp;&nbsp;
  - 外周阻力：主要指**小动脉和微动脉**对血流的阻力&nbsp;&nbsp;&nbsp;
- 辅助因素：主动脉和大动脉的**弹性储器作用**&nbsp;&nbsp;&nbsp;

##### 2. 动脉血压的正常值&nbsp;&nbsp;&nbsp;

- 收缩压：心室收缩时主动脉压力峰值，正常 **100~120 mmHg**&nbsp;&nbsp;&nbsp;
- 舒张压：心室舒张末期主动脉压最低值，正常 **60~80 mmHg**&nbsp;&nbsp;&nbsp;
- 脉搏压（脉压）：收缩压 - 舒张压，正常 **30~40 mmHg**&nbsp;&nbsp;&nbsp;
- 平均动脉压（MAP）：≈ **舒张压 + 1/3 脉压**，约 **100 mmHg**&nbsp;&nbsp;&nbsp;

##### 3. 影响动脉血压的因素&nbsp;&nbsp;&nbsp;

- （1）搏出量：主要导致**收缩压**改变 → 收缩压高低反映**搏出量**的多少&nbsp;&nbsp;&nbsp;
- （2）心率：变化对**舒张压**影响更明显 → 心率↑ → 舒张期缩短 → 舒张压**↑**&nbsp;&nbsp;&nbsp;
- （3）外周阻力：主要影响**舒张压** → 舒张压高低反映**外周阻力**的大小&nbsp;&nbsp;&nbsp;
- （4）主动脉和大动脉弹性储器作用：弹性↓（老年人）→ 收缩压↑、舒张压↓ → **脉压增大**&nbsp;&nbsp;&nbsp;
- （5）循环血量与血管容量比例：失血（血量↓）→ 血压↓；扩容（血管容量↑）→ 血压↓&nbsp;&nbsp;&nbsp;

### 四、静脉血压和静脉回心血量&nbsp;&nbsp;&nbsp;

#### （一）静脉血压&nbsp;&nbsp;&nbsp;

- 中心静脉压（CVP）：**右心房**和**胸腔内大静脉**的血压&nbsp;&nbsp;&nbsp;
  - 正常值：**4~12 cmH₂O**&nbsp;&nbsp;&nbsp;
  - 取决于**心室射血能力**与**静脉回心血量**之间的相互关系&nbsp;&nbsp;&nbsp;
  - 临床意义：监测**补液速度**和**补液量**的重要指标&nbsp;&nbsp;&nbsp;
  - CVP↓ → 血容量不足；CVP↑ → 心衰或输液过量&nbsp;&nbsp;&nbsp;
- 外周静脉压（PVP）：各器官静脉的血压&nbsp;&nbsp;&nbsp;

#### （二）静脉回心血量的影响因素&nbsp;&nbsp;&nbsp;

- 体循环平均充盈压↑ → 外周静脉压与中心静脉压差增大 → 回心血量**↑**&nbsp;&nbsp;&nbsp;
- 心肌收缩力↑ → 心脏舒张期**抽吸力**增大 → 回心血量**↑**&nbsp;&nbsp;&nbsp;
- 体位改变（卧位→立位）→ 身体低垂部分血管扩张 → 回心血量**↓**&nbsp;&nbsp;&nbsp;
- 骨骼肌挤压作用↑（肌肉运动）→ 挤压+抽吸 → 回心血量**↑**&nbsp;&nbsp;&nbsp;
- 呼吸运动（吸气→胸内负压↑）→ 腔静脉扩张 → 回心血量**↑**&nbsp;&nbsp;&nbsp;

### 五、微循环&nbsp;&nbsp;&nbsp;

- 定义：**微动脉**与**微静脉**之间的血液循环，是**物质交换**的场所&nbsp;&nbsp;&nbsp;
- 组成（七部分）：微动脉 → 后微动脉 → 毛细血管前括约肌 → 真毛细血管 → 通血毛细血管 → 动静脉吻合支 → 微静脉&nbsp;&nbsp;&nbsp;

#### 微循环的三条血流通路&nbsp;&nbsp;&nbsp;

- 迂回通路（营养通路）：微动脉 → 后微动脉 → 毛细血管前括约肌 → 真毛细血管 → 微静脉 → **完成物质和气体交换**&nbsp;&nbsp;&nbsp;
- 直捷通路：微动脉 → 后微动脉 → 通血毛细血管 → 微静脉 → **保证回心血量**&nbsp;&nbsp;&nbsp;
- 动静脉短路：微动脉 → 动静脉吻合支 → 微静脉 → **利于散热，调节体温**&nbsp;&nbsp;&nbsp;

### 六、组织液&nbsp;&nbsp;&nbsp;

- 定义：细胞外液中存在于组织细胞间隙的液体，呈胶冻状&nbsp;&nbsp;&nbsp;
- 生成动力：**有效滤过压**&nbsp;&nbsp;&nbsp;

#### 有效滤过压公式&nbsp;&nbsp;&nbsp;

- 有效滤过压 =（**毛细血管血压** + **组织液胶体渗透压**）-（**血浆胶体渗透压** + **组织液静水压**）&nbsp;&nbsp;&nbsp;
- 正值 → 滤过（生成组织液）；负值 → 重吸收（回流）&nbsp;&nbsp;&nbsp;

#### 影响组织液生成与回流的因素&nbsp;&nbsp;&nbsp;

- 毛细血管血压↑（微动脉扩张/静脉回流受阻）→ 生成**↑** → 水肿（如右心衰、炎症）&nbsp;&nbsp;&nbsp;
- 血浆胶体渗透压↓（肝病/肾病/营养不良→血浆蛋白↓）→ 生成**↑** → 水肿&nbsp;&nbsp;&nbsp;
- 毛细血管壁通透性↑（过敏/烧伤→蛋白漏出）→ 组织液胶渗压↑ → 生成**↑** → 水肿&nbsp;&nbsp;&nbsp;
- 淋巴回流受阻（丝虫病）→ 回流**↓** → 组织液积聚 → 水肿（象皮肿）&nbsp;&nbsp;&nbsp;

## 第四节 心血管活动的调节&nbsp;&nbsp;&nbsp;

### 一、神经调节&nbsp;&nbsp;&nbsp;

#### （一）心脏和血管的神经支配&nbsp;&nbsp;&nbsp;

##### 1. 心脏的神经支配&nbsp;&nbsp;&nbsp;

###### （1）心交感神经&nbsp;&nbsp;&nbsp;

- 起源：脊髓 **T1~T5** 中间外侧柱&nbsp;&nbsp;&nbsp;
- 支配区域：右交感→**窦房结**（变时为主），左交感→**房室交界**（变传导为主）&nbsp;&nbsp;&nbsp;
- 递质与受体：节后纤维释放**去甲肾上腺素（NE）**，与心肌细胞膜 **β₁ 受体**结合&nbsp;&nbsp;&nbsp;
- 生理作用（**三正作用**）：&nbsp;&nbsp;&nbsp;
  - 正性**变时**作用：心率**加快**&nbsp;&nbsp;&nbsp;
  - 正性**变力**作用：收缩力**增强**&nbsp;&nbsp;&nbsp;
  - 正性**变传导**作用：房室传导**加速**&nbsp;&nbsp;&nbsp;

###### （2）心迷走神经&nbsp;&nbsp;&nbsp;

- 起源：延髓**迷走神经背核**和**疑核**&nbsp;&nbsp;&nbsp;
- 支配区域：右迷走→**窦房结**，左迷走→**房室交界**&nbsp;&nbsp;&nbsp;
- 递质与受体：节后纤维释放**乙酰胆碱（ACh）**，与心肌细胞膜 **M 受体**结合&nbsp;&nbsp;&nbsp;
- 生理作用（**三负作用**）：&nbsp;&nbsp;&nbsp;
  - 负性**变时**作用：心率**减慢**&nbsp;&nbsp;&nbsp;
  - 负性**变力**作用：收缩力**减弱**&nbsp;&nbsp;&nbsp;
  - 负性**变传导**作用：房室传导**减慢**&nbsp;&nbsp;&nbsp;

##### 2. 血管的神经支配&nbsp;&nbsp;&nbsp;

###### （1）缩血管神经纤维（仅有交感神经）&nbsp;&nbsp;&nbsp;

- 递质与受体：节后纤维释放**去甲肾上腺素（NE）**，主要结合血管平滑肌 **α₁ 受体**（与α₁亲和力 > β₂），产生**缩血管效应**&nbsp;&nbsp;&nbsp;
- 支配特点：大多数血管仅接受交感缩血管神经支配；密度：**皮肤 > 骨骼肌/内脏 > 冠脉/脑血管**&nbsp;&nbsp;&nbsp;
- 生理意义：持续发放紧张性冲动，维持血管**基础张力**&nbsp;&nbsp;&nbsp;

###### （2）舒血管神经纤维（末梢释放 ACh）&nbsp;&nbsp;&nbsp;

- 交感舒血管神经纤维：支配**骨骼肌血管**&nbsp;&nbsp;&nbsp;
- 副交感舒血管神经纤维：支配脑膜、唾液腺、胃肠外分泌腺、外生殖器血管&nbsp;&nbsp;&nbsp;
- 脊髓背根舒血管神经纤维：支配皮肤微动脉&nbsp;&nbsp;&nbsp;

#### （二）心血管中枢&nbsp;&nbsp;&nbsp;

- 脊髓：最后传出通路，受高位中枢控制&nbsp;&nbsp;&nbsp;
- 延髓（**最基本的心血管中枢**）：&nbsp;&nbsp;&nbsp;
  - 缩血管区（延髓头端腹外侧部）：心血管交感紧张起源地&nbsp;&nbsp;&nbsp;
  - 舒血管区（延髓尾端腹外侧部）：抑制缩血管神经紧张性活动&nbsp;&nbsp;&nbsp;
  - 心抑制区（迷走背核和疑核）：心迷走紧张起源地&nbsp;&nbsp;&nbsp;
  - 传入神经接替站（孤束核）：接受传入信息，分配至高级中枢&nbsp;&nbsp;&nbsp;
- 延髓以上中枢（下丘脑等）：起**整合**作用&nbsp;&nbsp;&nbsp;

#### （三）心血管反射&nbsp;&nbsp;&nbsp;

##### 1. 颈动脉窦-主动脉弓压力感受性反射（减压反射）&nbsp;&nbsp;&nbsp;

**反射弧组成：**&nbsp;&nbsp;&nbsp;
- 感受器：颈动脉窦、主动脉弓**压力感受器**（感受血管壁**机械牵张**程度）&nbsp;&nbsp;&nbsp;
- 传入神经：窦神经（→舌咽神经）、迷走神经&nbsp;&nbsp;&nbsp;
- 中枢：延髓心血管中枢&nbsp;&nbsp;&nbsp;
- 传出神经：心交感神经、心迷走神经、交感缩血管神经&nbsp;&nbsp;&nbsp;
- 效应器：心脏和血管&nbsp;&nbsp;&nbsp;

**反射过程（血压升高时）：**&nbsp;&nbsp;&nbsp;
- 血压↑ → 动脉扩张 → 管壁牵张↑ → 压力感受器兴奋 → 传入冲动↑&nbsp;&nbsp;&nbsp;
- → 延髓：心迷走紧张**↑**、心交感紧张**↓**、交感缩血管紧张**↓**&nbsp;&nbsp;&nbsp;
- → 心率↓、心肌收缩力↓ → 心输出量**↓**&nbsp;&nbsp;&nbsp;
- → 血管舒张 → 外周阻力**↓**&nbsp;&nbsp;&nbsp;
- → **血压回降**&nbsp;&nbsp;&nbsp;

**特点与意义：**&nbsp;&nbsp;&nbsp;
- 对血压**突然变化**敏感（最敏感范围：MAP ≈ **100 mmHg**）&nbsp;&nbsp;&nbsp;
- **负反馈**调节，快速、双向&nbsp;&nbsp;&nbsp;
- 可进行**重调定**&nbsp;&nbsp;&nbsp;
- 生理意义：**缓冲血压变化**，维持血压**相对稳定**&nbsp;&nbsp;&nbsp;

##### 2. 颈动脉体-主动脉体化学感受性反射&nbsp;&nbsp;&nbsp;

- 适宜刺激：血液 **PO₂↓**、**PCO₂↑**、**[H⁺]↑**&nbsp;&nbsp;&nbsp;
- 反射效应：刺激化学感受器 → 呼吸加深加快（主要） + 外周阻力↑ → 血压↑&nbsp;&nbsp;&nbsp;
- 特点：平时对血压无明显调节作用，主要影响**呼吸**；在**严重缺氧、大失血**时才表现升压效应&nbsp;&nbsp;&nbsp;
- 生理意义：增加外周阻力，使血液**重新分配**，保证**心脑血液供应**&nbsp;&nbsp;&nbsp;

### 二、体液调节&nbsp;&nbsp;&nbsp;

#### （一）肾素-血管紧张素系统（RAS）&nbsp;&nbsp;&nbsp;

- 生成链：血管紧张素原（肝合成）→ 肾素（肾近球细胞）→ **AngⅠ**（十肽）→ ACE（肺血管）→ **AngⅡ**（八肽）→ 血管紧张素酶A → **AngⅢ**（七肽）&nbsp;&nbsp;&nbsp;
- **AngⅡ** 的生物学作用（最重要）：&nbsp;&nbsp;&nbsp;
  - ① 直接收缩全身**微动脉** → 外周阻力↑ → 血压↑&nbsp;&nbsp;&nbsp;
  - ② 作用于交感缩血管神经 → 促进**NE**释放&nbsp;&nbsp;&nbsp;
  - ③ 影响中枢 → 抑制压力感受性反射、促进血管升压素释放&nbsp;&nbsp;&nbsp;
  - ④ 刺激**醛固酮**释放 → 水钠重吸收↑ → 增加体液量&nbsp;&nbsp;&nbsp;
  - ⑤ 刺激下丘脑 → 增加**渴觉** → 饮水↑&nbsp;&nbsp;&nbsp;

#### （二）肾上腺素（Adr）和去甲肾上腺素（NE）&nbsp;&nbsp;&nbsp;

- 来源：肾上腺髓质分泌（约 80% Adr + 20% NE）&nbsp;&nbsp;&nbsp;
- 肾上腺素（Adr）：&nbsp;&nbsp;&nbsp;
  - 与 **β₁ 受体**结合（心肌）→ 心率↑、收缩力↑ → 心输出量↑ → **"强心"效应**&nbsp;&nbsp;&nbsp;
  - 与 **β₂ 受体**结合（冠脉/脑血管/骨骼肌血管）→ 血管**舒张**&nbsp;&nbsp;&nbsp;
  - 与 **α₁ 受体**结合（皮肤/粘膜/内脏血管）→ 血管**收缩**&nbsp;&nbsp;&nbsp;
- 去甲肾上腺素（NE）：&nbsp;&nbsp;&nbsp;
  - 主要结合 **α₁ 受体** → 全身血管广泛**收缩** → 外周阻力↑ → 血压↑ → **"缩血管/升压"效应**&nbsp;&nbsp;&nbsp;
  - 结合 **β₁ 受体** → 心输出量↑ → 但由于血压↑刺激压力感受器 → 心率反**↓**&nbsp;&nbsp;&nbsp;

#### （三）血管升压素（抗利尿激素，ADH）&nbsp;&nbsp;&nbsp;

- 来源：下丘脑**视上核、室旁核**&nbsp;&nbsp;&nbsp;
- 生理浓度 → 作用于肾远曲小管和集合管 **V₂ 受体** → 促进水重吸收 → 减少尿量&nbsp;&nbsp;&nbsp;
- 高浓度 → 作用于血管平滑肌 **V₁ 受体** → 血管收缩 → 血压↑&nbsp;&nbsp;&nbsp;
