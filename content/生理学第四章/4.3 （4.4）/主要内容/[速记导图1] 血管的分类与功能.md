# 血管的分类与功能 — 极简速记导图

- 弹性储器血管（大动脉：**主动脉、肺动脉主干**）
  - 管壁厚 -> 富含**弹性纤维** -> 弹性大、可扩张性强
  - 射血期：**扩张储存**血液 + 缓冲收缩压
  - 舒张期：**弹性回缩** -> 推动血液继续流动 + 维持舒张压
  - 核心作用：将**间断射血**转化为**连续血流** + **缓冲血压波动**
- 分配血管（中动脉）
  - 弹性储器血管后 -> 分支至小动脉前的管道
  - 富含**平滑肌** -> 收缩性强
  - 作用：将血液**输送**至各组织器官
- 毛细血管前阻力血管（**小动脉、微动脉**）
  - 口径**小** + 管壁**平滑肌多** -> 血流阻力**大**
  - 收缩 -> 口径↓ -> 阻力↑ -> 血流量↓
  - 舒张 -> 口径↑ -> 阻力↓ -> 血流量↑
  - 作用：形成**外周阻力** -> 维持动脉血压 + 调控**组织血流量**
- 毛细血管前括约肌
  - 真毛细血管起始部平滑肌
  - 舒缩 -> 调控毛细血管**开闭** -> 决定**开放数量**
- 交换血管（**真毛细血管**）
  - 管壁极薄：仅**单层内皮细胞** + 基膜 -> **通透性极高**
  - 血流速度最**慢** -> 为物质交换提供充足时间
  - 作用：血液与组织细胞进行**物质交换**的主要场所
- 毛细血管后阻力血管（微静脉）
  - 有一定阻力 -> 调节毛细血管前/后阻力比值 -> 改变**毛细血管压**
  - 影响**组织液滤过**和血容量
- 容量血管（**静脉系统**）
  - 数量多 + 管径大 + 壁薄 + **可扩张性大**
  - 储备**60%~70%** 全身循环血量
  - 作用：**储血库** + 调节**静脉回心血量**
- 动静脉短路（动-静脉吻合支）
  - 微动脉与微静脉**直接连通** -> 绕过毛细血管
  - 分布：手指/足趾/耳郭等皮肤多
  - 开放 -> 皮肤血流量↑ -> **散热↑**
  - 关闭 -> 皮肤血流量↓ -> **保温**
  - 作用：**体温调节**
- 血管功能分类速查口诀
  - 大动脉 = **弹性储器**（缓冲+连续流）
  - 小/微动脉 = **阻力血管**（外周阻力+调流量）
  - 毛细血管 = **交换血管**（物质交换）
  - 静脉 = **容量血管**（储血60~70%）
  - 动静脉吻合支 = **短路血管**（体温调节）
