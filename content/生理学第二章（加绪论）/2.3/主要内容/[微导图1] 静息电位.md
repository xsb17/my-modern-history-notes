# 静息电位 — 微型速记导图

- 核心概念
  - 极化：静息时**内负外正**状态
  - 去极化：RP**减小**（-70→-50mV）
  - 超极化：RP**增大**（-70→-90mV）
  - 复极化：向 RP 恢复
- 静息电位定义
  - 细胞静息时膜内外的**电位差**（内负外正）
  - 数值因细胞类型而异（神经/骨骼肌约 **-70~-90mV**）
- ★产生机制——三因素
  - ① **K⁺ 平衡电位**（最主要）
    - 静息时膜对 **K⁺ 高通透**（钾漏通道持续开放）
    - K⁺ 顺浓度差**外流** -> 膜内负电位↑
    - 实测 RP ≈ E_K（略小于理论值）
    - 改变膜两侧 K⁺ 浓度 -> RP 随之改变
  - ② **Na⁺ 内流**（次要）
    - 钠漏通道介导少量 Na⁺ 内流
    - 部分抵消 K⁺ 外流形成的负电位
    - -> 实测 RP **略小于** E_K
  - ③ **钠泵生电作用**（贡献 < 5%）
    - 泵出 **3Na⁺**/泵入 **2K⁺** -> 净泵出正电荷
    - -> 增大膜内负值
- ★影响静息电位的因素
  - 细胞外 **K⁺↑** -> 浓度差↓ -> K⁺ 外流↓ -> RP**↓**（去极化）★最重要
  - Na⁺ 通透性↑ -> Na⁺ 内流↑ -> RP**↓**
  - K⁺ 通透性↑ -> K⁺ 外流↑ -> RP**↑**（超极化）
  - 钠泵活动↓（缺血/缺氧/哇巴因）-> RP**↓**
- 膜内外主要离子浓度
  - Na⁺：外 **142** / 内 **10**（外高内低 -> 内向流趋势）
  - K⁺：外 **4** / 内 **140**（内高外低 -> 外向流趋势）
  - Ca²⁺：外 **2.4** / 内 **0.0001**
  - Cl⁻：外 **103** / 内 **4**
