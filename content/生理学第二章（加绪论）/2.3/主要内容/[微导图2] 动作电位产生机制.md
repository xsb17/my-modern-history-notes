# 动作电位产生机制 — 微型速记导图

- 动作电位定义
  - 在 RP 基础上接受**有效刺激** -> 快速可传播的膜电位波动
  - 锋电位 = **动作电位的标志**
- ★产生条件
  - 刺激三参数：**强度 / 时间 / 强度-时间变化率**
  - 阈刺激：引起 AP 的**最小强度**刺激
  - 阈强度（阈值）= 衡量**兴奋性**的指标（兴奋性 = 1/阈值）
  - 阈电位：引发 AP 的**临界膜电位**（比 RP 小 10~20mV）
- ★AP 形态分期
  - 锋电位：上升支（去极化 **-70→+30mV**）+ 下降支（复极化 +30→-70mV）
  - 后电位：负后电位（去极化后）+ 正后电位（超极化后）
- ★去极化相（上升支）机制
  - 前提：膜外 Na⁺ **高**/膜内 Na⁺ **低**（外高内低）
  - 刺激 -> 达**阈电位** -> 电压门控 **Na⁺ 通道**大量开放
  - -> **Na⁺ 大量内流** -> 膜去极化↑
  - -> 更多 Na⁺ 通道开放（**正反馈**）-> 膜内电位变为正值 -> **超射**
- ★复极化相（下降支）机制
  - **Na⁺ 通道失活**关闭 -> Na⁺ 内流**停止**
  - **K⁺ 通透性升高** -> **K⁺ 外流↑**
  - -> 膜内电位向负值恢复 -> 回到静息电位
- 后电位机制
  - 负后电位：胞外 K⁺ 一过性**蓄积** -> 阻碍 K⁺ 外流
  - 正后电位：**钠泵活动增强** -> 恢复 Na⁺/K⁺ 分布（泵出 3Na⁺/泵入 2K⁺）
- AP 离子机制口诀
  - 去极 = **Na⁺ 进**（正反馈）
  - 复极 = **K⁺ 出**
  - 正后电位 = **钠泵干活**
