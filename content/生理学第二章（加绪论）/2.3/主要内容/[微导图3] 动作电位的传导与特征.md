# 动作电位的传导与特征 — 微型速记导图

- 无髓纤维传导——局部电流学说
  - AP 区（膜内正/外负）与相邻静息区（膜内负/外正）-> **局部电流**
  - 局部电流 -> 静息区**去极化**达阈电位 -> 产生新 AP
  - 特点：**双向传导**、不衰减
- ★有髓纤维传导——跳跃式传导
  - 郎飞结特征：
    - 裸露无髓鞘 -> 可产生局部电流
    - **Na⁺ 通道密集** -> 阈电位**低**
    - 可产生 AP
  - 髓鞘区特征：
    - 胶质细胞多层包裹 -> 电阻大/电容小
    - Na⁺ 通道**稀疏** -> 阈电位高
    - **不能产生 AP**
  - 传导方式：AP 沿相邻**郎飞结**跳跃式产生
  - 意义：
    - **传导速度快**（空间常数大/时间常数小）
    - **节省能量**（仅郎飞结消耗 ATP 维持离子梯度）
- ★AP 三大特征（必考）
  - ① **"全或无"现象**
    - "无"：刺激 < 阈值 -> AP 不产生
    - "全"：刺激 ≥ 阈值 -> AP 立即达**最大幅度**
    - AP 幅度**不随**刺激强度变化
  - ② **不衰减传导**
    - AP 幅度**不因**传导距离增大而减小
    - 传导的"全或无"
  - ③ **不可总和**
    - 锋电位不融合 -> 脉冲式发放
    - 原因：**绝对不应期**的存在
- 无髓 vs 有髓对比
  - 无髓：局部电流连续传导/慢/耗能多
  - 有髓：跳跃式传导/快（郎飞结）/耗能少
