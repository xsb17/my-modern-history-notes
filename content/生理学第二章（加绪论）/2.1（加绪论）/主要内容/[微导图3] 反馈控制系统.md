# 反馈控制系统 — 微型速记导图（极高频）&nbsp;&nbsp;&nbsp;

- 控制系统三要素：**控制部分**（发出指令）-> **受控部分**（执行）-> **监视器**（感受反馈）&nbsp;&nbsp;&nbsp;
- ★负反馈（最常见）&nbsp;&nbsp;&nbsp;
  - 定义：反馈信息使受控部分活动朝**相反方向**改变&nbsp;&nbsp;&nbsp;
  - 意义：**维持内环境稳态** -> 纠偏/稳定&nbsp;&nbsp;&nbsp;
  - 必考实例：&nbsp;&nbsp;&nbsp;
    - **体温调节**（体温↑->散热↑/产热↓）&nbsp;&nbsp;&nbsp;
    - **血压调节/减压反射**（血压↑->迷走↑/交感↓->血压↓）&nbsp;&nbsp;&nbsp;
    - **血糖调节**（血糖↑->胰岛素↑->血糖↓）&nbsp;&nbsp;&nbsp;
    - 甲状腺激素分泌调节&nbsp;&nbsp;&nbsp;
- ★正反馈（打破平衡，推进完成）&nbsp;&nbsp;&nbsp;
  - 定义：反馈信息**促进和加强**控制部分活动 -> 朝**相同方向**改变&nbsp;&nbsp;&nbsp;
  - 意义：**打破平衡** -> 使某一活动**趋向完成**&nbsp;&nbsp;&nbsp;
  - ★必考五大例子（选择/填空极高频）：&nbsp;&nbsp;&nbsp;
    - **排尿反射**&nbsp;&nbsp;&nbsp;
    - **分娩过程**&nbsp;&nbsp;&nbsp;
    - **血液凝固**&nbsp;&nbsp;&nbsp;
    - **动作电位 Na⁺ 内流去极化**（0 期正反馈）&nbsp;&nbsp;&nbsp;
    - **排便反射**&nbsp;&nbsp;&nbsp;
- 前馈控制（预见性调节）&nbsp;&nbsp;&nbsp;
  - 定义：反馈信息到达**前** -> 控制部分已提前纠正偏差&nbsp;&nbsp;&nbsp;
  - 特点：**快、有预见性、适应性大** -> 避免负反馈的滞后和波动&nbsp;&nbsp;&nbsp;
  - 例：**赛跑前心率↑呼吸↑**、**望梅止渴**、**一朝被蛇咬十年怕井绳**&nbsp;&nbsp;&nbsp;
- 非自动控制系统（开环）&nbsp;&nbsp;&nbsp;
  - 受控部分不能反馈影响控制部分&nbsp;&nbsp;&nbsp;
  - 例：应激反应中糖皮质激素持续升高&nbsp;&nbsp;&nbsp;
- 反馈系统口诀&nbsp;&nbsp;&nbsp;
  - 负反馈 = 反向纠偏/维持稳态（体温/血压/血糖）&nbsp;&nbsp;&nbsp;
  - 正反馈 = 同向推进/打破平衡（排尿/分娩/凝血/AP去极）&nbsp;&nbsp;&nbsp;
  - 前馈 = 提前预防/有预见性（赛跑前/望梅止渴）&nbsp;&nbsp;&nbsp;
