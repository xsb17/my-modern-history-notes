# 细胞膜的被动转运 — 微型速记导图

- 细胞膜结构基础：**液态镶嵌模型** -> 脂质双分子层 + 膜蛋白 + 糖类
- ★单纯扩散
  - 方向：**高浓度 -> 低浓度**（顺浓度梯度）
  - 耗能：**不耗能**（被动转运）
  - 膜蛋白：**不需要**
  - 代表物质：**O₂、CO₂、NO、NH₃**、乙醇、甘油、尿素、类固醇激素
  - 水：可单纯扩散但速度极慢
  - 原理：**相似相溶**（脂溶性越高越快）
- ★易化扩散（共性）
  - 方向：**高浓度 -> 低浓度**
  - 耗能：**不耗能**
  - 膜蛋白：**必需**（载体或通道蛋白介导）
- ★经通道易化扩散
  - 转运对象：**带电离子**（Na⁺/K⁺/Ca²⁺/Cl⁻）+ H₂O
  - 特性：
    - **离子选择性**（每种通道只通透特定离子）
    - **门控特性**：
      - 电压门控通道（膜电位变化 -> Na⁺通道开放）
      - 化学门控通道（递质结合 -> 通道开放）
      - 机械门控通道（牵张 -> 通道开放）
      - 非门控通道（钾漏通道 -> 持续开放）
  - 速度：**快**（离子通道开放即通）
- ★经载体易化扩散
  - 转运对象：小分子有机物 -> **葡萄糖、氨基酸**
  - 三大特征：
    - **结构特异性**：每种载体只识别特定物质
    - **饱和现象**：载体数量有限 -> 转运速率有上限
    - **竞争性抑制**：结构相似物质竞争同一载体
  - 速度：**较慢**（载体需构象变化）
- 通道 vs 载体对比
  - 通道：离子/水/无饱和/无竞争/快/门控
  - 载体：葡萄糖/氨基酸/**有饱和**/**有竞争**/较慢/构象变化
