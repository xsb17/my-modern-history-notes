# 兴奋-收缩耦联 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★兴奋-收缩耦联（ECC）定义&nbsp;&nbsp;&nbsp;
  - 肌细胞的**兴奋**（AP）与**机械收缩**（肌丝滑行）联系起来的**中介过程**&nbsp;&nbsp;&nbsp;
- 结构基础&nbsp;&nbsp;&nbsp;
  - 骨骼肌：**三联管**（横管 + 两侧终池）&nbsp;&nbsp;&nbsp;
  - 心肌：**二联管**（横管 + 单侧终池）&nbsp;&nbsp;&nbsp;
- ★耦联关键因子：**Ca²⁺**&nbsp;&nbsp;&nbsp;
- ★耦联过程四个步骤（必考）&nbsp;&nbsp;&nbsp;
  - ① AP 传播：&nbsp;&nbsp;&nbsp;
    - AP 沿肌膜和**横管膜**传播到肌细胞深处&nbsp;&nbsp;&nbsp;
  - ② Ca²⁺ 释放：&nbsp;&nbsp;&nbsp;
    - 横管膜 **L 型钙通道**（电压敏感）激活&nbsp;&nbsp;&nbsp;
    - -> 激活终池膜 **钙释放通道**（RYR 受体）&nbsp;&nbsp;&nbsp;
    - -> 肌质网内 **Ca²⁺ 大量释放**入胞质&nbsp;&nbsp;&nbsp;
  - ③ 收缩触发：&nbsp;&nbsp;&nbsp;
    - 胞质 **Ca²⁺↑** -> Ca²⁺ + **肌钙蛋白** -> 原肌球蛋白扭转 -> 横桥位点暴露&nbsp;&nbsp;&nbsp;
    - -> 横桥结合肌动蛋白 -> 横桥扭动 -> **肌丝滑行 -> 收缩**&nbsp;&nbsp;&nbsp;
  - ④ Ca²⁺ 回收（舒张）：&nbsp;&nbsp;&nbsp;
    - 肌质网膜**钙泵**（Ca²⁺-ATP 酶）将胞质 Ca²⁺ **泵回**肌质网&nbsp;&nbsp;&nbsp;
    - -> 胞质 Ca²⁺↓ -> 肌钙蛋白与 Ca²⁺**解离** -> 原肌球蛋白复位&nbsp;&nbsp;&nbsp;
    - -> **肌肉舒张**&nbsp;&nbsp;&nbsp;
- 骨骼肌 vs 心肌 ECC 差异&nbsp;&nbsp;&nbsp;
  - 骨骼肌：三联管、L 型钙通道直接机械耦联激活 RYR（**不依赖胞外 Ca²⁺**）&nbsp;&nbsp;&nbsp;
  - 心肌：二联管、L 型钙通道 Ca²⁺ 内流触发 RYR（**钙诱导钙释放 CICR**，依赖胞外 Ca²⁺）&nbsp;&nbsp;&nbsp;
- ECC 口诀&nbsp;&nbsp;&nbsp;
  - AP 入横管 -> L 通道激活 RYR -> Ca²⁺ 释放 -> 肌钙结合 -> 收缩&nbsp;&nbsp;&nbsp;
  - 钙泵回收 -> Ca²⁺ 下降 -> 舒张&nbsp;&nbsp;&nbsp;
