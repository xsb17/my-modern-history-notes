# 兴奋-收缩耦联 — 微型速记导图

- ★兴奋-收缩耦联（ECC）定义
  - 肌细胞的**兴奋**（AP）与**机械收缩**（肌丝滑行）联系起来的**中介过程**
- 结构基础
  - 骨骼肌：**三联管**（横管 + 两侧终池）
  - 心肌：**二联管**（横管 + 单侧终池）
- ★耦联关键因子：**Ca²⁺**
- ★耦联过程四个步骤（必考）
  - ① AP 传播：
    - AP 沿肌膜和**横管膜**传播到肌细胞深处
  - ② Ca²⁺ 释放：
    - 横管膜 **L 型钙通道**（电压敏感）激活
    - -> 激活终池膜 **钙释放通道**（RYR 受体）
    - -> 肌质网内 **Ca²⁺ 大量释放**入胞质
  - ③ 收缩触发：
    - 胞质 **Ca²⁺↑** -> Ca²⁺ + **肌钙蛋白** -> 原肌球蛋白扭转 -> 横桥位点暴露
    - -> 横桥结合肌动蛋白 -> 横桥扭动 -> **肌丝滑行 -> 收缩**
  - ④ Ca²⁺ 回收（舒张）：
    - 肌质网膜**钙泵**（Ca²⁺-ATP 酶）将胞质 Ca²⁺ **泵回**肌质网
    - -> 胞质 Ca²⁺↓ -> 肌钙蛋白与 Ca²⁺**解离** -> 原肌球蛋白复位
    - -> **肌肉舒张**
- 骨骼肌 vs 心肌 ECC 差异
  - 骨骼肌：三联管、L 型钙通道直接机械耦联激活 RYR（**不依赖胞外 Ca²⁺**）
  - 心肌：二联管、L 型钙通道 Ca²⁺ 内流触发 RYR（**钙诱导钙释放 CICR**，依赖胞外 Ca²⁺）
- ECC 口诀
  - AP 入横管 -> L 通道激活 RYR -> Ca²⁺ 释放 -> 肌钙结合 -> 收缩
  - 钙泵回收 -> Ca²⁺ 下降 -> 舒张
