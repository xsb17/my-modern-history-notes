# 肌肉收缩形式与影响因素 — 微型速记导图&nbsp;&nbsp;&nbsp;

- ★两种收缩形式（必考）&nbsp;&nbsp;&nbsp;
  - 等长收缩：&nbsp;&nbsp;&nbsp;
    - 长度**不变** / 张力**增加**&nbsp;&nbsp;&nbsp;
    - 例：维持**姿势**、提重物未动时&nbsp;&nbsp;&nbsp;
  - 等张收缩：&nbsp;&nbsp;&nbsp;
    - **张力不变** / 肌肉**缩短**&nbsp;&nbsp;&nbsp;
    - 例：举起**重物**&nbsp;&nbsp;&nbsp;
  - 实际：大多数收缩是等长+等张的混合&nbsp;&nbsp;&nbsp;
- ★影响肌肉收缩效能的四大因素&nbsp;&nbsp;&nbsp;
  - 前负荷（初长度）：&nbsp;&nbsp;&nbsp;
    - 肌肉收缩**前**已存在的负荷 -> 决定**初长度**&nbsp;&nbsp;&nbsp;
    - 最适初长度：粗细肌丝**最佳重叠** -> 横桥数量最大 -> **最大张力**&nbsp;&nbsp;&nbsp;
    - Frank-Starling 机制：心肌的前负荷调节&nbsp;&nbsp;&nbsp;
  - 后负荷：&nbsp;&nbsp;&nbsp;
    - 肌肉收缩**中**承受的**负荷/阻力**&nbsp;&nbsp;&nbsp;
    - 后负荷↑ -> 缩短速度**↓** + 缩短程度**↓**&nbsp;&nbsp;&nbsp;
    - 后负荷过大 -> 只产生张力不能缩短（变为等长收缩）&nbsp;&nbsp;&nbsp;
  - 肌肉收缩能力：&nbsp;&nbsp;&nbsp;
    - 与前/后负荷**无关** -> 取决于肌肉自身**功能状态**&nbsp;&nbsp;&nbsp;
    - 影响：胞质 **Ca²⁺ 水平**↑/肌球蛋白 **ATP 酶活性**↑ -> 收缩力↑&nbsp;&nbsp;&nbsp;
    - 正性变力：儿茶酚胺/咖啡因 -> 收缩力↑&nbsp;&nbsp;&nbsp;
    - 负性变力：缺氧/酸中毒/疲劳 -> 收缩力↓&nbsp;&nbsp;&nbsp;
  - 收缩总和：&nbsp;&nbsp;&nbsp;
    - 多纤维总和：参与收缩**运动单位↑** -> 收缩力↑（空间总和）&nbsp;&nbsp;&nbsp;
    - 频率总和（强直收缩）：刺激**频率↑** -> 收缩波**融合** -> 张力↑&nbsp;&nbsp;&nbsp;
      - 单收缩 < 不完全强直收缩 < **完全强直收缩**（最大张力）&nbsp;&nbsp;&nbsp;
- 收缩形式对比&nbsp;&nbsp;&nbsp;
  - 等长 = 长不变/力变 -> 姿势维持&nbsp;&nbsp;&nbsp;
  - 等张 = 力不变/长变 -> 主动运动&nbsp;&nbsp;&nbsp;
- 影响因素口诀&nbsp;&nbsp;&nbsp;
  - 前负荷 = 初长度（最优重叠最大力）&nbsp;&nbsp;&nbsp;
  - 后负荷 = 阻力（↑则减速减幅度）&nbsp;&nbsp;&nbsp;
  - 收缩力 = 自身状态（Ca²⁺/ATP 酶）&nbsp;&nbsp;&nbsp;
  - 总和 = 多纤维+多频率（强直收缩力最大）&nbsp;&nbsp;&nbsp;
