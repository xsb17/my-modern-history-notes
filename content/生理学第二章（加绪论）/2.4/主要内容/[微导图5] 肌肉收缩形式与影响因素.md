# 肌肉收缩形式与影响因素 — 微型速记导图

- ★两种收缩形式（必考）
  - 等长收缩：
    - 长度**不变** / 张力**增加**
    - 例：维持**姿势**、提重物未动时
  - 等张收缩：
    - **张力不变** / 肌肉**缩短**
    - 例：举起**重物**
  - 实际：大多数收缩是等长+等张的混合
- ★影响肌肉收缩效能的四大因素
  - 前负荷（初长度）：
    - 肌肉收缩**前**已存在的负荷 -> 决定**初长度**
    - 最适初长度：粗细肌丝**最佳重叠** -> 横桥数量最大 -> **最大张力**
    - Frank-Starling 机制：心肌的前负荷调节
  - 后负荷：
    - 肌肉收缩**中**承受的**负荷/阻力**
    - 后负荷↑ -> 缩短速度**↓** + 缩短程度**↓**
    - 后负荷过大 -> 只产生张力不能缩短（变为等长收缩）
  - 肌肉收缩能力：
    - 与前/后负荷**无关** -> 取决于肌肉自身**功能状态**
    - 影响：胞质 **Ca²⁺ 水平**↑/肌球蛋白 **ATP 酶活性**↑ -> 收缩力↑
    - 正性变力：儿茶酚胺/咖啡因 -> 收缩力↑
    - 负性变力：缺氧/酸中毒/疲劳 -> 收缩力↓
  - 收缩总和：
    - 多纤维总和：参与收缩**运动单位↑** -> 收缩力↑（空间总和）
    - 频率总和（强直收缩）：刺激**频率↑** -> 收缩波**融合** -> 张力↑
      - 单收缩 < 不完全强直收缩 < **完全强直收缩**（最大张力）
- 收缩形式对比
  - 等长 = 长不变/力变 -> 姿势维持
  - 等张 = 力不变/长变 -> 主动运动
- 影响因素口诀
  - 前负荷 = 初长度（最优重叠最大力）
  - 后负荷 = 阻力（↑则减速减幅度）
  - 收缩力 = 自身状态（Ca²⁺/ATP 酶）
  - 总和 = 多纤维+多频率（强直收缩力最大）
