# 影响神经-肌接头传递的因素 — 微型速记导图

- ★影响 ACh 释放的因素
  - 胞外**低 Ca²⁺**/**高 Mg²⁺** -> Ca²⁺ 内流↓ -> ACh 释放**↓**
  - **肉毒杆菌毒素** -> 阻止 ACh 囊泡释放 -> **肌肉麻痹**（美容除皱）
- ★影响 ACh 受体的因素
  - **筒箭毒碱** -> 竞争性阻断 **N₂ 受体** -> 肌肉**松弛**（手术肌松药）
  - **α-银环蛇毒** -> 不可逆阻断 N₂ 受体 -> **呼吸肌麻痹**
  - ★**重症肌无力**：
    - 自身免疫病 -> 自身抗体**破坏 ACh 受体** -> 受体数量↓
    - 症状：**肌无力/易疲劳**（眼外肌/面部肌首发 -> 全身）
    - 治疗：**新斯的明**（胆碱酯酶抑制剂 -> 增加 ACh 浓度）
- ★影响 ACh 清除的因素
  - **有机磷农药** -> 使胆碱酯酶**失活**（磷酰化）
    - -> ACh 大量**蓄积** -> 持续刺激 N₂ 受体 -> **肌束震颤 -> 呼吸肌麻痹**
  - **新斯的明** -> 可逆性抑制胆碱酯酶 -> ACh↑
    - 临床应用：重症肌无力、术后腹胀/尿潴留
- 三步调控汇总
  - 释放环节：低 Ca²⁺/高 Mg²⁺/**肉毒素** -> 释放↓
  - 受体环节：**筒箭毒/α-蛇毒/重症肌无力** -> 受体受阻/破坏
  - 清除环节：**有机磷**（失活酶->蓄积）/**新斯的明**（抑酶->增强）
