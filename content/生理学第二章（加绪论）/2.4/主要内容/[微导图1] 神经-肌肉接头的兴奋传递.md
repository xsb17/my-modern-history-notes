# 神经-肌肉接头的兴奋传递 — 微型速记导图（高频）

- 神经-肌肉接头结构
  - 接头前膜：**ACh 突触小泡** + 线粒体 + 钙通道
  - 接头间隙
  - 接头后膜（终板膜）：**N 型 ACh 受体** + **胆碱酯酶** + 皱襞
- ★兴奋传递四个阶段（必考大题）
  - 第一阶段：**ACh 释放**
    - 运动神经 AP -> 前膜去极化 -> 电压门控 **Ca²⁺ 通道开放**
    - -> **Ca²⁺ 内流** -> 突触小泡胞吐 -> ACh **量子式释放**
  - 第二阶段：**终板电位（EPP）形成**
    - ACh 扩散 -> 结合终板膜 **N 型 ACh 受体**
    - -> 阳离子通道开放（**Na⁺ 内流为主**）
    - -> **终板电位**（本质 = **局部电位**，大量微终板电位总和）
  - 第三阶段：**肌膜 AP 形成**
    - EPP 电紧张扩布 -> 邻近普通肌膜 Na⁺ 通道开放
    - -> 去极化达**阈电位** -> 触发**动作电位**
  - 第四阶段：**ACh 降解**
    - **胆碱酯酶**水解 ACh -> 胆碱 + 乙酸 -> 终止传递
- ★传递特征（三特征）
  - **单向传递**（电->化学->电）
  - **时间延搁**（约 0.5~1.0ms）
  - **易疲劳、易受药物影响**
- EPP vs AP 对比
  - EPP = 局部电位（等级性/衰减/可总和/无不应期）
  - AP = 动作电位（全或无/不衰减/不可总和/有不应期）
