# 神经-肌肉接头的兴奋传递 — 微型速记导图（高频）&nbsp;&nbsp;&nbsp;

- 神经-肌肉接头结构&nbsp;&nbsp;&nbsp;
  - 接头前膜：**ACh 突触小泡** + 线粒体 + 钙通道&nbsp;&nbsp;&nbsp;
  - 接头间隙&nbsp;&nbsp;&nbsp;
  - 接头后膜（终板膜）：**N 型 ACh 受体** + **胆碱酯酶** + 皱襞&nbsp;&nbsp;&nbsp;
- ★兴奋传递四个阶段（必考大题）&nbsp;&nbsp;&nbsp;
  - 第一阶段：**ACh 释放**&nbsp;&nbsp;&nbsp;
    - 运动神经 AP -> 前膜去极化 -> 电压门控 **Ca²⁺ 通道开放**&nbsp;&nbsp;&nbsp;
    - -> **Ca²⁺ 内流** -> 突触小泡胞吐 -> ACh **量子式释放**&nbsp;&nbsp;&nbsp;
  - 第二阶段：**终板电位（EPP）形成**&nbsp;&nbsp;&nbsp;
    - ACh 扩散 -> 结合终板膜 **N 型 ACh 受体**&nbsp;&nbsp;&nbsp;
    - -> 阳离子通道开放（**Na⁺ 内流为主**）&nbsp;&nbsp;&nbsp;
    - -> **终板电位**（本质 = **局部电位**，大量微终板电位总和）&nbsp;&nbsp;&nbsp;
  - 第三阶段：**肌膜 AP 形成**&nbsp;&nbsp;&nbsp;
    - EPP 电紧张扩布 -> 邻近普通肌膜 Na⁺ 通道开放&nbsp;&nbsp;&nbsp;
    - -> 去极化达**阈电位** -> 触发**动作电位**&nbsp;&nbsp;&nbsp;
  - 第四阶段：**ACh 降解**&nbsp;&nbsp;&nbsp;
    - **胆碱酯酶**水解 ACh -> 胆碱 + 乙酸 -> 终止传递&nbsp;&nbsp;&nbsp;
- ★传递特征（三特征）&nbsp;&nbsp;&nbsp;
  - **单向传递**（电->化学->电）&nbsp;&nbsp;&nbsp;
  - **时间延搁**（约 0.5~1.0ms）&nbsp;&nbsp;&nbsp;
  - **易疲劳、易受药物影响**&nbsp;&nbsp;&nbsp;
- EPP vs AP 对比&nbsp;&nbsp;&nbsp;
  - EPP = 局部电位（等级性/衰减/可总和/无不应期）&nbsp;&nbsp;&nbsp;
  - AP = 动作电位（全或无/不衰减/不可总和/有不应期）&nbsp;&nbsp;&nbsp;
