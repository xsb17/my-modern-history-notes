# 横纹肌结构与肌丝滑行理论 — 微型速记导图

- 肌原纤维和肌节
  - 肌节 = 相邻两 **Z 线**之间的区域 -> 肌肉收缩的**基本单位**
  - 粗肌丝：**肌球蛋白**（含横桥，有 **ATP 酶活性**）
  - 细肌丝：**肌动蛋白**（横桥结合位点）+ **原肌球蛋白**（覆盖位点）+ **肌钙蛋白**（与 Ca²⁺ 结合）
- ★肌管系统（必考）
  - 横管（T 管）：与肌原纤维**垂直** -> 将 AP 传入细胞深处
  - 纵管（肌质网/SR）：末端膨大为**终池** -> 含高浓度 **Ca²⁺**
  - ★三联管（骨骼肌特有）：**横管 + 两侧终池** -> 兴奋-收缩耦联**关键部位**
  - 二联管（心肌）：横管 + 单侧终池
- ★肌丝滑行理论（核心学说）
  - 核心论点：肌肉收缩时**肌丝本身不缩短** -> **细肌丝**向粗肌丝**滑行** -> Z 线靠近 -> 肌节变短
  - 滑行过程链：
    - 胞质 **Ca²⁺↑** -> Ca²⁺ + **肌钙蛋白** -> 原肌球蛋白**扭转** -> 肌动蛋白位点**暴露**
    - -> 横桥 + 肌动蛋白 -> 横桥**扭动**（拉动细肌丝向 M 线）-> 横桥**复位**
    - -> **横桥周期**：结合 -> 扭动(ATP→ADP+Pi) -> 复位
  - 舒张过程：Ca²⁺ 被**钙泵**回收 -> Ca²⁺ 与肌钙蛋白**解离** -> 原肌球蛋白**重新覆盖**位点
- 力学表现
  - 张力 = 活化**横桥数目**的总和
  - 缩短速度 = 横桥周期越**短**越快（ATP 酶活性高 -> 收缩快）
- 口诀
  - 收缩 = Ca²⁺ -> 肌钙蛋白 -> 原肌球扭转 -> 横桥结合扭动
  - 舒张 = 钙泵回收 Ca²⁺ -> 原肌球复位覆盖
