# 第九章第一节：感觉器官（视觉器官为主）— 全集存档&nbsp;&nbsp;&nbsp;

## 第一节 概述&nbsp;&nbsp;&nbsp;

### 一、感受器和感觉器官&nbsp;&nbsp;&nbsp;

- 感受器：生物体内专门感受机体内、外环境变化的**结构或装置**&nbsp;&nbsp;&nbsp;
  - 例：游离神经末梢（痛觉）、环层小体、视杆/视锥细胞、毛细胞&nbsp;&nbsp;&nbsp;
- 感觉器官：感受器 + **附属结构** + 神经组织 → 眼、耳、鼻、舌&nbsp;&nbsp;&nbsp;

### 二、感受器的生理特性&nbsp;&nbsp;&nbsp;

#### 1. 适宜刺激&nbsp;&nbsp;&nbsp;

- 一种感受器通常只对**一种特定形式**的刺激最敏感，该刺激称为适宜刺激&nbsp;&nbsp;&nbsp;
- 例：视锥/视杆细胞 → **380~760nm 电磁波**；耳蜗毛细胞 → 声波&nbsp;&nbsp;&nbsp;

#### 2. 换能作用&nbsp;&nbsp;&nbsp;

- 将各种形式的**刺激能量**转换为传入神经的**动作电位**&nbsp;&nbsp;&nbsp;
- 感受器电位：在感受器上产生的过渡性**局部电位**&nbsp;&nbsp;&nbsp;
- 发生器电位：在初级传入神经末梢产生的过渡性电位&nbsp;&nbsp;&nbsp;
- 特点：无"全或无"特征、可总和、呈**分级式**&nbsp;&nbsp;&nbsp;

#### 3. 编码作用&nbsp;&nbsp;&nbsp;

- 把刺激信号中包含的各种信息（**强度、性质、持续时间**等）编排成神经冲动的不同序列&nbsp;&nbsp;&nbsp;
- 强度编码：刺激强 → AP 频率↑ + 兴奋的感受器数目↑&nbsp;&nbsp;&nbsp;

#### 4. 适应现象&nbsp;&nbsp;&nbsp;

- 恒定强度的刺激持续作用 → 感觉神经纤维上 AP 频率**逐渐降低**甚至消失&nbsp;&nbsp;&nbsp;
- 快适应感受器：皮肤触觉（环层小体）→ 意义：利于接受**新刺激**&nbsp;&nbsp;&nbsp;
- 慢适应感受器：痛觉、窦弓压力感受器 → 意义：利于**持续监测**&nbsp;&nbsp;&nbsp;

## 第二节 视觉器官——眼&nbsp;&nbsp;&nbsp;

### 一、概述&nbsp;&nbsp;&nbsp;

- 人脑从外界获得的信息 **70%** 来自于视觉&nbsp;&nbsp;&nbsp;
- 视觉感受器：**视锥细胞、视杆细胞**&nbsp;&nbsp;&nbsp;
- 适宜刺激：波长 **380~760nm** 电磁波&nbsp;&nbsp;&nbsp;
- 视觉形成：光线 → 折光系统成像于视网膜 → 感光细胞换能/编码 → 视神经 AP → 视觉中枢（**枕叶**）→ 视觉&nbsp;&nbsp;&nbsp;

### 二、眼的折光系统及其调节&nbsp;&nbsp;&nbsp;

#### （一）眼的折光与成像&nbsp;&nbsp;&nbsp;

- 折光系统：**角膜、房水、晶状体、玻璃体**&nbsp;&nbsp;&nbsp;
- **简化眼**：与正常眼在光学参数和屈光效果上相同的简单模型&nbsp;&nbsp;&nbsp;
  - 前后径 **20mm**，折射率 **1.333**，曲率半径 **5mm**&nbsp;&nbsp;&nbsp;
  - 节点(n)：在角膜后方 **5mm** 处，后主焦点相当于视网膜位置&nbsp;&nbsp;&nbsp;

#### （二）眼的调节——近反射（视近物时三大变化）&nbsp;&nbsp;&nbsp;

##### 1. 晶状体调节（最主要）&nbsp;&nbsp;&nbsp;

- 视远物（**6m以外**）：不需调节 → 睫状肌**舒张** → 悬韧带**紧张** → 晶状体**扁平**&nbsp;&nbsp;&nbsp;
- 视近物：成像在视网膜后（模糊）→ 动眼神经（+）→ **睫状肌收缩** → **悬韧带松弛** → 晶状体**变凸**（前面为主）→ 折光力↑ → 焦距缩短 → 物像前移回到视网膜上&nbsp;&nbsp;&nbsp;
- **近点**：眼尽最大能力调节时所能看清物体的**最近距离**（反映晶状体弹性）&nbsp;&nbsp;&nbsp;
  - 年龄↑ → 晶状体弹性↓ → 近点**远移**：10岁8.8cm→20岁10.4cm→40岁18.5cm→60岁83.3cm&nbsp;&nbsp;&nbsp;
- **老视**：年龄增长晶状体弹性**降低** → 视近物不清 → 矫正：**凸透镜**&nbsp;&nbsp;&nbsp;

##### 2. 瞳孔调节&nbsp;&nbsp;&nbsp;

- **瞳孔对光反射**：瞳孔随光照强弱而改变大小 → 强光缩小/弱光散大 → 调节入眼光量&nbsp;&nbsp;&nbsp;
  - 互感反应：**双侧性**（一侧光照，双侧瞳孔均缩小）&nbsp;&nbsp;&nbsp;
  - 临床意义：判断麻醉深度、病危程度&nbsp;&nbsp;&nbsp;
- **瞳孔近反射**：看近物时反射性**瞳孔缩小** → 减少球面像差和色相差&nbsp;&nbsp;&nbsp;

##### 3. 视轴会聚（辐辏反射）&nbsp;&nbsp;&nbsp;

- 双眼看近物时，两眼视轴同时向**鼻侧聚拢**&nbsp;&nbsp;&nbsp;
- 意义：物像落在两眼视网膜的**对称点**上，避免**复视**&nbsp;&nbsp;&nbsp;

#### （三）屈光异常及矫正&nbsp;&nbsp;&nbsp;

- **近视**：眼球前后径**过长**或折光力过强 → 远物成像于视网膜**前** → 矫正：**凹透镜**&nbsp;&nbsp;&nbsp;
- **远视**：眼球前后径**过短**或折光力过弱 → 近物成像于视网膜**后** → 矫正：**凸透镜**&nbsp;&nbsp;&nbsp;
- **散光**：角膜/晶状体表面在不同方向上曲率半径不同 → 矫正：**圆柱镜**&nbsp;&nbsp;&nbsp;
- **老视**：晶状体弹性**↓** → 近点远移 → 矫正：**凸透镜**&nbsp;&nbsp;&nbsp;

### 三、视网膜的两种感光换能系统&nbsp;&nbsp;&nbsp;

#### 视杆细胞系统（暗视觉系统）&nbsp;&nbsp;&nbsp;

- 分布：视网膜**周边部**（中央凹无）&nbsp;&nbsp;&nbsp;
- 功能：**暗视觉**（无**色觉**）→ 光敏度**高**&nbsp;&nbsp;&nbsp;
- 分辨率：**低**（多个细胞会聚到一个双极细胞）&nbsp;&nbsp;&nbsp;
- 感光色素：**视紫红质**（视蛋白 + 视黄醛）&nbsp;&nbsp;&nbsp;
- VitA 缺乏 → 视紫红质合成障碍 → **夜盲症**&nbsp;&nbsp;&nbsp;
- 暗适应机制：**视紫红质合成↑**&nbsp;&nbsp;&nbsp;

#### 视锥细胞系统（明视觉系统）&nbsp;&nbsp;&nbsp;

- 分布：**中央凹**高度密集（周边少）&nbsp;&nbsp;&nbsp;
- 功能：**明视觉**（有**色觉**）→ 光敏度**低**&nbsp;&nbsp;&nbsp;
- 分辨率：**高**（单线式联系）&nbsp;&nbsp;&nbsp;
- 感光色素：三种（红/绿/蓝敏感色素）→ **三原色学说**&nbsp;&nbsp;&nbsp;
- 色盲/色弱：某种视锥细胞色素基因缺陷&nbsp;&nbsp;&nbsp;

#### 两大系统对比&nbsp;&nbsp;&nbsp;

- 视杆：暗视觉/无色觉/高光敏度/低分辨率/周边/视紫红质/VitA→夜盲&nbsp;&nbsp;&nbsp;
- 视锥：明视觉/色觉/低光敏度/高分辨率/中央凹/三原色/基因→色盲&nbsp;&nbsp;&nbsp;

---&nbsp;&nbsp;&nbsp;

### 四、视觉相关的生理现象&nbsp;&nbsp;&nbsp;

#### 1. 暗适应&nbsp;&nbsp;&nbsp;

- 从**明处进入暗处**，视力逐渐恢复的过程&nbsp;&nbsp;&nbsp;
- 机制：视锥细胞快速适应（约 5~8min）+ 视杆细胞**视紫红质合成↑**（约 20~30min）&nbsp;&nbsp;&nbsp;
- 特点：初期快（视锥适应）→ 后期慢但幅度大（视杆适应）&nbsp;&nbsp;&nbsp;
- 临床：VitA 缺乏 → 视紫红质合成不足 → 暗适应**延长**&nbsp;&nbsp;&nbsp;

#### 2. 明适应&nbsp;&nbsp;&nbsp;

- 从**暗处进入明处**，视力迅速恢复的过程&nbsp;&nbsp;&nbsp;
- 机制：视紫红质在强光下**迅速分解** → 视杆细胞暂时失活 → 视锥细胞接替工作&nbsp;&nbsp;&nbsp;
- 特点：耗时**短**（约1分钟）&nbsp;&nbsp;&nbsp;

#### 3. 视野&nbsp;&nbsp;&nbsp;

- 单眼固定注视前方一点时能看到的**空间范围**&nbsp;&nbsp;&nbsp;
- 大小排名：**白色 > 黄色/蓝色 > 红色 > 绿色**&nbsp;&nbsp;&nbsp;
- 临床：检查视野可辅助诊断视网膜/视神经/视通路病变&nbsp;&nbsp;&nbsp;

#### 4. 视敏度（视力）&nbsp;&nbsp;&nbsp;

- 眼对物体**细微结构**的分辨能力，即能分辨的**最小视角**的倒数&nbsp;&nbsp;&nbsp;
- 正常视力 1.0 → 能分辨 **1分角**（1/60度）的视角&nbsp;&nbsp;&nbsp;
- 检查工具：视力表&nbsp;&nbsp;&nbsp;
