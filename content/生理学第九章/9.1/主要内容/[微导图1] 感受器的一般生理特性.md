# 感受器的一般生理特性 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 感受器 vs 感觉器官&nbsp;&nbsp;&nbsp;
  - 感受器：专门感受**内外环境变化**的结构（如游离神经末梢/环层小体/视细胞/毛细胞）&nbsp;&nbsp;&nbsp;
  - 感觉器官 = 感受器 + **附属结构** + 神经组织（眼/耳/鼻/舌）&nbsp;&nbsp;&nbsp;
- ★适宜刺激&nbsp;&nbsp;&nbsp;
  - 一种感受器只对**一种特定形式**的刺激最敏感&nbsp;&nbsp;&nbsp;
  - 视觉：**380~760nm** 电磁波&nbsp;&nbsp;&nbsp;
  - 听觉：声波（20~20000Hz）&nbsp;&nbsp;&nbsp;
- ★换能作用（必考）&nbsp;&nbsp;&nbsp;
  - 将**刺激能量** -> 转换为传入神经的 **AP**&nbsp;&nbsp;&nbsp;
  - **感受器电位**：感受器上产生的过渡性**局部电位** -> 无全或无/可总和/分级式&nbsp;&nbsp;&nbsp;
  - **发生器电位**：初级传入神经末梢产生的过渡性电位&nbsp;&nbsp;&nbsp;
  - 本质：**非AP** -> 属于**局部电位**&nbsp;&nbsp;&nbsp;
- 编码作用&nbsp;&nbsp;&nbsp;
  - 将刺激的**强度/性质/时间**等信息编排成 AP 不同序列&nbsp;&nbsp;&nbsp;
  - 强度编码：刺激强 -> **AP频率↑** + 兴奋感受器**数目↑**&nbsp;&nbsp;&nbsp;
- ★适应现象（必考）&nbsp;&nbsp;&nbsp;
  - 恒定强度刺激持续 -> AP频率**逐渐降低**甚至消失&nbsp;&nbsp;&nbsp;
  - 快适应感受器：&nbsp;&nbsp;&nbsp;
    - 例：**皮肤触觉**（环层小体）-> 刺激后很快不再发放冲动&nbsp;&nbsp;&nbsp;
    - 意义：利于接受**新刺激** -> 穿衣服后不再感觉&nbsp;&nbsp;&nbsp;
  - 慢适应感受器：&nbsp;&nbsp;&nbsp;
    - 例：**痛觉、窦弓压力感受器**、肌梭 -> 持续发放冲动&nbsp;&nbsp;&nbsp;
    - 意义：利于**持续监测** -> 维持姿势/血压监测&nbsp;&nbsp;&nbsp;
- 感受器四特性口诀&nbsp;&nbsp;&nbsp;
  - 适宜刺激 = 各有所爱&nbsp;&nbsp;&nbsp;
  - 换能作用 = 能量转换(局部电位)&nbsp;&nbsp;&nbsp;
  - 编码作用 = 编排信息序列&nbsp;&nbsp;&nbsp;
  - 适应现象 = 快适应(新刺激)/慢适应(持续监测)&nbsp;&nbsp;&nbsp;
