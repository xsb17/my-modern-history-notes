# 视网膜的两种感光换能系统 — 微型速记导图（必背对比）&nbsp;&nbsp;&nbsp;

- ★视杆细胞系统（暗视觉系统）&nbsp;&nbsp;&nbsp;
  - 分布：视网膜**周边部** -> 中央凹**无**视杆&nbsp;&nbsp;&nbsp;
  - 功能：**暗视觉** -> **无色觉**（黑白/灰阶）&nbsp;&nbsp;&nbsp;
  - 光敏度：**高** -> 微弱光即可兴奋&nbsp;&nbsp;&nbsp;
  - 分辨率：**低** -> 多个视杆会聚到一个双极细胞（高会聚）&nbsp;&nbsp;&nbsp;
  - 感光色素：**视紫红质** = 视蛋白 + **视黄醛**（VitA衍生物）&nbsp;&nbsp;&nbsp;
  - 临床联系：**VitA缺乏** -> 视紫红质合成障碍 -> **夜盲症**&nbsp;&nbsp;&nbsp;
  - 暗适应机制：**视紫红质合成↑**&nbsp;&nbsp;&nbsp;
- ★视锥细胞系统（明视觉系统）&nbsp;&nbsp;&nbsp;
  - 分布：**中央凹**高度密集 -> 周边少&nbsp;&nbsp;&nbsp;
  - 功能：**明视觉** -> 有色觉（**三原色学说**）&nbsp;&nbsp;&nbsp;
  - 光敏度：**低** -> 需较强光&nbsp;&nbsp;&nbsp;
  - 分辨率：**高** -> 单线式联系（一个视锥->一个双极细胞）&nbsp;&nbsp;&nbsp;
  - 感光色素：三种 -> 红敏/绿敏/蓝敏色素（均为视蛋白+视黄醛）&nbsp;&nbsp;&nbsp;
  - 临床联系：某种色素基因缺陷 -> **色盲/色弱**&nbsp;&nbsp;&nbsp;
- ★两大系统全面对比表（必考）&nbsp;&nbsp;&nbsp;
  - 分布：视杆=**周边** | 视锥=**中央凹**&nbsp;&nbsp;&nbsp;
  - 功能：视杆=**暗视觉** | 视锥=**明视觉**&nbsp;&nbsp;&nbsp;
  - 色觉：视杆=**无** | 视锥=**有**(三原色)&nbsp;&nbsp;&nbsp;
  - 光敏度：视杆=**高** | 视锥=**低**&nbsp;&nbsp;&nbsp;
  - 分辨率：视杆=**低**(高会聚) | 视锥=**高**(单线式)&nbsp;&nbsp;&nbsp;
  - 感光色素：视杆=**视紫红质** | 视锥=**三种**视蛋白&nbsp;&nbsp;&nbsp;
  - 缺乏症：视杆->VitA↓->**夜盲** | 视锥->基因缺陷->**色盲**&nbsp;&nbsp;&nbsp;
