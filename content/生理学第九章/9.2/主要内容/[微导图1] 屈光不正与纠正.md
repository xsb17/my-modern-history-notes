# 屈光不正与纠正 — 微型速记导图&nbsp;&nbsp;&nbsp;

- 屈光不正定义：折光力异常或眼球形态异常 -> 平行光不能聚焦在视网膜&nbsp;&nbsp;&nbsp;
- ★近视（myopia）&nbsp;&nbsp;&nbsp;
  - 原因：眼球前后径**过长** / 折光力**过强**&nbsp;&nbsp;&nbsp;
  - 视近物：辐散光可聚焦视网膜上 -> 看清&nbsp;&nbsp;&nbsp;
  - 视远物：平行光聚焦在视网膜**前方** -> 看不清&nbsp;&nbsp;&nbsp;
  - 矫正：配带**凹透镜**&nbsp;&nbsp;&nbsp;
- ★远视（hyperopia）&nbsp;&nbsp;&nbsp;
  - 原因：眼球前后径**过短** / 折光力**过弱**&nbsp;&nbsp;&nbsp;
  - 视远物：平行光聚焦在视网膜**后方**（需调节）&nbsp;&nbsp;&nbsp;
  - 视近物：物像更靠后 -> 更难看清&nbsp;&nbsp;&nbsp;
  - 矫正：配带**凸透镜**&nbsp;&nbsp;&nbsp;
- ★散光（astigmatism）&nbsp;&nbsp;&nbsp;
  - 原因：不同方位折光面曲率半径**不相等** -> 折光力**不一致**&nbsp;&nbsp;&nbsp;
  - 表现：各方向上的焦点不重合 -> 视物变形&nbsp;&nbsp;&nbsp;
  - 矫正：配带**圆柱镜**&nbsp;&nbsp;&nbsp;
- 三种屈光不正对比口诀&nbsp;&nbsp;&nbsp;
  - 近视 = 眼轴长/前聚焦 -> **凹透镜**发散&nbsp;&nbsp;&nbsp;
  - 远视 = 眼轴短/后聚焦 -> **凸透镜**会聚&nbsp;&nbsp;&nbsp;
  - 散光 = 曲率不均 -> **圆柱镜**定向补偿&nbsp;&nbsp;&nbsp;
- 临床联系&nbsp;&nbsp;&nbsp;
  - 青少年近视 -> 多为**轴性近视**（眼球前后径过长）&nbsp;&nbsp;&nbsp;
  - 老年人 -> 晶状体弹性↓ -> **老视**（不同于远视，也需凸透镜）&nbsp;&nbsp;&nbsp;
