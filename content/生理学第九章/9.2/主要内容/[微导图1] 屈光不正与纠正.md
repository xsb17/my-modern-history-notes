# 屈光不正与纠正 — 微型速记导图

- 屈光不正定义：折光力异常或眼球形态异常 -> 平行光不能聚焦在视网膜
- ★近视（myopia）
  - 原因：眼球前后径**过长** / 折光力**过强**
  - 视近物：辐散光可聚焦视网膜上 -> 看清
  - 视远物：平行光聚焦在视网膜**前方** -> 看不清
  - 矫正：配带**凹透镜**
- ★远视（hyperopia）
  - 原因：眼球前后径**过短** / 折光力**过弱**
  - 视远物：平行光聚焦在视网膜**后方**（需调节）
  - 视近物：物像更靠后 -> 更难看清
  - 矫正：配带**凸透镜**
- ★散光（astigmatism）
  - 原因：不同方位折光面曲率半径**不相等** -> 折光力**不一致**
  - 表现：各方向上的焦点不重合 -> 视物变形
  - 矫正：配带**圆柱镜**
- 三种屈光不正对比口诀
  - 近视 = 眼轴长/前聚焦 -> **凹透镜**发散
  - 远视 = 眼轴短/后聚焦 -> **凸透镜**会聚
  - 散光 = 曲率不均 -> **圆柱镜**定向补偿
- 临床联系
  - 青少年近视 -> 多为**轴性近视**（眼球前后径过长）
  - 老年人 -> 晶状体弹性↓ -> **老视**（不同于远视，也需凸透镜）
