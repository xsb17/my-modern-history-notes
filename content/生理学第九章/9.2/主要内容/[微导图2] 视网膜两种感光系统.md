# 视网膜两种感光系统 — 微型速记导图

- 视网膜分层：色素细胞 -> **感光细胞（视锥/视杆）**-> 双极细胞 -> 神经节细胞
- 生理盲点：**视神经乳头**处 -> **无感光细胞** -> 双眼视物对侧补偿
- 黄斑中央凹：距视乳头颞侧约 **3mm** -> 感光**最敏锐**处
- ★视锥系统（明视觉）——7 维度特征
  - 光敏感性：**低** -> 感受**强光**
  - 分辨率：**高** -> 视物精细
  - 色觉：**有** -> **三原色**（红/绿/蓝视锥）
  - 外形：短**圆锥**状
  - 分布：密集于**中央凹** -> 周边少
  - 突触联系：**单线式** -> 一个视锥-一个双极 -> 高分辨率
  - 代表动物：**鸡**（几乎只有视锥）
- ★视杆系统（暗视觉）——7 维度特征
  - 光敏感性：**高** -> 感受**弱光**（1个光子即可兴奋）
  - 分辨率：**低** -> 视物粗糙
  - 色觉：**无** -> 黑白灰阶
  - 外形：**长杆**状
  - 分布：视网膜**周边**多 -> 中央凹**无**
  - 突触联系：**聚合式** -> 多个视杆-一个双极 -> 弱刺激总和
  - 代表动物：**猫头鹰**（只有视杆）
- ★视杆细胞感光换能机制
  - 感光物质：**视紫红质** = 视蛋白 + **11-顺型视黄醛**
  - 暗处：cGMP↑ -> cGMP门控Na⁺通道**开放** -> Na⁺持续内流（**暗电流**）-> 静息电位 **-30~-40mV**
  - 光照：视紫红质光化学反应 -> G蛋白 -> 磷酸二酯酶 -> cGMP↓ -> Na⁺通道**关闭** -> K⁺外流 -> **超极化**
  - VitA缺乏 -> 视紫红质合成障碍 -> **夜盲症**
- 三原色学说
  - 红视锥（**564nm**）/ 绿视锥（**534nm**）/ 蓝视锥（**420nm**）
  - 不同波长 -> 三种视锥不同比例兴奋 -> 产生不同色觉
  - 红:绿:蓝 = **4:1:0** -> 红色 | **2:8:1** -> 绿色 | **4:1:18** -> 蓝色
- 色盲/色弱
  - 色盲：缺乏编码感光色素**基因**（遗传）-> 红/绿/蓝色盲
  - 色弱：视锥反应能力**弱**（后天）
