# 外耳与中耳的传音功能 — 微型速记导图

- 听觉概述
  - 感受器：**螺旋器**（Corti器）
  - 适宜刺激：**20~20000Hz** 声波
  - 听阈：刚能引起听觉的最小声强
  - 最大可听阈：能忍受的最强声压
  - 听域：感受的整个频率和强度范围
  - 最敏感频率：**1000~3000Hz**
  - 日常语言频率：**300~3000Hz**
- 外耳功能
  - 耳廓：**收集声波**（定向）
  - 外耳道：**传导声波**（共振增强 2000~3000Hz）
- ★中耳的增压效应（必考简答）
  - 听骨链：鼓膜 -> **锤骨** -> **砧骨** -> **镫骨** -> 前庭窗
  - 鼓膜面积与镫骨底面积比 ≈ **17.2 : 1**
  - 听骨链杠杆长臂:短臂 ≈ **1.3 : 1**
  - 总增压：17.2 × 1.3 = **22.4 倍**
  - 振幅减小约 **1/4**
  - 意义：补偿气体声阻抗 < 液体声阻抗的**能量衰减** -> 提高传递效率
- 咽鼓管功能
  - 连通**鼻咽部**和**鼓室** -> 吞咽/张口时开放
  - 调节鼓室内压力 = 大气压 -> 维持鼓膜正常位置/形状/振动
  - 儿童：咽鼓管**短/粗/平直** -> 鼻炎咽炎易致**中耳炎**
- ★声波传入内耳两条途径（必考选择）
  - 气传导（**主要途径**）：
    - 声波 -> 外耳道 -> 鼓膜 -> 听骨链 -> **前庭窗膜** -> 耳蜗
  - 骨传导：
    - 声波 -> 颅骨振动 -> 耳蜗内淋巴振动
    - 较强声波、自己说话时明显
  - 临床：传音性耳聋（外耳/中耳病变）-> 气传导↓/骨传导正常；感音性耳聋（内耳病变）-> 两者均↓
- 中耳功能口诀
  - 鼓膜听骨链 = 增压 **22.4** 倍
  - 咽鼓管 = 平衡气压保鼓膜
  - 气导为主/骨导为辅
