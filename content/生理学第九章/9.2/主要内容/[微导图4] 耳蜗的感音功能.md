# 耳蜗的感音功能 — 微型速记导图

- 耳蜗结构
  - 骨性管道约 **35mm** -> 盘旋 **2.5~2.75 圈**
  - 感受器：**螺旋器**（Corti器）-> 由毛细胞和支持细胞组成
- ★基底膜振动与行波学说（必考）
  - 感音换能链：
    - 声波 -> 鼓膜 -> 听骨链 -> 前庭窗膜振动 -> 耳蜗淋巴液 -> **基底膜振动**
    - -> 基底膜与盖膜间相对位置变化 -> **听毛弯曲** -> 毛细胞感受器电位 -> 蜗神经 AP
  - ★行波特征（频率编码）：
    - **高频**声波 -> 行波传播**近** -> 感受部位：耳蜗**底部**
    - **低频**声波 -> 行波传播**远** -> 感受部位：耳蜗**顶部**
    - 基底膜底部感受**高音**/顶部感受**低音**
- ★耳蜗两种生物电现象（必考）
  - 耳蜗内电位（内淋巴电位）：
    - 未受声波刺激时 -> 不同部位电位差
    - 毛细胞静息电位 **-70~-80mV**
    - 耳蜗内电位（内淋巴）**+80mV**
    - 毛细胞顶部的电位差 ≈ **150~160mV** -> 增加对声波敏感度
  - 耳蜗微音器电位：
    - 声波刺激 -> 类似微音器产生的电变化
    - 多个毛细胞**感受器电位的复合电位**
    - 特点：**可以总和** -> 触发蜗神经 AP；波形频率与声波**一致**；无不应期/无潜伏期/不易疲劳
- 听神经动作电位
  - 通过神经冲动的**节律、间隔时间**传递声音信息
  - 编码方式：频率编码 + 部位编码
- 临床联系
  - 耳蜗底部受损 -> **高频听力损失**（噪声性聋典型表现）
  - 耳蜗顶部受损 -> **低频听力损失**
  - 微音器电位可总和 -> 人工耳蜗的电刺激基础
