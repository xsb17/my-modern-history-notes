# 第三章核心逻辑：辛亥革命与君主专制制度的终结

> 本章讲述的是以孙中山为代表的资产阶级革命派如何举起近代民族民主革命的旗帜，通过辛亥革命推翻了两千多年的封建君主专制制度，建立了中国历史上第一个资产阶级共和政府——以及这场革命最终失败的原因与教训。

---

## 一、辛亥革命爆发的历史条件

### （一）民族危机加深，社会矛盾激化

**20世纪初，帝国主义的侵略进一步加剧：**

1. **掠夺铁路修筑权**
   > 日本《朝日新闻》指出："铁路所布，即权力所及。凡其他之兵权、商权、矿权、交通权，左之右之，存之亡之，操纵于铁路两轨，莫敢谁何。故夫铁道者，犹人之血管机关也，死生存亡系之。有铁路权，即有一切权；有一切权，则凡其地官吏，皆吾颐使之奴，其地人民，皆我俎上之肉。"

   **列强在华投资铁路情况：**

   | 国名 | 在中国投资的铁路 | 共计 |
   |------|-----------------|------|
   | 英国 | 2800英里 |  |
   | 俄国 | 1530英里 |  |
   | 德国 | 720英里 | **6420英里** |
   | 比利时 | 650英里 |  |
   | 法国 | 420英里 |  |
   | 美国 | 300英里 |  |

2. **攫取采矿权**
3. **对华贸易和资本输出恶性膨胀**
4. **1903—1904年英国侵入西藏**
5. **1904—1905年日、俄为争夺在华利益在中国东北进行战争，清政府宣称"局外中立"**

> 💡 **补充说明——"局外中立"的奇耻大辱：** 两个外国在中国领土上打仗，清政府却被迫宣布"局外中立"，任由战火在中国大地上燃烧。这充分说明清政府对国家主权的维护能力已彻底丧失，其统治的合法性受到严重动摇。

**国内民变频发与社会运动高涨：**
- 从1902年至1911年间，各地较大规模的民变多达 **1300余起**
- **拒俄运动**（抗议沙俄侵占东北）
- **抵制美货运动**（抗议美国排华）
- **收回利权运动**（收回被列强掠夺的路权、矿权）

### （二）清末"新政"及其破产

**时间线：**
| 时间 | 事件 |
|------|------|
| 1901.4 | 清政府成立**督办政务处**，宣布实行"新政" |
| 1906 | 宣布**"预备仿行宪政"** |
| 1908 | 颁布**《钦定宪法大纲》** |
| 1911 | 成立**责任内阁**（"皇族内阁"） |

**清末"新政"的内容：**
| 改革领域 | 具体措施 |
|----------|----------|
| **改革官制** | 设立商部、学部、巡警部等中央行政机构 |
| **改革军制** | 裁撤绿营，**建立新军** |
| **改革经济** | 颁布商法商律，奖励工商 |
| **改革教育** | 鼓励留学，颁布新的学制，下令从1906年起**正式废除科举考试** |
| **预备立宪** | 1906年宣布"预备仿行宪政"，1908年颁布《钦定宪法大纲》 |

> 💡 **补充说明——最后两场科举考试：** 1903年的癸卯会试和1904年的甲辰会试，是在河南贡院举行的最后两场全国科举考试。1905年，清政府正式下令废除科举制度，标志着延续1300多年的科举制度走到了尽头。但讽刺的是，废除科举并没有拯救清王朝，反而切断了传统士绅与清政府的联系纽带，加速了其统治基础的瓦解。

**五大臣出洋考察（1905年）：**
- 清政府派遣五大臣出洋考察宪政
- 考察报告总结立宪的"三大好处"：**"皇位永固"、"外患日轻"、"内乱可弥"**
- 清政府立宪的真实目的是维护皇权，而非真正实行民主

> 💡 **补充说明——五大臣出洋的真实目的：** 清廷派遣载泽、端方等五大臣出洋考察，表面上是考察各国宪政，实际上是想寻找一种既能维持满清皇室特权、又能平息革命风潮的"两全之策"。载泽在密折中明确说立宪可以使"皇位永固"——这揭示了清廷立宪的虚伪性。

**立宪派的活动：**
- 1906年"预备仿行宪政"谕旨后，各地出现各种立宪组织
- 活跃人物：**郑孝胥、张謇、汤化龙、谭延闿**等
- 立宪派最初寄希望于清政府的宪政改革

**皇族内阁（1911年5月）：**
- 成立责任内阁，13名大臣中**满族9人**，其中**皇族占5人**
- 被讥为**"皇族内阁"**
- 后果：立宪派大失所望；满汉矛盾激化；中央与地方的矛盾加深

**新政的意外后果——清王朝自掘坟墓：**
```
编练新军 → 袁世凯北洋势力崛起
派遣留学 → 长江流域新军革命化倾向 → 革命党人骨干
预备仿行宪政 → 立宪派转向共和革命
```
清政府原本想通过新政维护统治，结果却培养了自己的掘墓人。

> 💡 **补充说明——袁世凯与小站练兵：** 袁世凯在天津小站编练新军，采用西式训练方法，培养了段祺瑞、冯国璋、徐世昌等一大批北洋系军官。这支新军不但是清政府最后的武力支柱，也成了后来袁世凯逼宫清廷、窃取辛亥革命果实的资本。

### （三）资产阶级革命派的阶级基础和骨干力量

**经济基础：** 甲午战争后民族资本主义初步发展，中国资本每年新设工矿企业数量持续增长。代表人物如**张謇**（民族实业家）。

**阶级基础：** 民族资产阶级（主要是中下层）力量壮大，要求政治变革。

**骨干力量——资产阶级、小资产阶级知识分子：**
- **新式学堂学生**：接受近代教育，具有新思想
- **留学生**：在海外接触了民主共和思想（日本成为革命党人聚集地）
- **维新运动的影响**：戊戌变法的失败使更多知识分子转向革命道路

## 三、社会变革的十个标志

PPT 指出辛亥革命后中国社会发生的十大变化：

| # | 变化 | 意义 |
|---|------|------|
| 1 | **剪掉辫子** | 男人从此改头换面 |
| 2 | **不再缠足** | 女人终于可以大步走路 |
| 3 | **换新衣服** | 中山装替代了小马褂儿 |
| 4 | **改称谓** | 不再叫"老爷"请叫"先生" |
| 5 | **男女无别** | 可以一起坐车还能牵手逛街 |
| 6 | **公元纪年** | 采用阳历与世界接轨 |
| 7 | **禁除鸦片** | 再也不做"东亚病夫" |
| 8 | **不再三妻四妾** | 一夫一妻流行开来 |
| 9 | **不再跪拜** | 平等就是不要卑躬屈膝 |
| 10 | **改良新剧** | 女伶也可以参加公演 |

> 一个伟人的出现，两种制度的更替；三民主义的提出，四海人民的觉醒。

## 四、核心时间轴

| 时间 | 事件 | 意义 |
|------|------|------|
| **1894.11** | 孙中山在檀香山成立**兴中会** | 第一个民主革命团体；提出"振兴中华" |
| **1901.4** | 清政府成立督办政务处，宣布"新政" | 试图自救 |
| **1902—1911** | 各地民变达1300余起 | 社会矛盾全面激化 |
| **1904—1905** | 日俄战争，清政府"局外中立" | 国权丧尽 |
| **1905.8.20** | **中国同盟会**在东京成立 | 第一个全国性统一的资产阶级革命政党 |
| **1905.11** | 孙中山在《民报》发刊词中概括**三民主义** | 革命纲领确立 |
| **1905—1907** | 革命派与改良派大论战 | 划清革命与改良的界限 |
| **1906** | 清政府宣布"预备仿行宪政" | 虚伪立宪的开始 |
| **1906** | 正式废除科举考试 | 结束1300多年科举制 |
| **1908** | 颁布《钦定宪法大纲》 | 以宪法名义维护皇权 |
| **1911.4.27** | **黄花岗起义** | 72烈士葬于黄花岗 |
| **1911.5** | "皇族内阁"成立 | 立宪派彻底失望 |
| **1911.5—9** | **保路运动**（湘鄂粤川） | 四川最烈，清廷调湖北新军镇压 |
| **1911.10.10** | **武昌起义** | 辛亥革命爆发 |
| **1911年底** | 半数以上省份宣布独立 | 清王朝统治土崩瓦解 |
| **1912.1.1** | 孙中山在南京就任临时大总统 | **中华民国成立** |
| **1912.2.12** | 清帝退位 | 两千多年封建帝制终结 |
| **1912.3** | 颁布《中华民国临时约法》 | 第一部资产阶级共和国宪法性质的法典 |
| **1912.4.1** | 孙中山正式卸任 | 袁世凯窃取革命果实 |
| **1913** | **二次革命**（讨袁） | 失败 |
| **1915.12—1916.3** | 袁世凯称帝（83天）+ **护国运动** | 袁被迫取消帝制 |
| **1917.7—1918.5** | **护法运动** | 失败 |
