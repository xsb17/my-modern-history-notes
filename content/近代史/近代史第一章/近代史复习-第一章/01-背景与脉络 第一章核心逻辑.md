# 第一章核心逻辑：进入近代后中华民族的磨难与抗争

> 本章讲述的是 1840 年鸦片战争前后，中国从封闭自守的封建帝国被迫卷入世界资本主义体系的过程。核心逻辑是：**为什么曾经领先世界的中国会在近代落后挨打？西方列强是如何一步步侵略中国的？中国人民又是如何反抗和觉醒的？**

---

## 一、宏观历史坐标系：东西方的大分流

### （一）古代中国的辉煌

**中华民族具有悠久的文明历史：**
- 是世界上少有的**历史文化从未间断、一直延续至今**的国家
- 体现了中华民族的凝聚力和以爱国主义为核心的民族精神

**古代中国的科学技术长期处于世界领先地位：**

| 四大发明 | 世界影响 |
|----------|----------|
| **造纸术** | 促进文化传播与教育普及 |
| **指南针** | 打开世界市场并建立殖民地 |
| **火药** | 把骑士阶层炸得粉碎 |
| **印刷术** | 变成新教的工具，科学复兴的手段 |

> 马克思指出："火药、指南针、印刷术——是预告资产阶级社会到来的三大发明。火药把骑士阶层炸得粉碎，指南针打开了世界市场并建立了殖民地，而印刷术则变成新教的工具，总的来说，变成科学复兴的手段，变成对精神发展创造必要前提的最强大的杠杆。"

**李约瑟的评价：**
> "在3—13世纪，中国保持一个让西方人望尘莫及的科学知识水平。""中国古代的发明和发现往往是超过同时代的欧洲，特别是15世纪以前更是如此，这可以毫不费力地加以证明。"

**古代中国的哲学思想博大精深——"轴心时代"（公元前5世纪）：**

| 文明 | 代表人物 |
|------|----------|
| 希腊 | 苏格拉底、柏拉图、亚里士多德 |
| 以色列 | 犹太教的先知们 |
| 印度 | 释迦牟尼 |
| **中国** | **孔子、老子、墨子** |

> 地理轴心：北纬 25°—35°

**孙中山的评价：**
> "五百年前，中国是世界上顶强的国家。国家的地位，好像现在的英国、美国一样。英国、美国的强盛还是列强。中国从前的强盛是独强。对各弱小民族，中国完全是用王道感化他们。他们是怀中国之德，甘心情愿，自己来朝贡的。"

### （二）中国封建社会的基本特征

| 维度 | 特征 |
|------|------|
| **经济特征** | 封建地主土地所有制经济占主导地位 |
| **政治特征** | 高度中央集权的封建君主专制制度 |
| **文化特征** | 以儒家思想为正统 |
| **社会特征** | 族权和政权相结合的封建宗法等级制度 |

**政治特征——中央集权的君主专制制度（"屁股文化"的演变）：**
- 秦朝到北宋：**坐着上朝**（大臣有席位）
- 北宋以后：**站着上朝**（皇帝权威加强）
- 明清时期：**跪着上朝**（君主专制达到顶峰）

**文化特征——儒家思想的演变：**

| 阶段 | 时期 | 思想来源 |
|------|------|----------|
| **原儒** | 先秦 | 孔 + 孟 + 荀（儒家本原） |
| **汉儒** | 汉代 | 儒 + 墨 + 法（独尊儒术） |
| **宋儒** | 宋明 | 儒 + 释 + 道（理学） |
| **现代新儒家** | 近代以来 | 儒 + 西方文化 |

- 自汉武帝确立**独尊儒术**的政策，儒家思想开始成为中国封建社会的正统思想。
- 两千多年来，**国家的统一和各族间经济文化的紧密联系和相互交流**是中国历史的主流。

### （三）西方世界的崛起

**西方现代化的发端：**

```
文艺复兴 → 宗教改革 → 大航海时代 → 商业革命 → 工业革命 → 近代政治经济制度 → 发达的资本主义 → 对外殖民扩张与掠夺
```

- **1640年**：英国资产阶级革命标志着世界历史开始进入资本主义时代
- **18世纪**：继英国之后，美国、法国等先后通过资产阶级革命，建立了资产阶级政权

**地理大发现与世界格局的改变：**

| 中国 | 西方 |
|------|------|
| 郑和下西洋（"强而不欺，威而不霸"） | 哥伦布、达·伽马——殖民掠夺 |
| "郑和之后，再无郑和"（梁启超语） | 地理大发现 → 商业革命 |
| 朝贡体系，王道感化 | 殖民体系，武力征服 |

> 南非一位著名政治家曾说过：西方人来到我们面前时，手中拿着圣经，我们手中有黄金。后来就变成了，他们手中有黄金，我们手中拿着圣经。

**罪恶的奴隶贸易：** 装满黑奴的贸易船横跨大西洋，成为资本主义原始积累的重要来源。

**工业革命（18世纪中叶至19世纪中叶）：**
- 从英国开始，迅速推广到欧美各国
- 大机器生产取代了工场手工业
- 资本主义经济得到迅速发展

**英国煤产量、英国生铁产量、美国纺织业纱锭数量**均出现爆发式增长。

> 马克思、恩格斯对资本主义生产方式的评价："资产阶级在它的不到一百年的阶级统治中所创造的生产力，比过去一切世代创造的全部生产力还要多，还要大。"——《共产党宣言》

> 💡 **补充说明——钟表与工业革命：** 芒福德认为，对工业革命影响最大的发明或许并非蒸汽机，而是钟表。钟表令劳动不再以天（或季度），而是以小时（甚至分钟）为单位被组织起来。工业社会的时间表进而塑造了"一日三餐"和"按时睡觉"的习惯。

### （四）马嘎尔尼访华（1793年）——东西方碰撞的预演

- 1793年，英国特使**马嘎尔尼**以给乾隆皇帝祝寿为名访华
- 他带来了英国的最新工业产品，希望打开中国市场
- 乾隆皇帝以"天朝物产丰盈，无所不有"为由拒绝

> 马嘎尔尼在"失败"的中国之行后说：**"中华帝国只是一艘破旧的老船，很幸运。有一些谨慎的船长近150年没有沉没它。它巨大的身体使周围的邻居感到恐惧。如果一个不称职的人掌舵，船上的纪律和安全将结束了。"**

**核心启示：** 能否在历史关键节点判断准、利用好战略机遇，将对一个国家和民族的前途命运产生全局性、长远性、决定性影响。

### （五）19世纪初的中西力量对比

| 对比项 | 中国（清朝） | 英国（西方） |
|--------|-------------|-------------|
| 政治 | 封建君主专制，高度集权 | 资产阶级议会制 |
| 经济 | 小农经济，男耕女织 | 工业革命，大机器生产 |
| 军事 | 冷热兵器混用，木质帆船 | 全部热兵器，蒸汽铁甲舰 |
| 科技 | 传统手工技术 | 近代科学体系 |
| 外交 | 闭关锁国，朝贡体系 | 全球殖民扩张 |
| 思想 | 儒家纲常，天朝上国 | 启蒙思想，自由主义 |

## 二、鸦片战争爆发的历史必然性

### （一）根本原因：英国资本主义扩张的需要

- 19世纪初，英国已基本完成**工业革命**，成为世界最强大的资本主义国家，建立了号称**"日不落"**的殖民大帝国
- 继占领印度之后，**中国成为它主要的侵略目标**
- 英国对华贸易长期处于**入超状态**（贸易逆差）

### （二）直接原因：鸦片走私与虎门销烟

**特殊商品：鸦片**
- 英国以走私毒品**鸦片**作为牟取暴利及改变贸易逆差的手段
- 英国输入中国鸦片数量激增（每箱50—100千克）

**鸦片带来的三大危害：**
1. **造成了中国的白银大量外流和财政危机**
2. **导致银贵钱贱，加重了劳动人民的负担**
3. **直接毒害了中国人的身体和精神**

> 林则徐："鸦片为中国之巨害，其杀吾国民，甚于干戈、瘟疫、饥馑之患。"

**虎门销烟（1839年6月）：**
- 钦差大臣**林则徐**于1839年6月在广东虎门销毁所收缴鸦片
- 1839.6.3—25 在广州附近的虎门海滩，当众销毁英商鸦片 **2万多箱**
- 完全是维护国家利益和民族尊严的正义行动

> 💡 **补充说明——今昔对比的深意：** PPT 展示了 2021 年 8 月 10 日，辽宁省高级人民法院对加拿大籍被告人罗伯特·劳埃德·谢伦伯格走私毒品上诉一案依法进行二审公开宣判，裁定驳回上诉，维持原判。这凸显了"现在的中国，任何外籍身份都不是'护身符'"——与鸦片战争时期形成鲜明对比。

### （三）战争爆发

- **1840年4月**：英国国会通过对华战争的决定
- **1840年6月**：英国侵华舰队4000余人从印度等地陆续到达中国海面，封锁了珠江入海口和广东海面，**鸦片战争正式爆发**
- **鸦片战争以清政府的失败而告终**
- 1842年《南京条约》签订

### （四）鸦片战争的意义——中国近代史的起点

> 鸦片战争后，**中国的封建社会逐步变成了半殖民地半封建社会**。中国人民逐渐开始了反帝反封建的资产阶级民主革命。

## 三、近代中国社会的深刻变化

### （一）社会性质的变化：半殖民地半封建

| 变化 | 独立/封建的中国 | → | 半殖民地半封建的中国 |
|------|---------------|----|-------------------|
| **半殖民地** | 领土、领海、关税、司法等主权完整 | → | 主权被破坏，政治、经济、外交、军事被控制；但仍维持独立国家和政府的名义 |
| **半封建** | 纯粹的封建社会 | → | 出现了资本主义生产关系；但外国资本及官僚资本居于支配地位，农村封建生产关系仍占优势 |

**半殖民地半封建社会的六个特征：**
1. 资本-帝国主义侵略势力日益成为支配中国的决定性力量
2. 封建地主的土地所有制成为中国走向现代化和民主化的严重障碍
3. 近代中国各地区经济、政治和文化的发展是**极不平衡的**
4. 中国的封建势力日益衰败并同外国侵略势力勾结
5. 中国新兴的民族资本主义经济虽然已经产生，但是**发展很缓慢，力量很软弱**
6. 中国的广大人民过着**饥寒交迫和毫无政治权利的生活**

### （二）社会阶级关系的变动

**旧的阶级：**
| 阶级 | 变化 |
|------|------|
| **地主阶级** | 继续占有大量土地；部分转化为**城居地主**，兼营土地、高利贷和工商业；附股外资企业、入股洋务企业、创办或参股民营企业→**资本家** |
| **农民阶级** | 仍是人数最多的被剥削阶级；贫农或雇农；大量破产、失去土地，流入城市→**产业工人的后备军**；**近代中国民主革命的主力军** |

**新兴阶级：**
| 新阶级 | 特征 |
|--------|------|
| **工人阶级** | 来源：城乡破产失业的农民、手工业者和城市贫民；**身受帝国主义、封建势力、资产阶级三重压迫**；受剥削最深、革命性最强、组织纪律性强、集中、团结、与广大农民有天然联系→**近代中国最革命的阶级** |
| **资产阶级** | 主要由买办、商人、地主、官僚投资新式企业转化而成；分为**官僚买办资本家**和**民族资本家**（民族资本家在政治上具有**两面性**） |

### （三）社会主要矛盾和两大历史任务

**两对主要矛盾：**
1. **帝国主义和中华民族的矛盾**（最主要的矛盾）
2. **封建主义和人民大众的矛盾**

> 这两对主要矛盾及其斗争贯穿整个中国半殖民地半封建社会的始终，并对中国近代社会的发展变化起着决定性的作用。

**两大历史任务：**
1. **民族独立和人民解放**——必须推翻帝国主义、封建主义联合统治的半殖民地半封建的社会制度
2. **国家的富强和人民的富裕**——必须改变中国经济技术落后的面貌

## 四、核心时间轴

| 时间 | 事件 | 意义 |
|------|------|------|
| **1640** | 英国资产阶级革命 | 世界进入资本主义时代 |
| **18世纪中叶—19世纪中叶** | 工业革命 | 大机器生产取代工场手工业 |
| **1793** | 马嘎尔尼访华 | 东西方力量对比的预演 |
| **1839.6** | **虎门销烟** | 林则徐，维护民族尊严 |

| **1883—1885** | 中法战争 | "中国不败而败" |

| **1898—1899** | 列强划分势力范围 | 瓜分狂潮 |
