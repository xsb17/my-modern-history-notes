# 01-背景与脉络 日本侵华与全民族觉醒

> [!quote] 本章主线
> 第六章是全书最恢弘的一章——从**日本蓄谋已久的侵华战争**，到**全民族奋起抗战**，再到**正面与敌后两个战场**的协同作战，最终以**抗战伟大胜利**洗刷百年国耻。这既是中华民族的苦难史，更是从觉醒到崛起的转折史。

## 二、日本大陆政策的历史渊源

### （一）大陆政策的形成

**大陆政策**是日本自明治维新后开始形成的，立足于用战争手段侵吞周边大陆国家，进而征服世界的对外扩张政策。

| 时期 | 事件 | 意义 |
|------|------|------|
| 14世纪 | "倭寇"问题 | 早期侵扰 |
| 16世纪后期 | 丰臣秀吉统一日本 | 对外扩张思想萌芽 |
| 明治维新后 | 确立"脱亚入欧"和"开国进取"口号 | 逐步确立以武力征服世界的方针 |
| 1871年 | 《中日修好条约》 | 基本上对等的双边条约 |
| 1873年 | "征韩论"盛极一时 | 对朝鲜野心暴露 |
| 1874年 | 出兵台湾 | **近代日本第一次向海外派兵** |
| 1876年 | 《日朝修好条规》（江华条约） | 强迫朝鲜开埠 |
| 1879年 | 吞并琉球，建立冲绳县 | 第一步领土扩张 |
| 1894—1895 | **甲午战争**《马关条约》 | 割占台湾及附属列岛，赔款2.3亿两白银 |
| 1901年 | 《辛丑条约》 | 日本分得3479.31万两白银 |
| 1904—1905 | **日俄战争**《朴茨茅斯和约》 | 占领辽东半岛 |
| 1910年 | 《日韩合并条约》 | 吞并朝鲜，设置朝鲜总督府 |

> 💡 **补充说明：** 大陆政策不是某一天突然形成的，而是日本自明治维新以来根深蒂固的扩张国策。日本是一个岛国，对土地的占有欲望极强。明治维新后日本走上资本主义道路，逐渐发展为**军国主义国家**，实行"北进"（大陆政策）和"南进"（海洋政策）两条扩张路线。从1874年出兵台湾到1937年全面侵华，日本用**63年**步步紧逼，每步都服务于更大的侵略目标。

### （二）东方会议与《田中奏折》

| 项目 | 内容 |
|------|------|
| 时间 | **1927年** |
| 召集人 | 日本首相**田中义一** |
| 会议 | "东方会议" |
| 产物 | 《对华政策纲要》——企图把"满蒙"从中国本土彻底分割出去 |

> **《田中奏折》核心名言：**
> **「惟欲征服支那，必先征服满蒙；如欲征服世界，必先征服支那。支那完全被我国征服，其他为小亚细亚及印度南洋等异服之民族，必畏我而降于我。」**

### （三）1929—1933年经济大危机的影响

资本主义世界经济大危机使日本国内矛盾激化：
> 「帝国内部现正痛切体验……生产力之不足，生产设备之不足，原料之不足及蓄积资本之不足。凡此均需开始新的战争，俾有所补充；而国内不稳势力之高涨，罢工之狂澜，亦需发动内外举国之战争以期平靖。」
> ——30年代日本秘密文件

> 💡 **补充说明：** 这段话暴露了日本发动战争的两大动机：**①掠夺资源**（解决生产力不足、原料不足）；**②转移国内矛盾**（平息罢工和国内不稳势力）。这就是为什么日本会从"满蒙"开刀——东北地区丰富的煤铁资源正是日本急需的。

```

1914年  ─── 一战期间侵占山东

1931.9.18 ─── 九一八事变（抗日战争起点）
1932.1   ─── 一二八事变（十九路军抵抗）
1932.3   ─── 伪"满洲国"成立（溥仪傀儡政权）
```

### （一）九一八事变（1931年9月18日）

| 项目 | 内容 |
|------|------|
| 事件 | 日军炸毁南满铁路柳条湖段，诬陷中国军队 |
| 结果 | 至1932年2月，**中国东北全境沦陷** |
| 傀儡政权 | 1932年3月1日，建立伪"满洲国"，扶植**溥仪**为傀儡皇帝 |
| 历史定位 | **九一八事变是中国人民抗日战争的起点** |

> **确立"十四年抗战"的四大重要意义：**

| 序号 | 意义 |
|------|------|
| **1** | **有利于完整还原抗日战争的历史过程** |
| **2** | **有利于客观反映中国共产党的中流砥柱作用** |
| **3** | **有利于正确认识中国抗日战争在世界反法西斯战争中的重要地位和作用** |
| **4** | **有利于大力弘扬伟大的抗战精神** |

> 💡 **补充说明：** 过去长期使用"八年抗战"（从1937年七七事变算起），2017年起教材改为"十四年抗战"（从1931年九一八事变算起）。这一改动意义重大——它不仅更符合历史事实（东北人民从1931年起就在抵抗），也凸显了中国抗战在世界反法西斯战争中**开始最早、持续时间最长**的地位。同时，九一八事变"标志着第二次世界大战两个策源地之一——**远东战争策源地**的最早形成"。

### （二）一二八事变（1932年）

- **日军进攻上海**
- **十九路军**奋起抵抗（蔡廷锴、蒋光鼐指挥）
- 间岛芳子等日本间谍参与其中

### （三）华北事变（1935年）

日本在华北制造一系列事端，中国在**河北、察哈尔**两省的主权大部丧失。接着，日本策动**华北五省两市**（河北、察哈尔、绥远、山西、山东 + 北平、天津）"防共自治运动"，制造傀儡政权。

### （四）七七事变（1937年7月7日）

> 1936年8月，法西斯军人控制的**广田弘毅内阁**制定了"南攻南洋群岛、北攻西伯利亚""先打中国"的侵略计划。11月，日本同德国签订《反共产国际协定》，欧亚两个法西斯国家打着"反共"旗号结成反动同盟。

1937年7月7日夜，驻丰台日军在军事演习中，借口一名士兵失踪，要求进入宛平县城搜查，遭拒绝后即开炮轰城，向中国驻军进攻。**日本全面侵华战争由此开始。**

> 💡 **补充说明：** "七七事变"不是偶然事件，而是日本长期侵华计划的一个环节。1936年日本已经制定了全面的侵华方案，并与德国结成了法西斯同盟。所谓"士兵失踪"只是借口——事实上那名士兵很快就归队了，但日军仍然发动了攻击。

## 四、日本帝国主义的残暴统治

> 「两千年来，中国施之于日本者甚厚，有造于日本者至大，百年来日本报之于中国者极酷，为祸中国者独深。近代中国所遭受的创痛，虽然不能说全都来自于日本，但实际上以日本所给予的最多最巨。」
> ——郭廷以

### （一）南京大屠杀

| 项目 | 内容 |
|------|------|
| 时间 | 1937年12月，日军占领南京后 |
| 暴行 | 展开烧、杀、淫、掠"大竞赛" |
| 死难者 | **30万人以上** |

### （二）细菌战与化学战

| 暴行 | 具体内容 |
|------|----------|
| 细菌战 | 日军悍然实行细菌战、化学战 |
| 活体实验 | 对中国军民实行惨无人道的活体试验（**731部队**） |
| 毒气战 | 在战场上大量使用毒气 |
| 慰安妇 | 强迫大量中国妇女充当"慰安妇" |
| 劳工 | 掳掠和残害中国劳工 |

### （三）疯狂掠夺中国资源和财富

| 领域 | 手段 |
|------|------|
| **东北垄断企业** | "南满铁路株式会社""满洲重工业股份公司"独占全部重工业和铁路交通，控制东北经济命脉 |
| **关内垄断企业** | "华北开发股份公司""华中振兴股份公司"分别主管对华北和华中的经济掠夺 |
| **农业** | 强占耕地，低价收购农产品 |
| **工矿交通运输** | 通过直接控制、"军事管理""委托经营""中日合办"等方式控制中国工矿企业和交通运输业 |
| **金融** | 抢占银行，掠夺金银和现款，开设金融机构，滥发纸币，增加苛捐杂税 |
| **劳动力** | 强制劳役、强制征兵 |
| **资源数据** | 到1942年，东北的钢产量占日本总产量的**1/3**，铝产量占日本总产量的**40%**，日本所需的38种军需物资有**14种**由东北供给 |

> 💡 **补充说明：** 日本在东北的掠夺完全是为战争服务的。1936年11月，关东军确定了《满洲产业开发五年计划现地案》，计划重点开发与战争密切相关的铁、煤、石油、电力等重工业。东北实际上成了日本的**战争物资供应基地**。

### （四）强制推行奴化教育

| 手段 | 目的 |
|------|------|
| 强令学习日语，"中日亲善" | 泯灭中国民众的民族意识和反抗精神 |
| 强迫学生每顿饭前祈祷，感谢日本天皇 | 培养对天皇的效忠 |
| 台湾推行"皇民化"运动 | 使台湾同胞日本化，支撑侵略战争 |

> 💡 **补充说明：** PPT特别指出——台湾的"皇民化"运动"造就了一群敌视祖国的亲日'皇民'，其被奴化的思想是**第一代'台独'的重要思想来源**"。这是一个重要的考点，将殖民历史与"台独"的根源联系起来。

### （五）战争损失数据

> **据不完全统计，战争期间，中国军民伤亡人数超过3500万；按1937年的比价，中国直接经济损失1000多亿美元，间接经济损失5000多亿美元。**

## 六、抗日民族统一战线的建立与全民族抗战

### （一）一二·九运动（1935年12月9日）

北平学生喊出「**反对华北自治运动**」「**打倒日本帝国主义**」「**停止内战，一致对外**」等口号，举行示威游行。

- **意义：** 促进了中华民族觉醒，标志着**中国人民抗日救亡运动新高潮的到来**

### （二）西安事变（1936年12月12日）

| 项目 | 内容 |
|------|------|
| 背景 | 蒋介石到西安逼张学良、杨虎城"剿共" |
| 经过 | 张、杨在对蒋介石"哭谏"无效的情况下实行"**兵谏**"，扣留蒋介石 |
| 中共方针 | 派**周恩来**等到西安参加谈判，迫使蒋介石作出"**停止剿共、联红抗日**"的承诺 |
| 意义 | **西安事变的和平解决成为时局转换的枢纽，十年内战局面基本结束** |

### （三）国共合作，共赴国难

> 毛泽东：「每个共产党员和抗日的革命者应该沉着地完成切必要的准备，随时出动，到抗日前线。」

> 蒋介石：「我们希望和平而不求苟安，准备应战而决不求战。我们知道全国应战以后的时势，就只有牺牲到底，无丝毫侥幸求免之理。」

### （四）全民族同仇敌忾

| 人物 | 事迹 |
|------|------|
| **邓玉芬** | 北京"英雄母亲"，先后把丈夫和5个孩子送上前线，**全部战死沙场**，被誉为"当代的佘太君" |
| **马本斋** | 创建八路军冀中军区**回民支队**，与日伪军进行大小战斗**870多次**，毛泽东称其为"**百战百胜的回民支队**" |
